# ロール定義と権限

## 目的
- 役割ごとに利用できる機能/装置エージェントを明文化し、運用フローに利用する

## ロール一覧
- Administrator
  - 役割・装置エージェント割り付けの管理者
  - 全装置エージェントを常に利用可能
- Developer
  - 開発者向けの拡張権限
  - 全装置エージェントを常に利用可能
- Operator
  - 運用担当ロール
- Development
  - 開発部門共通ロール
- MechanicalDesign
  - 機械設計ロール
- ElectricalDesign
  - 電気設計ロール
- Manufacturing
  - 製造ロール

## 権限の付与方法
- ロール付与/削除: UI `ロール管理` からターゲットユーザーを選択してチェックを保存
- 装置エージェント利用許可:
  - Administrator / Developer は割り付けに関わらず全エージェントを選択可能
  - それ以外のロールは、ロール管理画面で装置エージェントを割り付けると選択肢に表示される

## 運用メモ
- ロールIDは大文字で正規化されるため、指定時の大小は不問
- 装置エージェント割り付けはユーザー単位で上書き保存され、既存の割り付けは保存時に置き換わる
