# ガイダンス（続き）

## 2. 原点復帰後の確認

原点復帰後、[指令位置]が0になっていることを確認します。

**ティーチング・リモート運転ウィンドウ表示：**
- 指令位置(CPOS)：0 [step]
- 検出位置：0 [step]
- 検出速度：0 [Hz]

**ステータス表示：**
- INPUT: C-ON, FREE, STOP
- OUTPUT: SRINT, MBC, MOVE, MPS

## ● 高速原点復帰運転の運転条件を変更したいとき

### 1. パラメータ設定画面へのアクセス

画面の左側にあるツリービューから、[パラメータ] - [モーター・機構(座標)/JOG/原点復帰]設定をクリックします。
モーター・機構パラメータが表示されます。

**ツリービュー構造：**
```
📁 データ
  📁 運転データ
  📁 運転プロパティ
  📁 運転データ一般属性設定
📁 パラメータ
  📁 基本設定
    📁 モーター・機構(座標)/JOG/原点復帰設定 ←選択
    📁 アラーム関連・その他設定
    📁 A/D入力・情報
    📁 Detect-IN関連・選択信号(IN)
    📁 Detect-OUT関連・選択信号(OUT)
    📁 Remote-I/O関連・接続信号(I/O)
    📁 EXT-I/O・WS-HUB・USB-OUT関連・拡張信号(EXT)
    📁 通信・RF通信
```

### 2. パラメータの設定変更

[JOG/HOME/Z-HOME運転 運転情報設定]パラメータを[マニュアル設定]に変更します。

### 3. ZHOME運転条件パラメータの変更

次の3種類のパラメータで、運転条件を変更します。

| パラメータ番号 | パラメータ名 | 設定値 |
|----------------|--------------|---------|
| 29 | (ZHOME)運転速度 [Hz] | 5000 |
| 30 | (ZHOME)加減速度 [kHz/s] | 1000.000 |
| 31 | (ZHOME)移動速度 [Hz] | 500 |

### 4. ドライバへのダウンロード

運転条件を変更したら、ツールバーの[データの書き込み]アイコンをクリックして、ドライバにダウンロードします。

**データ書き込みアイコン：** 📤📱💾

## STEP 3 ティーチング・リモート運転を終了します

ティーチング・リモート運転を終了するときは、[ティーチング・リモート運転を開始する]のチェックを外してください。

**ティーチング・リモート運転ウィンドウ（終了時）：**
- ☑ ティーチング・リモート運転を開始する ← チェックを外す
- 指令位置(CPOS)：0 [step]
- 検出位置：0 [step]
- 検出速度：0 [Hz]

**ステータス表示：**
- INPUT: C-ON, FREE, STOP
- OUTPUT: SRINT, MBC, MOVE, MPS
