## 接続ケーブルセット / 可動接続ケーブルセット
**（AZM46、AZM48、AZM66、AZM69用）**

モーターとドライバを接続するときに使用します。モーター用とエンコーダ用の2本組です。
電磁ブレーキ付モーター用は、モーター用、エンコーダ用、および電磁ブレーキ用の3本組です。

```mermaid
graph TD
    A[モーター] --> B[接続ケーブルセット]
    B --> C[ドライバ]

    subgraph "接続ケーブルセット"
        direction TB
        D[エンコーダ用※1]
        E[モーター用]
        F[電磁ブレーキ用※2]
    end

    A --> D
    A --> E
    A --> F
    D --> C
    E --> C
    F --> C
```

※1 エンコーダ用ケーブルは、長さが足りないときにお使いください。
※2 電磁ブレーキ付モーターのとき。

**(memo)** モーターを可動部分に取り付けるときは、可動ケーブルを使用してください。
