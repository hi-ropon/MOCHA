# ガイダンス（続き）

## JOG運転ボタンの説明

| ボタン | 説明 |
|--------|------|
| ⏮ | 「(JOG)運転速度 (高)」パラメータに設定した運転速度で一方向へ連続運転します。 |
| ◀ | 「(JOG)運転速度」パラメータに設定した運転速度で一方向へ連続運転します。 |
| ⏸ | JOG運転ボタンの「最小移動量」に設定した移動量で一方向へ位置決め運転します。 |
| ⏹ | モーターが即停止します。 |
| ⏯ | JOG運転ボタンの「最小移動量」に設定した移動量で+方向へ位置決め運転します。 |
| ▶ | 「(JOG)運転速度」パラメータに設定した運転速度で+方向へ連続運転します。 |
| ⏭ | 「(JOG)運転速度 (高)」パラメータに設定した運転速度で+方向へ連続運転します。 |

JOG運転ボタンをクリックしてもモーターが動作しないときは、次の点を確認してください。
- 主電源、モーター、およびMEXE02が正しく接続されているか。
- アラームが発生していないか。

## ● JOG運転の運転条件を変更したいとき

### 1. パラメータ設定画面へのアクセス

画面の左側にあるツリービューから、[パラメータ] - [モーター・機構(座標)/JOG/原点復帰]設定をクリックします。
モーター・機構パラメータが表示されます。

**ツリービュー構造：**
```
📁 運転データ
  📁 運転プロパティ
  📁 運転データ一般属性設定
📁 パラメータ
  📁 基本設定
    📁 モーター・機構(座標)/JOG/原点復帰設定 ←選択
    📁 →アラーム関連・その他設定
    📁 →A/D入力・情報
    📁 →Detect-IN関連・選択信号(IN)
    📁 →Detect-OUT・情報・選択信号(OUT)
    📁 →Remote-I/O関連・接続信号(I/O)
    📁 →EXT-I/O・WS-HUB・USB-OUT関連・拡張信号(EXT)
    📁 →通信・RF通信
```

### 2. パラメータの設定変更

[JOG/HOME/Z-HOME運転 運転情報設定]パラメータを[マニュアル設定]に変更します。

### 3. 運転条件パラメータの変更

次の種類のパラメータで、運転条件を変更します。

| パラメータ番号 | パラメータ名 | 設定値 |
|----------------|--------------|---------|
| 24 | (JOG)移動量 [step] | 1 |
| 25 | (JOG)運転速度 [Hz] | 1000 |
| 26 | (JOG)加速度 [Hz/s] | 1000 000 |
| 27 | (JOG)移動時速度 [Hz] | 500 |
| 28 | (JOG)運転高速度 [Hz] | 5000 |

### 4. ドライバへのダウンロード

運転条件を変更したら、ツールバーの[データの書き込み]アイコンをクリックして、ドライバにダウンロードします。
