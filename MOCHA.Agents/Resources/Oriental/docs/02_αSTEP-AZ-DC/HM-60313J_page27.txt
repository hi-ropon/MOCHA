# 8 入出力信号の説明

## 8-1 入力信号

ドライバの入力信号は、すべてフォトカプラ入力です。各種の入力端に関する電圧レベルで区切く、内部フォトカプラの ON/
期間「OFF非共通」を表示しています。

### ■ CW [PLS] 入力、CCW [DIR] 入力

パルス入力で運転することに使用する信号です。
2 パルス入力方式のときは CW パルス、CCW パルス、1 パルス入力方式のときは PLS パルスと DIR パルスになります。使用する上
位システムのパルス出力方式に合わせて、ドライバのパルス入力方式を設定してください。パルス処入力するときは、PLS-
RDY出力が ON になってから使用できます。

#### パルス入力信号の波形

```
PLS-RDY出力 ON  ___________________
           OFF ___|               |___

パルス入力     ON  _|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_
              OFF |  |  |  |  |  |  |  |  |  |  |
                  0.3μs以上
```

**memo** モーター停止時は、必ずパルス入力をフォトカプラ OFF にしてください。

#### ● 最大入力パルス周波数

• 上位システムがラインドライバ出力の場合：1 MHz（デューティー50 %）
• 上位システムがオープンコレクタ出力の場合：250 kHz（デューティー50 %）

#### ● 2 パルス入力方式（出力制御設定）

CW入力を OFF から ON するとき、モーターが CW 方向へ 1 ステップ回転します。
CCW入力を OFF から ON するとき、モーターが CCW 方向へ 1 ステップ回転します。

```
波形図:
CW入力   ON  _____|‾‾‾‾‾|_____
        OFF      |     |
                 5 μs以上

CCW入力  ON  _____|‾‾‾‾‾|_____
        OFF      |     |

モーターの動き:  CW ←→ CCW
```

**重要** CW入力と CCW入力を同時に入力しないでください。モーターが正常に動作しなくなります。

#### ● 1 パルス入力方式

DIR入力が ON のとき、PLS入力を OFF から ON するとき、モーター
が CW 方向へ 1 ステップ回転します。
DIR入力が OFF のとき、PLS入力を OFF から ON するとき、モーター
が CCW 方向へ 1 ステップ回転します。

```
波形図:
PLS入力  ON  _____|‾‾‾‾‾|_____     |‾‾‾‾‾|_____
        OFF      |     |          |     |
                 5 μs以上         5 μs以上

DIR入力  ON  _____|‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|_____________
        OFF                        |

モーターの動き:    CW ←→ CCW
```

### ■ START入力

位置決めの運転を開始する信号です。パルス入力で運転するときは使用しません。
運転データ No. を選択した後、START入力を ON するとき、位置決めの運転が開始されます。
