## ■ RS-485 通信仕様

| 項目 | 仕様 |
|---|---|
| 電気的特性 | EIA-485規格、ストレートケーブル |
| | ツイストペア線(TIA/EIA-568B CAT5e以上を推奨)を使用し、終端抵抗を50 mAとすること。※ |
| 通信方式 | 非同期半二重通信 |
| | 標準調歩方式(データ：8 ビット、ストップビット：1 ビット、パリティ：なし、偶数、奇数) |
| 伝送速度 | 9,600 bps、19,200 bps、38,400 bps、57,600 bps、115,200 bps、230,400 bpsから選択 |
| プロトコル | Modbus RTUモード |
| 接続台数 | ネットワーク上における最大接続台数数：10 台(終端抵抗3フレーム、バイアリ抵抗 |
| 総端抵抗 | 上位システム1 台に対して、最大31 台まで接続できます。 |

※記載の電気的仕様ケーブルを共通するケーブルから先端するメインの区間に応じては、シールドオスフェライ・
トラックを使用してください。
