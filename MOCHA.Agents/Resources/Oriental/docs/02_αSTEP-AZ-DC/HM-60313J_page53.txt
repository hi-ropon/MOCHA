## 中継ケーブルセット / 可動中継ケーブルセット
**（AZM14、AZM15、AZM24、AZM26用）**

モーターと接続ケーブルを中継するときに使用します。
ドライバとモーターの間に、複数のケーブルをつなぎたい場合や、既存の接続ケーブルの長さが足りないときに使用します。

```mermaid
graph LR
    A[モーター] --> B[中継ケーブルセット] --> C[接続ケーブルセット] --> D[ドライバ]

    subgraph "システム構成"
        direction TB
        B
        C
    end
```

※ 使用している接続ケーブルセットをお選びください。

**(memo)**
・モーターを可動部分に取り付けるときは、可動ケーブルを使用してください。
・中継ケーブルを接続ケーブルに直接呼して継手するときは、ケーブル全長を20 mU下にしてください。
