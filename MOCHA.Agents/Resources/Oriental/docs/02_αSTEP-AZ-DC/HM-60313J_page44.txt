# ガイダンス

## パルス列入力タイプの場合

1. SW1 の No.2 を ONにします。

2. ドライバの主電源を再投入します。
   変更した設定が有効になります。

## 高速原点復帰運転（ZHOME運転）

MEX502 または HOME PRESETスイッチで確定した原点に復帰します。

1. 主電源を投入します。
   READY出力と PLS-RDY出力が ONになります。

2. READY出力が ONであることを確認し、ZHOME入力を ONにします。
   READY出力が OFFになり、モーターが高速原点復帰運転を始めます。

3. READY出力が OFFになったことを確認し、ZHOME入力を OFFにします。
   原点に到達すると、運転が停止します。
   HOME-END出力と READY出力が ONになります。

