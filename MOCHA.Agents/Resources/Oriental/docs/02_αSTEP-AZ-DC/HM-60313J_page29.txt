# 8-2 出力信号

ドライバの出力信号には、フォトカプラ・オープンコレクタ出力、ライン出力（バランス力）があります。信号の電圧は信号の
電圧レベルについて、内部フォトカプラの「ON・通電」「OFF・非通電」を表しています。

## HOME-END出力

原点が確定したとき、または原点復帰運転が完了したとき、HOME-END出力がONになります。

## IN-POS出力

位置決め運転の終了後、ローターが静止位置に対してIN-POS出力が切り替え時間（パラメータの値）の収束したときに、IN-POS
出力がONになります。

```
目標値             IN-POS出力が切り替え時間
                   (初期値1.6)
IN-POS出力 ON
          OFF
```

## PLS-RDY出力

パルス入力を運転するときに使用する信号です。
パルス入力による運転の準備が完了すると、PLS-RDY出力がONになります。PLS-RDY出力がONになってから、パルス
を入力してください。

## READY出力

運転の準備が完了すると、READY出力がONになります。READY出力がONになってから、運転開始信号をドライバに入
力してください。

## MOVE出力

モーターの動作中、MOVE出力がONになります。

## ALM-B出力

アラーム発生すると ALM-B出力がOFFになり、モーターが停止します。同時にドライバのPOWER/ALARM LEDが赤
色に点滅します。ALM-B出力は白点（ノーマルクローズ）です。

## ASG出力、BSG出力

ASG出力は、モーターの運転に対してパルスを出力します。ASG出力のパルス数を数えると、モーターの回転数をモニタ
することが可能です。モーターが正転するときは、主にASG出力、モーターの逆転のときは主にBSG出力の信号レベルが反転します。
BSG出力は、ASG出力に対して90°の位相差があります。ASG出力のピッチよりも精密に、BSG出力の出力レベルを検出することで、
モーターの回転方向を判定できます。

```
           CW動転         CCW動転
ASG出力 ON
       OFF    90°
BSG出力 ON
       OFF
```

memo ASG出力、BSG出力は、モーターの動きに対して最大0.1 msの遅れがあります。停止直後の確認は
としてご使用してください。
