# 6　設置

ドライバの設置場所と設置方法について説明しています。

## 6-1　設置場所

ドライバは、電気機器から離れた場所に、設置されています。設置しやすく、保守が容易な次のような場所に設置してください。

• 屋内に設置され正常な換気ができるところ（屋内使用）
• 使用周囲温度0～+50℃で使用されるところ
• 使用周囲湿度85%以下で結露しないところ
• 激しい振動がないところ（振動加速度以下）、および衝撃がないところ
• 直射日光が当らないところ
• 腐食性ガスがないところ
• 水（作業水槽）、油（潤滑油）、およびそれらの液滴がかからないところ
• 塵・埃が少ないところ
• 金属粉や導電性微細な塵埃の浮遊がかからないところ
• 電磁ノイズ（溶接機、高周波設備など）が少ないところ
• 放射性物質や有害物質、および高電圧がないところ
• 高度1,000 m以下

## 6-2　設置方法

ドライバはレール幅35 mmのDINレールに取り付けてください。

ドライバを2台以上並べて設置するときは、水平方向は接触できます。
間隔は45 mm以上空けてください。

ドライバを3台以上並べて設置するときは、内側のドライバの発熱が大きいです。
ドライバ周辺の空気の温度を強制空調してください。また、周囲温度が-40℃、
停止電流50%以下で使用してください。

```
寸法図：
         35 mm
    ┌─────────────┐
    │             │ 90 mm
    │             │
    │             │
    └─────────────┘
         50 mm以上
```

memo：ドライバのDINレールにて最大負荷、各機器付きでの設置は設置している
でください。
• ドライバは、上位システム中枠の間に適い機器の下部に設置している
でください。
• ドライバの防護構造IP50で構築されるときは、ファンで冷却したり、ド
ライバ間に空隙を設けるなど、効果を検討してください。
• ドライバは、必要に応じ周囲を周辺機器でしてください。

### 設置手順：

1. ドライバのDINレール受けフックをロックし、背面にあるフックをDINレールに引きます。

2. ドライバをDINレールに押し当てて、DINレールをフック上げて設置します。

3. エンドプレートでドライバの両端を固定します。
