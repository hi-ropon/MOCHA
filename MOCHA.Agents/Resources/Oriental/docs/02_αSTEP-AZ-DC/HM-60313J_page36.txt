# ガイダンス

## 4. ティーチング・リモート運転を開始する

[ティーチング・リモート運転を開始する]をクリックします。
警告ウィンドウが表示されるので、[はい]をクリックしてください。

**警告ダイアログ：**
- タイトル：ティーチング・リモート運転を開始します
- メッセージ：ティーチング・リモート運転を開始します。よろしいですか？
- ボタン：[はい] [いいえ]

## 5. MEXE02データとドライバデータの同期

MEXE02のデータとドライバのデータを同期させるウィンドウが表示されるので、同期方法を選択して[OK]をクリックします。

**同期方法選択ダイアログ：**
- 同期方法を選択してください。
- 同期方法：
  - ○ 全データの読み出し（製品 → PC）
  - ● 全データの書き込み（PC → 製品）
  - ○ 差分データの書き込み（PC → 製品）
- ボタン：[OK] [キャンセル]

ティーチング・リモート運転が有効になり、POWER/ALARM LEDの赤色と緑色が同時に点滅します。（赤色と緑色が重なって、橙色に見えることがあります。）

## 6. JOG運転ボタンでモーターの位置を調整します

### JOG運転ボタンの説明

| ボタン | 説明 |
|--------|------|
| ⏮ | 「(JOG)運転速度 (高)」パラメータに設定した運転速度で一方向へ連続運転します。 |
| ◀ | 「(JOG)運転速度」パラメータに設定した運転速度で一方向へ連続運転します。 |
| ⏸ | JOG運転ボタンの「最小移動量」に設定した移動量で一方向へ位置決め運転します。 |
| ⏹ | モーターが即停止します。 |
| ⏯ | JOG運転ボタンの「最小移動量」に設定した移動量で+方向へ位置決め運転します。 |
| ▶ | 「(JOG)運転速度」パラメータに設定した運転速度で+方向へ連続運転します。 |
| ⏭ | 「(JOG)運転速度 (高)」パラメータに設定した運転速度で+方向へ連続運転します。 |
