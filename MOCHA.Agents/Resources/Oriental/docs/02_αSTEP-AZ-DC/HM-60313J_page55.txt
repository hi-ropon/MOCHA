## 15-2 接続ケーブル（コネクタタイプ用）

### 接続ケーブル / 可動接続ケーブル

モーターとドライバを接続するときに使用します。

```mermaid
graph LR
    A[モーター] --> B[接続ケーブル] --> C[ドライバ]

    subgraph "ケーブル構成"
        direction TB
        D[モーター/エンコーダ/電磁ブレーキ用]
    end

    B --> D
```

モーターからケーブルを引き出す方向によって、接続ケーブルの品名が異なります。図をご確認ください。

```mermaid
graph TB
    subgraph "ケーブル引き出し方向"
        A[ケーブル側面引き出し終端]
        B[ケーブル正面引き出し側面]
        C[ケーブル正面引き出し反対方終端]
    end

    A --> D[出力側端]
    B --> E[側面]
    C --> F[反対方終端]
```

**(memo)** モーターを可動部分に取り付けるときは、可動ケーブルを使用してください。
