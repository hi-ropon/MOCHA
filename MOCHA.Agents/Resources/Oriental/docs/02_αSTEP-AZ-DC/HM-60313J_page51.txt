# 15 ケーブル

## 15-1 接続ケーブル（ケーブルタイプ用）

### 接続ケーブルセット / 可動接続ケーブルセット
**（AZM14、AZM15、AZM24、AZM26用）**

モーターとドライバを接続するときに使用します。

```mermaid
graph LR
    A[モーター] --> B[接続ケーブルセット] --> C[ドライバ]

    subgraph "接続ケーブルセット"
        direction TB
        D[コネクタ部分]
        E[ケーブル]
        F[コネクタ部分]
        D --> E --> F
    end
```

**(memo)** モーターを可動部分に取り付けるときは、可動ケーブルを使用してください。
