## ■ ドライバの種類と概要

AZシリーズのドライバには、次の3種類があります。タイプによって、入力内容、記憶内容、およびLEDの表示が異なります。

### ● 位置決め機能内蔵タイプ

```mermaid
flowchart LR
    A[POWER/ALARM LED<br/>STATUS LED] --> B[機能設定スイッチ<br/>・プロトコル<br/>・各種設定LED<br/>可変編号<br/>通信速度]
    B --> C[CN-No.1, 2, 13, 14ピ<br/>ン運転入力]
    C --> D[ピンNo.1: DIN10[START]<br/>ピンNo.2: DIN12[M1]<br/>ピンNo.13: DIN11[M0]<br/>ピンNo.14: DIN13[M2]]
```

特徴:
- ● パラメータークを設定したい
- ● オペレーションウィザードパネルで
  プログラムを組みたい
- ● RS-485通信で駆動したい
- ● I/O制御で駆動したい

### ● RS-485通信付きパルス列入力タイプ

```mermaid
flowchart LR
    A[POWER/ALARM LED<br/>STATUS LED] --> B[機能設定スイッチ<br/>・プロトコル<br/>・各種設定LED<br/>可変編号<br/>通信速度]
    B --> C[CN-No.1, 2, 13, 14ピ<br/>ンパルス入力]
    C --> D[ピンNo.1: CW+[PLS+]<br/>ピンNo.2: CCW+[DIR+]<br/>ピンNo.13: CW-[PLS-]<br/>ピンNo.14: CCW-[DIR-]]
```

特徴:
- ● パラメータークを設定したい
- ● オペレーションウィザードパネルで
  設定を変更したい
- ● RS-485通信で駆動したい
- ● I/O制御で駆動したい
- ● パルス入力で駆動したい
