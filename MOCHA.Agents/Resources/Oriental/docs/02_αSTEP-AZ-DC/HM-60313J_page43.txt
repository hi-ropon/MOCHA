# ガイダンス

## RS-485 通信付きパルス列入力タイプ、パルス列入力タイプの場合

パルスを入力して、位置決め運転を行ないます。

1. 主電源を投入します。
   READY出力と PLS-RDY出力が ONになります。

2. PLS-RDY出力が ONであることを確認し、パルスを入力します。
   モーターが位置決め運転を始めます。
   パルスが停止して運転が終了すると、READY出力が ONになります。

```mermaid
sequenceDiagram
    participant 主電源 as 主電源
    participant パルス入力 as パルス入力
    participant READY出力 as READY出力
    participant PLS-RDY出力 as PLS-RDY出力
    participant MOVE出力 as MOVE出力
    participant IN-POS出力 as IN-POS出力
    participant モーター動作 as モーター動作

    主電源->>主電源: ON①
    READY出力->>READY出力: ON
    PLS-RDY出力->>PLS-RDY出力: ON

    パルス入力->>パルス入力: ON②パルス波形
    READY出力->>READY出力: OFF
    PLS-RDY出力->>PLS-RDY出力: OFF②
    MOVE出力->>MOVE出力: ON
    IN-POS出力->>IN-POS出力: ON

    モーター動作->>モーター動作: 動作中

    Note over パルス入力, モーター動作: パルス停止後、運転終了でREADY出力がONになります
```

## 1パルス入力方式で使用する場合

1パルス入力方式をお使いになるときは、事前に次の設定を行なってください。

### RS-485 通信付きパルス列入力タイプの場合

1. 画面の左側にあるツリービューから、「パラメータ」-「通信・I/F機能」をクリックします。
   通信・I/F機能パラメータが表示されます。

   **設定メニュー構造:**
   ```
   データ
   ├─ 運転データ
   │  ├─ 運転のイベント
   │  └─ 運転データ検出判定条件
   ├─ パラメータ
   │  ├─ 基本設定
   │  │  ├─ モーター・電磁ブレーキ/ログ・座標/到達時設定
   │  │  ├─ ETO・Alarm対応設定
   │  │  ├─ I/O割付・機能
   │  │  ├─ Direct-IN 機能設定(DIN)
   │  │  ├─ Direct-OUT機能設定(DOUT)
   │  │  ├─ Remote+I/O機能設定(R+I/O)
   │  │  └─ コモンライン設定、RS485通信設定(通信機能搭載機種)
   │  └─ 通信・I/F機能
   ```

2. 「PULSE-I/F動作」パラメータを「1-Pulse」に設定します。

   | 運転データ | 通信・I/F機能 |
   |----------|-------------|
   | 1        | PULSE-I/F動作 | 1-Pulse |

3. 運転条件を変更したら、ツールバーの「データの書き込み」アイコンをクリックして、ドライバにダウンロードします。

   **ツールバーアイコン:**
   ⬅️📥➡️

4. ドライバの主電源を再投入します。
   変更したパラメータが反映されます。
