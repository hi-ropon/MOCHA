## ガイダンス

### JOG運転ボタンの説明

| ボタン | 説明 |
|--------|------|
| ⏮⏮ | 「(JOG)運転速度(高)」パラメーターに設定した運転速度で－方向へ連続運転します。|
| ⏮ | 「(JOG)運転速度」パラメーターに設定した運転速度で－方向へ連続運転します。|
| ⏸ | JOG運転ボタンの「最小移動量」に設定した移動量で－方向へ位置決め運転します。|
| ⏹ | モーターが即停止します。|
| ➕ | JOG運転ボタンの「最小移動量」に設定した移動量で＋方向へ位置決め運転します。|
| ⏭ | 「(JOG)運転速度」パラメーターに設定した運転速度で＋方向へ連続運転します。|
| ⏭⏭ | 「(JOG)運転速度(高)」パラメーターに設定した運転速度で＋方向へ連続運転します。|

JOG運転ボタンをクリックしてもモーターが動作しないときは、次の点を確認してください。
• 主電源やモーターが正しく接続されているか。
• アラームが発生していないか。

### ● JOG運転の運転条件を変更したいとき

1. 画面の左側にあるツリービューから、[パラメーター]→[モーター・機構(座標/JOG/原点復帰)]設定をクリックします。
   モーター・機構パラメーターが表示されます。

   **ツリー構造:**
   ```
   ├─ エラー
   │   ├─ 通信エラー
   │   ├─ 通信エラーの応答時間設定
   │   └─ 通信エラー分析結果設定
   ├─ パラメーター
   │   ├─ 基本設定
   │   ├─ モーター・機構(座標/JOG/原点復帰)設定 ←
   │   ├─ I/O信号・結線
   │   ├─ Direct-IN端子時速度(P/N)
   │   ├─ Direct-IN端子角度(P/N+OUT)
   │   ├─ Remote-I/O端子速度(P/P+I/O)
   │   ├─ EXT-IN/WR-IN2(EXT-OUT端子速度制御)
   │   └─ 通信・制御
   ```

2. [JOG/HOME/ZHOME運転 運転情報設定]パラメーターを[マニュアル設定]に変更します。

3. 次の5種類のパラメーターで、運転条件を変更します。

   | 番号 | パラメーター | モーター機構定義JOG常有情報設定 | 設定値 |
   |------|-------------|----------------------------|-------|
   | 24 | (JOG)移動量 [step] | 1 |
   | 25 | (JOG)運転速度 [Hz] | 1000 |
   | 26 | (JOG)加速度 [kHz/s] | 1000 000 |
   | 27 | (JOG)移動最高速度 [Hz] | 500 |
   | 28 | (JOG)運転速度(高) [Hz] | 5000 |

4. 運転条件を変更したら、ツールバーの[データの書き込み]アイコンをクリックして、ドライバにダウンロードします。

   **ダウンロード操作:**
   ```
   🔄 📥 🔧  [データの書き込み]アイコン
   ```
