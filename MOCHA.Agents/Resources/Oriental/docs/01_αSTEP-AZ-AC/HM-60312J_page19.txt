## 7-3 回生抵抗の接続

高下降駆動などの上下駆動時、大電力で回生電流が流れ、内心が損傷対象となるときは、当社の回生抵抗 RG8100を接続してください。

### 接続方法

- 回生抵抗の黒いリード線2本（AWG22:0.3 mm²）はコネクタのドライバで CN1用コネクタを使用して、TH1とTH2に接続してください。
- 回生抵抗の太いリード線2本（AWG18:0.75 mm²）は接続レバーでつまんで CN4の RG1とRG2に接続してください。

### 重要な注意事項

CN4コネクタのRG1とRG2には、防水処理を何等ののためのシール処理っているものです。回生抵抗を接続するときはシールを外してください。主電源のリード線を誤って接続すると、故障の原因となる場合があります。

```
    シール
   ┌─────┐
   │ ●● │
   │ ●● │
   └─────┘
```

### 注記（memo）
- 回生抵抗を接続するときは、必ずジャンパー線を CN1用コネクタから外してください。
- 回生抵抗のサーミスタット端子を読みとるときは、サーミスタット外までについて、回生抵抗結果動作のアラームが発生します。回生抵抗結果動作のアラームが発生したときは、主電源と制御電源を切り、異常の内容を確認してください。
