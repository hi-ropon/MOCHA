### ● 高速原点復帰運転の運転条件を変更したいとき

1. 画面の左側にあるツリービューから、[パラメーター]→[モーター・機構(座標/JOG/原点復帰)]設定をクリックします。
   モーター・機構パラメーターが表示されます。

   **ツリー構造:**
   ```
   ├─ データ
   │   ├─ 通信エラー
   │   ├─ 通信エラーイベント
   │   └─ 通信エラー分析結果設定
   ├─ パラメーター
   │   ├─ 基本設定
   │   ├─ モーター・機構(座標/JOG/原点復帰)設定 ←
   │   ├─ I/O信号・結線
   │   ├─ I/O信号・結線
   │   ├─ Direct-IN 制御端子設定(P/N)
   │   ├─ Direct-OUT制御端子設定(P/OUT)
   │   ├─ Remote-I/O制御端子設定(P+I/O)
   │   ├─ EXT-IN/WR-IN(I/SOUT制御端子制御設定)
   │   └─ 通信・設定制御
   ```

2. [JOG/HOME/ZHOME運転 運転情報設定]パラメーターを[マニュアル設定]に変更します。

3. 次の3種類のパラメーターで、運転条件を変更します。

   **ZHOME運転パラメーター設定表:**
   | 番号 | パラメーター | モーター機構定義JOG常在情報設定 | 設定値 |
   |------|-------------|----------------------------|-------|
   | 29 | (ZHOME)運転速度 [Hz] | 5000 |
   | 30 | (ZHOME)加速度 [kHz/s] | 1000.000 |
   | 31 | (ZHOME)移動速度 [Hz] | 500 |

4. 運転条件を変更したら、ツールバーの[データの書き込み]アイコンをクリックして、ドライバにダウンロードします。

   **データ書き込み操作:**
   ```
   🔄 📥 🔧  [データの書き込み]アイコン
   ```
