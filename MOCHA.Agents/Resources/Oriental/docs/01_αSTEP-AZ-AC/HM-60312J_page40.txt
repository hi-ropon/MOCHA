<overview>
MEXE02での原点確定手順の続き。ティーチング・リモート運転の開始、同期方法選択、JOG運転ボタンでのモータ位置調整について説明。JOG運転ボタンの詳細な説明表が含まれている。
</overview>

4. [ティーチング・リモート運転を開始する] をクリックします。
   警告ウィンドウが表示されるので、[はい] をクリックしてください。

   ```
   警告: ティーチング・リモートを運転を開始します。
   よろしいですか？
   [はい(Y)] [いいえ(N)]
   ```

5. MEXE02のデータとドライバのデータを同期させるウィンドウが表示されるので、同期方法を選択して [OK] をクリックします。

   ```
   同期方法選択

   同期方法を選択してください。

   同期方法：
   ○ モータの位置を読み出し（製品 → PC）
   ○ モータの値を書き込む（PC → 製品）
   ○ 今回のデータで置き換え（PC → 製品）

   [OK] [キャンセル]
   ```

   ティーチング・リモート運転が有効になり、PWR/ALM LEDの赤色と緑色が同時に点滅します。（赤色と緑色が重なって、橙色に見えることがあります。）

6. JOG運転ボタンで、モータの位置を調整します。

   ```mermaid
   graph TB
       A[ティーチング・リモートコントロール]
       B[連続データNo.: 0]
       C[JOG運転ボタン群] --> D[位置確定ボタン]
       E[ドライバ設定]
       F[FREE出力] --> G[FREE-ON]
   ```

### JOG運転ボタンの説明

| ボタン | 説明 |
|--------|------|
| ⏪ | 「JOG」運転速度（高）パラメータに設定した運転速度で－方向へ連続運転します。 |
| ◀ | 「JOG」運転速度」パラメータに設定した運転速度で－方向へ連続運転します。 |
| ■ | JOG運転ボタンの「最小移動量」に設定した移動量で－方向へ位置決め運転します。 |
| ⏹ | モータが即停止します。 |
| ＋ | JOG運転ボタンの「最小移動量」に設定した移動量で＋方向へ位置決め運転します。 |
| ▶ | 「JOG」運転速度」パラメータに設定した運転速度で＋方向へ連続運転します。 |
| ⏩ | 「JOG」運転速度（高）パラメータに設定した運転速度で＋方向へ連続運転します。 |
