# 7-10 ノイズ対策

ノイズには、外部からドライバに侵入してドライバを誤動作させるノイズ、およびドライバから発散されて周りの機器を誤動作させるノイズの2種類があります。

外部から侵入するノイズに対しては、ドライバの誤動作を防ぐ対策を講じてください。特に電気ライン上ノイズの影響を受けやすいため、十分に対策が必要です。

ドライバから発散されるノイズに対しては、ノイズを抑制する対策を講じてください。

## ノイズ対策の方法

ノイズ対策の方法は、主に次の3種類があります。

### ノイズの抑制
- **リレー や電磁スイッチを使用するときは、ノイズフィルタやCR回路等サージを吸収してください。**
- **同一ケースや配電盤内のドライバの電源ケーブルを使用してください。モーターから発散されるノイズを抑制する効果があります。運搬ケーブルの延長：5～6 m程度で配線してください。**
- **アルミケースの電機機がドライバを置いてください。ドライバから発散されるノイズを遮断する効果があります。**

### シールド対策
- **ノイズフィルタをドライバの電源ケーブルに接続してください。**
- **ノイズフィルタは、ノーマルモードとコモンモードのケーブルの電源ケーブルは200 mm以上離し、または、平行に配線しないでください。制御ケーブルと電源ケーブルが交差するときは、直角に交差させてください。**
- **制御ケーブルは、ツイストペアシールドケーブルを使用してください。**
- **ケーブルは短縮で短縮し、数字での浮いた配線を避けより、束ねないでください。**
- **シールドケーブルのシールド線は、システム全体でシールドすること。ノイズを遮断する効果があります。また、接続した端末に電流が流れないよう、安全した制振に接続してください。当社でもケーブル数を多く取り付けた入出力信号用ケーブルを用意してください。**
- **ケーブルを敷設するときは、シールドの金属の接続端で全体に満を同一シールドケーブル、ケーブルクランプを使用し、さらなども接続的近くに固定してもらいたい。**

### ケーブルクランプ

### ノイズ伝播による影響の抑制
- **ノイズが伝播しているケーブルをフェライトコアに通すくださ。伝導したノイズがドライバに侵入したり、ドライバから外部機器を開設することを抑制します。フェライトコアの効果が感められる場合、複数回巻きつけることでフェライトコアの取付位置を調整してください。フェライトコアによるノイズ遮断の効果を高める場合は、ケーブルを数回巻き取って取付てください。**
- **パルス設備の場合などで、ノイズの影響を安定にしているラインドライバ方式に変更してください。また、システムのパルス信号にオープンコレクタ方式のときは、動作の遅いノイズ対処上コモンモード電源を使用してください。高速の範囲で調整できます。**

## ノイズ対策部品

### ノイズフィルタ
- **電源とノイズフィルタ（または福島電線）を電源ラインに接続してください。電源ライン免許上で伝導するノイズを防ぎます。**
**ノイズフィルタは、できるだけドライバの近くに取り付けてください。**

**推奨ノイズフィルタ仕様：**

| 項目 | 仕様 |
|------|------|
| 電流値 | ドライバの供給電流以上 |
| 額定電圧 | DC24V |

**推奨製品例：**
| メーカー | 型名 |
|----------|------|
| 双信電機株式会社 | HF2010A-UPF |
| Schaffner EMC | FN2070-10-06 | HF3010C-SZA |
|  |  | FN2025HP-10-71 |

- **ノイズフィルタの入出力ケーブルには AWG18（0.75 mm²）以上の電線を使用し、ケーブルが分かれないよう ケーブルクランプなどで確実に固定してください。**
- **ノイズフィルタを金属ケースへプレート に固定し取り付けし、定時に配線しないでください。ケーブルの磁気の影響が少なくなり、遮行に配線することで、最終的のノイズが相互相殺され電線電場ケーブルに伝導しません。ノイズ抑制効果が向上します。**
- **ノイズフィルタを接続する場合：ノイズフィルタのねじだけでなく、徹底した軽で固定してください。**
- **設備内でノイズフィルタを接続・金属部品は、ノイズフィルタの入力ケーブルと見なく配線しないでください。ノイズ抑制効果が低減します。**
