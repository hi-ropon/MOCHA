## 7-2 CN1の接続

### CN1用コネクタの結線方法

- 適用リード線：AWG24～16（0.2～1.25 mm²）
- 皮むき長さ：10 mm

#### 結線手順

1. リード線の皮をむきます。
2. マイナスドライバで橙色のボタンを押しながら、リード線を挿し入れます。
3. リード線を挿入したら、ボタンを離してリード線を固定します。

```
      橙色のボタン
         ↓
    ┌─────────┐
    │●●●●●●●│ ← リード線
    │●●●●●●●│
    └─────────┘
```

### ピンアサイン一覧

0V端子との短絡や配線の短絡などがおきますが、固くそれぞれの回線を確認してください。
因主抵抗や動力通断機能を使用するときは、ジャンパー線を取り外して接続してください。

#### コネクタピン配置

```
+24V ●●● 0V※1
TH1 ●●● MB2
TH2 ●●● TH2
HWTO1+ ●●● HWTO1+
HWTO2+ ●●● HWTO2-
EDM+ ●●● EDM-
```

### 端子詳細

| 表示 | 内容 |
| --- | --- |
| +24V, 0V※1 | 制御電源を接続します。<br>• 電磁ブレーキを使用しない場合：DC24 V±5 %、0.25 A<br>• 電磁ブレーキを使用する場合：DC24 V±5 %、0.5 A（AZM644d:0.33 A）<br>• 電磁ブレーキを使用し、ケーブルタイプのモーターとドライバ間の配線距離が20 m以下の場合：DC24 V±4 %、0.5 A（AZM644d:0.33 A） |
| MB1, MB2 | 電磁ブレーキ付きケーブルのリード線を接続します。<br>MB1：電磁ブレーキ－（黒）<br>MB2：電磁ブレーキ＋（白） |
| TH1, TH2 | 当社の回生抵抗 RG8100のサーミスタット端子を接続します。回生抵抗を使用しないときは、因生抵抗とジャンパー線で短絡してください。<br>回生抵抗を接続しない場合、動力電断機能を使用しないときは、因のようにジャンパー線で短絡させてください。 |
| HWTO1+, HWTO1-,<br>HWTO2+, HWTO2- | 外部機器を接続します。動力電断機能を使用しないときは、因のようにジャンパー線で短絡させてください。 |
| EDM+, EDM- | 外部機器を接続します。動力電断機能を使用しないときは、因のようにジャンパー線で短絡させてください。 |
| +V, 0V※2 | 内部機器用です。何も接続しないでください。動力電断機能を使用しないときは、因のようにジャンパー線で短絡させてください。 |

### 注記（memo）
動力電断機能の詳細は、AZシリーズ機器編でご確認ください。
