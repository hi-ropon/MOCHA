# 14 もっと便利に使いたい

MEXE02を活用すると、運転データを設定したり、CN5コネクタに割り付ける入出力信号を変更することができます。また、
運転の状況をモニタしたり、テスト運転を行うこともできます。
詳しい運転内容については、AZシリーズ 機能編をご覧ください。

```mermaid
flowchart TD
    A[機種に合わせた分解能を設定したい]
    B[I/Oの割り付けを変更したい]
    C[シーケンス機能を活用したい]
    D[保守やメンテナンスに便利な機能を活用したい]
    E[波形モニタで動作を確認したい]
    F[アラーム条件を変更したい]
    G[押し当て運転をしたい]
    H[FAネットワークで動作させたい※]

    style A fill:#d3d3d3
    style B fill:#d3d3d3
    style C fill:#d3d3d3
    style D fill:#d3d3d3
    style E fill:#d3d3d3
    style F fill:#d3d3d3
    style G fill:#d3d3d3
    style H fill:#d3d3d3
```

※ パルス列入力タイプを除く。
