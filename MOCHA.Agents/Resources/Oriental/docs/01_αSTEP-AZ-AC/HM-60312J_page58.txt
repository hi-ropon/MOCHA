## 15-2 接続ケーブル（コネクタタイプ用）

### 接続ケーブル／可動接続ケーブル

モーターとドライバを接続するときに使用します。

```mermaid
graph LR
    A[モーター] --- B[接続ケーブル<br/>モーター/エンコーダ/電磁ブレーキ用]
    B --- C[ドライバ]

    style B fill:#f9f9f9
```

モーターからケーブルを引き出す方向によって、接続ケーブルの品名が異なります。図でご確認ください。

```mermaid
graph TD
    subgraph "ケーブル引出し方向の種類"
        A[ケーブル引出方向<br/>出力軸側]
        B[ケーブル引出方向<br/>垂直]
        C[ケーブル引出方向<br/>反出力軸側]
    end

    style A fill:#f9f9f9
    style B fill:#f9f9f9
    style C fill:#f9f9f9
```

**メモ** モーターを可動部分に取り付けるときは、可動ケーブルを使用してください。
