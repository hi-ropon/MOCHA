## 10-5 高速原点復帰運転

高速原点復帰運転(ZHOME)を使うと、原点へ簡単に復帰できます。

### STEP 1　現在位置を確認します

ティーチング・リモート運転ウィンドウの[指令位置]を確認します。

**位置確認画面:**
```
┌─────────────────────────────────────────┐
│ 転送元│AZ パルス列入力/出力システムの機能内部/RS485通信データパルス列入力・運転・ティーチング...│
│                                       │
│ ☑ ティーチング・リモート運転を有効にする      │
│                                       │
│ ドライバ状況                            │
│ 指令位置(POS)      8500 [step]         │
│ 検出位置           8500 [step]         │
│ 検出速度           0    [Hz]           │
└─────────────────────────────────────────┘

INPUT状態表示:
☐ C-ON
☐ FREE
☐ STOP

OUTPUT状態表示:
☐ C-RNT
☐ MBC
☐ MOVE
☐ MPS
```

### STEP 2　高速原点復帰運転を行ないます

1. [ZHOME運転]をクリックします。
   警告ウィンドウが表示されるので、[はい]をクリックしてください。
   モーターが高速原点復帰運転を行ないます。

   **ZHOME運転確認画面:**
   ```
   運転設定:
   - 運転データNo.: 0
   - 位置決め運転開始ボタン
   - 原点復帰運転ボタン
   - ZHOME運転ボタン ←

   確認ダイアログ:
   ┌──────────────────┐
   │ 警告                │
   │ ZHOME運転を開始します。 │
   │ よろしいですか？        │
   │                    │
   │ [はい] [いいえ]      │
   └──────────────────┘
   ```
