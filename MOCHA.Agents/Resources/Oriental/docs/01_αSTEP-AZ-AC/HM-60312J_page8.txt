#### 4 法令・規格

#### 4-1 UL規格、CSA規格

この製品は、UL規格、CSA規格の認証を取得しています。
ドライバには、UL規格、CSA規格で規定されるモーター過熱保護機能とモーター過熱保護は組み込まれています。

#### 4-2 CEマーキング/UKCAマーキング

この製品は、次の指令/規則にもとづいてマーキングを表示しています。

#### ■ EU低電圧指令/UK電気機器（安全）規則

##### ● 設置条件

### 表

| 項目 | 内容 |
| --- | --- |
| 過電圧カテゴリー | Ⅱ |
| | 2 |
| 保護等級 | IP20（パルス列入力タイプ）<br>IP10（位置決め機能内蔵タイプ、RS-485通信付きパルス列入力タイプ） |
| 感電保護 | クラスI |

- 11定格周波では使用できません。
- モーターケーブルや電源ケーブルとの配力線ケーブルと、信号系のケーブルは、二重絶縁で分離してください。
- 設置条件によっては、ドライバの設置箱が50 ℃を超えることがあります。次のことを守ってください。
  - 必ず試運転を行ない、ドライバの温度を確認してください。
  - 通電部は材料として、規格値以上のものを使用してください。
  - ドライバに触れないでください。
- 本銘用電気部は、EN(IEC)規格に適合していない条件を使用してください。
- ドライバには、EN規格で規定されるモーター過熱保護機能とモーター過熱保護は組み込まれています。
- ドライバには、突験保護機能が内蔵されていません。配線するときは、スペースの確保確認を考慮した電源への配線時に従ってください。また、次のことを考慮してください。
  - 過電圧測定：定格電圧15 A、定格遮断電流10 mA
  - 過電圧カテゴリーⅡの環境に接続する場合は絶縁トランスを使用し、絶縁トランスの二次側（単相の場合はN、三相の場合は中性点）を接地する。
  - 設置ループインピーダンス（Zs）の値以下
