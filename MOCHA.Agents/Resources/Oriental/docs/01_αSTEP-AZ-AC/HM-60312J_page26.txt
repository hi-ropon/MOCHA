<overview>
本ページは電流ソース出力回路使用時のRS-485通信付きパルス列入力タイプおよびパルス列入力タイプの接続図を示している。3つの異なるパルス入力方式（ラインドライバ、オープンコレクタDC5V、オープンコレクタDC24V）の詳細な回路図と仕様が記載されている。
</overview>

<contents>
> **memo**
> - 入力電圧はDC24 Vで動作してください。
> - 出力電圧はDC12～24 V、10 mA以下で動作してください。電流値が10 mAを超えると、外部機器が破損して、10 mAに収まるように動作してください。
> - 出力機器の動作電圧は最大3Vです。

## RS-485通信付きパルス列入力タイプ、パルス列入力タイプの場合
ピンNo.1、2、13、14はパルス入力専用です。他の機能は割り付けできません。

### パルス入力ラインドライバのとき

**制御システム（電流ソース出力） ⟷ ドライバ接続仕様：**

```mermaid
graph LR
    A[制御システム<br/>電流ソース出力] --> B[ドライバ<br/>差動入力]

    subgraph "信号線"
        C[CW+ PLS+]
        D[CW- PLS-]
        E[CCW+ DIR+]
        F[CCW- DIR-]
    end

    A --> C
    A --> D
    A --> E
    A --> F
    C --> B
    D --> B
    E --> B
    F --> B
```

| 信号 | 抵抗値 | 電圧値 | 周波数 |
|------|-------|-------|---------|
| CW+(PLS+) | 470Ω | 270Ω | 5.6kΩ |
| CW-(PLS-) | 470Ω | 270Ω | - |
| CCW+(DIR+) | 470Ω | 270Ω | 5.6kΩ |
| CCW-(DIR-) | 470Ω | 270Ω | - |

**接続方式：** ツイストペア線を使用した差動入力

### パルス入力オープンコレクタのとき（パルス列信号の電圧がDC5 Vの場合）

**制御システム（電流ソース出力） → ドライバ接続仕様：**

**DC5V電源系統：**
| 信号 | 抵抗値 | 電圧値 | 周波数 |
|------|-------|-------|---------|
| CW+(PLS+) | 470Ω | 270Ω | 5.6kΩ |
| CW-(PLS-) | 470Ω | 270Ω | - |
| CCW+(DIR+) | 470Ω | 270Ω | 5.6kΩ |
| CCW-(DIR-) | 470Ω | 270Ω | - |

**接続方式：** DC5V電源と電流ソース出力

### パルス入力オープンコレクタのとき（パルス列信号の電圧がDC24 Vの場合）

**制御システム（電流ソース出力） → ドライバ接続仕様：**

**DC24V電源系統：**
| 信号 | 抵抗値（プルアップ） | 電圧値 | 周波数 |
|------|---------------------|-------|---------|
| CW+(PLS+) | R1: 1.2kΩ-2.2kΩ (0.5W以上) | 470Ω 270Ω | 5.6kΩ |
| CW-(PLS-) | - | 470Ω 270Ω | - |
| CCW+(DIR+) | R1: 1.2kΩ-2.2kΩ (0.5W以上) | 470Ω 270Ω | 5.6kΩ |
| CCW-(DIR-) | - | 470Ω 270Ω | - |

**接続方式：** DC24V電源とプルアップ抵抗付き電流ソース出力

```mermaid
graph TB
    A[パルス入力方式の選択]

    A --> B[ラインドライバ方式]
    A --> C[オープンコレクタDC5V]
    A --> D[オープンコレクタDC24V]

    B --> E[差動信号<br/>高速・高精度]
    C --> F[DC5Vシングルエンド<br/>簡単接続]
    D --> G[DC24Vシングルエンド<br/>プルアップ抵抗必要]

    subgraph "電流ソース出力の利点"
        H[安定した信号伝送]
        I[ノイズ耐性向上]
        J[長距離伝送対応]
    end
```

> **memo**
> CW(PLS入力)、CCW(DIR入力)は、DC5～24Vでお動きください。DC24 Vでお動きになるときは、外部抵抗R1（1.2kΩ～2.2 kΩ、0.5 W以上）を接続してください。DC5 Vでお動きになるときは、電圧を適度抑制してください。

### 電流ソース出力回路との接続の特徴

- 制御システム側が電流ソース出力回路を使用
- より安定したパルス信号伝送が可能
- ノイズ環境での信頼性向上
- 長距離配線に適している
