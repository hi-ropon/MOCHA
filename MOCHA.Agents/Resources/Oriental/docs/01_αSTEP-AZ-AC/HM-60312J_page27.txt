# 7-8 RS-485通信ケーブルの接続

RS-485通信で制御するときに接続します。

RS-485通信ケーブルをCN6コネクタまたはCN7コネクタに接続してください。空いた方のコネクタで、別のドライバと接続できます。

当社でもドライバ間接続用のケーブルをご用意しています。品名は59ページをご覧ください。

また、市販のLANケーブル（ストレートケーブル）でもドライバ間同士を接続できます。

[図: 複数のドライバをRS-485通信ケーブルで数珠つなぎ接続している様子]

## ピンアサイン一覧

| ピンNo. | 信号名 | 内容 |
|---------|---------|----------|
| 1 | N.C. | 未使用 |
| 2 | GND | GND |
| 3 | TR+ | RS-485通信用信号（+） |
| 4 | N.C. | 未使用 |
| 5 | N.C. | 未使用 |
| 6 | TR- | RS-485通信用信号（-） |
| 7 | N.C. | 未使用 |
| 8 | N.C. | 未使用 |

[コネクタ図: 8ピンRJ45コネクタのピン配置、ピン1から8まで番号付け]

## 内部入力回路

```mermaid
graph TD
    A[CN6/CN7コネクタ] --> B[内部入力回路]

    subgraph "回路構成"
        C[2 GND]
        D[3 TR+]
        E[6 TR-]
        F[TERM. No.1: 1kΩ]
        G[TERM. No.2: 1kΩ]
        H[120Ω終端抵抗]
    end

    C --> I[GND]
    D --> J[差動入力回路]
    E --> J
    J --> K[5V信号レベル]

    L[0V] --> M[基準電位]
```

**回路仕様：**
- 差動入力：TR+（ピン3）、TR-（ピン6）
- 内部終端抵抗：120Ω
- 信号レベル：5V
- 終端設定：TERM. No.1、TERM. No.2（各1kΩ）
- 基準電位：0V（GND）

# 7-9 USBケーブルの接続

次の仕様のUSBケーブルで、MEXE02をインストールしたパソコンをUSB通信コネクタに接続してください。

## ケーブル仕様

| 項目 | 仕様 |
|------|------|
| 仕様 | USB2.0（フルスピード） |
| ケーブル | 長さ：3 m以下<br/>形状：A to mini B |

```mermaid
graph LR
    A[パソコン<br/>MEXE02インストール済み] --> B[USBケーブル<br/>A to mini B]
    B --> C[ドライバ<br/>USB通信コネクタ]

    subgraph "接続要件"
        D[USB2.0フルスピード]
        E[ケーブル長3m以下]
        F[フェライトコア推奨]
    end
```

> **memo**
> - ドライバとパソコンは、USBケーブルで直接接続してください。
> - ノイズの影響が強いときは、フェライトコア付きUSBケーブルを使用するか、フェライトコアをUSBケーブルに装着してください。

### USBケーブル接続の注意事項

1. **直接接続**：ハブを介さずドライバとパソコンを直接接続
2. **ケーブル長**：3m以下の長さを守る
3. **ノイズ対策**：必要に応じてフェライトコア付きケーブルを使用
4. **ソフトウェア**：MEXE02が事前にインストールされていること
