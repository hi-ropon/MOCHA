# 8 入出力信号の説明

## 8-1 入力信号

ドライバの入力信号は、すべてフォトカプラ入力です。機器の外部配線時の電圧レベルではなく、内部フォトカプラのON/OFF制御の状態を表わしています。

### ■ CW[PLS]入力、CCW[DIR]入力

パルス入力で運転することに使用する信号です。
2パルス入力方式のときは、機器の配線モードでCW入力パルス数の2の倍 PLS入力とDIR入力になります。使用するときは、システムのパルス出力方法に合わせて、ドライバのパルス入力方式を設定してください。パルスを入力するときは、PLS-RDY出力がONになることを確認してください。

```
PLS-RDY出力 ON _____
              OFF      0.3μs以上

パルス入力 ON
          OFF
```

**memo**: モータの停止時間、およびパルス入力フォトカプラがOFFになってください。

#### • 最大入力パルス周波数

- 上位システムがオープンコレクタ出力の場合：1 MHz（デューティ 50 %）
- 上位システムがオープンプロック出力の場合：250 kHz（デューティ 50 %）

#### • 2パルス入力方式（出力側配線設定）

CW入力を OFFからONにすると、モータがCW方向へ1ステップ動作します。
CCW入力を OFFからONにすると、モータがCCW方向へ1ステップ動作します。

```
CW入力  ON
        OFF      5 μs以上
CCW入力  ON
         OFF      5 μs以上

            CW
        モータの動き
            CCW
```

**注意**: CWとCCW入力同時間に入力しないでください。モータの誤動作の原因になります。

#### • 1パルス入力方式

DIR入力がONのとき、PLS入力をOFFからONにすると、モータがCW方向へ1ステップ動作します。
DIR入力がOFFのとき、PLS入力をOFFからONにすると、モータがCCW方向へ1ステップ動作します。

```
              5 μs以上     5 μs以上
PLS入力 ON
        OFF

DIR入力 ON
        OFF

            CW
        モータの動き
            CCW
```

### ■ START入力

位置決めの運転を開始する信号です。パルス入力で運転するときには使用しません。
連結ケーブル No.を選択したとき、START入力がONになると、位置決めの運転が始まります。
