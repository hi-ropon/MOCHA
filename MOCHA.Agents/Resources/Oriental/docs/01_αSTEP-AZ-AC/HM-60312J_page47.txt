## ガイダンス

### ● RS-485通信付きパルス列入力タイプ、パルス列入力タイプの場合

パルスを入力して、位置決め運転を行っています。

**運転シーケンス:**

1. 主電源と制御電源を投入します。
   READY出力とPLS-RDY出力がONになります。

2. PLS-RDY出力がONであることを確認し、パルスを入力します。
   モーターが位置決め運転を始めます。
   パルスが停止して運転が終了すると、READY出力がONになります。

**タイミングチャート:**

```mermaid
gantt
    title パルス列入力タイプ位置決め運転タイミングチャート
    dateFormat X
    axisFormat %s

    section 制御電源
    ON  :done, power, 0, 10

    section 主電源
    ON  :done, main, 0, 10

    section パルス入力
    パルス列 :crit, pulse, 2, 6

    section READY出力
    OFF->ON->OFF->ON :ready, 1, 10

    section PLS-RDY出力
    OFF->ON :plsrdy, 1, 10

    section MOVE出力
    OFF->ON->OFF :move, 2, 6

    section IN-POS出力
    OFF->ON :inpos, 6, 10
```

**信号タイミング詳細:**

| 信号名 | 状態変化 | 説明 |
|--------|----------|------|
| 制御電源 | OFF→ON | ① 電源投入 |
| 主電源 | OFF→ON | ① 電源投入 |
| パルス入力 | パルス列入力 | ② 位置指令パルス |
| READY出力 | OFF→ON→OFF→ON | 運転準備完了/運転中/運転完了 |
| PLS-RDY出力 | OFF→ON | ② パルス受付準備完了 |
| MOVE出力 | OFF→ON→OFF | 運転中表示 |
| IN-POS出力 | OFF→ON | 位置決め完了 |
| モーター動作 | 停止→動作→停止 | パルス入力に同期した動作 |

### ● 1パルス入力方式で使用する場合

1パルス入力方式でお使いになるときは、事前に次の設定を行なってください。

#### RS-485通信付きパルス列入力タイプの場合

1. 画面の左側にあるツリービューから、[パラメーター]→[通信・I/F機能]をクリックします。
   通信・I/F機能パラメーターが表示されます。

   **ツリー構造:**
   ```
   ├─ データ
   │   ├─ 通信エラー
   │   ├─ 通信エラーイベント
   │   └─ 通信エラー分析結果設定
   ├─ パラメーター
   │   ├─ 基本設定
   │   ├─ モーター・機構(座標/JOG/原点復帰)設定
   │   ├─ ETO・Alarm その他設定
   │   ├─ I/O信号・結線
   │   ├─ Direct-IN 制御端子設定(P/N)
   │   ├─ Direct-OUT制御端子設定(P/OUT)
   │   ├─ Remote-I/O制御端子設定(P+I/O)
   │   ├─ EXT-IN/WR-IN2(SOUT制御端子制御設定)
   │   └─ 通信・I/F機能 ←
   ```

2. [PULSE-I/F動作]パラメーターを[1-Pulse]に設定します。

   **PULSE-I/F設定表:**
   | 番号 | パラメーター | 通信・I/F機能 | 設定値 |
   |------|-------------|---------------|--------|
   | 1 | PULSE-I/F動作 | 1-Pulse |

3. 運転条件を変更したら、ツールバーの[データの書き込み]アイコンをクリックして、ドライバにダウンロードします。

   **データ書き込み操作:**
   ```
   🔄 📥 🔧  [データの書き込み]アイコン
   ```
