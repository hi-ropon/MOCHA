### テキスト
#### ■ ドライバの種類と概要

AZシリーズのドライバには、次の3種類があります。タイプによって、入出力信号、設定内容、および LED が異なります。

### 位置決め機能内蔵タイプ

**特徴：**
- FAネットワークで動かしたい
- 上位システムやタッチパネルでモーター機器を動かしたい
- RS-485通信で動かしたい
- I/O制御で動かしたい

**接続概要：**
```mermaid
flowchart LR
    A[ドライバ<br/>位置決め機能内蔵タイプ] --> B[PWR/ALM LED<br/>C-DAT/C-ERR LED<br/>信号設定]
    A --> C[通信ポート<br/>プロトコル<br/>信号設定（拡張）]
    A --> D[ピンNo.1, 2, 13, 14は<br/>削除入力<br/>ピンNo.1: DIN0[START]<br/>ピンNo.2: DIN2[M1]<br/>ピンNo.13: DIN1[M0]<br/>ピンNo.14: DIN3[M2]]
```

### RS-485通信付きパルス列入力タイプ

**特徴：**
- FAネットワークで動かしたい
- 上位システムやタッチパネルでモーター機器を動かしたい
- RS-485通信で動かしたい
- I/O制御で動かしたい
- パルス入力で動かしたい

**接続概要：**
```mermaid
flowchart LR
    A[ドライバ<br/>RS-485通信付き<br/>パルス列入力タイプ] --> B[PWR/ALM LED<br/>C-DAT/C-ERR LED<br/>信号設定]
    A --> C[通信ポート<br/>プロトコル<br/>信号設定（拡張）]
    A --> D[ピンNo.1, 2, 13, 14は<br/>パルス入力<br/>ピンNo.1: CW+[PLS+]<br/>ピンNo.2: CCW+[DIR+]<br/>ピンNo.13: CW-[PLS-]<br/>ピンNo.14: CCW-[DIR-]]
```

### パルス列入力タイプ

**特徴：**
- パルス入力で動かしたい

**接続概要：**
```mermaid
flowchart LR
    A[ドライバ<br/>パルス列入力タイプ] --> B[PWR/ALM LED<br/>READY LED<br/>最本電流率]
    A --> C[指令フィルタ<br/>パルス入力方式<br/>分解能]
    A --> D[ピンNo.1, 2, 13, 14は<br/>パルス入力<br/>ピンNo.1: CW+[PLS+]<br/>ピンNo.2: CCW+[DIR+]<br/>ピンNo.13: CW-[PLS-]<br/>ピンNo.14: CCW-[DIR-]]
```
