## 10-2 原点の確定

工場出荷時は、原点が設定されていません。運転を行う前に、必ず原点を確定してください。

原点の確定は、機器ごとに1回だけ行ってください。いったん原点を設定すれば、その後は主電源と制御電源を遮断して停電状態であっても各情報は保持されています。

原点を変更するときは、次の2通りの方法があります。どちらの方法でも原点を設定してください。
- HOME PRESETスイッチで設定する。
- MEXE02で設定する。

**memo**: NVメモリに位置データが記録されます。NVメモリの書き込み可能回数は、約10万回です。
- 機器アクチュエータ、出荷時に原点が設定されています。変更しないときも、原点の確定を行ってください。

### ■ HOME PRESETスイッチで原点を確定する

1. 手動で、出力軸を原点にしたい位置まで動かします。

2. 主電源と制御電源を投入します。

3. 主電源と制御電源が投入されていることを確認し、HOME PRESETスイッチを1秒間以上押します（原点保持操作）。
   PWR/ALM LEDの点滅が終わり、同時に消灯します。
   （原点と確定後て、再点灯することがあります。）

   ```mermaid
   graph LR
       A[PWR/ALM LED] --> B[点滅終了・消灯]
       C[HOME PRESETスイッチ] --> D[1秒間以上押す]
   ```

4. PWR/ALM LEDが点滅状態になったら3秒以内に手を離し、手を離してから3秒以内にもう一度HOME PRESETスイッチを押します。
   PWR/ALM LEDの点滅が終わり終了時になり、停止した位置が原点になります。

   ```mermaid
   graph LR
       E[PWR/ALM LED] --> F[点滅状態]
       G[HOME PRESETスイッチ] --> H[3秒以内に押す]
   ```

5. 原点が確定されました。

**memo**: 手動の作業は、必ずPWR/ALM LEDが点滅状態になってから手を離し、3秒以内に行ってください。3秒を過ぎると、PWR/ALM LEDが点灯の点灯に戻ってしまいます。その場合は、もう一度手順1からやり直してください。

### ■ MEXE02で原点を確定する

1. 主電源と制御電源を投入します。

2. パソコンを起動し、MEXE02を起動します。

3. ツールバーの[ティーチング・リモート制御]アイコン、またはショートカットボタンの[ティーチング・リモート制御]をクリックします。
   ティーチング・リモート制御のウィンドウが表示されます。

   ```
   ティーチング・リモート制御
   ```
