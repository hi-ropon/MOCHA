## ガイダンス

7. 手動でモーターのON進路を操作することもでき、[FREE-ON]をクリックし、警告ウィンドウのはいをクリックします。
   出力解除フリーにして、手で出力軸を回転させることができます。
   警告解除は[FREE-OFF]をクリックして、モーターを励磁させてください。

   **画面操作例:**
   - チューニング設定ウィンドウで[FREE-ON]ボタンの表示
   - 警告ダイアログ: 「モーターの軸が回転可能になりますが、よろしいですか？」
   - OK/キャンセルボタンによる選択

8. モーターの回転位置を変えたいときは、位置の調整リセットをクリックし、警告ウィンドウのはいをクリックします。
   設定位置が確保され、ドライバに位置量設定されます。

   **画面操作例:**
   - フリーセット機能による位置リセット
   - 警告ダイアログ: 「位置がリセットされます。よろしいですか？」
   - 位置設定確認画面の表示

## 10-3 試運転

ここでは、MEX02を使用して回転実行を行う手順を紹介します。

⚠️ **重要**: モーターを動かすときは位置場所のパネルを確認し、安全を確保してから操作してください。

1. ツールバーの[ティーチング・リモート操作]アイコン、またはショートカットボタンの[ティーチング・リモート操作]をクリックします。
   ティーチング・リモート操作のウィンドウが表示されます。

   **操作ボタン:**
   ```
   🔧 ⭕ 🎯  または  👥ティーチング・リモート操作
   ```

2. [ティーチング・リモート操作を開始する]をクリックします。
   警告ウィンドウが表示されるので、はいをクリックしてください。

   **画面操作:**
   - 開始確認ダイアログ: 「ティーチング・リモート操作を開始しますが、よろしいですか？」
   - 操作開始ボタンの表示

   ティーチング・リモート操作を停止する際は、PWR/ALRM LEDの表示と制御の時間に注意して、
   適切に停止することを確認します。

3. JOG操作ボタンをクリックして、モーターを試運転します。

   **JOGパネル構成:**
   | ボタン | 機能 | 説明 |
   |--------|------|------|
   | ⏮⏮ | 高速逆転 | 高速でマイナス方向に回転 |
   | ⏮ | 低速逆転 | 低速でマイナス方向に回転 |
   | ⏸ | 停止 | モーター停止 |
   | ⏹ | 緊急停止 | 緊急時の停止 |
   | ⏭ | 低速正転 | 低速でプラス方向に回転 |
   | ⏭⏭ | 高速正転 | 高速でプラス方向に回転 |

   **表示項目:**
   - チューニング設定ウィンドウ
   - 回転位置表示
   - 動作状態インジケーター
   - FREE-ON操作ボタン
