### ■ 号機番号（デバイスアドレス）

号機設定スイッチ（ID）と機能設定スイッチ（SW1）のNo.1を併用して、号機番号（デバイスアドレス）を設定します。号機番号（デバイスアドレス）は重複しないように設定してください。

**出荷時設定**
- 位置決め機能内蔵タイプ: 号機番号0（ID:0、SW1-No.1:OFF）
- RS-485通信付きパルス列入力タイプ: 号機番号1（ID:1、SW1-No.1:OFF）

| ID | SW1-No.1 | 号機番号（デバイスアドレス） | ID | SW1-No.1 | 号機番号（デバイスアドレス） |
|----|----------|------------------------|----|----------|------------------------|
| 0  |          | 0※                     | 0  |          | 16                     |
| 1  |          | 1                      | 1  |          | 17                     |
| 2  |          | 2                      | 2  |          | 18                     |
| 3  |          | 3                      | 3  |          | 19                     |
| 4  |   OFF    | 4                      | 4  |   ON     | 20                     |
| 5  |          | 5                      | 5  |          | 21                     |
| 6  |          | 6                      | 6  |          | 22                     |

| A  |          | 10                     | A  |          | 26                     |

※ Modbusプロトコルの場合、号機番号（デバイスアドレス）0はブロードキャストで予約されているので、使用しないでください。

### ■ 通信速度

通信速度設定スイッチ（BAUD）で、通信速度を設定します。
通信速度は、上位システムの通信速度と同じ値を設定してください。

**出荷時設定**
- 位置決め機能内蔵タイプ: 7
- RS-485通信付きパルス列入力タイプ: 4

| BAUD | 通信速度 (bps) |
|------|---------------|
| 0    | 9,600         |
| 1    | 19,200        |
| 2    | 38,400        |
| 3    | 57,600        |
| 4    | 115,200       |
| 5    | 230,400       |
| 6    | 使用しません   |
| 7    | ネットワークコンバータ |
| 8～F | 使用しません   |

**memo**: 6、および8～Fの目盛りは設定しないでください。
