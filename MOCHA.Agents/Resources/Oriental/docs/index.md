### 02_αSTEP-AZ-DC（αSTEP-AZ-DC取扱説明書）
**フォルダ:** `02_αSTEP-AZ-DC/`
**ファイル数:** 56ページの取扱説明書
**対象ページ:** 2-57
**マニュアル型番:** HM-60313J

**主な内容:**
- **はじめに**: 製品使用前の注意事項、関連取扱説明書、一般仕様、RS-485通信仕様
- **DC電源仕様**: DC電源対応のαSTEPコントローラーシステム
- **一般仕様**: 使用温度0～+50℃、使用湿度85%RH以下、絶縁抵抗100MΩ以上
- **RS-485通信**: EIA-485規格、Modbus RTUプロトコル、最大31台接続対応
- **通信設定**: 9,600～230,400bps対応、ツイストペア線使用、終端抵抗対応
- **電気的特性**: ストレートケーブル、TIA/EIA-568B CAT5e以上推奨
- **接続仕様**: ネットワーク最大接続台数10台、上位システム1台対最大31台
- **設置環境**: 海抜1,000m以下、腐食性・易燃性ガス無環境、保存温度-25～+70℃
- **絶縁仕様**: DC500V 1分間印加で100MΩ以上（入力プラス－電源マイナス間）
- **通信プロトコル**: 非同期半二重通信、データ8bit、ストップビット1bit
