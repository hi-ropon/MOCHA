### テキスト
#### ストアードデータ(SD)運転
● 使用例  
運転データNo.0の絶対位置決め押し当て運転を実行する場合  
- 押し当てなしの場合：No.0の運転終了後、No.1の運転を開始します。（イベント発生なし）  
- 押し当てありの場合：TLC出力のONエッジを検出後、No.2の運転を開始します。（弱イベント発生）

#### 運転データの設定

#### 運転イメージ

- 左グラフは「速度」と「位置」の関係を示し、押し当ての有無による動作の違いを表現しています。  
- 右グラフは「速度」と「時間」の関係を示し、START信号とTLC信号のタイミングを示しています。

```mermaid
flowchart LR
    subgraph 左グラフ[速度-位置グラフ]
        A1[速度 1000] --> A2[位置 2000]

    end
    subgraph 右グラフ[速度-時間グラフ]
        B1[START 信号] --> B2[速度 1000]
        B2 --> B3[押し当てなし]
        B2 --> B4[押し当てあり 0.5秒]
        B4 --> B5[TLC ONエッジ]
    end
```

**図の説明:**  
- 左グラフは押し当ての有無で位置と速度の変化を示し、押し当てありの場合は速度が逆方向に変化することを示しています。  
- 右グラフはSTART信号から運転開始し、TLC信号のONエッジで押し当てありの運転が開始されるタイミングを示しています。
