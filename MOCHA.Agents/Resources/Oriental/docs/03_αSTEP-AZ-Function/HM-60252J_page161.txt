このページは「入出力信号の概要」に関する内容で構成されています。主にテキストと表で構成されており、各種入力機能（コンポジット入力機能、仮想入力機能）についてのパラメータ名、内容、初期値が表形式で整理されています。グラフや図は含まれていません。

### テキスト
#### コンポジット入力機能
- MEXE02ツリー表示: Direct-IN機能選択(DIN)
- パラメータ名: コンポジット入力機能
- 内容: DIN0～DIN9に、コンポジット入力機能として割り付ける入力信号を選択する。
  - 【設定範囲】164ページ「2-1 入力信号一覧」
- 初期値: 0（未使用）

#### 仮想入力
##### 仮想入力機能
- MEXE02ツリー表示: EXT-IN・VIR-IN・USR-OUT機能選択
- パラメータ名: 仮想入力機能
- 内容: VIR-IN0～VIR-IN3に割り付ける入力信号を選択する。
  - 【設定範囲】164ページ「2-1 入力信号一覧」
- 初期値: 0（未使用）

##### 仮想入力源選択
- MEXE02ツリー表示: EXT-IN・VIR-IN・USR-OUT機能選択
- パラメータ名: 仮想入力源選択
- 内容: VIR-IN0～VIR-IN3のトリガにする出力信号を選択する。
  - 【設定範囲】166ページ「2-2 出力信号一覧」
- 初期値: 128（CONST-OFF）

##### 仮想入力接点設定（信号反転）
- MEXE02ツリー表示: EXT-IN・VIR-IN・USR-OUT機能選択
- パラメータ名: 仮想入力接点設定（信号反転）
- 内容: VIR-IN0～VIR-IN3の接点を変更する。
  - 【設定範囲】
    - 0: 反転しない
    - 1: 反転する
- 初期値: 0

##### 仮想入力ON信号検出不感時間
- MEXE02ツリー表示: EXT-IN・VIR-IN・USR-OUT機能選択
- パラメータ名: 仮想入力ON信号検出不感時間
- 内容: VIR-IN0～VIR-IN3のON信号検出不感時間を設定する。
  - 【設定範囲】0～250 ms
- 初期値: 0

##### 仮想入力強制1shot
- MEXE02ツリー表示: EXT-IN・VIR-IN・USR-OUT機能選択
- パラメータ名: 仮想入力強制1shot
- 内容: VIR-IN0～VIR-IN3の強制1shot機能を有効にする。
  - 【設定範囲】
    - 0: 無効
    - 1: 有効
- 初期値: 0
