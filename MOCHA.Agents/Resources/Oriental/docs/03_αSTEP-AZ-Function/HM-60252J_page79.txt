### テキスト
#### 3-5 連続SD運転
連続SD運転とは、運転データに運転速度を設定して実行する運転である。運転速度を正の値にするとFWD方向、負の値にするとRVS方向へ一定の速度で運転し続ける。

##### 運転動作
- 0 < 運転速度（FWD方向）の場合  
  運転速度が正の値のときの動作を説明。

- 0 > 運転速度（RVS方向）の場合  
  運転速度が負の値のときの動作を説明。

##### memo
- 連続SD運転の目標位置は開始位置（指令位置）であり、運転データの「位置」は設定しない。
- 連続運転（トルク）を設定した場合は自起動運転となる。

### グラフ
#### 0 < 運転速度（FWD方向）の場合
- 速度と位置の関係グラフ：起動速度から起動・変速を経て運転速度に達し、その速度を維持しながら位置が増加。
- 電流と時間の関係グラフ：起動時に停止電流から運転電流に変化し、一定の運転電流を維持。

#### 0 > 運転速度（RVS方向）の場合
- 速度と位置の関係グラフ：起動速度から起動・変速を経て運転速度に達し、その速度を維持しながら位置が減少。
- 電流と時間の関係グラフ：起動時に停止電流から運転電流に変化し、一定の運転電流を維持。

（グラフは速度・位置・電流・時間の関係を示し、FWD方向とRVS方向で対称的な動作を表現）
