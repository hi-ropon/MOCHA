### テキスト
#### 簡易ダイレクトデータ運転コマンド
簡易ダイレクトデータ運転を行うときに使用するコマンドです。設定した値はRAMに保存されます。  
簡易ダイレクトデータ運転コマンドは、FAネットワーク専用です。

### 表
| FAネットワーク命令コード | 名称                           | 内容                                                                                  | 初期値 |
|--------------------------|--------------------------------|---------------------------------------------------------------------------------------|--------|
| READ 53 (0035h)           | WRITE 4149 (1035h)              | 簡易ダイレクトデータ運転モニタ0(NETC)                                               | 0      |
|                          |                                | 簡易ダイレクトデータ運転の目標位置を設定します。                                    |        |
|                          |                                | 【設定範囲】 -2,147,483,648～2,147,483,647 step                                      |        |
| READ 54 (0036h)           | WRITE 4150 (1036h)              | 簡易ダイレクトデータ運転モニタ1(NETC)                                               | 1,000  |
|                          |                                | 簡易ダイレクトデータ運転の運転速度を設定します。                                    |        |
|                          |                                | 【設定範囲】 -4,000,000～4,000,000 Hz                                               |        |
