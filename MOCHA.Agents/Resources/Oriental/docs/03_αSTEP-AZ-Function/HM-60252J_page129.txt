### テキスト
#### 複合JOG運転
複合JOG運転では、FW-JOG-C入力またはRV-JOG-C入力をONにしている間、インチング運転→JOG運転→高速JOG運転の順に運転が遷移します。FW-JOG-C入力またはRV-JOG-C入力がOFFからONになると運転を開始し、OFFになると減速停止します。

■運転イメージ
- 「JOG-C高速連続運転移行時間」パラメータで設定した時間が過ぎると高速JOG運転を開始する
- 「JOG-C連続運転移行時間」パラメータで設定した時間が過ぎるとJOG運転を開始する
- FW-JOG-C入力をONにするとFWD方向へインチング運転を始める
- FW-JOG-C入力をOFFにすると減速停止を始める

### 図
```mermaid
flowchart LR
    A[FW-JOG-C入力 (RV-JOG-C入力)] -->|ON| B[FWD方向へインチング運転開始]
    B --> C[モーター動作: インチング運転]
    C --> D[「JOG-C連続運転移行時間」経過後]
    D --> E[モーター動作: JOG運転開始]
    E --> F[「JOG-C高速連続運転移行時間」経過後]
    F --> G[モーター動作: 高速JOG運転開始]
    A -->|OFF| H[減速停止開始]
```
**図の説明:**
- FW-JOG-C入力がONになるとFWD方向へインチング運転を開始し、設定時間経過でJOG運転、さらに高速JOG運転へと段階的に移行する。
- FW-JOG-C入力がOFFになると減速停止を開始する。
