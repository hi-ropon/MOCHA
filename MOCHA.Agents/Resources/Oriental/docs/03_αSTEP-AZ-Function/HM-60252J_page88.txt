### グラフ（波形図）
複数の入出力信号のON/OFF状態を時間軸に沿って示した波形図です。各信号の名称とON/OFF状態が縦に並び、横軸は時間の経過を示しています。信号のON状態は灰色の塗りつぶしで示され、OFF状態は白色で示されています。

| 信号名       | 説明（推定）               |
|--------------|----------------------------|
| モーター動作 | モーターの稼働状態         |
| START入力    | スタート信号入力           |
| SSTART入力   | セカンドスタート信号入力   |
| M0～M7入力  | M0からM7までの入力信号     |
| MOVE出力     | 移動動作の出力信号         |
| READY出力   | 準備完了出力信号           |
| IN-POS出力  | 位置到達出力信号           |
| SEQ-BSY出力 | シーケンスビジー出力信号   |
| OPE-BSY出力 | 操作ビジー出力信号         |
| DELAY-BSY出力 | 遅延ビジー出力信号       |
| CRNT出力    | カレント出力信号           |
| MBC出力     | MBC出力信号                |
| D-END0出力 | D-END0出力信号             |
| D-END1出力 | D-END1出力信号             |
| D-END2出力 | D-END2出力信号             |
| M-ACT0～M-ACT7出力 | M-ACT0からM-ACT7までの出力信号 |
| M-CHG出力  | M-CHG出力信号              |
| AUTO-CD出力 | 自動CD出力信号             |

### 図の説明
- 各信号のON/OFF状態が時間軸に沿って変化し、モーター動作や各種入力・出力信号のタイミング関係を示している。
- 例えば、START入力がONになるとモーター動作が開始し、MOVE出力やREADY出力などの信号が連動して変化する様子が読み取れる。
- M0～M7入力やM-ACT0～M-ACT7出力は複数の信号をまとめて示している。

```mermaid
flowchart TD
    A[START入力 ON] --> B[モーター動作 ON]
    B --> C[MOVE出力 ON]
    C --> D[READY出力 ON]
    D --> E[IN-POS出力 ON]
    E --> F[SEQ-BSY出力 ON]
    F --> G[OPE-BSY出力 ON]
    G --> H[DELAY-BSY出力 ON]
    H --> I[CRNT出力 ON]
    I --> J[MBC出力 ON]
    J --> K[D-END0出力 ON]
    K --> L[D-END1出力 ON]
    L --> M[D-END2出力 ON]
    M --> N[M-ACT0～M-ACT7出力 ON]
    N --> O[M-CHG出力 ON]
    O --> P[AUTO-CD出力 ON]
```
