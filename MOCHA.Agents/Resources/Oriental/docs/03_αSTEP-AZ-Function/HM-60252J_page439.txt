### テキスト
#### CH設定の操作手順
3. [CH Settings]をクリックすると、CH設定ウィンドウが表示され、CHごとに測定条件を設定できる。

4. [RUN]をクリックすると波形の測定が開始される。

5. 測定中に[STOP]をクリックすると波形測定を終了する。TriggerのModeが[SINGLE]の場合は波形描画終了と同時に自動終了。

6. 波形測定を終了する際は、「波形モニタを開始する」のチェックを外す。

#### CH設定ウィンドウの説明
- CH1～CH4は指令速度や検出速度などの項目を設定。
- CH5～CH12は入出力信号の設定。

### 表
| 番号 | 説明 |
| :--- | :--- |
| 1 | 波形の表示位置を上下に移動させる。 |
| 2 | 測定した項目を反転表示させる。 |
| 3 | 測定する内容を選択。CH1～CH4は指令速度や検出速度など、CH5～CH12は入出力信号。 |
| 4 | 表示スケールを選択（CH1～CH4のみ）。⑤と組み合わせて拡大表示可能。 |
| 5 | 設定したオフセット値を表示に加算（CH1～CH4のみ）。④と組み合わせて拡大表示可能。 |

### 画像説明
CH Settingsウィンドウには12チャンネル(CH1～CH12)の設定項目があり、各CHに対して以下の設定が可能。
- Pos.（波形表示位置上下移動）
- Inv.（反転表示チェックボックス）
- 測定内容選択ドロップダウン（例：指令速度、検出速度、MOVE、READYなど）
- Scale（表示スケール）
- Offset（オフセット値）

各CHの設定は個別に行い、波形モニタの表示をカスタマイズできる。
