### テキスト
#### 9-3 直接参照（互換用）の概要
当社の既存製品からAZシリーズに置き換えるときに便利な入力方法です。  
方式、位置、速度などの設定項目ごとにまとめたアドレスで、アドレスなどが既存製品と同じなので特別な設定をしなくても使用可能です。（詳細は387ページ参照）

**重要**  
- 設定できる運転データはNo.0～No.63の64個まで。No.64以降（65個以上）は設定不可。  
- 対応できる設定項目は6種類（結合やループなど他の項目は設定不可）。  
  方式、位置、速度、起動・変速レート、停止レート、運転電流。

### 注釈
| 注釈番号 | 内容 |
| :------- | :--- |
| ※1 | 運転終了遅延、結合、エリア、ループ、イベントは設定できません。 |
| ※2 | 運転データNo.64以降は設定できません。 |
