### テキスト
#### モニタ機能
- 出力例として、機械原点からFWD方向へ700 step回転した場合のパラメータ設定例が示されています。
- ラウンド範囲が-500 step～499 stepのため、現在座標はその範囲内で出力されることが説明されています。

#### アラームコード
- 現在発生しているアラームのアラームコードを8bitのデータで送信することが説明されています。
- 例として過負荷アラーム（アラームコード30h）が発生している場合の2進数データが示されています。

#### 現在位置＋アラームコード
- 現在位置情報とアラームコードを連続して送信することが説明されています。

#### チェックサム
- 送信データを1バイトごとに区切り、1バイトずつ加算した結果の下位8bitをチェックサムとする説明。
- データが正しく出力されたか確認するための情報であることが記載されています。
- 出力例として、検出位置300 step、ハードウェアオーバートラベルのアラーム発生中（アラームコード66h）の場合のチェックサム計算例が示されています。

| 項目           | 内容                             |
| :------------- | :------------------------------- |
| 2進数          | 0000 0000 0000 0000 0000 0010 1011 1100 |
| 送信データ(LSBファースト) | 0011 1101 0100 0000 0000 0000 0000 0000 |

#### アラームコード（過負荷アラーム30h発生時）
| 項目           | 内容         |
| :------------- | :----------- |

| 送信データ(LSBファースト) | 0000 1100    |

#### チェックサム計算例
| 項目           | 内容                             |
| :------------- | :------------------------------- |
| 検出位置       | 300 step = 0000 0000 0000 0000 0000 0001 0010 1100 |
| アラームコード | 66h = 0110 0110                  |
| チェックサム   | 0000 0000 + 0000 0000 + 0000 0001 + 0010 1100 + 0110 0110 = 1001 0011 |
