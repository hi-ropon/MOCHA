### テキスト
#### ラッチ機能の例
- NEXT入力によるラッチの動作イメージを示しています。

#### 運転イメージの説明
- 運転速度、START入力、NEXT入力、LAT-CLR入力、NEXT-LAT出力の各信号のON/OFFタイミングと持続時間（2ms以上、2ms以下）が示されています。

### 図（運転イメージ）
```mermaid
flowchart TD
    subgraph 運転速度
        direction LR
        A1[No.0] --> A2[No.1]
    end
    START_INPUT[START入力] -->|ON 2ms以上| START_ON
    NEXT_INPUT[NEXT入力] -->|ON 2ms以上| NEXT_ON
    LAT_CLR_INPUT[LAT-CLR入力] -->|ON 2ms以上| LAT_CLR_ON
    NEXT_LAT_OUTPUT[NEXT-LAT出力]

    START_INPUT -.->|OFF| START_OFF
    NEXT_INPUT -.->|OFF| NEXT_OFF
    LAT_CLR_INPUT -.->|OFF| LAT_CLR_OFF
    NEXT_LAT_OUTPUT -.->|OFF| NEXT_LAT_OFF

    %% 矢印でタイミング関係を示す（概念的）
    START_ON --> NEXT_ON
    NEXT_ON --> LAT_CLR_ON
    LAT_CLR_ON --> NEXT_LAT_OUTPUT
```
**図の説明:**  
運転速度がNo.0からNo.1に変わる間に、START入力、NEXT入力、LAT-CLR入力がそれぞれ2ms以上ON/OFFされるタイミングを示し、NEXT-LAT出力がそれに応じて変化する様子を表しています。

### 画像（波形モニタ MEXE02）
- 波形モニタ画面のキャプチャがあり、START、NEXT、LAT-CLR、NEXT-LATの各信号の波形が色分けされて表示されています。
- 画面にはトリガ設定、タイムスケール、チャネル選択などの操作パネルがあり、信号の詳細なタイミング解析が可能です。
