### テキスト
#### AREA0～AREA7出力
モーターが設定したエリア内にあるとき、AREA出力がONになります。  
モーターの停止中でも、モーターがエリア内にあるときはONになります。

#### 関連するパラメータ
以下のパラメータがAREA出力に関連しています。

### 表
| MEXE02ツリー表示 | パラメータ名 | 内容 | 初期値 |
| :--- | :--- | :--- | :---: |
| I/O動作・機能 | AREA0＋位置/オフセット ～ AREA7＋位置/オフセット | AREA0～AREA7出力の＋方向位置、または目標位置からのオフセットを設定します。<br>【設定範囲】<br>-2,147,483,648～2,147,483,647 step | 0 |
|  | AREA0－位置/判定距離 ～ AREA7－位置/判定距離 | AREA0～AREA7出力の－方向位置、またはオフセット位置からの距離を設定します。<br>【設定範囲】<br>-2,147,483,648～2,147,483,647 step | 0 |
|  | AREA0 範囲指定方法 AREA7 範囲指定方法 | AREA0～AREA7の範囲指定方法を設定します。<br>【設定範囲】<br>0：絶対値で範囲指定<br>1：目標位置からのオフセット・幅を指定 | 0 |
|  | AREA0 位置判定基準 ～ AREA7 位置判定基準 | AREA0～AREA7の位置判定基準を設定します。<br>【設定範囲】<br>0：検出位置基準<br>1：指令位置基準 | 0 |

### 図
#### 「AREA範囲指定方法」パラメータが「絶対値で範囲指定」の場合の動作イメージ

```mermaid
flowchart LR
    subgraph 範囲指定1
        direction TB
        ON1[AREA出力 ON]
        OFF1[AREA出力 OFF]
        OFF1 -->|AREA－方向位置| ON1
        ON1 -->|AREA＋方向位置| OFF1
    end

    subgraph 範囲指定2
        direction TB
        OFF2[AREA出力 OFF]
        ON2[AREA出力 ON]
        ON2 -->|AREA＋方向位置| OFF2
        OFF2 -->|AREA－方向位置| ON2
    end

    subgraph 範囲指定3
        direction TB
        OFF3[AREA出力 OFF]
        ON3[AREA出力 ON]
        OFF3 -->|AREA＋方向位置 = AREA－方向位置| ON3
    end
```

- 「AREA＋位置/オフセット」パラメータ > 「AREA－位置/判定距離」パラメータのとき、モーターの位置が「AREA－位置/判定距離」以上、または「AREA＋位置/オフセット」以下のとき、AREA出力がONになる。  
- 「AREA＋位置/オフセット」パラメータ < 「AREA－位置/判定距離」パラメータのとき、モーターの位置が「AREA＋位置/オフセット」以下、または「AREA－位置/判定距離」以上のとき、AREA出力がONになる。  
- 「AREA＋位置/オフセット」パラメータ = 「AREA－位置/判定距離」パラメータのとき、AREA出力がONになる。

#### 「AREA範囲指定方法」パラメータが「目標位置からのオフセット・幅を指定」の場合の動作イメージ

```mermaid
flowchart TB
    モーター動作 --> モーター動作詳細[モーター動作範囲]
    モーター動作詳細 --> AREA出力[AREA出力 ON/OFF]
    AREA出力 -->|幅| 幅1[幅]
    AREA出力 -->|幅| 幅2[幅]
    AREA出力 -->|オフセット| オフセット[オフセット]
```

- モーター動作の範囲に対して、オフセットと幅を指定し、その範囲内でAREA出力がONになる。
