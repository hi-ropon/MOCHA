### テキスト
#### A相/B相出力
- ドライバのI/Oコネクタには、出荷時にA相（ASG）出力とB相（BSG）出力が割り付けられている。
- A相出力とB相出力はABZOセンサから出力されるパルス信号であり、これらを使うことでモーターの現在位置や回転方向を検出可能。
- A相/B相出力は差動出力であるため、外部機器の入力回路は差動出力に対応するものを接続する必要がある。

#### 重要
- A相/B相出力は差動出力であることを強調し、外部機器の入力回路は差動出力に対応したものを使用することが重要。

- ドライバのI/Oコネクタピン配置（抜粋）
  - Pin No.11: ASG+（A相出力+）
  - Pin No.12: BSG+（B相出力+）
  - Pin No.23: ASG-（A相出力-）
  - Pin No.24: BSG-（B相出力-）
  - Pin No.1～13はAC電源ドライバCN5、Pin No.1～24はDC電源ドライバCN4に対応

### メモ
- 「差動出力機能選択」パラメータで「A相/B相出力」を選択すると、現在の検出位置が位相差形式で出力される。
- A相出力とB相出力のパルスの分解能は電源投入時のモーター分解能と同じであり、モーター分解能を変更するとA相/B相出力の分解能も変わる。
