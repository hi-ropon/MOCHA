### テキスト
#### グループ機能の命令コードと名称
- READ命令コード：2351（092Fh）
- WRITE命令コード：6447（192Fh）
- 名称：R-IN15グループ動作モード初期状態（NETC）
- 内容：リモートI/Oの動作モードの初期状態を設定する。
  - 設定範囲：
    - 0：軸IDで動作
    - 1：グループIDで動作
- 初期値：0
- 備考：「R-INグループ動作モード初期状態（NETC）」パラメータの設定によって異なる。

#### R-INグループ動作モード（NETC）のbit配置
- bit15～bit0までの16bitがR-IN15～R-IN0に対応。
- bit15～bit8は上位バイト、bit7～bit0は下位バイトとして表記。
- 各bitに対応するR-IN番号が示されている。

#### 設定例

| Dec    | Hex    | 設定内容                                                                                         |
| :------| :-------| :-----------------------------------------------------------------------------------------------|
| 0      | 0000h  | すべてのbitを「0」に設定した場合。<br>R-IN0～R-IN15はすべてドライバごとに入力する。（初期状態） |
| 1      | 0001h  | bit0だけを「1」、その他を「0」に設定した場合。<br>R-IN0はグループに入力。R-IN1～R-IN15はドライバごとに入力。 |
| 2      | 0002h  | bit1だけを「1」、その他を「0」に設定した場合。<br>R-IN1はグループに入力。R-IN0とR-IN2～R-IN15はドライバごとに入力。 |
| 65,535 | FFFFh  | すべてのbitを「1」に設定した場合。<br>R-IN0～R-IN15はすべてグループ宛に入力。 |
