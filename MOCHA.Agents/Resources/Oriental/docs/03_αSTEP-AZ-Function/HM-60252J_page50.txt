テキスト部分では、ティーチング・リモート運転の開始方法、JOG運転ボタンの使用、ソフトウェアリミット検出時のアラーム対応について段階的に説明しています。

<contents>
### テキスト
#### ■動作確認

1. ショートカットボタンの[ティーチング・リモート運転]、またはツールバーの[ティーチング・リモート運転]アイコンをクリックします。  
2. [ティーチング・リモート運転を開始する]をクリックします。  
3. JOG運転ボタンでモーターを運転します。  
4. 設定したソフトウェアリミットが検出されると、アラームが発生します。  
　ドライバステータス欄の「現在のアラーム」に「67:ソフトウェアオーバートラベル」が表示されます。  
5. [アラームリセット]をクリックしてアラームを解除します。  
　アラーム解除後、[ZHOME運転]やJOG運転ボタンでソフトウェアリミットから脱出してください。

### 図
- ティーチング・リモート運転のショートカットボタンとツールバーアイコンの画像  
- 「ティーチング・リモート運転を開始する」ボタンのある画面  
- JOG運転ボタンの位置を示した画面  
- アラーム発生時の「現在のアラーム」表示画面（67:ソフトウェアオーバートラベル）  
- アラームリセットボタンの位置を示した画面

**図の説明:**  
- ティーチング・リモート運転を開始し、JOG運転ボタンでモーターを動かす。  
- ソフトウェアリミットに達するとアラームが発生。  
- アラームリセットで解除し、リミットから脱出する操作手順を示す。
