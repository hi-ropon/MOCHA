### テキスト
#### EtherCAT通信で制御する場合の設定手順
4. NETC01-ECTのスイッチ設定  
- RS-485通信速度を625 kbpsに設定し、SW1の7番をONにする。  
- EtherCATノードアドレスは1に設定し、ECAT IDの×10を「0」、×1を「1」に設定する。  
- 設定後は図のようなスイッチ配置になる。

5. NETC01-ECTの電源を切る。

##### memo
- 複数ドライバ接続時は、ドライバ数分だけ「接続（号機番号）」パラメータを設定する。  
- 変更したパラメータ反映には電源再投入が必要。  
- SW1は常時「7」に設定。0～6は使用不可。7以上の設定は通信スイッチ異常アラーム発生。  

#### STEP 3 ドライバのスイッチ設定
- ドライバのスイッチを以下のように設定する。  
- プロトコルは「OFF」（ネットワークコンバータ）を選択。  

### 表
#### NETC01-ECTスイッチ設定内容
| 設定内容               | スイッチ設定               | 出荷時設定           |
|------------------------|----------------------------|----------------------|
| RS-485通信速度: 625 kbps | SW1を[7]                  | 7                    |
| EtherCATノードアドレス: 1 | ECAT IDの×10を「0」、×1を「1」 | 1（×10:0、×1:1）     |

#### ドライバのスイッチ設定内容
| 設定内容               | スイッチ設定                                      |
|------------------------|--------------------------------------------------|
| プロトコル：ネットワークコンバータ | SW1のNo.2をOFF                                  |
| 号機番号：0             | SW1のNo.1をOFF、IDを0                             |
| 終端抵抗：ON            | AC電源ドライバ：TERMのNo.1とNo.2をON<br>DC電源ドライバ：SW1のNo.3とNo.4をON |

### 図
#### NETC01-ECTスイッチ配置図
```mermaid
flowchart LR
    A[NETC01-ECT本体] --> B[RS-485通信速度設定スイッチ(SW1)]
    A --> C[ノードアドレス設定スイッチ(ECAT ID)]
    B -.-> D[SW1の7番をON]
    C -.-> E[×10=0、×1=1に設定]
```

#### AC電源ドライバのスイッチ配置
```mermaid
flowchart TD
    A[AC電源ドライバ本体] --> B[号機番号設定スイッチ(ID)]
    A --> C[通信速度設定スイッチ(BAUD)]
    A --> D[機能設定スイッチ(SW1)]
    A --> E[終端抵抗設定スイッチ(TERM.)]

    D --> F[No.2:プロトコル ON]
    D --> G[No.1:号機番号 ON]
    E --> H[終端抵抗 ON]
```

#### DC電源ドライバのスイッチ配置
```mermaid
flowchart TD
    A[DC電源ドライバ本体] --> B[機能設定スイッチ(SW1)]
    A --> C[号機番号設定スイッチ(ID)]
    A --> D[通信速度設定スイッチ(BAUD)]

    B --> E[No.3, No.4:終端抵抗 ON]
    B --> F[No.2:プロトコル ON]
    B --> G[No.1:号機番号 ON]
```

**図の説明:**  
- NETC01-ECTはRS-485通信速度設定スイッチ(SW1)とノードアドレス設定スイッチ(ECAT ID)があり、設定値に従いスイッチを操作する。  
- AC電源ドライバはID設定スイッチ、通信速度設定スイッチ、機能設定スイッチ(SW1)、終端抵抗設定スイッチ(TERM.)がある。  
- DC電源ドライバは機能設定スイッチ(SW1)、ID設定スイッチ、通信速度設定スイッチがあり、SW1の各番号スイッチで終端抵抗、プロトコル、号機番号を設定する。
