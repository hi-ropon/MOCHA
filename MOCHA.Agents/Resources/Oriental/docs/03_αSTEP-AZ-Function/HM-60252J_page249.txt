### テキスト
#### USB-IDについて
USB-IDは、パソコンのUSBポート（COMポート番号）をドライバに紐付けるパラメータです。COMポート番号はMEXE02で通信ポートを設定する際に使用します。複数のドライバをパソコンに接続すると、空いているCOMポート番号を接続順に割り振ります。ドライバの電源を再投入したりUSBケーブルを抜き差しすると、パソコンが認識する接続順序が変わり、割り振られたCOMポート番号も変わる場合があります。

#### USB-IDを設定しない場合
COMポート番号と接続状況の一覧表が示されており、空きポートには電源投入順にドライバが割り当てられます。

#### COMポート番号と接続状況

| COMポート番号 | 接続状況 |
| :------------ | :------- |
| 1             | 接続済み |
| 2             | 接続済み |
| 3             | 空き     |
| 4             | 接続済み |
| 5             | 空き     |
| 6             | 空き     |

#### COMポート番号の割り当てと電源投入順序の関係

```mermaid
flowchart LR
    PC[MEXE02パソコン] -->|1| DriverA[ドライバA COM3]
    PC -->|2| DriverB[ドライバB COM5]
    PC -->|3| DriverC[ドライバC COM6]

    subgraph 電源投入順序変更後
        PC2[MEXE02パソコン]
        PC2 -->|3| DriverA2[ドライバA COM6]
        PC2 -->|1| DriverB2[ドライバB COM3]
        PC2 -->|2| DriverC2[ドライバC COM5]
    end

    note1[電源投入順序を変えるとCOMポート番号も変わる]
    PC -.-> note1
    PC2 -.-> note1
```

**図の説明:**
- 左側は電源投入順序1→2→3の場合のCOMポート割り当て。
- 右側は電源投入順序3→1→2に変えた場合のCOMポート割り当て。
- 電源投入順序の違いにより、ドライバに割り当てられるCOMポート番号が変わることを示している。
