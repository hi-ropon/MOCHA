### テキスト
#### 動力遮断機能の故障検出
HWTO1入力とHWTO2入力の状態およびEDM出力の状態を監視することで、動力遮断機能の故障を検出可能です。  
正常動作時の信号組み合わせは表の通りで、この組み合わせ以外は故障を示します。  
HWTO1またはHWTO2の片方だけがONまたはOFFの場合は外部機器や配線の故障が疑われ、EDM出力はOFFとなりモーターは無励磁状態になります。

**重要事項**  
- EDM出力がOFFのときは動力遮断機能を解除しないこと。  
- ドライバや外部機器の故障、配線異常が発生した場合は原因を確認し速やかに対処すること。  
- 動力遮断機能はISO 13849-1のカテゴリ3に分類されるが、EDM出力だけで全ての危険側故障を検出できるわけではない。

#### 7-5 使用例
セーフティユニットを用いて扉（ガード）を開けた際に動力遮断機能が動作する接続方法を示す。

### 図
#### セーフティユニットとドライバの接続図
```mermaid
flowchart LR
    扉(ガード) -->|開く| 安全スイッチ
    安全スイッチ --> セーフティユニット
    セーフティユニット -->|安全入力1| CN1_HWTO1+
    セーフティユニット -->|安全入力2| CN1_HWTO2+
    セーフティユニット -->|安全出力1| ドライバ_HWTO1+
    セーフティユニット -->|安全出力2| ドライバ_HWTO2+
    セーフティユニット -->|フィードバック入力| CN1_EDM+
    ドライバ -->|EDM出力| セーフティユニット
    電源 -->|DC24V| セーフティユニット
    電源 -->|0V| セーフティユニット
```

#### 正常時の動作フロー
```mermaid
flowchart TD
    subgraph 扉(ガード)を開けたとき
        A1[扉(ガード)を開ける。] --> A2[セーフティユニットの安全入力1と安全入力2がOFFになる。]
        A2 --> A3[セーフティユニットの安全出力1と安全出力2がOFFになる。]
        A3 --> A4[ドライバのHWTO1入力とHWTO2入力がOFFになる。]
        A4 --> A5[動力遮断機能が動作し、EDM出力がONになる。]
    end

    subgraph 扉(ガード)を閉めたとき
        B1[扉(ガード)を閉める。] --> B2[セーフティユニットの安全入力1と安全入力2がONになる。]
        B2 --> B3[フィードバック入力(ドライバのEDM出力)がONのため、セーフティユニットの安全出力1と安全出力2がONになる。]
        B3 --> B4[ドライバのHWTO1入力とHWTO2入力がONになり、動力遮断機能が解除される。]
    end
```

| HWTO1入力 | HWTO2入力 | EDM出力 |
| :-------- | :-------- | :------ |
| ON        | ON        | OFF     |
| OFF       | OFF       | ON      |
| ON        | OFF       | OFF     |
| OFF       | ON        | OFF     |
