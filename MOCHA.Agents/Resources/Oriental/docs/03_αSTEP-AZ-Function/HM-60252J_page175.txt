### テキスト
#### パルス列入力タイプ、RS-485通信付きパルス列入力タイプの場合
- ピンNo.1、2、13、14はパルス入力専用であり、他の機能は割り付けできない。
- 以下にパルス入力の接続方法を示す。

### 図
#### パルス入力がラインドライバのとき
```mermaid
flowchart LR
    subgraph コントローラ
        A1[ドライバ出力]
        A2[ドライバ出力]
    end
    subgraph ツイストペア線
        B1[ツイストペア線]
        B2[ツイストペア線]
    end
    subgraph ドライバ
        C1[470Ω] --> C2[270Ω] --> C3[270Ω] --> C4[5.6kΩ] --> C5[フォトカプラ]
        C6[470Ω] --> C7[270Ω] --> C8[270Ω] --> C9[5.6kΩ] --> C10[フォトカプラ]
    end
    A1 --> B1 --> C1
    A2 --> B2 --> C6
```
- ピン1: CW+[PLS+]
- ピン13: CW−[PLS−]
- ピン2: CCW+[DIR+]
- ピン14: CCW−[DIR−]
- 0V接地あり

#### パルス入力がオープンコレクタのとき（パルス入力信号の電圧がDC5Vの場合）
```mermaid
flowchart LR
    subgraph コントローラ
        A1[トランジスタ出力]
        A2[トランジスタ出力]
        V[DC5V]
    end
    subgraph ツイストペア線
        B1[ツイストペア線]
        B2[ツイストペア線]
    end
    subgraph ドライバ
        C1[470Ω] --> C2[270Ω] --> C3[270Ω] --> C4[5.6kΩ] --> C5[フォトカプラ]
        C6[470Ω] --> C7[270Ω] --> C8[270Ω] --> C9[5.6kΩ] --> C10[フォトカプラ]
    end
    V --> A1
    A1 --> B1 --> C1
    A2 --> B2 --> C6
    A2 --> 0V
```
- ピン1: CW+[PLS+]
- ピン13: CW−[PLS−]
- ピン2: CCW+[DIR+]
- ピン14: CCW−[DIR−]

#### パルス入力がオープンコレクタのとき（パルス入力信号の電圧がDC24Vの場合）
```mermaid
flowchart LR
    subgraph コントローラ
        V[DC24V]
        R1[抵抗 R1: 1.2kΩ〜2.2kΩ 0.5W以上]
        T1[トランジスタ出力]
        T2[トランジスタ出力]
        GND[0V]
    end
    subgraph ツイストペア線
        B1[ツイストペア線]
        B2[ツイストペア線]
    end
    subgraph ドライバ
        C1[470Ω] --> C2[270Ω] --> C3[270Ω] --> C4[5.6kΩ] --> C5[フォトカプラ]
        C6[470Ω] --> C7[270Ω] --> C8[270Ω] --> C9[5.6kΩ] --> C10[フォトカプラ]
    end
    V --> R1 --> T1
    T1 --> B1 --> C1
    T2 --> B2 --> C6
    T2 --> GND
```
- 抵抗R1は1.2kΩ〜2.2kΩ、0.5W以上を使用
- ピン1: CW+[PLS+]
- ピン13: CW−[PLS−]
- ピン2: CCW+[DIR+]
- ピン14: CCW−[DIR−]

### メモ
- CW[PLS]入力、CCW[DIR]入力はDC5V〜DC24Vで使用可能。
- DC24Vで使用する場合は外部抵抗R1（1.2kΩ〜2.2kΩ、0.5W以上）を接続すること。
- DC5Vで使用する場合は電圧を直接接続すること。
