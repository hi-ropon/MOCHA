### テキスト
#### オフセット（エリア）、幅（エリア）
オフセット（エリア）や幅（エリア）を設定することで、運転データごとにMAREA出力の範囲を設定できます。

#### 運転方向がFWD方向の場合
- 位置決め運転  
- 連続運転  

#### カウント（Loop）、位置オフセット（Loop）、終了（Loop）
カウント（Loop）、位置オフセット（Loop）、終了（Loop）を設定すると、ループ機能が有効になります。（詳細は94ページ「ループ機能」参照）

#### 弱イベント、強イベント
弱イベントと強イベントを設定すると、イベントジャンプ機能が有効になります。弱イベントと強イベントが同時に発生した場合は強イベントが優先されます。（詳細は98ページ「イベントジャンプ機能」参照）

### 図
#### 運転方向がFWD方向の場合の速度・位置・MAREA出力の関係

```mermaid
flowchart LR
    subgraph 位置決め運転
        A1[起動速度] --> A2[運転速度] --> A3[運転速度] --> A4[起動速度] --> A5[位置]
        A6[オフセット(エリア)] --> A7[幅] --> A8[幅]
        A9[MAREA出力] --> A10[ON/OFF]
    end

    subgraph 連続運転
        B1[起動速度] --> B2[運転速度] --> B3[運転速度] --> B4[位置]
        B5[目標位置] --> B6[オフセット(エリア)] --> B7[幅] --> B8[幅]
        B9[MAREA出力] --> B10[ON/OFF]
    end
```

**図の説明:**  
- 位置決め運転では、速度が起動速度から運転速度に上昇し、目標位置に達するまで一定速度を維持し、最後に速度が減速する。  
- 連続運転では、速度が起動速度から運転速度に上昇し、その速度を維持しながら位置が変化する。  
- MAREA出力はオフセット（エリア）と幅（エリア）で指定された範囲でON/OFFが切り替わる。
