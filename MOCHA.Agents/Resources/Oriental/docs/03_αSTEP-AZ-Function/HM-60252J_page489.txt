### テキスト
#### タイミングチャート
「T-MODE使用時停止中運転電流設定」パラメータが「停止電流」のときの動作タイミングを示しています。

- 指令速度、指令電流（運転電流と停止電流）、各種入力（T-MODE、CW、CCW）、各種出力（TLC、MOVE、PLS-RDY、READY、IN-POS、AUTO-CD）のON/OFF状態の変化を時間軸に沿って示しています。
- 注釈として、
  - ※1 負荷、運転速度、指令フィルタなどによって異なること
  - ※2 「オートカレントダウン判定時間」パラメータに設定した時間経過後に停止電流に切り替わること
が記載されています。

### 図（タイミングチャート）
```mermaid
flowchart TD
    A[指令速度] -->|時間経過| B[指令電流]
    B -->|運転電流/停止電流| C[T-MODE入力]
    C --> D[CW入力]
    C --> E[CCW入力]
    D --> F[TLC出力]
    E --> F
    F --> G[MOVE出力]
    G --> H[PLS-RDY出力]
    H --> I[READY出力]
    I --> J[IN-POS出力]
    J --> K[AUTO-CD出力]

    %% ON/OFFの切り替わりを示す矢印や注釈は省略し、信号の流れを示す構造図として表現
```

**図の説明:**
- 各信号のON/OFF状態が時間軸に沿って変化し、特にT-MODE入力のON期間中にCW入力やCCW入力が交互にON/OFFし、それに連動してTLC、MOVE、PLS-RDY、READY、IN-POS、AUTO-CDの出力が変化する様子を示しています。
- 指令電流は運転電流から停止電流に切り替わるタイミングがあり、これは「オートカレントダウン判定時間」パラメータに依存します。
