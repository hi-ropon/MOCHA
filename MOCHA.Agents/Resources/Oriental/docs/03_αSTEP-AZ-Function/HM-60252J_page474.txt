### テキスト
#### パルス入力方式の種類
パラメータで設定できるパルス入力方式には、以下の3種類があります。  
1. 1パルス入力方式  
2. 2パルス入力方式  
3. 位相差入力方式（パラメータで設定）

##### 1パルス入力方式
- PLS入力でパルスを入力し、DIR入力で回転方向を選択します。

##### 2パルス入力方式
- CW入力を入力すると正方向回転、CCW入力を入力すると負方向回転します。

##### 位相差入力方式（パラメータで設定）
- CW入力に対してCCW入力の位相が90°遅れると正方向回転、90°進むと負方向回転します。

#### 2パルス入力方式
```mermaid

    subgraph CW入力
        direction TB
        A[ON] --> B[OFF] --> C[ON] --> D[OFF]
    end
    subgraph CCW入力

    end
    CW入力 -->|5μs以上| CCW入力
    CCW入力 --> 回転方向[FWD方向 / RVS方向]
```

#### 位相差入力方式（1倍速、2倍速、4倍速）
```mermaid

    subgraph 1倍速

    end
    subgraph 2倍速
        CW2[CW入力 ON/OFF] --> CCW2[CCW入力 ON/OFF]

    end
    subgraph 4倍速
        CW4[CW入力 ON/OFF] --> CCW4[CCW入力 ON/OFF]
        CCW4 --> 回転方向4[FWD方向 / RVS方向]
    end
```

### 表

#### 位相差入力方式の波形比較

| 倍率 | CW入力波形 | CCW入力波形 | 回転方向 |
| :--- | :--------- | :---------- | :------- |
| 1倍速 | ON/OFFの波形（0.4μs以上の遅延） | ON/OFFの波形 | FWD方向 / RVS方向 |
| 2倍速 | ON/OFFの波形（0.4μs以上の遅延） | ON/OFFの波形 | FWD方向 / RVS方向 |
| 4倍速 | ON/OFFの波形（0.4μs以上の遅延） | ON/OFFの波形 | FWD方向 / RVS方向 |

---

**補足**  
- 「FWD方向」は正方向回転、「RVS方向」は逆方向回転を示します。  
- 各波形のON/OFFのタイミングと位相差により回転方向が決定されます。  
- 1パルス入力方式はDIR入力で回転方向を指定し、2パルス入力方式はCW/CCW入力のどちらかで回転方向を決定します。  
- 位相差入力方式はCW/CCWの位相差により回転方向を判定し、1倍速、2倍速、4倍速の3種類の設定が可能です。
