### テキスト
#### 3-7 シーケンス機能

■ 繰り返し運転  
2つ以上の運転データNo.を結合して繰り返し運転を行う方法が3つ紹介されています。  
- ループ機能を使用する（2～255回の範囲で繰り返し可能）  
- 拡張ループ機能を使用する（2～100,000,000回の範囲で繰り返し可能）  
- 結合機能を使用する（無限に繰り返し可能、位置のオフセット不可）

■ ループ機能  
結合した運転データNo.の運転を設定回数だけ繰り返す機能です。  
「カウント(Loop)」で設定した回数だけ繰り返し、終了後は「結合先」に設定した運転データNo.へ遷移します。

**重要**  
ループさせる運転データNo.の「結合」に「結合無」が含まれると運転が停止するため、すべての運転を「手動順送」「自動順送」または「形状接続」で結合する必要があります。

### 図
```mermaid
flowchart LR
    A[ループ開始] --> B[No.0]
    B --> C[No.1]
    C --> D[No.2]
    D --> E[ループ終了]
    E --> F[No.3]
    B -.->|ループ回数繰り返し| D
    E -->|ループ終了後、結合先の運転に遷移| F
```
**図の説明:**  
- No.0～No.2までの運転データをループ回数分繰り返す  
- ループ終了後、No.3の運転データへ遷移する
