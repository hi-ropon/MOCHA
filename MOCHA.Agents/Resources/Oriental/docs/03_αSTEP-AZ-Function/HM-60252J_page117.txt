### テキスト
#### SLIT入力やTIM信号を併用する場合
原点復帰運転が終了した後も、外部信号が検出されるまで運転を続けます。  
外部信号が検出されると、原点復帰運転が完了します。

### 表・図
| 原点検出信号 | 原点復帰運転の開始方向：＋側 | 原点復帰運転の開始方向：－側 |
| :--- | :--- | :--- |
| SLIT入力 | ![図1](図1) | ![図2](図2) |
| TIM信号またはZSG信号 | ![図3](図3) | ![図4](図4) |
| SLIT入力とTIM信号またはSLIT入力とZSG信号 | ![図5](図5) | ![図6](図6) |

#### 図の説明
- 各図はRV-LS（リミットセンサ左側）、FW-LS（リミットセンサ右側）を示す縦のグレー帯があり、その間での運転方向の変化を矢印で表しています。  
- 矢印の点線部分はリミットセンサから脱出して原点復帰戻り量だけ移動する動作を示しています。  
- SLIT入力やTIM出力（ZSG出力）のON/OFF信号のタイミングも下部に示されています。
