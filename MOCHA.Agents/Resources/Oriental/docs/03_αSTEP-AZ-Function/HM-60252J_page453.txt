### テキスト
#### アラーム発生時にモーターが無励磁になる場合
1. 異常が発生すると、ALM-B出力、MOVE出力、およびPLS-RDY出力がOFFになり、同時にモーターが停止します。  
2. アラームを解除するときはパルス入力を停止してください。パルス入力をしたまま解除すると、モーターが突然起動し、けがや装置破損の原因になります。  
3. アラームの要因を取り除いてから、ALM-RST入力をONにします。アラームが解除され、ALM-B出力、READY出力、およびPLS-RDY出力がONになります。  
4. ALM-B出力がONになったことを確認してから、ALM-RST入力をOFFにします。

### 図（タイミング詳細）
| 信号名       | ON期間・OFF期間の時間条件                         |
|--------------|--------------------------------------------------|
| アラーム要因 | 異常状態：1秒以上、異常解除後0秒以上             |
| ALM-RST入力  | ON：2ms以上、OFF：250ms以下                       |
| PLS-RDY出力  | OFF：2ms以下、ON：250ms以下                       |
| READY出力    | OFF：2ms以下、ON：250ms以下                       |
| MOVE出力     | OFF：2ms以下、ON：100ms以下                       |
| ALM-B出力    | OFF：200ms以下、ON：200ms以下                      |
| モーター励磁 | 無励磁：60ms以下、励磁：250ms以下                 |
| 電磁ブレーキ | 解放：250ms以下、保持                              |

**説明:**  
このタイミングチャートは、アラーム発生時から解除までの各信号のオンオフ状態とその時間条件を示しています。アラーム要因がONになると異常状態となり、各出力がOFFに切り替わりモーターは無励磁状態になります。アラーム解除時はALM-RST入力をONにし、各出力が順次ONに戻り、モーター励磁と電磁ブレーキの状態も復帰します。
