### テキスト
#### ETO解除無効時間
「ETO解除無効時間」パラメータの設定時間を過ぎるまでは、ETO-CLR入力をONにしてもモーターを励磁できません。

#### 関連するパラメータ
| パラメータID (Dec) | パラメータID (Hex) | 名称               | 内容                                                                                                                                                                                                                   | 初期値 |
|--------------------|--------------------|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| 408                | 0198h              | ETO解除無効時間    | HWTO1入力とHWTO2入力をONにした後に、ETO-CLR入力でモーターを励磁させる場合、ETO-CLR入力が無効になる時間を設定します。ここで設定した時間を過ぎるまでは、ETO-CLR入力をONにしてもモーターを励磁できません。<br>【設定範囲】0～100 ms | 0      |

### 図（タイミングチャート）

#### 「ETO解除無効時間」パラメータの設定時間が経過する前に、ETO-CLR入力をONにした場合（ONエッジで励磁する場合）
```mermaid
flowchart LR
    subgraph HWTO1入力、HWTO2入力
        A1(OFF) -->|ON| A2(ON)
    end
    subgraph ドライバの状態
        B1[動力遮断状態]
    end
    subgraph ETO-CLR入力
        C1(OFF) -->|ON| C2(ON)
    end
    subgraph モーター励磁
        D1[無励磁]
    end
    subgraph READY出力、PLS-RDY出力
        E1(OFF)
    end

    A2 -- 15ms以下 --> B1
    C2 -- ETO解除無効時間 --> B1
    C2 -- ETO解除無効時間内にONにしても励磁できない --> D1
```

#### 「ETO解除無効時間」パラメータの設定時間が経過してから、ETO-CLR入力をONにした場合（ONエッジで励磁する場合）
```mermaid
flowchart LR
    subgraph HWTO1入力、HWTO2入力
        A1(OFF) -->|ON| A2(ON)
    end
    subgraph ドライバの状態
        B1[動力遮断状態] -->|ETO解除無効時間| B2[動力遮断状態解除]
    end
    subgraph ETO-CLR入力
        C1(OFF) -->|ON| C2(ON)
    end
    subgraph モーター励磁
        D1[無励磁] -->|200ms以下| D2[励磁]
    end
    subgraph READY出力、PLS-RDY出力
        E1(OFF) -->|250ms以下| E2(ON)
    end

    A2 -- 15ms以下 --> B1
    B2 --> C2
    C2 --> D2
    D2 --> E2
```

**図の説明:**  
- 1つ目のタイミングチャートは、ETO解除無効時間の設定時間が経過する前にETO-CLR入力をONにしてもモーターは励磁されないことを示しています。  
- 2つ目のタイミングチャートは、ETO解除無効時間の設定時間が経過してからETO-CLR入力をONにするとモーターが励磁されることを示しています。
