### テキスト
#### AZシリーズでできる運転
モーターの運転速度や位置（移動量）などを運転データに設定して運転します。

### 図

#### 速度と時間の関係（ストアードデータ運転の3種類）

```mermaid
flowchart LR
    subgraph 位置決め運転
        A1[速度: 0 → 上昇 → 一定 → 下降 → 0]
        A2[START入力]
        A1 --> A2
    end
    subgraph 押し当て運転
        B1[速度: 0 → 一定 → 0]
        B2[押し当て開始]
        B3[押し当て終了]
        B1 --> B2 --> B3
        B4[START入力]
        B1 --> B4
    end
    subgraph 連続運転
        C1[速度: 0 → 上昇 → 一定]
        C2[START入力]
        C1 --> C2
    end
```

---

#### 回転機構の指令位置リセットイラスト
- 回転機構の指令位置を1回転ごとに「0」に戻す機構の図解（円盤と複数のセンサ）

---

#### サポートソフト MEXE02 設定画面
- 運転データの一覧表（名前、方式、位置[step]、速度[Hz]の列がある）

---

#### シーケンス機能の説明

- 結合された運転を決められた回数繰り返す。  
- ループオフセット機能により、繰り返すたびに移動量を増減可能。

```mermaid
flowchart LR
    A[絶対位置決め運転] --> B[絶対位置決め運転]
    B --> C[絶対位置決め運転]
    C --> D[位置オフセット]
    D --> A
```

---

#### ループ機能説明図

- 任意のI/O信号をトリガにして運転を遷移。  
- トリガ信号の有無で異なる運転に遷移可能。

```mermaid
flowchart TD
    Start[運転開始] -->|トリガあり| Continuous[連続運転]
    Start -->|トリガなし| Absolute[絶対位置決め運転]
    Continuous --> Sensor[センサ検出]
    Sensor -->|押し当てあり| Push[押し当て運転]
    Sensor -->|押し当てなし| Absolute
```
