### テキスト
#### 例2：「ラウンド設定範囲」が1,800 rev、分解能が1,000 P/Rの場合の座標
ラウンド設定範囲のパラメータを設定すると、図の座標でモーターを動かせるようになる。

##### 重要事項
- 「ラウンド(RND)設定」パラメータや「初期座標生成・ラウンド設定範囲」パラメータを変更すると絶対位置がずれる場合がある。変更時はプリセット(P-PRESET)または原点復帰運転を行うこと。
- 「ラウンド(RND)設定」パラメータが有効でも、「初期座標生成・ラウンド設定範囲」パラメータの設定条件を満たさない場合はラウンド設定異常のインフォメーションが発生し、電源再投入またはConfiguration実行時にアラームが発生する。

##### 「初期座標生成・ラウンド設定範囲」パラメータの設定条件
- 条件①：ラウンド設定範囲は整数であること（例：1,800）
- 条件②：ラウンド設定範囲 × 分解能 = ラウンド設定範囲 × (電子ギヤB / 電子ギヤA) × 1,000 が整数であること  
（取付角寸法が20mmと28mmのモーターは900）

##### 設定例1
- ラウンド設定範囲：100 rev  
- 分解能：1,000 P/R（電子ギヤA=1、電子ギヤB=1）  
- モーター：標準モーター（減速比1）

条件①：1,800 / 100 = 18  
条件②：100 × (1/1) × 1,000 = 100,000  
条件①、②とも整数なので設定条件を満たし、ラウンド可能。

| :---------------- | :------------------------------ | :------------- |
|                   | 初期座標生成・ラウンド座標設定 | マニュアル設定 |
|                   | ラウンド(RND)設定               | 有効           |
|                   | 初期座標生成・ラウンド設定範囲 | 1,800 rev      |
|                   | 初期座標生成・ラウンドオフセット比率設定 | 50.00          |
|                   | 初期座標生成・ラウンドオフセット値設定 | 0              |
| モーター・機構    | 電子ギヤA                       | 1              |
|                   | 電子ギヤB                       | 1              |
