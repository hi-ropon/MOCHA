### テキスト
#### 押し当て電流の設定
押し当て電流を設定すると、出力トルクを制限できます。例えば、押し当て電流を50%にすると、出力トルクも50%に制限されます。  
押し当て電流は運転データの運転電流で設定し、基本電流率を100%として、その割合で設定します。

**グラフ説明:**

- 左縦軸はトルク（%）  
- 右縦軸は電流率（%）  
- 押し当て運転のトルク値を50%に制限した場合のトルク特性を示している。

#### 関連するパラメータ
| MEXE02ツリー表示 | 名称                     | 内容                                                                                                         | 初期値 |
| :--------------- | :----------------------- | :----------------------------------------------------------------------------------------------------------- | ------ |
| 基本設定         | オートカレントダウン     | モーターが停止したときに、停止電流に切り替えるオートカレントダウン機能を設定します。<br>【設定範囲】0:無効、1:有効 | 1      |
| 基本設定         | オートカレントダウン判定時間 | モーター停止後、オートカレントダウン機能が働くまでの時間を設定します。<br>【設定範囲】0～1,000ms               | 100    |
| I/O動作・機能    | T-MODE使用時停止中電流設定 | T-MODE入力がONの状態でモーター停止時の指令電流を選択します。<br>【設定範囲】0:停止電流、1:運転電流               | 0      |

### メモ
- 押し当て運転中、停止状態でも保持力が低下しないよう、必要に応じてパラメータを設定してください。  
- オートカレントダウンを「有効」に設定すると、停止状態になったときにトルクが低下します。  
- オートカレントダウン判定時間が短いと、わずかな停止状態でもトルクが低下します。
