#### ラウンド機能とドライバ内部の32 bitカウンタの関係
ドライバ内部の32 bitカウンタは、ラウンド機能の有無にかかわらずモーターの位置情報をSTEP数で出力します。  
ラウンド機能が有効な場合、ラウンド座標と32 bitカウンタの関係は以下のようになります。

例として、モーターが同一方向へ180回転した場合のラウンド機能の動作を説明しています。  
モーターの現在位置は180回転ごとにプリセットされますが、32 bitカウンタはプリセットされません。  
32 bitカウンタの値はMEXE02のステータスモニタ画面で確認可能です。

32 bitカウンタは -2,147,483,648 から 2,147,483,647 の範囲で循環し、最大値の次は最小値が表示され、以降は昇順で表示されます。  
逆に最小値の次は最大値が表示され、以降は降順で表示されます。

### 表
| MEXE02ツリー表示 | パラメータ名                         | 設定値          |
| :---------------- | :---------------------------------- | :-------------- |
|                   | 初期座標生成・ラウンド座標設定       | マニュアル設定  |
| モーター・機構     | ラウンド（RND）設定                  | 有効            |
|                   | 初期座標生成・ラウンド設定範囲       | 18.0 rev       |
|                   | 初期座標生成・ラウンドオフセット比率設定 | 50.0 %         |
|                   | 初期座標生成・ラウンドオフセット値設定   | 0 step         |
