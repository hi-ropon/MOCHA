### テキスト
#### ストアードデータ（SD）運転
関連する入出力信号について示しています。

### グラフ
モーター動作に関連する各入出力信号のON/OFF状態を時間軸に沿って示したタイミングチャートです。

```mermaid
flowchart LR
    subgraph タイミングチャート
        direction TB
        モーター動作 -->|動作パターン| START入力
        START入力 --> M0～M7入力
        M0～M7入力 --> MOVE出力
        MOVE出力 --> READY出力
        READY出力 --> IN-POS出力
        IN-POS出力 --> SEQ-BSY出力
        SEQ-BSY出力 --> OPE-BSY出力
        OPE-BSY出力 --> DELAY-BSY出力
        DELAY-BSY出力 --> CRNT出力
        CRNT出力 --> MBC出力
        MBC出力 --> D-END0出力
        D-END0出力 --> D-END1出力
        D-END1出力 --> D-END2出力
        D-END2出力 --> M-ACT0～M-ACT7出力
        M-ACT0～M-ACT7出力 --> M-CHG出力
        M-CHG出力 --> AUTO-CD出力
    end
```

**信号一覧と動作概要**
| 信号名           | ON/OFF動作の概要                                   |
|------------------|--------------------------------------------------|
| START入力        | モーター動作開始のトリガー                         |
| M0～M7入力       | モーター制御用の入力信号                           |
| MOVE出力         | モーターの移動指示信号                             |
| READY出力        | モーターの準備完了信号                             |
| IN-POS出力       | モーターの位置到達信号                             |
| SEQ-BSY出力      | シーケンスのビジー状態                             |
| OPE-BSY出力      | 操作のビジー状態                                   |
| DELAY-BSY出力    | 遅延ビジー状態                                     |
| CRNT出力         | 現在の状態を示す信号                               |
| MBC出力          | モーター制御信号                                   |
| D-END0出力       | エンドスイッチ0の信号                             |
| D-END1出力       | エンドスイッチ1の信号                             |
| D-END2出力       | エンドスイッチ2の信号                             |
| M-ACT0～M-ACT7出力 | モーターアクチュエータの出力信号                   |
| M-CHG出力        | モーターの状態変更信号                             |
| AUTO-CD出力      | 自動制御信号                                       |

各信号のON/OFF状態はタイミングチャートで示されており、モーター動作の各段階における信号の変化が視覚的に理解できます。
</contents>
