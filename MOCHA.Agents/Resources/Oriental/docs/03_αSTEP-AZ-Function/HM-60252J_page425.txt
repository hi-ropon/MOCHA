### テキスト
#### 移動平均フィルタ
「指令フィルタ選択」パラメータで「移動平均フィルタ」を選択し、「指令フィルタ時定数」パラメータを設定します。  
これによりモーターの応答性を調整でき、位置決め運転時の残留振動を抑制し、位置決め時間を短縮可能です。  
「指令フィルタ時定数」パラメータは負荷や運転条件により最適値が異なるため、適切な値を設定してください。

#### スムースドライブ
スムースドライブ機能を使用するとモーターの振動を抑えられます。  
使用しない場合は低速域で振動が大きくなることがあるため、通常は「1:有効」に設定してください。

### 図（移動平均フィルタの動作イメージ）
```mermaid
flowchart LR
    subgraph 自起動駆動
        A1[設定速度] --> A2[モーター速度] --> A3[MOVE出力] --> A4[IN-POS出力]
    end
    subgraph 台形駆動
        B1[設定速度] --> B2[モーター速度] --> B3[MOVE出力] --> B4[IN-POS出力]
    end
    subgraph パラメータ0msの場合
        A1 --> A2 --> A3 --> A4
        B1 --> B2 --> B3 --> B4
    end
    subgraph パラメータ200msの場合
        C1[設定速度] --> C2[モーター速度] --> C3[MOVE出力] --> C4[IN-POS出力]
        C2 -. 遅延 200ms .-> C3
        C3 -. 遅延 200ms .-> C4
        D1[設定速度] --> D2[モーター速度] --> D3[MOVE出力] --> D4[IN-POS出力]
        D2 -. 遅延 200ms .-> D3
        D3 -. 遅延 200ms .-> D4
    end
```
**図の説明**  
- 左側は「指令フィルタ時定数」パラメータが0msの場合の速度・出力の変化  
- 右側は200msの場合で、モーター速度やMOVE出力、IN-POS出力に200msの遅延が生じている様子を示す  
- 自起動駆動と台形駆動の2種類の動作パターンを比較
