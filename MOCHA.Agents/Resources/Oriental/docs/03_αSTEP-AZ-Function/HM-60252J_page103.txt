### テキスト
#### 加減速の共通設定と独立設定
運転データ拡張用設定の「使用レート選択」で、ストアードデータ運転と連続マクロ運転における加減速を設定可能。
- 共通設定：共通起動・変速レートと共通停止レートパラメータの設定値に従う。
- 独立設定：運転データNo.に設定された加減速に従う。

#### 停止動作
##### 運転停止入力
モーターの動作中に運転停止信号を入力するとモーターが停止する。

### 表
#### 関連する運転データ拡張用設定

| MEXE02ツリー表示 | 名称               | 内容                                                                                                   | 初期値    |
|------------------|--------------------|------------------------------------------------------------------------------------------------------|-----------|
| 運転データ拡張用設定 | 使用レート選択       | 共通加減速または運転データの加減速のどちらを使用するか設定。<br>【設定範囲】0:共通レート使用(共通設定)、1:各運転データのレート使用(独立設定) | 1         |
|                  | 共通起動・変速レート | 共通設定における起動・変速レート、または起動・変速時間を設定。<br>【設定範囲】1～1,000,000,000(1=0.001 kHz/s, 1=0.001 s, または1=0.001 ms/kHz) | 1,000,000 |
|                  | 共通停止レート       | 共通設定における停止レート、または停止時間を設定。<br>【設定範囲】1～1,000,000,000(1=0.001 kHz/s, 1=0.001 s, または1=0.001 ms/kHz) | 1,000,000 |

> **memo**  
> パルス列運転のときは、ここで設定したパラメータは無効になる。

|------------------|----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| I/O動作・機能    | STOP・STOP-COFF入力停止方法 | STOP入力またはSTOP-COFF入力がONになったときのモーター停止方法を設定。<br>【設定範囲】<br>0:STOP入力、STOP-COFF入力ともに即停止<br>1:STOP入力は減速停止、STOP-COFF入力は即停止<br>2:STOP入力は即停止、STOP-COFF入力は減速停止<br>3:STOP入力、STOP-COFF入力ともに減速停止 | 3      |

> **memo**  
> パルス列運転のときは、常に即停止するため、ここで設定したパラメータは無効になる。
