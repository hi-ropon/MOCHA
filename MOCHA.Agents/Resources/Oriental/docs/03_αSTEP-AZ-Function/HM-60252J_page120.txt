### テキスト
#### 原点復帰運転
● 押し当て方式  
運転速度で原点復帰開始方向へ運転して、メカ端に設置したストッパなどに押し当たるとモーターが反転します。その後、押し当て原点復帰初回戻り量を移動して停止し、再びストッパに向かって原点検出速度で運転します。もう一度押し当てると反転し、押し当て原点復帰戻り量を移動して停止します。

##### 記号の説明
- VR: 原点復帰運転速度  
- VS: 原点復帰起動速度  
- VL: 原点検出速度  
- ・・・: 原点オフセットを設定した場合の軌跡

### 表・図
| 原点復帰運転の開始位置 | 原点復帰運転の開始方向：＋側 | 原点復帰運転の開始方向：－側 |
| :--------------------- | :--------------------------- | :--------------------------- |
| メカ端の間             | RVS側メカ端 → FWD側メカ端   | RVS側メカ端 → FWD側メカ端   |
|                        | +VR, +VS, +VL, -VL, -VS, -VR の速度変化を示す軌跡図 | +VR, +VS, +VL, -VL, -VS, -VR の速度変化を示す軌跡図 |

- ※1 メカ端から「(HOME)押し当て原点復帰初回戻り量」だけ移動します。  
- ※2 メカ端から「(HOME)押し当て原点復帰戻り量」だけ移動します。

### 図の説明
- 左図（＋側開始方向）と右図（－側開始方向）で、速度の変化と移動方向の軌跡を示している。
- 実線と破線で押し当てと戻りの動作を表現。
- 矢印はモーターの回転方向と速度変化を示す。

```mermaid
flowchart LR
    Start[原点復帰開始方向へ運転] --> Push[メカ端に押し当て]
    Push --> Reverse1[モーター反転]
    Reverse1 --> MoveBack1[初回戻り量だけ移動 ※1]
    MoveBack1 --> Stop1[停止]
    Stop1 --> Push2[再びストッパに向かって運転]
    Push2 --> Reverse2[モーター反転]
    Reverse2 --> MoveBack2[戻り量だけ移動 ※2]
    MoveBack2 --> Stop2[停止]
```

**図の説明:**  
- 運転開始から押し当て、反転、戻り、停止までの一連の動作フローを示す。
- ※1、※2はそれぞれ初回戻り量と戻り量の移動を指す。
