### テキスト
#### 出力信号の概要

■ ダイレクト出力  
ダイレクト出力（DOUT）は、I/Oケーブルをコネクタに配線して信号を直接出力する方法。  
コンポジット出力機能を使うと、2つの出力信号の論理結合結果を1つの信号で出力可能。

MEXE02の設定例：HOME-END出力とAREA0出力がONになったら、HOME-END（DOUT0）を出力する。  
（通常出力機能に「HOME-END」、コンポジット論理結合に「AND」、コンポジット出力機能に「AREA0」を設定すると、原点復帰完了（HOME-END）と指定位置到達（AREA0）を1つの出力信号（DOUT0）で確認可能）

■ ユーザー出力  
ユーザー出力（USR-OUT）は内部I/Oを使って信号を出力する方法。  
1つのユーザー出力に2種類の信号（AとB）を割り付け、AとBの論理結合が成立したらUSR-OUTが出力される。  
配線不要でダイレクトI/Oと併用可能。ユーザー出力は2つまで設定可能。

MEXE02の設定例：IN-POS出力とREADY出力がONになったら、USR-OUTを出力する。

### 表

#### ダイレクト出力設定項目

| 名称                     | 説明                                                                                     |
|--------------------------|------------------------------------------------------------------------------------------|
| (通常)出力機能           | DOUTに割り付ける出力信号を選択します。                                                  |
| 接点設定（信号反転）     | 接点の変更が行えます。                                                                   |
| OFF出力遅延時間          | 設定した時間を超えると、出力信号がOFFになります。ノイズ対策や機器間のタイミング合わせに使用。 |
| コンポジット論理結合     | コンポジット出力機能の論理結合（ANDまたはOR）を設定します。                             |
| コンポジット出力機能     | DOUTの信号と論理演算を行う出力信号を選択。2つの信号の論理結合が成立するとDOUTがONになる。 |
| コンポジット接点設定（信号反転） | コンポジット出力機能で選択した信号の接点を変更します。                                  |

#### ユーザー出力設定項目

| 名称                         | 説明                                                                                   |
|------------------------------|----------------------------------------------------------------------------------------|
| ユーザー出力源A-機能          | 出力機能Aを選択します。                                                                |
| ユーザー出力源A-接点設定（信号反転） | 出力機能Aの接点を変更します。                                                          |
| ユーザー出力源B-機能          | 出力機能Bを選択します。                                                                |
| ユーザー出力源B-接点設定（信号反転） | 出力機能Bの接点を変更します。                                                          |
| ユーザー出力論理結合選択      | 出力機能源AとBの論理結合（ANDまたはOR）を設定します。                                  |
