### テキスト
#### MEXE02で原点を確定する手順
1. ショートカットボタンの「ティーチング・リモート運転」、またはツールバーの「ティーチング・リモート運転」アイコンをクリックする。
2. 「ティーチング・リモート運転を開始する」をクリックする。
3. JOG運転ボタンを使ってモーターを原点位置まで運転する。  
   ドライバステータス欄の「指令位置（CPOS）」を確認しながら位置を調整する。  
   ※手動で原点位置を調整する際はFREE操作ボタンを使用する。  
   「FREE:ON」をクリックするとモーターが無励磁となり、外力でモーター出力軸を動かせるようになる。

### 表
#### JOG運転ボタンの内容

| ボタン画像 | 割り付けられている入力信号名 | 入力信号とパラメータの関係 |
| :---: | :--- | :--- |
| ◀◀ | RV-JOG-H | 「JOG運転速度（高）」パラメータに設定した運転速度で連続運転する。 |
| ◀ | RV-JOG | 「JOG運転速度」パラメータに設定した運転速度で連続運転する。 |
| － | RV-JOG-P | JOG運転ボタンの「最小移動量」に設定した移動量で位置決め運転する。 |
| ■ | STOP | モーターが即停止する。 |
| ＋ | FW-JOG-P | JOG運転ボタンの「最小移動量」に設定した移動量で位置決め運転する。 |
| ▶ | FW-JOG | 「JOG運転速度」パラメータに設定した運転速度で連続運転する。 |
| ▶▶ | FW-JOG-H | 「JOG運転速度（高）」パラメータに設定した運転速度で連続運転する。 |
