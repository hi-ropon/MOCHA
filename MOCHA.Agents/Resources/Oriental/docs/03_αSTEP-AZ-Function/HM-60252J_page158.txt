### テキスト
#### 1-1 入力信号の概要

■ ダイレクト入力  
ダイレクト入力（DIN）とは、I/Oケーブルをコネクタに配線して信号を直接入力する方法。  
コンポジット入力機能を使うと、1つの入力で2つの信号を同時にONにでき、省配線を実現可能。

■ 仮想入力  
仮想入力（VIR-IN）とは、仮想入力源に設定した信号の出力を使用して、仮想入力で設定した信号を入力する方法。  
内部のI/Oを使った入力方法のため配線不要で、ダイレクトI/Oと併用可能。仮想入力は最大4つまで設定可能。

### 表

#### ダイレクト入力の設定項目

| 名称               | 説明                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 入力機能           | DINに割り付ける入力信号を選択します。                                                 |
| 接点設定（信号反転） | 接点の変更が行えません。                                                               |
| ON信号検出不感時間 | 設定した時間を超えると、入力信号がONになります。ノイズ対策や機器間のタイミング合わせに使用。 |
| 強制1shot          | ONになった入力信号を、250μs後に自動でOFFにします。                                   |
| コンポジット入力機能 | DINがONになったら、ここで選択した信号も同時にONになります。                         |

#### ダイレクト入力の設定例

| DIN0（PULSE-I/タイプ除く） | 入力機能 | 接点設定（信号反転） | ON信号検出不感時間 [ms] | 強制1shot | コンポジット入力機能 |
|----------------------------|----------|----------------------|-------------------------|-----------|----------------------|
|                            | FW-POS   | 反転しない           | 0                       | 無効      | M0                   |

#### 仮想入力の設定項目

| 名称                     | 説明                                                                                   |
|--------------------------|----------------------------------------------------------------------------------------|
| 仮想入力機能             | VIR-INに割り付ける信号を選択。仮想入力源の信号が出力されたらVIR-INもONになる。          |
| 仮想入力源選択           | VIR-INのトリガにする出力信号を選択。                                                   |
| 仮想入力接点設定（信号反転） | 接点の変更が行えません。                                                               |
| 仮想入力ON信号検出不感時間 | 設定した時間を超えると、入力信号がONになる。ノイズ対策や機器間のタイミング合わせに使用。 |
| 仮想入力強制1shot        | ONになった入力信号を、250μs後に自動でOFFにします。                                   |

|-------------------------|---------------------------|-----------------------------------------|--------------------------------------------|------------------------------|
