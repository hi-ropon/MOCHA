### テキスト
#### STEP 2 NETC02-CCのスイッチを設定します
NETC02-CCの上面にあるスイッチを、以下のように設定してください。設定すると、図のようになります。

### 表
| 設定内容                     | スイッチ               | 出荷時設定          |
|----------------------------|-----------------------|---------------------|
| RS-485通信接続台数：2       | N-AXISを「2」          | 1                   |
| CC-Link伝送ボーレート：10 Mbps | B-RATEを「4」          | 0                   |
| CC-Link局番：1              | STATION No.の×1を「1」、×10を「0」 | 1（×1：1、×10：0）  |
| 動作モード：OFF             | No.1～No.4をすべて「OFF」 | すべてOFF           |

### 図
```mermaid
flowchart LR
    A[RS-485通信接続台数設定スイッチ (N-AXIS)] --> B[CC-Link伝送ボーレート設定スイッチ (B-RATE)]
    B --> C[CC-Link局番設定スイッチ (STATION No.)]
    C --> D[動作モード設定スイッチ (SW2)]
```

**図の説明:**
- 左上に「RS-485通信接続台数設定スイッチ (N-AXIS)」
- その下に「CC-Link伝送ボーレート設定スイッチ (B-RATE)」
- さらに下に「CC-Link局番設定スイッチ (STATION No.)」
- 左側に「動作モード設定スイッチ (SW2)」が配置されているNETC02-CCの上面スイッチ配置図
