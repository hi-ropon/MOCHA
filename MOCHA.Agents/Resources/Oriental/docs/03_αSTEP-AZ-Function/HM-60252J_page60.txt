### テキスト
#### 3-2 データの設定
ストアードデータ運転に関する設定は以下の3種類がある。
- 運転データ  
  運転方式、目標位置、運転速度、加減速レート、運転電流などを設定する。
- 運転I/Oイベント  
  イベントジャンプ機能に必要なイベント発生条件や、イベント発生時の運転の結合先・結合方法を設定する。イベントジャンプ機能使用時に利用。
- 運転データ拡張用設定  
  拡張ループ機能に必要なループ開始位置や終了位置、ループ回数などを設定。256回以上のループ運転時に使用。

運転データは最大256点（No.0～255）まで設定可能。

| 位置       | 目標位置（移動量）を設定。連続SD運転では使用しない。<br>【設定範囲】<br>-2,147,483,648～2,147,483,647 step                                                                                                                                                                                                                                           | 0          |
| 速度       | 運転速度を設定。位置決め運転と押し当て運転は絶対値の速度で運転。正の値でFWD方向、負の値でRVS方向回転。<br>【設定範囲】<br>-4,000,000～4,000,000 Hz                                                                                                                                                                                             | 1,000      |

| 停止レート   | 停止時の減速レートまたは減速時間を設定。<br>【設定範囲】<br>1～1,000,000,000 (1=0.001 kHz/s, 1=0.001 s, または1=0.001 ms/kHz)                                                                                                                                                                                                                         | 1,000,000  |
