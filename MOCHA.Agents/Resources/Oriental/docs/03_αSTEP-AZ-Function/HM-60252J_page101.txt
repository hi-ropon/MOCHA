### テキスト
#### 運転データ拡張用設定
運転データの仕様を拡張する機能について説明しています。

##### ■拡張ループ機能
拡張ループ機能は、運転データで設定できない回数（256回以上）のループ運転を実行するための機能です。  
耐久試験のように単純な繰り返し運転を行う際に利用します。  
「繰り返し開始運転番号」から「繰り返し終了運転番号」までの範囲で、設定した回数だけ運転データを繰り返します。  
繰り返し回数が終了すると、「結合先」に設定した運転データ番号へ遷移します。  
拡張ループ機能を使う場合、繰り返し開始運転番号から繰り返し終了運転番号の運転データは固定値となります。

### 表
#### 拡張ループ機能で使用する運転データ項目

| MEXE02ツリー表示 | 名称               | 固定値                                                                                   |
| :---------------- | :----------------- | :--------------------------------------------------------------------------------------- |
| 運転データ        | 結合先             | ↓ (+1)                                                                                  |
|                   | オフセット(エリア) | 0                                                                                       |
|                   | 幅(エリア)         | -1                                                                                      |
|                   | カウント(Loop)     | 繰り返し開始運転番号：繰り返し回数<br>その他：-                                         |
|                   | 位置オフセット(Loop)| 0                                                                                       |
|                   | 終了(Loop)         | 繰り返し終了運転番号：End<br>その他：-                                                 |
|                   | 弱イベント         | -                                                                                       |
|                   | 強イベント         | -                                                                                       |

#### 関連する運転データ

| MEXE02ツリー表示 | 名称   | 内容                                                                                         | 初期値 |
| :---------------- | :----- | :------------------------------------------------------------------------------------------- | :----- |
| 運転データ        | 結合   | 結合方法を設定します。<br>【設定範囲】<br>0：結合無<br>1：手動順送<br>2：自動順送<br>3：形状接続 | 0      |
|                   | 結合先 | 結合先を設定します。<br>【設定範囲】<br>-256：Stop<br>-2：↓↓ (+2)<br>-1：↓ (+1)<br>0～255：運転データNo. | -1     |

### 重要事項
ループさせる運転データNo.の「結合」に「結合無」が含まれていると、設定した運転データNo.で運転が停止します。  
必ずすべての運転を「手動順送」「自動順送」または「形状接続」で結合してください。
