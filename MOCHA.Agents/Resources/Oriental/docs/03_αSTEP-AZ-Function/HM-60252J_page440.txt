### テキスト
#### 波形モニタを活用する
ドライバやMEXE02のバージョンによって、以下の項目もモニタ可能です。
- パルス波形（ドライバVer.4.00以降で可能）
  - 外部から入力されているパルスの波形をモニタできる。パルス入力されているダイレクトI/OをCH5～CH12の中から選択可能。
- 過負荷タイマ（MEXE02 Ver.3.33以降で可能）
  - 過負荷状態の継続時間をカウントする。過負荷タイマはアナログ出力のためCH1～CH4の中から選択可能。
  - 過負荷状態になるとTLC出力がONとなり、同時にタイマがカウント開始。
  - カウント値が「過負荷アラーム」の設定値に達するとアラームが発生。
  - 過負荷状態が解除されるとTLC出力はOFFになるが、タイマは0に戻らず、解除された時間をカウント値から減算していく。
  - これにより、過負荷状態が再度発生した場合、カウント値が途中から加算されるため、設定時間より早くアラームが発生することがある。
  - 過負荷アラームの原因特定に役立つ。

### 画像説明
波形モニタ画面のキャプチャがあり、以下の説明が付されています。
- 過負荷タイマは過負荷時間を積算し、過負荷状態が解除されると減算する。
- TLC出力は過負荷になるとON、解除されるとOFFになる。

### グラフ（タイミングチャート）
```mermaid
flowchart TD
    A[TLC出力] -->|ON| B[過負荷タイマ積算開始]
    B --> C{過負荷タイマ積算値}
    C -->|設定値超過| D[アラーム発生]
    A -->|OFF| E[過負荷状態解除]
    E --> F[過負荷タイマ減算開始]

    subgraph モーター負荷率
        G[0%] --> H[100%]
    end

    subgraph ALM-A出力
        I[OFF] --> J[ON]
    end
```

**グラフ説明:**
- TLC出力がONになると過負荷タイマが積算を開始し、設定値を超えるとアラームが発生。
- 過負荷状態解除時はタイマが減算される。
- モーター負荷率は0%から100%の間で変動し、過負荷状態に影響。
- ALM-A出力はアラーム発生時にONになる。
