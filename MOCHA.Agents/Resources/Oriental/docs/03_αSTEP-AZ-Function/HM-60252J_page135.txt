### テキスト
#### 速度制御押し当て運転
- FW-PSH入力またはRV-PSH入力がONの間、モーターは選択されている運転データNo.の運転速度で連続運転する。  
- 運転中に運転データNo.を変更すると変速する。  
- FW-PSH入力またはRV-PSH入力をOFFにするとモーターは減速停止する。減速中に同じ回転方向の信号をONにすると再加速して運転を続ける。  
- FW-PSH入力とRV-PSH入力が両方ONになるとモーターは減速停止する。

### 図（運転イメージ）
```mermaid
flowchart TD
    subgraph 位置偏差
        A1[位置偏差増大] --> A2[一定値で固定]
        A2 --> A3[押し当て状態で位置偏差増大]
        A3 --> A4[一定トルクで加圧継続]
    end

    subgraph TLC出力
        B1[信号ON/OFF]
    end

    subgraph モーター動作
        C1[速度制御押し当て運転開始(FW-PSH ON)]
        C2[減速停止開始(入力OFF)]
        C3[速度制御押し当て運転開始(RV-PSH ON)]
    end

    subgraph 入力信号
        D1[FW-PSH入力]
        D2[M0入力]
        D3[RV-PSH入力]
    end

    D1 --> C1
    D3 --> C3
    C1 --> C2
    C3 --> C2
```
- FW-PSH入力ONでFWD方向へ速度制御押し当て運転開始。  
- RV-PSH入力ONでRVS方向へ速度制御押し当て運転開始。  
- それぞれの入力をOFFにすると減速停止が始まる。  
- 位置偏差は一定値まで増大すると固定され、押し当て状態になるとさらに増大し、一定トルクで加圧し続ける。
