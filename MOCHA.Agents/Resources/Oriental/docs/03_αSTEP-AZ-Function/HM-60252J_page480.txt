### テキスト
#### 4-4 タイミングチャート
1. PLS-RDY出力がONになっていることを確認します。  
2. CWパルスを入力します。モーターがCW方向へ運転を始めます。  
3. 運転が終わると、READY出力がONになります。

### 図
```mermaid
flowchart LR
    subgraph 信号波形
        direction TB
        A[PLS-RDY出力] ---|ON| B[ON状態維持]
        C[CW入力] ---|パルス入力| D[パルス継続] ---|パルス入力| E[パルス継続]
        F[READY出力] ---|OFF| G[ONに変化]
    end
    subgraph 内部速度指令
        H[速度指令開始] --> I[速度指令維持] --> J[速度指令終了]
    end
    A --> C
    C --> F
    F --> H
```

**図の説明:**  
- PLS-RDY出力がON状態を維持している間にCW入力のパルスが入力される。  
- CW入力のパルスが続く間、モーターはCW方向に運転を開始。  
- 運転終了後、READY出力がONに変わる。  
- 内部速度指令はパルス入力に合わせて上昇し、一定時間維持された後、下降する。
