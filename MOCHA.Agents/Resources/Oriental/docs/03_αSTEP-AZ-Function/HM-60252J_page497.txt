### テキスト
#### 4-1 ドライバシミュレーションモードの準備と操作手順

■準備  
●モーターを接続しない場合  
**重要**  
電動アクチュエータのシミュレーションを行うときは、必ずアクチュエータをドライバに接続してください。（498ページ参照）

### 図
#### AC電源入力のときの接続例
```mermaid
flowchart LR
    制御電源 -->|+24V、0Vに接続| ドライバ
    ドライバ -->|USB通信コネクタに接続| USBケーブル --> MEXE02(パソコン)
    ドライバ -->|ネットワークコンバータ<br>RS-485通信で制御するときに接続してください。| ネットワークコンバータ
    ドライバ -->|CN5に接続| 入出力信号 --> 上位システム
    ドライバ --> 接地
```

#### DC電源入力のときの接続例
```mermaid
flowchart LR
    主電源 --> ドライバ
    ドライバ -->|USB通信コネクタに接続| USBケーブル --> MEXE02(パソコン)
    ドライバ -->|ネットワークコンバータ<br>RS-485通信で制御するときに接続してください。| ネットワークコンバータ
    ドライバ -->|CN4に接続| 入出力信号 --> 上位システム
    ドライバ --> 接地
```

**図の説明**  
- AC電源入力時は制御電源の+24V、0Vをドライバに接続し、USB通信コネクタをパソコンに接続する。ネットワークコンバータはRS-485通信制御時に使用。入出力信号はCN5に接続し上位システムと連携。  
- DC電源入力時は主電源をドライバに接続し、USB通信コネクタをパソコンに接続。ネットワークコンバータは同様にRS-485通信制御時に使用。入出力信号はCN4に接続し上位システムと連携。  
- いずれの場合もドライバは接地されている。
