### テキスト
#### 製品情報を確認する
MEXE02にはユニット情報モニタが用意されており、このモニタ機能を使うことで製品名、機番、スイッチの設定状況などの製品情報を確認できる。またパラメータの設定値も確認可能。

#### ユニット情報モニタ画面の見方
ユニット情報モニタ画面は複数のエリアに分かれており、それぞれのエリアで主なモニタ項目と説明がある。

---

### 表
#### ユニット情報モニタ画面のエリア別主なモニタ項目と説明

| エリア | 主なモニタ項目 | 説明 |
| :---: | :--- | :--- |
| 1 製品情報エリア | ユーザー名称 | パラメータで任意の名称を付けることができる。 |
|  | 製品名称 | MEXE02に接続されている製品名が表示される。 |
|  | 機番 | 製品ごとに割り当てられているシリアルナンバー。工場出荷時に書き込まれており変更不可。 |
| 2 ドライバ情報エリア | 制御電源投入回数 | 制御電源を投入した回数（制御電源を接続しないドライバは主電源投入回数を表示）。 |
|  | 主電源投入回数 | AC電源ドライバ：突入抑制リレーがONになった回数。DC電源ドライバ：モーター接続状態で主電源投入回数。 |
|  | 主電源通電時間 | 主電源が通電された合計時間。 |

---

### 画像（ユニット情報モニタ画面の構成）
- 画面は3つの番号付きエリアに分かれている。
  1. 製品情報エリア（ユーザー名称、製品名称、機番など）
  2. ドライバ情報エリア（CPU情報、通信情報、分解能など）
  3. 機構情報設定エリア（電子ギヤ、モーター回転方向、機構リード、初期原点生成設定、機構リミットパラメータなど詳細設定項目）
