### テキスト
#### 4-3 モニタ
シミュレーション中にステータスモニタで確認できる内容について説明しています。  
表示される内容が通常時と異なる項目についても解説しています。

### 画像説明
ステータスモニタ画面のスクリーンショットに3つの注釈枠があり、それぞれ以下の情報を示しています。

1. モーター座標情報  
2. モーターステータス情報  
3. 計算値情報

### 表
| 番号 | 項目名             | 説明                                                                                   |
| :--- | :----------------- | :------------------------------------------------------------------------------------- |
| ①   | モーター座標情報   | ABZOセンサで検出している座標情報を表示。モーターの接続/未接続に関わらず、座標情報は指令に追従する。 |
| ②   | モーターステータス情報 | ABZOセンサの情報を表示。モーターの接続/未接続に関わらず、シミュレーション中も常に更新される。          |
| ③   | 計算値情報         | ドライバの指令情報とモーターの検出情報から算出された値を表示。モーターの接続/未接続に関わらず、シミュレーション中は以下の状態になる。<br>・積算負荷、トルク：不定<br>・位置偏差、モーター負荷率：常に「0」を表示 |
