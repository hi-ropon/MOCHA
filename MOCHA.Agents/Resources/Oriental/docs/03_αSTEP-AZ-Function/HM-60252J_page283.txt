### テキスト
Modbus RTUモードによるデータの設定例として、START信号をONにするクエリ送信から位置決め運転開始、終了までの手順が説明されています。

1. STARTをONにするためのクエリ送信
2. 位置決め運転開始
3. 位置決め運転完了後、STARTをOFFに戻すためのクエリ送信

STARTは初期設定でドライバ入力指令(007Dh)のbit3に割り付けられており、2進数で1,000=16進数で0008hに相当します。

|----------------------|---------|---------------------------------|

| データ               |         |                                 |

| - レジスタアドレス(下位) | 7Dh     |                                 |
| - ライト値(上位)     | 00h     | レジスタアドレスに書き込む値=START ON(0008h) ※       |
| - ライト値(下位)     | 08h     |                                 |
| エラーチェック(下位) | 18h     | CRC-16の計算結果                |
| エラーチェック(上位) | 14h     |                                 |
