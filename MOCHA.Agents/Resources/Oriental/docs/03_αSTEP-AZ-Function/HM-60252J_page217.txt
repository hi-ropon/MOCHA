### テキスト
#### 動力遮断機能
動力遮断機能とは、モーターへの電力供給をハードウェアで遮断する機能です。  
2チャンネルの入力（HWTO1入力、HWTO2入力）によって、モーター電流を制御するインバータ回路の駆動信号を遮断します。  
これにより、モーターへの電力供給が遮断された状態（動力遮断状態）になります。  
動力遮断機能は、装置可動部の動作範囲内で作業しなければならない場合に、可動部の予期しない起動を防止する目的で使用されます。

**重要**  
必ずモーターが停止していることを確認してから、動力遮断機能を実行してください。  
運転中に動力遮断機能を実行すると、モーター、ドライバ、装置が破損するおそれがあります。

### 図
```mermaid
flowchart LR
    subgraph 入力信号
        HWTO1+ -->|光カプラ| HWTO1
        HWTO2+ -->|光カプラ| HWTO2
        EDM+ -->|トランジスタ| EDM-
    end

    HWTO1 --> 遮断1[遮断]
    HWTO2 --> 遮断2[遮断]

    遮断1 --> 上アーム駆動用電源
    遮断2 --> 下アーム駆動用電源

    上アーム駆動用電源 -->|駆動信号| インバータ回路
    下アーム駆動用電源 -->|駆動信号| インバータ回路

    インバータ回路 --> モーター

    制御回路 --> インバータ回路
```

**図の説明**  
- HWTO1入力をOFFにすると、インバータ回路の上アーム駆動信号が遮断される。  
- HWTO2入力をOFFにすると、インバータ回路の下アーム駆動信号が遮断される。  
- EDMは外部監視信号として接続されている。
