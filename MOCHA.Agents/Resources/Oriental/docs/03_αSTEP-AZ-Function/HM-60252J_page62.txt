### テキスト
#### ストアードデータ(SD)運転
- 位置、速度、起動・変速レート、停止レート、運転電流、運転終了遅延  
  ストアードデータ運転に必要な目標位置、運転速度、加減速レート（加減速時間）、運転電流を設定します。

##### 運転パターン
- 位置決め運転  
  起動速度から運転速度へ加速し、目標位置で停止。停止後に運転終了遅延がある。  
- 連続運転  
  起動速度から運転速度へ加速し、その速度を維持。停止はしない。  
- 起動速度＜運転速度の場合  
  起動速度から運転速度へ加速し、目標位置で停止。  
- 運転速度≦起動速度の場合  
  運転速度で一定の位置移動を行い、停止。

##### memo
押し当て運転でトルク制限を行う場合は、運転データの「運転電流」で設定。  
励磁最大静止トルクを100%として設定し、例えばトルク値を50%に制限したい場合は運転電流を50%に設定する。

#### 結合・結合先
- 結合無  
  1つの運転データNo.で運転を1回実行（単独運転）。  
- 手動順送  
  SSTART入力を入力するたびに、「結合先」に設定した運転データNo.の運転を実行。SSTART入力はREADY出力がONのときに有効。  
- 自動順送  
  「運転終了遅延」に設定した時間だけ停止後、「結合先」に設定した運転データNo.の運転が自動で開始。  
- 形状接続  
  モーターを停止せずに、「結合先」に設定した運転データNo.の運転を続けて実行。

#### 運転パターンのタイムチャート
```mermaid
flowchart TD
    subgraph 位置決め運転
        A1[起動速度] --> A2[運転速度]
        A2 --> A3[停止]
        A3 --> A4[運転終了遅延]
        A1 -->|時間| A2
        A2 -->|時間| A3
        A3 -->|時間| A4
    end

    subgraph 連続運転
        B1[起動速度] --> B2[運転速度]
        B1 -->|時間| B2
    end

    subgraph 起動速度＜運転速度の場合
        C1[起動速度] --> C2[運転速度]
        C2 --> C3[停止]
        C1 -->|時間| C2
        C2 -->|時間| C3
    end

    subgraph 運転速度≦起動速度の場合
        D1[運転速度一定]
        D1 -->|時間| D1
    end
```
