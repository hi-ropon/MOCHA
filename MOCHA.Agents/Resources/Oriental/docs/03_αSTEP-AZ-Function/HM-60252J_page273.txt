<overview>
このページは、Modbus RTU制御（RS-485通信）に関する設定手順を説明しています。構成は以下の通りです。
- テキスト部分：STEPごとの操作説明（スイッチ設定、通信パラメータ設定、電源再投入）
- 表部分：スイッチ設定内容の一覧、通信パラメータの詳細
- 図部分：AC電源ドライバとDC電源ドライバのスイッチ配置図と機能説明
</overview>

<contents>
### テキスト
#### STEP 2 スイッチを設定します
ドライバのスイッチを指定の通りに設定します。設定後の状態は図の通りです。

#### STEP 3 電源を投入し、通信パラメータを設定します
MEXE02で通信パラメータを確認し、通信が確立しない場合はドライバの通信パラメータを見直します。

#### STEP 4 電源を再投入します
ドライバのスイッチや通信パラメータは電源の再投入後に有効になります。

### 表
#### スイッチ設定内容
| 設定内容           | スイッチ設定内容                          |
| :----------------- | :-------------------------------------- |
| プロトコル         | Modbusプロトコル                         |
| 号機番号           | 1                                      |
| 通信速度           | 115,200 bps                            |
| 終端抵抗           | ON                                     |
| スイッチ設定       | SW1のNo.2をON、SW1のNo.1をOFF、IDを1    |
| 通信速度設定       | BAUDを4                                |
| 終端抵抗設定       | AC電源ドライバ：TERMのNo.1とNo.2をON   |
|                    | DC電源ドライバ：SW1のNo.3とNo.4をON    |

#### 通信パラメータ（MEXE02ツリー表示）
| 通信・I/F機能       | パラメータ名                                      |
| :------------------ | :----------------------------------------------- |
|                     | 通信パリティ【初期値：1（偶数パリティ）】         |
|                     | 通信ストップビット【初期値：0（1ビット）】         |
|                     | 送信待ち時間【初期値：30（3.0 ms）】               |
|                     | サイレントインターバル【初期値：0（自動）】        |

### 図
#### AC電源ドライバのスイッチ配置
```mermaid
flowchart LR
    A[号機設定スイッチ(ID)] --> B[通信速度設定スイッチ(BAUD)]
    B --> C[機能設定スイッチ(SW1)]
    C --> D[No.2: プロトコル]
    C --> E[No.1: 号機番号]
    A --> F[終端抵抗設定スイッチ(TERM.)]
```

#### DC電源ドライバのスイッチ配置
```mermaid
flowchart LR
    G[機能設定スイッチ(SW1)] --> H[No.3, No.4: 終端抵抗]
    G --> I[No.2: プロトコル]
    G --> J[No.1: 号機番号]
    K[号機設定スイッチ(ID)] --> L[通信速度設定スイッチ(BAUD)]
```

**図の説明:**
- AC電源ドライバは号機設定スイッチ、通信速度設定スイッチ、機能設定スイッチ、終端抵抗設定スイッチで構成される。
- DC電源ドライバは機能設定スイッチ（終端抵抗、プロトコル、号機番号）、号機設定スイッチ、通信速度設定スイッチで構成される。
