### テキスト
#### スイッチの設定
- 図は出荷時設定の状態を示す。  
- 【重要】スイッチを設定する際は必ずドライバの電源を切ること。電源投入状態での設定は無効。

#### AC電源ドライバ
- 位置決め機能内蔵タイプとRS-485通信付きパルス列入力タイプの2種類がある。  
- 各種スイッチの名称と位置が図示されている。  
  - 号機設定スイッチ（ID）  
  - 通信速度設定スイッチ（BAUD）  
  - 機能設定スイッチ（SW1）  
  - 終端抵抗設定スイッチ（TERM.）

#### DC電源ドライバ
- 位置決め機能内蔵タイプとRS-485通信付きパルス列入力タイプの2種類がある。  
- 各種スイッチの名称と位置が図示されている。  
  - 機能設定スイッチ（SW1）  
  - 号機設定スイッチ（ID）  
  - 通信速度設定スイッチ（BAUD）

### 表
#### 出荷時設定
| 種類 | 設定状態 |
| :--- | :--- |
| 位置決め機能内蔵タイプ | OFF |
| RS-485通信付きパルス列入力タイプ | ON |

#### SW1-No.2 プロトコル設定
| SW1-No.2 | プロトコル内容 |
| :--- | :--- |
| ON | Modbus RTUプロトコル |
| OFF | ネットワークコンバータに接続 |
