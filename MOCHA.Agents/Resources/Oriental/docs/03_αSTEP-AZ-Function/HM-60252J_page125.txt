### テキスト
#### 高速JOG運転
高速JOG運転は、FW-JOG-H入力またはRV-JOG-H入力がONの間、モーターが高速で一方向へ連続運転を行います。入力信号をOFFにすると減速停止し、運転停止信号を入力しても運転を停止できます。

### 図
```mermaid
flowchart TD
    FW[FW-JOG-H入力 ON] -->|ON| M[モーター高速FWD方向運転開始]
    RV[RV-JOG-H入力 ON] -->|ON| M2[モーター高速RVS方向運転開始]
    FW -->|OFF| D1[減速停止開始]
    RV -->|OFF| D2[減速停止開始]
    D1 --> S[モーター停止]
    D2 --> S
```
**図の説明:**
- FW-JOG-H入力がONになると、モーターはFWD方向へ高速JOG運転を開始する。
- RV-JOG-H入力がONになると、モーターはRVS方向へ高速JOG運転を開始する。
- それぞれの入力がOFFになると減速停止が始まり、最終的にモーターは停止する。
