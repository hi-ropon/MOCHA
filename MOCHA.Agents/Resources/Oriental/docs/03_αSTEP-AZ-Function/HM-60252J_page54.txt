### テキスト
#### 位置決め運転に必要な設定のながれ
- ドライバの取扱説明書にてモーター・ドライバの設置、配線、原点確定を行う。
- 第3編でI/Oの割り付け、入出力条件、現在値の出力、省配線に役立つ機能などを紹介。
- 座標と分解能の設定方法やラウンド機能については140ページ、電子ギヤ利用時の分解能設定は55ページを参照。
- 運転方法を選択し、データを設定する。
- 運転方法には「ストアードデータ運転＋シーケンス機能」「マクロ運転」「高速原点復帰・原点復帰運転」がある。
- 第4編でパラメータ設定を行う。
- 第9編でインフォメーションやアラームに関する設定を行う。
- 最後に設定の終了。

### 図
```mermaid
flowchart TD
    A[ドライバの取扱説明書]
    A --> B[第3編 I/Oを割り付けます]
    B --> C[座標と分解能を設定します]
    C --> D[運転方法を選択し、データを設定します]
    D --> E[ストアードデータ運転＋シーケンス機能]
    D --> F[マクロ運転]
    D --> G[高速原点復帰・原点復帰運転]
    E --> H[運転データを設定して運転。ジャンプやループなど簡単なシーケンス機能も実行可能]
    F --> I[JOG運転や連続運転など特定信号入力で運転]
    G --> J[原点に復帰します]
    D --> K[第4編 パラメータを設定します]
    K --> L[第9編 インフォメーションやアラームに関する設定を行います]
    L --> M[設定の終了]
```

**図の説明:**
- ドライバの取扱説明書から始まり、第3編でI/O割り付け、第4編でパラメータ設定、第9編でインフォメーションやアラーム設定を行い、最終的に設定を終了する流れを示す。
- 運転方法はストアードデータ運転＋シーケンス機能、マクロ運転、高速原点復帰・原点復帰運転の3種類がある。
