# Keyence（キーエンス）機器マニュアル インデックス

このファイルはKeyence社製機器のマニュアルインデックスです。
Keyence Document Readerとkeyence_agentが各マニュアルページを検索・読み込むために使用されます。

## マニュアル構成概要

Keyence機器シリーズは産業用自動化・安全制御・検査機器に特化した制御機器群です。各製品は高精度な制御・監視・安全機能を提供し、製造業の安全性向上と生産性向上に貢献します。

## 機器別マニュアル詳細

### 01_GC-1000（セーフティコントローラ）
**フォルダ:** `01_GC-1000/`  
**ファイル数:** 380ページの詳細マニュアル  
**対象ページ:** 1-380  
**マニュアル型番:** GC-1000-SC-UM-001

**主な内容:**
- **取扱い上の注意**: 機能安全規格（IEC 61508 SIL3、ISO13849-1 Cat.4 PL e、UL1998）対応、設置環境条件
- **システム構成**: GC-1000メインコントローラ、拡張ユニット、リモートI/Oモジュール、専用ソフトウェア
- **安全制御機能**: セーフティ入力16点、セーフティ出力6点、AUX出力4点、テスト出力4点
- **通信機能**: EtherNet/IP、PROFINET、Modbus/TCP、MCプロトコル、UDP、5つの通信プロトコル対応
- **GC-Link機能**: KEYENCE安全機器との簡単配線、専用クイックコネクタケーブル対応
- **プログラミング機能**: Category 4安全回路構築、プログラムフリー設定、ドラッグ&ドロップ画面
- **表示・監視**: 1.77インチカラーLCD、設備停止原因表示、イベント履歴、タイミングチャート
- **拡張性**: 最大10台拡張ユニット、最大4台リモートI/Oモジュール対応
- **応答性能**: 世界初一定応答時間、超高速5.5ms応答
- **診断・メンテナンス**: セーフティデバイス状態監視、エラー・アラート表示、過去イベント履歴
- **トラブルシューティング**: 故障診断、安全回路異常対応、復旧手順、配線チェック

**検索キーワード:** GC-1000、セーフティコントローラ、安全制御、SIL3、Category4、PLe、GC-Link、安全回路、セーフティI/O、プログラミング、EtherNet/IP、PROFINET、配線、設定、トラブル
