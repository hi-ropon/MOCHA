# Mitsubishi（三菱電機）機器マニュアル インデックス

このファイルはMitsubishi Electric（三菱電機）社製機器のマニュアルインデックスです。
Mitsubishi Document Readerとmitsubishi_agentが各マニュアルページを検索・読み込むために使用されます。

## マニュアル構成概要

Mitsubishi Electric機器シリーズは産業用オートメーション・制御機器に特化した制御機器群です。各製品は高精度な位置制御・速度制御・トルク制御機能を提供し、製造業の自動化と生産性向上に貢献します。
