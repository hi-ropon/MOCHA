### テキスト
#### 機能
(d)で指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトします。シフト後、(s)から(n2)点分を空きデータ部分に格納します。
- (s)に定数を指定した場合、シフト後の最下位から(n2)点分は指定した値になります。
- (n2)で指定した値が00の場合は無処理となります。

#### 例
(n1)=4、(n2)=2の場合の動作を示しています。

#### エラー
| エラーコード(SD0) | 内容 |
|------------------|------|
| 2821H            | (s)から(n2)点分の範囲と、(d)から(n1)点分の範囲が重複しているとき。 |
| 3405H            | (n1)、(n2)に指定した値が(n1) < (n2)のとき。 |

### 図
データシフトの動作を示す図解です。
- 上段に元のデータ範囲(d)から(d)+15までの倍精度実数が並んでいます。
- (n1)=4点分のデータ範囲が示され、その中で左へ(n2)=2点分シフトが行われます。
- シフト後、元のデータのうち左端の2点分が削除され、右端に(s)からの2点分のデータ（例では9.87654321と-9.87654321）がコピーされます。
- 図中の矢印で①が左へ2点分シフト、②がコピー動作を表しています。
