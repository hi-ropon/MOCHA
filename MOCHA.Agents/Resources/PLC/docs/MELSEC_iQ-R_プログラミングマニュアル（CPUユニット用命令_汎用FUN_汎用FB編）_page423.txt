### テキスト
#### 機能
- (s)で指定された単精度実数を符号付きBIN32ビットデータに変換し、(d)で指定されたデバイスに格納する。
- 変換後のデータは、(s)で指定された単精度実数の小数点以下1桁目を四捨五入した値となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### プログラム例の説明
- M0がONしたとき、D0～D1に格納された単精度実数を符号付きBIN32ビットデータに変換し、D100～D101へ格納する。

### 図
- ラダー図：SM402の接点、M0の接点、EMOVP命令（D0にE-123456.7を設定）、FLT2DINT命令（D0の値をD100に変換格納）が示されている。
- ビット構成図：
  - (s)側（単精度実数）D1, D0のビット構成例（例：-123456.7 → C7F1H 205AH）
  - (d)側（符号付きBIN32ビットデータ）D101, D100のビット構成例（例：-123457 → FFFEH 1DBFH）
