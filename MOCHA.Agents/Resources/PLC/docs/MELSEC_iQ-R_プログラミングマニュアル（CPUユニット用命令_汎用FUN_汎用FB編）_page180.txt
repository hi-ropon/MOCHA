### テキスト
#### デバイスのリセット(アナンシエータを除く)
RST命令は指定したデバイスを0（OFF）にします。タイマやカウンタについては現在値を0にし、接点やコイルをOFFにします。

#### 実行条件
RST命令の実行条件はパルス信号（立ち上がり）です。

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。
- Fを使用する場合は別の命令RST Fを参照する必要があります。

### 図
- **ラダー図**：RST命令のラダー回路例が示されており、(d)のビットデバイスをリセットする形を表現。
- **ST（構造化テキスト）**：`ENO:=RST(EN,d);`という命令文でRST命令の使用例を示す。
- **FBD/LD**：ファンクションブロック図またはラダー図で、EN（実行条件）入力、ENO（実行結果）出力、d（リセット対象デバイス）を持つブロックの形状を示す。
