### 表
#### 命令処理時間一覧表
本表は各命令名ごとに、指定された条件下での処理時間（マイクロ秒単位）をCPUユニット別に示しています。

- **命令名**: LDDT<=、LDDT<、LDDT>=、ANDDT=、ANDDT<>、ANDDT>、ANDDT<=、ANDDT<、ANDDT>=、ORDT= など複数の命令が列挙されています。
- **条件**: 「指定の日付と比較:導通時」「指定の日付と比較:非導通時」「現在の日付と比較:導通時」「現在の日付と比較:非導通時」「無実行時」など、命令の実行条件が記載されています。
- **処理時間(μs)**: 
  - R00CPU、R01CPU、R02CPU、左記以外のCPUユニットに分かれており、それぞれ最小値と最大値が示されています。
  - 例えば、LDDT<=命令の「指定の日付と比較:導通時」条件では、R00CPU・R01CPUの処理時間は最小4.200μs、最大11.400μs、R02CPUは最小3.200μs、最大10.400μs、その他CPUユニットは最小2.700μs、最大9.900μsとなっています。
- **特徴**:
  - 無実行時の処理時間は非常に短く（例:0.18816μsなど）、実行時は条件により処理時間が変動します。
  - 処理時間はCPUユニットによって異なり、R02CPUやその他CPUユニットの処理時間が比較的短い傾向があります。
  - 条件によって処理時間の幅が大きく変わる命令もあります。

この表は命令の性能評価や最適化、CPU負荷の見積もりに利用されることが想定されます。
</contents>
