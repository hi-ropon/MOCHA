### テキスト
#### 文字列挿入 STRINS(P)
任意の文字列データを、文字列データの任意挿入位置に挿入する命令です。

#### 実行条件
- STRINS命令は立下りエッジで実行されます。
- STRINSP命令は立上りエッジで実行されます。

### 図
- **ラダー図**  
  挿入対象の文字列(s1)、挿入文字列(d)、挿入位置(s2)を入力し、結果を出力する構造を示しています。  
- **FBD/LD図**  
  EN（実行条件）、s1、s2を入力し、dを出力するブロック図が示されています。

### 表
#### 実行条件
| 命令   | 実行条件       |
|--------|----------------|
| STRINS | 立下りエッジ    |
| STRINSP| 立上りエッジ    |

| オペランド | 内容                                   | 範囲       | データ型 | データ型(ラベル)       |
|------------|--------------------------------------|------------|----------|------------------------|
| (s1)       | 挿入文字列または、挿入文字列が格納されている先頭デバイス | —          | 文字列   | ANYSTRING_SINGLE       |
| (d)        | 挿入文字列を格納する先頭デバイス     | —          | 文字列   | ANYSTRING_SINGLE       |
| (s2)       | 挿入位置（バイト単位）                | 1〜16383   | 符号付きBIN16ビット | ANY16                  |
| EN         | 実行条件                             | —          | ビット   | BOOL                   |
| ENO        | 実行結果                             | —          | ビット   | BOOL                   |
