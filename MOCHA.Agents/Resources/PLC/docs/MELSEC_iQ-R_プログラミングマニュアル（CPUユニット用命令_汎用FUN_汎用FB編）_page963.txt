### テキスト
#### BCD8桁平方根命令 BDSQRT(P)
BCD値（8桁）で指定した値の平方根を演算します。

#### 機能
(s)で指定したBCD8桁データの平方根を計算し、(d)で指定したデバイスに演算結果を格納します。

- (s)で指定する値はBCD値で最大8桁（0～99999999）です。
- (d)の演算結果はBCD値で0～9999.9999を格納します。
- 演算結果は小数部5桁目を切り捨てた値となります。

平方根計算の数式は以下の通りです。

\[
\sqrt{(s)+1 (s)} = (d) (d)+1
\]

ここで、(s)+1と(s)は2ワードデータ、(d)は整数部、(d)+1は小数部を示します。

### 図
- ラダー図：BDSQRT命令のラダー表現。EN（実行条件）、s（入力データ）、d（出力データ）を示す。
- FBD/LD図：BDSQRT命令のファンクションブロック図。EN（入力）、ENO（出力）、s（入力データ）、d（出力データ）を示す。
- 実行条件のタイミング図：BDSQRTは立下りエッジで実行、BDSQRTPは立上りエッジで実行。
