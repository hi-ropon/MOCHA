### テキスト
#### 機能
・(d)で指定したワードデバイスの(n)ビット目をリセット（0）します。  
・(n)が15を超えた場合は、下位4ビットのデータで実行されます。

#### エラー
演算エラーはありません。

### 図
・ワードデバイスD10のビット列の例が示されています。  
・上段はリセット前のビット状態で、b15からb0までのビットが1または0で表現されています。  
・下段はリセット後の状態で、D10の11ビット目（b11）が0にリセットされていることが矢印で示されています。  
・この図は「D10の11ビット目をリセット(0)する」動作を視覚的に説明しています。
