### テキスト
#### BIN32ビットデータ平均値算出
「DMEAN(P)(_U)」命令は、指定したデバイスからn点分（32ビットデータ単位）の平均値を算出します。

#### 実行条件
- DMEAN、DMEAN_U命令は立ち下がりエッジで実行
- DMEANP、DMEANP_U命令は立ち上がりエッジで実行

### 図
#### ラダー図
- ラダー図では、(s)、(d)、(n)の3つのオペランドを持つ命令ブロックが示されています。
- ST言語では、ENO:=DMEAN(EN,s,n,d);やENO:=DMEAN_U(EN,s,n,d);のように記述されます。

#### FBD/LD図
- FBD/LD図では、EN（入力）、ENO（出力）、s、d、nの各端子がある命令ブロックが描かれています。

### 表

| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
| (s) | 平均値を算出するデータが格納されている先頭デバイス | — | 符号付きBIN32ビット / 符号なしBIN32ビット | ANY32_S / ANY32_U |
| (d) | 平均値を格納する先頭デバイス | — | 符号付きBIN32ビット / 符号なしBIN32ビット | ANY32_S / ANY32_U |
| (n) | データ数またはデータ数が格納されたデバイス番号 | 0〜65535 | 符号なしBIN16ビット | ANY16 |
| EN | 実行条件 | — | ビット | BOOL |
| ENO | 実行結果 | — | ビット | BOOL |

※ラベルで設定する場合は、動作に必要な領域が確保できるよう配列を定義し、その配列型ラベルの要素を指定する必要があります。
