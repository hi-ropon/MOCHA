### テキスト
#### 機能
OUT命令は、演算結果を指定されたデバイスへ出力します。

#### 条件
- ビットデバイス使用時
- ワードデバイスのビット指定時

#### エラー
演算エラーはありません。

| ビットデバイス使用時  | OFF      | OFF               |
|                      | ON       | ON                |
| ワードデバイスのビット指定時 | OFF      | 0                 |
|                      | ON       | 1                 |
