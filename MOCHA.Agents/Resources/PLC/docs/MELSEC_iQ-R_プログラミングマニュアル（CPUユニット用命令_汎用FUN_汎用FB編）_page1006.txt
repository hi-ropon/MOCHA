### テキスト
#### 機能
- 実行指令がONしている時間を秒単位で測定し、(s)で指定された乗数を掛けた値を(d)で指定されたデバイスに格納します。
- 実行指令がOFFからONに変わったとき、(d)+0、(d)+1で指定されたデバイスをクリアします。
- (s)で指定できる乗数は0～2までで、それ以外の値は無処理となります。

#### Point（注意点）
- TTMR命令実行時に時間を計測するため、JMP命令などで飛ばさないこと。飛ばすと正確な測定ができなくなります。
- TTMR命令実行中は(s)で指定した乗数を変更しないこと。変更すると正確な値になりません。
- (d)+1で指定されたデバイスはCPUユニットのシステムで使用するため、値を変更しないこと。変更すると正確な値になりません。

### 表
| (s) | 乗数 |
|-----|------|
| 0   | 1    |
| 1   | 10   |
| 2   | 100  |

### エラー
- 演算エラーはありません。
