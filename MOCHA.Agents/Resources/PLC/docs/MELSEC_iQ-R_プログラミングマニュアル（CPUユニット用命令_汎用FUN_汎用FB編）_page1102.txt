### テキスト
#### 機能
(s1)で指定された時刻データから、(s2)で指定された時間データを減算し、その結果を(d)で指定されたデバイス番号以降に格納する機能について説明しています。

#### 計算例
- 例1：10時40分20秒875から3時50分10秒500を減算した場合の計算結果は6時50分10秒375となります。
- 例2：演算結果が負の数になった場合は、そのデータに24を加えた値が演算結果となります。例えば、4時50分32秒875から10時42分12秒500を減算した場合、-6時8分20秒375ではなく18時8分20秒375となります。

#### Point
- (s1)+3、(s2)+3、(d)+3のデバイスは演算には使用しません。
- S(P).DATERD命令で読み出した時計データは、そのまま減算可能です。
- S(P).DATERD命令で読み出した場合、「秒」と「1/1000秒」の間に「曜日」が入りますが、S(P).DATE命令では演算しないため、そのまま減算できます。
