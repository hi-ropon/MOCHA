### テキスト
#### 機能
- (s)の下位(n)ビットで指定されたBIN値に対応した、(d)のビット位置をONにします。
- (n)は1～8が指定可能。
- (n)=0のときは無処理となり、(d)で指定されたデバイスの内容は変化しません。
- ビットデバイスは1ビット、ワードデバイスは16ビットとして扱われます。

##### 図の説明
- (s)の下位3ビット「110」（2進数で6）に対応して、(d)のビット位置6がONになる様子を示しています。
- (d)のビット位置は0～7まであり、ビット6がONになっていることを矢印で示しています。

### 表
#### エラー
| エラーコード (SD0) | 内容               |
|--------------------|--------------------|
| 3401H              | (n)が0～8以外のとき。|

この表は、指定された(n)の値が範囲外の場合に発生するエラーコードとその内容を示しています。
