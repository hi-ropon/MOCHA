### テキスト・表
#### 二安定ファンクションブロック
- ■セット優先
  - ファンクションブロック記号: SR, SR_E
  - 処理内容: 2つの入力値を判別し、1(TRUE)または0(FALSE)を出力する。
  - 参照先: 1496ページ SR(_E)

- ■リセット優先
  - ファンクションブロック記号: RS, RS_E
  - 処理内容: 2つの入力値を判別し、1(TRUE)または0(FALSE)を出力する。
  - 参照先: 1498ページ RS(_E)

#### エッジ検出ファンクションブロック
- ■立ち上りエッジ検出
  - ファンクションブロック記号: R_TRIG, R_TRIG_E
  - 処理内容: 信号の立ち上がりを検出してパルス信号を出力する。
  - 参照先: 1500ページ R_TRIG(_E)

- ■立ち下りエッジ検出
  - ファンクションブロック記号: F_TRIG, F_TRIG_E
  - 処理内容: 信号の立ち下がりを検出してパルス信号を出力する。
  - 参照先: 1502ページ F_TRIG(_E)

#### カウンタファンクションブロック
- ■アップカウンタ
  - ファンクションブロック記号: CTU, CTU_E
  - 処理内容: 信号の立ち上がり回数をカウントアップする。
  - 参照先: 1504ページ CTU(_E)

- ■ダウンカウンタ
  - ファンクションブロック記号: CTD, CTD_E
  - 処理内容: 信号の立ち上がり回数をカウントダウンする。
  - 参照先: 1506ページ CTD(_E)

- ■アップダウンカウンタ
  - ファンクションブロック記号: CTUD, CTUD_E
  - 処理内容: 信号の立ち上がり回数をカウントアップ/カウントダウンする。
  - 参照先: 1508ページ CTUD(_E)
