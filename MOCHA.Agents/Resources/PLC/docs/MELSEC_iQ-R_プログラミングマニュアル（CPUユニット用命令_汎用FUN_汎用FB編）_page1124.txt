### テキスト
#### DTO(P)命令の概要
- (s2)で指定したデバイスから(n)×2点分のデータを、(U/H)で指定したユニットまたは自号機CPUユニット内のバッファメモリの(s1)で指定したアドレス以降に書き込みます。
- (s2)に定数を指定した場合は、同一データ((s2)で指定した値)を指定したバッファメモリアドレスから(n)×2ワード分書き込みます。
- 命令実行時に対象ユニットの異常や処理中などでアクセスできない場合は無処理となります。

   - CPUユニットの(s2)で指定した範囲の(n)×2点分のデータが、ユニットバッファメモリの(s1)で指定したアドレス以降に書き込まれる様子を示す。
   - 矢印でCPUユニットからユニットバッファメモリへデータが移動している。

2. **自号機CPUユニットへのダブルワードデータライト（デバイス/ラベルメモリ→自号機CPUバッファメモリ）**
