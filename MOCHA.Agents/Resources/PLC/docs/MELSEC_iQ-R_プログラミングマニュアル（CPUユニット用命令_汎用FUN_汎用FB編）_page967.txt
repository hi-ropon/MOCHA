### テキスト
#### 倍精度実数べき乗演算 POWD(P)
倍精度実数のべき乗を演算する命令です。

#### 実行条件
- POWD命令は立ち上がりエッジで実行されます。
- POWDP命令は立ち下がりエッジで実行されます。

#### 設定データ
##### 内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (s1) | 被べき乗データ、または被べき乗データを格納している先頭デバイス | 0, 2^-1022 ≤ |s1| < 2^1024 | 倍精度実数 | ANYREAL_64 |
| (s2) | べき乗データまたは、データが格納されている先頭デバイス | 0, 2^-1022 ≤ |s2| < 2^1024 | 倍精度実数 | ANYREAL_64 |
| (d) | 演算結果を格納する先頭デバイス | — | 倍精度実数 | ANYREAL_64 |
| EN | 実行条件 | — | ビット | BOOL |
| ENO | 実行結果 | — | ビット | BOOL |

### 図
- ラダー図：POWD命令のラダー回路例。入力s1、s2、出力dを示す。
- ST言語例：`ENO:=POWD(EN,s1,s2,d);`および`ENO:=POWDP(EN,s1,s2,d);`
- FBD/LD：命令ブロック図。入力EN、s1、s2、出力ENO、dを示す。

| (s2) | 対象外 | T, ST, C, D, W, SD, SW, FD, R, ZR, RD | 対象外 | Z, LT, LST, LC | ○ | 対象外 |
