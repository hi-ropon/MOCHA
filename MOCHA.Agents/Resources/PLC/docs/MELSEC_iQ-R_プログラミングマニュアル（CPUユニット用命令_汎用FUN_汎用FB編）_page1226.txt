### テキスト
#### 26 PID制御命令
PID制御を行うための命令には「PID演算命令」と「PID制御命令」の2種類があり、それぞれの使い分けについては1211ページを参照するよう案内されています。

#### 26.1 概要
PID制御命令における演算方式、PID制御手順、その他の機能について説明しています。

**Point**
- PID制御命令には不完全微分と完全微分の2種類がある。
- 不完全微分は微分項の入力に一次遅れフィルタを入れたPID制御で、以下のような場合に有効。
  - 高周波ノイズの影響を受けやすい制御
  - 完全微分方式でステップ状変化があったときに操作端を動作させるだけの有効なエネルギーが与えられない場合
- 完全微分は微分項の入力をそのまま使用するPID制御。

### 表
| 演算方式       | 内容                                                                                     |
| -------------- | ---------------------------------------------------------------------------------------- |
| 速度形演算     | PID演算で操作量(MV)の変化分を計算する方式。実際の操作量はサンプリング周期ごとに計算した操作量の変化分が累積された値となる。 |
| 測定値微分形   | PID演算で微分項に測定値(PV)を使用して演算する方式。微分項に偏差を使用していないため、設定値変更による偏差の変化時に微分動作による出力の急変を軽減できる。 |
