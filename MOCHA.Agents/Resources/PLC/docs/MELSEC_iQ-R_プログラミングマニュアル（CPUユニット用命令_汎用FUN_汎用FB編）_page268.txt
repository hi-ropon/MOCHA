### テキスト
#### DB-(P) [オペランドが3個の場合]
指定した2つのBCD8桁データを減算する命令について説明しています。

- ラダー言語では、(s1)、(s2)の2つのBCD8桁データを減算し、結果を(d)に格納します。
- ST言語では、`ENO:=DBMINUS(EN,s1,s2,d);` または `ENO:=DBMINUSP(EN,s1,s2,d);` と記述します。
- FBD/LDでは、EN（実行条件）、s1、s2（減算対象データ）、d（結果格納先）を入力とし、ENO（実行結果）を出力とするブロック図が示されています。

#### 実行条件
- 命令DB-は立ち下がりエッジで実行されます。
- 命令DB-Pは立ち上がりエッジで実行されます。
