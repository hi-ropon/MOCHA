### テキスト
#### BOOL型→STRING型変換（BOOL_TO_STRING(_E)）
BOOL型データをSTRING型データへ変換する関数です。

##### 対応CPU
RnCPU、RnENCPU、RnPCPU（プロセス、三重化、一般）、RnPSFCPU（安全、一般）

##### 機能概要
- 入力されたBOOL型データ(s)をSTRING型データ(d)に変換して出力します。
- 入力値がFALSEの場合はSTRING型のデータ値で0を出力。
- 入力値がTRUEの場合はSTRING型のデータ値で1を出力。

##### 演算処理
- (s)への入力値はBOOL型のデータ値です。
- EN（実行条件）がTRUEの場合に演算を実行し、FALSEの場合は停止します。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の表の通りです。

##### エラー
演算エラーはありません。

### 図
- ラダー回路、FBD/LD、ST言語での関数使用例の図があり、EN/ENO付き・なしの違いを示しています。
- BOOL型（FALSE/TRUE）からSTRING型（0/1）への変換イメージ図があり、視覚的に変換内容を説明しています。

### 表
#### 設定データ（引数の内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | BOOL |
| ENO | 出力状態(TRUE:正常実行、FALSE:異常または停止) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | STRING |

#### 実行条件と演算結果
| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE（演算実行） | TRUE | 演算出力値（0または1） |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、(d)のデータを使用しないようプログラム処理が必要。
