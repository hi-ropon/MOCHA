### テキスト
- (214) ループ2に対するPID制御用データを設定します。
- (386) D500～D521に設定されているPID制御用データを設定します。
- (389)～(400) R60DA4を出力許可に設定します。

- MOV命令は定数値を指定されたデバイスに代入する命令で、PID制御用の各種パラメータ設定に使用されています。
- PIDINIT命令はPID制御の初期化を行い、D500番地に対して実行されます。
- SETおよびRST命令はY0A9出力のセット・リセットを行い、制御のON/OFFを切り替えています。
- 条件付きで命令が実行される部分もあり、SM402やX0A0などの条件ビットが指定されています。

以上より、このページはPID制御ループ2に関する制御データの設定と出力許可の制御を行うプログラム例の一部であることがわかります。
