### テキスト
#### 機能
(s)で指定された符号なしBIN32ビットデータを、符号付きBIN16ビットデータに変換し、(d)で指定されたデバイスに格納します。  
プログラム例では、M0がONしたときに、D0～D1に格納された符号なしBIN32ビットデータを符号付きBIN16ビットデータに変換し、D100へ格納する動作を示しています。

#### エラー
演算エラーはありません。

### 図
プログラム例の図は以下の内容を示しています。

- SM402の接点が閉じるとM0がONになる。
- M0がONのとき、UDINT2INT命令が実行される。
- D0～D1に格納された符号なしBIN32ビットデータ（例：5678H 9ABCH）が符号付きBIN16ビットデータに変換される。
- 変換後のデータ（例：9ABCH、-25924）がD100に格納される。

図中のビット表記は、(s)が符号なし32ビットデータ、(d)が符号付き16ビットデータを示し、具体的なビット位置（b31～b0、b15～b0）と16進数表記が対応しています。

(1)は下位16ビットへの格納を示しています。
