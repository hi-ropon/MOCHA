### テキスト
#### 機能
- (s)で指定された角度のTAN（正接）値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- 演算結果の角度はラジアン単位で格納される。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### エラー
- エラーコード「3402H」について説明。
- 内容は「(s)に指定した値が-0、非正規化数、非数、±∞のとき」に発生する。

#### Point
- ラジアンと角度変換については、RADD(P)命令、DEGD(P)命令を参照すること。
- 参照ページは945ページ（RADD(P)）、947ページ（DEGD(P)）。

### 図
- TANの逆関数（TAN⁻¹）を使い、(s)の4つの連続した倍精度実数から角度を計算し、(d)の4つの連続した倍精度実数に結果を格納するイメージ図がある。
