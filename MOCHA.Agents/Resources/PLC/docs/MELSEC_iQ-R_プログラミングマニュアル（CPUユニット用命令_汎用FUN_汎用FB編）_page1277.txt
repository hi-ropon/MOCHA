### テキスト
#### BMOV(P)命令の概要
BMOV(P)命令は、指定したブロックのステップ活性状態を、指定ステップから指定ワード分一括で読み出す命令です。  
対応CPUは、RnCPU、RnENCPU、プロセスCPU（二重化含む）などで、バージョンによる制約が記載されています。  
- R00CPU、R01CPU、R02CPUは制約なし。  
- RnENCPUはファームウェアバージョン12以降で使用可能。  
- プロセスCPU（プロセスモード）はバージョン03以降。  
- プロセスCPU（二重化モード）はバージョン18以降。  

### 図
#### ラダー図
- ラダー図では、(s)(d)(n)の3つのオペランドを持つ命令ブロックが示されており、ST言語では  
  ENO:=BMOV(EN,s,n,d);  
  ENO:=BMOVP(EN,s,n,d);  
  と表記されます。

#### FBD/LD図
- FBD/LD図では、EN、s、nを入力、ENO、dを出力とする命令ブロックの構造が示されています。
