### テキスト
#### 機能
(s1)で指定した単精度実数データと、(s2)で指定した単精度実数データを比較し、その結果（小、一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONします。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s1)、(s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |

### 図
ラダー図で以下の条件が示されています。  
- (d)がONになる条件: [(s1), (s1)+1] > [(s2), (s2)+1] のときON  
- (d)+1がONになる条件: [(s1), (s1)+1] = [(s2), (s2)+1] のときON  
- (d)+2がONになる条件: [(s1), (s1)+1] < [(s2), (s2)+1] のときON  

この図はX0入力がONのときにECMP命令が実行され、比較結果に応じて(d)、(d)+1、(d)+2のいずれかがONになる動作を示しています。
