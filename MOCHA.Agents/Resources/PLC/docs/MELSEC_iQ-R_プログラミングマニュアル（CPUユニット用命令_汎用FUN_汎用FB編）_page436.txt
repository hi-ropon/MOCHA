### テキスト
#### 符号付きBIN16ビットデータ→符号付きBIN32ビットデータ変換
命令名：INT2DINT(P)

指定した符号付きBIN16ビットデータを符号付きBIN32ビットデータに変換する命令です。

対応ツールバージョン：バージョン"1.035M"以降のエンジニアリングツールで対応しています。

#### ■実行条件
- INT2DINT命令は立ち上がりエッジで実行
- INT2DINTP命令は立ち上がりエッジで実行（詳細は図参照）

### 図
#### ラダー図
- ラダー図では、(s)に入力される符号付きBIN16ビットデータを(d)に符号付きBIN32ビットデータとして出力する命令ブロックが示されています。
- ST命令の例として、ENO:=INT2DINT(EN,s,d); ENO:=INT2DINTP(EN,s,d); の記述があります。

#### FBD/LD図
- FBD/LD図では、EN（実行条件）、s（入力）、d（出力）、ENO（実行結果）端子を持つ命令ブロックが示されています。
