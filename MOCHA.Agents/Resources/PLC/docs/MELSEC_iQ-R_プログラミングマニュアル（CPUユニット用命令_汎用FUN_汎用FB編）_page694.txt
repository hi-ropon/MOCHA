### テキスト
#### BIN32ビットデータ不感帯制御 DBAND(P)(_U)
指定した入力値（BIN32ビット値）が不感帯の上下限範囲内か否かにより、出力値を制御する命令について説明しています。

#### 実行条件
命令ごとに実行条件の信号パターンが示されています。
- DBAND、DBAND_Uはパルス状の実行条件
- DBANDP、DBANDP_Uは立ち上がりエッジの実行条件

### 図
#### ラダー図
- 入力s1, s2, s3と出力dの関係を示すラダー図が掲載されています。
- EN（実行条件）、ENO（実行結果）も含まれています。

- ブロック図形式でEN（入力実行条件）、s1, s2, s3（入力値）、d（出力値）、ENO（出力実行結果）の接続が示されています。
