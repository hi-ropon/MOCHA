### テキスト
#### インデックスレジスタ命令
- インデックスレジスタの一括退避、復帰
  - ZPUSH、ZPUSHP：インデックスレジスタの内容を指定領域に退避
  - ZPOP、ZPOPP：退避されているデータをインデックスレジスタに読み出し
- インデックスレジスタ/ロングインデックスレジスタの選択退避、復帰
  - ZPUSH、ZPUSHP：指定したインデックスレジスタ、ロングインデックスレジスタの内容を指定領域に退避
  - ZPOP、ZPOPP：退避されているデータをインデックスレジスタ、ロングインデックスレジスタに読み出し

#### ファイルレジスタ操作命令
- ファイルレジスタのブロックNo.切換え
  - RSET、RSETP：プログラム中で使用するファイルレジスタのブロックNo.を指定されたデバイスに格納されているブロックNo.に変更
- ファイルレジスタ用ファイルのセット
  - QDRSET、QDRSETP：プログラム中で使用するファイルレジスタのファイル名を変更
- ファイルレジスタの1バイト単位のリード/ライト命令
  - 1バイトデータリード（ZRRDB、ZRRDBP）：指定した通しバイトNo.のファイルレジスタ内容を読み出す
  - 1バイトデータライト（ZRWRB、ZRWRBP）：指定したデバイスの下位ビットの内容を通しバイトNo.のファイルレジスタに書き込む

### 図
- ファイルレジスタの1バイトデータリード命令のビット構成図
  - 0,2ビットが下位8ビット、1,3ビットが上位8ビットとしてZR0、ZR1に対応
  - (s)で指定された8ビットを(d)に読み出すイメージ
- ファイルレジスタの1バイトデータライト命令のビット構成図
  - 指定デバイスの下位ビットを通しバイトNo.のファイルレジスタに書き込む
  - (s1)、(s2)のビット指定があり、ZR0、ZR1の上下位8ビットに対応
