### テキスト
#### 機能
- (d)で指定したデバイスから、(n)点分の単精度実数を左へ1点シフトする。
- 最下位から1単精度実数分は0になる。
- (n)で指定された値が0の場合は無処理となる。

#### エラー
- 演算エラーは発生しない。

### 図
#### 例：(n)=6の場合のデータシフト
- 上段に元のデータが6点分並んでいる。
- 下段は左へ6点分シフトした結果を示す。
- シフトにより、元の最下位の6点分は0に置き換わっている。
- 具体的な数値例として、1.2345、123.45678、-1.234567、-12.3456、-123.456789、12.345が左へ6点シフトされ、123.45678、-1.234567、-12.3456、-123.456789、12.345、0となっている。
