### テキスト
#### DOR(P) [オペランドが3個の場合]
指定した2つのBIN32ビットデータの論理和を行う命令です。

#### ■実行条件
- DOR命令は立ち下がりエッジで実行
- DORP命令は立ち上がりエッジで実行

#### ■注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定の安全デバイスおよび安全ラベルのみ使用可能

### 図
- ラダー図：3つのオペランド(s1, s2, d)を用いた論理和回路を示す
- ST言語：ENO:=DOR(EN,s1,s2,d); ENO:=DORP(EN,s1,s2,d);
- FBD/LD：EN, ENO, s1, s2, dの入出力ポートを持つ命令ブロック図
