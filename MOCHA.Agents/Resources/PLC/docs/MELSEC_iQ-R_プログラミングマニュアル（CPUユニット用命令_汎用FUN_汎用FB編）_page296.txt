### テキスト
#### 6.3 論理演算命令 16ビットデータ論理積
WAND(P)命令は、指定した2つのBIN16ビットデータの論理積を行う命令です。対応CPUはRnCPU、RnENCPU（プロセス、二重化、一般、安全）です。

- ラダー回路での表現は簡単な論理積の形で示されています。
- ST言語やFBD/LD言語には対応していません（3オペランドの場合は別ページ参照）。

#### 実行条件
- WAND命令は立ち上がりパルスで実行
- WANDP命令は立ち下がりパルスで実行

### 表
#### 設定データ：内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
| (s) | 論理積を行うデータまたはデータ格納デバイス | -32768～32767 | 符号付きBIN16ビット | ANY16 |
| (d) | 論理積結果を格納するデバイス | -32768～32767 | 符号付きBIN16ビット | ANY16 |

- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、上記の安全デバイスおよび安全ラベルのみ使用可能。
