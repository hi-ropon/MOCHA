### テキスト
#### BINデータ→BCD8桁変換
命令「DBCD(P)」は、指定したBIN（2進）32ビットデータをBCD（8桁）データに変換します。

#### 実行条件
命令「DBCD」は立下りエッジで実行され、「DBCDP」は立上りエッジで実行されます。

### 図
#### ラダー図
ラダー図では、(s)と(d)の2つのオペランドを持つ命令ブロックが示されています。
ST言語では、ENO:=DBCD(EN,s,d); または ENO:=DBCDP(EN,s,d); と記述されます。

#### FBD/LD図
FBD/LD図では、EN（実行条件）、ENO（実行結果）、s（BINデータ）、d（BCDデータ）を入力・出力とする命令ブロックが示されています。

### 表
#### 実行条件
| 命令  | 実行条件       |
|-------|----------------|
| DBCD  | 立下りエッジ   |
| DBCDP | 立上りエッジ   |
