### テキスト
#### 機能
- (s)で指定された値の自然対数eを底としたときの対数を演算し、(d)で指定されたデバイスに演算結果を格納する。
- (s)で指定する値は正の数のみ可能で、負の数では演算できない。
- 演算結果が-0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

### 表
#### エラーコード（SD0）
| エラーコード | 内容 |
| :--- | :--- |
| 3402H | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |
| 3405H | (s)に指定可能な範囲外のデータを設定したとき。<br>・指定した値が負の数<br>・指定した値が0 |
