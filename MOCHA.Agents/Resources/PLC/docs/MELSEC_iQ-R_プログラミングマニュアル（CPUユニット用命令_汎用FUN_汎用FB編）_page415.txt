### テキスト・表
#### エラーコード一覧
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3401H              | (s)の各桁に0～9以外の値があるとき。*1 |

*1の注釈として、SM754（BIN、DBIN命令エラー不可フラグ）をONにしておくことで、本エラーを検出しないように設定可能であることが記載されています。また、範囲外の数値を設定した場合、SM754のON/OFFに関わらず命令は実行されません。さらに、BIN(P)命令の場合は、エラーの有無に関わらず指令（実行条件）をOFFからONに切り替えるまで次の演算は実行されないことが説明されています。
</contents>
