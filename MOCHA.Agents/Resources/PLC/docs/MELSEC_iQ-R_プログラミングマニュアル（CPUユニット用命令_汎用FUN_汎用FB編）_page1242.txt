### テキスト
#### 指定ループNo.の演算停止
S(P).PIDSTOP命令は、指定したループNo.のPID演算を停止します。プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります。

停止したループNo.のPID演算は、S(P).PIDINIT命令を実行しても再開しません。停止中は演算データを保持します。

#### 機能
- (s)で指定されたループNo.のPID演算を停止する。
- 停止したループはPIDINIT命令を実行しても再開しない。
- 停止中は演算データを保持。

### 図
- **ラダー図**  
  ラダー図では、(s)入力に停止するループNo.を指定し、ENO出力で実行結果を示す。

- **FBD/LD図**  
  ブロック図では、EN入力で命令の実行を制御し、s入力に停止するループNo.を指定、ENO出力で実行結果を返す。

- **実行条件の波形図**  
  S.PIDSTOP命令は立下りエッジで実行、SP.PIDSTOP命令は立上りエッジで実行される。
