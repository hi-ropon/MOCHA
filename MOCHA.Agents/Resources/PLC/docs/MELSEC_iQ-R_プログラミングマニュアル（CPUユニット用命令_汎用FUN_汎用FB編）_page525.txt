### テキスト
#### 機能
- (s)で指定されたBIN32ビットデータをビットごとに反転し、その結果を(d)で指定されたデバイスに転送する。
- (s)が桁指定したビットデバイスの場合、桁指定で指定されたビットが対象となり、対象外のビットには0が指定されているものとして反転し転送する。

#### エラー
- 演算エラーはありません。

### 図
- 上段に(s)で指定されたビット列（b27からb0までの32ビットの一部）が示されている。
- 下段に(d)で指定されたデバイスに転送されるビット列が示されており、(s)のビット列がビットごとに反転されている様子が矢印で表現されている。
- (1)の注釈として、(s)が32ビット未満の場合、0が指定されているものとして反転されることが説明されている。
