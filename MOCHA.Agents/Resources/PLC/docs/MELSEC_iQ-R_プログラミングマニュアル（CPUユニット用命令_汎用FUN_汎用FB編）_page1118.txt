### テキスト
#### 使用可能デバイス
オペランドごとに使用可能なビット、ワード、ダブルワード、間接指定、定数、その他のデバイスの種類が表形式で示されています。  
(U/H)はユニットおよびCPUユニットの先頭入出力番号を16進数4桁で表したときの上3桁で指定します。

#### 機能
CPUユニットの先頭入出力番号は、読み出すユニットの先頭入出力番号をK4またはH4で指定し、具体的な番号は表で示された通りです。

### 図
CPUユニットの構成を示すブロック図があり、PW、CPU、RX10が並び、その中にRJ71EN71が含まれています。  
下部には16進数のアドレス（0000H、0010H、0020H、0030H、0040H、0060H）が示され、0040Hの部分がRJ71EN71に対応していることが矢印で示されています。
