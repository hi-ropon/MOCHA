### テキスト
#### 機能
- (s)で指定した符号付きBIN32ビットデータ(ANY32_S)を、符号なしBIN32ビットデータ(ANY32_U)に変換し、(d)で指定したラベルに格納する。
- DINT2UDINT(P)命令の使用例を示す。

#### 例
- D+_U命令はオペランドにANY32_Uを指定する必要があるため、D+_U命令を実行する前にDINT2UDINT命令を使用して、ANY32_SのdLabel0をANY32_UのudLabel1へ変換する。
- 値としてはdLabel0をそのままudLabel1へ格納する。

#### エラー
- 演算エラーはありません。

### 図
- ラダー図：bSwitchAビットがONのとき、DINT2UDINT命令でdLabel0の符号付き32ビットデータをudLabel1の符号なし32ビットデータに変換し、その後D+_U命令でudLabel0からudLabel1へ演算を行う流れを示す。
- ビット配置図：DINT2UDINT命令の(s)側と(d)側のビット列(b31～b0)が同じ値(8000H 0001H)であることを示し、変換後も値はそのまま格納されることを表現。

### 用語説明
- bSwitchA：ビット
- dLabel0：ダブルワード[符号付き]
- udLabel0、udLabel1：ダブルワード[符号なし]/ビット列[32ビット]
- (1)：そのまま格納することを示す。
- (2)：D+_U命令のオペランドのデータ型に合わせてから演算することを示す。
