### テキスト
#### BCD4桁除算
指定した2つのBCD4桁データを除算する命令について説明しています。

#### 実行条件
- 命令「B/」は立ち上がりパルスで実行されます。
- 命令「B/P」は立ち上がりエッジで実行されます。

### 図
#### ラダー図
- ラダー図では、2つのBCD4桁データ（s1、s2）を入力し、結果をdに出力する形で表現されています。
- ST（ステートメント）は対応していません。

#### FBD/LD図
- ブロック図形式で、EN（実行条件）、s1、s2（入力）、d（出力）、ENO（実行結果）を持つ命令ブロックが示されています。
- ブロック内にはBDIVISIONまたはBDIVISIONPが入ることが記載されています。
