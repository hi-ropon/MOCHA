### テキスト
#### 割込みプログラムからの復帰 - IRET
割込みプログラムの処理終了を示す命令「IRET」について説明しています。

##### 対応CPU
RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）に対応しています。

##### ラダー
対応していません。

##### FBD/LD
対応していません。

##### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| IRET | 常時実行 |

##### 機能
- 割込みポインタ(I)による割込みプログラムの処理終了を示します。
- IRET命令実行後、シーケンスプログラムへ処理を戻します。

### 表
#### エラーコード(SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 33A0H | 割込みNo.に対応するポインタが存在しないとき。 |
| 33A1H | 割込み発生後IRET命令を実行する前に、END命令、FEND命令、GOEND命令、STOP命令を実行したとき。 |
| 33A2H | 割込みポインタ(I)による割込みプログラムを実行する前にIRET命令を実行したとき。 |
| 33A3H | 定周期実行タイププログラム中でIRET命令を実行したとき。 |
| 33A4H | イベント実行タイププログラム中でIRET命令を実行したとき。 |
</contents>
