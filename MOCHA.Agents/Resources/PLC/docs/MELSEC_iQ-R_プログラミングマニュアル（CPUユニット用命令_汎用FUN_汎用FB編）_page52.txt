### テキスト
#### ローカルデバイスの設定例
ローカルデバイスをM0～M63、D0～D11の範囲で設定している場合の説明です。  
転送先はプログラムAのローカルデバイスM48～M239に該当する範囲であり、M64～M239が存在しなくてもデータは書き込まれます。D0～D10のデータは上書きされます。

#### ラベルの配列設定例
ラベルを下記の配列で設定している場合の説明です。  
転送先はlabelA[1]から10点分に該当する範囲で、labelA[2]～labelA[9]が存在しなくてもデータは書き込まれ、abc[0]～abc[8]のデータは上書きされます。

### 表
| ラベル名 | データ型           | 配列要素数 |
| -------- | ------------------ | ---------- |
| labelA   | ビット列[16ビット] | 2          |
| abc      | ビット列[16ビット] | 10         |

### 図
- **プログラム例（ラダー図）**  
  - BMOV命令でD0からK4#M48までの範囲にデータを転送する例。  
  - BMOV命令でD0からlabelA[1]までの範囲にデータを転送する例。

- **デバイス/ラベルメモリの割付イメージ**  
  - ローカルデバイス領域の割付イメージ図：M15～M0、M31～M16、M63～M48、D0～D11などの範囲が示されている。  
  - ラベルメモリの割付イメージ図：labelA[0]、labelA[1]、abc[0]～abc[9]までの配列要素が縦に並んでいる。

### 注釈
- 転送先の範囲に存在しないデバイスやラベルがあっても、データは書き込まれるが、既存のデータは上書きされることが明記されている。
