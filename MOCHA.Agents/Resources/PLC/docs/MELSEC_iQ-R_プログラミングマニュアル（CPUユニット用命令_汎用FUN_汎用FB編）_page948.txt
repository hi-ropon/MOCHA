### テキスト
#### 機能
- 角度の大きさの単位を、(s)で指定されたラジアン単位から度単位（DEG.単位）に変換し、(d)で指定されたデバイス番号に格納します。
- ラジアン単位→度単位変換は以下の式で行います。
  \[
  \text{度単位} = \text{ラジアン単位} \times \frac{180}{\pi}
  \]
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### 注意事項
- 演算結果の丸め誤差に注意が必要。
