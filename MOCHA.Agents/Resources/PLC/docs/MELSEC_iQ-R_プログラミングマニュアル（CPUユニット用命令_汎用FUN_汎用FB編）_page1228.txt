### テキスト
#### Point
- PID制御用データは、プログラムのスキャンごとに登録または変更できます。ただし、登録または変更した場合は必ずPIDINIT命令を実行してください。PIDINIT命令を実行しないと、登録または変更したデータがPIDCONT命令実行時に反映されません。
- PIDPRMW命令で1ループごとにPID制御用データを変更する場合は、PIDINIT命令を実行する必要はありません。
</contents>
