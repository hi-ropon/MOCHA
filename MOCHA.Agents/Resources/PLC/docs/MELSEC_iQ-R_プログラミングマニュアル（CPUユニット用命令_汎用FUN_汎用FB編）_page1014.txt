### テキスト
#### 機能
- (s1)で指定されたON時間と、(s2)で指定された周期のパルスを、(d)で指定された出力ユニットに出力します。
- (s1)、(s2)の設定時間は1～65535（16進数で0001H～FFFFH）で指定可能です。
- (s1)で指定する値は(s2)で指定された値未満にしてください。
- 以下の場合は無処理となります。
  - (s1)、(s2)が0のとき
  - (s1)≧(s2)のとき
  - PWM命令を2回以上実行したとき

#### エラー
- 演算エラーはありません。

#### Point（注意点）
- PWM命令は引数のデバイスデータをCPUユニットのワークエリアに登録し、実際の出力動作はシステム割込みで処理されます。CPUユニットのワークエリアに登録されたデバイスデータは、実行指令をOFFまたはSTOP→RUNにてクリアされます。
- CPUユニットの割込み間隔は1msであるため、PWM命令はCPUユニットで実行している全プログラムで1回のみ使用可能です。
- PWM命令によりパルス出力中（実行指令がON中）は、PWM命令の引数を変更しないでください。引数を変更する場合は、実行指令をOFFにしてから行ってください。

### 図
- PWMパルスの波形図が示されており、OFFからONに変わる時間が(s1) ms、1周期の長さが(s2) msであることを表しています。波形は矩形波で、ON時間と周期が明確に区別されています。
