### テキスト
■使用可能デバイスについての説明があり、コントロールデータ(d1)、ファイル名(s2)、書き込むデータ(s3)は使用するCPUユニットによって異なる旨の注意書きがあります。

### 表
#### 使用可能デバイス一覧表
- オペランドごとに、ビット、ワード、ダブルワード、間接指定、定数、その他のデバイスの使用可否を示しています。
- ビットデバイスはX, Y, M, L, SM, F, B, SB, FX, FYが対象。
- ワードデバイスはJロ￥口、T, ST, C, D, W, SD, SW, FD, R, ZR, RD、Uロ￥Gロ、Jロ￥口、U3Eロ￥(H)Gロが対象。
- ダブルワードはZ、LT, LST, LCが対象。
- 間接指定はK, Hが対象。
- 定数はEが対象。
- その他は$ (U)が対象。
- 各オペランド(U, s1, d1, s2, s3, d2)ごとに使用可能か「○」または「－」で示されています。

#### 使用するCPUユニットと参照先
- RnCPU、RnENCPU、RnPCPU、RnPSFCPU、RnSFCUの各CPUユニットごとに対応するCPUモデル名と参照先ページが記載されています。
- RnCPUはR01CPU、R02CPU、R04CPU、R08CPU、R16CPU、R32CPU、R120CPUが含まれ、ファームウェアバージョンによって参照先ページが異なります。
- RnENCPU、RnPCPU、RnPSFCPU、RnSFCUも同様にファームウェアバージョンやタイプによって参照先ページが指定されています。

以上の内容から、このページはファイル操作命令における使用可能なデバイスの種類とCPUユニットごとの対応状況を示した技術的な一覧表となっています。
</contents>
