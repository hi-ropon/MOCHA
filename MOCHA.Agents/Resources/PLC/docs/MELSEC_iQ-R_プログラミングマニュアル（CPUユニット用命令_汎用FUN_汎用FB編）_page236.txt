### テキスト
#### BIN16ビット減算
指定した2つのBIN16ビットデータを減算する命令について説明しています。  
対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）となっています。

#### 実行条件
命令には「-」「-U」「-P」「-P_U」の4種類があり、それぞれの実行条件が図で示されています。  
- 「-」「-U」は立下りパルスで実行  
- 「-P」「-P_U」は立上りパルスで実行

### 図
- ラダー回路の簡単なイメージ図があり、2つのBIN16ビットデータを減算することを示しています。  
- ST（ステータス）出力は対応していない旨が記載されています。

### 表
#### 実行条件（命令と実行条件の対応）
| 命令 | 実行条件 |
| :--- | :--- |
| - / -U | 立下りパルス |
| -P / -P_U | 立上りパルス |
