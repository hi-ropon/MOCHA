### テキスト
#### EN/ENO付きファンクションブロックの実行条件と演算結果
ENがTRUE（演算実行）の場合、ENOはTRUEとなり、演算出力値(d1, d2, d3)が出力されます。  
ENがFALSE（演算停止）の場合、ENOはFALSEとなり、前回の出力値が保持されます。

#### タイミングチャートの説明（n=3の場合）
- (1) EN=ONかつ(s1)がOFF→ONで(d3)がカウントアップされる。
- (2) EN=ONかつ(s3)がOFF→ONで(d3)が初期化される。
- (3) EN=ONかつ(s2)がOFF→ONで(d3)がカウントダウンされる。
- (4) EN=ONかつ(s4)がOFF→ONで(d3)が初期化される。

#### エラー
演算エラーはありません。

### 図（タイミングチャート）
- EN、s1～s4、ENO、d1～d3の信号のON/OFF状態を時間軸に沿って示している。
- ENがONの期間中に各s信号がOFFからONに変化するタイミングでd3の値が変化（カウントアップ、カウントダウン、初期化）する様子を視覚的に表現。
- d1、d2はON/OFFの状態を示し、d3は数値（0～3）で変化を示している。
