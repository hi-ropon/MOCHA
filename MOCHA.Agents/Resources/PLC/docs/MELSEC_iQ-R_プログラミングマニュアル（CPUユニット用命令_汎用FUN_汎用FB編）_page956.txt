### テキスト
#### 機能
- (s)で指定された値の指数を演算し、(d)で指定されたデバイスに演算結果を格納する。
- 指数演算の底(e)は「2.71828」として計算される。
- 演算結果が0またはアンダーフローが発生した場合、結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細な注意事項は47ページを参照。

#### Point
- EXPD(P)命令は自然対数の底を用いた演算を行う。
- 常用対数の値を求める場合は、常用対数に対応する値0.43429で割った値を(s)に指定する必要がある。
- 指数計算の数式は以下の通り：
  \[
  10^X = e^{\frac{X}{0.43429}}
  \]
