### テキスト
#### INT型→BOOL型変換
関数名：INT_TO_BOOL(_E)
この関数はINT型データをBOOL型データへ変換します。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）です。

#### 機能
- 演算処理
  - 入力されたINT型データ(s)をBOOL型データに変換し、出力(d)します。
  - 入力値が0の場合はFALSEを出力します。
  - 入力値が0以外の場合はTRUEを出力します。
  - 入力値(s)はINT型データ値です。

- 演算結果
  1. EN/ENOなしファンクション
     - 演算処理を実行し、(d)から演算出力値が出力されます。
  2. EN/ENO付きファンクション
     - 実行条件と演算結果は以下の通りです。

#### エラー
- 演算エラーはありません。

### 表
#### 設定データ（引数、内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL |

#### 実行条件と演算結果
| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)のデータを使用しないようプログラム処理を行う必要があります。

### 図
- ラダー、FBD/LDの回路図例
  - EN/ENOなし：s（INT型入力）からd（BOOL型出力）へ変換するシンプルなブロック図
  - EN/ENO付き：EN入力、ENO出力を持つ変換ブロックの図
- ST言語例
  - EN/ENOなし：d := INT_TO_BOOL(s);
  - EN/ENO付き：d := INT_TO_BOOL_E(EN, ENO, s);
- INT→BOOL変換イメージ
  - 入力値0 → FALSE出力
  - 入力値1567（0以外）→ TRUE出力
