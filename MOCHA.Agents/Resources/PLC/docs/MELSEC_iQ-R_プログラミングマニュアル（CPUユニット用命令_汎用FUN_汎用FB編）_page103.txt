### テキスト・表
#### BIN16ビット/32ビットデータ→16進アスキー変換
- BINHA, BINHAP, DBINHA, DBINHAP命令は、指定したBIN値を16進アスキー値に変換し、指定したワードデバイス番号以降に格納する。

#### BIN16ビット/32ビットデータ→文字列変換
- STR, STRP, STR_U, STRP_U命令は、指定したBIN値を全桁数と小数部桁数の10進文字列に変換し、指定デバイスに格納する。
- DSTR, DSTRP, DSTR_U, DSTRP_U命令も同様に2ワードBIN値を10進文字列に変換し格納。

#### BCD4桁/8桁データ→10進アスキーコード変換
- BCDDA, BCDDAP, DBCDDA, DBCDAP命令は、指定したBCD値を10進アスキー値に変換し、指定ワードデバイス番号以降に格納。

#### 単精度実数→文字列変換
- ESTR, ESTRP命令は、単精度実数データを文字列に変換し、指定デバイスに格納。

#### 16進BINデータ→16進アスキーコード変換
- INT2ASC, INT2ASCP命令は、指定デバイス番号以降の1ワードBIN値を16進アスキーに変換し、指定ワードデバイス番号以降に格納。

#### Unicode文字列→シフトJIS文字列変換
- WS2SJIS, WS2SJISP命令は、Unicode文字列をシフトJIS文字列に変換し、格納。

#### シフトJIS文字列→Unicode文字列変換（バイトオーダーマークなし）
- SJIS2WS, SJIS2WSP命令は、シフトJIS文字列をUnicode文字列に変換し、格納。

#### シフトJIS→Unicode変換（バイトオーダーマークあり）
- SJIS2WSB, SJIS2WSBP命令は、シフトJIS文字列をUnicode文字列に変換し、先頭にバイトオーダーマークを付加して格納。

#### 文字列の長さ検出
- LEN, LENP命令は、指定デバイスに格納されている文字列データの長さ（文字数）を指定デバイスに格納。

### 表構成
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| 各種命令（例：BINHA, STR, BCDDAなど） | 指定したデータを特定の形式に変換し、指定デバイスに格納する処理内容の説明 | 参照ページ番号と命令名 |

以上のように、各命令の名称、処理内容、参照先が整理されている。
