### テキスト
#### 倍精度実数符号反転（EDNEG(P)）
倍精度実数データの符号を反転する命令です。RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）に対応しています。

#### 機能
(d)で指定されたデバイスに格納されている倍精度実数データの符号を反転し、同じデバイスに格納します。正負の符号を反転させる際に使用します。

#### エラー
- エラーコード3402H：(d)で指定した内容が-0、非正規化数、非数、±∞のときに発生します。

### 図
- ラダー図：符号反転命令のラダー回路表現。入力(d)に対して符号反転を行う。
- FBD/LD図：EN（実行条件）入力、ENO（実行結果）出力、d（倍精度実数データ）入力のブロック図。
- 符号反転のイメージ図：倍精度実数の値が正から負に反転する様子を示す。
