### テキスト
#### 間接アドレス読出し
「ADRSET(P)」命令は、指定したデバイスの間接アドレスを読み出す命令です。

#### 実行条件
- ADRSET命令は立ち上がりエッジで実行されます。
- ADRSETP命令は立ち下がりエッジで実行されます。

#### 設定データ
##### 内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
| (s) | 間接アドレスを読み出すデバイス番号 | なし | デバイス名 | ANY_ELEMENTARY |
| (d) | (s)で指定されたデバイスの間接アドレスを格納するデバイス番号 | なし | 符号付きBIN32ビット | ANY32 |
| EN | 実行条件 | なし | ビット | BOOL |
| ENO | 実行結果 | なし | ビット | BOOL |

### 図
- ラダー図：間接アドレス読出し命令のラダー回路記述例。sとdのオペランドを持ち、ENO出力がある。
- FBD/LD図：EN入力、ENO出力、s入力、d出力を持つ命令ブロックの図。
