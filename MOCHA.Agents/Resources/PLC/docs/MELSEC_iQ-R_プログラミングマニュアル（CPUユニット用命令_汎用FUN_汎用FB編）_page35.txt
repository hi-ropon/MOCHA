### テキスト
#### 総称データ型
いくつかの基本データ型をまとめたラベルのデータ型について説明しています。ファンクションやファンクションブロックの引数や戻り値などで複数のデータ型が許される場合に総称データ型が使用されます。総称データ型で定義されたラベルは、下位のデータ型のどの型でも使用可能です。

#### 総称データ型（配列）
総称データ型の配列について説明しています。配列要素数を定義する必要があることが記載されています。

### 表
#### 総称データ型のデータ型（ラベル）と指定可能なデータ型
| データ型(ラベル) | 指定可能なデータ型 |
|------------------|--------------------|
| ANY*1            | ANY_ELEMENTARY     |
|                  | ANY_BIT            | ANY_BOOL（ビット）<br>ANY_BITADDR（ビット）<br>ANY16_U（ワード[符号なし]/ビット列[16ビット]）<br>ANY32_U（ダブルワード[符号なし]/ビット列[32ビット]） |
|                  | ANY_WORDADDR       | ANY_NUM            | ANY_INT            | ANY16              | ANY16_S（ワード[符号付き]）<br>ANY16_U（ワード[符号なし]/ビット列[16ビット]） |
|                  |                    |                    | ANY32              | ANY32_S（ダブルワード[符号付き], 時間）<br>ANY32_U（ダブルワード[符号なし]/ビット列[32ビット]） |
|                  |                    |                    | ANY_REAL           | ANYREAL_32（単精度実数）<br>ANYREAL_64（倍精度実数） |
|                  | ANY_STRING         | ANYSTRING_SINGLE（文字列）<br>ANYSTRING_DOUBLE（文字列[Unicode]） |
|                  | ANY16_OR_STRING_SINGLE | ANY16_S（ワード[符号付き]）<br>ANY16_U（ワード[符号なし]/ビット列[16ビット]）<br>ANYSTRING_SINGLE（文字列） |
|                  | ANY_DT             | ワード[符号付き]、ワード[符号なし]/ビット列[16ビット] |
|                  | ANY_TM             | ワード[符号付き]、ワード[符号なし]/ビット列[16ビット] |
| ANY_STRUCT*1     | 構造体             |
| STRUCT           | 構造体             |

*1 配列としても使用可能。
