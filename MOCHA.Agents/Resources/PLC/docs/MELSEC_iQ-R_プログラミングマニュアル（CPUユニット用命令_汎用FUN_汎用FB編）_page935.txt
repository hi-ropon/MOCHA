### テキスト
#### BCD型SIN⁻¹演算 BASIN(P)
BCD値で指定した値のSIN⁻¹（逆正弦）値を演算します。

#### 実行条件
- BASIN命令は立ち上がりエッジで実行
- BASINP命令は立ち下がりエッジで実行

#### 機能
- (s)で指定された値のSIN⁻¹（逆正弦）値を演算し、(d)で指定されたデバイスに演算結果（角度）を格納します。
- (s)は符号、(s)+1は整数部、(s)+2は小数部をそれぞれBCD値で格納します。
- (s)の符号は演算データの符号を設定し、正のときは0、負のときは1を格納します。
- (s)+1、(s)+2には演算するデータの整数部、小数部をBCD値で格納（0～1.00000まで設定可能）。
- (d)に格納される演算結果はBCD値で0～90°、270～360°（度単位）の範囲内の値です。
- 演算結果は小数部を四捨五入した値となります。

### 図
- ラダー回路図：BASIN命令のラダー表現（(s)、(d)の入力）
- FBD/LD回路図：BASIN命令のFBD/LD表現（EN、ENO、s、dの入出力）
- 演算結果のBCD格納イメージ：SIN⁻¹の符号、整数部、小数部をそれぞれBCD値で格納する様子を示す。
