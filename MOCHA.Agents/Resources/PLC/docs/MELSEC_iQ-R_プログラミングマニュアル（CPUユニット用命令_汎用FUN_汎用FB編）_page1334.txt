### テキスト
#### 32.17 DWORD型→TIME型変換
関数「DWORD_TO_TIME(_E)」はDWORD型データをTIME型データへ変換するための関数です。対応CPUはRnCPU、RnENCPU、RnFSPCPUなど複数の種類が示されています。

#### ラダー、FBD/LD、STでの使用例
- ラダー、FBD/LDではEN/ENOなしとEN/ENO付きの2種類のブロック形状が示されています。
- ST言語ではEN/ENOなしの場合は「d := DWORD_TO_TIME(s);」、EN/ENO付きの場合は「d := DWORD_TO_TIME_E(EN, ENO, s);」と記述します。

### 表
#### 設定データ（内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | DWORD |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | TIME |

### 機能
#### 演算処理
- 入力されたDWORD型データ(s)をTIME型データに変換し、出力(d)します。
- 入力値はDWORD型の数値です。

#### 演算結果
1. EN/ENOなしファンクション
   - 演算処理を実行し、dから演算出力値が出力されます。
2. EN/ENO付きファンクション
   - 実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となるため、dのデータを使用しないようプログラム処理を行う必要があります。

### エラー
- 演算エラーはありません。

### 図・説明
- ラダー/FBD/LDの関数ブロック形状（EN/ENOなし、EN/ENO付き）
- ST言語での関数呼び出し例
- DWORD型の数値例（0、1234567）をTIME型（T#0ms、T#20m34s567ms）に変換するイメージ図（矢印付き）
