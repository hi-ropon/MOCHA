### テキスト
#### 2相入力アップ/ダウンカウンタ（UDCNT2）
指定したA相パルスとB相パルスの状態により、カウンタの現在値を更新する命令です。

#### 実行条件
命令「UDCNT2」はパルスの立ち上がりで実行されます。

### 図
#### ラダー図
- (s1)、(d)、(s2)の3つのオペランドを持つ命令ブロックが示されています。

#### FBD/LD図
- EN（入力）、s1、s2（入力）、d（出力）、ENO（出力）を持つ命令ブロックの形状が示されています。
