<overview>
このページは主に表形式で構成されており、BIN16ビットおよび32ビットデータに関する各種命令の一覧とその処理内容、参照先がまとめられています。テキスト部分は命令の説明や処理内容の詳細を記述しており、表部分は命令記号、処理内容、参照先の3列で整理されています。グラフや図はなく、命令の動作を示す簡単なイラスト（ビットデータの範囲や構造を示す図）が一部に含まれています。
</overview>

<contents>
### テキスト・表
#### BIN16ビット/32ビットデータサーチ（最小・同一・最大）
- 命令記号：SERMM、SERMMP、DSERMM、DSERMMP
- 処理内容：指定したBIN16ビットまたはBIN32ビットデータから、(n)点分、(s2)で指定したデータと同一のデータおよび最大値、最小値を検索する。
- 参照先：SERMMは714ページ、DSERMMは716ページ

#### BIN16ビット/32ビットデータビットチェック
- 命令記号：SUM、SUMP、DSUM、DSUMP
- 処理内容：指定したデバイスのBIN16ビットまたはBIN32ビットデータのうち、1になっているビットの総数を格納する。
- 参照先：SUMは718ページ、DSUMは719ページ

#### BIN16ビット/32ビットデータのビット判定
- 命令記号：BON、BONP、DBON、DBONP
- 処理内容：指定したデバイスの(n)ビットの状態がONかOFFかをチェックし、結果を指定したデバイスへ格納する。
- 参照先：BONは721ページ、DBONは723ページ

#### BIN16ビット/32ビットデータ最大値検索
- 命令記号：MAX、MAXP、MAX_U、MAXP_U、DMAX、DMAXP、DMAX_U、DMAXP_U
- 処理内容：指定したデバイスから(n)点分のデータを16ビットまたは32ビット単位で検索し、最大値を指定のデバイスに格納する。
- 参照先：MAXは725ページ、DMAXは727ページ

#### BIN16ビット/32ビットデータ最小値検索
- 命令記号：MIN、MINP、MIN_U、MINP_U、DMIN、DMINP、DMIN_U、DMINP_U
- 処理内容：指定したデバイスから(n)点分のデータを16ビットまたは32ビット単位で検索し、最小値を指定のデバイスに格納する。
- 参照先：MINは729ページ、DMINは731ページ

### 図（簡易イラスト）
- 各命令の処理内容説明において、BIN16ビットやBIN32ビットデータの範囲やビット位置を示す簡単な図が添えられている。例えば、(s1)で指定したデータの範囲や、(s2)で指定した比較対象のデータの位置関係を示す矢印付きの図。
