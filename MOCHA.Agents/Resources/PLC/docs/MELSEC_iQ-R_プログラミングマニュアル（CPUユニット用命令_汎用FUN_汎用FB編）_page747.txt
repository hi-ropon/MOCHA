### テキスト
#### 動作例
並べ替え元データ（データ行数n1=5、データ列数n2=4）を各条件で並べ替えたときの動作例を示しています。

**Point**
1列目に管理番号などの通し番号を入れておくと、その内容で元の行番号が判断できます。

#### 例1
列番(n3)=2、SM774=OFF（昇順）での並べ替え結果を示しています。

#### 例2
列番(n3)=3、SM774=ON（降順）での並べ替え結果を示しています。
