### テキスト
#### 出力値上下限設定: (s3)+1 ビット5
出力値上下限設定によって出力値がどのように変化するかを説明しています。出力値上下限設定は、PID制御の積分項の増大を抑える効果があり、この機能を使用する場合は、(s3)+1のビット2を必ずOFFに設定する必要があります。

### グラフ
- 横軸は時間(t)、縦軸は出力値(MV)を示しています。
- グラフには「出力値上下限なし」と「出力値上下限あり」の2つの波形が描かれています。
- 「出力値上下限なし」の波形は上下限を超えて出力される部分があるのに対し、「出力値上下限あり」の波形はULV（出力上限値）とLLV（出力下限値）の範囲内に制限されており、上下限を超えた部分は出力されないことが示されています。
- MV：出力値、ULV：出力上限値、LLV：出力下限値、t：時間を表しています。

### 表
| 設定項目       | 内容           | 設定範囲               |
| -------------- | -------------- | ---------------------- |
| 動作設定(ACT)  | (s3)+1 ビット2 | 出力変化量警報         |
|                | (s3)+1 ビット5 | 出力値上下限設定       |
|                |                | ON: 使用する           |
|                |                | OFF: 使用しない        |

- ビット2は必ずOFFに設定する必要があります。
