### テキスト
#### 32.51 TIME型→DINT型変換
TIME_TO_DINT(_E)関数は、TIME型データをDINT型データへ変換します。

##### 対応CPU
RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）

##### 機能
- (s)に入力されたTIME型データをDINT型データに変換し、(d)から出力します。
- (s)への入力値はTIME型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :---------- | :----------- | :-- |
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE | 不定値 |

※ ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)のデータを使用しないようプログラム処理を行う必要があります。

##### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDの関数ブロック図  
　[EN/ENOなし]はs入力とd出力のみのシンプルな構造。  
　[EN/ENO付き]はEN入力、ENO出力が追加されている。

- ST（構造化テキスト）での記述例  
　[EN/ENOなし]：d := TIME_TO_DINT(s);  
　[EN/ENO付き]：d := TIME_TO_DINT_E(EN, ENO, s);

### 表
#### 設定データ（引数、内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :---- | :---- | :---- | :---- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | TIME |
| ENO | 出力状態(TRUE:正常実行、FALSE:異常または停止) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | DINT |
