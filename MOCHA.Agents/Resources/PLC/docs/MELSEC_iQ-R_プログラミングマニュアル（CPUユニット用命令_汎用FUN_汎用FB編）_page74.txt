<overview>
本ページは命令コードに関する説明で構成されており、主に16ビットおよび32ビットのビットデータに対する排他的論理和（XOR）および否定排他的論理和（XNOR）命令について記載されています。ページ全体はテキストと表で構成されており、表には命令コード、処理内容、参照先が整理されています。また、処理内容の説明には簡単な図解が含まれています。グラフは含まれていません。
</overview>

### テキスト・表
#### 16ビットブロックデータ排他的論理和
- **命令コード**: BKXOR, BKXORP
- **処理内容**: 指定した2つのBIN16ビットデータブロックの排他的論理和を行う。
- **図解**: (s1)と(s2)の2つの入力ビットブロックから排他的論理和を取り、(d)に出力するイメージが示されている。
- **参照先**: 324ページ BKXOR(P)

#### 16ビット/32ビットデータ否定排他的論理和
- **命令コード**: WXNR, WXNRP, WXNR, WXNRP, DXNR, DXNRP, DXNR, DXNRP
- **処理内容**: 
  - 16ビットデータの否定排他的論理和を行う命令（オペランド2個または3個の場合）
  - 32ビットデータの否定排他的論理和を行う命令（オペランド2個または3個の場合）
- **参照先**:
  - 326ページ WXNR(P) [オペランド2個の場合]
  - 328ページ WXNRP(P) [オペランド3個の場合]
  - 330ページ DXNR(P) [オペランド2個の場合]
  - 332ページ DXNRP(P) [オペランド3個の場合]

#### 16ビットブロックデータ否定排他的論理和
- **命令コード**: BKXNR, BKXNRP
- **処理内容**: 指定した2つのBIN16ビットデータブロックの否定排他的論理和を行う。
- **図解**: (s1)と(s2)の2つの入力ビットブロックから否定排他的論理和を取り、(d)に出力するイメージが示されている。
- **参照先**: 334ページ BKXNR(P)
</contents>
