### テキスト
#### 機能
- (d)で指定されたデバイスの内容（BIN32ビットデータ）を+1します。
- 図解では、ビット列b31からb0までのデータが73500（BIN）から73501（BIN）に+1される様子を示しています。
- (d)で指定されたデバイスの内容が2147483647のときにDINC(P)命令を実行すると、-2147483648が(d)で指定されたデバイスに格納されます（符号付きの場合）。
- (d)で指定されたデバイスの内容が4294967295のときにDINC(P)_U命令を実行すると、0が(d)で指定されたデバイスに格納されます（符号なしの場合）。

#### エラー
- 演算エラーはありません。
</contents>
