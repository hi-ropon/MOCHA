### テキスト
#### 32.38 BCD型→DINT型変換
関数名：BCD_TO_DINT(_E)
対応CPU：RnCPU、RnECPU（プロセス、三重化、一般、安全）に対応（安全CPUは非対応）
説明：BCD型データをDINT型データへ変換する関数です。

### 図
- ラダー図（EN/ENOなし）：入力sから出力dへBCD_TO_DINT変換を行うシンプルな構成。
- ラダー図（EN/ENO付き）：実行条件EN、出力状態ENOを含む構成で、sからdへ変換。
- ST言語：EN/ENOなしの場合は d := BCD_TO_DINT(s); EN/ENO付きの場合は d := BCD_TO_DINT_E(EN, ENO, s); と記述。

### 表
#### 設定データ（引数一覧）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件（TRUE:実行、FALSE:停止） | 入力変数 | BOOL |
| s(IN) | 入力データ（BCD型） | 入力変数 | ANY_BIT |
| ENO | 出力状態（TRUE:正常、FALSE:異常） | 出力変数 | BOOL |
| d | 出力データ（DINT型） | 出力変数 | DINT |

この関数は、BCD形式の入力データをDINT形式の整数データに変換し、実行条件に応じて動作します。
