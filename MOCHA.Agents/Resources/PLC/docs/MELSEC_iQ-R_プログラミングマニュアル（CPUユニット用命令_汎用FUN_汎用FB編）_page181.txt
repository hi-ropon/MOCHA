### テキスト
#### 機能
実行指令がONになると、指定されたデバイスは以下の状態になります。

- 実行指令がOFFの場合はデバイスの状態は変化しません。
- RST命令でワードデバイス指定時の機能は、下記の回路と同じ動作をします。

#### エラー
演算エラーはありません。

### 表
| デバイス | デバイスの状態 |
| :--- | :--- |
| ビットデバイス | コイル、接点をOFFにします。 |
| タイマ、カウンタ | 現在値を0にし、コイル、接点をOFFにします。 |
| ワードデバイスのビット指定 | 指定ビットを0にします。 |
| タイマ、カウンタ以外のワードデバイス | 内容を0にします。 |

### 図（回路例）
- 左側の回路は、X10入力がONのときにRST命令でD50をリセットする回路を示しています。
- 右側の回路は、X10入力がONのときにMOV命令で定数K0をD50に設定する回路を示しています。
- これにより、RST命令でワードデバイスを指定した場合の動作がMOV命令で0を代入する動作と同じであることを示しています。
