### 表
#### 使用可能デバイス
この表は、各オペランドに対して使用可能なデバイスの種類を示しています。表の構成は以下の通りです。

- 行：オペランドの種類（s1, s2, s3, s4, d1, d2）
- 列：デバイスの種類（ビット、ワード、ダブルワード、間接指定、定数、その他）

各セルには「○」や「－」で使用可否が示されています。

##### 主な内容
- ビットデバイスは「X, Y, M, L, SM, F, B, SB, FX, FY」などがあり、オペランドd1のみ使用可能。
- ワードデバイスは「T, ST, C, D, W, SD, SW, FD, R, ZR, RD」などがあり、s1, s2, s3, s4, d2で使用可能。
- ダブルワードデバイスは「Z, LT, LST, LZ, LC」などがあり、全オペランドで使用不可。
- 間接指定はs1～d2すべてで「○」となっている。
- 定数は「K, H, E」でs2のみ「○」、他は「－」。
- その他の項目はすべて「－」で使用不可。

この表は、CPUユニット内蔵のデータベースアクセス命令において、どのオペランドでどのデバイスが使用可能かを示すものと推測されます。
