<overview>
本ページはCPUユニット用命令の基本命令に関する説明で構成されています。ページ全体は主に表形式で命令の種類ごとに区分されており、各命令の命令記号、処理内容、参照先が記載されています。テキスト部分は命令の説明文で、表部分は命令記号と処理内容を整理したものです。さらに、各命令の動作を示す簡単な図（ビットデータの転送や交換のイメージ）が含まれています。グラフはなく、図は命令の動作イメージを示す簡易的なものに留まっています。
</overview>

<contents>
### テキスト
本ページはBIN16ビットおよびBIN32ビットのデータブロック転送やデータ交換に関する命令を説明しています。各命令は、指定されたデバイス間でビットデータを転送または交換する処理を行います。

### 表
#### 同一BIN16ビットデータブロック転送（32ビット）
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| FMOVL / FMOVLP | BIN16ビットデータを指定されたデバイスから(n)点に転送します。 | 536ページ FMOVL(P) |

#### 同一BIN32ビットデータブロック転送（16ビット）
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DFMOV / DFMOVP | BIN32ビットデータを、指定したデバイスから(n)点に転送します。 | 538ページ DFMOV(P) |

#### 同一BIN32ビットデータブロック転送（32ビット）
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DFMOVL / DFMOVLP | BIN32ビットデータを、指定したデバイスから(n)点に転送します。 | 540ページ DFMOVL(P) |

#### BIN16ビット/32ビットデータ交換
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| XCH / XCHP | 指定されたBIN16ビットデータを交換します。 | 542ページ XCH(P) |
| DXCH / DXCHP | 指定されたBIN32ビットデータを交換します。 | 544ページ DXCH(P) |

#### BIN16ビットデータブロック交換
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| BXCH / BXCHP | 指定されたデバイスから(n)点のBIN16ビットデータを交換します。 | 546ページ BXCH(P) |

#### BIN16ビットデータ上下バイト交換
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| SWAP / SWAPP | 指定したデバイスの上下8ビットの値を交換します。 | 548ページ SWAP(P) |

### 図
- 各命令の処理内容を示す簡易図があり、例えばビットデータの転送元と転送先の関係や、データのビット単位の交換イメージが描かれています。
- SWAP命令の図では、16ビットデータの上位8ビット(b15～b8)と下位8ビット(b7～b0)が入れ替わる様子が示されています。
