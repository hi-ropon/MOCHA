### テキスト
#### データテーブルからの後入データリード
- 命令記号：FPOP、FPOPP
- 処理内容：テーブルの最後に格納したデータを指定されたデバイスに格納する。

#### データテーブルへのデータライト
- 命令記号：FIFW、FIFWP
- 処理内容：BIN16ビットデータを指定されたデータテーブルに格納する。

#### データテーブルのデータ挿入、削除
- 命令記号：FINS、FINSP
- 処理内容：BIN16ビットデータを指定されたデータテーブルの(n)番目に挿入する。
- 命令記号：FDEL、FDELP
- 処理内容：データテーブルの(n)番目のデータを削除する。

#### データリード/ライト命令
- データメモリからのデータリード
  - 命令記号：S.DEVLD、SP.DEVLD
  - 処理内容：データメモリ上のデバイスデータ格納用ファイルからデータを読み出す。
- データメモリへのデータライト
  - 命令記号：SP.DEVST
  - 処理内容：データメモリ上のデバイスデータ格納用ファイルに書き込む。

### 図（命令動作イメージ）
- 各命令の処理内容に対応した簡単な図があり、データの流れや挿入・削除の位置を示している。
  - 例：FPOP命令では、データテーブルの最後のデータをデバイスに格納する流れを矢印で示す。
  - FINS命令では、指定位置(n)にデータを挿入する様子を示す。
  - FDEL命令では、指定位置(n)のデータを削除し、後続データが詰められる様子を示す。
