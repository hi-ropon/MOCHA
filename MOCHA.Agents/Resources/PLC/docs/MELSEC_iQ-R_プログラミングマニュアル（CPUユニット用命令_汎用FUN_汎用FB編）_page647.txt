### テキスト
#### 前回読出し位置の続きから読み出します
- 実行/完了タイプ：CSV形式変換読出し（10進数（BIN16ビットデータ））
- 列数指定：4H
- データタイプ指定：ワード
- ファイル位置：FFFFFFFFH（前回の読出し位置から継続）
- 読出し先頭デバイス：D7
- 読出し結果データ数：5H

### 図の説明
- 図は5行4列のCSVデータを示し、各セルに数値と区切り文字（カンマ、CR、LF）が配置されている。
- 1～2行目は既に読み出されたデータを示し、3行目から読み出しを再開する様子を矢印で表現。
- (1)は開始行を示し、3行目の11と12のデータが読み出される。
- (2)は次回の開始位置を示し、4行目の13～16のデータが次に読み出されることを示す。
- (3)はデバイス格納データの対応を示し、D0～D13のデバイスにK5～K15の値が格納される。
- Nはデータ数を示し、D7から5個のデータが読み出されることを意味する。

### Point（注意点）
- 前回の読出し位置から継続して読み出しを行う場合、「実行/完了タイプ」「列数指定」「データタイプ指定」を前回と異なる設定にすると、正常に追加読み出しができない。
- 読出し途中に他のSP.FREAD命令やSP.FWRITE命令を実行すると、前回の読出し位置から正常に追加読み出しができなくなる。
