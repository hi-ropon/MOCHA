### テキスト
#### 機能
(s1)で指定したBIN32ビットデータと、(s2)で指定したBIN32ビットデータを比較し、その結果（小、 一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONします。

#### エラー
演算エラーはありません。

### 図
#### 比較動作の説明図（左側）
- X0が入力されると、DCMP命令が動作し、(s1)、(s2)、(d)の3つのビットデータを比較します。
- 比較結果により、以下のいずれかのビットがONになります。
  - [(s1)、(s1)+1] > [(s2)、(s2)+1] のとき (d)
  - [(s1)、(s1)+1] = [(s2)、(s2)+1] のとき (d)+1
  - [(s1)、(s1)+1] < [(s2)、(s2)+1] のとき (d)+2

#### タイミングチャート（右側）
- X0のパルスに合わせて、(s1)、(s2)、(d)、(d)+1、(d)+2のビットのON/OFF状態が示されています。
- (s1)の値が49から50に変化し、その比較結果に応じて(d)、(d)+1、(d)+2のいずれかがONし、一定時間保持される様子が描かれています。
- 保持時間は(d)、(d)+1、(d)+2それぞれ異なり、比較結果に応じて適切にビットがONされる動作を示しています。
