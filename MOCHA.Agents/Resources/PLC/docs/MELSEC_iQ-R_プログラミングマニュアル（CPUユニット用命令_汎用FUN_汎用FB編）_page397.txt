### テキスト
#### 機能
- (d)で指定したデバイスから単精度実数(n1)点分のデータ範囲内で、右へ(n2)点分シフトする。
- シフト後、(s)から(n2)点分は空きデータ部分に格納される。
- (s)に定数を指定した場合、シフト後の最上位から(n2)点分は指定した値になる。
- (n2)で指定した値が0の場合は無処理となる。

#### 例
- (n1)=9、(n2)=4の場合の動作例を示す。

### 図
- データの並びを示す図で、(d)から始まる9点のデータと(s)から始まる4点のデータが表示されている。
- 矢印で(n2)点分の右シフトとコピーの動作を示している。
- シフト後のデータ配置が具体的な数値で示されている。

| 3405H | (n1)、(n2)に指定した値が(n1) < (n2)のとき。 |
