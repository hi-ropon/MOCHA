<overview>
このページは主にテキストと表で構成されており、PLCのプログラムスキャン実行登録とデータ処理に関するローテーション命令について説明しています。テキスト部分では命令の概要や処理内容が記載され、表部分では命令記号、処理内容、参照先が整理されています。さらに、ビットデータの右ローテーションおよび左ローテーション、BIN16およびBIN32ビットデータのローテーション動作を示す簡単な図（ビットの回転方向を示す矢印付き）が含まれています。グラフはなく、図は説明用の簡易的なビット回転の模式図のみです。
</overview>

### テキスト
#### プログラムスキャン実行登録
- 命令記号「PSCAN」「PSCANP」は、指定したプログラムをスキャン実行タイプとして登録するための命令である。
- 参照先は605ページのPSCAN(P)に記載。

#### データ処理
##### ローテーション命令
- BIN16ビットデータの右ローテーション
  - 命令記号：ROR、RORP
  - 処理内容：BIN16ビットデータをキャリフラグを含めずに(n)ビット右へ回転させる。
  - 図：b15からb0までのビットが右方向に回転し、SM700に出力される模式図。
  - 参照先：607ページ ROR(P)、RCR(P)

- BIN16ビットデータの右ローテーション（キャリフラグ含む）
  - 命令記号：RCR、RCRP
  - 処理内容：BIN16ビットデータをキャリフラグを含めて(n)ビット右へ回転させる。
  - 図：同上、キャリフラグを含む回転を示す。
  - 参照先：607ページ ROR(P)、RCR(P)

- BIN16ビットデータの左ローテーション
  - 命令記号：ROL、ROLP
  - 処理内容：BIN16ビットデータをキャリフラグを含めずに(n)ビット左へ回転させる。
  - 図：b0からb15までのビットが左方向に回転し、SM700に出力される模式図。
  - 参照先：610ページ ROL(P)、RCL(P)

- BIN16ビットデータの左ローテーション（キャリフラグ含む）
  - 命令記号：RCL、RCLP
  - 処理内容：BIN16ビットデータをキャリフラグを含めて(n)ビット左へ回転させる。
  - 図：同上、キャリフラグを含む回転を示す。
  - 参照先：610ページ ROL(P)、RCL(P)

- BIN32ビットデータの右ローテーション
  - 命令記号：DROR、DRORP
  - 処理内容：BIN32ビットデータをキャリフラグを含めずに(n)ビット右へ回転させる。
  - 図：b31からb0までのビットが右方向に回転し、SM700に出力される模式図。
  - 参照先：613ページ DROR(P)、DRCR(P)

- BIN32ビットデータの右ローテーション（キャリフラグ含む）
  - 命令記号：DRCR、DRCRP
  - 処理内容：BIN32ビットデータをキャリフラグを含めて(n)ビット右へ回転させる。
  - 図：同上、キャリフラグを含む回転を示す。
  - 参照先：613ページ DROR(P)、DRCR(P)

### 図（模式図の説明）
- 各ローテーション命令にはビットの回転方向を示す矢印付きの模式図があり、ビット番号b0からb15（またはb31まで）を示し、どの方向にビットが回転するかを視覚的に表現している。
- キャリフラグを含む場合は、キャリフラグが回転に含まれることを示す矢印が追加されている。
