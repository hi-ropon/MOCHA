### テキスト
#### 符号付きBIN32ビットデータ→符号なしBIN32ビットデータ変換
命令名は「DINT2UDINT(P)」で、指定した符号付きBIN32ビットデータを符号なしBIN32ビットデータに変換します。

対応環境は、バージョン"1.035M"以降のエンジニアリングツールです。

##### ■実行条件
- DINT2UDINT命令は立ち上がりエッジで実行されます。
- DINT2UDINTP命令は立ち上がりレベルで実行されます。

##### ■Point
DINT2UDINT(P)命令はラベルでプログラミングする場合に、命令のオペランドに指定可能なデータ型と、指定されたラベルのデータ型を一致させるために使用します。デバイスでプログラミングする場合は、この命令を使用する必要はありません。

### 図
- **ラダー図**：命令のラダー表現として、(s)と(d)のオペランドを持つ命令ブロックが示されています。
- **FBD/LD図**：EN（実行条件）、ENO（実行結果）、s（入力データ）、d（出力データ）を持つ命令ブロックの図が示されています。
