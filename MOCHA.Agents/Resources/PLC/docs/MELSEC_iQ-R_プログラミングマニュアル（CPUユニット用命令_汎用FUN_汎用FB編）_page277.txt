### テキスト
#### 機能
(s1)で指定したBCD8桁データと、(s2)で指定したBCD8桁データの除算を行い、その除算結果を(d)で指定したデバイスに格納します。

- 除算結果は64ビットを使用して商と余りを格納します。  
  - 商（BCD8桁）は下位32ビットに格納。  
  - 余り（BCD8桁）は上位32ビットに格納。  
- (d)をビットデバイスで指定した場合、除算結果の余りは格納されません。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3400H              | (s2)で指定した値（除数）が0のとき。 |
| 3405H              | (s1)のBCDデータが0～99999999以外のとき。<br>(s2)のBCDデータが0～99999999以外のとき。 |

### 図
- BCD8桁データのビット配置を示す図があり、(s1)と(s2)のデータがそれぞれ8桁で表現されている。  
- 除算の商は(d)の下位4桁に、余りは(d)の上位4桁に格納されることを示している。  
- (s2)のBCDデータの一部に0が入ることが図で説明されている。  
- 商、余りの格納位置が(d)+1、(d)、(d)+3、(d)+2の4つの位置に分かれていることが示されている。
