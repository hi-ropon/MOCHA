### テキスト
#### プログラムファイル間サブルーチンプログラム出力OFFコール（EFCALL(P)）
- プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約がある旨の注意書きがあります。
- 指定したプログラムファイル名の指定されたポインタのサブルーチンプログラムの非実行処理を行う命令です。

#### 実行条件
- EFCALL命令は立ち上がりエッジで実行されます。
- EFCALLP命令は立ち下がりエッジで実行されます。

### 図
#### ラダー図
- (FILE)(P)の形式で指定し、(s1)...(s5)の引数を渡すことが可能。
- STは対応していません。

#### FBD/LD図
- EN（実行条件）、ENO（実行結果）、FILE（ファイル名）、P（ポインタ）を入力とするブロック図。
