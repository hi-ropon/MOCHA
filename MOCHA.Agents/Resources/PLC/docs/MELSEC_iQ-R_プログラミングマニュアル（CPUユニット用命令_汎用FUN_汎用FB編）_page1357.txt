### テキスト
#### 32.35 DINT型→TIME型変換
関数名：DINT_TO_TIME(_E)  
対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）  
説明：DINT型データをTIME型データへ変換する関数です。

#### 機能
- 演算処理  
  入力されたDINT型のデータ(s)をTIME型のデータ(d)に変換し出力します。  
  sの入力値はDINT型のデータ値です。

- 演算結果  
  1. EN/ENOなしファンクション  
  演算処理を実行し、dから演算出力値が出力されます。  
  2. EN/ENO付きファンクション  
  実行条件と演算結果は以下の通りです。

#### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDの関数ブロック図  
  EN/ENOなし：s入力、d出力のシンプルなブロック  
  EN/ENO付き：EN入力、ENO出力、s入力、d出力のブロック

- ST（構造化テキスト）記述例  
  EN/ENOなし：d := DINT_TO_TIME(s);  
  EN/ENO付き：d := DINT_TO_TIME_E(EN, ENO, s);

- 演算処理イメージ図  
  DINT型の数値（例：1234567）をTIME型の時間表現（例：T#20m34s567ms）に変換する流れを示す。

### 表
#### 設定データ（引数一覧）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | DINT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | TIME |

#### EN/ENO付きファンクションの演算結果
| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、(d)のデータを使用しないようプログラム処理を行う必要がある。
