### テキスト
#### 機能
(s3)で指定した入力値（BIN32ビット値）が、(s1)、(s2)で指定した上下限リミット値の範囲により、(d)で指定したデバイスに格納される出力値を制御します。  
出力値は以下の条件に従って制御されます。

- 下限リミット値(s1)、(s1)+1 > 入力値(s3)、(s3)+1 のとき → 出力値は下限リミット値(s1)、(s1)+1  
- 上限リミット値(s2)、(s2)+1 < 入力値(s3)、(s3)+1 のとき → 出力値は上限リミット値(s2)、(s2)+1  
- 下限リミット値(s1)、(s1)+1 ≤ 入力値(s3)、(s3)+1 ≤ 上限リミット値(s2)、(s2)+1 のとき → 出力値は入力値(s3)、(s3)+1  

また、  
- 上限リミット値のみ制御する場合は、(s1)で指定する下限リミット値に設定データ範囲の最小値を設定する。  
- 下限リミット値のみ制御する場合は、(s2)で指定する上限リミット値に設定データ範囲の最大値を設定する。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3405H              | (s1)で指定した下限リミット値が、(s2)で指定した上限リミット値よりも大きいとき。 |

### 図
入力値(s3)と上下限リミット値(s1)、(s2)の関係を示すグラフ。  
- 横軸は入力値(s3)+1、縦軸は出力値(d)+1。  
- 入力値が下限リミット値より小さい場合、出力値は下限リミット値に固定される。  
- 入力値が上限リミット値より大きい場合、出力値は上限リミット値に固定される。  
- 入力値が上下限リミット値の間にある場合、出力値は入力値に等しい。

|------|----------------------|
