### テキスト
#### 機能
・(d)で指定したデバイスから(n1)ダブルワード分のデータ範囲内で、右へ(n2)ダブルワード分シフトします。シフト後、(s)から(n2)点分を空きデータ部分に格納します。  
・(s)に定数が指定された場合、シフト後の最上位から(n2)点分は指定した値になります。  
・(n2)で指定された値が0の場合は無処理となります。

#### 例
(n1)=9、(n2)=4の場合のシフト動作を示しています。  
図では、(d)から始まる9ダブルワードのデータが右に4ダブルワード分シフトされ、空いた左側4ダブルワード分に(s)のデータがコピーされる様子が示されています。

#### エラー
| エラーコード(SD0) | 内容 |
|------------------|------|
| 2821H            | (s)から(n2)点分の範囲と、(d)から(n1)点分の範囲が重複しているとき。 |
| 3405H            | (n1)、(n2)に指定された値が(n1)<(n2)のとき。 |

#### 注意事項
(s)と(d)をビットの桁指定で指定する場合、(s)と(d)の桁数を合わせる必要がある。

### 図
(n1)=9、(n2)=4の場合のデータシフトの動作を示す図。  
- 上段に(s)から4ダブルワード分のデータ（11111111H、22222222H、33333333H、44444444H）が示されている。  
- 中段に(d)から9ダブルワード分のデータ（30FFFFH、1EEEEH、20000H、0H、1FFFFH、10000H、1EEEEH、7FFFFH、2AAAAH）が示されている。  
- 矢印で右に4ダブルワード分シフトされる動作を表現し、シフト後のデータ配置が下段に示されている。  
- シフト後、左側4ダブルワード分は(s)のデータがコピーされている。
