### テキスト・表

#### 時計データの減算
- 命令記号：DATE-、DATE-P
- 処理内容：時間データ（時、分、秒）を減算する。
- 図解：2つの時間データ（hour, minute, second）から減算し、結果の時間データを得るイメージ。

#### 時間データの変換（時分秒→秒）
- 命令記号：TIME2SEC、TIME2SECP
- 処理内容：時間データ（時、分、秒）を秒のデータに換算する。
- 図解：時間データ（hour, minute, second）を秒（second）に変換するイメージ。

#### 時間データの変換（秒→時分秒）
- 命令記号：SEC2TIME、SEC2TIMEP
- 処理内容：秒のデータを時間データ（時、分、秒）に換算する。
- 図解：秒（second）を時間データ（hour, minute, second）に変換するイメージ。

#### 日時データの変換
- 命令記号：
  - 秒→日時データ（年、月、日、時、分、秒）への変換：DATE2SEC、DATE2SECP、DATE2SEC_U、DATE2SECP_U
  - 日時データ→秒への変換：SEC2DATE、SEC2DATEP、SEC2DATE_U、SEC2DATEP_U
- 処理内容：
  - 日時データを秒のデータに換算する。
  - 秒のデータを日時データ（年、月、日、時、分、秒、曜日）に換算する。
- 図解：
  - 年、月、日、時、分、秒の日時データを秒に変換するイメージ。
  - 秒のデータを年、月、日、時、分、秒、曜日に変換するイメージ。

#### 日付比較
- 命令記号：LDDT=、ANDDT=、ORDT=、LDDT<>、ANDDT<>、ORDT<>、LDDT>、ANDDT>、ORDT>、LDDT<=、ANDDT<=、ORDT<=、LDDT<、ANDDT<、ORDT<、LDDT>=、ANDDT>=、ORDT>=
- 処理内容：指定した日付データ同士、または日付データと現在日付を比較する。
- 参照先：1086ページに詳細説明あり。

### まとめ
本ページは、時間・日時データの減算や変換、日付比較に関するCPU命令の一覧とその処理内容を表形式で整理し、理解しやすいように簡単な図で変換イメージを示しています。命令の使い方や詳細は別ページにて参照可能です。
