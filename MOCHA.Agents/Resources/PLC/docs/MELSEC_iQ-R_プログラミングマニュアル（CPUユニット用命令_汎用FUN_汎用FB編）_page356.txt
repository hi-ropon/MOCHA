### テキスト
#### nワードデータの1ワード左シフト（DSFL(P)）
指定したデバイスから(n)点分を左へ1ワードシフトさせます。シフト後の空きデータには0が格納されます。  
対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）です。

### 図
#### ラダー図
- (d)と(n)の2つのオペランドを持つ命令ブロックが示されています。

#### FBD/LD図
- EN（入力）、ENO（出力）、n、dの4つの端子を持つ命令ブロックが示されています。

| 命令   | 実行条件の波形例       |
|--------|------------------------|
| DSFL   | 立ち下がりエッジ       |
