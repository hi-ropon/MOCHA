### テキスト
#### 文字列操作命令
文字列の右側または左側からの取り出し、任意の位置からの取り出しや置換え、文字列の検索、挿入、削除に関する命令が説明されています。

#### 実数処理
浮動小数点数の比較命令について、単精度実数比較と倍精度実数比較の2種類が紹介されています。
