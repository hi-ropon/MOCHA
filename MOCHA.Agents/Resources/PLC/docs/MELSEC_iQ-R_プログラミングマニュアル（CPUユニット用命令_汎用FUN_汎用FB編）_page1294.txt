### テキスト
#### 28.2 SFC専用命令 移行条件ダミー出力
- 対応CPUはR00CPU、R01CPU、R02CPUで、バージョンによる制約はありません。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン12以降で使用可能。エンジニアリングツールはバージョン1.015R以降対応。
- プロセスCPU（プロセスモード）はファームウェアバージョン03以降で使用可能。エンジニアリングツールはバージョン1.020W以降対応。
- プロセスCPU（二重化モード）はファームウェアバージョン18以降で使用可能。エンジニアリングツールはバージョン1.050C以降対応。
- 本命令は移行条件を成立させるためのダミー出力である。

#### 参照
- 移行条件の詳細は「MELSEC iQ-R プログラミングマニュアル（プログラム設計編）」を参照すること。
