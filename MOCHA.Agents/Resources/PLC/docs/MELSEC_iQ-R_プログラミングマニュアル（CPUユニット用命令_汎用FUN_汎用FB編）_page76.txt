本ページはCPUユニット用命令の基本命令に関する説明で構成されています。ページ全体は主に表形式で整理されており、命令記号、処理内容、参照先の3つのカラムから成っています。テキスト部分は命令の説明文で、表部分は命令の一覧と詳細を示しています。グラフや図はなく、命令の処理内容を示す簡単な矢印付きのフローチャート的な図が各命令ごとに含まれています。

### テキスト
本ページはBCD4桁/8桁データや単精度実数、倍精度実数をBIN16ビットまたはBIN32ビットデータに変換する命令について説明しています。

### 表
#### BCD4桁/8桁→BIN16ビット/32ビットデータ変換命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| BIN / BINP | 指定したBCD4桁データ(0～9999)をBIN16ビットデータに変換する。 | 414ページ BIN(P) |
| DBIN / DBINP | 指定したBCD8桁データ(0～99999999)をBIN32ビットデータに変換する。 | 416ページ DBIN(P) |

処理内容の図解では、BCDからBINへの変換を示す矢印があり、(s)がBCD、(d)がBINを表しています。

#### 単精度実数→符号付きBIN16ビット/32ビットデータ変換命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| FLT2INT / FLT2INTP | 指定した単精度実数(-32768～32767)を符号付きBIN16ビットデータに変換する。 | 418ページ FLT2INT(P) |
| FLT2DINT / FLT2DINTP | 指定した単精度実数(-2147483648～2147483647)を符号付きBIN32ビットデータに変換する。 | 422ページ FLT2DINT(P) |

処理内容の図解では、(s)が実数、(d)がBINデータを示し、変換の流れを矢印で表しています。

#### 単精度実数→符号なしBIN16ビット/32ビットデータ変換命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| FLT2UINT / FLT2UINTP | 指定した単精度実数(0～65535)を符号なしBIN16ビットデータに変換する。 | 420ページ FLT2UINT(P) |
| FLT2UDINT / FLT2UDINTP | 指定した単精度実数(0～4294967295)を符号なしBIN32ビットデータに変換する。 | 424ページ FLT2UDINT(P) |

こちらも同様に実数からBINデータへの変換を示す矢印付きの図解があります。

#### 倍精度実数→符号付きBIN16ビット/32ビットデータ変換命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBL2INT / DBL2INTP | 指定した倍精度実数(-32768～32767)を符号付きBIN16ビットデータに変換する。 | 426ページ DBL2INT(P) |
| DBL2DINT / DBL2DINTP | 指定した倍精度実数(-2147483648～2147483647)を符号付きBIN32ビットデータに変換する。 | 430ページ DBL2DINT(P) |

処理内容の図解は複数の(s)+1、(s)+2などの実数入力からBINデータへの変換を示しています。

### 図
各命令の処理内容は簡単なフローチャート形式で示されており、(s)が入力データ（BCDや実数）、(d)が出力データ（BIN16ビットまたはBIN32ビット）を表しています。矢印で変換の流れを示し、符号付き/符号なしの違いやデータサイズの違いを視覚的に理解できるようになっています。
