### テキスト
#### 算術演算ファンクション
算術演算ファンクションは加算、乗算、減算、除算、剰余、べき乗、代入の各処理を行います。

#### ビットシフトファンクション
ビットシフトファンクションはビットの左シフト・右シフトおよび左ローテーション・右ローテーションの処理を行います。
