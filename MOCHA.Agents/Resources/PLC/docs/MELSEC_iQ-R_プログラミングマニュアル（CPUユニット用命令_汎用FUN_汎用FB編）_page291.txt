### テキスト
#### 機能
- (d)で指定されたデバイス（BIN16ビットデータ）を-1します。
- (d)で指定されたデバイスの内容が-32768のときにDEC(P)命令を実行すると、32767が(d)で指定されたデバイスに格納されます。（符号付き指定の場合）
- (d)で指定されたデバイスの内容が0のときにDEC(P)_U命令を実行すると、65535が(d)で指定されたデバイスに格納されます。（符号なし指定の場合）

#### エラー
- 演算エラーはありません。

### 図
- ビットデータの変換例として、16ビットのビット列（b15～b0）で5678（BIN）が-1されて5677（BIN）になる様子を示しています。
