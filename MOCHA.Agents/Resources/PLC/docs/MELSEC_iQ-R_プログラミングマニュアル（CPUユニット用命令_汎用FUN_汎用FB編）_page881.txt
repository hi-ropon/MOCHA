### テキスト
#### 符号付きBIN16ビットデータ→単精度実数変換
命令INT2FLT(P)は、符号付きBIN16ビットデータを単精度実数に変換します。

#### 実行条件
- INT2FLT命令は立ち上がりエッジで実行
- INT2FLTP命令は立ち下がりエッジで実行
※バージョン1.035M以降のエンジニアリングツールで対応

#### 機能
(s)で指定した符号付きBIN16ビットデータを単精度実数に変換し、(d)に格納します。

#### エラー
演算エラーはありません。

### 図
#### ラダー図
- INT2FLT命令のラダー図例。入力(s)から単精度実数(d)へ変換し出力。

#### FBD/LD図
- EN（実行条件）入力に対し、s（入力データ）からd（出力データ）へ変換するブロック図。

#### ビットデータ変換イメージ
- (s)の16ビット符号付きBINデータ（例：-1234）がFB2EH（16進数）で表現されている。
- 変換後の単精度実数は(d)のD101、D100に格納され、16進数でC49AH、4000Hとなる（-1234.0を表す）。

#### 命令使用例ラダー図
- SM402がONのとき、MOVP命令でK-1234をD0にセット。
- M0がONのとき、INT2FLT命令でD0の整数値を単精度実数に変換しD100に格納。
