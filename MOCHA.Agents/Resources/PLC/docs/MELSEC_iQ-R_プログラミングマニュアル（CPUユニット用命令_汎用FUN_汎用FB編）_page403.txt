### テキスト
#### 機能
- 指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、右へ(n2)点分シフトする。
- 最上位から(n2)点分は0になる。
- (n1)または(n2)で指定した値が0のときは無処理となる。
- (n1)≦(n2)の場合は、指定デバイスから(n1)点分のデータはすべて0になる。

#### 例
- (n1)=4、(n2)=2の場合のデータシフトの動作を図で示している。
- 元のデータは(d)+15から(d)までの範囲に倍精度実数が並んでいる。
- 右へ2点シフトした結果、最上位2点((d)+15、(d)+12)は0になり、残りのデータは2点分右にずれている。

#### エラー
- 演算エラーはありません。

### 図（例の説明）
- 上段は元のデータ配置を示し、(d)+15から(d)までの倍精度実数が並んでいる。
- 下段は右へ2点シフトした後のデータ配置を示し、最上位2点は0に置き換わり、残りのデータが右に2点分ずれている様子を表している。
