### テキスト
#### 機能
(s)で指定されたデバイスから(n)点のBIN32ビットデータの中で最大値を検索し、(d)、(d)+1で指定されたデバイスに最大値を格納します。  
具体的には、(s)で指定したデバイスから検索し、最初に検出した最大値に対応するデバイス番号を(s)から何点目かを(d)+2に格納し、最大値の個数を(d)+3に格納します。

#### エラー
演算エラーはありません。

### 図
ビットデータの例として、(s)+1から(s)+9までのデバイスにそれぞれ以下のBIN32値が格納されています。  
- (s)+1, (s): 54321000 (BIN)  
- (s)+3, (s)+2: 4321000 (BIN)  
- (s)+5, (s)+4: 3254000 (BIN)  
- (s)+7, (s)+6: 54321000 (BIN)  
- (s)+9, (s)+8: 12345678 (BIN)  

これらの中で最大値は54321000 (BIN)であり、(d)と(d)+1にこの最大値が格納されます。  
さらに、(d)+2には最大値が最初に検出された位置（1）が、(d)+3には最大値の個数（2）が格納されます。

### 用語説明
- (d), (d)+1: 最大値を格納するデバイス  
- (d)+2: 最大値の位置（何点目か）  
- (d)+3: 最大値の個数
</contents>
