### テキスト
#### 倍精度実数最大値検索 EDMAX(P)
倍精度実数のブロックデータの最大値を検索する命令です。

##### ■説明
- EDMAX命令はST命令やFBD/LD命令では対応していません。代わりに汎用関数のMAXを使用してください（1457ページ参照）。
- 実行条件はEDMAXが立ち上がり、EDMAXPが立下りです。

### 図
- **ラダー命令の構成**  
  ラダー図では、(s)、(d)、(n)の3つのオペランドがあり、ST命令形式で「ENO:=EDMAXP(EN,s,n,d);」と記述されています。

- **FBD/LD命令の構成**  
  ブロック図では、EN（入力）、ENO（出力）、s、d、nの5つの端子がある四角形の命令ブロックとして示されています。

### 表
#### 実行条件
| 命令   | 実行条件           |
|--------|--------------------|
| EDMAX  | 立ち上がり信号     |
| EDMAXP | 立ち下がり信号     |

| オペランド | 内容                                                                                     | 範囲                          | データ型       | データ型(ラベル)          |
|------------|------------------------------------------------------------------------------------------|-------------------------------|----------------|---------------------------|
| (s)        | 最大値を検索する倍精度実数または、倍精度実数が格納されている先頭デバイス                 | 0, 2^-1022 ≤ |s| < 2^1024     | 倍精度実数     | ANYREAL_64*1              |
| (d)        | 検索結果を格納する先頭デバイス<br>(d)～(d)+3:最大値<br>(d)+4:位置<br>(d)+5:個数           | —                             | 倍精度実数     | —*2 (ANY_REAL_64_ARRAY)   |
| (n)        | 倍精度実数ブロックデータの点数                                                         | 0～65535                      | 符号なしBIN16ビット | ANY16                     |
| EN         | 実行条件                                                                               | —                             | ビット         | BOOL                      |
| ENO        | 実行結果                                                                               | —                             | ビット         | BOOL                      |

*1 ラベルで設定する場合は、動作に必要な領域が確保できるよう配列を定義し、その配列ラベルの要素を指定してください。  
*2 使用するプログラム言語にかかわらず、デバイスで指定します。ラベルは指定しないでください。
