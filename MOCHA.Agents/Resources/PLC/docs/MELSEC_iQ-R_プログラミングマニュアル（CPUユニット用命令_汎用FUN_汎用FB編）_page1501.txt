### テキスト
#### 機能
- 演算処理  
  (s)がONすると、(d)を1スキャンのみONする動作を行う。

#### 演算結果
1. EN/ENOなしファンクションブロック  
   演算処理を実行し、(d)から演算出力値が出力される。  
   ・タイミングチャートにより、(s)の立ち上がりで(d)がONし、次スキャンで(d)がOFFになる動作を示す。  
   (1) (s)の立ち上がりで(d)をON  
   (2) 次スキャンで(d)をOFF

2. EN/ENO付きファンクションブロック  
   実行条件と演算結果は以下の表の通り。  

| 実行条件 EN | 演算結果 ENO | (d) |
|-------------|--------------|-----|
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE | 前回出力値 |

- タイミングチャートにより、ENがONかつ(s)の立ち上がりで(d)がONし、次スキャンで(d)がOFFになる動作を示す。  
- EN=OFFの場合は(d)は前回のスキャンの出力を保持する。  
- (1) EN=ONかつ(s)の立ち上がりで(d)をON  
- (2) 次スキャンで(d)をOFF  
- (3) EN=OFFの場合は(d)は前回の出力を保持

#### エラー
- 演算エラーはない。

### 表
#### 実行条件と演算結果の対応表（EN/ENO付きファンクションブロック）

| 実行条件 EN | 演算結果 ENO | (d)の動作内容 |
|-------------|--------------|---------------|
| TRUE（演算実行） | TRUE | 演算出力値を出力 |
| FALSE（演算停止） | FALSE | 前回の出力値を保持 |

### 図（グラフ）
#### タイミングチャート
- EN/ENOなしファンクションブロック  
  (s)のONタイミングに合わせて(d)が1スキャンだけONし、その後OFFになる波形を示す。  
- EN/ENO付きファンクションブロック  
  ENがONかつ(s)の立ち上がりで(d)がONし、次スキャンで(d)がOFFになる波形。  
  ENがOFFの場合は(d)の出力が前回の値を保持する動作を示す。  
- 各信号（EN、(s)、ENO、(d)）のON/OFF状態が時間軸に沿って示されている。
