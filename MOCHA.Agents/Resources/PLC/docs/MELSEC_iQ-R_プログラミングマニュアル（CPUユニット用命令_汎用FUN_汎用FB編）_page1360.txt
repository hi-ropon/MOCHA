### テキスト
#### 32.37 BCD型→INT型変換
関数名：BCD_TO_INT(_E)  
BCD型データをINT型データへ変換します。

#### ラダー、FBD/LDのシンボル
- EN/ENOなしの場合とEN/ENO付きの場合のシンボル図が示されています。
- ST言語での使用例も記載されています。

#### 設定データ
##### 内容、種別、データ型
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | WORD |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | INT |

#### 機能
##### 演算処理
- 入力されたBCD型(WORD型)のデータをINT型のデータに変換し、出力します。
- 具体的には、(s)に入力されたBCDデータ（例：9999H）をINT型の9999に変換して(d)から出力します。

### 図
- BCD(WORD)のビット構成とINTのビット構成の対応を示す図があります。
- 9999HのBCDデータの各桁（千の位、百の位、十の位、一の位）がどのビットに対応しているかを示し、INT型のビットに変換される様子をビット単位で説明しています。
- 各桁のビット位置は以下の通り：
  1. 千の位
  2. 百の位
  3. 十の位
  4. 一の位
- (s)への入力値はWORD型のデータ値で0H～9999H（各桁は0～9の範囲）であることが明記されています。
