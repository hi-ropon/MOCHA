### テキスト
#### 他形式プロジェクトからの置換え
GX Works2形式プロジェクトをGX Works3のプロジェクトとして使用する場合、特定の命令は自動的に互換用命令やファンクション/ファンクションブロック（FB）に置き換えられます。  
- 互換用命令/ファンクションはGX Works3のバージョンが「1.047Z」以降で置き換えられる。  
- 互換用ファンクションブロックはGX Works3のバージョンが「1.055H」以降で置き換えられる。  
- 引数のデータ型や順番はGX Works2と同じ。

#### 互換用命令（基本命令/応用命令）
ST（インラインST含む）、構造化ラダー/FBDのプログラムで使用されている基本命令や応用命令が互換用命令に置き換えられます。
