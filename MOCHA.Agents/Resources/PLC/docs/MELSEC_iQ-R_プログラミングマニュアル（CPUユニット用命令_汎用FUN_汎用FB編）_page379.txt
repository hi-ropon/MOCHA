### テキスト
#### 機能
- 指定したデバイスから(n1)ワード分のデータ範囲内で、右へ(n2)ワード分シフトする。
- 最上位から(n2)ワード分は0Hになる。
- (n1)または(n2)で指定した値が0Hの場合は無処理となる。
- (n1)が(n2)以下の場合は、指定デバイスから(n1)ワード分のデータはすべて0Hになる。

#### エラー
- 演算エラーは発生しない。

### 図
#### 例：(n1)=9、(n2)=4の場合
- 元のデータは(d)から(d)+8までの9ワード分が示されている。
- 右へ4ワード分シフトすると、(d)から(d)+4までは元のデータが4ワード分ずつ右にずれ、(d)+5から(d)+8までは0Hになる。
- 図中の各ワードには16進数の値が記載されており、シフト前後のデータの変化が視覚的に示されている。
