### 表
#### 命令処理時間一覧
この表は各命令名に対して、条件別の処理時間（μs）をCPUユニットごとに最小値と最大値で示しています。

- **命令名**: 各種命令の名称（例: PALERT(P), PABORT, LD$= など）
- **条件**: 命令実行時の条件（例: 導通時/非導通時、無実行時、実行時など）
- **処理時間(μs)**:
  - R00CPU, R01CPU
  - R02CPU
  - 左記以外のCPUユニット
 それぞれのCPUユニットについて最小値と最大値が記載されています。

##### 主なポイント
- LD$=、LD$<>、LD$> などの命令は導通時/非導通時の条件で処理時間が示されており、R00CPUやR01CPUでは最小3.1μs、最大6.0μs程度。
- AND$=、AND$<>、AND$> などの論理演算命令は無実行時と実行時の処理時間が分けて記載されており、無実行時は0.18816μs程度、実行時は3.1～6.0μs程度。
- OR$=、OR$<>、OR$> なども同様に無実行時と実行時の処理時間が記載。
- $+ (s) (d)、$+ (s1) (s2) (d) などの加算命令は実行時の処理時間が4.0～9.7μs程度。
- $MOV、$MOV_WS は文字数（0文字、32文字）によって処理時間が異なり、最大で15.7μs程度。
- BINDA、BINDA_U、DBINDA、DBINDA_U、BINHA、DBINHA などの命令はパラメータ(s)の値によって処理時間が異なり、最小4.1μs～最大8.9μs程度。
- STR命令は条件なしで4.3～8.8μs程度。

この表はCPUの種類や命令の条件によって処理時間が異なることを詳細に示しており、性能評価や最適化に役立つ情報となっています。
