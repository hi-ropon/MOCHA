### テキスト
#### BCD型SIN演算（BSIN(P)）
BCD値で指定した角度のSIN（正弦）値を演算する命令について説明しています。

- BSIN命令はRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）で使用可能です。
- SIN値の演算は、指定した角度（BCD値）に対して行います。

#### 実行条件
- BSIN命令は立ち上がりエッジで実行されます。
- BSINP命令は立ち下がりエッジで実行されます。

#### 機能
- (s)で指定した角度のSIN値を演算し、(d)で指定したデバイスに演算結果の符号を格納します。
- (d)+1に整数部、(d)+2に小数部を格納します。
- 角度は0～360度（DEG.単位）をBCD値で設定します。
- 符号は演算結果が正のとき0、負のとき1を格納します。
- 演算結果はBCD値で-1.000～1.000の範囲内の値です。

### 図
