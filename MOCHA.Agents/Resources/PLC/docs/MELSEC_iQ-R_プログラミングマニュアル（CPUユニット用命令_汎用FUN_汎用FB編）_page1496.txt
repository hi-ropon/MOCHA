### テキスト
#### 二安定ファンクションブロック(セット優先)
SR(_E)という名称の二安定ファンクションブロックについて説明しています。このブロックは2つの入力値を判別し、出力として1(TRUE)または0(FALSE)を返します。

### 図
- **ラダー、FBD/LD**
  - EN/ENOなしの場合のブロック図：s1（セット入力）、s2（リセット入力）、d（出力）を示す。
  - EN/ENO付きの場合のブロック図：EN（実行条件入力）、ENO（実行状態出力）、s1、s2、dの接続を示す。

- **ST言語**
  - EN/ENOなし：インスタンス名の例としてS1:=s1,R:=s2,Q1:=dを示す。
  - EN/ENO付き：EN:=en,ENO:=eno,S1:=s1,R:=s2,Q1:=dのインスタンス名例を示す。

### 表
#### 設定データ（内容、種別、データ型）

| 引数   | 内容                               | 種別     | データ型 |
|--------|----------------------------------|----------|----------|
| EN     | 実行条件(TRUE:実行、FALSE:停止)  | 入力変数 | BOOL     |
| s1(S1) | セット指令                       | 入力変数 | BOOL     |
| s2(R)  | リセット指令                     | 入力変数 | BOOL     |
| ENO    | 出力状態(TRUE:正常、FALSE:異常または停止) | 出力変数 | BOOL     |
| d(Q1)  | 出力                             | 出力変数 | BOOL     |

この表は、各引数の役割とデータ型を明確に示しており、二安定ファンクションブロックの入出力仕様を理解するための重要な情報となっています。
