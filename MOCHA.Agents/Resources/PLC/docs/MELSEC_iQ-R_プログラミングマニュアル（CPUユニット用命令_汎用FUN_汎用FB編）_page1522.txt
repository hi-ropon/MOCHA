### テキスト
#### TIMER_100_FB_M
- (s1)の実行条件がONになると計測を開始し、(s3)×100msから計測開始、(s2)×100msまで計測値が到達すると(d2)がONし、現在の計測値は(d1)に出力される。
- (s1)がOFFになると現在値は0となり、(d2)もOFFになる。
- 低速タイマの計測単位（タイマ時限設定）をエンジニアリングツールのデフォルト値から変更した場合、変換/全変換時にワーニングが発生する。
- (s2)には0～32767の値が指定可能。
- (s3)には-32768～32767の値が指定可能。ただし負の値を指定した場合、0が初期値となる。

#### TIMER_HIGH_FB_M
- 計測単位が0.1～100msの高速タイマで、(s1)の実行条件がONになると計測を開始する。
- (s3)×0.1～100ms（可変、パラメータで設定）から計測開始し、(s2)×0.1～100msまで計測値が到達すると(d2)がONし、現在の計測値は(d1)に出力される。
- (s1)がOFFになると現在値は0となり、(d2)もOFFになる。
- 高速タイマの計測単位はデフォルト10msで、0.01ms～100msの範囲で変更可能。
- (s2)には0～32767の値が指定可能。
- (s3)には-32768～32767の値が指定可能。ただし負の値を指定した場合、0が初期値となる。

### 図
#### TIMER_100_FB_M 回路例
- Coil、Preset、ValueIn、ValueOut、Statusの5つの端子を持つブロック図。
- M0、10、1が入力、D10、M10が出力として接続されている。

#### TIMER_100_FB_M タイミングチャート
- M0がONになるとタイマが動作開始し、D10が1となり900ms後にM10がONになる。
- M0がOFFになるとタイマ動作停止。

#### TIMER_HIGH_FB_M 回路例
- TIMER_100_FB_Mと同様の構成で、Coil、Preset、ValueIn、ValueOut、Statusの端子を持つ。

#### TIMER_HIGH_FB_M タイミングチャート
- M0がONになるとタイマが動作開始し、D10が1となり90ms後にM10がONになる。
- M0がOFFになるとタイマ動作停止。
