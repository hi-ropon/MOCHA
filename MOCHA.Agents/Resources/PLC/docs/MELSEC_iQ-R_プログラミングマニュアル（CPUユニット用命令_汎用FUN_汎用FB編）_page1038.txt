### テキスト
・各値のデータサイズは、各フィールドのデータ型のサイズに従うことを説明しています。

### 表
#### フィールドのデータ型とデータサイズ
| フィールドのデータ型 | データサイズ（単位：ワード） |
|----------------------|-----------------------------|
| BOOL                 | 1                           |
| WORD                 | 1                           |
| DWORD                | 2                           |
| INT                  | 1                           |
| DINT                 | 2                           |
| REAL                 | 2                           |
| LREAL                | 4                           |
| STRING               | 文字数が偶数の場合：(文字数÷2)+1<br>文字数が奇数の場合：文字数÷2の切り上げ<br>【例】<br>STRING: 32の場合：(32÷2)+1=17<br>STRING: 15の場合：(15÷2)=7.5の切り上げ→8 |
| WSTRING              | 文字数+1<br>【例】<br>WSTRING: 32の場合：32+1=33 |

### 説明
- STRING型は文字数に応じてデータサイズが変わり、偶数文字の場合は文字数を2で割って1を足した値、奇数文字の場合は2で割った値を切り上げた値となります。
- WSTRING型は文字数に1を足したサイズとなります。
