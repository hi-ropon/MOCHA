### テキスト
#### 機能
- (s)で指定されたBIN32ビットデータを、(d)で指定されたデバイスに転送する。
- (s)がビットデバイスの場合、桁指定で指定されたビットが転送対象となる。転送対象外のビットは0として転送される。

#### エラー
- 演算エラーは存在しない。

### 図
- 上段の図は、(s)のビットデータの例を示している。b27からb0までのビットが並び、特定のビットが1または0で示されている。
- 下段の図は、(d)のデバイスに転送されたビットデータを示している。b31からb0までのビットがあり、(s)のビットデータが対応する位置に転送されている。b28からb31の上位ビットは0で埋められている。
- 図中の注釈では、(s)が32ビット未満の場合、0が指定されているものとして転送されることが説明されている。
