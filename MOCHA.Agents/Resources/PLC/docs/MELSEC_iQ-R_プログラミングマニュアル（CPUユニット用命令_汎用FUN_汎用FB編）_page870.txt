### テキスト・表
#### エラーコード(SD0)と内容

| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H | (s)で指定したデータが-0、非正規化数、非数、±∞のとき。<br>(d)で指定したデータが-0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が下記範囲を超えるとき。（オーバーフローが発生したとき。）<br>|(d)| < 2^1024 |

- 「3402H」は、sまたはdで指定したデータが-0、非正規化数、非数、または無限大（±∞）の場合に発生するエラーです。
- 「3403H」は、dの値が指定された範囲（絶対値が2の1024乗未満）を超えた場合、すなわちオーバーフローが発生した場合に発生するエラーです。

ページ下部には「11 実数処理」「11.1 浮動小数点命令」という章タイトルが記載されています。
