### テキスト
#### 機能
(s)で指定されたデバイスから(n)点までのBIN16ビットデータの中から最小値を検索し、(d)で指定されたデバイスに最小値を格納します。また、(s)で指定されたデバイスから検索し、最初に検出した最小値が格納されているデバイス番号が(s)から何点目かを(d)+1に格納し、最小値の個数を(d)+2に格納します。

#### エラー
演算エラーはありません。

### 図（説明）
左側に複数のデバイス番号とそのBIN16ビットデータの値が縦に並んでいます。例として、5015、6192、5571、5015、5571の値が示されています。これらは(s)から(s)+(n)-1までの範囲を表しています。

矢印の右側には、(d)で指定されたデバイスに最小値5015が格納され、その下の(d)+1には位置「1」、(d)+2には個数「2」が格納されていることが示されています。

図の下には各記号の意味が記載されています。
- (d): 最小値
- (d)+1: 位置
- (d)+2: 個数
