### テキスト
#### 符号なしBIN32ビットデータ→符号なしBIN16ビットデータ変換
命令名は「UDINT2UINT(P)」で、指定した符号なしBIN32ビットデータを符号なしBIN16ビットデータに変換します。

※対応はエンジニアリングツールのバージョン"1.035M"以降。

#### 実行条件
- UDINT2UINT命令はパルス状の実行条件
- UDINT2UINTP命令は立ち上がりエッジで実行

### 図
- ラダー図：命令のラダー表現として、入力(s)と出力(d)の接続イメージ
- FBD/LD図：EN（有効化入力）、ENO（有効化出力）、s（入力データ）、d（出力データ）の接続イメージ
