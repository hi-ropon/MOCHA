### テキスト
#### 機能
- (s1)で指定されたデバイスから(n)点分の内容と、(s2)で指定されたデバイスから(n)点分の内容の論理積を求め、その結果を(d)で指定されたデバイス以降に格納する。
- (s2)には-32768～32767の符号付きBIN16ビットの定数も指定可能。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 2821H | (s1)から(n)点分のデバイス範囲と、(d)から(n)点分のデバイス範囲の一部が重複している場合。ただし、(s1)と(d)に同一デバイスを指定した場合は除く。<br>(s2)から(n)点分のデバイス範囲と、(d)から(n)点分のデバイス範囲の一部が重複している場合。ただし、(s2)と(d)に同一デバイスを指定した場合は除く。 |

### 図（ビット列の説明）
- (s1)と(s2)の各ビット列が縦に(n)点分並び、それぞれのビットごとにAND演算を行う様子を示している。
- 上段の図は(s1)と(s2)のビット列の例を示し、AND演算を行った結果が(d)に格納される。
- 下段の図は(s2)に定数（16ビット符号付きBIN）を指定した場合の例で、(s1)のビット列と定数のビット列のAND演算結果が(d)に格納される。
- ビット列はb15からb0までの16ビットで表現されており、各ビットの0/1の状態が具体的に示されている。

以上より、本ページは論理積AND演算の機能説明、動作例（ビット列による具体例）、および関連するエラーコードの説明で構成されています。
