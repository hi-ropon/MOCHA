### テキスト
#### 35.1 nビット左シフト
SHL(_E)命令は、入力値をnビット分左へシフトして出力します。

#### 命令の説明
- 入力値を指定されたビット数だけ左にシフトし、その結果を出力します。

### 図
- ラダー、FBD/LD言語でのシフト命令のブロック図が2種類示されています。
  - EN/ENOなしの場合：入力s、シフト数n、出力dの3つの端子がある単純なブロック。
  - EN/ENO付きの場合：EN（実行条件入力）、ENO（実行状態出力）が追加されたブロック。
- ST言語での使用例も示されており、
  - EN/ENOなし：d := SHL(s, n);
  - EN/ENO付き：d := SHL_E(EN, ENO, s, n);
 という構文で記述します。

### 表
#### 設定データ（内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力値 | 入力変数 | ANY_BIT |
| n(N) | シフトビット数指定 | 入力変数 | ANY_BIT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力値 | 出力変数 | ANY_BIT |

以上の内容により、SHL命令はビット単位での左シフトを行い、実行条件の有無に応じて動作を制御できることがわかります。
