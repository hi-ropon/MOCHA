### 表
#### 命令処理時間一覧表
本表は各命令の処理時間を示しており、以下の項目で構成されています。

- 命令名：各命令の名称
- 条件：命令実行時の条件やパラメータの説明
- 処理時間(μs)：CPUユニット別に最小値と最大値が記載されている
  - R00CPU、R01CPU
  - R02CPU
  - 左記以外のCPUユニット

##### 主な内容説明
- TCPおよびUDPのデータ量（最小1バイト、最大10238バイト）に応じた処理時間が命令ごとに示されている（例：SP.SOCRCV、S.SOCRCVS、SP.SOCSNDなど）。
- モード切替（通常モード⇔固定長モード）やデータ量（最小1ワード、最大5120ワード）に関する処理時間も記載（例：SP.SOCRMODE、S.SOCDATA）。
- FTP関連命令（SP.FTPPUT、SP.FTPGET）はファイル名＋サーバパス文字列の長さ（32文字、64文字）やユニットパラメータ指定の有無による処理時間の違いが示されている。
- その他、ファイル名指定の命令（PSTOP、POFF、PSCAN）やループ回数による処理時間変化（S.PIDINIT、S.PIDCONT）も記載。

##### 処理時間の特徴
- 同一命令でも条件やCPUユニットによって処理時間に差がある。
- データ量が多いほど処理時間が長くなる傾向がある。
- FTP関連命令はファイル名やパスの文字数によって処理時間が変動する。

以上の内容から、本表は命令ごとの処理時間の目安をCPUユニット別に詳細に示した技術資料であることがわかります。
</contents>
