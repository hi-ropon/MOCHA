### テキスト
#### BIN16ビットデータビットチェック
指定したデバイスのBIN16ビットデータのうち、1になっているビットの総数を格納する命令です。

#### 実行条件
| 命令 | 実行条件 |
| --- | --- |
| SUM | 立ち上がりエッジ |
| SUMP | 立ち下がりエッジ |

#### 機能
(s)で指定されたデバイスのBIN16ビットデータのうち、1になっているビットの総数を(d)で指定されたデバイスに格納します。  
ビットの総数は2進数（BIN）で格納されます。例として、8個のビットが1の場合は「00001000」として格納されます。

#### エラー
演算エラーはありません。

### 図
- ラダー図：SUM命令のラダー表現。入力(s)、出力(d)が示されている。
- FBD/LD図：SUM命令のFBD/LD表現。EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）が示されている。
- ビット配置例：  
  (s)のビット列の中で1になっているビットをカウントし、その総数を(d)にBIN形式で格納する様子を示す。
