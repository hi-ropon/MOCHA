### テキスト
#### 設定データ
##### 内容、範囲、データ型
オペランドごとに内容、範囲、データ型、データ型（ラベル）が表形式で示されています。

- (d): タイマデバイスまたはタイマ型ラベル、範囲なし、ビット型、ANY_BOOLラベル
- Coil: 実行条件、ビット型、BOOLラベル
- Value: タイマの設定値、範囲0～32767、符号なしBIN16ビット、ANY16ラベル
- EN: 実行条件、ビット型、BOOLラベル
- ENO: 実行結果、ビット型、BOOLラベル

注記として、ST、FBD、LDの場合はデータ型がANY_INTになること、SIL2プロセスCPUおよび安全CPUの安全プログラムでは安全デバイスおよび安全ラベルのみ使用可能であることが記載されています。

##### 使用可能デバイス
オペランドごとにビット、ワード、ダブルワード、間接指定、定数、その他のデバイスの使用可否が表形式で示されています。

- 一般的なビットデバイス（X, Y, M, L, SMなど）は(d)では使用不可、CoilはT, STのみ使用可、ValueはT, ST, Cは不可など細かい制限あり。
- SIL2プロセスCPUおよび安全CPUでは特定の安全デバイス（SA¥X, SA¥Yなど）と定数のみ使用可能。
- 注釈で使用可能なデバイスの詳細制限が補足されています。

#### 機能
- OUT命令の演算結果がONのとき、(d)（ST, FBD, LDの場合はCoil）で指定したタイマのコイルがONし設定値まで計測。
- タイムアップ（現在値＝設定値）するとa接点は導通、b接点は非導通になる。
- OUT命令の演算結果がON→OFFに変化した場合の動作説明。

##### タイマの種類と動作
| タイマの種類 | タイマコイル | タイマの現在値 | タイムアップ前 a接点 | タイムアップ前 b接点 | タイムアップ後 a接点 | タイムアップ後 b接点 |
|--------------|--------------|----------------|----------------------|----------------------|----------------------|----------------------|
| 低速タイマ   | OFF          | 0              | 非導通               | 導通                 | 非導通               | 導通                 |
| 高速タイマ   |              |                |                      |                      |                      |                      |
| 低速積算タイマ| OFF          | 現在値を保持   | 非導通               | 導通                 | 導通                 | 非導通               |
| 高速積算タイマ|              |                |                      |                      |                      |                      |

- タイムアップ後、積算タイマの現在値クリアと接点OFFはRST命令で行う。
- 設定値が0のときはOUT命令実行時にタイムアップ。
- OUT命令実行時に以下の処理を行う：
  - OUT命令、OUTH T命令、OUT ST命令、OUTH ST命令のコイルON/OFF
  - 上記命令の接点ON/OFF
  - 上記命令の現在値の変更
- OUT命令群がON中にJMP命令などでOUT命令を飛ばした場合、現在値更新および接点ON/OFFは行わない。
- 同一OUT命令群を同一スキャン内で2回以上実行した場合は、実行回数分の現在値を更新。
