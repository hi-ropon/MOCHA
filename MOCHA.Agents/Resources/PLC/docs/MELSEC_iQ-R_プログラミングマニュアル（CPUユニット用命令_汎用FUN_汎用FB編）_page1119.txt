### テキスト
#### FROM(P)
- (U/H)で指定したユニット、または他号機CPUユニット内のバッファメモリの(s)で指定したアドレスから、(n)ワード分のデータを読み出し、(d)で指定したデバイス以降に格納します。
- ユニットからのワードデータリードの説明があり、ユニットバッファメモリからCPUユニットへ(n)ワード分のデータ転送を行います。
- 他号機CPUユニットからのワードデータリードも可能で、他号機のCPUバッファメモリからデバイス/ラベルメモリへ(n)ワード分のデータ転送を行います。
- 読出しデータ(n)が0の場合は無処理となります。
- 命令実行時に対象ユニットの異常や処理中などでアクセスできない場合は無処理となります。

#### DFROM(P)
- (U/H)で指定したユニット、または他号機CPUユニット内のバッファメモリの(s)で指定したアドレスから、(n)×2ワード分のデータを読み出し、(d)で指定したデバイス以降に格納します。
- ユニットからのダブルワードデータリードの説明があり、ユニットバッファメモリからCPUユニットへ(n)×2ワード分のデータ転送を行います。
- 他号機CPUユニットからのダブルワードデータリードも可能で、他号機のCPUバッファメモリからデバイス/ラベルメモリへ(n)×2ワード分のデータ転送を行います。
- 読出しデータ(n)が0の場合は無処理となります。
- 命令実行時に対象ユニットの異常や処理中などでアクセスできない場合は無処理となります。

### 図
- FROM(P)命令の図は、ユニットバッファメモリの(s)アドレスから(n)ワード分のデータがCPUユニットの(d)アドレス以降に転送される様子を示すブロック図。
- 他号機CPUユニットからのワードデータリードの図は、他号機CPUバッファメモリの(s)アドレスから(n)ワード分のデータがデバイス/ラベルメモリの(d)アドレス以降に転送される様子を示すブロック図。
- DFROM(P)命令の図は、ユニットバッファメモリの(s)アドレスから(n)×2ワード分のデータがCPUユニットの(d)アドレス以降に転送される様子を示すブロック図。
- 他号機CPUユニットからのダブルワードデータリードの図は、他号機CPUバッファメモリの(s)アドレスから(n)×2ワード分のデータがデバイス/ラベルメモリの(d)アドレス以降に転送される様子を示すブロック図。

以上より、FROM(P)はワード単位のデータ読み出し、DFROM(P)はダブルワード単位のデータ読み出しを行う命令であることが理解できます。
