### 表
#### GX Works2命令名と互換用命令名一覧
- 表は4つの列で構成されている。
  1. **GX Works2命令名**: 元の命令名が記載されている。
  2. **互換用命令名**: 互換性のある命令名が記載されている。
  3. **使用プログラム**: ST（Structured Text）とFBD（Function Block Diagram）のどちらで使用可能かを○で示している。
  4. **参照先**: 詳細説明が記載されているマニュアルのページ番号と該当箇所が記載されている。

- 命令名はDMINUSP_M、DMINUSP_3_M、DMULTI_3_Mなど多くの種類があり、それぞれに対応する互換用命令名が設定されている。
- 使用プログラム欄はほとんどの命令でST、FBDともに○が付いており、両方のプログラムで使用可能であることを示している。
- 参照先はページ番号と該当命令の説明ページが記載されており、例えば「244ページ D-(P)(_U) [オペランドが2個の場合]」など具体的な説明箇所が示されている。
- 一部の命令は参照先が「MELSEC iQ-R プログラミングマニュアル(ユニット専用命令編)」と記載されている。

### テキスト
- ページ下部に「付録」「付6 他形式プロジェクトからの置換え」と記載されている。

以上より、本ページはGX Works2の命令とその互換命令の対応表であり、使用可能なプログラム形式と詳細説明の参照先を一覧化した資料であることがわかる。
