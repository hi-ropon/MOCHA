### テキスト
#### ブロック再開
指定ブロックの一時停止を解除し、停止ステップから実行を再開させる命令について説明しています。

#### ステップ起動
指定ステップを活性化させる命令について説明しています。

#### ステップ終了
指定ステップを非活性にする命令について説明しています。

#### 対象ブロック切替え
SFC制御命令の対象ブロックを指定する命令について説明しています。

#### SFC専用命令
移行条件ダミー出力に関する命令について説明しています。

### 表
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| RSTART [BLD] | 指定ブロックの一時停止を解除し、停止ステップから実行を再開させます。 | 1286ページ RSTART [BLD] |
| SET [S□/BLD￥S□] | 指定ステップを活性化させます。 | 1288ページ SET [S□/BLD￥S□] |
| RST [S□/BLD￥S□] | 指定ステップを非活性にします。 | 1290ページ RST [S□/BLD￥S□] |
| BRSET | SFC制御命令の対象ブロックを指定します。 | 1292ページ BRSET |
| TRAN | 移行条件を成立させるダミー出力です。 | 1294ページ TRAN |

以上の内容は、CPUユニット用命令の応用命令の一部として、各命令の概要と参照ページを示したものです。
