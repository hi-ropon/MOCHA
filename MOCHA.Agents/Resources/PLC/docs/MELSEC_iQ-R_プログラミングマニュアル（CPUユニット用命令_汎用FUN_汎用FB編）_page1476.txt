### 表
#### 実行条件と演算結果

| 実行条件 | 演算結果 |
| --- | --- |
| EN | ENO | (d) |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となる。その場合は(d)から出力されたデータを使用しないようプログラム処理を行うこと。
