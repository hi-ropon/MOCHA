### テキスト
#### 使用可能デバイス
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび定数のみ使用可能。
- (s1)、(s2)で指定したデバイスのBIN32ビットデータをa接点扱いで比較演算を行う。

#### 機能
- (s1)で指定したデバイスのBIN32ビットデータと、(s2)で指定したデバイスのBIN32ビットデータを比較演算する。
- 各命令の比較演算結果は表に示す通り。
- LDD命令、ANDD命令、ORDD命令で16進数の定数を指定する場合、最上位ビット(b31)が1になる数値(8～F)を指定すると負の数とみなして比較演算を行う。
- 比較に使用するデータは32ビットデータを扱う命令(DMOV(P)命令など)で指定すること。16ビットデータを扱う命令(MOV(P)命令など)では正常に大小比較できない。
- LDD命令をFBD/LDで使用した場合、ENに左母線、または常時ONの変数部品/定数部品を指定する。
- ORDD命令をFBD/LDで使用した場合、ENにTRUEを指定するとENOはONとなり、ENは実行条件にならない。

#### エラー
- 演算エラーはない。
