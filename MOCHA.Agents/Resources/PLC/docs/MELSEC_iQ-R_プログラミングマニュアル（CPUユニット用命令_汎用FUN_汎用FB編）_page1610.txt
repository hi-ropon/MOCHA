### 表
#### GX Works2命令名と互換用命令名一覧
- 表は5つの列で構成されている。
  1. GX Works2命令名
  2. 互換用命令名
  3. 使用プログラム（ST）
  4. 使用プログラム（FBD）
  5. 参照先

- 「使用プログラム」列はST（構造化テキスト）とFBD（ファンクションブロックダイアグラム）での使用可否を○（使用可）または×（使用不可）で示している。

- 参照先は主に「MELSEC iQ-R プログラミングマニュアル（ユニット専用命令編）」や特定のページ番号が記載されている。

- 命令例：
  - GETE_M → G_GETE_M2、ST・FBD共に使用可、参照先はMELSEC iQ-Rマニュアル
  - GP_BIDIN → GP_BIDIN_M2、ST・FBD共に使用可
  - LD< → LD_LT_M2、STでは使用不可、FBDで使用可、参照先は214ページのLD□(_U)、AND□(_U)、OR□(_U)
  - INC → INC_M2、ST・FBD共に使用可、参照先は288ページのINC(P)(_U)
  - GRY → GRY_M2、ST・FBD共に使用可、参照先は458ページのGRY(P)(_U)

- 命令名は主に通信やデバイス操作に関するものが多く、互換用命令名は元の命令名に「_M2」や「G_」が付加された形で対応している。

- ページ下部に「付録 付6 他形式プロジェクトからの置換え」と記載されている。

以上の内容から、本ページはGX Works2の命令とその互換命令の対応表であり、プログラム形式ごとの使用可否や詳細参照先を示す技術資料の一部である。
