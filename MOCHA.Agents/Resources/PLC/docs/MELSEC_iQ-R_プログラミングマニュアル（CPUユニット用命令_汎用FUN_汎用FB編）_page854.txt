### テキスト
#### 機能
下限値(s1)の単精度実数と上限値(s2)の単精度実数の帯域に対して、比較データ(s3)の単精度実数で比較し、その結果（下、帯域内、上）に応じて(d)、(d)+1、(d)+2のいずれかがONします。

ラダー図の説明：
- (d) は [(s1), (s1)+1] > [(s3), (s3)+1] のときON
- (d)+1 は [(s1), (s1)+1] ≤ [(s3), (s3)+1] ≤ [(s2), (s2)+1] のときON
- (d)+2 は [(s3), (s3)+1] > [(s2), (s2)+1] のときON

#### 注意事項
- 下限値(s1)は上限値(s2)より小さな値に設定してください。
- 下限値(s1)を上限値(s2)より大きな値とした場合、上限値(s2)は下限値(s1)と同じ値として扱います。
