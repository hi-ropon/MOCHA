### テキスト
#### FTPサーバ名
FTPサーバ名（ドメイン名）をASCII文字列で設定します。文字数は1～256文字（半角英数字、半角記号）です。  
1文字目を(s1)+5の下位バイトに、2文字目を(s1)+5の上位バイトに格納し、最後の文字の後に00Hを格納して文字列の終端を示します。  
なお、(s1)+4に"0000H: FTPサーバアドレス"を指定する場合はFTPサーバ名の設定は不要です。

#### FTPサーバアドレス
FTPサーバのIPアドレスを000.000.000.001～223.255.255.254の範囲で設定します。  
IPアドレスは4オクテットに分けて、(s1)+135の上位バイト、(s1)+135の下位バイト、(s1)+134の上位バイト、(s1)+134の下位バイトに格納します。  
FTPサーバ名を指定する場合は設定不要です。

#### ログイン名
FTPサーバにログインするためのログイン名をASCII文字列で設定します。文字数は1～32文字（半角英数字、半角記号）です。  
1文字目を(s1)+136の下位バイトに、2文字目を(s1)+136の上位バイトに格納し、最後の文字の後に00Hを格納して文字列の終端を示します。

#### パスワード
FTPサーバにログインするためのパスワードをASCII文字列で設定します。文字数は0～32文字（半角英数字、半角記号）です。  
1文字目を(s1)+153の下位バイトに、2文字目を(s1)+153の上位バイトに格納し、最後の文字の後に00Hを格納して文字列の終端を示します。
