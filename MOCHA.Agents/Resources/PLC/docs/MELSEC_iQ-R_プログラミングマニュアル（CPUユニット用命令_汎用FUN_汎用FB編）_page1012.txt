#### Point
- PLSY命令は、引数のデバイスデータをCPUユニットのワークエリアに登録し、実際の出力動作はシステム割込みで処理される。CPUユニットのワークエリアに登録されたデバイスデータは、実行指令をOFFまたはSTOP→RUNにてクリアされる。
- 出力できるパルスはONおよびOFF時間がCPUユニットの割込み間隔以上必要で、CPUユニットの割込み間隔は1msである。
- PLSY命令によるパルス出力中（実行指令がON中）は、PLSY命令の引数を変更しないこと。引数の変更は実行指令をOFFしてから行う。
- PLSY命令はCPUユニットで実行している全プログラムで1回のみ使用可能で、2回目以降は無処理となる。
