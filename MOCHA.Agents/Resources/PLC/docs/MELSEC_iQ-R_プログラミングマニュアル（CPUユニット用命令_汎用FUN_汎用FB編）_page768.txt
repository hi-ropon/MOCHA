### テキスト
#### 運転停止異常命令 PABORT
プログラムの実行を停止し、CPUユニットを停止エラーにします。

#### 機能
- PABORT命令を実行すると、プログラムの実行を停止し、CPUユニットを停止エラー（エラーコード：3070H）にします。
- 実行時にSM0（最新自己診断エラー（アナンシエータONを含む））がONし、CPUユニットのERROR LEDが点滅します。
- SM1（最新自己診断エラー（アナンシエータONを含まない））はONしません。
- (s)で指定したBIN16ビットデータをエラーコード3070Hの詳細情報2に格納します。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| PABORT | 立ち上がりエッジ（パルス） |

### 図
- **ラダー図**: (s)のビットデバイスを入力としてPABORT命令を実行するラダー回路。
- **ST言語**: `ENO:=PABORT(EN,s);`
- **FBD/LD**: EN入力、ENO出力、s入力のPABORT命令ブロック。
