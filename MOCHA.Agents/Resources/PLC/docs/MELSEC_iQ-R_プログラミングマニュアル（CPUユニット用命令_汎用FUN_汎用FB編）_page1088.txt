<overview>
本ページは、命令の比較演算結果に関する説明で構成されています。テキスト部分では、比較命令の種類と条件、比較演算結果の説明があり、例として日付の比較に関する解説が含まれています。表部分は、比較命令の条件と結果をまとめた表が2つ掲載されており、比較対象ごとの導通・非導通の状態を示しています。図部分は、日付A、B、Cの比較を示すタイムライン形式の図が1点あります。
</overview>

<contents>
### テキスト
- 各命令の比較演算結果について説明しています。
- 比較命令記号（ラダー、FBD/LD）と条件、比較演算結果の関係を示しています。
- 例として、日付A、B、Cの比較を示し、同じ条件でも比較対象によって結果が異なることを説明しています。
- FBD/LDの場合、導通はENOがON、非導通はENOがOFFを示すことが記載されています。
- 比較する日付が実在しなくても、設定範囲内の日付であれば条件に従って比較演算することが可能である旨が説明されています。
- LDDT_□命令やORDT_□命令の使用時のENOの挙動についても補足されています。

### 図
- 日付A、B、Cの比較を示すタイムライン図
  - 2006/1/1から2009/1/1までの期間を示し、Aは2006/9/22、Bは2007/6/23、Cは2008/8/8の日付を示しています。
  - これにより、日付の大小関係を視覚的に理解できるようになっています。
