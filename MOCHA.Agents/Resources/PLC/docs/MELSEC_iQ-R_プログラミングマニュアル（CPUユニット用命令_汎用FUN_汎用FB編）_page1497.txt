### テキスト
#### 機能
- 演算処理の動作条件について説明。
  - (s1)がONになると(d)をSETし、(s1)がOFFで(s2)がONになると(d)をRESETする。
  - (s1)がONのときに(s2)をONにしても(d)はRESETされない。

#### 演算結果
1. EN/ENOなしファンクションブロック
- 演算処理を実行し、(d)から演算出力値が出力される。

2. EN/ENO付きファンクションブロック
- 実行条件と演算結果の関係を表で示す。
- ENがTRUEのときは演算実行、FALSEのときは演算停止。
- ENOはENの状態を反映し、(d)は演算出力値または前回出力値を示す。

#### エラー
- 演算エラーは存在しない。

### 図（タイミングチャート）
1. EN/ENOなしファンクションブロックのタイミングチャート
- (s1)、(s2)、(d)の信号のON/OFF状態を時間軸に沿って示している。
- (s1)がOFFからONになると(d)がONになる。
- (s1)がOFFかつ(s2)がOFFからONになると(d)がOFFになる。

2. EN/ENO付きファンクションブロックのタイミングチャート
- EN、(s1)、(s2)、ENO、(d)の信号のON/OFF状態を示す。
- ENがONかつ(s1)がOFFからONになると(d)がONになる。
- ENがONかつ(s1)がOFFかつ(s2)がOFFからONになると(d)がOFFになる。

### 表
| 実行条件 | 演算結果 |
| :--- | :--- |
| EN | ENO | (d) |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE | 前回出力値 |
