### テキスト
#### 32.49 TIME型→DWORD型変換
TIME型データをDWORD型データへ変換する関数「TIME_TO_DWORD(_E)」について説明しています。  
対応CPUはRnCPU、RnENCPU、RnPSFCPU（プロセス、安全、一般）など複数あります。

##### 機能
- (s)に入力されたTIME型のデータをDWORD型のデータに変換し、(d)から出力します。  
- (s)への入力値はTIME型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。  
2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |  
| --- | --- |  
| EN | ENO | (d) |  
| TRUE（演算実行） | TRUE | 演算出力値 |  
| FALSE（演算停止） | FALSE*1 | 不定値 |  

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となります。その場合は(d)から出力されたデータを使用しないようプログラムで処理してください。

##### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDでの関数ブロック図  
　EN/ENOなし：s（TIME型入力）→d（DWORD型出力）  
　EN/ENO付き：EN（実行条件）、s（TIME型入力）、ENO（出力状態）、d（DWORD型出力）  
- ST言語での記述例  
　[EN/ENOなし] d := TIME_TO_DWORD(s);  
　[EN/ENO付き] d := TIME_TO_DWORD_E(EN, ENO, s);

### 表
#### 設定データ（引数、内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |  
| --- | --- | --- | --- |  
| EN | 実行条件（TRUE:実行、FALSE:停止） | 入力変数 | BOOL |  
| s(IN) | 入力 | 入力変数 | TIME |  
| ENO | 出力状態（TRUE:正常、FALSE:異常） | 出力変数 | BOOL |  
| d | 出力 | 出力変数 | DWORD |

### イラスト説明
- TIME型の入力例：T#20m34s567ms  
- DWORD型の出力例：1234567（TIMEの値をDWORDの数値に変換したイメージ）
