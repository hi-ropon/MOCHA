### テキスト
#### 機能
- (s)で指定された角度のSIN（正弦）値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- (s)で指定するSIN値は-1.0～1.0の範囲で設定可能。
- (d)に格納する角度（演算結果）はラジアン単位で格納される。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### 注意事項
- エンジニアリングツールからの入力値設定時の注意事項は47ページを参照。

### 表
#### エラーコード一覧

| エラーコード (SD0) | 内容                                   |
|--------------------|--------------------------------------|
| 3402H              | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |
| 3405H              | (s)に指定した値が-1.0～1.0の範囲外のとき。         |

### ポイント
- ラジアンと角度の変換については、RADD(P)命令、DEGD(P)命令を参照すること。
- 参照ページは945ページ（RADD(P)）、947ページ（DEGD(P)）。

### 図（テキスト内イメージ）
- SIN^-1の計算イメージとして、(s)の4つのデバイスに倍精度実数のSIN値が格納され、(d)の4つのデバイスに倍精度実数の角度（ラジアン）が格納される流れが示されている。
