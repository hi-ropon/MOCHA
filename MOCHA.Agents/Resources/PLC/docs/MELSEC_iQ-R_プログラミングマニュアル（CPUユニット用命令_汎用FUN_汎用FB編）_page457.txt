### テキスト
#### 機能
- (s)で指定した符号なしBIN32ビットデータ(ANY32_U)を、符号付きBIN32ビットデータ(ANY32_S)に変換し、(d)で指定したラベルに格納する。
- UDINT2DINT(P)命令の使用例を示す。

#### 例
- D+命令はオペランドにANY32_Sを指定する必要があるため、D+命令実行前にUDINT2DINT命令を使用し、ANY32_UのudLabel0をANY32_SのdLabel1へ変換する。
- 変換後の値はudLabel0の値をそのままdLabel1へ格納する。

#### エラー
- 演算エラーはありません。

### 図
- bSwitchAビットの入力により、UDINT2DINT命令がudLabel0（符号なしダブルワード32ビット）をdLabel1（符号付きダブルワード）に変換。
- 図中のビット構成は、b31からb0までの32ビットを示し、udLabel0の値「8000H 0001H」がdLabel1にそのまま格納される流れを示す。
- (1)はそのまま格納する動作、(2)はD+命令のオペランドのデータ型に合わせて演算することを示す。

### 用語説明
- bSwitchA: ビット
- dLabel0, dLabel1: ダブルワード[符号付き]
- udLabel0: ダブルワード[符号なし]/ビット列[32ビット]
