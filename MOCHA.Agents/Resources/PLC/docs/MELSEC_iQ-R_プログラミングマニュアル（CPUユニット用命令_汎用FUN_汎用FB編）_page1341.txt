### テキスト
#### INT型→BCD型変換
INT_TO_BCD(_E)関数は、INT型データをBCD型データ（WORD型）へ変換します。

#### 設定データ
■内容、種別、データ型  
| 引数 | 内容 | 種別 | データ型 |
| --- | --- | --- | --- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | WORD |

#### 機能
■演算処理  
(s)に入力されたINT型のデータをBCD型(WORD型)のデータに変換し、(d)から出力します。

■演算結果  
1. EN/ENOなしファンクション  
演算処理を実行し、(d)から演算出力値が出力されます。  
2. EN/ENO付きファンクション

### 表
#### 実行条件と演算結果
| EN | ENO | (d) |
| --- | --- | --- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)から出力されたデータを使用しないようプログラム処理を行う必要があります。
