### テキスト
#### 40.1 加算
関数名：ADD_TIME(_E)
対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPI（一般）、RnSFCPI（安全）
説明：入力値（TIME型）の和((s1)+(s2))を出力します。

#### 使用例
- ラダー、FBD/LDでの使用例
  - EN/ENOなしの場合とEN/ENO付きの場合のブロック図が示されている。
- ST言語での使用例
  - EN/ENOなし：d := ADD_TIME(s1, s2);
  - EN/ENO付き：d := ADD_TIME_E(EN, ENO, s1, s2);

### 表
#### 設定データ：内容、種別、データ型
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件（TRUE:実行、FALSE:停止） | 入力変数 | BOOL |
| s1(IN1)、s2(IN2) | 入力 | 入力変数 | TIME |
| ENO | 出力状態（TRUE:正常、FALSE:異常） | 出力変数 | BOOL |
| d | 出力 | 出力変数 | TIME |

以上より、ADD_TIME関数は2つのTIME型入力の加算を行い、実行条件と出力状態を管理しつつ結果をTIME型で出力する関数であることがわかります。
