### テキスト
#### 機能
- (d)で指定したデバイスから単精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトする。
- シフト後、(s)から(n2)点分は空きデータ部分に格納される。
- (s)に定数を指定した場合、シフト後の最下位から(n2)点分は指定した値になる。
- (n2)で指定した値が0の場合は無処理となる。

#### 例
- (n1)=9、(n2)=4の場合の動作を説明。

### 図
- データのシフト動作を示す図解。
- 元データは(d)+17から(d)までの9点で構成されている。
- 左へ4点分シフトし、空いた部分には(s)から4点分の値（1.1、2.2、3.3、4.4）がコピーされる。
- 図中に「① (n2)点分、左へシフト」「② コピー」の動作が示されている。

| 3405H | (n1)、(n2)に指定した値が(n1) < (n2)のとき。 |
