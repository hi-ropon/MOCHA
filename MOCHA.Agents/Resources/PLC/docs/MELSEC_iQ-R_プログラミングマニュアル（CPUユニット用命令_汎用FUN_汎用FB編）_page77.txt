#### 倍精度実数→符号なしBIN16ビット/32ビットデータ変換
- 命令コード: DBL2UINT, DBL2UINTP
- 処理内容: 指定した倍精度実数（0～65535）を符号なしBIN16ビットデータに変換。数式で(s)+3, (s)+2, (s)+1, (s)から(d)へ変換しBIN化。
- 参照先: 428ページ DBL2UINT(P)

- 命令コード: DBL2UDINT, DBL2UDINTP
- 処理内容: 指定した倍精度実数（0～4294967295）を符号なしBIN32ビットデータに変換。
- 参照先: 432ページ DBL2UDINT(P)
