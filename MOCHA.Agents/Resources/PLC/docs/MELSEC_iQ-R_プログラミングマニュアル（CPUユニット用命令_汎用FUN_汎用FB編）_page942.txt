### テキスト
#### 機能
- 角度の大きさの単位を、(s)で指定された度単位（DEG.単位）からラジアン単位に変換し、(d)で指定されたデバイス番号に格納する。
- 度単位からラジアン単位への変換は以下の式で行う。
  ラジアン単位 = 度単位 × (π / 180)
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細な注意事項は47ページを参照。

#### エラー
- エラーコード(SD0)と内容の説明
  - 3402H: (s)に指定した値が-0、非正規化数、非数、±∞のとき。
  - 3403H: (d)が指定範囲を超えたとき（オーバーフロー発生時）。|d| < 2^128 の範囲内である必要がある。

### 図
- 度単位の単精度実数(s)がラジアン単位の単精度実数(d)に変換されるイメージ図
  - (s)+1、(s) → 度（°）
  - (d)+1、(d) → ラジアン（rad）
  - 矢印で度からラジアンへの変換を示している。
