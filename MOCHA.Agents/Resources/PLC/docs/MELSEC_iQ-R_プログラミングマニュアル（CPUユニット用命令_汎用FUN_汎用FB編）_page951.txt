### テキスト
#### 倍精度実数平方根（EDSQRT(P)）
倍精度実数で指定した値の平方根を演算する命令です。

#### 実行条件
- EDSQRT命令は立ち上がりエッジで実行されます。
- EDSQRTP命令は立ち下がりエッジで実行されます。

#### 機能
- オペランド(s)で指定された値の平方根を計算し、(d)で指定されたデバイスに演算結果を格納します。
- (s)に指定できる値は正の数のみで、負の数は演算不可です。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する際、丸め誤差が発生する場合があります。詳細は47ページの注意事項を参照してください。

### 図
- ラダー図：EDSQRT命令のラダー表現。入力(s)、出力(d)の接続例。
- ST言語：ENO:=EDSQRT(EN,s,d); ENO:=EDSQRTP(EN,s,d); の記述例。
- FBD/LD図：EN入力、ENO出力、s入力、d出力のブロック図。
