### テキスト
#### 符号なしBIN32ビットデータ→符号付きBIN32ビットデータ変換
UDINT2DINT(P)命令は、指定した符号なしBIN32ビットデータを符号付きBIN32ビットデータに変換します。

#### ■実行条件
- UDINT2DINT命令は立下りエッジで実行
- UDINT2DINTP命令は立上りエッジで実行

#### ■Point
UDINT2DINT(P)命令はラベルでプログラミングする場合に、命令のオペランドに指定可能なデータ型と、指定されたラベルのデータ型を一致させるために使用します。  
デバイスでプログラミングする場合は、UDINT2DINT(P)命令を使用する必要はありません。

### 図
- **ラダー図**  
  (s)に符号なしBIN32ビットデータまたはラベルを入力し、(d)に符号付きBIN32ビットデータのラベルを出力する構成を示す。

- **FBD/LD図**  
  EN（実行条件）入力、ENO（実行結果）出力、s（入力データ）、d（出力データ）を持つ命令ブロックの形状を示す。
