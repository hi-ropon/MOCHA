### テキスト
#### 倍精度実数減算
ED-(P)命令は倍精度実数の減算を行います。ここではオペランドが2個の場合の説明です。

#### 実行条件
- ED-命令は立下りエッジで実行
- ED-P命令は立上りエッジで実行

#### 機能
(d)で指定された倍精度実数から(s)で指定された倍精度実数を減算し、その結果を(d)で指定されたデバイスに格納します。

- (s)、(d)には0から2^-1022以上、2^1024未満の値が指定・格納可能
- エンジニアリングツールから入力値を設定する際は丸め誤差が発生する場合があるため注意が必要

### 表
#### 実行条件
| 命令 | 実行条件 |
| --- | --- |
| ED- | 立下りエッジ |
| ED-P | 立上りエッジ |

#### 設定データ：内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| --- | --- | --- | --- | --- |
| (s) | 減算データまたは減算データが格納されている先頭デバイス | 0, 2^-1022 ≤ |s| < 2^1024 | 倍精度実数 | ANYREAL_64 |
| (d) | 減算されるデータが格納されている先頭デバイス | 0, 2^-1022 ≤ |d| < 2^1024 | 倍精度実数 | ANYREAL_64 |

#### 使用可能デバイス

| (d) | なし | ○ (T, ST, C, D, W, SD, SW, FD, R, ZR, RD) | なし | ○ (LT, LST, LZ) | なし | なし |

- ラダー回路の簡単な図示：ED-(P)命令のラダー表現（sとdのオペランドを示す）
- 実行条件の波形イメージ：ED-は立下り、ED-Pは立上りの波形で実行されることを示す
- 倍精度実数の減算イメージ：(d)の3つの連続するデバイスからなる倍精度実数から(s)の3つの連続するデバイスの倍精度実数を減算し、結果を(d)の3つのデバイスに格納する流れを図示
