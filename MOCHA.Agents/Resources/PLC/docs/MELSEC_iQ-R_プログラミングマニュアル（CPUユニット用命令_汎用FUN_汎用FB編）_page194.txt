### テキスト
#### 5.4 シフト命令 ビットデバイスシフト SFT(P)
1つ若いデバイスのON/OFF状態を指定したデバイスにシフトし、1つ若いデバイスをOFFにする命令です。

#### 実行条件
- SFT命令は立下りエッジで実行されます。
- SFTP命令は立上りエッジで実行されます。

### 図
- **ラダー図**  
  (d)というオペランドを持つシフト命令のラダー図が示されています。  
  ENO:=SFT(EN,d);  
  ENO:=SFTP(EN,d);  
  というST言語の記述も併記されています。

- **FBD/LD図**  
  EN（実行条件）、ENO（実行結果）、d（シフトするデバイス）を持つ命令ブロックの図が示されています。
