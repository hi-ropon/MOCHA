### デバッグ，故障診断命令
#### エラー表示またはアナンシエータリセット
- **LEDR**: CPUユニットのアナンシエータ表示や運転を継続できる自己診断エラーの表示をリセットする命令。
- 参照ページ: 764ページ LEDR

#### 運転継続異常命令
- **PALERT, PALERTP**: CPUユニットを続行エラーにする命令。
- 参照ページ: 766ページ PALERT(P)

#### 運転停止異常命令
- **PABORT**: プログラムの実行を停止し、CPUユニットを停止エラーにする命令。
- 参照ページ: 768ページ PABORT

### 文字列処理命令
#### 文字列比較
- 命令記号: LD$=, AND$=, OR$=, LD$<>, AND$<>, OR$<>, LD$>, AND$>, OR$>, LD$<=, AND$<=, OR$<=, LD$<, AND$<, OR$<, LD$>=, AND$>=, OR$>=
- 処理内容: (s1)の文字列と(s2)の文字列を1文字ずつ比較する。
- 参照ページ: 770ページ LD$, AND$, OR$

#### 文字列の結合
- 命令記号: $+, $+P
- 処理内容: (d)で指定の文字列に(s)で指定の文字列を結合し、(d)以降に格納する。
- 参照ページ: 774ページ $+(P) [オペランドが2個の場合]、776ページ $+(P) [オペランドが3個の場合]

#### 文字列転送
- 命令記号: $MOV, $MOVP, $MOV_WS, $MOVP_WS
- 処理内容: (s)で指定の文字列を(d)で指定のデバイス以降に転送する。Unicode文字列転送も可能。
- 参照ページ: 778ページ $MOV(P)、780ページ $MOV(P)_WS

#### BIN16ビット/32ビットデータ→10進アスキー変換
- 命令記号: BINDA, BINDAP, BINDA_U, BINDAP_U
- 処理内容: (s)で指定した1ワードBIN値を10進アスキー値に変換し、(d)で指定したワードデバイス番号以降に格納する。
- 参照ページ: 782ページ BINDA(P)(_U)

- 命令記号: DBINDA, DBINDAP, DBINDA_U, DBINDAP_U
- 処理内容: (s)で指定した2ワードBIN値を10進アスキー値に変換し、(d)で指定したワードデバイス番号以降に格納する。
- 参照ページ: 786ページ DBINDA(P)(_U)
