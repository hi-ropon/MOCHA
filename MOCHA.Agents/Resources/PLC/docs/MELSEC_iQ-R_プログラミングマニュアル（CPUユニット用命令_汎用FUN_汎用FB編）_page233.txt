### テキスト
#### 機能
(d)で指定したBIN16ビットデータと、(s)で指定したBIN16ビットデータの加算を行い、その加算結果を(d)で指定したデバイスに格納します。

- 加算結果にオーバーフローが生じた場合、桁上げは無視されます。この場合、SM700（キャリフラグ）はONしません。

#### エラー
演算エラーはありません。

### 図
- 加算の具体例として、(d)と(s)のBIN16ビットデータの値が示され、それらの加算結果が矢印で表現されています。

1. 正の数の加算例
   - (d) 5678 (BIN) + (s) 1234 (BIN) = (d) 6912 (BIN)

2. 負の数を含む加算例（符号付き）
   - (d) -12345 (BIN) + (s) -23456 (BIN) = (d) 29735 (BIN)
   - (d) 12345 (BIN) + (s) 23456 (BIN) = (d) -29735 (BIN)

3. 符号なし加算例
   - (d) 56789 (BIN) + (s) 12345 (BIN) = (d) 3598 (BIN)

これらの図は、BIN16ビットデータの加算処理の動作例を示しており、オーバーフロー時の動作も含めて理解を助ける内容となっています。
</contents>
