### テキスト
#### 機能
(s)から(n)点分の倍精度実数のブロックデータに対して、最小値を検索し、(d)で指定したデバイスへ格納します。  
最初に最小値を検出したデータを、(s)から何点目かを(d)+4で指定したデバイスへ格納し、最小値と同じ値の個数を(d)+5で指定したデバイスへ格納します。  
検索結果（位置）の値については、(s)で指定したブロックデータの先頭を1点目として扱います。  

・(d)、(d)+1、(d)+2、(d)+3には最小値が格納されます。  
・(d)+4には位置（何点目か）が格納されます。  
・(d)+5には個数が格納されます。  
・(n)で指定した値が0の場合は無処理となります。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)で指定したブロックデータに、倍精度実数以外の値が存在する場合。 |

### 図
左側の図は、(s)から(n)点の範囲のブロックデータの例を示しています。  
例として、1.2345、123.45、-1.2345、-12.345、-123.45の5点が並んでいます。  
右側の図は、(d)で指定したデバイスに格納される結果のイメージです。  
(d)、(d)+1、(d)+2、(d)+3に最小値-123.45が格納され、(d)+4に位置5（5点目）、(d)+5に個数1が格納されることを示しています。
