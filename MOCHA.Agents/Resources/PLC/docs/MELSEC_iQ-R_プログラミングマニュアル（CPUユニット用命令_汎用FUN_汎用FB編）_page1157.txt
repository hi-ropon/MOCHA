### テキスト
#### コネクションの切断
命令「SP.SOCCLOSE」は指定したコネクションに対してクローズ処理を行う命令です。

#### 実行条件
命令「SP.SOCCLOSE」は実行条件が成立した場合に動作します。

### 図
#### ラダー図
- ラダー図では、入力として(U)、(s1)、(s2)、(d)の4つのオペランドがあり、命令が実行される様子を示しています。

#### FBD/LD図
- FBD/LD図では、ブロックの左側にEN（実行条件）、U、s1、s2の入力があり、右側にENO（実行結果）とdの出力がある構成です。
