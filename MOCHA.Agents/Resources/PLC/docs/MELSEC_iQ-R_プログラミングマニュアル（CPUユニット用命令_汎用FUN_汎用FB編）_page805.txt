### テキスト
#### BCD4桁データ→10進アスキーコード変換
BCD4桁データをアスキーコードに変換する命令「BCDDA(P)」について説明しています。

##### 命令概要
- BCD4桁データをアスキーコードに変換します。

##### 実行条件
- BCDDA命令は立ち上がりエッジで実行されます。
- BCDDAP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー図では、(s)と(d)の2つのオペランドを持つ命令ブロックが示されています。

- **FBD/LD図**  
  FBD/LD図では、EN（入力）、ENO（出力）、s（入力BCDデータ）、d（出力文字列）のポートがある命令ブロックが示されています。
