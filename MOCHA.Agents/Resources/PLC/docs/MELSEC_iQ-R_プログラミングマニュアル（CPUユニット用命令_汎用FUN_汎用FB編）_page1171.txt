### テキスト
#### コネクションの受信モード変更
命令名：SP.SOCRMODE  
指定したコネクションに対して、TCP受信モード（UDP通信のコネクションには無効）および受信データサイズを変更します。

#### 実行条件
命令：SP.SOCRMODE  
実行条件は特に指定されていません。

### 図
#### ラダー
- ラダー図には、3つの入力（U、s1、s2）を持つ命令ブロックが示されています。
- 出力はありません。
- ST言語では、`ENO:=SP_SOCRMODE(EN,U,s1,s2);` と記述されます。

#### FBD/LD
- ブロック図は、EN（実行条件）、U、s1、s2の4つの入力とENO（実行結果）の出力を持つ命令ブロックを示しています。
