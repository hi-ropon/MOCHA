### テキスト
#### エラー
演算エラーはありません。

#### Point
- MPS命令、MRD命令、MPP命令を使用した場合と使用しない場合の回路は以下のようになります。
- MPS命令、MPP命令の使用数は同一にしてください。使用数が異なる場合、エンジニアリングツールの回路モードで正常な回路表示ができなくなります。

### 図（回路図）
#### [MPS命令、MRD命令、MPP命令を使用した回路]
- X0、X1、X2の3つの入力が直列に接続され、その後にY10が接続されている。
- X3とX4が直列に接続され、その後にY11が接続されている。
- X5が単独でY12に接続されている。

#### [MPS命令、MRD命令、MPP命令を使用しない回路]
- X0、X1、X2が直列に接続され、その後にY10が接続されている。
- X0、X1、X3、X4が直列に接続され、その後にY11が接続されている。
- X0、X1、X5が直列に接続され、その後にY12が接続されている。

以上のように、命令の使用有無によって回路の構成が異なることが示されています。
