### テキスト
#### 機能
- (s1)で指定したワードデバイスから、(s2)で指定した位置のビットデータを取り出し、(d)で指定したビットデバイスに書き込みます。
- (1)の図は、(s1)のワードデバイスのビット位置を示しており、b15からb0までのビットが並んでいます。
- (d)で指定したビットデバイスは、該当ビットが「0」のときOFF、「1」のときONになります。
- (s2)はワードデータのビット位置(0～15)を指定し、16以上を指定した場合は16で割った余りの値がビット位置となります。

##### 例
- (s2)=18のとき、18÷16=1余り2のため、b2のデータになります。

#### エラー
- 演算エラーはありません。

### 図
- ワードデバイスのビット位置を示す図で、b15からb0までのビットが横に並んでいます。
- (1)は(s2)ビット位置の例として、(s2)=5の場合を示しています。
- 矢印で(s1)のワードデバイスからビットを取り出し、(d)のビットデバイスに書き込む流れを表現しています。
