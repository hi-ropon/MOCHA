### テキスト
#### 機能
- 指定したデバイスから(n1)ダブルワード分のデータ範囲内で、右へ(n2)ダブルワード分シフトする。
- 最上位から(n2)ダブルワード分は0Hになる。
- (n1)または(n2)で指定した値が0の場合は無処理となる。
- (n1)が(n2)以下の場合は、指定デバイスから(n1)ダブルワード分のデータはすべて0Hになる。

#### エラー
- 演算エラーは発生しない。

### 図（例）
- (n1)=9、(n2)=4の場合のデータシフトの例を示す。
- 上段に(d)から(d)+17までのデバイスアドレスと対応する16進数データが並ぶ。
- 下段は右へ4ダブルワード分シフトした結果で、最上位4ダブルワード分は0Hに置き換わっている。
- 具体的には、元のデータの(d)+13から(d)+17までのデータが消え、代わりに0Hが入る。
- 残りのデータは右に4ダブルワード分ずれている。
