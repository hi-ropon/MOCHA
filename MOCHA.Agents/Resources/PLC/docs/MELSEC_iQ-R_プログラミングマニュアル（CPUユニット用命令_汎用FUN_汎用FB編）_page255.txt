### テキスト
#### 使用可能デバイス
- オペランドごとに使用可能なデバイスの種類が表形式で示されています。
- ビット、ワード、ダブルワード、間接指定、定数の各カテゴリに分かれており、(s1)、(s2)、(d)の3種類のオペランドに対して使用可否が○や―で示されています。
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、特定の安全デバイスおよび定数のみ使用可能であることが記載されています。

#### 機能
- (s1)で指定したBIN32ビットデータと、(s2)で指定したBIN32ビットデータの除算を行い、その結果を(d)で指定したデバイスに格納します。
- 図では、(s1)に「567890 (BIN)」、(s2)に「123456 (BIN)」を指定し、除算結果が「4 (BIN)」として(d)+1に格納される例が示されています。
- 除算の商は(d)+1に、余りは(d)+3に格納されます。
- ワードデバイスの場合、64ビットを使用して商と余りが格納され、商は下位32ビット、余りは上位32ビットに分けて格納されます。
- ビットデバイスの場合は32ビットを使用し、商のみが格納されます。

#### エラー
- エラーコード「3400H」は、(s2)で指定した値（除数）が0のときに発生します。
