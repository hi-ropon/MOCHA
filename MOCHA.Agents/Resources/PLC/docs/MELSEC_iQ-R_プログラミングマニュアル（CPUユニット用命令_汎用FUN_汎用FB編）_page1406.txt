### テキスト
#### 32.64 DINT型→ビット配列変換
関数名：DINT_TO_BITARR(_E)
この関数は、DINT型データの下位nビットをビット配列に出力します。

##### 対応CPU
RnCPU、RnENCPU、RnFSPCPU（プロセス、二重化、一般、安全）に対応。

##### 機能
- 演算処理：入力sに指定されたANY32型の下位nビットを出力dに出力します。指定されたビット数以上の出力ビットは変更されません。
- 演算結果：
  1. EN/ENOなしファンクション：演算処理を実行し、dから演算出力値が出力される。
  2. EN/ENO付きファンクション：実行条件と演算結果は表の通り。

##### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LD言語での関数ブロック図
  - EN/ENOなし：s（入力ANY32）、n（入力BOOL）、d（出力BOOL配列）
  - EN/ENO付き：EN（入力BOOL）、ENO（出力BOOL）、s、n、d
- ST言語での記述例
  - EN/ENOなし：d := DINT_TO_BITARR(s, n);
  - EN/ENO付き：d := DINT_TO_BITARR_E(EN, ENO, s, n);

### 表
#### 設定データ（引数、内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s | 入力 | 入力変数 | ANY32 |
| n | 4,8,12,16,20,24,28,32の定数のみ指定可能 | 入力変数 | BOOL |
| ENO | 出力状態(TRUE:正常実行、FALSE:異常または停止) | 出力変数 | BOOL |
| d | 出力(要素指定は変数も可) | 出力変数 | BOOL配列要素 |

#### 実行条件と演算結果

| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、(d)から出力されたデータを使用しないようプログラム処理を行う必要がある。
