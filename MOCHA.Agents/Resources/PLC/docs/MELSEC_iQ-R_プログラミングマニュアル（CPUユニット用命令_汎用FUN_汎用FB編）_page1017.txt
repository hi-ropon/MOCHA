### テキスト
#### プログラム例
10分割されたロータリーテーブル上に置かれた品物を窓口で出し入れする場合に、ロータリーテーブルを近回りで回転するように制御するプログラム例です。  
品物の番号はD2で指定し、窓口の番号はD1で指定します。  
前後2区間はテーブルを低速回転させます。

#### エラー
演算エラーはありません。

### 図
- ロータリーテーブルは10分割されており、0～9の番号が付けられた品物が配置されています。
- 0番窓口と1番窓口があり、それぞれにロボットアームが設置されています。
- 検出スイッチが2箇所（X000、X001）と0点検出スイッチ（X002）が配置されており、テーブルの位置検出に使用されます。
- ロータリーテーブルは正転方向に回転し、品物の出し入れを行います。
- プログラムラダー図が示されており、X0～X10の入力とM0～M2の内部リレーを使い、ROT命令でD0の値をK10（10分割）でK2（回転区間）制御し、M0で回転動作を制御しています。
