### テキスト
#### オートチューニング(ステップ応答法)のみのプログラム例の説明
- (0)ステップ
  - 目標値設定は50℃未満
  - オートチューニング用出力値設定は1.8秒ON
  - サンプリングタイム(Ts)は3秒以下に設定
  - 動作設定(ACT)でオートチューニング開始
  - 入力フィルタ(α)設定
  - 微分ゲイン(KD)は0%に設定
  - 出力値上限設定は2秒ON
  - 出力値下限設定は0秒ON
  - オートチューニング開始の指示

- (180)ステップ
  - PID命令動作

- (191)ステップ
  - A/D変換ユニットより測定値(PV)を入出力データエリアに設定

- (225)ステップ
  - PID動作初期化

### ラダー図（プログラム構成）
- X2入力がトリガとなり、複数のMOVP命令で各種定数や設定値をD500～D533のデータレジスタに書き込む。
  - 例：K500をD500に、K1800をD502に、K3000をD510に、H31をD511に、K70をD512に、K0をD515に、K2000をD532に、K0をD533に設定。
- M0のパルス信号(PULSE)発生とM1のセット命令が続く。
- SM400の信号でMOV命令によりU8¥G400の値をD501に移動。
- X2とM1の条件でD502のリセット命令(RST)が実行される。

このプログラムは、ステップ応答法によるオートチューニングのための初期設定、PID制御の開始、測定値の取得、PID動作の初期化を順に行う構成となっている。
