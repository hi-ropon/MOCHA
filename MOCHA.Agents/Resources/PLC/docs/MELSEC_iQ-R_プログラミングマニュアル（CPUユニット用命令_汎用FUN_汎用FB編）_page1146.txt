### テキスト
#### ルーチング情報の登録
指定した転送先ネットワークNo.のエリアにルーチング情報を登録する命令「S(P).RTWRITE」について説明しています。

#### 実行条件
命令ごとの実行条件波形が示されています。
- S.RTWRITE：立ち下がりエッジで実行
- SP.RTWRITE：立ち上がりエッジで実行
