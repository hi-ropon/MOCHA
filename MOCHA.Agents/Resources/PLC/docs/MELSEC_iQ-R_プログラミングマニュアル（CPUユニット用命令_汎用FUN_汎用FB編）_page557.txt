### テキスト
#### 7.1 プログラム分岐命令 ポインタ分岐
- 対象命令はCJ、SCJ、JMP。
- CJ：同一プログラムファイル内の指定されたポインタ番号のプログラムを実行。
- SCJ：次のスキャンから同一プログラムファイル内の指定されたポインタ番号のプログラムを実行。
- JMP：無条件に同一プログラムファイル内の指定されたポインタ番号のプログラムを実行。
- ラダー、ST、FBD/LD言語での対応状況が記載されており、ラダーのみ対応、STとFBD/LDは非対応。

#### 実行条件
| 命令 | 実行条件 |
| --- | --- |
| CJ, SCJ | 入力信号がONのとき実行 |
| JMP | 常時実行 |

#### 設定データ
- 内容、範囲、データ型についての説明。
- オペランド(P)はジャンプ先のポインタ番号を指定し、デバイス名でPOINTER型。

#### 使用可能デバイス
- オペランド(P)はその他(P)欄に○があり使用可能。
- ビット、ワード、ダブルワード、間接指定、定数などの各デバイスタイプに対しての使用可否が表形式で示されている。

#### 機能
- CJ命令の動作説明：
  - 実行指令がONのとき、指定されたポインタ番号のプログラムを実行。
  - 実行指令がOFFのときは次のステップのプログラムを実行。
- 図でON/OFF信号とCJ命令の実行タイミングを示し、(1)が実行指令、(2)が毎スキャン実行であることを説明。

### 図
