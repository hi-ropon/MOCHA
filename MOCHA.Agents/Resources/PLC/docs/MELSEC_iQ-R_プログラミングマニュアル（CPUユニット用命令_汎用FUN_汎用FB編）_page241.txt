### テキスト
#### 機能
(d)で指定したBIN32ビットデータと、(s)で指定したBIN32ビットデータの加算を行い、その結果を(d)で指定したデバイスに格納します。

- 加算結果にオーバーフローが発生した場合でも桁上げは無視されます。
- この場合、SM700（キャリフラグ）はONしません。

#### エラー
演算エラーはありません。

### 図
- BIN32ビットデータの加算例（符号付き・符号なし）
  - 例1: 567,890 (BIN) + 123,456 (BIN) = 691,346 (BIN)
  - 例2: 12,345,678,90 (BIN) + 9,876,543,21 (BIN) = -2,072,745,085 (BIN)（オーバーフロー発生時）
  - 例3: -12,345,678,90 (BIN) + -9,876,543,21 (BIN) = 2,072,745,085 (BIN)（オーバーフロー発生時）
  - 例4（符号なし）: 3,456,789,012 (BIN) + 1,234,567,890 (BIN) = 3,938,960,06 (BIN)

各図はビット位置（b31～b0）を示し、加算前後の値をBIN（2進数）表記で示しています。
