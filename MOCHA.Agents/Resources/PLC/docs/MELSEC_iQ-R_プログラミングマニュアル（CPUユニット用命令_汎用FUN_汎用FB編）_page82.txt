### テキスト・表
#### ダブルワードデータの1ダブルワード右シフト、左シフト命令
- 命令コード: DDSFR / DDSFRP（右シフト）、DDSFL / DDSFLP（左シフト）
- 処理内容: 指定したデバイスからダブルワード分のデータを1ダブルワード分右または左にシフトする。シフト後の空きデータには0が格納される。
- 参照先: DDSFRは358ページ、DDSFLは360ページ

#### 単精度実数n点の1点右シフト、左シフト命令
- 命令コード: ESFR / ESFRP（右シフト）、ESFL / ESFLP（左シフト）
- 処理内容: 指定したデバイスから単精度実数n点分のデータを1点右または左にシフトする。シフト後の空きデータには0が格納される。
- 参照先: ESFRは362ページ、ESFLは364ページ

#### 倍精度実数n点の1点右シフト、左シフト命令
- 命令コード: EDSFR / EDSFRP（右シフト）、EDSFL / EDSFLP（左シフト）
- 処理内容: 指定したデバイスから倍精度実数n点分のデータを1点右または左にシフトする。シフト後の空きデータには0が格納される。
- 参照先: EDSFRは366ページ、EDSFLは368ページ

### 図解説明
- 各命令の処理内容の下に、シフト動作を示す図がある。
- 図はデータの並びを示す長方形のブロックで表現されており、(n)はデータ数、(d)は開始位置を示す。
- 右シフトの場合はデータが右に1単位ずれ、左端に0が入る。
- 左シフトの場合はデータが左に1単位ずれ、右端に0が入る。
- 倍精度実数のシフトでは、開始位置(d)が(d)+3～(d)と表記され、倍精度のデータ幅を示している。

以上より、本ページは各種データ型に対するシフト命令の概要と動作を表形式と図解でわかりやすく説明している内容となっています。
