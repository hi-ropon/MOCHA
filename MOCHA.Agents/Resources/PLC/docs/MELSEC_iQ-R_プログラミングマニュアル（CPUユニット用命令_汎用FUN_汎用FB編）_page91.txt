### テキスト
#### 構造化命令
- ■FOR～NEXT  
  - 命令記号: FOR, NEXT  
  - 処理内容: FOR～NEXT命令間の処理を(n)回実行する。  
  - 参照先: 574ページ FOR, NEXT

- ■FOR～NEXT強制終了  
  - 命令記号: BREAK, BREAKP  
  - 処理内容: FOR～NEXT命令による繰り返し処理を強制的に終了し、指定したポインタに実行を移す。  
  - 参照先: 576ページ BREAK(P)

- ■サブルーチンプログラムコール  
  - 命令記号: CALL, CALLP  
  - 処理内容: 入力条件成立で(P)のサブルーチンプログラムを実行する。((s1)～(s5)はサブルーチンプログラムへの引数)  
  - 参照先: 578ページ CALL(P)

- ■サブルーチンプログラムからのリターン  
  - 命令記号: RET  
  - 処理内容: サブルーチンプログラムの終了を示す。  
  - 参照先: 582ページ RET

- ■サブルーチンプログラムの出力OFFコール  
  - 命令記号: FCALL, FCALLP  
  - 処理内容: 入力条件不成立時に(P)のサブルーチンプログラムの非実行処理を行う。((s1)～(s5)はサブルーチンプログラムへの引数)  
  - 参照先: 583ページ FCALL(P)

- ■プログラムファイル間サブルーチンコール  
  - 命令記号: ECALL, ECALLP  
  - 処理内容: 入力条件成立で指定プログラムの(P)の、サブルーチンプログラムを実行する。((s1)～(s5)はサブルーチンプログラムへの引数)  
  - 参照先: 587ページ ECALL(P)

- ■プログラムファイル間サブルーチンプログラム出力OFFコール  
  - 命令記号: EFCALL, EFCALLP  
  - 処理内容: 入力条件不成立時に指定プログラムの(P)の、サブルーチンプログラムの非実行処理を行う。((s1)～(s5)はサブルーチンプログラムへの引数)  
  - 参照先: 592ページ EFCALL(P)

- ■サブルーチンプログラムコール  
  - 命令記号: XCALL  
  - 処理内容: 入力条件成立で(P)のサブルーチンプログラムを実行し、入力条件不成立時に非実行処理を行う。((s1)～(s5)はサブルーチンプログラムへの引数)  
  - 参照先: 596ページ XCALL

#### プログラム制御用命令
- ■プログラム待機  
  - 命令記号: PSTOP, PSTOPP  
  - 処理内容: 指定したプログラムを待機タイプにする。  
  - 参照先: 601ページ PSTOP(P)

- ■プログラム出力OFF待機  
  - 命令記号: POFF, POFFP  
  - 処理内容: 指定したプログラムのOUT命令のコイルをOFFして、待機タイプにする。  
  - 参照先: 603ページ POFF(P)
</contents>
