### テキスト
#### データベースエクスポート(DBEXPORT)
指定したデータベースに設定されているデータをUnicodeテキストファイルに書き出す命令です。  
使用できないCPUはR00CPU、R01CPU、R02CPUです。

#### 実行条件
- DBEXPORT：立ち上がりエッジで実行
- DBEXPORTP：立ち下がりエッジで実行

### 図
- **ラダー図**  
  EN（入力）、S（データベースフォルダパス）、d1（完了デバイス）、d2（完了ステータス）、ENO（実行結果）が接続された命令ブロックの形状を示しています。

- **FBD/LD図**  
  EN（Enable）、S（フォルダパス）、d1（完了デバイス）、d2（完了ステータス）、ENO（Enable Out）端子を持つ命令ブロックの図。
