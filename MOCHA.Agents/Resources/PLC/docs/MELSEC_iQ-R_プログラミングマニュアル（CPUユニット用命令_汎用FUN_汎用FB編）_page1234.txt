### テキスト
#### PID制御命令(不完全微分)
不完全微分は、微分項の入力に一次遅れフィルタを入れたPID制御方式です。以下の場合に有効とされています。
- 高周波ノイズの影響を受けやすい制御の場合
- 完全微分方式でステップ状変化があったときに操作端を動作させるだけの有効なエネルギーが与えられない場合

### 図
#### 不完全微分の構成図
入力PV（プロセス変数）に対して、一次遅れフィルタを通し、その後微分項の演算を行う構成です。微分ゲインは1/ηで減衰されます。

#### PID演算ブロック図
- SV（設定値）とPV（測定値）の差を入力とし、
- 比例項（Kp）、積分項（1/Ti・s）、不完全微分項（Td×s/(1+ (Td/Kd)・s)）の3つの演算ブロックを経て、
- 制御対象Pに対して操作量MVを出力します。
- 外乱Wや検出雑音Vも考慮されています。
