<overview>
このページは、ユニットアクセス命令における使用可能デバイスの一覧表、機能説明、CPUユニットの先頭入出力番号の指定方法について構成されています。主に表形式でデバイスの種類と対応オペランドを示し、図でユニット構成とアドレスの関係を説明し、最後にCPUユニットごとの先頭入出力番号を表でまとめています。
</overview>

### テキスト
#### 使用可能デバイス
ユニットアクセス命令で使用可能なデバイスの種類を表形式で示しています。ビット、ワード、ダブルワード、間接指定、定数、その他のカテゴリに分かれており、各オペランド（U/H、s1、s2、n）に対して使用可否が○や―で示されています。

#### 機能
(U/H)はユニットおよびCPUユニットの先頭入出力番号を16進数4桁で表したときの上3桁で指定することを説明しています。

#### CPUユニットの先頭入出力番号指定
ユニット構成図が示されており、CPUユニットの先頭入出力番号はK4またはH4で指定することが記載されています。

### 図
CPUユニットの構成図があり、PW、CPU、RX10などのユニットが並んでいます。各ユニットの先頭入出力番号が16進数で示されており、CPUユニットの先頭入出力番号は0040Hの位置にあります。
