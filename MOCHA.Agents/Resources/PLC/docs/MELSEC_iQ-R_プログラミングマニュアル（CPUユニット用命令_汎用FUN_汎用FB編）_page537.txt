### テキスト
#### 機能
- (s)で指定されたデバイスのBIN16ビットデータと同一のデータを、(d)で指定されたデバイスへ(n)点分転送する。
- (s)がワードデバイスで(d)がビットデバイスの場合、ワードデバイスはビットデバイスの桁指定で指定されたビット数が対象となる。

##### 例
- (d)にK1Y30が指定された場合、(s)で指定されたワードデバイスの下位4ビットが対象となる。
- (s)、(d)が両方ともビットデバイスの場合は、(s)、(d)の桁数を同一にする必要がある。

#### エラー
- 演算エラーは発生しない。

### 図
- 左側に(s)で指定されたデバイスの16ビットデータ（例：3456H）が示されている。
- 右側に(d)で指定されたデバイスの連続したアドレスに同じデータがn点分転送される様子が示されている。
- 例の図では、(s)のワードデバイスD100の下位4ビット「1011」が(d)のビットデバイスK1Y30の4ビットに対応して転送されるイメージが描かれている。
