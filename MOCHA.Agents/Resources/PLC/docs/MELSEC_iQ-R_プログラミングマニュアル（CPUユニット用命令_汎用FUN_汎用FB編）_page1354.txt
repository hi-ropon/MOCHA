#### エラー
- (d)がWORD型の場合  
  エラーコード「3401H」が発生する条件は、(s)で設定した符号付きBIN32ビットデータが-32768～32767の範囲外のとき、または(s)のデータが0～9999以外のときです。

- (d)がDWORD型の場合  
  エラーコード「3401H」が発生する条件は、(s)のデータが0～99999999以外のときです。

#### 注意事項
* ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となります。その場合は(d)から出力されたデータを使用しないようプログラムの処理を行ってください。
