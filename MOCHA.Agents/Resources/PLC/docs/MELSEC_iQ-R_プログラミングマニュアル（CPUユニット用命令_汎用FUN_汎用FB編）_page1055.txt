### テキスト
#### 注意事項
以下の場合に、完了デバイス(d2)+1の異常完了信号がONとなり、(d3)の完了ステータスにエラーコードが格納されます。
- データベースアクセス命令実行中にDBSELECT(P)命令を実行した場合
- (s1)で不正な識別番号が指定された場合
- (s1)で未オープンの識別番号を指定した場合
- ファームウェアバージョンが"28"より前のシーケンサCPUで、(s1)にファームウェアバージョンが"28"以降のシーケンサCPUで構築または操作したデータベースを指定した場合
- (s2)で指定したテーブル名が存在しない場合
- (s3)で検索するフィールド数が最大値を超えた場合
- (s3)+1～(s3)+ロで指定したフィールド名に範囲外のフィールド名を設定した場合
- (s4)で検索するレコード数が最大値を超えた場合
- (s4)で検索する対象のフィールドのサイズ値が範囲外の場合
- データベースのSELECT処理に失敗した場合
- (d1)に出力するレコードの件数が、(s4)で指定した件数を上回った場合
- (d1)に出力するレコードのサイズが、(s4)で指定したサイズを上回った場合
- (s4)で指定する条件間演算値が範囲外の場合
- (s4)+37で指定する条件間演算値が1または2で、(s4)+1に0を設定している場合
- (s4)に0を設定し、(s4)+1に0以外の値を設定した場合
