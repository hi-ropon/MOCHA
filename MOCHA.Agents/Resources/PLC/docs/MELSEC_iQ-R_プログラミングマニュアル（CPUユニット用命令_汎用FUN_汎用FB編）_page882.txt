### テキスト
#### 符号なしBIN16ビットデータ→単精度実数変換
命令「UINT2FLT(P)」は符号なしBIN16ビットデータを単精度実数に変換します。  
対応エンジニアリングツールのバージョンは1.035M以降です。

##### 実行条件
- UINT2FLT命令は立ち上がりパルスで実行
- UINT2FLTP命令は立ち上がりレベルで実行

##### 機能
(s)で指定した符号なしBIN16ビットデータを単精度実数に変換し、(d)に格納します。

##### エラー
演算エラーはありません。

#### 動作例
- MOVP命令でK1234をD0に設定し、UINT2FLT命令でD0の16ビット符号なし整数（04D2H=1234）を単精度実数に変換し、D100に格納。
- 変換後のD100は449AH 4000Hの16進数で表され、これは単精度実数1234.0を示す。
