### 表
#### 命令処理時間一覧
この表は各命令名に対して、条件別の処理時間（単位：マイクロ秒）がCPUユニットごとに最小値と最大値で示されています。

- **命令名**: 各行に命令名が記載されています（例: ANDD<>, ANDD>, ORD= など）。
- **条件**: 無実行時、実行時（導通時/非導通時）などの条件が記載されています。
- **処理時間(μs)**: 
  - R00CPU、R01CPU、R02CPU、左記以外のCPUユニットに分けて、最小値と最大値が示されています。
  - 例えば、ANDD<>命令の無実行時の処理時間は、R00CPU・R01CPUで最小0.12544μs、最大0.01568μs、左記以外のCPUユニットで最小0.00392μsとなっています。
- 条件は主に「無実行時」と「実行時:導通時/非導通時」に分かれており、実行時の方が処理時間が長い傾向にあります。

この表はCPUの種類や命令の実行状態による処理時間の違いを詳細に示しており、命令の性能評価や最適化に役立つ情報となっています。
</contents>
