### テキスト
#### 機能
指定ファイルからデータを読み出す機能について説明しています。コントロールデータの実行や完了タイプにより、ファイルの読出し形式を指定可能です。

- 読出し対象はSDメモリカードのみ。
- 処理完了(d3)のビットデバイスは、SP.FREAD命令の処理完了を検出したスキャンのEND命令実行時に自動的にONし、次のスキャンのEND命令でOFFになる。
- SP.FREAD命令が異常完了した場合は、異常完了(d3)+1のデバイスが処理完了(d3)のデバイスと同じタイミングでON/OFFされる。
- SP.FREAD命令実行中はSM753（ファイルアクセス中）がONになる。
- SM753がON中はSP.FREAD命令は実行できず、実行した場合は無処理となる。
- 命令実行時にエラーを検出した場合は、処理完了(d3)、異常完了(d3)+1およびSM753はONしない。
- 奇数バイト指定時のデータ格納方法についても説明があり、指定した最後のb8～b15には命令実行前のデータが格納されている。

### 図（ビット配置表）
- データは(d2)から(d2)+5までの6ワードに格納される。
- 各ワードはb0からb15までのビットで構成されている。
- 奇数バイト指定時は、指定した最後のb8～b15には命令実行前のデータが格納されている。

この図は、データのビット単位の格納位置を示しており、奇数バイト指定時のデータの扱いを視覚的に理解できるようになっています。
