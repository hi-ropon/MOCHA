### テキスト
#### 機能
(d)で指定されたデバイスのBIN32ビットデータと、(s)で指定されたデバイスのBIN32ビットデータのビットごとの論理和を求め、その結果を(d)で指定されたデバイスに格納します。

- ビットデバイスの場合、桁指定による点数以降のビットデバイスは0として演算します。

#### エラー
演算エラーはありません。

### 図
図は、(d)と(s)で指定された32ビットのビットデータのビットごとの論理和演算の流れを示しています。

- 上段に(d)のビット列（b31からb0まで）が示され、その下に(s)のビット列が並びます。
- それぞれのビット列は1と0のビットで構成されており、論理和（OR）演算が行われます。
- 演算結果は(d)のビット列に格納され、各ビットは(d)または(s)のいずれかが1であれば1となります。
- 図中のビット位置はb31（最上位ビット）からb0（最下位ビット）まで示されており、ビットごとの論理和の結果が視覚的に理解できるようになっています。
