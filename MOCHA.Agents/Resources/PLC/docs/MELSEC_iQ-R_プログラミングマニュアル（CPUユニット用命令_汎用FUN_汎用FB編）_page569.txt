### テキスト
#### 機能
- (s)で指定されたデバイスから16点分のビットパターンにより、指定された割込みポインタ番号の割込みプログラムを実行許可状態または実行禁止状態にします。
  - 1(ON): 割込みプログラムの実行許可状態
  - 0(OFF): 割込みプログラムの実行禁止状態
- 各ビットに対応する割込みポインタ番号は表の通り。

- 電源投入時またはCPUユニットをリセットした場合、I0～I31、I44～I1023の割込みプログラムは実行許可状態になり、I32～I43の割込みプログラムは実行禁止状態になる。
- (s)～(s)+15のデバイスの状態は、SD1400～SD1463(IMASK命令マスクパターン)に格納される。

#### Point
- IMASK命令では、I0～I255の割込みポインタを一括で実行許可状態または実行禁止状態にできる。
- I256～I1023の割込みポインタを実行許可状態または実行禁止状態にする場合は、プログラムでIMASK命令をSIMASK命令へ代替する必要がある。
- SIMASK命令の詳細は570ページ参照。
