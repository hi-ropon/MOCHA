### テキスト
#### パルス否定演算開始、パルス否定直列接続、パルス否定並列接続命令
- **LDPI**: 立上りパルス否定演算開始命令。指定デバイスがOFF時、ON時、立下り時(ON→OFF)の場合に導通する。
- **LDFI**: 立下りパルス否定演算開始命令。指定デバイスの立上り時(OFF→ON)、OFF時、ON時に導通する。
- **ANDPI / ANDFI**: 立上り/立下りパルス否定直列接続命令。これまでの演算結果とAND演算を行う。
- **ORPI / ORFI**: 立上り/立下りパルス否定並列接続命令。これまでの演算結果とOR演算を行う。

### ラダー図
- 各命令のラダー回路図が示されている。LDPI、LDFIは単独のパルス否定演算開始接点、ANDPI、ANDFIは直列接続、ORPI、ORFIは並列接続の形で表現されている。

### ST言語
- 各命令に対応するST言語の記述例が示されている。
```
ENO := LDPI(EN,s);
ENO := LDFI(EN,s);
ENO := ANDPI(EN,s);
ENO := ANDFI(EN,s);
ENO := ORPI(EN,s);
ENO := ORFI(EN,s);
```

### FBD/LD
- 機能ブロック図として、EN（入力）、ENO（出力）、s（パルス入力）を持つブロックの図が示されている。
