### テキスト
#### 機能：演算処理
(s1)に入力された文字列の任意位置から指定文字数分のデータを、(s2)に入力された文字列へ置換し、(d)から出力します。  
- 置換する文字数は(n1)の入力値で指定  
- 置換開始位置は(n2)の入力値で指定  

#### 入力値の範囲
- (s1)、(s2)はSTRING型で0～255バイトの範囲  
- (n1)はINT型で0～255の範囲（ただし、(s1)の文字数以内）  
- (n2)はINT型で1～255の範囲（ただし、(s1)の文字数以内）  

#### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力される。  
2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通り。

### 図
#### 文字列置換の例（(n1)、(n2)が5の場合）
- (s1)に「ABCDEFGH123」が入力されているメモリ配置（上位バイト・下位バイトで16進数表記）  
- (s2)に「123456」が入力されているメモリ配置  
- (n2)＝5文字目から、(n1)＝5文字分を(s2)の文字列で置換  
- 置換後の(d)のメモリ配置が示されており、5文字目以降が「12345」に置換されていることがわかる  

### 表
| 実行条件 | 演算結果 ENO | 演算結果 (d) |
| :--- | :---: | :--- |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOがFALSEの場合、(d)から出力されるデータは不定値となるため、(d)のデータを使用しないようプログラムで処理する必要がある。
