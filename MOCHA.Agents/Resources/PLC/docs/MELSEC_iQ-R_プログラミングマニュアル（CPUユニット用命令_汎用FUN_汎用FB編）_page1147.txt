本ページは「パラメータ設定操作」の一部で、主に転送先ネットワークNo.に関するルーチングデータの登録方法とその内容の説明、及び関連するエラーコードの説明で構成されています。ページ上部にテキスト説明があり、中段に2つの表（転送先デバイスの内容とネットワーク種別の指定範囲）、下部にエラーコードの説明表が配置されています。グラフや図は含まれていません。
</overview>

### テキスト
#### 機能
- パラメータの(s1)で指定される転送先ネットワークNo.のエリアに、(s2)以降のルーチングデータを登録する。
- (s2)以降に格納されるデータの内容は表に示されている。
- (s2)+1の指定範囲は別表の通り。
- (s1)で指定される転送先ネットワークNo.のデータがパラメータで設定されている場合、(s2)以降のデータに変更される。
- (s2)以降のデータ((s2)～(s2)+2)がすべて0の場合、(s1)で指定される転送先ネットワークNo.のデータはパラメータより削除される。

### 表1：先頭デバイスの内容
| 先頭デバイス | 項目           | 範囲          |
|--------------|----------------|---------------|
| (s2)         | 中継先ネットワークNo. | 1～239        |
| (s2)+1       | 中継先局番     | 下表参照      |
| (s2)+2       | ダミー         | －            |

### 表2：ネットワーク種別の指定範囲
| ネットワーク種別                         | 指定範囲    |
|----------------------------------------|------------|
| MELSECNET/H                           | 1～64      |
| CC-Link IEコントローラネットワーク       | 1～120     |
| CC-Link IEフィールドネットワーク(マスタ局) | 125固定    |
| CC-Link IEフィールドネットワーク(ローカル局) | 1～120     |
| CC-Link IE TSN(マスタ局)               | 125固定    |
| CC-Link IE TSN(ローカル局)             | 1～120     |

### 表3：エラーコード(SD0)
| エラーコード | 内容                                                                                     |
|--------------|------------------------------------------------------------------------------------------|
| 3405H        | - (s1)が1～239以外のとき。                                                               |
|              | - (s2)以降のデータが下記の範囲を超えているとき。                                          |
|              |   ・(s2): 1～239                                                                        |
|              |   ・(s2)+1: 1～120、125                                                                 |
|              | - ネットワークユニットのパラメータで登録したルーチング情報の登録数と、RTWRITE命令で登録したルーチング情報の登録数の合計が最大登録数238を超えたとき。 |
|              | - パラメータに登録されていない転送先ネットワークNo.を削除指定したとき。                   |
|              | - (s2)、(s2)+1のどちらか一方だけに0指定している場合。                                   |
