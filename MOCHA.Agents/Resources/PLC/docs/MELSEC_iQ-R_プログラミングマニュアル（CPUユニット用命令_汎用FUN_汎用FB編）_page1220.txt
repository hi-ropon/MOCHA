### テキスト
#### オートチューニング
PID制御で良好な制御結果を得るためには、制御対象に合った各定数（比例ゲイン、積分時間、微分時間）の最適値を求める必要があります。オートチューニング機能は、これらの重要な定数を自動的に設定し、PID制御を最適化するための機能です。

PID演算命令のオートチューニングには「リミットサイクル法」と「ステップ応答法」の2種類があり、このページではリミットサイクル法の手順を説明しています。

**Point**
- オートチューニングはシステムが安定した状態から開始する必要があり、不安定な状態から開始すると正しく動作しない場合があります。

#### リミットサイクル法
リミットサイクル法の概要は別ページ（1582ページ）に記載されています。

##### オートチューニング（リミットサイクル法）で設定される制御用データ
- 比例ゲイン(Kp): (s3)+3
- 積分時間(Ti): (s3)+4
- 微分時間(Td): (s3)+6

##### オートチューニング手順
1. **正動作/逆動作の設定**  
   (s3)+1(動作設定(ACT))のビット0で正動作か逆動作かを設定。

2. **リミットサイクル法の選択**  
   (s3)+1(動作設定(ACT))のビット6をONにしてリミットサイクル法を選択。OFF時はステップ応答法。

3. **オートチューニング実行フラグのON**  
   (s3)+1(動作設定(ACT))のビット4をON。

4. **入力フィルタの設定**  
   (s3)+2(入力フィルタ(α))を設定。

5. **サンプリングタイムの設定**  
   (s3)(サンプリングタイム(Ts))を設定。

6. **出力値上限(ULV)の設定**  
   (s3)+26(出力値上限(ULV))に最大出力値を設定。

7. **出力値下限(LLV)の設定**  
   (s3)+27(出力値下限(LLV))に最小出力値を設定。

8. **PV値スレッショルド(ヒステリシス)幅(SHPV)の設定**  
   (s3)+25(PV値スレッショルド(ヒステリシス)幅(SHPV))を設定。

9. **目標値(SV)の設定**  
   PID演算命令の(s1)に目標値(SV)を設定。

10. **オートチューニングの開始**  
    PID演算命令の起動接点をONにすると、測定値(PV)によりオートチューニングが開始。  
    チューニング完了時点で、(s3)+1(動作設定(ACT))のオートチューニングフラグ（ビット4、ビット6）がOFFになる。
