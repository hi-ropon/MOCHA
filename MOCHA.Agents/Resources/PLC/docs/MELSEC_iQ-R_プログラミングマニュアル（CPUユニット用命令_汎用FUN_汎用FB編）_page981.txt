### テキスト
#### 乱数命令
乱数命令は0～32767未満の乱数を発生させ、指定したデバイスに格納します。RND(P)命令は特定の算出式に従って乱数を生成し、前回の算出結果を係数として使用します。

#### 機能
0～32767の乱数を発生し、(d)で指定されたデバイスに格納します。RND(P)命令は算出式に従い乱数を生成します。

#### エラー
演算エラーはありません。

### 図
- **ラダー図**  
  (d)デバイスに乱数を格納する命令のラダー表現が示されています。

- **ST言語例**  
  ```
  ENO:=RND(EN,d);
  ENO:=RNDP(EN,d);
  ```
  と記載され、RNDおよびRNDP命令の使用例が示されています。

- **FBD/LD**  
  EN（実行条件）、ENO（実行結果）、d（乱数格納デバイス）を持つ命令ブロックの図が示されています。
