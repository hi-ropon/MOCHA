### テキスト
#### 機能
- (d)で指定したデバイスから単精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトする。
- 最下位から(n2)点分は0になる。
- (n1)または(n2)で指定した値が0のときは無処理となる。
- (n1)≦(n2)の場合は、(d)で指定したデバイスから(n1)点分のデータはすべて0となる。

#### エラー
- 演算エラーはありません。

### 図
- 図は(n1)=9、(n2)=4の場合の例を示している。
- 上段に元のデータ配列があり、(d)から始まるデバイスアドレスに対応した9点の単精度実数データが並んでいる。
- 下段は左へ4点シフトした後のデータを示し、シフトされた分の右端4点は0に置き換わっている。
- 矢印と括弧で(n1)と(n2)の範囲を示し、どの範囲のデータがシフトされるかを視覚的に説明している。

以上より、このページはデータシフト命令の動作原理と具体例をテキストと図でわかりやすく解説している。
