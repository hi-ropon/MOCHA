### テキスト
#### nダブルワードデータの1ダブルワード右シフト（DDSFR(P)）
- 指定したデバイスからnダブルワード分を右へ1ダブルワードシフトします。シフト後の空きデータには0が格納されます。
- 対応CPUはR00CPU、R01CPU、R02CPUで、ファームウェアバージョン"06"以降で使用可能。エンジニアリングツールはバージョン"1.047Z"以降で使用可能。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）とRnENCPUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降で使用可能。

#### 実行条件
- DDSFR命令は立下りエッジで実行
- DDSFRP命令は立上りエッジで実行

### 図
- ラダー図：EN入力、ENO出力、n（シフトするデータ数）、d（シフトする先頭デバイス）を示す。
- ST言語例：
  ```
  ENO:=DDSFR(EN,n,d);
  ENO:=DDSFRP(EN,n,d);
  ```
- FBD/LDブロック図：EN、ENO、n、dの入出力端子を持つ命令ブロックを示す。
