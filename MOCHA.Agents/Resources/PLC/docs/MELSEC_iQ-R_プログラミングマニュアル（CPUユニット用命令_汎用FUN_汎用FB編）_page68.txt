### テキスト
#### 2.2 基本命令
本節では比較演算命令について説明しています。

##### ■BIN16ビットデータ比較
指定した2つのBIN16ビットデータをa接点扱いで比較します。

##### ■BIN32ビットデータ比較
指定した2つのBIN32ビットデータをa接点扱いで比較します。

##### ■BIN16ビットデータ比較出力
(s1)で指定したBIN16ビットデータと、(s2)で指定したBIN16ビットデータを比較し、その結果（小、 一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONします。
