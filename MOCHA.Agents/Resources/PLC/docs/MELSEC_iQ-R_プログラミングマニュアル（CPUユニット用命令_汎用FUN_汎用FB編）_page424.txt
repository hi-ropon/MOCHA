### テキスト
#### 単精度実数→符号なしBIN32ビットデータ
命令名：FLT2UDINT(P)

指定した単精度実数を符号なしBIN32ビットデータに変換する命令です。

対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

#### 実行条件
- FLT2UDINT命令は立ち上がりエッジで実行
- FLT2UDINTP命令は立ち上がりエッジで実行

### 図
- ラダー図：入力(s)と出力(d)の関係を示すシンプルな命令ブロック
- FBD/LD図：EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）を持つ命令ブロック
