### テキスト
#### 機能
- 指定したデバイスから(n)点分を左へ1ビットシフトさせる動作を行います。
- 最下位ビットから1ビット分は0になります。
- SM700では、シフト範囲の一番左の1ビットが格納されます。
- (n)に0を指定した場合は処理されません。

#### エラー
- 演算エラーはありません。

### 図
- ビットシフトの動作を示す図が2つあります。
  - 上段は元のビット配列で、(d)から(d)+(n)-1までのビットが示されています。
  - 下段はSM700のシフト後のビット配列で、左へ1ビットシフトされた結果が示されています。
  - 具体的には、元のビット列「1100...011」が左に1ビットシフトされ、「1001...110」となり、最下位ビットは0に設定されていることがわかります。
