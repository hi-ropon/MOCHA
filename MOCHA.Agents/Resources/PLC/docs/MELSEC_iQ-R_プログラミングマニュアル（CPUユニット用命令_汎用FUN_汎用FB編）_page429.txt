### テキスト
#### 機能
- (s)で指定された倍精度実数を符号なしBIN16ビットデータに変換し、(d)で指定されたデバイスに格納する。
- 変換後のデータは、(s)で指定された倍精度実数の小数点以下1桁目を四捨五入した値となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### プログラム例の説明
- M0がONしたとき、D0～D3に格納された倍精度実数を符号なしBIN16ビットデータに変換し、D100へ格納する。

### 図
- プログラム例の動作を示す図で、SM402の接点、M0の接点、EDMOVP命令、DBL2UINT命令の流れが描かれている。
- EDMOVP命令はE5432.1の倍精度実数をD0に移動。
- DBL2UINT命令はD0の倍精度実数を符号なしBIN16ビットデータに変換し、D100に格納。
- ビット構成の説明もあり、D3～D0の各ビットがどのように16ビットデータに変換されるかを示している。
