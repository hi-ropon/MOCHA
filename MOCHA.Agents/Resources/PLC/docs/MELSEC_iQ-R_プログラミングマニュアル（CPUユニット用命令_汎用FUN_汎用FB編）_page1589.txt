### テキスト
#### 付5 PID制御プログラム例
PID制御命令を使用したPID制御プログラムの例を示しています。プログラム例のシステム構成について説明しています。

#### 自動モードによるPID制御プログラム例
R60AD4から取り込んだデジタル値を測定値（PV）としてPID演算を行い、PID演算で求めた操作量（MV）をR60DA4から出力して外部機器を制御するプログラム例です。プログラム条件を表で示しています。

### 図
CPUユニット内のPID演算構成図が示されています。  
- PID演算はループ1とループ2の2つがあり、それぞれPV（測定値）とMV（操作量）がR60DA4のCH.1、CH.2に接続されています。  
- R60DA4のCH.1、CH.2は外部機器に接続され、外部機器からの信号が再びR60AD4のCH.1、CH.2に戻るループ構成です。  
- R60AD4の入出力番号はX/Y80～X/Y8F、R60DA4の入出力番号はX/YA0～X/YAFとなっています。
