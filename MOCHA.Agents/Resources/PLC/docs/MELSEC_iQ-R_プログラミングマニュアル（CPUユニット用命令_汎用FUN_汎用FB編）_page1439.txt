#### エラー
- (s1)、(s2)がINT型/WORD型の場合
  - エラーコード「3400H」: (s2)で指定した値（除数）が0のときに発生するエラー。

- (s1)、(s2)がDINT型/DWORD型の場合
  - エラーコード「3400H」: (s2)で指定した値（除数）が0のときに発生するエラー。
