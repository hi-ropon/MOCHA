### テキスト
#### 機能
- データメモリ上のデバイスデータ格納用ファイルの指定した読み出しオフセット(s)から、指定された点数(n)分のデバイスデータを読み出し、指定したデバイス(d)に格納する。
- 読み出しオフセット(s)は、デバイスデータ格納用ファイル先頭からのオフセットで、ワードオフセット（16ビットごとに+1する単位）で指定する。
- S(P).DEVLD命令を使用する場合は、デバイスデータ格納用ファイル設定が必要（627ページ参照）。
- デバイスデータ格納用ファイルへの書き込みはSP.DEVST命令で行う（630ページ参照）。

### 図
- 左図はデータメモリ内のデバイスデータ格納用ファイルから、読み出しオフセット(s)で指定した位置からn点分のデータを読み出し、デバイス(d)に格納する流れを示す。
- 右図はデバイスデータ格納用ファイルの読み出しオフセットのイメージで、16ビット単位で+0、+1、+2…とオフセットが増加していることを示している。

| 3405H | (n)の設定データが0のとき。 |
