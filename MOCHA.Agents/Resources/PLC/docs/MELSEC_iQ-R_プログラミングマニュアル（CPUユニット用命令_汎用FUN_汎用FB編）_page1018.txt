### テキスト
#### 傾斜信号命令 RAMPQ
指定した初期値から最終値に向けて、指定回数(n回)で値を移行させる命令です。

#### 実行条件
RAMPQ命令は立ち上がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー図では、(s1)、(s2)、(d1)、(n)、(d2)の5つのオペランドを持つ命令として表現されています。  
- **FBD/LD図**  
  ファンクションブロックとしてEN（入力）、ENO（出力）、s1、s2、d1、d2、nの各端子が示されています。

### 表
#### 設定データ：内容、範囲、データ型

| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (s1) | 初期値 | -32768～32767 | 符号付きBIN16ビット | ANY16 |
| (s2) | 最終値 | -32768～32767 | 符号付きBIN16ビット | ANY16 |
| (d1) | (d1)+0：現在値<br>(d1)+1：実行回数 | ― | 符号付きBIN16ビット | ANY16_ARRAY（要素数：2） |
| (n) | 移行回数 | 1～32767 | 符号付きBIN16ビット | ANY16 |
| (d2) | (d2)+0：完了デバイス<br>(d2)+1：完了時データ保持選択ビット | ― | ビット | ANYBIT_ARRAY（要素数：2） |
| EN | 実行条件 | ― | ビット | BOOL |
| ENO | 実行結果 | ― | ビット | BOOL |
