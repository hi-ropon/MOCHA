### テキスト
#### 単精度実数自然対数演算 LOG(P)
単精度実数で指定した値の自然対数(eを底とした対数)を演算します。

- LOG命令はST、FBD/LDでは対応していません。汎用ファンクションのLOGを使用してください（1419ページ参照）。
- 実行条件はLOGとLOGPの2種類があります。

#### ■実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| LOG | 立下りエッジ |
| LOGP | 立上りエッジ |

### 図
- ラダー図：LOGP命令のラダー回路例。EN入力、s（入力値）、d（出力値）を示す。
- FBD/LD図：EN（入力）、ENO（出力）、s（入力値）、d（出力値）を持つブロック図。
