### テキスト
#### 機能
- (s)で指定された角度のCOS（余弦）値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- (s)で指定するCOS値は-1.0～1.0の範囲で設定可能。
- (d)に格納される角度（演算結果）はラジアン単位。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)に指定した値が0、非正規化数、非数、±∞のとき。 |
| 3405H              | (s)に指定した値が-1.0～1.0の範囲外のとき。 |

#### Point
- ラジアンと角度の変換については、RAD(P)命令、DEG(P)命令を参照すること。
- 参考ページ：941ページ（RAD(P)）、943ページ（DEG(P)）

### 図
- COS⁻¹の入力と出力の関係を示す簡単な図があり、(s)+1、(s)が単精度実数のCOS値入力、(d)+1、(d)が単精度実数の角度出力を表している。
