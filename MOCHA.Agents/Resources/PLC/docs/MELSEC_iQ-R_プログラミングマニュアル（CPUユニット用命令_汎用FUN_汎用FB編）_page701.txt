### テキスト
#### 機能概要
(s2)で指定したスケーリング用変換データ（16ビットデータ単位）に対し、(s1)で指定した入力値からスケーリングを行い、(d)で指定されたデバイス番号に演算結果を格納します。スケーリング変換は、(s2)で指定したデバイス以降に格納したスケーリング用変換データに基づいて実施されます。

- 演算結果が整数値にならない場合は、小数点以下1桁目を四捨五入します。
- スケーリング用変換データのX座標データは昇順に設定してください。
- (s1)はスケーリング用変換データ範囲内((s2)のデバイス値)で設定してください。
- 複数のポイントが同一X座標を指定している場合は、ポイントNo.が最も大きいポイントのY座標値を出力します。
- (s2)で指定するスケーリング用変換データの座標ポイント数は1～65535にしてください。

### 表
#### 設定項目とデバイス割付
| 座標ポイント数 | (s2) |
| --- | --- |
| ポイント1 | X座標: (s2)+1<br>Y座標: (s2)+2 |
| ポイント2 | X座標: (s2)+3<br>Y座標: (s2)+4 |
| ... | ... |
| ポイントn | X座標: (s2)+2n-1<br>Y座標: (s2)+2n |

この表は、(s2)で指定する座標ポイント数に応じて、各ポイントのX座標とY座標がどのデバイスに割り付けられるかを示しています。

### 図（グラフ）
- X-Y座標平面上に複数のポイント（ポイント1、ポイント2、ポイント3、...、ポイントn）が配置されています。
- X軸は入力値(s1)の範囲を示し、Y軸はスケーリング変換後の値(d)を示します。
- 演算可能範囲はポイント1からポイントnまでのX座標間であり、その範囲外は演算エラーとなります。
- ポイント間は線で結ばれており、スケーリングはこの線形補間に基づいて行われます。
- 演算エラーはX軸の範囲外で発生し、グラフ上に「演算エラー」として示されています。
