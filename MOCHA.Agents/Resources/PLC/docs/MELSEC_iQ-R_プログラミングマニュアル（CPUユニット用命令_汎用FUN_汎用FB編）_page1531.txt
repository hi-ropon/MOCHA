### 表
#### 命令処理時間一覧
命令名ごとに、条件（n=1やn=96、実行時など）に応じた処理時間（単位はマイクロ秒）がCPUユニット別に示されています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例：BK-、DBK+、INCなど） |
| 条件 | 実行条件や繰り返し回数(n=1、n=96、実行時など) |
| 処理時間(μs) | R00CPU、R01CPU、R02CPU、その他CPUユニットごとの最小値と最大値 |

- R00CPU、R01CPU、R02CPUはそれぞれのCPUユニットの処理時間を示し、最小値と最大値が記載されています。
- 左記以外のCPUユニットについても最小値と最大値が示されています。
- 処理時間は命令の種類や条件によって大きく異なり、例えばBK-命令はn=1で最小3.1μs～最大8.7μs、n=96で最大22.9μs程度となっています。
- INCやDECなどの命令は「実行時」のみの記載で、処理時間は非常に短く0.00392μs～0.12544μs程度です。

この表はCPUの性能評価や命令実行の時間見積もりに利用されることを想定しています。
