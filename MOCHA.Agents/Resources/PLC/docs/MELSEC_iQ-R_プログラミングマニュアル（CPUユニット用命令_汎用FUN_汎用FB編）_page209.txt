### テキスト
#### 5.8 停止命令 シーケンスプログラム停止
CPUユニットの演算を停止する命令について説明しています。スイッチをSTOP側にした場合と同様の動作を行います。

##### 機能
- 実行指令がONになると、出力(Y)をリセットし、CPUユニットの演算を停止します。これはスイッチをSTOP側にした場合と同じ動作です。
- STOP命令実行後にCPUユニットの演算を再開させるには、スイッチをRUN→STOPに戻した後、再度RUNの位置にします。

##### 実行条件
- STOP命令は立ち上がりエッジ（パルス）で実行されます。

### 図
- **ラダー図**：停止命令はラダー図上で「ENO:=STOP(EN);」と記述され、入力ENに対して出力ENOが停止命令の実行結果を示します。
- **FBD/LD図**：停止命令はEN入力とENO出力を持つブロックとして表現されています。

### 表
#### エラーコード (SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 3340H | FOR命令実行後、NEXT命令実行前にSTOP命令を実行した場合 |
| 3381H | CALL(P)、FCALL(P)、ECALL(P)、EFCALL(P)、XCALL命令実行後、RET命令実行前にSTOP命令を実行した場合 |
| 33A1H | 割込みポインタ(II)による割込みプログラム中でIRET命令実行前にSTOP命令を実行した場合 |
| 33A3H | 定周期実行タイププログラム中でSTOP命令を実行した場合 |
| 33A4H | イベント実行タイププログラム中でSTOP命令を実行した場合 |

これらのエラーは、STOP命令の不適切なタイミングでの実行を示しています。
