### テキスト
#### 日時データの変換(秒→日時)
命令名は「SEC2DATE(P)(_U)」で、秒のデータを日時に換算する命令です。

対応CPUとファームウェアバージョンの条件が記載されており、以下の通りです。
- R00CPU、R01CPU、R02CPUはファームウェアバージョン"06"以降で使用可能。エンジニアリングツールはバージョン"1.047Z"以降。
- RnENCPIUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降。
- RnPCPUはファームウェアバージョン"21"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降。

#### 実行条件
- SEC2DATE、SEC2DATE_Uは立ち上がりエッジで実行。
- SEC2DATEP、SEC2DATEP_Uは立ち下がりエッジで実行。

#### ラダー図
秒データ(s)を入力し、日時データ(d)を出力する命令ブロックが示されています。
