### テキスト
#### 倍精度実数n点のn点右シフト、左シフト命令

- **SFTEDR / SFTEDRP**
  - 指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、右へ(n2)点分シフトする命令。
  - シフト後の空きデータには0が格納される。
  - 参照先は402ページのSFTEDR(P)。

- **EDSFTR / EDSFTRP**
  - 指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、右へ(n2)点分シフトする命令。
  - シフト後の空きデータには指定データが格納される。
  - 参照先は404ページのEDSFTR(P)。

- **SFTEDL / SFTEDLP**
  - 指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトする命令。
  - シフト後の空きデータには0が格納される。
  - 参照先は406ページのSFTEDL(P)。

- **EDSFTL / EDSFTLP**
  - 指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトする命令。
  - シフト後の空きデータには指定データが格納される。
  - 参照先は408ページのEDSFTL(P)。

### 図（ビットシフトの動作説明）
- 各命令の処理内容には、n1点分のデータ範囲とn2点分のシフト範囲を示す図が添えられている。
- 右シフトの場合は、データが右方向にずれ、空いた左側の領域に0または指定データが入る。
- 左シフトの場合は、データが左方向にずれ、空いた右側の領域に0または指定データが入る。
- 図中の(d)+3～(d)はデータのビット範囲を示し、(s)は指定データを表している。
