### テキスト
#### 倍精度実数除算
ED/(P)命令は倍精度実数を除算する命令です。

- 対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）
- バージョン対応：バージョン"1.035M"以降のエンジニアリングツールで対応

#### 実行条件
- ED/命令は立下りエッジで実行
- ED/P命令は立上りエッジで実行

### 図
#### ラダー図
- (s1)、(s2)の2つの入力オペランドと(d)の出力オペランドを持つ命令ブロックの形状を示す。

#### FBD/LD図
- EN（実行条件）、s1、s2（入力）、d（出力）を持つブロック図。
- ブロック内にはEDDIVISION、EDDIVISIONPが入る。
