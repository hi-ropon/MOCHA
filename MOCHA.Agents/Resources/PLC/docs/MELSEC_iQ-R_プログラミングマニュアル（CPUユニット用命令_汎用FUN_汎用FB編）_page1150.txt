### テキスト
#### トリガロギングリセット（LOGTRGR）
指定したロギング設定No.のトリガ条件をリセットする命令です。R00CPUでは使用できません。

#### 機能
- (s)で指定したロギング設定No.のLOGTRG命令をリセットします。
- LOGTRGR命令を実行すると、指定したトリガロギング設定No.のLOGTRG命令が無効になります。
- LOGTRGR命令を実行すると、(s)で指定したロギング設定No.の特殊リレー（ロギング完了、ロギングトリガ、ロギングトリガ後）がOFFになります。
- バッファ内のデータをSDメモリカードに保存中でも命令を実行可能ですが、すべてのデータ保存が完了するまで命令の処理は待たれます。

#### 注意事項
割込みプログラムでLOGTRG命令を使用するとトリガ条件が発生しない場合があります。
