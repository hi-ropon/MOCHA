### テキスト・表
#### nワードデータのnワード右シフト、左シフト命令
- **命令コード**: SFTWR / SFTWRP, WSFR / WSFRP, SFTWL / SFTWLP, WSFL / WSFLP
- **処理内容**:
  - SFTWR / SFTWRP: 指定したデバイスからnワード分のデータを右へnワードシフト。シフト後の空きデータには0が格納される。
  - WSFR / WSFRP: 指定したデバイスから(n1)ワード分のデータ範囲内で右へ(n2)ワードシフト。シフト後の空きデータには指定データが格納される。
  - SFTWL / SFTWLP: 指定したデバイスからnワード分のデータを左へnワードシフト。シフト後の空きデータには0が格納される。
  - WSFL / WSFLP: 指定したデバイスから(n1)ワード分のデータ範囲内で左へ(n2)ワードシフト。シフト後の空きデータには指定データが格納される。
- **参照先**: 各命令ごとに378ページ〜384ページの詳細説明が示されている。

#### nダブルワードデータのnダブルワード右シフト、左シフト命令
- **命令コード**: SFTDWR / SFTDWRP, DWSFTR / DWSFTRP
- **処理内容**:
  - SFTDWR / SFTDWRP: 指定したデバイスから(n1)ダブルワード分のデータ範囲内で右へ(n2)ダブルワードシフト。シフト後の空きデータには0が格納される。
  - DWSFTR / DWSFTRP: 指定したデバイスから(n1)ダブルワード分のデータ範囲内で右へ(n2)ダブルワードシフト。シフト後の空きデータには指定データが格納される。
- **参照先**: 386ページ〜388ページの詳細説明が示されている。

### 図（ビットシフト動作のイメージ）
- 各命令の説明に対応して、データのビットまたはワードの並びを示す図があり、シフト前後のデータ配置や空き部分に0または指定データが格納される様子を視覚的に表現。
- 図は(n1)、(n2)、(d)、(s)などのパラメータを用いてシフト範囲やシフト量を示している。
