### テキスト
#### BIN16ビットデータ不感帯制御
指定した入力値（BIN16ビット値）が不感帯の上下限範囲内か否かにより、出力値を制御する命令について説明しています。

#### 実行条件
命令ごとに実行条件が異なり、BANDおよびBAND_Uは立ち下がり、BANDPおよびBANDP_Uは立ち上がりで実行されることが示されています。

#### 設定データの内容、範囲、データ型
- (s1)：不感帯（無出力帯域）の下限値。符号付きBIN16ビット（-32768～32767）または符号なしBIN16ビット（0～65535）。
- (s2)：不感帯の上限値。符号付きまたは符号なしBIN16ビット。
- (s3)：不感帯制御により制御する入力値。符号付きまたは符号なしBIN16ビット。
- (d)：不感帯制御により制御した出力値を格納するデバイス。符号付きまたは符号なしBIN16ビット。
- EN：実行条件（ビット型）。
- ENO：実行結果（ビット型）。

### 図
#### ラダー図
- BAND命令のラダー図が示されており、s1、s2、s3の3つの入力とdの出力がある構造です。

#### FBD/LD図
- EN（入力）、s1、s2、s3（入力）、ENO（出力）、d（出力）を持つ命令ブロックの図が示されています。
