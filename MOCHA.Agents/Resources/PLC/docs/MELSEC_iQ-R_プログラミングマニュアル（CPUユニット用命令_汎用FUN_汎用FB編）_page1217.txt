### テキスト
#### 積分時間: (s3)+4
- 設定範囲は0～32767（×100ms）、0の場合は∞（積分なし）として扱う。
- PID制御の積分動作に関する説明。
- 積分動作は偏差発生後、積分動作の出力が比例動作の出力になるまでの時間を積分時間（Ti）で表す。
- Tiを小さくすると積分動作が強くなる。

#### 微分ゲイン: (s3)+5
- 設定範囲は0～200[%]。
- PID制御の微分動作に関する説明。
- 微分動作による出力にフィルタをかける。
- 微分ゲイン(KD)は微分動作のみに影響。
- KDを小さくすると外乱などによる測定値(PV)の変化に瞬時に応答。
- KDを大きくすると外乱などによるPVの変化に長時間かけて応答。

##### Point
- 最初は微分ゲイン(KD)を0に設定し、入力フィルタ(α)で調整すること。
- 出力の変化が外乱に対して応答が良すぎる場合は、KDの値を大きくする。

### グラフ
#### 冷房（正動作）の場合でのPI動作
- 上段グラフは温度(℃)の時間変化を示し、比例動作での測定値とPI動作での測定値の2つの線がある。
- 下段グラフは出力(MV)の時間変化を示し、比例動作での出力とPI動作での出力の2つの線がある。
- 積分時間(Ti)の関係は0 < TI3 < TI2 < TI1で示されている。
- グラフの横軸は時間(t)、縦軸はそれぞれ温度(℃)と出力(MV)。
- SVは目標値を示す。
