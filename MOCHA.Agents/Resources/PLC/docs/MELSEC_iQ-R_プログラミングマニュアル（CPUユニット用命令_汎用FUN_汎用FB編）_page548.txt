### テキスト
#### BIN16ビットデータ上下バイト交換
「SWAP(P)」命令は、指定したデバイスの上下各8ビットの値を交換する命令です。

#### 実行条件
- SWAP命令は立ち上がりエッジで実行されます。
- SWAPP命令は立ち下がりエッジで実行されます。

#### 機能
(d)で指定したデバイスの上下8ビットの値を交換します。

#### エラー
演算エラーはありません。
