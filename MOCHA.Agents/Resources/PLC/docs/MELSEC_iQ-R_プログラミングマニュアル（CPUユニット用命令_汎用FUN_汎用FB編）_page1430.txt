#### エラー

### 表
#### 実行条件と演算結果

| 実行条件 | 演算結果 |
| --- | --- |
| EN | ENO | (d) |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、(d)から出力されたデータを使用しないようプログラム処理を行う必要があります。

#### エラーコード（REAL型）

| エラーコード(SD0) | 内容 |
| --- | --- |
| 3402H | (s1)～(s28)で指定したデータが0、非正規化数、非数、±∞のとき。<br>(d)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が下記範囲を超えるとき（オーバーフロー発生時）<br>|(d)| < 2^128 |
