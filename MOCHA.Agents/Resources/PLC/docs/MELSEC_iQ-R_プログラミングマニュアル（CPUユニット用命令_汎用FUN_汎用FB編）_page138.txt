このページは主に表形式で構成されており、DINT型、BCD型、REAL型などのデータ型変換に関する汎用ファンクションの一覧を示しています。テキスト部分は各変換の説明と参照ページ番号を含み、表部分はファンクション記号、処理内容、参照先の3列で構成されています。グラフや図は含まれていません。

### 表
#### データ型変換用汎用ファンクション一覧

| ファンクション記号           | 処理内容                                   | 参照先                          |
|-----------------------------|--------------------------------------------|--------------------------------|
| **DINT型→BCD型変換**         | DINT型データをBCD型データへ変換します。    | 1353ページ DINT_TO_BCD(_E)     |
| **DINT型→REAL型/LREAL型変換** | DINT型データをREAL型またはLREAL型データへ変換します。 | 1355～1356ページ DINT_TO_REAL(_E), DINT_TO_LREAL(_E) |
| **DINT型→TIME型変換**         | DINT型データをTIME型データへ変換します。   | 1357ページ DINT_TO_TIME(_E)    |
| **DINT型→STRING型変換**       | DINT型データをSTRING型データへ変換します。 | 1358ページ DINT_TO_STRING(_E)  |
| **BCD型→INT型/DINT型変換**    | BCD型データをINT型またはDINT型データへ変換します。 | 1360～1362ページ BCD_TO_INT(_E), BCD_TO_DINT(_E) |
| **BCD型→STRING型変換**        | BCD型データをSTRING型データへ変換します。 | 1365ページ BCD_TO_STRING(_E)   |
| **REAL型→INT型/DINT型変換**   | REAL型データをINT型またはDINT型データへ変換します。 | 1367～1369ページ REAL_TO_INT(_E), REAL_TO_DINT(_E) |
| **REAL型→LREAL型変換**        | REAL型データをLREAL型データへ変換します。 | 1371ページ REAL_TO_LREAL(_E)   |
| **REAL型→STRING型変換**       | REAL型データをSTRING型（指数型式）データへ変換します。 | 1373ページ REAL_TO_STRING(_E)  |

- 各変換は基本的に2種類のファンクション記号があり、通常版と拡張版（_E）が存在します。
- 参照先ページは詳細な説明や使用方法が記載されているページ番号を示しています。

### テキスト
- 各変換の処理内容は「○○型データを△△型データへ変換します。」と簡潔に記載されています。
- ファンクション記号はプログラム内で使用される識別子であり、変換の種類ごとに異なります。
