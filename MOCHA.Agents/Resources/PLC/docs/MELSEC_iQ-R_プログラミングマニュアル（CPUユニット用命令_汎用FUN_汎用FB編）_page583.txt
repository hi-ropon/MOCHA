### テキスト
#### サブルーチンプログラムの出力OFFコール（FCALL(P)）
- FCALL(P)命令は、指定されたポインタのサブルーチンプログラムの非実行処理を行う命令です。
- プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります（詳細はMELSEC iQ-R CPUユニットユーザーズマニュアル応用編参照）。

#### 実行条件
- FCALL命令は特定の実行条件波形で動作します。
- FCALLP命令も同様に実行条件があります。

### 図
#### ラダー図
- (P)というポインタを指定してサブルーチンプログラムを呼び出す形のラダー図が示されています。
- ST（ストラクチャードテキスト）では対応していません。

#### FBD/LD図
- EN（Enable）とENO（Enable Output）端子を持つFCALL(P)命令のブロック図が示されています。
