### テキスト
#### 33.4 常用対数演算 LOG(_E)
入力値の常用対数（底が10の対数）の演算結果を出力する関数について説明しています。

##### 機能
- 入力されたREAL型/LREAL型データの10を底とした対数を計算し、出力します。
- 入力値をA、演算出力値をBとすると、B = log₁₀A となります。
- 入力値(s)はREAL型またはLREAL型のデータ値で、正の数のみ設定可能（負の数は演算不可）。
- 演算結果が0またはアンダーフローの場合は、演算結果を0として出力します。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、dから演算出力値が出力されます。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |  
| --- | --- |  
| EN | ENO | (d) |  
| TRUE（演算実行） | TRUE | 演算出力値 |  
| FALSE（演算停止） | FALSE*1 | 不定値 |  

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、そのデータを使用しないようプログラム処理を行う必要があります。

### 図
- ラダー図およびFBD/LDの関数ブロックの構造図が示されています。  
  - EN/ENOなしの場合は、入力sから出力dへ直接つながる単純な構造。  
  - EN/ENO付きの場合は、EN入力、ENO出力が追加され、s入力からd出力へ接続されています。

### 表
#### 設定データ（引数、内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |  
| --- | --- | --- | --- |  
| EN | 実行条件（TRUE: 実行、FALSE: 停止） | 入力変数 | BOOL |  
| s(IN) | 入力値 | 入力変数 | ANY_REAL |  
| ENO | 出力条件（TRUE: 正常、FALSE: 異常） | 出力変数 | BOOL |  
| d | 出力 | 出力変数 | ANY_REAL |
