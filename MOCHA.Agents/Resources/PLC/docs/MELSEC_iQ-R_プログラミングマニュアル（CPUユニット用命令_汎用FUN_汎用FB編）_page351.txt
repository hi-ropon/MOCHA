### テキスト
#### 機能
- 指定したデバイスのビット列から(n)点分を右へ1ビットシフトする動作を説明しています。
- 最上位の1ビットは0に設定されます。
- SM700には、シフト範囲から一つ右の1ビットが格納されます。
- (n)で指定した値が0の場合は処理されません。

#### エラー
- 演算エラーはありません。

### 図
- ビット列のシフト動作を示す図があり、上段に元のビット列、下段にシフト後のビット列が描かれています。
- 各ビットの位置は(d)から(d)+(n)-1まで示され、シフト後の最上位ビットは0に置き換わり、SM700に1ビット分の値が格納される様子が矢印で示されています。
</contents>
