### テキスト
#### BIN32ビットデータ転送（DMOV(P)）
指定されたデバイスのBIN32ビットデータを転送する命令です。対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）など複数の種類が示されています。

#### 実行条件
- DMOV命令は立下りエッジで実行されます。
- DMOVP命令は立上りエッジで実行されます。

#### 注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、データ型の安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**  
  転送元デバイス（s）から転送先デバイス（d）へデータを転送する命令のラダー表現が示されています。

- **FBD/LD図**  
  EN（実行条件入力）、ENO（実行結果出力）、s（転送元）、d（転送先）を持つ命令ブロックの図が示されています。
