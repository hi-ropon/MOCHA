このページは主にテキストと図で構成されています。テキスト部分では、BIN16ビットデータの論理演算機能の説明とエラーに関する記述があります。図はビットごとの排他的論理和（XOR）演算の例を示しており、ビットの並びと演算結果を視覚的に表現しています。表やグラフは含まれていません。
</overview>

<contents>
### テキスト
#### 機能
- (d)で指定されたデバイスのBIN16ビットデータと、(s)で指定されたデバイスのBIN16ビットデータのビットごとの排他的論理和（XOR）を求め、その結果を(d)で指定されたデバイスに格納する。
- ビットデバイスの場合、桁指定による点数以降のビットデバイスは0として演算する。

#### エラー
- 演算エラーはありません。

### 図
- 図は3行のビット列を示している。
  - 1行目は(d)で指定されたデバイスのビット列（b15からb0まで）。
  - 2行目は(s)で指定されたデバイスのビット列（b15からb0まで）。
  - 3行目は1行目と2行目のビットごとのXOR演算結果を示している。
- 具体的には、各ビット位置ごとにXORを計算し、その結果を(d)のビット列に格納する様子を表している。

</contents>
