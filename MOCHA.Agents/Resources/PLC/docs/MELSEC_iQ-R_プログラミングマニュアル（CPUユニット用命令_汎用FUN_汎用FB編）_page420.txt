### テキスト
#### 単精度実数→符号なしBIN16ビットデータ変換命令（FLT2UINT(P)）
指定した単精度実数を符号なしBIN16ビットデータに変換する命令です。  
対応エンジニアリングツールのバージョンは1.035M以降となっています。

#### ■実行条件
- FLT2UINT命令は立ち上がりエッジで実行されます。
- FLT2UINTP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー命令は、(s)に単精度実数、(d)に符号なしBIN16ビットデータを指定し、ENO出力があります。  
- **FBD/LD図**  
  ブロック図では、EN入力、s入力（単精度実数）、d出力（符号なしBIN16ビットデータ）、ENO出力があります。
