### テキスト
#### 機能
(s)で指定されたデバイス番号以降に格納されている時間データ（hour、minute、second）を秒に換算し、その換算結果を(d)で指定されたデバイスに格納する機能について説明しています。

#### 例
(s)に4時間29分31秒が指定された場合、これを秒に換算すると16171秒となり、その値が(d)で指定されたデバイスに格納されることを示しています。

### 図
- 時間データの構成を示す図
  - (s)にhour（0～23）
  - (s)+1にminute（0～59）
  - (s)+2にsecond（0～59）が格納されていることを示しています。
- これらの時間データを秒に換算し、(d)で指定されたデバイスに格納する流れを矢印で表現しています。
- 例として、4時間29分31秒が16171秒に換算される様子を図示しています。

### 表
#### エラー
| エラーコード (SD0) | 内容                     |
|--------------------|--------------------------|
| 3405H              | (s)のデータが範囲外のとき。 |

- (s)に指定された時間データが範囲外の場合に発生するエラーコードとその説明が記載されています。
