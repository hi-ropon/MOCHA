### テキスト
#### 機能
(s1)で指定した倍精度実数を、(s2)で指定した倍精度実数でべき乗演算し、(d)で指定したデバイスに演算結果を格納します。

- (s1)、(s2)に指定できる値、および格納できる値は0、2^-1022以上かつ2^1024未満の範囲です。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s1)または(s2)に指定した値が0、非正規化数、非数、±∞のとき。 |

### 図
倍精度実数のべき乗演算のデータ配置と演算の流れを示す模式図です。

- 被べき乗データは(s1)、(s1)+1、(s1)+2、(s1)+3の4つの連続した倍精度実数データで構成されます。
- べき乗データは(s2)、(s2)+1、(s2)+2、(s2)+3の4つの連続した倍精度実数データで構成されます。
- これらのデータを用いてべき乗演算を行い、結果を(d)、(d)+1、(d)+2、(d)+3の4つの連続したデバイスに格納します。

図は、被べき乗データとべき乗データの配置を示し、それらが演算されて結果が格納される流れを矢印で表現しています。
