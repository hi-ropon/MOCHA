### 表
#### 基本ステップ数とサブセット処理の可否一覧
命令名ごとに、以下の2つの項目が表形式でまとめられています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例: OR$>=, $+ (s) (d), $MOVなど） |
| 基本ステップ数 | その命令を実行するのに必要な基本ステップ数（例: 3, 4, 5など） |
| サブセット可否 | その命令がサブセット処理可能かどうかを示す。○は可能、―は不可を意味する。 |

表の内容は多くの命令がリストアップされており、基本ステップ数は3～5の範囲で設定されています。サブセット処理が可能な命令は限られており、主に「OR$>=」や「$+ (s) (d)」など一部の命令に○が付いていますが、多くの命令は不可（―）となっています。
