### テキスト
#### 単精度実数除算
単精度実数を除算する命令です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

#### 実行条件
- E/命令は立下りエッジで実行
- E/P命令は立上りエッジで実行

#### バージョン対応
バージョン"1.035M"以降のエンジニアリングツールで対応しています。

### 図
- ラダー図：3つのオペランド(s1, s2, d)を持つ命令ブロックの形状を示しています。
- FBD/LD図：EN（実行条件）、s1、s2（除算対象）、d（結果格納先）を持つ命令ブロックの形状を示しています。
（図中の□にはEDIVISION、EDIVISIONPが入ります）
