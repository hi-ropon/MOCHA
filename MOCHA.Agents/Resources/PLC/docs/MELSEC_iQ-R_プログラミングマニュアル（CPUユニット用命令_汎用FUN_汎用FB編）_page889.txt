### テキスト
#### 符号付きBIN16ビットデータ→倍精度実数変換
命令名：INT2DBL(P)

符号付きBIN16ビットデータを倍精度実数に変換します。

##### 実行条件
- INT2DBL命令は立ち上がりエッジで実行
- INT2DBLP命令は立ち上がりレベルで実行

##### 機能
(s)で指定した符号付きBIN16ビットデータを倍精度実数に変換し、(d)に格納します。

##### エラー
演算エラーはありません。

### 図
- ラダー命令のシンボル：入力(s)、出力(d)を持つシンプルな命令図
- FBD/LD命令のシンボル：EN（実行条件入力）、ENO（実行結果出力）、s（入力）、d（出力）
- 動作例：
  - (s)の16ビットデータ（例：EF1FH = -4321）をINT2DBL命令で倍精度実数に変換
  - (d)に64ビットの倍精度実数として格納（例：C0B0H E100H 0000H 0000H = -4321.0）
