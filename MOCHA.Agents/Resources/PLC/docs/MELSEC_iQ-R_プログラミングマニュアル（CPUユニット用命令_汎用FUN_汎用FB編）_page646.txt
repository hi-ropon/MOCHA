このページは「CSV形式変換読出し」について説明しています。構成は以下の通りです。
- テキスト部分：CSV形式変換読出しの概要と設定項目の説明
- 図部分：CSV形式のデータ読み出しのイメージ図（データの配置と読み出しの流れを示す）

<contents>
### テキスト
- CSV形式変換読出しでは、データを複数回に分けて読み出すことが可能であることを説明しています。
- 読出しを開始したい行を指定する必要があることが記載されています。
- 実行/完了タイプは「CSV形式変換読出し（10進数(BIN16ビットデータ)）」であること。
- 列数指定は4H、データタイプはワード、ファイル位置は2H、読出し先頭デバイスはD0、読出し結果データ数は6Hと設定例が示されています。

### 図
- 図は5行4列のCSVデータの読み出しイメージを示しています。
- 各セルには数値とカンマ「,」が交互に配置され、行末にはCR（キャリッジリターン）とLF（ラインフィード）が付いています。
- 矢印と番号で以下のポイントが示されています。
  1. 開始行：読み出しを開始する行を指定（例では1行目から）
  2. 次回開始位置：次に読み出しを開始する位置を示す（例では3行目の途中）
  3. デバイス格納データ：読み出したデータが格納されるデバイス（D0からD6まで）と対応するK6〜K10のビットが示されている。
- Nはデータ数を示しています。
- 矢印はデータの読み出し順序と格納先の関係を視覚的に表現しています。

### 注釈
- (1)は開始行を示す。
- (2)は次回開始位置を示す。
- (3)はデバイス格納データ（読出しデータ）を示す。
- Nはデータ数を表す。

このページは、CSV形式のデータを複数回に分けて読み出す際のデータ配置と読み出しの流れを具体的に示した内容となっています。
</contents>
