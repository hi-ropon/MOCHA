### テキスト
#### ステップ終了命令 RST [S□/BL□￥S□]
- RnCPU、RnENCPU、プロセスCPU（プロセスモード）、プロセスCPU（二重化モード）における使用制限やファームウェアバージョンの条件が記載されています。
- 指定したステップを非活性化する命令です。

### 図
#### ラダー図
- 指定ステップを終了させるためのラダー回路図が示されています。
- (d)というビットデバイスをリセットする形で表現。

#### FBD/LD図
- EN（入力）、ENO（出力）、d（ステップ番号）を持つブロック図で表現。
