### テキスト
#### 注意事項
- 動作中はオペランドやデータの内容を変更しないこと。並び替え完了時に正しい順序にならない場合がある。
- (s)と(d)に同一デバイスを指定した場合、並び替え元のデータは並び替え後のデータ順に書き換えられるため、実行完了（SD774のビット9がONする）まで(s)の内容を変更しないこと。
- (s)と(d)を指定する際は、同一デバイス範囲を指定するか、範囲が重複しないデバイスを指定すること。

### 図
- 3つの図でデバイス範囲の指定例を示す。
  - 左図：元データD10～D30を並び替え後もD10～D30に書き換え → ○（正しい）
  - 中図：元データD10～D30を並び替え後D35～D55に書き換え → ○（正しい）
  - 右図：元データD10～D30を並び替え後D2～D22に書き換え → ×（誤り）
