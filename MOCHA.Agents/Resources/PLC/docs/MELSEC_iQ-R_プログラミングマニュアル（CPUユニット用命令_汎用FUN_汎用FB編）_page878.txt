### テキスト
#### 機能
(s1)で指定された倍精度実数と、(s2)で指定された倍精度実数の乗算を行い、その乗算結果を(d)で指定されたデバイスに格納します。

- (s1)、(s2)、(d)にはそれぞれ0、2^-1022≦指定値(格納値)＜2^1024の値が指定および格納可能です。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### 図解
倍精度実数の乗算のイメージとして、(s1)と(s2)の各4つのデバイスに分割された倍精度実数が掛け合わされ、結果が(d)の4つのデバイスに格納される様子が示されています。

| 3403H | (d)が下記範囲を超えるとき（オーバーフローが発生したとき）。<br>|(d)| < 2^1024 |
