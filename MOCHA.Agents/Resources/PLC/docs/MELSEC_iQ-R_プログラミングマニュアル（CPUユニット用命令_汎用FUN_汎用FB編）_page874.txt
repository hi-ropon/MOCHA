### テキスト
#### 機能
(s1)で指定された単精度実数と、(s2)で指定された単精度実数の乗算を行い、その結果を(d)で指定されたデバイスに格納します。

- (s1)、(s2)、(d)には0から2^126以上2^128未満の値が指定および格納可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

#### 注意事項
- 入力値の範囲や丸め誤差に関する注意が記載されています。

### 図
- 単精度実数の乗算処理のイメージ図があり、(s1)+1、(s1)、(s2)+1、(s2)の単精度実数同士を乗算し、結果を(d)+1、(d)に格納する流れを示しています。

### 表
#### エラーコード(SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 3402H | (s1)で指定したデータが0、非正規化数、非数、±∞のとき。<br>(s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が指定範囲を超えるとき（オーバーフローが発生したとき）。<br>|(d)| < 2^128 |

- エラーコード3402Hは入力データの異常値に関するエラー。
- エラーコード3403Hは乗算結果が格納可能範囲を超えた場合のオーバーフローエラーを示す。
