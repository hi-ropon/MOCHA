### テキスト
#### データメモリのデータリード/ライト
データメモリへのデータライト命令は、任意のデバイスデータをデータメモリへ書き込む命令です。これにより、演算で使用する固定値や演算結果をデータメモリへ書き込むことで、バッテリ低下時のデータ消失を防止できます。また、データメモリへ書き込まれたデータは、データメモリからのデータリード命令を使って任意のタイミングで読み出すことが可能です。

##### ■実行方法
デバイスデータのデータメモリへの書込みは、SP.DEVST命令で行います。また、データメモリに書き込まれたデバイスデータは、S(P).DEVLD命令で指定デバイスに読み出します。

##### ■設定方法
SP.DEVST命令、S(P).DEVLD命令を使用する場合は、デバイスデータ格納用ファイル設定が必要です。
設定は、CPUパラメータの[ファイル設定]→[デバイスデータ格納用ファイル設定]で行います。

### 図
CPUパラメータ設定画面の「デバイスデータ格納用ファイル設定」部分のスクリーンショットがあり、以下の項目が表示されています。
- 使用有無設定：使用する
- 容量：1ワード
- ファイル名：DEVSTORE

### 表
| 項目   | 内容               |
|--------|--------------------|
| 容量   | 1K～512K（ワード） |
| ファイル名 | DEVSTORE（固定）  |

### 注意事項
- データメモリにデバイスデータ格納用ファイル作成に必要な空き容量がない場合、エラー（エラーコード：21A1H）が発生します。
- STOP→RUN時に実際のデバイスデータ格納用ファイルがパラメータ設定と一致しているかチェックを行い、不一致の場合はエラー（エラーコード：21A0H）が発生します。
