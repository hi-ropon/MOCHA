<overview>
このページは、PLCの命令におけるX0とX1のOFFからONへの同一タイミング時の動作を示したタイミングチャート（グラフ）と、その動作説明のテキストで構成されています。テキスト部分では、タイミングチャートの各ポイントに対応した動作説明が番号付きで記載されています。表はなく、図はタイミングチャートのみです。
</overview>

<contents>
### テキスト
- 「X0とX1のOFF→ONが同一タイミングのとき」の動作について説明しています。
- (1) X0がOFFからONに変わるため、M0はONになります。
- (2) X1がOFFからONに変わるため、M0はOFFのままになります。
- (3) X0がOFF→ON以外のため、M0はOFFになります。
- (4) X1がOFF→ON以外のため、M0はOFFのままになります。
- PLS命令で出力(Y)を指定すると、1スキャンの最後に実行したPLS命令のON/OFF状態が出力されることが記載されています。

### 図（タイミングチャート）
- 横軸は時間軸で、X0、X1、M0の3つの信号のON/OFF状態を示しています。
- X0とX1はそれぞれOFFからONに変化するタイミングが同じであることを示しています。
- M0はX0のOFF→ONに同期してONになり、その後OFFになる動作を示しています。
- 各信号のON/OFF変化点に番号(1)～(4)が付されており、テキストの説明と対応しています。
- PLS命令（パルス命令）を用いた場合の動作を視覚的に表現しています。
</contents>
