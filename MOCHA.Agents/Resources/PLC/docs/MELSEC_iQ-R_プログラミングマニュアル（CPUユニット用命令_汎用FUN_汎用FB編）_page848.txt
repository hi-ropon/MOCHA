### テキスト
#### 注意事項
エンジニアリングツールで入力できる実数の桁数は最大15桁であるため、この命令では有効桁数16桁以上の実数との比較はできません。  
有効桁数16桁以上の実数との一致・不一致を判断する場合は、比較する実数の前後の近似値との大小比較で判断する必要があります。

### 図（命令の動作例）
#### 例1: 有効桁数16桁の実数と倍精度実数の一致を判断する場合
- 命令は「E1.234567890123456+10」と倍精度実数の一致を判定。
- 図は、D0～D3の範囲内にあるかどうかをチェックする動作を示す。
- 範囲は「E1.23456789012345+10」から「E1.23456789012346+10」までで、境界値は範囲に含まれない。
- 命令の動作は、D0～D3がこの範囲内にあるかを判定し、一致ならY10がONになる。

#### 例2: 有効桁数16桁の実数と倍精度実数の不一致を判断する場合
- 命令は「E1.234567890123456+10」と倍精度実数の不一致を判定。
- 図は、D0～D3の範囲内にあるかどうかをチェックし、範囲外なら不一致と判断。
- 範囲は「E1.23456789012345+10」から「E1.23456789012346+10」までで、境界値は範囲に含まれる。
- 不一致の場合、Y20がONになる。

### 図の説明
- 図中の四角形は命令の構成要素（比較演算子、実数値、デバイス）を示す。
- 丸や矢印は数値の範囲や判定条件を視覚的に表現。
- (1)の注釈は、D0～D3が指定範囲内かどうかのチェック内容を説明している。
