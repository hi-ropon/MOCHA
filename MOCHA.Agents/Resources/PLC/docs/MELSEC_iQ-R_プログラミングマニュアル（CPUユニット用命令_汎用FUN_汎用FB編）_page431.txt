### テキスト
#### 機能
- (s)で指定された倍精度実数を符号付きBIN32ビットデータに変換し、(d)で指定されたデバイスに格納します。
- 変換後のデータは、(s)で指定された倍精度実数の小数点以下1桁目を四捨五入した値となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### プログラム例の説明
- M0がONのとき、D0～D3に格納された倍精度実数を符号付きBIN32ビットデータに変換し、D100～D101に格納します。
- 図では、SM402のM0接点がONになると、EDMOVP命令でD0の倍精度実数（E-765432.1）をD3、D2、D1、D0に分割し、符号付きBIN32ビットデータに変換しています。
- その後、DBL2DINT命令でD0のデータをD100に格納しています。

### 図
- プログラム例のフローチャートが示されており、M0がONのときにEDMOVP命令とDBL2DINT命令が実行される流れを表しています。
- ビット構成として、D3、D2、D1、D0の各ビット（b63～b0）が符号付きBIN32ビットデータに変換され、D100、D101に格納される様子が示されています。
