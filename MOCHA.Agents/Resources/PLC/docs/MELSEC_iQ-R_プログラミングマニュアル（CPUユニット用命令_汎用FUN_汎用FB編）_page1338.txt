### テキスト
#### 32.21 INT型→DWORD型変換
関数名：INT_TO_DWORD(_E)

INT型データをDWORD型データへ変換する関数です。

対応CPU：RnCPU、RnENCPU（プロセス、三重化、一般、安全）に対応していません。

#### 使用例（ラダー、FBD/LD）
- EN/ENOなしの場合：d := INT_TO_DWORD(s);
- EN/ENO付きの場合：d := INT_TO_DWORD_E(EN, ENO, s);

#### 設定データ
##### 引数の内容、種別、データ型
| 引数 | 内容 | 種別 | データ型 |
| --- | --- | --- | --- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | DWORD |

#### 機能
##### 演算処理
- 入力されたINT型データ(s)をDWORD型データ(d)に変換して出力します。
- 変換後、上位16ビットは0になります。
- 入力値はINT型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション
- 演算処理を実行し、dから演算出力値が出力されます。

2. EN/ENO付きファンクション
- 実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |
| --- | --- |
| EN | ENO | (d) |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となります。その場合はdから出力されたデータを使用しないようプログラム処理を行う必要があります。
