#### 16ビットブロックデータ論理積
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |

（図説明）
- (s1) と (s2) の2つのビットブロックが論理積（AND）され、結果が (d) に格納されるイメージ図が示されています。
- 縦方向のビット数は (n) で表されています。
