### テキスト
#### 機能
- (s1)で指定したデバイスから(n)点分のBIN32ビットデータと、(s2)で指定したデバイスから(n)点分のBIN32ビットデータを比較し、(d)で指定したデバイス以降に演算結果を格納する。
- 比較条件が成立した場合は(d)の該当デバイスがON、成立しない場合はOFFとなる。
- 比較演算は32ビット単位で行う。
- (s1)には定数も指定可能。
- (d)は(s1)から(n)点分のデバイス範囲と(s2)から(n)点分のデバイス範囲以外で指定する。
- (d)から(n)点に格納される比較演算結果がすべてON(1)のとき、SM704（ブロック比較信号）がONになる。
- (n)で指定した値が0の場合は無処理。

#### 比較演算の動作イメージ（図解）
- (s1)と(s2)の各ビットデータを32ビット単位で比較し、結果を(d)の対応するビットに格納する。
- 例として、(s1)のビットデータと(s2)のビットデータを比較し、条件に応じて(d)のビットがON/OFFになる様子を示している。
