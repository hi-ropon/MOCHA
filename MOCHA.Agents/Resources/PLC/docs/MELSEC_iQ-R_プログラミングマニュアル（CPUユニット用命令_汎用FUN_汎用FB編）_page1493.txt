### テキスト
#### エラー
このセクションでは、特定のエラーコード（SD0）に対応するエラー内容が説明されています。

### 表
#### エラーコードと内容の一覧

| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3400H              | (s2)へ入力される値が0のとき。（0割り） |
| 3402H              | (s2)の内容が下記範囲外のとき。<br>-2^1024 < (s2) < -2^1022、0、2^-1022 < (s2) < 2^1024<br>(E-1.7976931348623157+308～E-2.2250738585072014-308、0、E2.2250738585072014-308～E1.7976931348623157+308) |
| 3405H              | (s2)の内容が-2147483648～2147483647の範囲外のとき。 |

- 3400Hは、s2に0が入力された場合の0割りエラーを示します。
- 3402Hと3405Hは、s2の値が指定された範囲外の場合のエラーです。特に3402HはLREAL型の場合の範囲外エラーを示しています。

以上がこのページの内容です。
