### 表
#### GX Works2命令と互換用命令の対応一覧
- 表は5つの列で構成されている。
  1. **GX Works2命令名**: 元の命令名を示す。
  2. **互換用命令名**: 互換性を持つ別の命令名。
  3. **使用プログラム ST**: ステートメントリスト（ST）での使用可否（○：使用可、×：使用不可）。
  4. **使用プログラム FBD**: ファンクションブロックダイアグラム（FBD）での使用可否（○：使用可、×：使用不可）。
  5. **参照先**: 詳細説明が記載されているマニュアルのページ番号や資料名。

- 命令例:
  - BKCMP< は互換命令 BKCMP_LT_M2 で、STでは使用不可、FBDでは使用可能。参照先は227ページ。
  - BKMINUS_M は BKMINUS_M2 と互換性があり、ST・FBD両方で使用可能。参照先は280ページ。
  - COS は COS_M2 と互換性があり、ST・FBD両方で使用可能。参照先は1423ページ。
  - D* は DMULTI_M2 と互換性があり、STでは使用不可、FBDでは使用可能。参照先は252ページ。

- 参照先にはページ番号のほか、「MELSEC iQ-R プログラミングマニュアル（ユニット専用命令編）」などの資料名も含まれている。

- ページ下部に「付録 付6 他形式プロジェクトからの置換え 1605」と記載されており、本表は付録の一部であることがわかる。

### テキスト
- 表の見出しや注釈として、各列の意味や使用可否の記号説明は明示されていないが、表の内容から推測可能。
- ページ番号は参照先の詳細説明があるページを示している。
