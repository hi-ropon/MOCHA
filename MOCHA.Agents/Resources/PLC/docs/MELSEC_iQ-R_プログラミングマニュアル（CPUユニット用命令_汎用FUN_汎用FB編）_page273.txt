このページは、BCD4桁データの除算機能について説明しています。ページ構成は以下の通りです。
- テキスト部分：機能の説明と除算結果の格納方法について記載。
- 図部分：BCD4桁データの除算例を示す図。
- 表部分：エラーコードとその内容をまとめた表。
</overview>

<contents>
### テキスト
#### 機能
(s1)で指定したBCD4桁データと(s2)で指定したBCD4桁データの除算を行い、除算結果を(d)で指定したデバイスに格納します。

- 除算結果は32ビットを使用して商と余りを格納します。
  - 商（BCD4桁）は下位16ビットに格納されます。
  - 余り（BCD4桁）は上位16ビットに格納されます。

#### エラー
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3400H             | (s2)で指定した値（除数）が0のとき。 |
| 3405H             | (s1)のBCDデータが0～9999以外のとき。<br>(s2)のBCDデータが0～9999以外のとき。 |

### 図
BCD4桁データの除算例を示しています。

- (s1)に「5678」、(s2)に「0876」を指定。
- 除算結果は(d)に格納され、商は「0006」、余りは(d)+1に「0422」と格納される。
- 0が入ることを示す矢印付きの説明あり。

この図は、除算の入力と出力の関係を視覚的に表現しています。
