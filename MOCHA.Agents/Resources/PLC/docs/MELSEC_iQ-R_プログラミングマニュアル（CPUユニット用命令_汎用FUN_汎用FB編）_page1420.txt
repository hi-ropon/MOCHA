### テキスト
#### エラー
REAL型およびLREAL型の(s)に関するエラーコードとその内容について説明しています。

#### ・(s)がREAL型の場合
- エラーコード「3402H」: (s)に指定した値が0、非正規化数、非数、±∞のときに発生するエラー。
- エラーコード「3405H」: (s)に指定可能な範囲外のデータを設定したときに発生するエラー。
  - 指定した値が負の数
  - 指定した値が「0」

#### ・(s)がLREAL型の場合
- エラーコード「3402H」: (s)に指定した値が0、非正規化数、非数、±∞のときに発生するエラー。
- エラーコード「3405H」: (s)に指定可能な範囲外のデータを設定したときに発生するエラー。
  - 指定した値が負の数
  - 指定した値が「0」
