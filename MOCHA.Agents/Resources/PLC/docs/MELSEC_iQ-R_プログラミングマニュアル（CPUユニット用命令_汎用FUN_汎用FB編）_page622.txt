### テキスト
#### 機能
(s)で指定されたBIN16ビットデータを、(d)で指定されたデータテーブルに格納します。(d)にはテーブルに格納されているデータ数が入り、(d)+1以降に(s)で指定されたデータを順次格納していきます。

- N：データ数
- D_Tr：データテーブル範囲（ユーザで管理）

注意点：
- FIFW(P)命令をはじめて実行する場合は、(d)で指定するデバイスの値をクリアしてください。
- (d)+1以降にデータが順次格納されるため、データテーブルの範囲には注意が必要です。
- データ数(d)には0～65535が格納されます。

#### エラー
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3405H             | (d)の値がFFFFHでFIFW(P)命令を実行したとき。 |

### 図
データテーブルへの格納イメージを示しています。

- 左側の図：(d)の位置に「2」、(d)+1に「5432」、(d)+2に「1234」、(d)+3に「0」、(d)+4に「0」が格納されている状態。
- (s)には「4321」がある。
- 矢印で右側の図に変化し、(d)の位置が「3」に変わり、(d)+1以降に「5432」「1234」「4321」「0」が順次格納されていることを示しています。

この図は、(d)に格納されているデータ数が1増え、(d)+1以降に(s)のデータが追加されていることを視覚的に表現しています。
