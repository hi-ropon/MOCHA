### テキスト
#### 無処理(NOPLF)
無処理の命令で、それまでの演算に何も影響を与えません。

#### 対応CPU
- RnCPU
- RnENCPU
- RnPCPU（プロセス）
- RnPCPU（二重化）
- RnPSFCPU（一般）
- RnPSFCPU（安全）
- RnSFCU（一般）
- RnSFCU（安全）

#### ラダー
- ラダー回路の図があり、無処理命令の記号が示されています。
- ST（シーケンス命令）は対応していません。

#### FBD/LD
- 対応していません。

#### 実行条件
| 命令   | 実行条件   |
|--------|------------|
| NOPLF  | 常時実行   |

#### 機能
- 無処理の命令で、それまでの演算に何も影響を与えません。

#### エラー
- 演算エラーはありません。
