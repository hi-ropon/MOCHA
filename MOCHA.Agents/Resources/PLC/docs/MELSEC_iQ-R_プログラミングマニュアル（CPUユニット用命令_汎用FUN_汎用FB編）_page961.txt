### テキスト
#### BCD4桁平方根（BSQRT(P)）
BCD値（4桁）で指定した値の平方根を演算する命令です。

#### 機能
(s)で指定したBCD4桁データの平方根を演算し、(d)で指定したデバイスに演算結果を格納します。

- (d)は整数部(d)と小数部(d)+1の2つのデバイスに分けて格納します。
- (s)で指定する値はBCD値で最大4桁（0～9999）です。
- (d)の演算結果はBCD値で0～9999.9999を格納します。
- 演算結果は小数部5桁目を切り捨てた値となります。

### 図
- ラダー図：BSQRT命令のラダー表現。入力(s)、出力(d)の接続例が示されています。
- FBD/LD図：BSQRT命令のFBD/LD表現。EN（実行条件）、ENO（実行結果）、入力s、出力dの端子が描かれています。
