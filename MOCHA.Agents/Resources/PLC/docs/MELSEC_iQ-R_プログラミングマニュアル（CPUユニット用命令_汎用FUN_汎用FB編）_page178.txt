### テキスト
#### デバイスのセット(アナンシエータを除く)
SET命令は指定したビットをONにする命令です。  
対応CPUはRnCPU、RnENCPU（プロセス、二重化、一般、安全）など複数のCPUに対応しています。

#### 実行条件
SET命令の実行条件はパルス信号です。

#### 注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- ラダー図：  
  - 左側に入力条件(d)があり、右側にSET命令のST文「ENO:=SET(EN,d);」が示されています。  
- FBD/LD図：
