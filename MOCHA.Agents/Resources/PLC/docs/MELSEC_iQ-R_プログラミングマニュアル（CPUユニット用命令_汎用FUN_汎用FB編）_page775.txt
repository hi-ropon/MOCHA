### 表
#### エラーコード一覧
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 2820H | (s)で指定したデバイス番号以降、デバイス/ラベルメモリの各設定エリアの間に00Hが存在しないとき。<br>(d)で指定したデバイス番号以降、デバイス/ラベルメモリの各設定エリアの間に00Hが存在しないとき。 |
| 2821H | (s)と(d)で指定された文字列の格納デバイス番号が重複しているとき。 |
| 3405H | (s)の文字列が16383文字を超えるとき。<br>(d)の文字列が16383文字を超えるとき。 |
| 3406H | (s)+(d)の文字列が16383文字を超えるとき。<br>(d)で指定されたデバイス番号以降、該当デバイス/ラベルメモリの各設定エリアの最終番号までの点数に結合した文字列がすべて格納できないとき。 |
