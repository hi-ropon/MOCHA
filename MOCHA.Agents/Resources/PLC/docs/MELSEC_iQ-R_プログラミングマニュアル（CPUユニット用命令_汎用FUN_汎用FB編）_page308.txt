### テキスト
#### WOR(P) [オペランドが3個の場合]
指定した2つのBIN16ビットデータの論理和を行う命令です。  
対応CPUはRnCPU、RnENCPU（プロセス、二重化、一般、安全）です。

#### ■実行条件
- WOR命令は立ち下がりエッジで実行されます。
- WORP命令は立ち上がりエッジで実行されます。

#### ■注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー**  
  ラダー回路図では、3つのオペランド(s1, s2, d)が示され、s1とs2の論理和の結果をdに格納します。

- **ST言語**  
  ENO:=WOR(EN, s1, s2, d);  
  ENO:=WORP(EN, s1, s2, d);

- **FBD/LD**  
  ブロック図では、EN（実行条件）、s1、s2（入力）、d（出力）、ENO（実行結果）が示されています。
