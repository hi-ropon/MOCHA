### テキスト
#### 機能
- パラメータで設定されたルーティング情報に基づき、(s)で指定された転送先ネットワークNo.のデータを読み出し、(d)以降に格納する。
- (s)で指定される転送先ネットワークNo.のデータがパラメータで設定されていない場合は、(d)以降に0を格納する。
- (d)以降に格納されるデータの内容は以下の通り。

### 表
#### 先頭デバイス
| 項目       | 内容               | 範囲       |
|------------|--------------------|------------|
| (d)        | 中継先ネットワークNo. | 1～239     |
| (d)+1      | 中継先局番           | 下表参照   |
| (d)+2      | ダミー               | －         |

#### ネットワーク種別と指定範囲
| ネットワーク種別                     | 指定範囲  |
|------------------------------------|-----------|

| CC-Link IEフィールドネットワーク(マスタ局) | 125固定   |
| CC-Link IEフィールドネットワーク(ローカル局) | 1～120    |
| CC-Link IE TSN(マスタ局)           | 125固定   |
| CC-Link IE TSN(ローカル局)         | 1～120    |
