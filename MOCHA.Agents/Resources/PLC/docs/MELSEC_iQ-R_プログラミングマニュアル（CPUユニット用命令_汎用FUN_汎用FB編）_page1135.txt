### テキスト
#### ユニット形名読出し（TYPERD(P)）
指定したスロットのユニット形名を読み出す命令です。

#### 実行条件
- TYPERD命令は立ち上がりエッジで実行されます。
- TYPERDP命令は立ち下がりエッジで実行されます。

### 図
- ラダー図：TYPERD命令のラダー表現。オペランドに(H)と(d)を指定し、ST言語では「ENO:=TYPERD(EN,H,d);」のように記述。
- FBD/LD図：EN（入力）、ENO（出力）、H、dの4つの端子を持つ命令ブロックの形。

### 表

| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
| (H) | ユニット形名を読み出すユニットの先頭入出力番号（16進数4桁で表したときの上3桁） | 0H～FFH、3E0H～3E3H | 符号なしBIN16ビット | ANY16 |
| (d) | (d)+0：命令実行結果、(d)+1～(d)+9：ユニット形名 | ― | ワード | ANY16_ARRAY（要素数：10） |
| EN | 実行条件 | ― | ビット | BOOL |
| ENO | 実行結果 | ― | ビット | BOOL |
