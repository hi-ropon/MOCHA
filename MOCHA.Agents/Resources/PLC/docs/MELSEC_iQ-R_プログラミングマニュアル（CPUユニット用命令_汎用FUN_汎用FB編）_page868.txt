### テキスト
#### 機能
(s1)で指定された倍精度実数と、(s2)で指定された倍精度実数の加算を行い、加算結果を(d)で指定されたデバイスに格納します。

- (s1)、(s2)、(d)にはそれぞれ0から2^-1022以上2^1024未満の値が指定および格納可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H | (s1)で指定したデータが0、非正規化数、非数、±∞のとき。<br>(s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が指定範囲を超えるとき（オーバーフロー発生時）。<br>|(d)| < 2^1024 |

### 図
- 浮動小数点数の加算処理のビット配置を示す図で、(s1)と(s2)の倍精度実数の各ビットが加算され、結果が(d)に格納される様子を表しています。
- 各倍精度実数は4つの連続したビットで表現されており、(s1)+3、(s1)+2、(s1)+1、(s1)のビットが加算され、同様に(s2)のビットも加算されます。結果は(d)+3、(d)+2、(d)+1、(d)に格納されます。
