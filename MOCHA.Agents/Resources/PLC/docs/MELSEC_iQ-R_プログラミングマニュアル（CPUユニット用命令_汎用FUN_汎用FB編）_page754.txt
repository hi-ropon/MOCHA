### テキスト
#### 機能
(s)で指定されたデバイスから(n)点分のBIN16ビットデータの平均値を計算し、(d)で指定されたデバイスに格納します。

- (1)は平均値を示す。
- 計算結果が整数値にならない場合、小数点以下は切り捨てられます。
- (n)で指定した値が0の場合は処理されません。

#### エラー
演算エラーはありません。

### 図
- 左側に縦に並んだデバイスの範囲を示すボックスがあり、(s)から(s)+(n)-1までの範囲が示されています。
- 矢印(1)がその範囲の平均値を計算し、右側の(d)で指定されたデバイスに格納する流れを表しています。
