### テキスト
#### nダブルワードデータの1ダブルワード左シフト（DDSFL(P)）
- 対応CPUはR00CPU、R01CPU、R02CPUで、ファームウェアバージョンが"06"以降で使用可能。エンジニアリングツールはバージョン1.047Z以降で使用可能。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン1.040S以降で使用可能。
- 指定したデバイスから(n)ダブルワード分を左へ1ダブルワードシフトし、シフト後の空きデータには0が格納される。

#### 実行条件
| 命令   | 実行条件           |
|--------|--------------------|
| DDSFL  | 立ち上がりエッジ    |
| DDSFLP | 立ち下がりエッジ    |

### 図
- ラダー図：シフトする先頭デバイス(d)とシフトするデータ数(n)を入力し、命令が実行される様子を示す。
- ST言語：`ENO:=DDSFL(EN,n,d);`および`ENO:=DDSFLP(EN,n,d);`の記述例。
- FBD/LD：EN、ENO、n、dの入出力ポートを持つ命令ブロックの図。
