### テキスト
#### 機能
- (s1)で指定されたデバイスに格納されている単精度実数データを、(s2)で指定された小数部桁数に基づいてBCD型浮動小数点フォーマットに分解し、(d)で指定されたデバイス番号以降に格納します。
- (d)の符号および(d)+3の指数部符号には、正のとき0、負のとき1が格納されます。
- (d)+4のBCD指数には0～38が格納されます。
- (s2)は(s1)の単精度実数データの小数部桁を指定し、例として3が示されています。
- (d)+1、(d)+2に格納されているBCDの有効桁は7桁目を四捨五入した6桁です。
- (s2)の小数部桁は0～7まで設定可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要です。

#### エラー
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3401H             | (s2)で指定した小数部桁数が0～7以外のとき。 |
| 3402H             | (s1)に設定したデバイス、ラベルの内容が-0、非正規化数、非数、±∞のとき。 |

### 図
- 単精度実数データと小数部桁数を入力として、BCD型浮動小数点フォーマットに変換し、符号、BCD7桁、指数部符号、BCD指数の4つの要素に分解して(d)以降のデバイスに格納する流れを示す。
- 具体例として、単精度実数3.25427を小数部桁数3で変換し、BCD形式の符号付き数値（3254270H）として格納する様子を図示。
- さらに、1.23456789を小数部桁数3で変換し、BCDの有効桁を四捨五入して6桁にした例も示されている。
