### テキスト
#### 機能
(d1)で指定されたデバイスと(d2)で指定されたデバイスから、(n)点分のBIN16ビットデータを交換する機能について説明しています。

### 図
ビットデータの交換を示す4つのビット配列の図があります。
- 左上と右上の図は、(d1)と(d2)で指定されたデバイスのビット配列を示し、それぞれb15からb0までの16ビットがn点分並んでいます。
- 矢印で示された下の2つの図は、上のビット配列から交換後のビット配列を示しています。
- 具体的には、(d1)のビット配列と(d2)のビット配列が入れ替わる様子がビット単位で表現されています。

### 表
#### エラーコード（SD0）
| エラーコード | 内容                             |
|--------------|--------------------------------|
| 2821H        | (d1)、(d2)のデバイスが重複しているとき。 |

この表は、デバイス指定が重複した場合に発生するエラーコードとその内容を示しています。
