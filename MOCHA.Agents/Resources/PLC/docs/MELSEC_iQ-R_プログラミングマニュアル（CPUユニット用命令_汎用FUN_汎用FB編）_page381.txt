### テキスト
#### 機能
(d)で指定したデバイスから(n1)ワード分のデータ範囲内で、右へ(n2)ワード分シフトします。シフト後、(s)から(n2)点分を空きデータ部分に格納します。
- (s)に定数を指定した場合、シフト後の最上位から(n2)点分は指定した値になります。
- (n2)で指定した値が00の場合は無処理となります。

#### 例
(n1)=9、(n2)=4の場合の動作を図で示しています。

#### エラー
| エラーコード(SD0) | 内容 |
|------------------|------|
| 2821H            | (s)から(n2)点分の範囲と、(d)から(n1)点分の範囲が重複しているとき。 |
| 3405H            | (n1)、(n2)に指定した値が(n1)<(n2)のとき。 |

#### 注意事項
(s)と(d)をビットの桁指定で指定する場合、(s)と(d)の桁数を合わせる必要がある。

### 図
- データシフトの動作を示す図解で、(d)から始まる9ワードのデータが右に4ワード分シフトされる様子を示しています。
- シフト後、空いた左側4ワード分は(s)からの値で埋められています。
- 矢印と番号でシフトとコピーの順序を示しています。
