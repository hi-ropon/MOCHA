### テキスト
#### 注意事項
ZPUSH(P)命令の(d)+1、(d)+2はシステムで使用されるため、値を変更しないように注意喚起しています。また、(d)で指定した領域内に格納されるZ、LZ退避種別もシステムで使用されるため、値を変更しないように指示しています。これらの変更は誤動作の原因となると説明しています。

| 3405H | (s)に指定可能な範囲外のデータを入力した場合に発生。具体的には以下のケースが挙げられています。<br>・指定した値が1～3以外<br>・インデックスレジスタ点数が0点の場合に(s)に「2」を指定<br>・ロングインデックスレジスタ点数が0点の場合に(s)に「3」を指定<br>また、(d)で指定した領域のシステム使用エリアに格納された値が書き換えられた場合にも発生します。 |
