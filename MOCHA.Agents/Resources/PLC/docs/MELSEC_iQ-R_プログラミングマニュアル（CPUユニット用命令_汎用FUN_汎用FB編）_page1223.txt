### テキスト
#### 各引数の設定項目
- (s1) 目標値(SV): 目標値を設定。オートチューニングのリミットサイクル法の場合は、バイアス値を加算した値を設定し、オートチューニングフラグがOFFの時点で実際の目標値を設定する。
- (s2) 測定値(PV): PID演算の入力値を設定。
- (s3) 制御用データ: オートチューニングのリミットサイクル法やステップ応答法に応じて先頭デバイスの占有点数が異なる。ステップ応答法の場合、動作設定ビットにより25点または20点分のデバイスを占有。
- (d) 出力値(MV): PID制御時は命令実行前に初期出力値をセット。オートチューニングのリミットサイクル法では出力上限値(ULV)または下限値(LLV)が出力され、終了後に所定の出力値が設定される。ステップ応答法では命令実行前にステップ出力値をセットし、オートチューニング中はPID命令側で変更。

### 表
#### 制御用データの設定項目（オペランド:s3）
| デバイス | 項目 | 内容 | 備考 |
| --- | --- | --- | --- |
| +0 | サンプリングタイム(Ts) | 1～32767[ms] | 演算周期より短い値での実行不可 |
| +1 | 動作設定(ACT) | ビット0: 0=正動作、1=逆動作<br>ビット1: 0=入力変化量警報なし、1=有効<br>ビット2: 0=出力変化量警報なし、1=有効<br>ビット3: 使用不可<br>ビット4: 0=オートチューニング不動作、1=実行<br>ビット5: 0=出力値上下限設定なし、1=有効<br>ビット6: 0=ステップ応答法、1=リミットサイクル法<br>ビット7～15: 使用不可 | ビット2とビット5は同時ON禁止 |
| +2 | 入力フィルタ定数(α) | 0～99[%] | 0指定で入力フィルタなし |
| +3 | 比例ゲイン(Kp) | 1～32767[%] | なし |
| +4 | 積分時間(Ti) | 1～32767[×100ms] | 0指定で積分なし（∞扱い） |
| +5 | 微分ゲイン(KD) | 0～200[%] | 0指定で微分ゲインなし |
| +6 | 微分時間(TD) | 1～32767[×10ms] | 0指定で微分なし |
| +7～+19 | PID演算の内部処理用（データ変更禁止） | なし | なし |
| +20 | 入力変化量(増側)警報設定値 | 0～32767 | 動作方向(ACT)のビット1=1時有効 |
| +21 | 入力変化量(減側)警報設定値 | 0～32767 | 動作方向(ACT)のビット1=1時有効 |
| +22 | 出力変化量(増側)警報設定値 | 0～32767 | 動作方向(ACT)のビット2=1かつビット5=0時有効 |
| 出力上限設定値 | -32768～32767 | 動作方向(ACT)のビット2=0かつビット5=1時有効 |
| +23 | 出力変化量(減側)警報設定値 | 0～32767 | 動作方向(ACT)のビット2=1かつビット5=0時有効 |
| 出力下限設定値 | -32768～32767 | 動作方向(ACT)のビット2=0かつビット5=1時有効 |

**備考**  
- 動作設定ビットの詳細な意味と注意点（特にビット2とビット5の同時ON禁止）  
- サンプリングタイムは演算周期より短い値は不可  
- 各種警報設定値は動作方向ビットの状態により有効・無効が切り替わる
