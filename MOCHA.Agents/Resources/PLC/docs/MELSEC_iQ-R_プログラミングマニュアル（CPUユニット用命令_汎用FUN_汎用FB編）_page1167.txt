### テキスト
#### コネクション情報の読出し
命令「SP.SOCCINF」は、(s1)で指定したコネクションのコネクション情報を読み出し、(d)で指定されたデバイス以降に格納する。

#### 実行条件
- 命令：SP.SOCCINF
- 実行条件は特に指定なし（表にて空欄）

### 図
- ラダー図：命令の入力として(U)、(s1)、(s2)、(d)の4つのオペランドがあり、出力はENOとなっている。
- FBD/LD図：四角の命令ブロックにEN、U、s1、s2が入力、ENOとdが出力される構成。
