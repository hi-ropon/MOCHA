### テキスト
#### 注意事項
同一コネクションに対する受信データの読出しを行う場合、S.SOCRCVS命令と併用しないよう注意喚起しています。

#### エラー発生時の動作
異常完了時には、完了時の状態表示デバイス(d2)+1がONになり、完了ステータス(s2)+1にエラーコードが格納されます。エラーコードの詳細は下記の表を参照してください。

#### Point
- 膨大なデータを受信しないために、SP.SOCRMODE命令で受信データサイズを設定し、受信データ量を抑制可能。
- SP.SOCRCV命令の完了デバイスをb接点で実行指示に接続することで、データを連続受信した場合でも連続して読み出し可能。

### 表
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 2820H             | 受信したデータが受信データ格納先デバイスのデバイス/ラベルメモリの各設定エリアを超えるとき。 |
| 3405H             | (s1)で指定するコネクション番号が1～16以外のとき。 |

以上の内容は、MELSEC iQ-R Ethernetユーザーズマニュアル（応用編）を参照するよう案内されています。
