### テキスト
#### BCD型COS演算（BCOS(P)）
BCD値で指定した角度のCOS（余弦）値を演算します。

##### 機能
- (s)で指定された値（角度）のCOS（余弦）値を演算し、(d)で指定されたワードデバイスに演算結果の符号を、(d)+1、(d)+2で指定されたワードデバイスに演算結果を格納します。
- COS(s)の構成は、(d)に符号、(d)+1に整数部、(d)+2に小数部を格納します。
- (s)で指定する値は0～360°（度単位）をBCD値で設定します。
- (d)に格納する演算結果の符号は、演算結果が正のとき0、負のとき1を格納します。
- (d)+1、(d)+2に格納する演算結果はBCD値で-1.000～1.000の範囲内の値です。

### 図
- ラダー図：BCOS命令のラダー表現。入力(s)、出力(d)を示す。
- FBD/LD図：BCOS命令のファンクションブロック図。EN（実行条件）、ENO（実行結果）、入力s、出力dの関係を示す。
- 実行条件チャート：BCOS命令は立ち上がりエッジで実行、BCOSP命令は立ち下がりエッジで実行されることを示す。
