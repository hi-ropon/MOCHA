### 表
#### GX Works2命令名と互換用命令名一覧
- 表は5列で構成されている。
  1. GX Works2命令名
  2. 互換用命令名
  3. 使用プログラム（ST）
  4. 使用プログラム（FBD）
  5. 参照先ページ

- 「使用プログラム」列はST（ステートメントリスト）とFBD（ファンクションブロックダイアグラム）での使用可否を○で示している。
- 参照先ページは各命令の詳細説明が記載されているページ番号と該当資料の略称が記載されている。

#### 表の内容例
- DBKPLUS_M → 互換用命令名：DBKPLUS_M2、ST・FBDともに使用可能、参照先282ページ DBK+(P)(_U)
- DDEC → 互換用命令名：DDEC_M2、ST・FBDともに使用可能、参照先294ページ DDEC(P)(_U)
- DMAX → 互換用命令名：DMAX_M2、ST・FBDともに使用可能、参照先1457ページ MAX(_E)、MIN(_E)
- DMINUS_3_M → 互換用命令名：DMINUS_M2、ST・FBDともに使用可能、参照先246ページ D-(P)(_U) [オペランドが3個の場合]

#### 備考
- 一部の命令は複数の互換用命令名が存在し、複数行にわたって記載されている。
- 参照先ページは命令の詳細な仕様や使い方が記載されている資料のページを示しているため、命令の理解や使用時に参照することが推奨される。

この表は、他形式プロジェクトからの命令置換え時に参照するための一覧表として機能している。
</contents>
