### テキスト
#### 符号付きBIN32ビットデータ→倍精度実数変換
命令「DINT2DBL(P)」は、符号付きBIN32ビットデータを倍精度実数に変換します。

#### 実行条件
- DINT2DBL命令は立下りエッジで実行
- DINT2DBLP命令は立上りエッジで実行

#### 機能
(s)で指定した符号付きBIN32ビットデータを倍精度実数に変換し、(d)に格納します。

#### エラー
演算エラーはありません。
