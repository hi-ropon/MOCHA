### テキスト
#### nビットデータのnビット右シフト、左シフト命令
本ページでは、ビットデータの右シフトおよび左シフトを行う命令について説明しています。各命令は指定したデバイス内のビットデータを、指定されたビット数だけ右または左にシフトし、シフト後の空きビットには特定の値が格納されます。

### 表
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| SFTBR / SFTBRP | 指定したデバイスからnビット分のデータを右へnビットシフトさせます。シフト後の空きデータには0が格納されます。 | 370ページ SFTBR(P) |
| SFTR / SFTRP | 指定したデバイスから(n1)ビット分のデータ範囲内で、右へ(n2)ビットシフトさせます。シフト後の空きデータには指定データが格納されます。 | 372ページ SFTR(P) |
| SFTBL / SFTBLP | 指定したデバイスからnビット分のデータを左へnビットシフトさせます。シフト後の空きデータには0が格納されます。 | 374ページ SFTBL(P) |
| SFTL / SFTLP | 指定したデバイスから(n1)ビット分のデータ範囲内で、左へ(n2)ビットシフトさせます。シフト後の空きデータには指定データが格納されます。 | 376ページ SFTL(P) |

### 図（ビットシフトの動作説明）
- **右シフト（SFTBR, SFTR）**  
  指定したビット数分、データを右にシフトし、空いた左側のビットには0または指定データが入る。  
  図ではビット列の右側から左側へビットが移動し、空いた部分に0または指定値が入る様子を示している。

- **左シフト（SFTBL, SFTL）**  
  指定したビット数分、データを左にシフトし、空いた右側のビットには0または指定データが入る。  
  図ではビット列の左側から右側へビットが移動し、空いた部分に0または指定値が入る様子を示している。

これらの命令は、ビット単位でのデータ操作を行う際に使用され、シフト方向や空きビットの値の指定により柔軟なビット操作が可能です。
