### 表
#### 命令処理時間一覧
この表は各命令に対して、特定の条件下での処理時間（単位はマイクロ秒）をCPUユニット別に示しています。

- **命令名**: 各行に命令の名前が記載されています（例：SCL2_U、DSCL2、UDCNT1など）。
- **条件**: 命令実行時の条件が記載されており、SM755のON/OFF状態やポイント番号、nの値などが指定されています。
- **処理時間(μs)**: 
  - R00CPU、R01CPU、R02CPU、及びその他のCPUユニットに分けて、最小値と最大値が示されています。
  - 例えば、SCL2_U命令の「SM755=ON、ポイントNo.1<(s1)<ポイントNo.2」の条件では、R00CPU/R01CPUの処理時間は最小4.700μs、最大10.300μs、R02CPUは最小3.700μs、最大9.300μs、その他CPUユニットは最小3.200μs、最大8.800μsとなっています。

- **特記事項**:
  - 一部の命令は条件が「—」となっており、条件指定なしの処理時間が示されています（例：UDCNT1、TTMRなど）。
  - CCD命令やSERDATA命令などは、SM772のON/OFFやnの値によって処理時間が変動しています。
  - SERMMやDSERMM命令は、同一データの有無や個数によって処理時間が異なることが記載されています。

この表は、命令の実行性能評価や最適化に役立つ詳細な処理時間データを提供しています。
</contents>
