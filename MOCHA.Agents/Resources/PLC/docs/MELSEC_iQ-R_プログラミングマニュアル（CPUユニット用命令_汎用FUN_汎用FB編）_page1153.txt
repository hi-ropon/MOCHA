### テキスト
#### 機能
- レコーディング機能では、(s)で指定した設定No.のデータを収集します。
- (s)には1～4が設定可能です。
- レコーディング設定の収集間隔設定で「トリガ命令」を選択した場合に有効です。
- 以下の場合は、DATATRG命令を実行しても無処理となります。
  - レコーディング機能が設定されていない設定No.を指定した場合
  - レコーディング設定の収集間隔設定で「トリガ命令」以外を選択した設定No.を指定した場合
  - 同一設定No.に対してDATATRG命令を命令実行中に実行した場合
  - 指定した設定No.のレコーディング開始トリガがOFFの状態で実行した場合
  - 指定した設定No.のレコーディング機能が準備中または保存中で実行した場合
