### テキスト・表
#### 倍精度実数符号反転
- 命令記号: EDNEG, EDNEGP
- 処理内容: 倍精度実数データの符号を反転する。
- 数式例: (d)+3, (d)+2, (d)+1, (d) → (d)+3, (d)+2, (d)+1, (d)
- 参照先: 902ページ EDNEG(P)

#### 単精度実数データ転送
- 命令記号: EMOV, EMOVP
- 処理内容: 単精度実数データを指定されたデバイスへ転送する。
- 数式例: (s)+1, (s) → (d)+1, (d)
- 参照先: 903ページ EMOV(P)

#### 倍精度実数データ転送
- 命令記号: EDMOV, EDMOVP
- 処理内容: 倍精度実数データを指定されたデバイスへ転送する。
- 数式例: (s)+3, (s)+2, (s)+1, (s) → (d)+3, (d)+2, (d)+1, (d)
- 参照先: 904ページ EDMOV(P)

#### 単精度実数SIN演算

- 処理内容: 単精度実数で指定した角度のSIN（正弦）値を演算する。
- 参照先: 905ページ SIN(P)
