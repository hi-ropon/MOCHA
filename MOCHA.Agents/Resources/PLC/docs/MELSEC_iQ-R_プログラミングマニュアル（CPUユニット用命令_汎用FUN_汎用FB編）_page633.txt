<overview>
このページは主に表形式で構成されており、使用可能なデバイスの種類と対応状況を示す表と、使用するCPUユニットごとの対応状況を示す表の2つのセクションで構成されています。テキスト部分は少なく、表の説明が中心です。グラフや図は含まれていません。
</overview>

<contents>
### 表1: 使用可能デバイス一覧
- オペランドごとに、ビット、ワード、ダブルワード、間接指定、定数、その他の使用可否を「○」や「―」で示しています。
- ビットデバイスにはX, Y, M, L, SM, F, B, SB, FX, FYが含まれます。
- ワードデバイスはT, ST, C, D, W, SD, SW, FD, R, ZR, RD、Jロ￥口、Uロ￥G口、Jロ￥口、U3Eロ￥(H)G口などに分類されています。
- ダブルワードはZ、LT, LST, LCに分かれています。
- 間接指定はLT, LZに対応しています。
- 定数はK, H, E, $の種類があります。
- その他(U)は特別な扱いで、(U)は○、その他は―で示されています。

### 表2: 使用するCPUユニットと参照先
- RnCPU、RnENCPU、RnPCPU、RnPSFCPU、RnSFCUの各CPUユニットごとに対応するCPUモデルと参照ページが示されています。
- RnCPUはR01CPU、R02CPU、R04CPU、R08CPU、R16CPU、R32CPU、R120CPUが含まれ、ファームウェアバージョンによって参照先ページが異なります（バージョン28以降は634ページタイプ1、27以前は636ページタイプ2）。
- RnENCPUも同様にファームウェアバージョンで参照先が分かれています。
- RnPCPU、RnPSFCPU、RnSFCUは636ページタイプ2を参照します。

### 補足
- コントロールデータ(d1)、ファイル名(s2)、読み出したデータ(d2)は使用するCPUユニットによって異なることが記載されています。
</contents>
