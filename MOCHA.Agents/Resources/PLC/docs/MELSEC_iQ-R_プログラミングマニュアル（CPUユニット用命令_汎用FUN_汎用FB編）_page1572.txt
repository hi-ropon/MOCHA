### 表
#### 基本ステップ数とサブセット処理の可否一覧
この表は各命令名に対して、基本ステップ数とサブセット処理が可能かどうかを示しています。

- **命令名**: 各命令の名称が記載されています。例として「E+P (s1) (s2) (d)」「ED+P (s1) (s2) (d)」「INT2FLT」など多くの命令が並んでいます。
- **基本ステップ数**: 各命令の基本的なステップ数が数字で示されています。例えば「4」「3」「5」など。
- **サブセット可否**: サブセット処理が可能な命令は「〇」、不可の命令は「―」で表記されています。

この表により、命令ごとの処理負荷（ステップ数）とサブセット処理の対応状況を一目で把握できます。
