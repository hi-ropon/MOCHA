### テキスト
#### WDTリセット
ウォッチドッグタイマをリセットする命令WDT(P)について説明しています。

#### 機能
- プログラム中でウォッチドッグタイマをリセットします。
- スキャンタイムがウォッチドッグタイマの設定値を超える場合に使用します。スキャンタイムが超える場合は、エンジニアリングツールのパラメータ設定でウォッチドッグタイマの設定値を変更してください。
- 0ステップからWDT(P)命令までの時間をt1、WDT(P)命令からEND(FEND)命令までの時間をt2とし、t1とt2の合計がウォッチドッグタイマの設定値を超えないようにします。
- WDT(P)命令は1スキャン中に2回以上使用可能ですが、異常発生時の出力OFFまでの時間がかかるため注意が必要です。
- WDT(P)命令を実行しても、特殊レジスタに格納されるスキャンタイムの値はクリアされません。そのため、特殊レジスタのスキャンタイム値はパラメータ設定のウォッチドッグタイマ設定値より大きくなることがあります。

#### エラー
- 演算エラーはありません。
