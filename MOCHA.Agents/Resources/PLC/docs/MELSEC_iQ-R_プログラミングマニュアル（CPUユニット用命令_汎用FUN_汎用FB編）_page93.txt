### テキスト
#### BIN32ビットデータの左ローテーション
BIN32ビットデータを左方向に回転させる命令について説明しています。キャリフラグを含めるか含めないかの2種類の命令があり、それぞれの動作内容が記載されています。

##### 命令一覧と処理内容
- **DROL / DROL P**  
  BIN32ビットデータをキャリフラグを含めずに(n)ビット左へ回転させます。  
  図では、SM700のビットb31からb0までが左方向に回転する様子が示されています。

- **DRCL / DRCL P**  
  BIN32ビットデータをキャリフラグを含めて(n)ビット左へ回転させます。  
  同様にビットの左回転の動作が図示されています。

#### データテーブル操作命令
##### データテーブルからの先入データリード
- **FIFR / FIFRP**  
  テーブルに最初に格納したデータを指定されたデバイスに格納する命令です。  
  図では、データテーブルの先頭からN個のデータが順にデバイスに格納されるイメージが示されています。
