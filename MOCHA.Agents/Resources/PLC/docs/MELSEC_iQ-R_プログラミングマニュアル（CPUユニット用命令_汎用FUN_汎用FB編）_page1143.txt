### テキスト
#### ユニット装着状態の判定方法
入出力点数が32点以上のユニットについて、先頭入出力番号以外の入出力番号を(H)で指定した場合、ユニット情報のユニット装着状態のみON、ユニット装着状態以外がOFFのユニット情報を格納します。

#### プログラム例の説明
- M0がONのとき、入出力番号が0050H以降のユニット情報をD0以降に格納する例を示しています。
- RJ71EN71の後半16点の入出力番号を(H)で指定しているため、ユニット装着状態のみONとなります。

#### ビットフィールドの説明
- D0～D3の各ビットがユニットの状態を示しており、ビットごとに意味が詳細に説明されています。
- 例えば、D2のビットはユニットの装着有無、スロットの先頭かどうか、機種管理番号、シリーズ名、ユニット種別、点数単位などを示しています。
- D3のビットはユニットアクセス可否、エラー状態、ユニット準備完了などの状態を示しています。
