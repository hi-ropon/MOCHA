### テキスト
#### 機能
CPUユニットの時計素子から「年、月、日、時、分、秒、曜日、1/1000秒」を読み出し、(d)で指定されたデバイス以降にBIN値で格納します。

- (d)の「年」は西暦4桁を格納します。
- (d)+6の「曜日」は「日～土」を「0～6」で格納します。
- うるう年は自動的に補正されます。

#### 注意事項
- S(P).DATERD命令は、CPUユニットに誤った時計データを設定している場合でも、時計データを読み出してデバイスに格納します。例えば2月30日などの不正な日付を設定しないように注意してください。
- 1/1000秒の時計データ読み出し時の誤差は最大2msです。これはCPUユニット内部の時計素子で記憶しているデータと、S(P).DATERD命令で読み出したデータの誤差によるものです。

#### エラー
演算エラーはありません。
