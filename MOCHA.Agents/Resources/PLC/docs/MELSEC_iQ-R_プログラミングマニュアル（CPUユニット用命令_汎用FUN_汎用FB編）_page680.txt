### 表
#### 使用可能デバイス
オペランドごとに使用可能なデバイスの種類を示した表です。ビット、ワード、ダブルワード、間接指定、定数、その他のカテゴリに分かれており、各オペランド(U, s1, d1, s2, s3, d2)がどのデバイスを使用できるかを○や―で示しています。

- ビットデバイス例: X, Y, M, L, SM, F, B, SB, FX, FY
- ワードデバイス例: T, ST, C, D, W, SD, SW, FD, R, ZR, RD
- ダブルワードデバイス例: Z, LT, LST, LC, LZ
- 間接指定、定数、その他の使用可否も記載

#### コントロールデータ(d1)
オペランド(d1)に関する詳細なコントロールデータの説明表です。

| デバイス | 項目 | 内容 | 設定範囲 | セット側 |
| :--- | :--- | :--- | :--- | :--- |
| +0 | 使用用途設定エリア | b0: 対象種別設定<br>指定対象がファイルかフォルダかを指定<br>・0: ファイル指定<br>・1: フォルダ指定 | 左記 | ユーザ |
| +1 | 完了ステータス | 命令完了時に完了ステータスを格納<br>・0000H: 正常完了<br>・0000H以外: 異常完了（エラーコード） | ― | システム |

- 「使用用途設定エリア」では、b0ビットで対象がファイルかフォルダかを指定可能。
- 「完了ステータス」では、命令の完了状態を示し、正常完了は0000H、異常完了はエラーコードが格納される。

### テキスト
- b0の説明として、対象種別設定がファイルかフォルダかを指定することが明記されています。
- 完了ステータスの説明では、正常完了と異常完了のコードの意味が記載されています。
- エラーコードの詳細は別ページ（687ページ）に記載されていることが示されています。
