### テキスト
#### 単精度実数COS演算
単精度実数で指定した角度のCOS（余弦）値を演算します。

対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）です。

COS命令はST、FBD/LDでは対応していないため、汎用ファンクションのCOSを使用する必要があります（1423ページ参照）。

#### 実行条件
- COS命令はパルス信号で実行されます。
- COSP命令は立ち上がりエッジで実行されます。

### 図
- ラダー図：COS命令のラダー回路の簡略図が示されており、(s)と(d)のオペランドがあることがわかります。
- FBD/LD図：EN（入力）、ENO（出力）、s（角度データ）、d（演算結果）のポートを持つブロック図が示されています。
