### テキスト
#### E+(P)命令の概要
単精度実数を加算する命令です。オペランドが3個の場合の動作を示しています。

#### 実行条件
- E+命令はパルス信号の立下りで実行されます。
- E+P命令はパルス信号の立上りで実行されます。

#### 機能
(s1)で指定された単精度実数と、(s2)で指定された単精度実数の加算を行い、加算結果を(d)で指定されたデバイスに格納します。
(s1)、(s2)、(d)には0から2^-126 ≤ |指定値(格納値)| < 2^128の範囲の値が指定および格納可能です。

### 図
- ラダー図：s1、s2の2つの入力とdの出力を持つ加算命令のラダー表現。
- FBD/LD図：EN（実行条件）、s1、s2の入力とdの出力を持つブロック図。EPLUSまたはEPLUSPが入る。
- 実行条件図：E+はパルスの立下り、E+Pはパルスの立上りで命令が実行されることを示すタイミングチャート。
