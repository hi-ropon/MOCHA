### テキスト
#### 単精度実数符号反転（ENEG(P)）
単精度実数データの符号を反転する命令です。指定されたデバイスの単精度実数の符号を反転し、同じデバイスに格納します。正の符号を反転する際に使用します。

#### 機能
(d)で指定されたデバイスの単精度実数の符号を反転し、(d)で指定されたデバイスに格納します。

例：
- 入力値 1.2345 → 出力値 -1.2345

#### エラー
- エラーコード3402H：指定した内容が0、非正規化数、非数、±∞のときに発生します。

### 図
- ラダー図：ENEG命令のラダー回路表現。入力(d)に対して符号反転を行い、結果を(d)に格納。
- FBD/LD図：ENEG命令のファンクションブロック図。EN（実行条件）、ENO（実行結果）、d（データ）端子が示されている。
- 符号反転イメージ図：単精度実数の符号が正から負に反転する様子を示す。
