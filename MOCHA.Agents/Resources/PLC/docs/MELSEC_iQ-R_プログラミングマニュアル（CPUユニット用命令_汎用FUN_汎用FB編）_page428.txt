### テキスト
#### 倍精度実数→符号なしBIN16ビットデータ変換命令 DBL2UINT(P)
指定した倍精度実数を符号なしBIN16ビットデータに変換する命令です。

#### 実行条件
- DBL2UINT命令は立下りエッジで実行
- DBL2UINTP命令は立上りエッジで実行

#### 使用可能デバイス
- (s)オペランド（倍精度実数または倍精度実数が格納されている先頭デバイス）は、ワードデバイス（T, ST, C, D, W, SD, SW, FD, R, ZR, RD）に対応
- (d)オペランド（BINデータを格納するデバイス）は、ビットデバイス（X, Y, M, L, SM, F, B, SB, FX, FY）とワードデバイス（T, ST, C, D, W, SD, SW, FD, R, ZR, RD）、ダブルワードデバイス（Z, LT, LST, LC）に対応

### 図
- ラダー図：命令のラダー回路表現。左側に(s)入力、右側に(d)出力があり、命令名DBL2UINT(P)が記載されている。
- FBD/LD図：命令ブロック図。EN（入力）、ENO（出力）、s（入力）、d（出力）のポートがある。
