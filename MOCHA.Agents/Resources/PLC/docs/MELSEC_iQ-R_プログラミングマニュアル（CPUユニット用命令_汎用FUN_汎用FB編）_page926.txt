### テキスト
#### 機能
- (s)で指定されたCOS(余弦)値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納します。
- (s)で指定するCOS値は-1.0～1.0の範囲で設定可能です。
- (d)に格納する角度（演算結果）はラジアン単位で格納されます。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

### 図
- COS値の格納場所は(s)、4つの連続したデバイス(s)、(s)+1、(s)+2、(s)+3に倍精度数として格納されている。
- 演算結果の角度は(d)、4つの連続したデバイス(d)、(d)+1、(d)+2、(d)+3に倍精度数として格納される。

### 表
#### エラーコード (SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 3402H | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |
| 3405H | (s)で指定した値が-1.0～1.0の範囲外のとき。 |

### ポイント
- ラジアンと角度の変換については、RADD(P)命令、DEGD(P)命令を参照すること。
- 参照ページ：945ページ（RADD(P)）、947ページ（DEGD(P)）
