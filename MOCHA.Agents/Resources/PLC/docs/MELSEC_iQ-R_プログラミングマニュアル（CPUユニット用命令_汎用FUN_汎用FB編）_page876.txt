### テキスト
#### 機能
(s1)で指定された単精度実数と、(s2)で指定された単精度実数の除算を行い、その除算結果を(d)で指定されたデバイスに格納します。

- 除算の計算式は以下の通りです。
  (s1)+1 (単精度実数) ÷ (s2)+1 (単精度実数) → (d)+1 (単精度実数)
- (s1)、(s2)、(d)には、0から2^126未満および2^128未満の値が指定・格納可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生する可能性があるため注意が必要です。
- 詳細な注意事項は47ページを参照してください。

### 表
#### エラーコード一覧
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3400H | (s2)で指定したデータ（除数）が0のとき。 |
| 3402H | (s1)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3402H | (s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が指定範囲を超えるとき（オーバーフロー発生時）。ただし |(d)| < 2^128。 |
