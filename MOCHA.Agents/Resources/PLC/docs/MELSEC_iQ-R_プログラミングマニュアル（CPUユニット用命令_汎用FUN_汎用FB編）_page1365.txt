### テキスト
#### 32.39 BCD型→STRING型変換
関数名：BCD_TO_STRING(_E)  
BCD型データをSTRING型データへ変換します。

#### 対応CPU
RnCPU、RnENCPU、RnPCPU（三重化・一般・安全）、RnSPFCPU（一般・安全）

#### ラダー、FBD/LD表記
- EN/ENOなし  
  ラダー図にて、入力s、出力dの接続を示す。
- EN/ENO付き  
  EN、ENO、s、dの接続を示す。
- ST表記  
  EN/ENOなし：d:=BCD_TO_STRING(s);  
  EN/ENO付き：d:=BCD_TO_STRING_E(EN,ENO,s);

#### 設定データ
| 引数 | 内容 | 種別 | データ型 |
| --- | --- | --- | --- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | ANY_BIT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | STRING(8) |

#### 機能
##### 演算処理
- 入力されたBCD型(WORD型またはDWORD型)のデータをSTRING型のデータに変換し、出力(d)する。
- WORD型指定時の例：9999H → "9999"（STRING）
- DWORD型指定時の例：99999999H → "99999999"（STRING）
- 入力はWORD型、DWORD型が指定可能。BOOL型は不可。
- 文字列の最後には、SM701(出力文字数切換)がOFFの場合、00Hを格納。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、dから演算出力値が出力される。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通り。

| 実行条件 | 演算結果 |
| --- | --- |
| EN | ENO | (d) |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)のデータを使用しないようプログラム処理を行うこと。

### 図
- ラダー図（EN/ENOなし、EN/ENO付き）  
  入力s、出力d、EN、ENOの接続を示す簡単なラダー回路図。
- BCD→STRING変換イメージ図  
  WORD型指定時：9999H（BCD）→"9999"（STRING）  
  DWORD型指定時：99999999H（BCD）→"99999999"（STRING）
