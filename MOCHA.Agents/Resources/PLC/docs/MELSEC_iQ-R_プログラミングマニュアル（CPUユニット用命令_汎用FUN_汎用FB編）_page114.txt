- パルス系命令：パルス密度の測定、定周期パルス出力、パルス幅変調の3つの命令について表形式で説明。
- 近回り制御：ロータリテーブルの近回り制御命令を表で説明。
- 傾斜信号：傾斜信号命令の説明。
- マトリクス入力：マトリクス入力命令の説明。
各セクションは命令記号、処理内容、参照先の3列の表で構成されており、命令の概要と詳細な動作内容が記載されています。図やグラフはなく、テキストと表のみの構成です。
</overview>

<contents>
### テキスト・表

#### パルス系命令
- ■パルス密度の測定
  - 命令記号：SPD
  - 処理内容：(s1)で指定したデバイスのパルス入力を(s2)で指定した時間カウントし、(d)で指定のデバイスに格納する。
  - 参照先：1009ページ SPD

- ■定周期パルス出力
  - 命令記号：PLSY
  - 処理内容：(s)で指定された周波数のパルスを、(n)で指定された回数分、(d)で指定された出力番号(Y)に出力する。
  - 参照先：1011ページ PLSY

- ■パルス幅変調
  - 命令記号：PWM
  - 処理内容：(s1)で指定されたON時間と、(s2)で指定された周期のパルスを、(d)で指定された出力番号(Y)に出力する。
  - 参照先：1013ページ PWM

#### 近回り制御
- ■ロータリテーブルの近回り制御
  - 命令記号：ROTC
  - 処理内容：(n1)で分割されたロータリテーブルで、停止位置から(s)+1で指定の位置へ近回りで回転させる。
  - 参照先：1015ページ ROTC

#### 傾斜信号
- ■傾斜信号命令
  - 命令記号：RAMPQ
  - 処理内容：(s1)で指定された値から(s2)で指定された値に、(n)で指定された回数で移行する。現在値は、(d1)+0で指定されたデバイスに格納される。
  - 参照先：1018ページ RAMPQ

#### マトリクス入力
- ■マトリクス入力命令
  - 命令記号：MTR
  - 処理内容：(s)で指定のデバイスから16点×(n)列のデータを順次取り込み、(d2)で指定のデバイス以降に格納する。
  - 参照先：1022ページ MTR
</contents>
