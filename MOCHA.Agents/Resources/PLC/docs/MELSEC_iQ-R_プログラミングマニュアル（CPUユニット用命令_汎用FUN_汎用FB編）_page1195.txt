### テキスト
#### 機能概要
- ユニットパラメータでFTPサーバーを指定する場合、特定のビット設定によりCPUユニットのファイルをFTPサーバのフォルダパスへ送付する。
- SP.FTPPUT命令実行時に、ユニットパラメータのFTPクライアント設定で指定したFTPサーバに接続し、ファイル送付後に接続を切断する。
- FTPサーバ指定は、コントロールデータの特定ビットにより有無を判別し、FTPサーバ名、ログイン名、パスワードはASCII文字列で指定し、末尾に"00H"を格納する。

#### 設定例
- FTPサーバ名、ログイン名、パスワードはそれぞれコントロールデータの先頭デバイスをD0として、(s1)+オフセットの位置に16進数で文字コードを格納する。
- FTPサーバアドレスは1バイトごとに"."で区切られたデータを16進数で指定し、(s1)+134～135に格納する。

### 図（プログラム例）
- FTPサーバ名 "SERV001" の設定例
  - (s1)+5～8に対応する16進数コードを示し、プログラム上では `$MOV "SERV001" D5` と入力。
- ログイン名 "USER001" の設定例
  - (s1)+136～139に対応する16進数コードを示し、プログラム上では `$MOV "USER001" D136` と入力。
- パスワード "PW1234" の設定例
  - (s1)+153～156に対応する16進数コードを示し、プログラム上では `$MOV "PW1234" D153` と入力。
- FTPサーバアドレスの設定例
  - (s1)+134～135に16進数でIPアドレスを格納し、プログラム上では `DMOV HC0A83319 D134` と入力。
