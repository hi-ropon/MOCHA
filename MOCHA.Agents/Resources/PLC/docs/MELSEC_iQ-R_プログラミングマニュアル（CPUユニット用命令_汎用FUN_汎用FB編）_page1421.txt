### テキスト
#### 33.5 指数演算 EXP(_E)
入力値の指数演算結果を出力する関数EXP(_E)について説明しています。

##### 機能
- (s)に入力されたREAL型/LREAL型データの指数を演算し、(d)から出力します。
- 入力値をA、演算出力値をBとすると、B = e^A（底eは2.71828）で計算されます。
- (s)への入力値はREAL型/LREAL型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は表の通りです。

### 図
- ラダー/FBD/LDの回路例  
  - EN/ENOなし：s入力からd出力へ直接指数演算を行う回路図  
  - EN/ENO付き：EN入力、ENO出力が付いた回路図

### 表
#### ラダー、FBD/LDの使用例

| 種類 | 内容 |
| :--- | :--- |
| EN/ENOなし | d := EXP(s); |
| EN/ENO付き | d := EXP_E(EN, ENO, s); |

#### 設定データ（引数、内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | ANY_REAL |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANY_REAL |

#### 実行条件と演算結果

| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、使用しないようプログラム処理が必要。

#### エラーコード（SD0）

| エラーコード | 内容 |
| :--- | :--- |
| 3402H | (s)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が下記範囲を超えるとき（オーバーフロー発生時）<br> |d| < 2^128 |
