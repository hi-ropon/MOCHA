### テキスト
#### 機能
- (s)で指定された値の指数を演算し、(d)で指定されたデバイスに演算結果を格納する。
- 指数演算では底(e)を「2.71828」として計算する。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が下記範囲を超えるとき（オーバーフローが発生したとき）。|d| < 2^128 |

### 図
- 指数演算の計算式として、常用対数を求める場合は、EXP(P)命令は自然対数で演算を行うため、常用対数に変換するには常用対数に対応する0.43429で割った値を(s)に指定する必要があることを示す。
- 数式：10^X = e^(X / 0.43429)
