### テキスト
#### 機能
(s)で指定されたデバイス番号以降に格納されている文字列データのうち、文字列の左（文字列の先頭）から(n)文字分のデータを、(d)で指定されたデバイス番号以降に格納します。

- 文字列の最終を示すNULLコード（00H）が、文字列データの最後に自動付加されます。
- (n)で指定する文字数が0の場合は、(d)にNULLコード（00H）が格納されます。

#### エラー
| エラーコード(SD0) | 内容 |
|------------------|------|
| 2820H | (s)で指定したデバイス番号以降、デバイス/ラベルメモリの各設定エリアに00Hが設定されていないとき。 |
| 3405H | (s)に指定可能な範囲外のデータを設定したとき。<br>・(s)の文字列が16383文字を超える<br>・(s)の文字列が0文字<br>・(n)が(s)で指定の文字数を超えているとき。 |

### 図
- 文字列データの格納イメージ図：
  - (s)から始まるデバイス番号以降に文字列の各文字のASCIIコードが格納されている様子を示す。
  - (d)から始まるデバイス番号以降に、(s)の左から(n)文字分の文字列が格納されるイメージ。
  - 文字列の終端にはNULLコード（00H）が付加される。

- 例：(n)=7の場合の具体例：
  - (s)のデバイス番号以降に"ABCDEF12345"のASCIIコードが格納されている。
  - (d)のデバイス番号以降には、左から7文字分"ABCDEF1"のASCIIコードが格納される。
  - 7文字目の後にNULLコード（00H）が付加されている。

### 表
| エラーコード(SD0) | 内容 |
| :--- | :--- |
| 2820H | (s)で指定したデバイス番号以降に00Hが設定されていない場合。 |
| 3405H | (s)の文字列が範囲外、文字数が0、または(n)が(s)の文字数を超えている場合。 |
