### テキスト
#### 日付比較命令（LDDT□、ANDDT□、ORDT□）
指定した日付データ同士、または日付データと現在日付を比較する命令です。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| LDDT□、ANDDT□、ORDT□ | 常時実行 |

*注釈*
- バージョン1.035M以降のエンジニアリングツールで対応。
- 比較演算子はEQ（等しい）、NE（等しくない）、GT（より大きい）、LE（以下）、LT（未満）、GE（以上）が使用可能。

### 図
#### ラダー回路図
- LD、AND、OR命令で日付比較を行う構造を示しています。
- 各命令は3つのオペランド(s1, s2, s3)を持ち、比較対象の日付データを指定します。
- 比較演算子はDT=、DT<>、DT>、DT<=、DT<、DT>=が使用可能。

#### FBD/LDブロック図
- EN（入力）、s1、s2、s3（比較対象）、ENO（出力）端子を持つブロック図。
- 比較演算子はLDDT_、ANDDT_、ORDT_とEQ、NE、GT、LE、LT、GEの組み合わせ。
