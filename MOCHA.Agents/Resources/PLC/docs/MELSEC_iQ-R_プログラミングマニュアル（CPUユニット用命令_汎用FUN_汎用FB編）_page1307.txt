### テキスト
#### 機能
(s)で指定したファイルから安全データ同一性チェック用のファイル識別子を読み出し、(d2)で指定したデバイスに格納します。ファイル名は拡張子を含め64文字以内で指定します。詳細は「MELSEC iQ-R CPUユニットユーザーズマニュアル（応用編）」を参照してください。

- (s)で指定可能なファイルは以下の通りです。
- 処理完了(d3)のビットデバイスは、SP.SIDRD命令の処理完了を検出したスキャンのEND命令実行時に自動的にONし、次のスキャンのEND命令でOFFになります。
- SP.SIDRD命令が異常完了時は、異常完了(d3)+1のデバイスが処理完了(d3)のデバイスと同じタイミングでON/OFFします。
- 命令実行時に演算エラーを検出した場合は、処理完了(d3)、異常完了(d3)+1はONしません。
- SP.SIDRD命令実行中はSM753（ファイルアクセス中）がONします。SM753がONの状態でSP.SIDRD命令を実行すると無処理となります。
- SP.SIDRD命令実行時の完了デバイスの動作は図の通りです。

#### 注意事項
- SP.SIDRD命令は割込みプログラムで実行しないこと。割込みプログラムで実行すると誤作動の原因となる。
- SP.SIDRD命令とファイルアクセスする他機能を同時に実行しないこと。
- 命令実行中にCPUユニットの状態をRUN→STOPに変更しても、命令処理は完了するまで継続する。

#### エラー
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3405H             | (s)で指定したファイル名文字列が読み出せない場合。<br>・ファイル名が1文字以上指定されていない。<br>・ファイル名が65文字以上指定された。 |
