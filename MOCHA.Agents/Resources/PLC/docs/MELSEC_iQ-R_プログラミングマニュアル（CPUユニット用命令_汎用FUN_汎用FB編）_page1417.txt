### テキスト
#### 33.2 平方根 (SQRT(_E))
入力値の平方根を演算する関数です。

##### 機能
- (s)に入力されたREAL型/LREAL型データの平方根を演算し、演算結果を(d)に格納します。
- 入力値をA、演算出力値をBとすると、B = √A となります。
- (s)への入力値はREAL型/LREAL型の正の数の範囲内である必要があります。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
|-------------|--------------|-----|
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、そのデータを使用しないようプログラムで処理してください。

### 図
- ラダー、FBD/LD、ST言語での平方根関数の使用例が示されています。  
  - ラダー/FBD/LDでは、入力(s)と出力(d)の接続図があり、EN/ENO付きの場合はENとENO端子もあります。  
  - ST言語では、EN/ENOなしの場合は d := SQRT(s);、EN/ENO付きの場合は d := SQRT_E(EN, ENO, s); と記述します。

### 表
#### 設定データ：内容、種別、データ型

| 引数 | 内容 | 種別 | データ型 |
|------|------|------|----------|
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | ANY_REAL |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANY_REAL |

#### エラー

| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3405H | 入力された値が負の数のとき。 |
