### テキスト
#### 単精度実数SIN演算
単精度実数で指定した角度のSIN（正弦）値を演算します。

対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）です。

##### 実行条件
SIN命令はST、FBD/LDでは対応していません。汎用ファンクションのSINを使用してください（1422ページ参照）。

■実行条件命令一覧
| 命令 | 実行条件 |
| :--- | :--- |
| SIN | パルス信号（立ち下がり） |
| SINP | パルス信号（立ち上がり） |

### 図
#### ラダー図
- (s)と(d)の2つのオペランドを持つSIN命令のラダー図が示されています。
- ST命令形式：ENO:=SINP(EN,s,d);

#### FBD/LD図
- EN（入力）、ENO（出力）、s（角度データ）、d（演算結果格納先）の4端子を持つブロック図。

### 表

| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |

| (s) | SIN（正弦）演算を行う角度データまたは角度データが格納されている先頭デバイス | 0, 2^-126 ≤ |s| < 2^128 | 単精度実数 | ANYREAL_32 |
| (d) | 演算結果を格納する先頭デバイス | — | 単精度実数 | ANYREAL_32 |
| EN | 実行条件 | — | ビット | BOOL |
| ENO | 実行結果 | — | ビット | BOOL |
