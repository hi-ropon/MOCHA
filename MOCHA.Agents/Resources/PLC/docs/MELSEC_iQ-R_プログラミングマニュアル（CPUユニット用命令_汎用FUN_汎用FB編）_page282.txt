### テキスト
#### BIN32ビットブロックデータ加算
指定した2つのBIN32ビットデータブロックを加算する命令について説明しています。

#### 実行条件
- DBK+、DBK+_U命令は立下りエッジで実行
- DBK+P、DBK+P_U命令は立上りエッジで実行

#### 使用可能デバイス
命令のオペランドごとに使用可能なビット、ワード、ダブルワード、間接指定、定数の種類が表で示されています。

### 図
- **ラダー図**  
  s1, s2, d, nの4つの入力を持つ命令ブロックの構造を示しています。  
- **FBD/LD図**  
  EN（実行条件）、s1, s2, n（入力）、ENO（実行結果）のポートを持つ命令ブロックの構造を示しています。  
  （DBKPLUS、DBKPLUSP、DBKPLUS_U、DBKPLUSP_Uのいずれかが入る）
