### テキスト
#### 機能
- (s)で指定した符号付きBIN32ビットデータを単精度実数に変換し、(d)へ格納する。
- 単精度実数は32ビットで処理され、2進数表現時の有効桁数は24ビット、10進数表現時は約7桁。
- 整数値が-16777216～16777215（24ビットBIN値）の範囲を超えた場合、変換結果に誤差が生じる。
- 変換結果は整数値の上位ビットから25ビット目を0挿入し、26ビット目以降を切り捨てた値となる。

#### エラー
- 演算エラーは存在しない。

### 図
- 図は2つのビット列の例を示し、上位ビットから25ビット目に0を挿入し、26ビット目以降を切り捨てる処理を視覚的に説明。
- 1つ目の例はビット列「b31～b0」の一部を示し、変換後の値が「222030032」となる。
- 2つ目の例は別のビット列の例で、変換後の値が「372588928」となる。
- 矢印と注釈で「(1)切り捨て」「(2)0挿入」の処理箇所を明示。

- また、ラダー図風の命令例として、SM402の接点からDMOVP命令でK-123456をD0に、DINT2FLT命令でD0をD100に変換格納する流れが示されている。
