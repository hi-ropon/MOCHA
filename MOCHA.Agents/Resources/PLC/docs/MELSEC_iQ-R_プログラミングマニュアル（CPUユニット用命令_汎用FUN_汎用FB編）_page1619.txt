### テキスト
#### 互換用ファンクションブロック（汎用ファンクションブロック）
ST（インラインST含む）、構造化ラダー/FBDのプログラムで使用されている汎用ファンクションブロックが互換用ファンクションブロックに置き換えられることを説明しています。

#### 動作の差異
互換用ファンクションブロックごとに、オフディレイタイマ時間、オンディレイタイマ時間、パルス幅時間の有効範囲がロングタイマの設定で異なることを説明しています。  
- 最小値はロングタイマの設定値（ms）と同じで、1ms未満の場合は1msとなる  
- 最大値はオフディレイタイマ時間(ms) ≤ 2147483647(ms) × ロングタイマの設定値(ms)  
- 有効範囲を超えると演算エラー（エラーコード：3401H）が発生し、ロングタイマの設定値を見直す必要がある
