### テキスト
#### FOR～NEXT強制終了（BREAK(P)）
FOR～NEXT命令による繰り返し処理を強制的に終了し、指定したポインタに実行を移します。

#### 実行条件
- BREAK命令は立ち上がりエッジで実行されます。
- BREAKP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  BREAK(P)命令のラダー表現が示されていますが、ST言語は対応していません。

- **FBD/LD図**  
  EN（実行条件）入力、ENO（実行結果）出力、P（分岐先ポインタ）、d（繰返し残数）を持つ命令ブロックの図が示されています。
