### テキスト
#### 機能
- (d)で指定したデバイスから、(n)点分の単精度実数を右へ1点シフトする。
- 最上位から1単精度実数分は0になる。
- (n)で指定された値が0のときは無処理となる。

#### エラー
- 演算エラーは発生しない。

### 図（例）
- (n)=6の場合の動作例を示す。
- 上段に元のデバイスの値が並び、下段にシフト後の値が並ぶ。
- 具体的には、(d)+11から(d)+1までの7つのデバイスにそれぞれ数値が入っている。
- 右へ1点シフトするため、(d)+11の値は0に置き換わり、(d)+10の値が(d)+11に、(d)+9の値が(d)+10に…と1つずつ右にずれる。
- (d)+1の値はシフト後は消失する（図中で×印で示されている）。

このように、指定した範囲の単精度実数を右に1点シフトし、最上位は0クリアされる動作を説明している。
