### テキスト
#### 倍精度実数SIN演算 SIND(P)
倍精度実数で指定した角度のSIN（正弦）値を演算します。

##### 注意事項
SIND命令はST、FBD/LDでは対応していません。汎用ファンクションのSINを使用してください。（1422ページ参照）

### 図
- ラダー図：SINDP命令のラダー回路図が示されており、(s)と(d)の2つのオペランドが入力される形状。
- FBD/LD図：EN（入力）、ENO（出力）、s（角度データ）、d（演算結果）を持つブロック図。

### まとめ
SINDP命令は倍精度実数の角度データから正弦値を計算し、結果を倍精度実数で出力します。ST命令では使用できず、FBD/LDでも対応していないため、汎用ファンクションSINを利用する必要があります。実行条件はパルスまたは立ち上がりエッジで指定します。
