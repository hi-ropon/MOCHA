### テキスト
#### ステップ起動
「SET [S□/BL□￥S□]」命令について説明しています。  
対応CPUはR00CPU、R01CPU、R02CPUでバージョン制約はありませんが、RnCPU（R00CPU、R01CPU、R02CPUを除く）はファームウェアバージョン12以降で使用可能です。  
プロセスCPU（プロセスモード）はファームウェアバージョン03以降、プロセスCPU（二重化モード）はファームウェアバージョン18以降で使用可能です。  
エンジニアリングツールのバージョン制約も記載されています。

指定したステップを活性化（起動）させる命令です。

### 図
- **ラダー図**  
  ラダー回路の一部として、(d)というビットデバイスを使い、ENO:=SET(EN,d);という命令が示されています。

- **FBD/LD図**  
  ブロック図でEN（入力）、ENO（出力）、d（ステップ番号）を示しています。
