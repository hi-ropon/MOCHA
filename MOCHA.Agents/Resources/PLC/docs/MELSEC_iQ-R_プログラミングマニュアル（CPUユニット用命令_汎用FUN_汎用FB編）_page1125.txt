#### Point
- CPUバッファメモリの読み出し/書込み可能エリアのリフレッシュエリアについて、リフレッシュ設定しない場合は読み出し/書込み指定可能エリアとして使用可能。
- リフレッシュ設定した場合でも、リフレッシュ送信範囲以降は読み出し/書込み指定可能エリアとして使用可能。
- CPUバッファメモリへのデータ書込みは、CPUバッファメモリアクセスデバイスが使用可能。
- TOP(P)命令、DTO(P)命令では64K以下のバッファメモリアドレスにデータを書き込める。
- 64Kを超えるバッファメモリアドレスに書き込む場合は、TOD(P)命令、DTOD(P)命令を使用する。

### 図
- 縦軸にアドレスを示し、上部に「システムエリア」、その下に「リフレッシュエリア」があり、リフレッシュエリアの下から下端までが「読み出し/書込み指定可能エリア」として示されている。
- 数値は0、2048、524287が示されている。
