### テキスト
#### TIMER_CONT_FB_M/TIMER_CONTHFB_Mの概要
- 積算タイマは、変数がONしている時間を計測するタイマである。
- (s1)の実行条件がONになると計測を開始し、積算タイマには低速積算タイマ（TIMER_CONT_FB_M）と高速積算タイマ（TIMER_CONTHFB_M）の2種類がある。
- (s3)は1000msまでの可変設定が可能（高速積算タイマ時は0.1～100ms）。
- パラメータ設定により計測開始し、(s2)の値が到達すると(d2)がONし、計測値は(d1)に出力される。
- (s1)の実行条件がOFFでも(d1)、(d2)のON/OFF状態は保持される。再度(s1)がONになると保持していた計測値から再開。
- 計測単位は低速・高速タイマで同じ。
- (s2)は0～32767の値指定可能。
- (s3)は-32768～32767の値指定可能だが、負値指定時は0が初期値。
- (d1)のリセットはFBの(s1)を直接リセットする。

### 図
#### ラダープログラム例
- RST命令でM0をリセットし、TIMER_CONT_FB_M_1.Coilに出力。

#### STプログラム例
- RST(M0,TIMER_CONT_FB_M_1.Coil)の記述例。

#### 回路例
- TIMER_CONT_FB_MのブロックにX0（Coil）、200（Preset）、0（ValueIn）を入力し、D10（ValueOut）、M10（Status）を出力。

#### タイミングチャート
- X0のON/OFFによりタイマが動作。
- X0がONで15秒間計測し、D10は0から150まで増加。
- 15秒経過後X0がOFFになると5秒間計測継続し、D10は151から200まで増加。
- M10は計測完了時にONになる。

### エラー
- 演算エラーはなし。
