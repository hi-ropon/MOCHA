### テキスト
#### TIMER_LOW_FB_Mの概要
- 計測単位が1～1000msの低速タイマです。
- (s1)の実行条件がONになると現在値の計測を開始し、(s3)×1～1000msの範囲で変更可能なパラメータで計測を開始します。
- (s2)×1～1000msまで計測値が到達すると(d2)がONになります。
- 現在の計測値は(d1)に出力されます。
- (s1)の実行条件がOFFになると現在値は0となり、(d2)もOFFになります。
- 低速タイマの計測単位（タイマ時限設定）はデフォルト値が100msで、1ms単位で1～1000msの範囲で変更可能です。
- (s2)には0～32767の値が指定可能です。
- (s3)には-32768～32767の値が指定可能ですが、負の値を指定した場合は0が初期値となります。

### 図
#### 回路例
- TIMER_LOW_FB_Mの機能ブロックに対して、M0がコイル入力、10がプリセット値、1がValueIn、D10がValueOut、M10がStatusとして接続されています。

#### タイミングチャート
- M0のONでタイマが動作開始し、D10が1から900msの間に変化し、M10はOFFからONに変わるタイミングを示しています。
- M0がOFFになるとタイマはリセットされ、D10は1に戻り、M10もOFFになります。
