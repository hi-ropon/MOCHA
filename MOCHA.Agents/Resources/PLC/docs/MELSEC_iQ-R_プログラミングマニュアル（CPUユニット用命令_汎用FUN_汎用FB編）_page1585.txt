### テキスト
#### オートチューニング(ステップ応答法)+PID制御のプログラム例

- (0) 目標値設定<50℃>、入力フィルタ(α)設定、微分ゲイン(KD)設定<0%>、出力値上限設定<2秒ON>、出力値下限設定<0秒ON>
- (88) オートチューニング開始
- (104) オートチューニング動作フラグ、オートチューニング用サンプリングタイム(Ts)設定<3秒>、動作設定(ACT)オートチューニング開始、オートチューニング用出力値設定<1.8秒ON>
- (191) 通常動作時のサンプリングタイム(Ts)設定<500ms>
- (224) A/D変換ユニットより測定値(PV)を入出力データエリアに設定
- (258) PID動作初期化

### 図（ラダー図）
- SM402、X2、X3、M0、M1、SM400などの内部リレーや入力信号を条件に、MOVP、PLS、SET、MOV、RST命令が使用されている。
- MOVP命令で定数(K500、K70、K0、K2000など)をデータレジスタ(D500、D512、D515、D532、D533)に設定。
- PLS命令でM0をパルス出力。
- SET命令でM1をセット。
- MOV命令でK3000、H31、K1800、K500、U8&G400などの値を各データレジスタ(D510、D511、D502、D510、D501)に移動。
- RST命令でD502をリセット。
- ラダー図はオートチューニング開始からPID制御の初期化までの一連の動作を示している。

以上より、本ページはオートチューニングとPID制御のプログラム構成例をラダー図と対応する動作説明で示している。
