### テキスト
#### 機能
- (d)で指定したデバイスから単精度実数(n1)点分のデータ範囲内で、右へ(n2)点分シフトさせる。
- 最上位から(n2)点分は0になる。
- (n1)または(n2)で指定した値が0のときは無処理となる。
- (n1)≦(n2)の場合は、(d)で指定したデバイスから(n1)点分のデータはすべて0となる。

#### エラー
- 演算エラーはありません。

### 図
- 例として、(n1)=9、(n2)=4の場合のデータシフトの動作を示している。
- 上段に元のデータが(d)+17(d)+16 ... (d)のアドレスに対応して配置されている。
- 下段は右へ4点分シフトした結果で、左側4点分は0に置き換わり、残りのデータが右にずれている。
- データ例として、1.2345、123.45678、-1.234567、-12.3456、-123.45678、12.345、3.456789、45.678901、567.89が示されている。
