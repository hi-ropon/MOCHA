### テキスト
#### 倍精度実数データ転送 EDMOV(P)
倍精度実数データを指定されたデバイスへ転送する命令です。

##### 命令形式
- ラダー回路: ENO:=EDMOV(EN,s,d); ENO:=EDMOVP(EN,s,d);
- FBD/LD回路: EN（入力）、ENO（出力）、s（転送元）、d（転送先）

##### 実行条件
| 命令   | 実行条件の説明                   |
|--------|--------------------------------|
| EDMOV  | 立ち上がりエッジで実行          |
| EDMOVP | 立ち下がりエッジで実行          |

##### 設定データ
- (s): 転送するデータまたは転送元データが格納されている先頭デバイス。範囲は0から2^1022 ≤ |s| < 2^1024。データ型は倍精度実数（ANYREAL_64）。
- (d): 転送先のデータを格納する先頭デバイス。データ型は倍精度実数（ANYREAL_64）。
- EN: 実行条件（ビット型）
- ENO: 実行結果（ビット型）

##### 使用可能デバイス
| オペランド | ビットデバイス | ワードデバイス                         | ダブルワードデバイス | 間接指定 | 定数 | その他 |
|------------|----------------|--------------------------------------|----------------------|----------|------|--------|
| (s)        | -              | T, ST, C, D, W, SD, SW, FD, R, ZR, RD | Z, LT, LST, LC       | ○        | -    | ○      |
| (d)        | -              | -                                    | -                    | ○        | -    | -      |

##### 機能
(s)で指定されたデバイスに格納されている倍精度実数データを(d)で指定されたデバイスへ転送します。

##### エラー
演算エラーはありません。

### 図
- ラダー回路の命令表現: (s)と(d)のオペランドを持つ命令ブロック。
- FBD/LD回路の命令表現: EN入力、ENO出力、s入力、d出力のブロック図。
- 倍精度実数データ転送のイメージ: (s)の3連続デバイスに格納された倍精度実数が転送され、(d)の3連続デバイスに同じ値が格納される様子を示す。
