### テキスト
#### 符号付きBIN32ビットデータ→符号なしBIN16ビットデータ変換
命令名は「DINT2UINT(P)」で、指定した符号付きBIN32ビットデータを符号なしBIN16ビットデータに変換します。

対応しているCPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）など複数の種類です。

バージョン1.035M以降のエンジニアリングツールで対応しています。

#### 実行条件
- DINT2UINT命令は立ち下がりエッジで実行
- DINT2UINTP命令は立ち上がりエッジで実行

### 図
#### ラダー図
- (s)入力（符号付きBIN32ビットデータ）
- (d)出力（符号なしBIN16ビットデータ）

#### FBD/LD図
- EN（実行条件入力）
- ENO（実行結果出力）
- s（入力データ）
- d（出力データ）
