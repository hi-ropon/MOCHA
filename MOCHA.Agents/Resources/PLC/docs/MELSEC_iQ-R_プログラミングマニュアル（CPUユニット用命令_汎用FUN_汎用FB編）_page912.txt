### テキスト
#### 機能
- (s)で指定された角度のSIN（正弦）値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- (s)で指定するSIN値は-1.0～1.0の範囲で設定可能。
- (d)に格納する角度（演算結果）はラジアン単位で格納される。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### 注意事項
- ラジアンと角度の変換については、RAD(P)命令、DEG(P)命令を参照すること。

### 表
#### エラーコード一覧

| エラーコード (SD0) | 内容                                   |
|--------------------|--------------------------------------|
| 3402H              | (s)に指定した値が0、非正規化数、非数、±∞のとき。 |
| 3405H              | (s)に指定した値が-1.0～1.0の範囲外のとき。       |

### ポイント
- ラジアンと角度の変換は別命令（RAD(P)、DEG(P)）を参照すること。
