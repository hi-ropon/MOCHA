### テキスト
#### BCD4桁→BIN16ビットデータ変換
指定したBCD4桁データをBIN16ビットデータに変換する命令です。

#### 実行条件
- BIN命令：立下りエッジで実行
- BINP命令：立上りエッジで実行

#### 機能
(s)で指定されたデバイスのBCD4桁データ（0～9999）をBIN16ビットデータに変換し、(d)で指定されたデバイスに格納します。

### 図
- ラダー図：BIN命令のラダー回路例が示されており、入力(s)と出力(d)がある。
- FBD/LD図：FBD/LD形式での命令ブロックの形状が示されている。EN（実行条件）、ENO（実行結果）、s（BCD4桁データ）、d（BIN16ビットデータ）端子がある。
- BCD4桁→BIN16ビット変換のビット対応図：
  - (s)のBCD4桁データ「9999」の各桁のビット配置（10^3～10^0の重み）を示し、
  - (d)のBIN16ビットデータのビット配置（2^15～2^0の重み）に変換される様子を図示。
  - なお、BINデータの最上位ビットは必ず0になることが注記されている。
