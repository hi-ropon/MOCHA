### テキスト
#### 機能
■DROR(P)
- (d)で指定されたデバイスのBIN32ビットデータを、SM700（キャリフラグ）を含めないで(n)ビット右へ回転させる命令。
- SM700（キャリフラグ）は、DROR(P)命令の実行前の状態によりONまたはOFFになる。
- (d)にビットデバイスが指定された場合は、桁指定で指定されたデバイス範囲内で回転する。
- 実際に回転するビット数は、(n)を桁指定で指定された点数で割った余りとなる。例として(n)=31、点数=24ビットの場合、31÷24の余り7となり7ビット右へ回転。
- (n)は0～31の値を指定可能。32以上の値は32で割った余りで回転。例として(n)=34の場合、34÷32の余り2となり2ビット右へ回転。

■DRCR(P)
- (d)で指定されたデバイスのBIN32ビットデータを、SM700（キャリフラグ）を含めて(n)ビット右へ回転させる命令。
- SM700（キャリフラグ）は、DRCR(P)命令の実行前の状態によりONまたはOFFになる。
- (d)にビットデバイスが指定された場合は、桁指定で指定されたデバイス範囲内で回転する。
- 実際に回転するビット数は、(n)を桁指定で指定された点数で割った余りとなる。例として(n)=31、点数=24ビットの場合、31÷24の余り7となり7ビット右へ回転。
- (n)は0～31の値を指定可能。32以上の値は32で割った余りで回転。例として(n)=34の場合、34÷32の余り2となり2ビット右へ回転。

#### エラー
- 演算エラーはありません。
