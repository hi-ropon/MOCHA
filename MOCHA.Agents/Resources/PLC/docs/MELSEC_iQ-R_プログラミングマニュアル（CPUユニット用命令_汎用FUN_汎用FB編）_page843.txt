### テキスト
#### 11.1 浮動小数点命令
単精度実数をa接点扱いで比較演算する命令「LDE□」「ANDE□」「ORE□」について説明しています。

- 命令は単精度実数を比較し、結果をビットで出力します。
- バージョン1.035M以降のエンジニアリングツールで対応。
- 比較演算子はEQ（等しい）、NE（等しくない）、GT（大きい）、LE（以下）、LT（未満）、GE（以上）が使用可能。

##### 実行条件
| 命令       | 実行条件   |
|------------|------------|
| LDE□, ANDE□, ORE□ | 常時実行 |

##### 設定データ
| オペランド | 内容                                       | 範囲                                   | データ型       | データ型(ラベル) |
|------------|--------------------------------------------|--------------------------------------|----------------|------------------|
| (s1)       | 比較データまたは比較データが格納されている先頭デバイス | 0, 2^-126 ≤ |s1| < 2^128           | 単精度実数     | ANYREAL_32       |
| (s2)       | 比較データまたは比較データが格納されている先頭デバイス | 0, 2^-126 ≤ |s2| < 2^128           | 単精度実数     | ANYREAL_32       |
| EN         | 実行条件                                   | —                                    | ビット         | BOOL             |
| ENO        | 実行結果                                   | —                                    | ビット         | BOOL             |

### 図
- **ラダー回路図**  
  - LD□、AND□、OR□の3種類の比較命令のラダー表現を示す。  
  - 各命令は2つの入力(s1, s2)を比較し、条件に応じて出力を制御する。

- **FBD/LD命令ブロック図**  
  - 入力EN、s1、s2を受け取り、出力ENOを返す命令ブロックの構造を示す。  
  - 比較演算子はLDE□、ANDE□、ORE□とEQ、NE、GT、LE、LT、GEの組み合わせが可能。
