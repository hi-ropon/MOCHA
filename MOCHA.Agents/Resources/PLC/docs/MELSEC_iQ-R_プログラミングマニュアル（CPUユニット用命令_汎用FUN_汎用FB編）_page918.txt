### テキスト
#### 機能
- (s)で指定された角度のSIN（正弦）値を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- 角度はラジアン単位（角度×π÷180）で設定する。
- 演算結果が0またはアンダーフローが発生した場合、結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |

#### Point
- 角度とラジアンの変換については、RADD(P)命令、DEGD(P)命令を参照。
- 参照ページ：945ページ（RADD(P)）、947ページ（DEGD(P)）

### 図
- SIN関数の引数と結果のデバイス番号の対応を示す。
  - (s)+3、(s)+2、(s)+1、(s)の4つのデバイスに角度の倍精度実数が格納される。
  - 演算結果は(d)+3、(d)+2、(d)+1、(d)の4つのデバイスに倍精度実数として格納される。
