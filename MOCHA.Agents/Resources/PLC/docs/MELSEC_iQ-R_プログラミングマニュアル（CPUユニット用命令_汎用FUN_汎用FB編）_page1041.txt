### テキスト
#### 注意事項
完了デバイス(d1)+1の異常完了信号がONになり、(d2)の完了ステータスにエラーコードが格納される場合について説明しています。具体的には以下のケースが該当します。
- データベースアクセス命令実行中にDBINSERT(P)命令を実行した場合
- (s1)で範囲外の識別番号を指定した場合
- (s1)で未オープンの識別番号を指定した場合
- ファームウェアバージョンが"28"より前のシーケンサCPUで、(s1)にファームウェアバージョンが"28"以降のシーケンサCPUで構築または操作したデータベースを指定した場合
- (s2)で指定したテーブル名が存在しない場合
- (s2)で指定したテーブル名の文字数が32文字を超える場合
- (s3)で追加するフィールド数を範囲外の値に設定した場合
- (s3)+1～(s3)+口で指定したフィールド名に範囲外のフィールド名を設定した場合
- (s4)で追加するレコード数を範囲外の値に設定した場合
- データベースのINSERT処理に失敗した場合
- (s4)+2で設定する1レコード分のデータの範囲が、(s4)+1で指定したサイズと一致していない場合

### 表
| エラーコード (SD0) | 内容 |
| :----------------- | :--- |
| 2820H              | (s2)、(s3)、(s4)、(d1)で指定した範囲が該当デバイス/ラベルを超える場合 |

オペランドの完了ステータスに格納されるエラーコードについては、1066ページの「データベースアクセス命令で発生するエラーコード」を参照するよう案内されています。
