### テキスト
#### アップダウンカウンタ CTUD(_E)
信号の立ち上がり回数をカウントアップおよびカウントダウンします。対応CPUはRnCPU、RnENCPU、RnPCPU（三重化、一般）、RnPSFCPU（安全、一般）、RnSFCPU（安全、一般）ですが、一部のCPUでは使用できません。

### 図
- **ラダー図（EN/ENOなし）**  
  左側に「s1～s4、n」と「d1～d3」の端子が並んだシンプルなラダー図が示されています。  
- **FBD/LD（EN/ENO付き）**  
  ブロック図形式で、EN（実行条件入力）、ENO（出力状態）、s1～s4（入力信号）、n（最大値/開始値）、d1～d3（出力信号）が示されています。

### 表
#### 設定データ（内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s1(CU) | カウントアップ信号入力 | 入力変数 | BOOL |
| s2(CD) | カウントダウン信号入力 | 入力変数 | BOOL |
| s3(R) | カウント値リセット | 入力変数 | BOOL |
| s4(LD) | カウント値セット | 入力変数 | BOOL |
| n(PV) | カウント最大値/開始値 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常または停止) | 出力変数 | BOOL |
| d1(QU) | カウントアップ完了 | 出力変数 | BOOL |
| d2(QD) | カウントダウン完了 | 出力変数 | BOOL |
| d3(CV) | 現在カウント値 | 出力変数 | INT |

このアップダウンカウンタは、入力信号の立ち上がりを検出してカウントアップまたはカウントダウンを行い、リセットやセットも可能です。出力にはカウント完了信号や現在のカウント値が含まれます。
