### テキスト
#### 機能
(s1)で指定した単精度実数を(s2)で指定した単精度実数でべき乗演算し、(d)で指定したデバイスに演算結果を格納します。

- (s1)、(s2)に指定できる値および格納できる値は、0、2^-126 ≤ 設定値（格納値） < 2^128 の範囲です。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

#### 注意事項
- 演算結果が0またはアンダーフローが発生した場合は、結果は0となる。
- エンジニアリングツールからの入力値設定時に丸め誤差が生じる可能性がある。

### 図
べき乗演算のデータの流れを示す図があり、以下のように構成されています。

- (s2)+1 と (s2) が「べき乗データ」として扱われる。
- (s1)+1 と (s1) が「被べき乗データ」として扱われる。
- (d)+1 と (d) に演算結果が格納される。
- 図中に「単精度実数」として (s1)+1 (s1) と (s2)+1 (s2) のべき乗を行うことが示されている。
