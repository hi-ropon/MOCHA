### テキスト
#### 39.6 文字列の削除 DELETE(_E)
文字列の任意の範囲を削除して出力する関数です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス・三重化・一般）、RnSPFCPU（安全・一般）です。

##### ラダー、FBD/LDでの使用例
- EN/ENOなしの場合とEN/ENO付きの場合の関数ブロック図が示されています。
- ST形式では、EN/ENOなしの場合は d:=DELETE(s,n1,n2); EN/ENO付きの場合は d:=DELETE_E(EN,ENO,s,n1,n2); と記述します。

#### 設定データ
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力文字列 | 入力変数 | ANYSTRING_SINGLE |
| n1(L) | 削除文字数指定 | 入力変数 | INT |
| n2(P) | 削除先頭位置指定 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力文字列 | 出力変数 | ANYSTRING_SINGLE |

#### 機能
##### 演算処理
- (s)へ入力された文字列の任意位置から指定文字数分のデータを削除し、残りの文字列を(d)から出力します。
- 削除する文字数は(n1)への入力値で指定し、1以上の値をとります。
- 削除する文字列の先頭位置は(n2)への入力値で指定します。

##### 例
(n1)、(n2)への入力値が5の場合の動作例が示されています。

- 入力文字列(s): "ABCDEF12345"
- 削除先頭位置(n2): 5文字目（0から数えた位置）
- 削除文字数(n1): 5文字

図では、上位バイト・下位バイトに分けて16進数表記でバイト単位のデータが示され、削除前後の状態が比較されています。削除後は指定位置から5文字分が削除され、残りの文字列が(d)に出力されます。

##### 入力値の範囲
- (s)への入力値はSTRING型のデータ値で0～255バイトの範囲内。
- (n1)への入力値はINT型で0～255の範囲内。ただし(s)の文字列長以内に限る。
- (n2)への入力値はINT型で1～255の範囲内。ただし(s)の文字列長以内に限る。
