### テキスト
#### 単精度実数平方根（ESQRT(P)）
単精度実数で指定した値の平方根を演算する命令です。

#### 実行条件
- ESQRT命令は立ち上がりエッジで実行されます。
- ESQRTP命令は立ち下がりエッジで実行されます。

#### 機能
- オペランド(s)で指定された値の平方根を計算し、(d)で指定されたデバイスに結果を格納します。
- (s)で指定する値は正の数のみ設定可能で、負の数では演算できません。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

### 図
- **ラダー図**：ESQRT命令のラダー回路表現。入力(s)、出力(d)が示されています。
- **FBD/LD図**：ファンクションブロック図でのEN（実行許可）、ENO（実行結果）、s（入力値）、d（出力値）の関係を示すブロック図。
- **数式図**：単精度実数の平方根計算のイメージ。√((s)+1) → (d)+1 の形で演算結果を格納する様子を示しています。
