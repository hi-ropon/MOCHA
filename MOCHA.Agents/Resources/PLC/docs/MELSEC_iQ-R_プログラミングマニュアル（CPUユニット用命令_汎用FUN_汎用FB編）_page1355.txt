### テキスト
#### 32.33 DINT型→REAL型変換
「DINT_TO_REAL(_E)」関数は、DINT型データをREAL型データへ変換します。

##### 機能
- 入力されたDINT型データ(s)をREAL型データ(d)に変換して出力します。
- 入力値はDINT型のデータ値で、REAL型データは32ビット単精度で処理され、有効桁数は約7桁です。
- 整数値が-16777216～16777215の範囲を超えると、変換値に誤差（丸め誤差）が生じます。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、dから演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |  
| :--- | :--- |  
| EN | ENO | (d) |  
| TRUE（演算実行） | TRUE | 演算出力値 |  
| FALSE（演算停止） | FALSE*1 | 不定値 |  

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)のデータを使用しないようプログラムで処理してください。

##### エラー
演算エラーはありません。

### 図
- ラダー回路およびFBD/LDでのDINT_TO_REAL関数のシンボル図が示されています。  
  - EN/ENOなしの場合は、s（入力）からd（出力）への変換を示すシンプルな図。  
  - EN/ENO付きの場合は、EN（実行条件）、ENO（出力状態）、s（入力）、d（出力）を含む図。

### 表
#### 設定データ（引数一覧）

| 引数 | 内容 | 種別 | データ型 |  
| :--- | :--- | :--- | :--- |  
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |  
| s(IN) | 入力 | 入力変数 | DINT |  
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |  
| d | 出力 | 出力変数 | REAL |
