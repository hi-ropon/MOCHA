### テキスト
#### 単精度実数ラジアン→角度変換（DEG(P)）
角度の大きさの単位を、単精度実数で指定したラジアン単位から度単位に変換する命令です。

#### 実行条件
- DEG命令は立ち下がりエッジで実行されます。
- DEGP命令は立ち上がりエッジで実行されます。

### 図
- **ラダー回路**  
  ラジアン値(s)を入力し、度単位の値(d)を出力する命令のラダー表現が示されています。  
  (s)が入力、(d)が出力です。

- **FBD/LD回路**  
  ブロック図でEN（実行条件）、ENO（実行結果）、s（ラジアン入力）、d（度出力）の接続が示されています。
