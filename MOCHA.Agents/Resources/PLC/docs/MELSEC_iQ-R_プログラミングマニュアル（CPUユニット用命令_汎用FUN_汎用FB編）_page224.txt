### テキスト
#### 機能
- 下限値(s1)のBIN16ビットデータと上限値(s2)のBIN16ビットデータの範囲に対し、比較データ(s3)のBIN16ビットデータで比較し、その結果に応じて(d)、(d)+1、(d)+2のいずれかがONになる。

#### 図の説明
- X0接点からZCP接点を経て、(s1)、(s2)、(s3)、(d)の各ビットが並ぶ。
- (d)は(s1) > (s3)のときON
- (d)+1は(s1) ≤ (s3) ≤ (s2)のときON
- (d)+2は(s3) > (s2)のときON

#### 注意事項
- 下限値(s1)は上限値(s2)より小さい値に設定すること。
- もし下限値(s1)が上限値(s2)より大きい場合、上限値(s2)は下限値(s1)と同じ値として扱われる。

#### エラー
- 演算エラーは発生しない。
