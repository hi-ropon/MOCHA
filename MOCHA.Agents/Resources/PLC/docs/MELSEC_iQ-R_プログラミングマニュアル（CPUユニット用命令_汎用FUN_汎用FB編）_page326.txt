### テキスト
#### 16ビットデータ否定排他的論理和 WXNR(P) [オペランドが2個の場合]
指定した2つのBIN16ビットデータの否定排他的論理和（XORの否定）を行う命令です。

- 対応CPU：RnCPU、RnENCPU（プロセス、三重化、一般、安全）、RnSPFCPU（一般、安全）
- ラダー回路での表現あり（ST、FBD/LDは対応していません）
- ST命令は対応していません（3オペランドの場合は別ページ参照）
- FBD/LDも対応していません（3オペランドの場合は別ページ参照）

#### 実行条件
| 命令   | 実行条件のイメージ |
|--------|--------------------|
| WXNR   | パルス信号          |
| WXNRP  | 立ち上がり信号      |
