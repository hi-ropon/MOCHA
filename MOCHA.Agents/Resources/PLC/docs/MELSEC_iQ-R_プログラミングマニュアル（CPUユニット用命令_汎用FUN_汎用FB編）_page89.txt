### テキスト
#### BIN32ビットデータ上下バイト交換
指定したデバイスの上下8ビットの値を交換する命令について説明しています。

#### 1ビットデータ転送
指定したビットデータを特定のアドレスに格納する命令について説明しています。

#### nビットデータ転送
複数のビットデータを一括して転送する命令について説明しています。

### 表
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DSWAP / DSWAPP | 指定したデバイスの上下8ビットの値を交換します。<br>図解では、16ビットデータの上位8ビット(b15～b8)と下位8ビット(b7～b0)が入れ替わる様子を示しています。 | 549ページ DSWAP(P) |
| MOVB / MOVBP | (s)で指定したビットデータを、(d)へ格納します。 | 551ページ MOVB(P) |
| BLKMOVB / BLKMOVBP | (s)から(n)点分のビットデータを、(d)から(n)点分のビットデータへ一括転送します。 | 553ページ BLKMOVB(P) |

### 図解説明
- BIN32ビットデータ上下バイト交換命令の図解では、16ビットのデータの上位8ビットと下位8ビットが入れ替わる様子を示しています。  
- b15～b8が上位8ビット、b7～b0が下位8ビットであり、(d)と(d)+1のアドレスに格納されたデータの上下バイトが交換されます。
