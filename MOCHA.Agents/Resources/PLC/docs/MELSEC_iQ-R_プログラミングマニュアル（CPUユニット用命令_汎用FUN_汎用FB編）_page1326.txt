### テキスト
#### 32.11 WORD型→TIME型変換
関数「WORD_TO_TIME(_E)」はWORD型データをTIME型データへ変換します。対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）です。

#### ラダー、FBD/LDでの使用例
- EN/ENOなしの場合：d = WORD_TO_TIME(s);
- EN/ENO付きの場合：d = WORD_TO_TIME_E(EN, ENO, s);

#### 設定データ
##### 引数の内容・種別・データ型
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | WORD |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | TIME |

#### 機能
##### 演算処理
- 入力されたWORD型データ(s)をTIME型データに変換し、dから出力します。
- sへの入力値はWORD型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、dから演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となります。その場合はdから出力されたデータを使用しないようプログラム処理を行ってください。

#### エラー
演算エラーはありません。

### 図
- ラダー/FBD/LDでの関数ブロック例  
  EN/ENOなしはs入力からd出力へ、EN/ENO付きはEN入力、ENO出力を持つブロック図が示されています。

- WORD型からTIME型への変換イメージ  
  WORD値0はTIME型でT#0msに、WORD値1234はT#1s234msに変換される例が示されています。
