### テキスト
#### 39.7 文字列の置換
REPLACE(_E)関数は、文字列の任意の範囲を置換して出力します。対応するCPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）など複数あります。

### 図
- ラダー、FBD/LD形式の関数ブロック図では、入力としてs1、s2、n1、n2があり、出力がdとなっています。
- ST言語では、[EN/ENOなし]の場合は d:=REPLACE(s1,s2,n1,n2); と記述し、[EN/ENO付き]の場合は d:=REPLACE_E(EN,ENO,s1,s2,n1,n2); と記述します。

### 表
#### 設定データ（内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s1(IN1)、s2(IN2) | 入力 | 入力変数 | ANYSTRING_SINGLE |
| n1(L) | 置換文字数指定 | 入力変数 | INT |
| n2(P) | 置換先頭位置指定 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANYSTRING_SINGLE |

この関数は、指定した文字列s1の中で、s2をn1文字分、n2の位置から置換し、その結果をdに出力します。実行条件ENにより動作が制御され、正常終了時はENOがTRUEとなります。
