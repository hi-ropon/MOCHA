### テキスト
#### エラーコード一覧と内容説明
このページでは、REAL型→LREAL型変換に関するエラーコードとその内容が記載されています。

##### エラーコード 3402H
- 内容: (s)の内容が-0、または指定された範囲内でない場合に発生するエラー。
- 範囲条件:
  - -2^128 < (s) ≤ -2^126
  - 0
  - 2^126 < (s) < 2^128
- 数値表記例: (E-3.40282347+38～E-1.17549435-38、0、E1.17549435-38～E3.40282347+38)

##### エラーコード 3403H
- 内容: (d)が指定された範囲を超えた場合に発生するオーバーフローエラー。
- 範囲条件: |(d)| < 2^128
