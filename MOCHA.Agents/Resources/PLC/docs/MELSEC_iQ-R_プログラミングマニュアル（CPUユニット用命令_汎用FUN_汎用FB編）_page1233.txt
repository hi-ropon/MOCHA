### テキスト
#### 手動モード時の測定値の設定値への転送機能
手動モード時にもPIDCONT命令を実行可能で、SM792（PIDバンプレス処理完全微分用）およびSM794（不完全微分用）のON/OFFにより、PIDCONT命令実行時にA/D変換ユニットから取り込んだ測定値を設定値に転送するか選択できます。デフォルトはOFFです。設定値および測定値はPIDCONT命令で指定したデバイスまたはラベルに格納されます。

- OFFの場合：
  - PIDCONT命令実行時に測定値を設定値に転送します。
  - 手動モードから自動モードに切り換えた際、手動モード時の操作量出力を継続可能。
  - 手動モードで設定値を変更すると、出力していた操作量から設定値への制御が可能。

- ONの場合：
  - PIDCONT命令実行時に測定値を設定値に転送しません。
  - 手動モードから自動モードに切り換えると、手動モード時の操作量出力から設定値への制御が可能。
  - 自動モードへの切換え前に設定値を格納する必要あり。

**Point**
- OFF時は測定値を設定値に転送するため、モード切換え時に測定値と設定値に差がなく操作量の急激な変化が起きない。ただし切換え後の設定値は自動モードの目標値と異なるため、プログラムで段階的に変化させる必要がある。
- ON時は測定値を設定値に転送しないため、モード切換え時に測定値と設定値に差が生じる。差が大きい場合は操作量の急激な変化が発生することがあるため、測定値を設定値に近づけてから切換えるシステムで使用推奨。プログラムで段階的に変化させる必要はなく、すぐに自動モードで実施可能。

#### PID制御用データ、入出力データの設定範囲の変更機能
PID制御用データおよび入出力データの設定範囲は任意の範囲に変更可能です。設定を有効にするには、SD792～SD793（PIDリミット制限設定完全微分用）、SD794～SD795（不完全微分用）の該当ループに対するビットをONにします。

### 表
#### SM792/SM794 動作内容

| SM792/SM794 | 動作内容 |
|-------------|----------|
| OFF         | ・PIDCONT命令実行時に測定値を設定値に転送<br>・手動モード→自動モード切換え時、手動モード時の操作量出力を継続可能<br>・手動モードで設定値を変更すると出力していた操作量から設定値への制御が可能 |
| ON          | ・PIDCONT命令実行時に測定値を設定値に転送しない<br>・手動モード→自動モード切換え時、手動モード時の操作量出力から設定値への制御が可能<br>・自動モードへの切換え前に設定値を格納する必要あり |

#### PIDリミット制限設定（不完全微分用SD794/SD795、完全微分用SD792/SD793）

| 不完全微分 | 完全微分 | 動作内容 |
|------------|----------|----------|
| SD794      | SD792    | b0～b15の各ビットがループ1～ループ16に対応。ビットが1の場合はPIDリミット制限なし、0は制限あり（デフォルト）。 |
| SD795      | SD793    | b0～b15の各ビットがループ17～ループ32に対応。ビットが1の場合はPIDリミット制限なし、0は制限あり（デフォルト）。 |

- ビットの意味：
  - 0：PIDリミット制限あり（デフォルト）
  - 1：PIDリミット制限なし
