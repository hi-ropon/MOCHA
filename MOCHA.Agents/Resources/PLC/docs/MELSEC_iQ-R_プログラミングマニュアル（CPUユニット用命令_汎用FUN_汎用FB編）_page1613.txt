### 表
#### GX Works2命令名と互換用命令名一覧
この表は、GX Works2で使用される命令名と、それに対応する互換用命令名を示しています。さらに、それらの命令が使用可能なプログラム形式（ST：構造化テキスト、FBD：ファンクションブロックダイアグラム）を○印で示し、参照先のマニュアルやページ番号も記載されています。

| 項目 | 内容 |
| :--- | :--- |
| GX Works2命令名 | GX Works2での命令名 |
| 互換用命令名 | 互換性を持つ別の命令名 |
| 使用プログラム | ST（構造化テキスト）、FBD（ファンクションブロックダイアグラム）での使用可否を○で表示 |
| 参照先 | 詳細説明が記載されているマニュアル名やページ番号 |

#### 表の特徴的な内容例
- 「PRR_M」命令は互換用命令「G_PRR_M2」と対応し、ST・FBD両方で使用可能。参照先は「MELSEC iQ-R プログラミングマニュアル（ユニット専用命令編）」。
- 「RIGHT」命令は互換用命令「RIGHT_M2」と対応し、ST・FBD両方で使用可能。参照先は「1471ページ LEFT(_E), RIGHT(_E)」。
- 「ROL」命令は互換用命令「ROL_M2」と対応し、ST・FBD両方で使用可能。参照先は「610ページ ROL(P), RCL(P)」。
- 「SIN」命令は互換用命令「SIN_M2」と対応し、ST・FBD両方で使用可能。参照先は「1422ページ SIN(_E)」。
- 「SORT」命令は互換用命令「SORT_M2」と対応し、ST・FBD両方で使用可能。参照先は「733ページ SORTD(_U)」。

このように、各命令の互換性や使用可能なプログラム形式、詳細な説明が記載されているマニュアルのページ番号が一覧で確認できます。
