### テキスト
#### 機能
■EGP命令
- EGP命令は演算結果をエッジリレー（V）で記憶します。
- 演算結果が立ち上がり（OFF→ON）時にON（導通状態）になります。
- 立ち上がり以外の変化（ON→ON、OFF→OFF、ON→OFF）の場合はOFF（非導通状態）になります。
- サブルーチンプログラムやFOR～NEXT命令でインデックス修飾されたプログラムのパルス演算に使用可能。
- AND命令扱いで使用できる。
- サブルーチンプログラムで使用した場合の動作は図の通り。

■EGF命令
- EGF命令も演算結果をエッジリレー（V）で記憶します。
- 演算結果が立ち下がり（ON→OFF）時にON（導通状態）になります。
- 立ち下がり以外の変化（OFF→ON、ON→ON、OFF→OFF）の場合はOFF（非導通状態）になります。
- サブルーチンプログラムやFOR～NEXT命令でインデックス修飾されたプログラムのパルス演算に使用可能。
- AND命令扱いで使用できる。

#### エラー
- 演算エラーはありません。

#### Point
- EGP命令、EGF命令は、これらの命令の直前にあるLD命令から命令直前までの演算結果で動作するため、AND命令（演算開始、直列接続、並列接続）と同一位置で使用すること。
- LD、OR命令の位置では使用不可。
- 下記の回路ブロックの位置では使用不可。

### 図
1. EGP命令の動作タイミングチャート
- X0、X1の立ち上がり・立ち下がりに応じてV0、V1がON/OFFになる様子を示す。
- 具体的には、X0の立ち上がりでON、X0の立ち下がりでOFF、X1の立ち上がりでONになる動作を示している。

2. 使用禁止の回路ブロック例（ラダー図）
- X0、X1、X2の入力とV0のAND回路の後にM0をSETする回路。
- この位置でEGP命令、EGF命令は使用できないことを示す。
