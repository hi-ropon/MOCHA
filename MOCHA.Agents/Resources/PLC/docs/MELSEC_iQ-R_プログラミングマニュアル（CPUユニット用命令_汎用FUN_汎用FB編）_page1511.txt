### テキスト
#### 43.4 カウンタファンクションブロック
「COUNTER_FB_M」というカウンタファンクションブロックについて説明している。  
実行条件が成立するとカウントアップを実行する。

### 図
#### ラダー、FBD/LD
カウンタファンクションブロックのラダー図が示されている。  
入力はs1、s2、s3、出力はd1、d2で構成されている。  
ST（構造テキスト）では、インスタンス名のパラメータとしてCoil:=s1、Preset:=s2、ValueIn:=s3、ValueOut:=d1、Status:=d2が設定されている。

### 表
#### 設定データ（内容、種別、データ型）
| 引数       | 内容                         | 種別   | データ型 |
|------------|------------------------------|--------|----------|
| s1(Coil)   | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL     |
| s2(Preset) | カウンタ設定値               | 入力変数 | INT      |
| s3(ValueIn)| カウンタ初期値               | 入力変数 | INT      |
| d1(ValueOut)| カウンタ現在値              | 出力変数 | ANY16    |
| d2(Status) | 出力                         | 出力変数 | BOOL     |

以上のように、カウンタファンクションブロックは実行条件に応じてカウントアップを行い、設定値や初期値を入力として受け取り、現在値や状態を出力する構成となっている。
