### テキスト
#### 機能
- (d)で指定したデバイスから、(n)ダブルワード分を右へ1ダブルワードシフトする。
- 最上位から1ダブルワード分は0になる。
- (n)で指定された値が0のときは無処理となる。

#### エラー
- 演算エラーは発生しない。

### 図（例）
- (n)=6の場合のシフト動作を示す。
- 上段は元のデバイス値の並びで、(d)+11から(d)までの7つのダブルワードの値が並んでいる。
- 下段は右へ1ダブルワードシフトした結果で、最上位の(d)+11の値は0になり、他は1つ右の位置の値が移動している。
- 具体的な数値例として、元の値は「555555、212121、325476、100000、50000、400000」であり、シフト後は「0、555555、212121、325476、100000、50000」となっている。
