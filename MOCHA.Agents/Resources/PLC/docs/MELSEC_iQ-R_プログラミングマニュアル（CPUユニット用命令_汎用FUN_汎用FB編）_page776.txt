### テキスト
#### $+(P) [オペランドが3個の場合]
文字列データを連結する命令について説明しています。

##### ラダー図
- ラダー図では、3つのオペランド (s1, s2, d) を用いた命令の形が示されています。

##### ST言語
- ENO := STRINGPLUS(EN, s1, s2, d);
- ENO := STRINGPLUSP(EN, s1, s2, d);

##### FBD/LD図
- ブロック図にはEN（実行条件）、s1、s2（連結する文字列）、d（連結結果）が入力・出力として示されています。
- ブロック名はSTRINGPLUSまたはSTRINGPLUSPが入ります。

##### 実行条件
| 命令 | 実行条件 |
|------|----------|
| $+   | 立ち上がりパルス |
| $+P  | 立ち上がりエッジ |
