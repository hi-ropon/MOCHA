### テキスト
#### 演算処理
- **MAX(_E)**  
  (s1)～(s28)に入力されたBOOL型/INT型/DINT型/WORD型/DWORD型/REAL型/LREAL型/STRING型/TIME型データの最大値を、(d)から(s)と同一のデータ型で出力します。

- **MIN(_E)**  
  (s1)～(s28)に入力された上記同様のデータ型の最小値を、(d)から(s)と同一のデータ型で出力します。

- **入力値の型**  
  (s1)～(s28)への入力値は、上記のデータ型である必要があります。

- **文字列比較条件**  
  - 一致の条件：すべての文字列が一致した場合  
  - 大きい文字列の条件：異なる文字列の場合は文字コードが大きい文字列、文字列長が異なる場合は長い文字列  
  - 小さい文字列の条件：異なる文字列の場合は文字コードが小さい文字列、文字列長が異なる場合は短い文字列

#### 演算結果
1. **EN/ENOなしファンクション**  
   演算処理を実行し、(d)から演算出力値が出力されます。

2. **EN/ENO付きファンクション**  
   実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |  
| :--- | :--- |  
| EN | ENO | (d) |  
| TRUE（演算実行） | TRUE | 演算出力値 |  
| FALSE（演算停止） | FALSE | 不定値 |  

※ ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)から出力されたデータを使用しないようプログラム処理を行う必要があります。
