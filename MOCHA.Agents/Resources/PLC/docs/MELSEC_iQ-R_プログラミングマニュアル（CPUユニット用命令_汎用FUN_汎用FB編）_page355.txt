### テキスト
#### 機能
- 指定したデバイス(d)から(n)点分のデータを右へ1ワードシフトする動作を行います。
- 最上位の1ワード分は0になります。
- (n)で指定した値が0のときは処理を行いません。

#### エラー
- 演算エラーはありません。

### 図
- 図はデータのシフト動作を示しています。
- 上段は元のデータ配列で、(d)から(d)+(n)-1までのデータが並んでいます。
- 下段はシフト後のデータ配列で、元のデータが右に1ワードずつずれ、最上位のワードは0に置き換わっています。
- 矢印で各データの移動先が示されており、(d)の位置は空白（0）になっていることが視覚的に理解できます。
</contents>
