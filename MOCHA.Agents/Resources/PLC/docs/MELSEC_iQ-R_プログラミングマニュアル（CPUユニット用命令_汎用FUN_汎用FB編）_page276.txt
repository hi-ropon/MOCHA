### テキスト
#### BCD8桁除算(DB/(P))
指定した2つのBCD8桁データを除算する命令です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

#### 実行条件
- DB/命令は立ち上がりエッジで実行されます。
- DB/P命令は立ち上がりレベルで実行されます。

### 図
#### ラダー図
- ラダー図では、2つのBCD8桁データ(s1, s2)を入力し、結果(d)を出力する形で表現されています。
- STは対応していません。

#### FBD/LD図
- FBD/LD図では、EN（実行条件）入力、s1, s2のBCD8桁データ入力、dの演算結果出力、ENO（実行結果）出力のブロックで示されています。
- ブロック名はDBDIVISIONまたはDBDIVISIONPが入ります。
