### テキスト
#### ブロック再開 RSTART [BL□]
- RSTART命令は指定したブロックの一時停止を解除し、停止ステップから実行を再開させる命令です。
- 対応CPUはR00CPU、R01CPU、R02CPUで、バージョンによる制約はありません。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン12以降で使用可能。エンジニアリングツールはバージョン1.015R以降対応。
- プロセスCPU（プロセスモード）はファームウェアバージョン03以降で使用可能。ツールはバージョン1.020W以降対応。
- プロセスCPU（二重化モード）はファームウェアバージョン18以降で使用可能。ツールはバージョン1.050C以降対応。

### 図
- ラダー図：命令の入力(d)が接点として表現され、ST欄に「ENO:=RSTART(EN,d);」と記述。
- FBD/LD図：EN入力、ENO出力、d入力の3端子構成の命令ブロックとして示されている。
