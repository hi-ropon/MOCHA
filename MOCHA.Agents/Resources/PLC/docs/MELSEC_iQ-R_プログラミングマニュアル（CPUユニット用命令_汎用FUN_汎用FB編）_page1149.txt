### テキスト
#### 機能
- (s)で指定したロギング設定No.のトリガロギングにトリガを発生させる。
- (s)には1～10が設定可能。
- LOGTRG命令を実行すると、(s)で指定したロギング設定No.の特殊リレー（ログイントリガ）がONし、トリガ後に設定したレコード数分トリガロギングを実行。その後データをラッチしてトリガロギングを停止。
- トリガ条件で「トリガ命令実行時」を選択した場合に有効。
- 以下の場合はLOGTRG命令を実行しても無処理となる：
  - トリガ条件で「トリガ命令実行時」以外を選択したロギング設定No.を指定した場合。
  - 設定を行っていないロギング設定No.を指定した場合。
  - 連続ロギングを実行しているロギング設定No.を指定した場合。
  - LOGTRG命令実行後に再度LOGTRG命令を実行せずにいる場合。

#### 注意事項
- 割込みプログラムでLOGTRG命令を使用するとトリガ条件が発生しない場合がある。
