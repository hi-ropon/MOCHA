### テキスト
#### BIN32ビットデータ比較出力（DCMP(P)(_U)）
RnCPU、RnENCPU、RnPCPUなどのCPUで使用可能な比較命令です。R00CPU、R01CPU、R02CPUはバージョン制約なしで使用可能。RnCPU（R00CPU、R01CPU、R02CPU除く）、RnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降で対応。

(s1)で指定したBIN32ビットデータと(s2)で指定したBIN32ビットデータを比較し、その結果（小、一致、大）に応じて(d)、(d)+1、(d)+20のいずれかがONになる。

### 図
- ラダー図：3つの入力(s1)(s2)(d)を持つ比較命令の簡易表現
- FBD/LD図：EN（実行条件）、s1、s2の入力とd、ENO（実行結果）の出力を持つブロック図
