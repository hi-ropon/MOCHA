### テキスト
#### 使用可能デバイス
- (s1)および(s2)で指定可能なビット、ワード、ダブルワード、間接指定、定数の種類が表形式で示されています。
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、特定の安全デバイスおよび定数のみ使用可能である旨が記載されています。

#### 機能
- (s1)で指定したデバイスのBIN16ビットデータと、(s2)で指定したデバイスのBIN16ビットデータをa接点扱いで比較演算を行うことが説明されています。
- 各命令の比較演算結果は表にまとめられており、条件に応じて導通状態（ENOはON）または非導通状態（ENOはOFF）となることが示されています。
- LDD、AND、OR命令で16進数の定数を指定する場合、最上位ビット(b15)が1になる数値(8～F)は負の数とみなして比較演算を行うことが注意書きされています。
- LDD命令をFBD/LDで使用する場合は、ENに左母線または常時ONの変数部品/定数部品を指定する必要があること。
- ORD命令をFBD/LDで使用する場合は、ENにTRUEを指定するとENOはONになるが、ENは実行条件にはならないことが説明されています。

#### エラー
- 演算エラーは存在しない旨が簡潔に記載されています。
