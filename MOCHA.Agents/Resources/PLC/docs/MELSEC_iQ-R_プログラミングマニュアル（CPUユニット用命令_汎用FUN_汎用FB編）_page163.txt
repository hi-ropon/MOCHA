### テキスト
#### エッジリレー演算結果パルス化
- **EGP**: エッジリレー演算結果パルス化命令（立上り）  
  演算結果の立ち上がり（OFF→ON）時にエッジリレー（V）で記憶し、導通させる。  
- **EGF**: エッジリレー演算結果パルス化命令（立下り）  
  演算結果の立ち下り（ON→OFF）時にエッジリレー（V）で記憶し、導通させる。

#### ラダー図
- EGPは立ち上がりエッジを検出する記号（上向き矢印）付きの接点で表現。  
- EGFは立ち下がりエッジを検出する記号（下向き矢印）付きの接点で表現。

#### ST言語例
```
ENO := EGP(EN, d);
ENO := EGF(EN, d);
```
ENが入力、dがエッジリレー番号、ENOが出力。

### 図
- **ラダー図**: EGP、EGFのエッジ検出を示す接点記号（立上り・立下り）  
- **FBD/LD**: EN入力、ENO出力、d（エッジリレー番号）を持つブロック図  
- **実行条件図**: EGPは立上りエッジ、EGFは立下りエッジの波形イメージ
