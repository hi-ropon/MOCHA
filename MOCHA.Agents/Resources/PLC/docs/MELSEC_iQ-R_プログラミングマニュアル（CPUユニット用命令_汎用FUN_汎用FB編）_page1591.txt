### 表
このページの表は、PLCの命令コード「MOV」とそれに対応する定数値およびデバイスアドレスの組み合わせを一覧で示しています。具体的には、以下のような内容です。

- 命令コード: MOV（データ転送命令）
- 定数値: K1、K100、K30000、K0、K2000、K800 など複数の定数が設定されている
- デバイスアドレス: D516からD529までの連続したデバイス番号

表の構成は、左端に条件（SM402）があり、その右に複数の空白セルを挟んで、命令コード、定数値、デバイスアドレスが横並びで配置されています。

### テキスト
ページ下部に「ループ2に対するPID制御用データを設定します。」という説明文があります。これは表に示されたMOV命令群が、PID制御のためのパラメータ設定を行っていることを示しています。

### 図・グラフ
このページには図やグラフは含まれていません。
