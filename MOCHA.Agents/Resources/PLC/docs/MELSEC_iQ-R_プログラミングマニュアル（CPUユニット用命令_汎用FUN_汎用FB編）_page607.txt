### テキスト
#### ローテーション命令（ROR(P), RCR(P)）
BIN16ビットデータの右ローテーションを行う命令です。
- ROR(P)：BIN16ビットデータをキャリフラグを含めずに(n)ビット右へ回転させます。
- RCR(P)：BIN16ビットデータをキャリフラグを含めて(n)ビット右へ回転させます。

※ST、FBD/LDではROR命令は対応しておらず、汎用ファンクションのRORを使用する必要があります。

#### 実行条件
- ROR、RCR命令は立ち下がりエッジで実行。
- RORP、RCRP命令は立ち上がりエッジで実行。

### 図
- ラダー図：左側に(d)と(n)の入力があり、右側に命令が記述されたラダー回路。
- FBD/LD図：EN、ENO、n、dの入出力端子を持つ命令ブロックの図。
