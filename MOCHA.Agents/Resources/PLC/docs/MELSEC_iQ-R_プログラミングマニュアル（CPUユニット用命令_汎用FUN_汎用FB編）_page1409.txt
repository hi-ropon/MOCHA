### テキスト
#### ラベルのビット指定を使ったプログラム作成の簡略化
ラベルのビット指定を使用すると、GET_BIT_OF_INTと同じ動作のプログラムを簡潔に作成できます。  
例として、GET_BIT_OF_INTを使った場合と同様に、変数g_int1のビット5（b5）の値をg_bool1に読み出す方法を示しています。

### 図（プログラム例）
- **ラダーの場合**  
  g_bool1 := g_int1.5;  
  これはg_int1の5ビット目の値をg_bool1に代入する記述例です。

- **ST（構造化テキスト）場合**  
  g_bool1 := g_int1.5;  
  ST言語でも同様にビット指定で値を読み出しています。

- **FBD/LD（ファンクションブロック図/ラダー図）場合**  
  図にはg_int1.5からg_bool1へ接続された線が描かれており、ビット指定の値を読み出す動作を視覚的に示しています。
