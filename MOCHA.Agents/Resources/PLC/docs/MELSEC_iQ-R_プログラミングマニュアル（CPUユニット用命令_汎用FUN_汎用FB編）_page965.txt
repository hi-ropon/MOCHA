### テキスト
#### 単精度実数べき乗演算（POW）
単精度実数のべき乗を演算する命令について説明しています。

#### 実行条件
- POW命令は立ち上がりエッジで実行されます。
- POWP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  べき乗演算の入力として、(s1)、(s2)の2つのオペランドと結果格納先(d)を示すシンボルが描かれています。

- **FBD/LD図**  
  ブロック図で、EN（入力許可）、s1、s2（入力データ）、ENO（出力許可）、d（出力データ）の接続が示されています。

- **ST言語**  
  ENO:=POW(EN,s1,s2,d);  
  ENO:=POWP(EN,s1,s2,d);  
  のように記述し、べき乗演算を実行する例が示されています。
