### テキスト
#### 単精度実数SIN⁻¹演算（ASIN(P)）
単精度実数で指定したSIN値から角度を演算する命令です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

ASIN命令はST、FBD/LDでは対応しておらず、汎用ファンクションのASINを使用する必要があります（1425ページ参照）。

#### 実行条件
- ASIN命令は立ち上がりエッジで実行されます。
- ASINP命令は立ち上がりレベルで実行されます。

### 図
- ラダー図：ASINP命令のラダー表現。入力(s)、出力(d)があり、ST命令形式は「ENO:=ASINP(EN,s,d);」と記述。
- FBD/LD図：EN（入力）、ENO（出力）、s（SIN値入力）、d（演算結果出力）の4端子を持つブロック図。
