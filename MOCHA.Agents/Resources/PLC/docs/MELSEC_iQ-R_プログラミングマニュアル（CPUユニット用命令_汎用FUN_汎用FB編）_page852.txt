### テキスト
#### 機能
(s1)で指定した倍精度実数データと、(s2)で指定した倍精度実数データを比較し、その結果（小、 一致、 大）に応じて(d)、(d)+1、(d)+2のいずれかがONします。

#### ラダー図（回路図）
- X0が入力されると、EDCMP命令が実行されます。
- EDCMP命令は(s1)と(s2)の値を比較し、以下の条件で出力をONにします。
  - (d)：(s1)～(s1)+3 > (s2)～(s2)+3 のときON
  - (d)+1：(s1)～(s1)+3 = (s2)～(s2)+3 のときON
  - (d)+2：(s1)～(s1)+3 < (s2)～(s2)+3 のときON
