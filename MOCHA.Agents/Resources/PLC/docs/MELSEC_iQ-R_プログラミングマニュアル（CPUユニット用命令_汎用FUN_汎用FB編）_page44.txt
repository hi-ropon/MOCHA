### テキスト
#### 実数データ（浮動小数点データ）
実数データには、32ビットで表現する単精度実数データと、64ビットで表現する倍精度実数データがあります。  
実数データはビットデバイス以外のデバイス、または単精度実数型ラベル、倍精度実数型ラベルにのみ格納可能です。

#### 単精度実数データの構成
単精度実数データは符号部、仮数部、指数部により構成されます。  
内部表現のビット構成は以下の通りです。  
- 符号部（1ビット）: 数値の正負を示し、0は正数または0、1は負数を示します。  
- 仮数部（23ビット）: 単精度実数を2進数で1.XXXXX...×2^Nと表現したときのXXXXX...の部分です。  
- 指数部（8ビット）: 単精度実数を2進数で1.XXXXX...×2^Nと表現したときのNを示します。

### 表
#### データサイズと値の範囲

| データ名称                     | データサイズ           | 値の範囲                         |
|------------------------------|-----------------------|---------------------------------|
| 単精度実数データ（単精度浮動小数点データ） | 正数: 32ビット（2ワード） | 2^-126 < 実数 < 2^128           |
|                              | ゼロ                  | 0                               |
|                              | 負数                  | -2^128 < 実数 < -2^-126         |
| 倍精度実数データ（倍精度浮動小数点データ） | 正数: 64ビット（4ワード） | 2^-1022 < 実数 < 2^1024         |
|                              | ゼロ                  | 0                               |
|                              | 負数                  | -2^1024 < 実数 < -2^-1022       |

#### 指数部（8ビット）と指数Nの関係

| 指数部(b24～b30) | FFH | FEH | FDH | ... | 81H | 80H | 7FH | 7EH | ... | 02H | 01H | 00H |
|------------------|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| N                | 未使用 | 127 | 126 | ... | 2   | 1   | 0   | -1  | ... | -125| -126| 未使用 |

### 図
単精度実数データのビット構成図は以下のように示されています。  
- 符号部（1ビット）: b31  
- 指数部（8ビット）: b30～b23  
- 仮数部（23ビット）: b22～b0  

この構成により、単精度実数データは符号、指数、仮数の3つの部分で表現されることが理解できます。
