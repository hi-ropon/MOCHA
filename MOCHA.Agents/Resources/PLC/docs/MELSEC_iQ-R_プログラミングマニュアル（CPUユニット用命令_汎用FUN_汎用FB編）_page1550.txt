### 表
#### 命令処理時間一覧表
命令名ごとに、条件別の処理時間（マイクロ秒単位）がCPUユニット別に示されています。CPUユニットはR00CPU、R01CPU、R02CPU、及びそれ以外のCPUユニットに分かれています。各CPUユニットごとに最小値と最大値が記載されています。

- **命令名**: ORTM<、ORTM>=、TCMP、TZCP、S.DATERD、S.DATE+、S.DATE-、DUTY、TIMCHK、HOURM、DHOURM、RFS(X)、RFS(Y)、COMなど多岐にわたる命令が列挙されています。
- **条件**: 無実行時、指定の時刻と比較（導通時・非導通時）、桁上げなし・あり、桁下げなし・あり、I/Oリフレッシュの選択時、CC-Link IEコントローラネットワークリフレッシュの選択時（管理局側、マスタ局側、ローカル局側、通常局側）、マルチCPUシステムのCPUバッファメモリ使用時など、詳細な条件が設定されています。
- **処理時間（μs）**: 各条件に対して、R00CPU、R01CPU、R02CPU、左記以外のCPUユニットの最小値・最大値が数値で示されています。例えば、ORTM<命令の無実行時はR00CPUで0.18816μs、R02CPUで0.02352μs、左記以外のCPUユニットで0.00588μsとなっています。
- **特記事項**: RFS(X)、RFS(Y)命令はn=1およびn=64の条件で処理時間が異なり、COM命令はネットワークの種類や役割によって処理時間が大きく変動しています。

この表は、各命令の処理性能をCPUユニット別に比較検討する際に利用される技術資料です。
</contents>
