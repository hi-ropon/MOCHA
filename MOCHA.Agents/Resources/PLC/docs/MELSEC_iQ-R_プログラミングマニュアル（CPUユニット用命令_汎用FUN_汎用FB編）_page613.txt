### テキスト
#### BIN32ビットデータの右ローテーション
- DROR(P)、DRCR(P)命令は、BIN32ビットデータを右方向に回転させる命令です。
- DROR(P): キャリフラグを含めずに(n)ビット右回転。
- DRCR(P): キャリフラグを含めて(n)ビット右回転。

#### 命令の使用例（ラダー、FBD/LD）
- ラダー図では、EN（イネーブル）入力に対してDRORP、DRCRP、DRCRP命令を実行する例が示されています。
- FBD/LD図はEN、n、dの入力とENOの出力を持つブロック図で、ST言語やFBD/LDでの使用例を示しています。
- 注記として、DROR命令はST、FBD/LDでは対応しておらず、汎用ファンクションのRORを使用するよう案内があります。
