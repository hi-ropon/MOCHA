### テキスト
#### エラーコード2820Hの説明
- ブロック指定をする場合に、指定ブロックNo.が指定可能なブロックNo.の範囲外のときに発生するエラー。
- ブロック指定をする場合に、指定ステップNo.が指定可能なステップNo.の範囲外のときに発生するエラー。

#### Point
ステップは桁指定で指定します。
- SFCプログラムで自ブロック内のステップを指定する場合は、K8S口で指定します。
- SFCプログラムで他ブロックのステップを指定する場合は、BL□￥K8S口で指定します。
- シーケンスプログラムでステップを指定する場合は、BL□￥K8S口で指定します。
