### テキスト
#### 機能
- ファイル名で指定されたデバイスに格納されているファイル名のプログラムをスキャン実行タイプにします。
- プログラムメモリに格納されているプログラムのみスキャン実行タイプにできます。
- 指定されたプログラムは、END処理でスキャン実行タイプになります。

#### 例
プログラムA、B、Cが存在し、プログラムA内でプログラムDに対してPSCANN(P)命令を実行した場合の動作を示しています。

- スキャンはプログラムAのA、B、C、ENDまで行われ、その後プログラムDが実行されます。
- パラメータで実行形式を指定していてもPSCANN(P)命令が優先されます。
- ファイル名に拡張子(.PRG)を指定する必要はなく、.PRGファイルのみ対象となります。

### 図
- プログラムのスキャン実行の流れを示すタイムチャート
  - プログラムAのセクションA、B、C、ENDがスキャンされる（Sc）
  - その後、プログラムDが実行される（(1)の箇所）

| 2840H | (ファイル名)で指定したファイル名のプログラムが存在しないとき。 |

| 3424H | (ファイル名)で指定したファイル名がSFCプログラムで、すでに別ファイル名のSFCプログラムが起動しているとき。 |
