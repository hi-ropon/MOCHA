### テキスト
#### ブロック活性チェック
対象命令はLD、LDI、AND、ANI、OR、ORIで、これらは指定ブロックの活性/非活性と、それまでの演算結果のANDまたはOR演算を演算結果とします。

- RnCPU（R00CPU、R01CPU、R02CPU）はバージョンによる制約なし。
- RnENCPUはファームウェアバージョン12以降で使用可能。エンジニアリングツールはバージョン1.015R以降対応。
- プロセスCPU（プロセスモード）はファームウェアバージョン03以降で使用可能。ツールはバージョン1.020W以降対応。
- プロセスCPU（二重化モード）はファームウェアバージョン18以降で使用可能。ツールはバージョン1.050C以降対応。

命令の意味：
- LD: a接点命令 / LDI: b接点命令
- AND: a接点直列接続命令 / ANI: b接点直列接続命令
- OR: a接点1個の並列接続命令 / ORI: b接点1個の並列接続命令

#### 実行条件
命令LD、LDI、AND、ANI、OR、ORIは常時実行されます。
