### テキスト
#### BIN32ビット加算
「D+(P)(_U)」命令は、指定した2つのBIN32ビットデータを加算する命令です。  
対応するCPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）です。

- ラダー命令は対応していますが、ST命令は対応していません（3オペランドの場合は別ページ参照）。  
- FBD/LD命令は対応していません（3オペランドの場合は別ページ参照）。

#### 実行条件
| 命令       | 実行条件の波形例       |
|------------|------------------------|
| D+ / D+_U  | 立ち上がりパルス       |
| D+P / D+P_U| 立ち下がりパルス       |

#### 注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。
