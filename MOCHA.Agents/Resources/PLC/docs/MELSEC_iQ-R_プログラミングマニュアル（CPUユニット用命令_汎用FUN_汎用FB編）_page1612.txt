### 表
#### GX Works2命令と互換用命令の対応表
- 表は5つの列で構成されている。
  1. **GX Works2命令名**: GX Works2で使用される命令の名称。
  2. **互換用命令名**: 互換性のある別の命令名。
  3. **使用プログラム**: ST（Structured Text）およびFBD（Function Block Diagram）での使用可否を○（使用可）または×（使用不可）で示す。
  4. **参照先**: 詳細な説明が記載されているマニュアルやページ番号。

- 命令例:
  - `OGLOAD_U_M` は互換命令 `G_OGLOAD_M2` で、ST・FBD両方で使用可能。参照先は「MELSEC iQ-R プログラミングマニュアル（ユニット専用命令編）」。
  - 論理演算命令（例：`OR<`、`OR<=`など）はSTでは使用不可（×）、FBDでは使用可能（○）。参照先にページ番号と関連命令名が記載されている。
  - 算術命令（例：`PLUS_M`、`PLUS_3_M`など）はST・FBD両方で使用可能で、参照先にページ番号とオペランド数の説明がある。

- 参照先のページ番号は214ページ、217ページ、232ページ、234ページなどがあり、関連する詳細説明や使用例が記載されていることが示されている。

- 表の下部にページ番号「1612」と付録名「付6 他形式プロジェクトからの置換え」が記載されている。

以上より、本ページはGX Works2の命令とその互換命令の対応関係、使用可能なプログラム形式、及び詳細参照先を一覧で示した技術資料の一部である。
