### テキスト
#### ユニット形名の格納方法
- 読出し対象ユニットが形名を保持している場合、(d)+1以降に格納するユニット形名は9ワード使用し、ASCII文字で格納する。
- 18文字目は00Hを格納し、文字数が17文字に満たない場合は残りの文字にも00Hを格納する。
- ユニットが保持している形名を格納する（定格銘板に記載されている形名とは異なる場合があるため注意）。

##### 例（格納する形名の例）
| 対象ユニット       | 格納する形名の例  |
|--------------------|------------------|
| CPUユニット        | R04CPU           |
| 入出力ユニット     | INPUT_16         |
| ネットワークユニット | RJ71GP21-SX      |

**Point**  
I/O割付設定で設定されている形名と実装されているユニットの形名が異なる場合、実装されているユニットが保持している形名を格納する。

#### ユニット形名を保持していない場合の文字列格納方法
- 9ワード使用し、ASCII文字で格納。
- 18文字目は00Hを格納。
- 文字数が17文字に満たない場合は残りの文字にも00Hを格納。
- 「ユニット種別を示す文字列」と「点数を示す文字列」を組み合わせた文字列を格納。

##### 例（格納する文字列の例）
| 対象ユニット       | 格納する文字列の例 |
|--------------------|-------------------|
| 入力ユニット       | INPUT_16          |
| 出力ユニット       | OUTPUT_32         |
| 入出力混合ユニット | MIXED_64          |
| インテリジェント機能ユニット | INTELLIGENT_128 |

##### 点数を示す文字列の例
| 点数   | 点数を示す文字列の例 |
|--------|---------------------|
| 16点   | _16                 |
| 32点   | _32                 |
| 48点   | _48                 |
| 64点   | _64                 |
| 128点  | _128                |
| 256点  | _256                |
| 512点  | _512                |
| 1024点 | _1024               |

**Point**  
I/O割付設定で設定されている点数と実装されているユニットの点数が異なる場合、実装されているユニットの点数を格納する。

#### CPUユニットがRnENCPUの場合の格納文字列
- 入出力番号を指定してユニット形名を読み出す場合の格納文字列例。

##### 例（格納する文字列の例）
| 対象ユニットの入出力番号 | 格納する文字列の例    |
|--------------------------|----------------------|
| 3E00H                    | R120ENCPU            |
| 0000H                    | _RJ71EN71(E+IEF)*1   |

*1 RnENCPU（ネットワーク部）の格納する文字列は、エンジニアリングツールのユニット構成図の部品選択ウィンドウで選択したユニットになる。
