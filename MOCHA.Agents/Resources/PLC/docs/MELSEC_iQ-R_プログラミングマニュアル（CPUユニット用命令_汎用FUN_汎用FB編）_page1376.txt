### テキスト
#### LREAL型→INT型変換（LREAL_TO_INT(_E)）
LREAL型データをINT型データへ変換する関数です。対応CPUはRnCPU、RnECPU、RnPCPU（プロセス、三重化、一般）、RnSPFCPU（安全、一般）ですが、一部のCPUでは使用不可となっています。

#### 機能
- 演算処理として、入力されたLREAL型データをINT型データに変換し、出力します。
- 入力値はLREAL型のデータ値であり、変換後のデータは小数点以下第1桁を四捨五入した整数値となります。

#### 演算結果
1. EN/ENOなしファンクション
   - 演算処理を実行し、出力(d)から演算結果が得られます。
2. EN/ENO付きファンクション
   - 実行条件と演算結果は以下の通りです。

| 実行条件 EN | ENO | (d) 演算結果 |
|-------------|-----|--------------|
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、そのデータを使用しないようプログラム処理を行う必要があります。

### 図
- ラダー、FBD/LD、ST言語での関数使用例が示されています。
  - ラダー/FBD/LD：[EN/ENOなし]と[EN/ENO付き]の2種類のブロック図があり、入力(s)と出力(d)の接続が示されています。
  - ST言語：[EN/ENOなし]は d:=LREAL_TO_INT(s);、[EN/ENO付き]は d:=LREAL_TO_INT_E(EN,ENO,s); の記述例が示されています。

### 表
#### 設定データ（引数の内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
|------|------|------|----------|
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | LREAL |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | INT |
