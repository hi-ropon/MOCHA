### 表
#### 安全システム用命令で発生するエラーコード一覧

| エラーコード(d1) | エラー内容                         | 処置方法                                         |
|------------------|----------------------------------|------------------------------------------------|
| 8000H            | 指定したファイルが存在しない。   | 指定したファイルが存在するかを確認してください。 |
| 8001H            | 指定したファイルの拡張子が不正である。 | 指定したファイルの拡張子を確認してください。     |
| 8002H            | ファイルアクセスする他機能を実行中である。 | ファイルアクセスする他機能を完了後、再度命令を実行してください。 |
| 8004H            | 指定したファイルが壊れている。   | CPUユニットに指定したファイルを再度書き込んでください。 |

以上のように、各エラーコードに対応したエラー内容とその対処方法が示されています。
