### テキスト
#### 実行フェーズの終了（PHASEEND命令）
実行中のフェーズを終了する命令「PHASEEND」について説明しています。

- 対応CPUは、ファームウェアバージョン24以降のR00CPU、R01CPU、R02CPU（ただしこれらは除く）、およびバージョン57以降のRnENCPUです。エンジニアリングツールはバージョン1.075D以降で使用可能です。
- 実行中のフェーズを終了する命令です。

#### ラダー回路（Ladder）
- 対応していません。

#### FBD/LD
- 対応していません。

#### 実行条件
| 命令       | 実行条件           |
|------------|--------------------|
| PHASEEND   | 立ち上がりエッジ（パルス） |

#### 機能
- 実行中のフェーズを終了し、直前のPHASE命令で指定したデバイスをOFFにします。
- PHASE命令に接続したフェーズ処理の非実行処理は、本命令実行後の次のスキャン時に行われます。

#### 注意事項
- PHASEEND命令はフェーズ処理内で実行してください。フェーズ処理外で実行した場合、最後にON実行したPHASE命令の指定デバイスをOFFにします。
- PHASE命令をON実行してからPHASEEND命令を実行するまでの間に、RUN中の回路ブロック変更やシーケンサへの書き込み（シーケンスプログラム、FBファイル、グローバルラベル設定ファイルのいずれか）が行われた場合は例外となります。

#### エラー
- 演算エラーはありません。
