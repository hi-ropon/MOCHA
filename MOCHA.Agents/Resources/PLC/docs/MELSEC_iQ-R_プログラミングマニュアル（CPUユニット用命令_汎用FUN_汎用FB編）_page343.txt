### テキスト
#### 機能
(s1)で指定されたダブルワードデバイスの中から、(s2)で指定した位置のビットデータを取り出し、(d)で指定したビットデバイスに書き込みます。

- (d)で指定したビットデバイスは、該当ビットが「0」のときOFF、「1」のときONになります。
- (s2)では、2ワードデータの各ビット位置(0〜31)を指定します。もし(s2)に32以上の値を指定した場合は、(s2)を32で割った余りの値がビット位置になります。

##### 例
(s2)=34のときは、34÷32=1余り2のため、b2のデータになります。

#### エラー
演算エラーはありません。

### 図
ビット位置の指定と書き込みのイメージ図が示されています。

- ダブルワードデバイスはb31からb0までのビットで構成されており、(s1)で指定した位置と(s1)+1の位置に分かれています。
- (s2)ビット位置が21の場合の例が示されており、(d)で指定したビットデバイスに対応するビットが書き込まれる流れを表しています。
