### テキスト
#### プログラムファイル間サブルーチンコール ECALL(P)
指定プログラムファイル名のポインタのサブルーチンプログラムを実行します。  
プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります。

#### 実行条件
- ECALL命令は立下りエッジで実行
- ECALLP命令は立上りエッジで実行

### 図
- ラダー表現例  
  - (FILE)(P)  
  - (FILE)(P)(s1)...(s5)  
- FBD/LD表現例  
  - EN, ENO, FILE, Pの入力端子を持つ命令ブロック
