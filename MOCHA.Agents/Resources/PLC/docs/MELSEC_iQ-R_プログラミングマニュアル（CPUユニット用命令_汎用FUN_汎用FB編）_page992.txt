### テキスト
#### 機能
プログラム中で使用するファイルレジスタのブロックNo.を(s)で指定されたデバイスに格納されているブロックNo.に変更します。ブロックNo.変更後は、シーケンスプログラムで使用しているすべてのファイルレジスタは変更後のブロックNo.のファイルレジスタを対象に処理されます。

#### 注意事項
ファイルレジスタの制約事項については、64ページの「ファイルレジスタ使用時の制約事項」を参照してください。

### 図
「ファイルレジスタに対する処理」のフローチャート図が示されています。
- (s)で指定されたブロックNo.設定があり、
- 現在使用しているファイルレジスタ（ブロックNo.0）から、
- RSET命令後に使用されるファイルレジスタ（ブロックNo.1）、
- さらにブロックNo.nのファイルレジスタへと処理が移る流れを表しています。
各ブロックはR0からR32767までの範囲のレジスタを示しています。

### 表
#### エラー
| エラーコード(SD0) | 内容                             |
|-------------------|----------------------------------|
| 2820H             | ファイルレジスタが存在しないとき。 |
| 3405H             | (s)で指定したブロックNo.が存在しないとき。 |
