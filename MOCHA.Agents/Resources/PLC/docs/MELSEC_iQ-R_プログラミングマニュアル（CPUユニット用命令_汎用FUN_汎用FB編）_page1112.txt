### テキスト
#### 機能
- 1スキャン途中に該当するデバイスのみをリフレッシュし、外部入力の取り込みや出力ユニットへの出力を行う機能。
- 入力の取り込みや外部への出力はプログラムのEND命令実行後に一括で行われるため、1スキャン中に外部にパルス信号を出力することはできない。
- I/Oリフレッシュ命令を実行すると、プログラム実行途中に該当する入力(X)または出力(Y)を強制的にリフレッシュし、1スキャン中にパルス信号を外部に出力可能。
- 入力(X)または出力(Y)を1点単位でリフレッシュする場合は、ダイレクトアクセス入力(DX)、ダイレクトアクセス出力(DY)を使用。
- プロセスCPU（二重化モード）では、待機系から増設ベースユニット上のユニットに割り付けられている入力(X)、出力(Y)を指定した場合は無処理となる。

#### 例
- RFS命令によるプログラムをダイレクトアクセスに置き換えた例を示す。
- ラダー図でX0のリフレッシュとY20のリフレッシュを行う命令が示されている。
- それぞれのリフレッシュをダイレクトアクセス入力DX0、ダイレクトアクセス出力DY20に置き換えた例。
