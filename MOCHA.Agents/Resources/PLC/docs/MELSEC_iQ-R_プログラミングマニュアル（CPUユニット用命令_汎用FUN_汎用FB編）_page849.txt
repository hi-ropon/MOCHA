### テキスト
#### 単精度実数比較出力 ECMP(P)
- 対応CPUはR00CPU、R01CPU、R02CPUで、バージョンによる制約はありません。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降で使用可能。
- (s1)で指定した単精度実数データと、(s2)で指定した単精度実数データを比較し、その結果（小、一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONになる。

### 図
#### ラダー図
- 入力：(s1)、(s2)、(d)
- 出力：ST命令でENO:=ECMP(EN,s1,s2,d)、ENO:=ECMPP(EN,s1,s2,d)と記述。

#### FBD/LD図
- ブロック図でEN、s1、s2を入力、ENO、dを出力する構成。
