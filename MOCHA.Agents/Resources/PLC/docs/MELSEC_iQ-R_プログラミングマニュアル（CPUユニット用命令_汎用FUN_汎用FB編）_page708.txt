### テキスト
#### BIN32ビット単位スケーリング(X/Y別座標データ)
スケーリング用変換データ（32ビットデータ単位）に対し、指定した入力値でスケーリング（X/Y別座標データ）を行う命令について説明しています。

#### 実行条件
命令ごとに実行条件の波形が示されており、DSCL2/DSCL2_Uは立ち下がりエッジ、DSCL2P/DSCL2P_Uは立ち上がりエッジで実行されることが示されています。

#### 設定データ
##### 内容、範囲、データ型
- (s1) : スケーリングを行う入力値または入力値が格納されている先頭デバイス  
  範囲は符号付きBIN32ビットで-2147483648～2147483647、符号なしBIN32ビットで0～4294967295  
- (s2) : スケーリング用変換データが格納されている先頭デバイス  
- (d) : スケーリングにより制御した出力値を格納する先頭デバイス  
- EN : 実行条件（ビット型）  
- ENO : 実行結果（ビット型）  

注記として、(s2)～(s2)+1の座標ポイント数は符号なしBIN32ビットとなること、ラベルで設定する場合は必要な領域を確保した配列型ラベルを指定する必要があることが記載されています。

### 図
- ラダー図：入力(s1)、(s2)、出力(d)の接続イメージとST言語での命令例が示されています。  
- FBD/LD図：EN（実行条件）、s1、s2（入力）、d（出力）、ENO（実行結果）の接続を示すブロック図が描かれています。
