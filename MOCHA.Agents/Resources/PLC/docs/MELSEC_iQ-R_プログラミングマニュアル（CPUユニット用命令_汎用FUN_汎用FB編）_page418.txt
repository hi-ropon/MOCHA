### テキスト
#### 単精度実数→符号付きBIN16ビットデータ変換命令「FLT2INT(P)」
指定した単精度実数を符号付きBIN16ビットデータに変換します。

対応ツールバージョンは1.035M以降のエンジニアリングツールです。

#### ■実行条件
- FLT2INT命令は立ち上がりエッジで実行されます。
- FLT2INTP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  (s)単精度実数入力、(d)符号付きBIN16ビット出力の命令ブロックを示しています。  
  ST命令例：ENO:=FLT2INT(EN,s,d); ENO:=FLT2INTP(EN,s,d);

- **FBD/LD図**  
  EN（実行条件）、s（単精度実数入力）、d（出力）端子を持つ命令ブロックの図。
