### テキスト
#### サブルーチンプログラムコール（XCALL）
- XCALL命令はサブルーチンプログラムの実行と非実行処理を行う命令です。
- 条件が成立した場合はサブルーチンCALLし、条件がONからOFFに変わるとサブルーチンをFCALLします。
- プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります。

### 図
- ラダー図：XCALL命令のラダー回路図が2種類示されており、(P)というパラメータを持つ形で表現されている。
- FBD/LD図：EN（入力）、ENO（出力）、P（パラメータ）を持つ命令ブロックの形で示されている。
