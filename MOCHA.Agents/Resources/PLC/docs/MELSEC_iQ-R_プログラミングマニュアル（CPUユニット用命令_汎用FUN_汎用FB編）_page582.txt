### テキスト
#### サブルーチンプログラムからのリターン（RET命令）
- RET命令はサブルーチンプログラムの終了を示します。
- RET命令を実行すると、サブルーチンプログラムを呼び出したCALL(P)命令、FCALL(P)命令、ECALL(P)命令、EFCALL(P)命令、XCALL命令の次のステップに戻ります。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| RET | 常時実行 |

### 図
- ラダー図：RET命令はラダー図では対応していません。
- FBD/LD図：対応していません。
- フローチャート図：  
  - A（メインルーチンプログラム）からCALL(P)命令でB（サブルーチンプログラム）を呼び出し、BのRET命令でAのCALL(P)命令の次のステップに戻る流れを示しています。

### 表
#### エラーコード(SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 3381H | CALL(P)命令、FCALL(P)命令、ECALL(P)命令、EFCALL(P)命令、XCALL命令実行後、RET命令を実行する前にEND命令、FEND命令、GOEND命令、STOP命令を実行したとき。 |
| 3382H | CALL(P)命令、FCALL(P)命令、ECALL(P)命令、EFCALL(P)命令、XCALL命令を実行する前にRET命令を実行したとき。 |
