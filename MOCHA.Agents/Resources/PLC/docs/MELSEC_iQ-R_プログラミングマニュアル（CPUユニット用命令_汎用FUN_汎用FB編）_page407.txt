### テキスト
#### 機能
- (d)で指定したデバイスから倍精度実数(n1)点分のデータ範囲内で、左へ(n2)点分シフトする。
- 最下位から(n2)点分は0になる。
- (n1)または(n2)で指定した値が0のときは無処理となる。
- (n1)≦(n2)の場合は、(d)で指定したデバイスから(n1)点分のデータはすべて0になる。

#### エラー
- 演算エラーはありません。

### 図
#### 例：(n1)=4、(n2)=2の場合
- 上段の図は、(d)から(d)+15までのデータのうち、4点分のデータ範囲(n1)と2点分のシフト量(n2)を示している。
- 元のデータは、(d)+15から(d)までの各デバイスに実数値が格納されている。
- 下段の図はシフト後のデータを示し、左へ2点分シフトされているため、(d)+15から(d)+8のデータが左に2点分ずれている。
- シフトされた右端の2点分((d)+3、(d))は0に置き換わっている。

この図により、データシフト命令の動作が視覚的に理解できる。
