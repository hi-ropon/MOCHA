### テキスト
本ページはCPUユニット用命令の応用命令の一部を説明しています。主に数値演算や乱数生成に関する命令が対象です。

#### 単精度実数常用対数演算
単精度実数で指定した値の常用対数（底10の対数）を演算する命令です。

#### 倍精度実数常用対数演算
倍精度実数で指定した値の常用対数（底10の対数）を演算する命令です。

#### 単精度実数最大値検索
指定した範囲の単精度実数のブロックデータから最大値を検索し、指定した検索結果格納先に最大値を格納します。

#### 倍精度実数最大値検索
指定した範囲の倍精度実数のブロックデータから最大値を検索し、指定した検索結果格納先に最大値を格納します。

#### 単精度実数最小値検索
指定した範囲の単精度実数のブロックデータから最小値を検索し、指定した検索結果格納先に最小値を格納します。

#### 倍精度実数最小値検索
指定した範囲の倍精度実数のブロックデータから最小値を検索し、指定した検索結果格納先に最小値を格納します。

#### 乱数命令
乱数の発生および乱数系列の変更に関する命令です。

##### 乱数発生、系列変更
- RND, RNDP: 0～32767未満の乱数を発生し、指定したデバイスに格納します。
- SRND, SRNDP: 指定されたデバイスに格納されている16ビットBINデータの内容に従い、乱数系列を変更します。
