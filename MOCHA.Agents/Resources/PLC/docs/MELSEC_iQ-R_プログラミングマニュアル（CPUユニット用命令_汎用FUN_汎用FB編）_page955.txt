### テキスト
#### 倍精度実数指数演算(EXPD(P))
倍精度実数で指定した値の指数を演算します。

- 対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）
- ラダー命令はSTで「ENO:=EXPDP(EN,s,d);」の形式で使用
- FBD/LDでは対応しておらず、汎用ファンクションのEXPを使用する必要あり（1421ページ参照）

##### ■実行条件
- EXPD命令は立下りエッジで実行
- EXPDP命令は立上りエッジで実行

### 図
- ラダー回路図：入力(s)、出力(d)の接続イメージ
- FBD/LD回路図：EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）の接続イメージ
