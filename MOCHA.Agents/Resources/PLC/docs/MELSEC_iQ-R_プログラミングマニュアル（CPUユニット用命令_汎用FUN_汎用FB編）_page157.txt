### テキスト
#### ORPI、ORFI命令について
- ORPI命令は立上りパルス否定並列接続命令、ORFI命令は立下りパルス否定並列接続命令であり、これらはそれまでの演算結果とOR演算を行い、演算結果とします。
- ORPI命令、ORFI命令で使用するON/OFF情報は表に示されています。
- ORPI命令をSTおよびFBD/LDで使用した場合、ENと指定ビットデバイス(s)立上りのOR演算結果がON以外の時、ENOはONします。ENは実行条件にはなりません。
- ORFI命令をSTおよびFBD/LDで使用した場合、ENと指定ビットデバイス(s)立下りのOR演算結果がON以外の時、ENOはONします。ENは実行条件にはなりません。

#### LDPI、LDFI、ANDPI、ANDFI、ORPI、ORFIの組み合わせによる動作
- これらの命令の組み合わせによる動作例は、LD、AND、ORの組み合わせによる動作例と同様であることが記載されています。
- 詳細は150ページのLD、LDI、AND、ANI、OR、ORIの説明を参照するよう案内されています。

#### エラー
- 演算エラーはありません。
