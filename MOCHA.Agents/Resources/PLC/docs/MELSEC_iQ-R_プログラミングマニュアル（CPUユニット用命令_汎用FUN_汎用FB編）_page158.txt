### テキスト
#### 5.2 結合命令
回路ブロックの直列接続および並列接続を行う命令について説明しています。

- **ANB**: 回路ブロックの直列接続命令で、AブロックとBブロックのAND演算を行います。
- **ORB**: 回路ブロックの並列接続命令で、AブロックとBブロックのOR演算を行います。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| ANB | 常時実行 |
| ORB | 常時実行 |

#### 機能
- **ANB**
  - AブロックとBブロックのAND演算を行い、その演算結果を出力します。
  - ANB命令のシンボルは接点シンボルではなく、接続シンボルです。

- **ORB**
  - AブロックとBブロックのOR演算を行い、その演算結果を出力します。
  - ORB命令は2接点以上の回路ブロックを並列接続します。1接点のみの並列接続はOR命令、ORI命令を使用し、ORB命令は不要です。
  - ORB命令のシンボルも接点シンボルではなく、接続シンボルです。

#### エラー
- 演算エラーはありません。

### 図
- **ラダー回路図**
  - ANB: AブロックとBブロックが直列に接続されている回路図。
  - ORB: AブロックとBブロックが並列に接続されている回路図。
  - 図中のAはAブロック、BはBブロックを示しています。

### 表
- 実行条件をまとめた表があり、ANBとORBは共に「常時実行」と記載されています。

### その他
- ST言語は対応していません。
- FBD/LD言語も対応していません。
