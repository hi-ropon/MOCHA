### テキスト
#### 機能
- (s)で指定された倍精度実数を符号なしBIN32ビットデータに変換し、(d)で指定されたデバイスに格納する。
- 変換後のデータは、(s)で指定された倍精度実数の小数点以下1桁目を四捨五入した値となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### 注意事項
- 47ページに注意事項が記載されている。
- プログラム例では、M0がONしたときにD0～D3に格納された倍精度実数を符号なしBIN32ビットデータに変換し、D100～D101に格納する。

### 図（プログラム例）
- SM402の接点M0がONのとき、EDMOVP命令でD0の倍精度実数（765432.1）をD3、D2、D1、D0に分割して格納。
- DBL2UDINT命令でD0の倍精度実数を符号なしBIN32ビットデータに変換し、D100に格納。
- 各データのビット構成や16進数表記（例：4127H、5BF0H、3333Hなど）が示されている。
