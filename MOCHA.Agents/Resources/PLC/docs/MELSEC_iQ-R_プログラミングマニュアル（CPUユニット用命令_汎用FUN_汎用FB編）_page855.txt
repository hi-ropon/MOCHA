### テキスト
#### 倍精度実数帯域比較出力（EDZCP(P)）
- 対応CPUはR00CPU、R01CPU、R02CPUでバージョン制限なし。RnCPU（R00CPU、R01CPU、R02CPU除く）およびRnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降が必要。
- 命令は、下限値(s1)と上限値(s2)の倍精度実数の帯域に対し、比較データ(s3)の倍精度実数と比較し、その結果に応じて(d)、(d)+1、(d)+2のいずれかのビットをONにする。

### 図
- ラダー図：比較対象の3つの入力(s1, s2, s3)と出力(d)の関係を示す。
- ST言語例：
  ```
  ENO := EDZCP(EN, s1, s2, s3, d);
  ENO := EDZCPP(EN, s1, s2, s3, d);
  ```
- FBD/LDブロック図：EN, s1, s2, s3を入力し、ENO, dを出力する構成。
