### テキスト
#### 37.2 最大値，最小値選択
- MAX(_E): 入力値の最大値を出力します。
- MIN(_E): 入力値の最小値を出力します。

### 図
#### ラダー、FBD/LD
- EN/ENOなしの場合のラダー図では、s1、s2の2つの入力からdに出力される構成。
- EN/ENO付きの場合のラダー図では、EN（実行条件）、s1、s2の入力からENO（出力状態）、dに出力される構成。

#### ST言語
- EN/ENOなしの場合
  - d := MAX(s1, s2);
  - d := MIN(s1, s2);
- EN/ENO付きの場合
  - d := MAX_E(EN, ENO, s1, s2);
  - d := MIN_E(EN, ENO, s1, s2);

### 表
#### 設定データ：内容、種別、データ型

| 引数            | 内容                             | 種別       | データ型           |
|-----------------|--------------------------------|------------|--------------------|
| EN              | 実行条件(TRUE: 実行、FALSE: 停止) | 入力変数   | BOOL               |
| s1(IN1)～s28(IN28) | 入力                           | 入力変数   | ANY_ELEMENTARY*1   |
| ENO             | 出力状態(TRUE: 正常、FALSE: 異常) | 出力変数   | BOOL               |
| d               | 出力                           | 出力変数   | ANY_ELEMENTARY*1   |

*1 安全プログラムではREAL型/LREAL型/STRING型データを指定しないこと。指定した場合はエラーとなる。
