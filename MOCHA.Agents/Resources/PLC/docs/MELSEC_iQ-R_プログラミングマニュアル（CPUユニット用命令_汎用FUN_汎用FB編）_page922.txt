### テキスト
#### 機能
- (s)で指定された角度のTAN（正接）値を演算し、(d)で指定されたデバイス番号に演算結果を格納します。
- 角度はラジアン単位（角度×π÷180）で設定します。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

#### 注意事項
- (s)で指定する角度がπ/2ラジアン、(3/2)πラジアンの場合、ラジアン値に演算誤差が生じるためエラーは発生しません。

#### エラー
- エラーコード「3402H」： (s)に指定した値が±0、非正規化数、非数、±∞のときに発生します。

#### Point
- 角度とラジアンの変換については、RADD(P)命令、DEGD(P)命令を参照してください。
  - RADD(P)命令は945ページ
  - DEGD(P)命令は947ページに記載

### 図
- TAN関数の引数と結果の格納先を示す図
  - (s)+3、(s)+2、(s)+1、(s)の4つの倍精度実数で角度を指定
  - 演算結果は(d)+3、(d)+2、(d)+1、(d)の4つの倍精度実数のデバイスに格納される
