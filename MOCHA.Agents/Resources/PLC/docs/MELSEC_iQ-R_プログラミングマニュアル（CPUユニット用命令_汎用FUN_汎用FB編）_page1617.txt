### 表
#### GX Works2ファンクションと互換用ファンクションの対応一覧

| 項目 | 内容 |
| :--- | :--- |
| GX Works2ファンクション名 | 元のGX Works2で使用されるファンクション名 |
| 互換用ファンクション名 | 互換性を持たせた別のファンクション名（末尾に「_M2」が付く） |
| 使用プログラム | ST（Structured Text）およびFBD（Function Block Diagram）での使用可否を○×で示す |
| 参照先 | 詳細説明が記載されているマニュアルのページ番号とファンクション名 |

#### 内容説明
- 表は主にGX Works2の各種関数（例：INSERT_E、INT_TO_BCD_Eなど）と、それに対応する互換用関数（例：INSERT_E_M2、INT_TO_BCD_E_M2など）を一覧化しています。
- 「使用プログラム」欄では、ST形式での使用はすべて「○」で可能である一方、FBD形式での使用はすべて「×」で不可となっています。
- 「参照先」欄には、各関数の詳細説明が記載されているマニュアルのページ番号が示されており、関数名も併記されています。
- 一部の関数は参照先が「—」となっており、詳細説明ページがないか、該当しないことを示しています。

#### 例
- INSERT_Eは互換用関数INSERT_E_M2があり、STで使用可能、FBDでは不可。詳細は1477ページに記載。
- INT_TO_BCD_Eは互換用関数INT_TO_BCD_E_M2があり、STで使用可能、FBDでは不可。詳細は1341ページに記載。

この表は、他形式プロジェクトからの関数置換え時に参照するための一覧表として機能しています。
</contents>
