**Point**  
演算結果がデータ型の範囲を超える場合は、入力値のデータ型を変換してから演算を実行してください。

#### 実行条件と演算結果

| 実行条件 | 演算結果 |
| --- | --- |
| EN | ENO | (d) |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |
