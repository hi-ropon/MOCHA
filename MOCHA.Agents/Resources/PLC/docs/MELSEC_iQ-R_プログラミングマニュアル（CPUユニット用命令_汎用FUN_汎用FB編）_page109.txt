### テキスト・表
#### BCD型SIN⁻¹演算
- 命令記号: BASIN, BASINP
- 処理内容: BCD値で指定した値のSIN⁻¹（逆正弦）値を演算する。
- SIN⁻¹(s)の引数(d)は符号、整数部、小数部に分かれる。
- 参照先: 935ページ BASIN(P)

#### BCD型COS⁻¹演算
- 命令記号: BACOS, BACOSP
- 処理内容: BCD値で指定した値のCOS⁻¹（逆余弦）値を演算する。
- COS⁻¹(s)の引数(d)は符号、整数部、小数部に分かれる。
- 参照先: 937ページ BACOS(P)

#### BCD型TAN⁻¹演算
- 命令記号: BATAN, BATANP
- 処理内容: BCD値で指定した値のTAN⁻¹（逆正接）値を演算する。
- TAN⁻¹(s)の引数(d)は符号、整数部、小数部に分かれる。
- 参照先: 939ページ BATAN(P)

#### 単精度実数角度→ラジアン変換
- 命令記号: RAD, RADP
- 処理内容: 角度の大きさの単位を単精度実数で指定し、度単位からラジアン単位に変換する。
- 変換式: (s)+1, (s) → (d)+1, (d)
- 参照先: 941ページ RAD(P)

#### 単精度実数ラジアン→角度変換
- 命令記号: DEG, DEGP
- 処理内容: 角度の大きさの単位を単精度実数で指定し、ラジアン単位から度単位に変換する。
- 変換式: (s)+1, (s) → (d)+1, (d)
- 参照先: 943ページ DEG(P)

#### 倍精度実数角度→ラジアン変換
- 命令記号: RADD, RADDP
- 処理内容: 角度の大きさの単位を倍精度実数で指定し、度単位からラジアン単位に変換する。
- 変換式: (s)+3, (s)+2, (s)+1, (s) → (d)+3, (d)+2, (d)+1, (d)
- 参照先: 945ページ RADD(P)
