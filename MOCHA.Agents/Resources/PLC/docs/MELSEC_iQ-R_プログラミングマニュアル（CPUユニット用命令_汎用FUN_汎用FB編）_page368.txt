### テキスト
#### 倍精度実数n点の1点左シフト（命令名：EDSFL(P)）
- R00CPU、R01CPU、R02CPUはファームウェアバージョン"06"以降で使用可能。エンジニアリングツールはバージョン"1.047Z"以降で使用可能。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降で使用可能。
- 指定したデバイスから倍精度実数(n)点分を左へ1点シフトし、シフト後の空きデータには0が格納される。

### 表
#### ラダー命令構文
| ラダー | ST |
| :--- | :--- |
| 図（d,n） | ENO:=EDSFL(EN,n,d); ENO:=EDSFLP(EN,n,d); |

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| EDSFL | 立下りエッジ |
| EDSFLP | 立上りエッジ |

#### 設定データ
##### 内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (d) | シフトする先頭デバイス | － | 倍精度実数 | ANYREAL_64 |
| (n) | シフトするデータの数 | 0～65535 | 符号なしBIN16ビット | ANY16 |

##### 使用可能デバイス
| オペランド | ビット | ワード | ダブルワード | 間接指定 | 定数 | その他 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| (d) | － | － | ○ | － | － | － |
| (n) | ○*1 | ○ | ○ | ○ | ○ | － |

*1 FX、FYは使用不可。
