### テキスト・図
- 小数部桁数を0以外に設定した場合、指定された小数部桁数+1桁目に自動的に2EH(.)が格納される。小数部桁数が0のときは2EH(.)は格納されない。
- 指数部の符号は、指数が正のときは2BH(+)を、負のときは2DH(-)を格納する。指数部は2桁固定で、1桁の場合は符号の間に30H(0)を格納する。
- 変換した文字列の最後には自動的に00Hが格納される。
- 図は、(s1)に格納された数値と(s2)の小数部桁数設定に基づき、どのように文字列が変換されるかを示している。具体的には全桁数、小数部桁数、符号の自動付加の位置を示す。

### エラーコード表
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3401H | (s1)が指定範囲外（0 ≤ (s1) < 2^128）である場合。<br>(s2)に変換できない不正なデータを設定した場合。<br>・(s2)で指定する形式指定が0または1以外。<br>・小数点形式で(s2)+1で指定する全桁数が範囲外。<br>・小数部桁数が0のときの全桁数や指数部の範囲外指定。<br>・指数形式での全桁数や小数部桁数の範囲外指定。<br>・小数点形式の場合、(s1)に指定できる32ビット浮動小数点実数データの整数部桁数が16桁を超える場合。<br>・(s2)+2で指定する小数部桁数が0～7以外。<br>・全桁数が24を超える値を指定した場合。 |
| 3402H | (s1)で指定したデバイスの内容が0、非正規化数、非数、±∞のとき。 |

- 注釈として、(s1)で指定する32ビット浮動小数点実数データの整数部桁数を示す旨の説明あり。
