### テキスト
#### 1ビットデータ転送 MOVB(P)
指定した1ビットデータを転送する命令です。対応CPUはRnCPU、RnENCPU（プロセス、二重化、一般、安全）など複数種類に対応しています。

#### 実行条件
- MOVB命令は立下りエッジで実行されます。
- MOVBP命令は立上りエッジで実行されます。

#### 注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**：スイッチのような記号で(s)と(d)のビットデバイス間の転送を示しています。
- **ST言語例**：
  ```
  ENO := MOVB(EN, s, d);
  ENO := MOVBP(EN, s, d);
  ```
- **FBD/LD図**：EN（実行条件入力）、ENO（実行結果出力）、s（転送元）、d（転送先）を持つブロック図。
