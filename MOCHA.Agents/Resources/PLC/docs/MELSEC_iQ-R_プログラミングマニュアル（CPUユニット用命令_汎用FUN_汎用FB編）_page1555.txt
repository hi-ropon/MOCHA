### 表
#### 命令処理時間一覧
この表は各命令の処理時間を示しており、CPUユニットごとに最小値と最大値が記載されています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例：DMOV(P)、BMOV(P)、SET、RSTなど） |
| 条件 | 命令実行時の条件（例：n=1、無変化時、変化時など） |
| 処理時間(μs) | CPUユニット別の処理時間。R00CPU、R01CPU、R02CPU、その他CPUユニットに分かれ、最小値と最大値が示されている。 |

#### 主な命令と処理時間の特徴
- **DMOV(P) [K8S□]**  
  R00CPU、R01CPUで最小4.1μs、最大7.7μs。R02CPUは最小3.1μs、最大6.7μs。他CPUユニットは最小2.6μs、最大6.2μs。
- **BMOV(P) [K4S□]**  
  n=1の条件でR00CPU、R01CPUは最小5.7μs、最大15.6μs。R02CPUは最小4.7μs、最大14.6μs。他CPUユニットは最小4.2μs、最大14.1μs。
- **SET [S□]**  
  無変化時はR00CPU、R01CPUで最小4.9μs、最大9.9μs。変化時は最小7.1μs、最大15.1μs。
- **RST [S□]**  
  無変化時は最小3.8μs、最大7.9μs。変化時は最小6.2μs、最大9.0μs。
- **PAUSE [BL□]**  
  全CPUユニットで最小3.3μs、最大4.6μs程度。
- **SP.CONTSW**  
  SM1646がON時は10.6～16.0μs、OFF時は137.2～169.7μsと大幅に処理時間が増加。
- **DCONTSW、ECONTSW**  
  許可・不許可の切り替え時に1.4～2.0μs程度。
- **CONTWR**  
  書込みデータ点数により処理時間が増加し、1点で18.3～21.2μs、1024点で27.2～29.9μs。
- **SP.SIDRD**  
  30.8～37.4μsと比較的長い処理時間。

この表は命令の種類と条件に応じたCPUユニット別の処理時間を詳細に示し、性能評価や最適化の参考となる情報を提供しています。
