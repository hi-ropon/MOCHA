### テキスト
#### SEL(_E)関数（選択値）
選択した入力値を出力する関数です。

##### 対応CPU
RnCPU、RnENCPU（プロセス、三重化）、RnPSFCPU（安全、一般）、RnSFCCPU（一般）、RnSFCFU（安全）

##### 使用言語
ラダー、FBD/LD、ST

##### ラダー/FBD/LDの構成
- EN/ENOなしバージョン：s1, s2, s3の3つの入力からdへ出力
- EN/ENO付きバージョン：EN入力、ENO出力付きでs1, s2, s3からdへ出力

##### ST言語の構成
- EN/ENOなし：d := SEL(s1, s2, s3);
- EN/ENO付き：d := SEL_E(EN, ENO, s1, s2, s3);

#### 設定データ（引数の内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件（TRUE:実行、FALSE:停止） | 入力変数 | BOOL |
| s1(G) | 出力条件（TRUE: s3出力、FALSE: s2出力） | 入力変数 | BOOL |
| s2(IN0) | 入力 | 入力変数 | ANY*1 |
| s3(IN1) | 入力 | 入力変数 | ANY*1 |
| ENO | 出力状態（TRUE:正常、FALSE:異常） | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANY*1 |

*1 安全プログラムではREAL型、LREAL型、STRING型データは指定不可。指定した場合はエラーとなる。

#### 機能
##### 演算処理
- (s1)の入力値に従い、(s2)、(s3)のいずれかの値を(d)から出力する。
- (s1)がFALSE(0)の場合、(s2)の入力値を(d)に出力。
- (s1)がTRUE(1)の場合、(s3)の入力値を(d)に出力。

##### 例
- (s2)、(s3)のデータ型がINT型の場合、引数名の(s2)、(s3)は(s1)のビット値(0または1)に対応。
- (s1)の入力値はBOOL型データ値。
- (s2)、(s3)への入力値はBOOL型、INT型、DINT型、WORD型、DWORD型、REAL型、LREAL型、STRING型、TIME型、構造体型、配列型のいずれか。
