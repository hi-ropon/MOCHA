### テキスト
#### 機能
(s)で指定されたBIN32ビットデータをBIN32ビットグレイコードデータに変換し、(d)で指定されたデバイスに格納します。

- (s)+1: 上位16ビット
- (s): 下位16ビット

#### エラー
エラーコード(SD0)とその内容を示しています。

### 図
BIN32ビットデータのビット配置を示す図があり、以下のように説明されています。

- (s) BIN: 元のBIN32ビットデータのビット配列（b31からb0まで）
- (d): 変換後のBIN32ビットグレイコードデータのビット配列
- (s)+1: 上位16ビット、(s): 下位16ビットとして分割されていることを示す

### 表

| :--- | :--- |
| 3401H | DGRY(P)命令使用時、(s)で指定した値が0～2147483647以外のとき。 |

この表は、指定範囲外の値が入力された場合のエラーコードとその説明を示しています。
