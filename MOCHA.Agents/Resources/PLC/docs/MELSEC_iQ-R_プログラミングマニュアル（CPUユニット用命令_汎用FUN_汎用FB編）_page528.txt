### テキスト
#### 1ビットデータ反転転送 CMLB(P)
指定したビットデータを反転し、転送する命令です。

##### 実行条件
- CMLB命令は立ち上がりエッジで実行
- CMLBP命令は立ち下がりエッジで実行

##### 機能
(s)で指定されたビットデータを反転し、その結果を(d)で指定されたデバイスへ格納します。

### 図
- **ラダー図**  
  ラダー図では、(s)と(d)のオペランドを持つ命令ブロックが示されており、ST言語では  
  `ENO:=CMLB(EN,s,d);`  
  `ENO:=CMLBP(EN,s,d);`  
  と記述されます。

- **FBD/LD図**  
  入力EN、出力ENO、入力s、出力dのポートを持つ命令ブロックが示されています。

- **ビット反転転送の動作イメージ図**  
  転送前の(s)のビット列の1が反転されて0となり、その結果が(d)に転送される様子が示されています。
