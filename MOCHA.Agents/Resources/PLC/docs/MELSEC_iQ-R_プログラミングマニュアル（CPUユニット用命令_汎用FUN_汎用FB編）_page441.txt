### テキスト
#### 機能
- (s)で指定した符号なしBIN16ビットデータ(ANY16_U)を、符号付きBIN16ビットデータ(ANY16_S)に変換し、(d)で指定したラベルに格納する。
- UINT2INT(P)命令の使用例を示す。

#### 例
+命令はオペランドにANY16_Sを指定する必要があるため、+命令を実行する前にUINT2INT命令を使用して、ANY16_UのuLabel0をANY16_SのwLabel1へ変換する。
値としてはuLabel0をそのままwLabel1へ格納する。

- bSwitchA: ビット
- wLabel0, wLabel1: ワード[符号付き]
- uLabel0: ワード[符号なし/ビット列16ビット]

(1) uLabel0の値をそのまま格納する。
(2) +命令のオペランドのデータ型に合わせて演算する。

#### エラー
演算エラーはありません。

### 図
- ブロック図で、bSwitchAの信号がUINT2INT命令に入力され、uLabel0からwLabel1へ変換される流れを示す。
- (s)側のuLabel0は符号なし16ビットデータ(C000H)で、(d)側のwLabel1は符号付き16ビットデータ(C000H)として格納される。
- +命令はwLabel0とwLabel1をオペランドとして使用し、wLabel1に結果を格納する。
