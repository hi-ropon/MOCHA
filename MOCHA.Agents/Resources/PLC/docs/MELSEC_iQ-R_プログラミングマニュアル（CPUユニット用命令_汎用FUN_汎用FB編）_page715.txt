### テキスト
#### 機能
- (s1)を先頭とする(n)個のデータに対し、(s2)のBIN16ビットデータと同一のデータを検索し、結果を(d)～(d)+4に格納する。
- 同一データがあった場合、(d)を先頭とする5点のデバイスには、同一データの個数、初回/最終位置、最大値、最小値の位置が格納される。
- 同一データがなかった場合、(d)を先頭とする5点のデバイスには、同一データの個数、初回/最終位置や最大値、最小値の位置が格納される。ただし、(d)を先頭とする3点のデバイス（同一データの個数、初回/最終位置）には0が格納される。
- データの中に最小値、最大値が複数ある場合は、それぞれ後側の位置が格納される。
- (n)に0を指定した場合は無処理となる。
- 検索結果テーブルの構成とデータ例を示す（n=10）。

#### エラー
- 演算エラーはない。
