### テキスト
#### 2.3 応用命令 プログラム制御
プログラムの制御に関する応用命令について説明しています。

#### プログラム分岐命令
- ポインタ分岐命令として、同一プログラムファイル内の指定されたポインタ番号のプログラムを実行する命令が紹介されています。
- ENDへジャンプ命令として、同一プログラムファイル内のFENDまたはEND命令へジャンプする命令が説明されています。

#### プログラム実行制御命令
- 割込み禁止、割込み許可に関する命令があり、割込みプログラムの実行禁止や禁止状態の解除を行います。
- 指定優先度以下の割込み禁止命令は、指定優先度以下の割込みプログラムの起動要因があっても、許可命令が実行されるまで割込みを禁止します。
- 割込みプログラムマスク命令は、指定された割込みポインタ番号の割込みプログラムの実行許可状態や実行禁止状態を設定します。
- 指定割込みポインタの禁止/許可命令も同様に割込みプログラムの実行許可状態や禁止状態を設定します。
- 割込みプログラムからの復帰命令は、割込みプログラムの処理終了を示します。
- WDTリセット命令はウォッチドッグタイマをリセットします。
