### テキスト
#### タイマの時間設定と注意事項
- タイマの時間設定はエンジニアリングツールのパラメータ設定で行う。
- 低速タイマ/低速積算タイマの設定範囲は1～1000ms（デフォルト100ms、設定単位1ms）。
- 高速タイマ/高速積算タイマの設定範囲は0.01～100.0ms（デフォルト10.0ms、設定単位0.01ms）。
- タイマの計数方法については「MELSEC iQ-R CPUユニットユーザズマニュアル（応用編）」を参照。

#### 注意事項
- タイマの接点で別のタイマの計測を行うプログラムを作成する場合は、計測するタイマの順にプログラミングすること。
- 以下の場合、計測順にプログラミングしないとすべてのタイマが同一スキャンでONしてしまう。
  - スキャンタイムより短い設定値のとき
  - 設定値が1のとき

### 図（ラダープログラム例）
#### 例1：T0～T2のタイマを後から計測するタイマから順にプログラミングした場合
- T1の接点がONした次のスキャンからT2のタイマが計測開始。
- T0の接点がONした次のスキャンからT1のタイマが計測開始。
- X0がONするとT0のタイマが計測開始。

図は3つのラダー回路で、X0→T0→T1→T2の順に接点が配置されている。

#### 例2：T0～T2のタイマを計測順にプログラミングした場合
- X0がONするとT0のタイマが計測開始。
- T0の接点がONするとT1、T2のタイマの接点もON。

図はX0→T0→T1、T2の接点が順に配置されている。

### エラー
- 演算エラーはなし。
