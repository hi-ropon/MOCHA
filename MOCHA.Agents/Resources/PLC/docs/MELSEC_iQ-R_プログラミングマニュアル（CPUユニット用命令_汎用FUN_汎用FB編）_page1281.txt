### 表
#### エラーコード(SD0)と内容
| エラーコード(SD0) | 内容 |
| :--- | :--- |
| 31A1H | 指定ブロックが存在しないとき。SFCプログラムが存在しない（スキャン実行タイプのSFCプログラムが存在しない）、またはSM321（SFCプログラムの起動/停止）をOFFに設定して命令を実行したとき。 |
| 31A2H | 指定ブロックNo.が指定可能なブロックNo.の範囲外のとき。 |

この表は、SFCプログラムに関連するエラーコードとその発生条件を示しています。エラーコード31A1Hは指定されたブロックが存在しない場合に発生し、31A2Hは指定されたブロック番号が許容範囲外の場合に発生します。
</contents>
