### テキスト
#### 機能
- (d)で指定されたデバイスの内容（BIN32ビットデータ）を-1します。
- 図解では、(d)のビット構成（b31～b0）を示し、73500（BIN）が-1されて73499（BIN）になる例を示しています。
- (d)で指定されたデバイスの内容が-2147483648のとき、DDEC(P)命令を実行すると2147483647が(d)で指定されたデバイスに格納されます（符号付き指定の場合）。
- (d)で指定されたデバイスの内容が0のときDDEC(P)命令を実行すると、-1が(d)で指定されたデバイスに格納されます（符号付き指定時）。
- (d)で指定されたデバイスの内容が0のとき、DDEC(P)_U命令を実行すると4294967295が(d)で指定されたデバイスに格納されます（符号なし指定の場合）。

#### エラー
- 演算エラーはありません。

### 図
- ビットデータの構成を示す図があり、b31からb0までのビットが並んでいる。
