### テキスト
#### 機能
(s)で指定したデバイス以降に格納している時計データをCPUユニットの時計素子に書き込みます。

- 各項目の設定はBIN値で行います。
- 年は1980年から2079年までの西暦4桁で設定します。
- 月は1～12（1月～12月）で設定します。
- 日は1～31で設定します。
- 時は0～23（24時間計）で設定します。
- 分は0～59で設定します。
- 秒は0～59で設定します。
- 曜日は0～6で「日～土」を表します。

#### エラー
- 3405H: (s)～(s)+6の各項目に設定範囲外のデータを設定した場合のエラー。
- 3425H: (s)～(s)+6に夏時間開始時刻から1時間未満を設定した場合のエラー。

#### Point
時計データを変更すると、イベント履歴に「時計設定」（イベントコード：24000）が保存されます。したがって、本命令を実行した場合、イベント履歴に「時計設定」が保存されます。
