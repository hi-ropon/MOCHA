### テキスト
#### BIN16ビットデータ交換
指定されたBIN16ビットデータを交換する命令について説明しています。

#### 実行条件
- XCH命令は立下りエッジで実行されます。
- XCHP命令は立上りエッジで実行されます。

#### 機能
(d1)と(d2)のBIN16ビットデータを交換します。

### 図
- **ラダー図**  
  ラダー回路で(d1)と(d2)のデータを交換する命令の記述例が示されています。  
  ST言語では、`ENO:=XCH(EN,d1,d2);` または `ENO:=XCHP(EN,d1,d2);` と記述します。

- **FBD/LD図**  
  ブロック図でEN（実行条件）、ENO（実行結果）、d1、d2の入出力ポートが示されています。

- **ビットデータ交換イメージ図**  
  (d1)と(d2)のビット列（b15～b0）が交換される様子をビット単位で示しています。  
  例えば、(d1)のビット列が`0111100000001111`、(d2)のビット列が`1111000011110000`の場合、交換後は(d1)が`1111000011110000`、(d2)が`0111100000001111`となります。
