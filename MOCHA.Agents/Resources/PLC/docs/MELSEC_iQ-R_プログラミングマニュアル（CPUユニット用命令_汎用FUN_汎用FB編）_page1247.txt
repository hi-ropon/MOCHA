### テキスト
#### 演算式
PID制御命令の正動作および逆動作の演算式が示されています。

- 正動作の演算式
  - EVₙ = PVₙ - SV
  - ΔMV = Kₚ{(EVₙ - EVₙ₋₁) + (Tₛ / Tᵢ)・EVₙ + Dₙ}
  - Dₙ = (Tᴰ / Tₛ)(PVₙ - 2PVₙ₋₁ + PVₙ₋₂)
  - MVₙ = ΣΔMV

- 逆動作の演算式
  - EVₙ = SV - PVₙ
  - ΔMV = Kₚ{(EVₙ - EVₙ₋₁) + (Tₛ / Tᵢ)・EVₙ + Dₙ}
  - Dₙ = (Tᴰ / Tₛ)(-PVₙ + 2PVₙ₋₁ - PVₙ₋₂)
  - MVₙ = ΣΔMV

#### 演算式中の記号の意味
以下の記号の意味が表形式で説明されています。

| 記号 | 意味 |
| :--- | :--- |
| EVₙ | 今回のサンプル時の偏差 |
| EVₙ₋₁ | 1周期前の偏差 |
| SV | 設定値 |
| PVₙ | 今回のサンプル時の測定値（フィルタ後） |
| PVₙ₋₁ | 1周期前の測定値（フィルタ後） |
| PVₙ₋₂ | 2周期前の測定値（フィルタ後） |
| ΔMV | 出力変化量 |
| MVₙ | 今回の操作量 |
| Dₙ | 今回の微分項 |
| Tₛ | サンプリング周期 |
| Kₚ | 比例定数 |
| Tᵢ | 積分定数 |
| Tᴰ | 微分定数 |

#### 補足説明
- 今回のサンプル時の測定値（フィルタ後）PVₙは、入力データの測定値を下式で演算した値であり、フィルタ係数が設定されていない場合は入力データの測定値（PV）と同じ値となる。
- PVₙ = PVₙ + α(PVₙ₋₁ - PVₙ)
- PVₙ：今回のサンプル時の測定値、α：フィルタ係数、PVₙ₋₁：1周期前の測定値（フィルタ後）
- PVₙは入出力データエリアに格納される。
