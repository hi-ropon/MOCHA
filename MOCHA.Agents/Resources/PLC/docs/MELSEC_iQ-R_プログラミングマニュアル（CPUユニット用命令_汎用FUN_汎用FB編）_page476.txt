### テキスト
#### エラーコード 3401H の説明
このエラーは、(s)～(s)+5に変換できない不正なデータを入力した場合に発生します。

- 1文字目のアスキーコードが、2DH、30H～39H、20H、00H以外の場合にエラーとなります。
- 2文字目以降のアスキーコードが、30H～39H、20H、00H以外の場合にエラーとなります。
- DDABIN(P)命令使用時は、アスキーデータが-2147483648～2147483647以外の場合にエラーとなります。
- DDABIN(P)_U命令使用時は、アスキーデータが0～4294967295以外の場合にエラーとなります。

##### 補足
* SM705（変換桁数切換）がOFFの場合は、1文字目のアスキーコードに何を設定してもエラーにはなりません。
