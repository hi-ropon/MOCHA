### テキスト
#### 指数形式の場合の実行方法
(s)で指定した文字列が指数形式の場合の変換処理について説明しています。

- (s)で指定された文字列は、符号、小数点、指数部を除いて6桁が有効で、7桁目以降は切り捨てられます。
- 指数部の符号は2BH(+)または2DH(-)で指定し、省略すると正の値として扱います。
- 文字列中の最初の0以外の数値の間に20H(スペース)または30H(0)が存在する場合は、それらを無視して変換します。
- 「E」と数値の間に30H(0)が存在する場合も無視して変換します。

#### 単精度実数の変換例
- 具体的な16進数のバイト配列を示し、それがどのように指数形式の実数に変換されるかを例示しています。
- 例1では「-1.3201E+10」、例2では「-1.35034E-2」、例3では「-1.0453E+3」と変換される様子を示しています。

### 表
#### 16進数バイト配列の例
| アドレス | b15〜b8 | b7〜b0 | 内容説明 |
| :------- | :------ | :----- | :------- |
| (s)      | 20H(スペース) | 2DH(-) | 文字列の先頭部分 |
| (s)+1    | 2EH(.)  | 31H(1) | 小数点以下の数字 |
| (s)+2    | 32H(2)  | 33H(3) | 数字部分 |
| (s)+3    | 31H(1)  | 30H(0) | 数字部分 |
| (s)+4    | 2BH(+)  | 45H(E) | 指数部の符号と'E' |
| (s)+5    | 30H(0)  | 31H(1) | 指数部の数字 |
| (s)+6    | 00H     |        | 文字列終端 |

- 上記のようなバイト配列が実際の文字列を16進数で表現しており、これを基に実数に変換しています。

### 変換過程の説明
- 矢印で示された変換後の実数表記は、指数形式の数値を示し、符号、小数点、指数部を含んでいます。
- 変換時に無視される文字（スペースや0）についても明記されています。
