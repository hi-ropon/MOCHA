### テキスト
#### 32.1 BOOL型→WORD型変換
「BOOL_TO_WORD(_E)」関数は、BOOL型データをWORD型データへ変換します。対応CPUはRnCPU、RnENCPU、RnPSFCPU（一般・安全）、RnSFCPU（一般・安全）です。

##### ■概要
- BOOL型データをWORD型データに変換して出力します。
- 入力値がFALSEの場合はWORD型のデータ値で0Hを出力。
- 入力値がTRUEの場合はWORD型のデータ値で1Hを出力。

##### ■引数の説明
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | BOOL |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | WORD |

##### ■機能・演算処理
- (s)に入力されたBOOL型のデータをWORD型のデータに変換して(d)から出力。
- 入力がFALSEなら0H、TRUEなら1Hを出力。
- (s)への入力値はBOOL型のデータ値。

##### ■演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力される。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通り。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)から出力されたデータを使用しないようプログラム処理を行う必要がある。

### 図
- ラダー、FBD/LDでの使用例  
　[EN/ENOなし]はsからdへ直接変換するシンプルな図。  
　[EN/ENO付き]はEN入力とENO出力を持ち、sからdへ変換する図。

- ST言語での使用例  
　[EN/ENOなし]：d := BOOL_TO_WORD(s);  
　[EN/ENO付き]：d := BOOL_TO_WORD_E(EN, ENO, s);
