### テキスト
#### エラーコードの説明
異常完了時には、完了ステータス(s2)+1にエラーコードが格納されます。完了ステータス(s2)+1に格納されるエラーコードについては、別途「MELSEC iQ-R Ethernetユーザーズマニュアル(応用編)」を参照するよう案内されています。

#### Point
TCP固定長受信モードにおいて、受信状態信号がONしない場合でも、現時点で受信しているデータをSP.SOCRDATA命令で読み出すことが可能です。これにより、相手機器からの送信データが不足しているかどうかを確認できます。

### 表
| エラーコード(SD0) | 内容                             |
|-------------------|----------------------------------|
| 3405H             | (s1)で指定するコネクション番号が1～16以外のとき。 |
