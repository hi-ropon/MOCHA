### テキスト
#### 機能
- (d)で指定したデバイスから(n1)ビット分のデータ範囲内で、右へ(n2)ビット分シフトする。
- シフト後、(s)から(n2)点分は空きデータ部分に格納される。
- (s)に定数0を指定した場合、シフト後の最上位から(n2)点分は0になる。
- (s)に定数1を指定した場合、シフト後の最上位から(n2)点分は1になる。
- (n2)で指定した値が0の場合は無処理となる。

#### 例
- (n1)=10、(n2)=4の場合のビットシフト動作を図で説明。

### 図
- ビット列のシフト動作を示す図。
- (d)から(n1)ビット分のデータが右へ(n2)ビット分シフトされる様子をビット単位で示している。
- シフト後の空いた上位ビット部分は(s)の指定に従い0または1で埋められる。
- 図中に「① (n2)ビット分、右へシフト」「② コピー」の動作手順が示されている。

### 表
#### エラーコード(SD0)と内容
| エラーコード | 内容 |
| :--- | :--- |
| 2821H | (s)から(n2)点分の範囲と、(d)から(n1)点分の範囲が重複しているとき。 |
| 3405H | (s)の定数指定時に、0または1以外の定数を指定したとき。<br>また、(n1)、(n2)に指定した値が(n1) < (n2)のとき。 |
