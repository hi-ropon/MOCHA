### テキスト
#### SCJ命令
- 実行指令がOFFからONに変化した次のスキャンから、同一プログラムファイル内の指定されたポインタ番号のプログラムを実行する。
- 実行指令がOFFおよびONからOFFに変化したときは、次のステップのプログラムを実行する。

#### JMP命令
- 無条件に、同一プログラムファイル内の指定されたポインタ番号のプログラムを実行する。

#### 注意事項
- タイマのコイルをONした後、CJ命令、SCJ命令、JMP命令でコイルのONしているタイマをジャンプさせると正常な計測ができなくなる。
- CJ命令、SCJ命令、JMP命令でOUT命令をジャンプさせるとスキャンタイムは短くなる。
- CJ命令、SCJ命令、JMP命令で後ろにジャンプさせるとスキャンタイムは短くなる。
- CJ命令、SCJ命令、JMP命令は、実行中のステップより若いステップへジャンプ可能。ただしウォッチドッグタイマがタイムアップしないようにループから抜け出す方法を考慮する必要がある。
- CJ命令、SCJ命令、JMP命令で飛ばされたデバイスは変化しない。
- ラベル（P□）は1ステップを占有する。
- ジャンプ命令は同一プログラムファイル内のポインタ番号のみ指定可能。

### 図

- ラダー図例1：X3がONの間ループし、X7がONになるとループから抜け出す構造を示す。

- ラダー図例3：ラベルP9が1ステップを占有していることを示すラダー図。
