### テキスト
#### 演算結果プッシュ，読出し，ポップ
- MPS：演算結果プッシュ命令。MPS命令直前の演算結果（ON/OFF）を記憶します。
- MRD：演算結果読出し命令。MPS命令で記憶した演算結果を読み出します。
- MPP：演算結果ポップ命令。MPS命令で記憶した演算結果を取り出します。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| MPS, MRD, MPP | 常時実行 |

#### 機能
- ■MPS
  - MPS命令直前の演算結果(ON/OFF)を記憶します。
  - MPS命令は連続で16回まで使用可能。MPP命令を途中で使用した場合は、MPS命令の使用数を-1します。
- ■MRD
  - MPS命令で記憶した演算結果を読み出し、その演算結果で次のステップから演算を行います。
- ■MPP
  - MPS命令で記憶した演算結果を読み出し、その演算結果で次のステップから演算を行います。
  - MPS命令で記憶した演算結果をクリアします。
  - MPS命令の使用数を-1します。

### 図・表
- ラダー図：MPS、MRD、MPPの命令ブロックを示す。各命令はパルス状の記憶・読出し・ポップ動作を表現。
- ST（構造化テキスト）：
  ```
  ENO := MPS(EN);
  ENO := MRD(EN);
  ENO := MPP(EN);
  ```
- FBD/LD：EN（入力）とENO（出力）を持つ命令ブロックの図示。

- 表：命令と実行条件の対応表（上記参照）

（注）回路表示の場合、MPS、MRD、MPPは表示されません。
