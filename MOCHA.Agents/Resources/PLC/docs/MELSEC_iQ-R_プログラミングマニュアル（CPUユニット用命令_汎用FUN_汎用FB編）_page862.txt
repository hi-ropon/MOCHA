### テキスト
#### エラーコードの説明
- 3402H: (s)で指定したデータが0、非正規化数、非数、±∞のときに発生するエラー。
- 3402H: (d)で指定したデータが0、非正規化数、非数、±∞のときに発生するエラー。
- 3403H: (d)が指定範囲を超えるときに発生するエラー（オーバーフローが発生したとき）。具体的には |(d)| < 2^128 の範囲を超えた場合。
