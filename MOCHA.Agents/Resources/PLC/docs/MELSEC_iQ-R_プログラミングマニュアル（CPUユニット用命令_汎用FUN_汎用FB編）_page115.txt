### テキスト
#### CPUユニット内蔵データベースアクセス機能
CPUユニットに内蔵されたデータベースにアクセスするための命令について説明している。

#### データベースアクセス命令
データベース接続、切断、レコード追加、更新、検索、削除、インポート、エクスポート、トランザクション開始、コミットの各機能に対応した命令がある。

### 表
#### データベース接続
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBOPEN / DBOPENP | 指定フォルダに格納されたデータベースに接続し、使用可能な状態にする。 | 1031ページ DBOPEN(P) |

#### データベース切断
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBCLOSE / DBCLOSEP | 指定した識別番号とデータベースの割当てを解除する。 | 1033ページ DBCLOSE(P) |

#### データベースレコード追加
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBINSERT / DBINSERTP | 指定識別番号に対応するデータベースの指定テーブルにレコードを追加する。 | 1035ページ DBINSERT(P) |

#### データベースレコード更新
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBUPDATE / DBUPDATEP | 指定識別番号のデータベース内の指定テーブルで条件に合致する全レコードを更新する。 | 1042ページ DBUPDATE(P) |

#### データベースレコード検索
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBSELECT / DBSELECTP | 指定識別番号のデータベースの指定テーブルのレコードを検索する。 | 1048ページ DBSELECT(P) |

#### データベースレコード削除
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBDELETE / DBDELETEP | 指定識別番号のデータベースの指定テーブルで条件に一致するレコードを削除する。 | 1056ページ DBDELETE(P) |

#### データベースインポート
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBIMPORT / DBIMPORTP | 指定パスのUnicodeテキストファイルからデータを取り込み、データベースを構築する。 | 1026ページ DBIMPORT(P) |

#### データベースエクスポート
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBEXPORT / DBEXPORTP | 指定パスのUnicodeテキストファイルにデータベースのデータを書き出す。 | 1029ページ DBEXPORT(P) |

#### データベーストランザクション開始
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBTRANS / DBTRANSP | 指定識別番号のデータベースでトランザクション開始を宣言する。 | 1060ページ DBTRANS(P) |

#### データベースコミット
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DBCOMMIT / DBCOMMITP | 指定識別番号のデータベースに関するトランザクションを確定する。 | 1062ページ DBCOMMIT(P) |
