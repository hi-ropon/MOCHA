### テキスト
#### 単精度実数加算
「E+(P)」命令はオペランドが2個の場合の単精度実数の加算を行います。

- ラダー、ST、FBD/LDでの対応状況が記載されており、ラダーでは簡単な記号で表現されているが、STおよびFBD/LDでは対応していません（3個の場合は859ページ参照）。
- 実行条件として、命令「E+」は立ち上がりエッジ、「E+P」は立ち下がりエッジで実行されることが示されています。

#### 機能
(d)で指定された単精度実数と、(s)で指定された単精度実数の加算を行い、加算結果を(d)で指定されたデバイスに格納します。

- (s)、(d)には0から2^126以下の値が指定・格納可能。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生する可能性があるため注意が必要（詳細は47ページ参照）。
