### テキスト
#### 32.36 DINT型→STRING型変換
関数名：DINT_TO_STRING(_E)

この関数はDINT型データをSTRING型データへ変換します。

対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）で使用可能ですが、RnSFCPU（一般、安全）では使用不可となっています。

#### 使用例
- ラダー、FBD/LDでの使用例
  - EN/ENOなしの場合：入力sから出力dへ変換するシンプルな形
  - EN/ENO付きの場合：EN入力、ENO出力を持つ形でsからdへ変換
- ST言語での使用例
  - EN/ENOなし：d := DINT_TO_STRING(s);
  - EN/ENO付き：d := DINT_TO_STRING_E(EN, ENO, s);

### 表
#### 設定データ：内容、種別、データ型

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | DINT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | STRING(11) |

この表は関数の引数の役割とデータ型を示しており、ENは実行条件、sは変換対象のDINT型入力、ENOは実行結果の状態出力、dは変換後のSTRING型出力を表しています。

### 図
- ラダー、FBD/LDのブロック図
  - EN/ENOなし：s入力からd出力へ変換する単純なブロック
  - EN/ENO付き：EN入力、ENO出力を持ち、s入力からd出力へ変換するブロック

これらの図は関数の使用イメージを視覚的に示しています。
