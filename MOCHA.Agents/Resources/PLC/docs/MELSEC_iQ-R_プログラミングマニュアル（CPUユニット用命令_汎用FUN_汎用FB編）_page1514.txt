### テキスト
#### 演算結果
1. EN/ENOなしファンクションブロック
- 演算結果は以下の通り。
- 演算エラーなしの場合は演算出力値が(d1)、(d2)に出力される。
- 演算エラーありの場合は(d1)、(d2)は不定値となる。

2. EN/ENO付きファンクションブロック
- 実行条件と演算結果は以下の通り。
- ENがTRUE（演算実行）の場合、ENOがTRUEなら演算出力値を(d1)、(d2)に出力し、ENOがFALSEなら不定値となる。
- ENがFALSE（演算停止）の場合、ENOはFALSEとなり、(d1)、(d2)は前回出力値を保持する。

### 表
#### 1. EN/ENOなしファンクションブロック 演算結果
| 演算結果       | (d1)、(d2)       |
| -------------- | ---------------- |
| 演算エラーなし | 演算出力値       |
| 演算エラーあり | 不定値           |

#### 2. EN/ENO付きファンクションブロック 実行条件と演算結果
| 実行条件 | 演算結果 | (d1)、(d2)       |
| -------- | -------- | ---------------- |
| EN       | ENO      |                  |
| TRUE     | TRUE     | 演算出力値       |
| TRUE     | FALSE    | 不定値           |
| FALSE    | FALSE    | 前回出力値       |

### グラフ（タイミングチャート）
#### 1. EN/ENOなしファンクションブロック（n=T#5sの場合）
- (s)のOFF→ONで(d1)をONにし、(s)のOFF→ONで(d2)の時間計測を開始。
- (d2)が指定時間に達すると(d1)をOFFにする。
- (s)がOFFかつ(d1)がOFFで(d2)を初期化。

#### 2. EN/ENO付きファンクションブロック（n=T#5sの場合）
- EN=ONかつ(s)のOFF→ONで(d1)をONにし、EN=ONかつ(s)のOFF→ONで(d2)の時間計測を開始。
- 計測開始後はEN=ONで計測時間がアップ。
- (d2)が指定時間に達すると(d1)をOFFにする。
- EN=ONかつ(s)=OFFかつ(d1)=OFFで(d2)を初期化。

タイミングチャートは各信号（EN、(s)、ENO、(d1)、(d2)）のON/OFF状態を時間軸に沿って示し、信号の変化とそれに伴う動作を視覚的に理解できるようにしています。
