### テキスト
#### 機能
(s)で指定されたデバイスのBIN32ビットデータ（0～99999999）をBCD8桁データに変換し、(d)で指定されたデバイスに格納します。

#### 注意事項
(1) 上位5ビットは必ず0にしてください。

### 図
* BIN32ビットデータ（s）のビット配置を示しています。上位16ビットと下位16ビットに分かれており、具体的なビットの並びが示されています。  
* 変換後のBCD8桁データ（d）は、10^7から10^0までの各桁に対応するビットが並び、各桁は8,4,2,1の重み付けで表現されています。  
* 図中に「(s)+1(上位16ビット)」「(s)(下位16ビット)」「(d)+1(上位4桁)」「(d)(下位4桁)」の表記があり、データの格納位置を示しています。

### 表
#### エラーコード一覧
| エラーコード (SD0) | 内容                                    |
|--------------------|---------------------------------------|
| 3401H              | (s)のデータが0～99999999以外のとき。  |
