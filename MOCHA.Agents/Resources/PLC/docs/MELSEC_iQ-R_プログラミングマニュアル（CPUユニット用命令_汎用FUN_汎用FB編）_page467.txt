### テキスト
#### 機能
(s)で指定したデバイスから、(n)点分のBIN16ビットデータ（0～9999）をBCD変換し、(d)で指定したデバイス以降に格納します。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 2821H              | (s)、(d)のデバイスが重複しているとき。 |
| 3401H              | (s)のデバイスから(n)点分のデータが0～9999以外のとき。 |

### 図
#### BIN16ビットデータからBCD変換のビット配置
- (s)から始まるBIN16ビットデータの各ビットが示されており、上位ビットから下位ビットまでの配置が表記されています。
- 例として、BIN 1234、BIN 5678、BIN 1545などのビット列が示されている。
- これらのBINデータがBCD形式に変換され、(d)で指定したデバイスに格納される様子が矢印で示されている。
- BCD変換後の各ビットは、1000、100、10、1の重みを持つ4ビット単位で表されている。
- 注記として、(1)のビットは必ず0にする必要があることが記載されている。
