### テキスト
#### 機能
- 実行指令がONしたとき、(d)で指定されたアナンシエータをONにします。
- アナンシエータ(F)をONにした場合の動作は以下の通りです。
  - USER LEDが点灯します。
  - ONしたアナンシエータ番号(F番号)が特殊レジスタ(SD64～SD79)に格納されます。
  - SD63の内容が+1されます。
- ただし、SD63の内容が16の場合（すでに16個のアナンシエータがONしている場合）は、新たにアナンシエータがONしても、ONするアナンシエータ番号がSD64～SD79には格納されません。

#### エラー
- 演算エラーはありません。

### 図
- 左側の図は、SD63～SD79の特殊レジスタの値を示しています。SD63は16、SD64は233、SD65は90、SD66は700、SD78は145、SD79は1027となっています。
- 中央に「F30 ON」という矢印があり、アナンシエータONの動作を示しています。
- 右側の図は、アナンシエータON後のSD63～SD79の値を示していますが、左側と同じ値で変化がないことを示しています。
- 図の注釈(1)には「変化しません」と記載されており、SD63の値が16の場合はレジスタの値が変わらないことを示しています。
