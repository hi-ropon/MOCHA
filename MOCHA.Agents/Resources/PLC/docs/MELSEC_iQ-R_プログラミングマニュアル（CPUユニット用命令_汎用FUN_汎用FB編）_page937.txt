### テキスト
#### BCD型COS⁻¹演算（BACOS(P)）
BCD値で指定した値のCOS⁻¹（逆余弦）値を演算します。

##### 命令概要
- BACOS、BACOSP命令は、入力されたBCD形式の値に対して逆余弦演算を行い、その結果を指定デバイスに格納します。

##### 機能
- (s)で指定された値のCOS⁻¹（逆余弦）値を演算し、(d)で指定されたデバイスに演算結果（角度）を格納します。
- COS⁻¹の計算は符号、整数部、小数部に分けてBCD値で格納されます。
- (s)には演算するデータの符号を設定し、正のときは0、負のときは1を格納します。
- (s)+1、(s)+2には整数部、小数部をそれぞれBCD値で格納します（0～1.0000まで設定可能）。
- (d)に格納される演算結果はBCD値で0～180°（DEG.単位）の範囲内の値です。
- 演算結果は小数部を四捨五入した値となります。

### 図
- ラダー図：BACOS命令のラダー記述例が示されており、(s)と(d)のデバイスを入力としている。
- FBD/LD図：BACOS命令のFBD/LDブロック図が示されており、EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）端子がある。
- COS⁻¹演算のデータ格納イメージ：符号、整数部、小数部をそれぞれ(s)、(s)+1、(s)+2に格納する様子が図示されている。
