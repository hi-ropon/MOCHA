### テキスト
#### エラーコード一覧（SD0）

| エラーコード (SD0) | 内容 |
|--------------------|------|
| 31A1H              | 指定ブロックが存在しないとき。<br>具体的には、SFCプログラムが存在しない（スキャン実行タイプのSFCプログラムが存在しない）、またはSM321（SFCプログラムの起動/停止）がOFFに設定されて命令を実行した場合。 |
| 31A2H              | 指定ブロックNo.が指定可能なブロックNo.の範囲外の場合。 |

この表は、SFCプログラムに関連するエラーコードとその原因を示しており、プログラムの存在確認やブロック番号の範囲チェックに関するエラーを説明しています。
