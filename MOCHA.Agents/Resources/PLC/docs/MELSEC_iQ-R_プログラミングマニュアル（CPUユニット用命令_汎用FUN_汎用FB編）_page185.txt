### テキスト
#### 機能
- 実行指令がONしたとき、(d)で指定されたアナンシエータをOFFにします。
- OFFにしたアナンシエータ番号（F番号）は、特殊レジスタ（SD64～SD79）から削除され、SD63の内容が-1されます。
- SD63の内容が16のとき、RST命令によりSD64～SD79からアナンシエータ番号が削除されます。また、SD64～SD79に登録されていない番号のアナンシエータがONしていれば、その番号を新たに登録します。
- SD64～SD79のアナンシエータ番号がすべてOFFすると、CPUユニット前面のUSER LEDが消灯します。

#### エラー
- 演算エラーはありません。

### 図（例）
- 左側の表はSD63～SD79のレジスタの初期値を示しています。SD63は16、SD64は233、SD65は90、SD66は700、SD67は28、SD78は145、SD79は1027です。
- 矢印(1)はF90をリセットする動作を示しています。
- 右側の表はリセット後の状態を示し、SD63は16、SD64は233、SD65は700（元のSD66の値）、SD66は28、SD77は145、SD78は1027、SD79は空欄（登録されていないF番号が格納される場所）となっています。
- 矢印(2)はSD66に格納されていたF番号がSD65に移動したことを示しています。
- 矢印(3)は登録されていないF番号がSD79に格納されることを示しています。
