### テキスト
#### 32.28 DINT型→BOOL型変換
関数名：DINT_TO_BOOL(_E)  
この関数はDINT型データをBOOL型データへ変換します。対応するCPU種別がアイコンで示されています。

##### 機能
- (s)に入力されたDINT型のデータをBOOL型のデータに変換し、(d)から出力します。
- 入力値が0の場合はFALSEを出力します。
- 入力値が0以外の場合はTRUEを出力します。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。  
2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

##### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDの関数ブロック図  
　EN/ENOなしとEN/ENO付きの2種類のブロック図が示されています。  
- ST（構造化テキスト）での使用例  
　EN/ENOなし：d := DINT_TO_BOOL(s);  
　EN/ENO付き：d := DINT_TO_BOOL_E(EN, ENO, s);  
- DINT型からBOOL型への変換イメージ  
　入力(s)が0の場合はFALSE、0以外の数値（例：12345678）の場合はTRUEを出力(d)することを示す矢印付きの図。

### 表
#### 設定データ（引数の内容・種別・データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | DINT |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL |

#### 演算結果（EN/ENO付きファンクションの条件と結果）
| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)から出力されたデータを使用しないようプログラムで処理してください。
