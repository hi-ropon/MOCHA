### テキスト
#### 倍精度実数→符号なしBIN32ビットデータ変換命令 DBL2UDINT(P)
指定した倍精度実数を符号なしBIN32ビットデータに変換する命令です。

- 対応ツールバージョンは1.035M以降のエンジニアリングツールです。

#### 実行条件
- DBL2UDINT命令は立ち上がりエッジで実行されます。
- DBL2UDINTP命令は立ち下がりエッジで実行されます。

### 図
#### ラダー図
- (s)に倍精度実数入力、(d)に符号なしBIN32ビットデータ出力を示すラダー命令の図が掲載されています。

#### FBD/LD図
- EN（実行条件）、s（入力）、d（出力）、ENO（実行結果）を持つ命令ブロックの図が示されています。
