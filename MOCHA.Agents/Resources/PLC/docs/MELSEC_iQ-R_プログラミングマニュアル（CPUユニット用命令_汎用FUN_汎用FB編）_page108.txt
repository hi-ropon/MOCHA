### テキスト
このページでは、倍精度実数の三角関数演算命令とBCD型の三角関数演算命令について説明しています。各命令は、指定した角度のCOS（余弦）、TAN（正接）、SIN（正弦）値の演算や、逆三角関数の演算を行います。
