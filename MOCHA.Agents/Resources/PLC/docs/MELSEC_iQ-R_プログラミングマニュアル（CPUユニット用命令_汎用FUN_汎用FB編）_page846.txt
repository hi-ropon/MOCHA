### テキスト
#### 倍精度実数比較命令（LDED□、ANDED□、ORED□）
倍精度実数をa接点扱いで比較演算する命令について説明しています。  
対応する命令はLDED□、ANDED□、ORED□で、EQ（等しい）、NE（等しくない）、GT（より大きい）、LE（以下）、LT（未満）、GE（以上）などの比較が可能です。  
バージョン1.035M以降のエンジニアリングツールで対応しています。

#### 実行条件
- LDED□、ANDED□、ORED□命令は常時実行されます。

### 図
#### ラダー図
- LD、AND、ORの各命令で、(s1)、(s2)の2つの比較データを入力し、比較演算を行う構成が示されています。

#### FBD/LD命令ブロック
- EN（実行条件）、s1、s2（比較データ）、ENO（実行結果）を持つ命令ブロックの構成図が示されています。
