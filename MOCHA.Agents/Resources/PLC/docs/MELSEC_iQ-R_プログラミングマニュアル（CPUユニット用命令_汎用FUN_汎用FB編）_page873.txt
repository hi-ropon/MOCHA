### テキスト
#### 単精度実数乗算 E*(P)
単精度実数を乗算する命令です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

* バージョン1.035M以降のエンジニアリングツールで対応しています。

#### 実行条件
- E*命令は立ち上がりエッジで実行されます。
- E*P命令は立ち下がりエッジで実行されます。

### 図
#### ラダー図
- ラダー図では、(s1)、(s2)の2つの入力と(d)の出力を持つ命令ブロックが示されています。

#### FBD/LD図
- FBD/LD図では、EN（実行条件）、s1、s2（入力）、ENO（実行結果）、d（出力）端子を持つ命令ブロックが示されています。
- 図中の□にはEMULTI、EMULTIPが入ります。
