#### エラー
エラーコード(SD0)とその内容について説明しています。

### 表
| エラーコード(SD0) | 内容 |
| :--- | :--- |
| 2821H | (s1)から(n)点分のデバイス範囲と、(d)から(n)点分のデバイス範囲が重複しているとき。<br>（(s1)と(d)に同一デバイスを指定した場合を除く。）<br>(s2)から(n)点分のデバイス範囲と、(d)から(n)点分のデバイス範囲が重複しているとき。<br>（(s2)と(d)に同一デバイスを指定した場合を除く。） |

内容は、デバイス範囲の重複に関するエラーコード2821Hの説明であり、特定の範囲指定が重複している場合に発生することを示しています。
