### 表
#### GX Works2命令と互換用命令の対応表
- 表は「GX Works2命令名」「互換用命令名」「使用プログラム（ST、FBD）」「参照先」の4つの列で構成されています。
- 「GX Works2命令名」には元の命令名が記載されています。
- 「互換用命令名」には対応する互換命令名が記載されています。
- 「使用プログラム」欄はST（Structured Text）とFBD（Function Block Diagram）での使用可否を「〇」「×」で示しています。
- 「参照先」には詳細説明があるページ番号と関連命令名が記載されています。

#### 表の内容例
- ANDD< 命令は互換用命令ANDD_LT_M2で、STでは使用不可（×）、FBDでは使用可能（〇）。参照先は217ページのLDD、ANDD、ORD命令。
- ASIN命令はASIN_M2が互換用命令で、ST・FBD共に使用可能（〇）。参照先は1425ページのASIN(_E)。
- BAND命令はBAND_M2が互換用命令で、ST・FBD共に使用可能（〇）。参照先は692ページのBAND(P)(_U)。
- BK+命令はBKPLUS_M2が互換用命令で、STでは使用不可（×）、FBDでは使用可能（〇）。参照先は278ページのBK+(P)(_U)。

#### 備考
- 参照先ページは命令の詳細説明が記載されているマニュアル内のページ番号を示しています。
- 一部の命令はユニット専用命令編など別マニュアルの参照を示しています。
- 使用プログラム欄の「〇」「×」は命令がそのプログラム形式で使用可能かどうかを示しています。

以上のように、本ページはGX Works2の命令とその互換命令の対応関係、使用可能なプログラム形式、参照先を一覧で示した命令リファレンスの一部です。
