### テキスト
#### 倍精度実数比較出力（EDCMP(P)）
本命令は、(s1)で指定した倍精度実数データと、(s2)で指定した倍精度実数データを比較し、その結果（小・一致・大）に応じて(d)、(d)+1、(d)+2のいずれかのビットをONにします。

対応CPUはR00CPU、R01CPU、R02CPUで制約はなく、RnCPU（R00CPU、R01CPU、R02CPUを除く）およびRnENCPUはファームウェアバージョン17以降で使用可能です。エンジニアリングツールはバージョン1.020W以降で対応しています。

#### 実行条件
- EDCMP命令は立下りエッジで実行
- EDCMPP命令は立上りエッジで実行

### 図
#### ラダー図
- (s1)、(s2)、(d)の3つのオペランドを持つ命令ブロックが示されている。
- ST言語では、ENO:=EDCMP(EN,s1,s2,d); または ENO:=EDCMPP(EN,s1,s2,d); と記述。

#### FBD/LD図
- EN、s1、s2を入力、ENO、dを出力とする命令ブロックの図。
