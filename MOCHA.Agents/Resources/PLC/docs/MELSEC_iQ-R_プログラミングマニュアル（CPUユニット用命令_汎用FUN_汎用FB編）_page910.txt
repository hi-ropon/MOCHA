### テキスト
#### 機能
- (s)で指定された角度のTAN（正接）値を計算し、(d)で指定されたデバイス番号に演算結果を格納する。
- 角度はラジアン単位（角度×π÷180）で設定する。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### 注意事項
- (s)で指定する角度がπ/2ラジアン、(3/2)πラジアンの場合、ラジアン値に演算誤差が生じるがエラーは発生しない。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)に指定した値が0、非正規化数、非数、±∞のとき。 |

#### Point
- 角度とラジアン変換については、RAD(P)命令、DEG(P)命令を参照すること。
- 参照ページは941ページ（RAD(P)）、943ページ（DEG(P)）。
