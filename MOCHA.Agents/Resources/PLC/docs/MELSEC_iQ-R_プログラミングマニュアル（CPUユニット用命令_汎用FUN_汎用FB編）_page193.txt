### テキスト
#### 機能
(d)で指定したダイレクトアクセス出力(DY)をパルス出力します。特に(d)にDY0を指定した場合は、SET命令、RST命令を使用した回路と同じ動作になります。  
DELTA命令を使用した回路を、SET/RST命令を使用した回路に置き換えた場合の例が示されています。

#### 注意事項
DELTA(P)命令は、インテリジェント機能ユニットへの立ち上げ実行の指令に使用します。  
出力ユニットを使用した実出力指令には使用できません。

#### エラー
演算エラーはありません。

#### Point
DELTA(P)命令は、高速カウンタユニットのプリセット値の設定などに使用します。  
例として、X20がONしたときに、ベースユニットのスロット0に装着されている高速カウンタユニット（RD62P2）のCH1のプリセットを行うプログラム例が示されています。

### 図
- DELTA命令を使用した回路例  
  X100の接点がONすると、DELTA DY0がパルス出力される回路図。  
  これをSET DY0、RST DY0命令を使った回路に置き換えた例も示されています。

- 動作タイミング波形図  
  X100のON/OFFに連動してDELTA DY0がパルス出力される様子がタイムチャートで示されています。  
  X100がOFFからONに変わるとDELTA DY0がONのパルスを出し、その後OFFに戻る動作を繰り返します。

- プログラム例（ラダー図）  
  X20がONしたときに、DMOVP命令でRD62P2のバッファメモリ0、1のアドレスにプリセット値(0)を格納し、DELTAP命令でプリセット指令を出力するラダー図が示されています。  
  (1) DMOVP K0 U0¥G0  
  (2) DELTAP DY1  
  (3) END
