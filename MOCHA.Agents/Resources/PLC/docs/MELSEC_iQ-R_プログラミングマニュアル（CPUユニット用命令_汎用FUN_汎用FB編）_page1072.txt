### テキスト
#### 時計データの書込み
「DATEWR(P)」命令は、指定したデバイス以降に格納している時計データをCPUユニットの時計素子に書き込みます。

#### 実行条件
- DATEWR命令は立ち下がりエッジで実行されます。
- DATEWRP命令は立ち上がりエッジで実行されます。

### 図
- **ラダー図**  
  時計データ書込み命令のラダー図が示されており、(s)というオペランドが入力として使われています。

- **FBD/LD図**  
  EN（実行条件）とENO（実行結果）を持つ命令ブロックの構造が示されています。

|---------|----------------|
| DATEWR  | 立ち下がりエッジ |
