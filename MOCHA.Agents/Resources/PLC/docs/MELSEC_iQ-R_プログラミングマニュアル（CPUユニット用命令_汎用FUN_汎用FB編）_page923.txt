### テキスト
#### 倍精度実数SIN⁻¹演算（ASIND(P)）
倍精度実数で指定したSIN値から角度を演算する命令です。  
対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）となっています。

#### 実行条件
- ASIND命令はパルス立ち上がりで実行されます。
- ASINDP命令は立ち上がりレベルで実行されます。

#### 注意事項
- ASIND命令はST、FBD/LDでは対応していません。汎用ファンクションのASINを使用してください。（1425ページ参照）

### 図
- ラダー回路図：ASINDP命令の使用例。入力s、出力dを持つ。
- FBD/LD回路図：EN（実行条件）、ENO（実行結果）、s（SIN値入力）、d（演算結果出力）の4端子構成。
