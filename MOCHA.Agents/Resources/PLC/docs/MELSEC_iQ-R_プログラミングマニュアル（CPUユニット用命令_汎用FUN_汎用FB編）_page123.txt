### テキスト・表

#### 通信プロトコル支援機能命令
- ■登録プロトコル実行
  - 命令記号: SP.ECPRTCL
  - 処理内容: エンジニアリングツールの通信プロトコル支援機能で設定したプロトコルを実行する。
  - 参照先: 1177ページ SP.ECPRTCL

#### SLMPフレーム送信命令
- ■SLMPフレーム送信
  - 命令記号: SP.SLMPSND
  - 処理内容: SLMP対応機器に対してSLMPの伝文を送信する。
  - 参照先: 1184ページ SP.SLMPSND

#### ファイル転送機能用命令
- ■FTPクライアントファイル送付
  - 命令記号: SP.FTPPUT
  - 処理内容: (s2)で指定したCPUユニットのファイルを、(s3)で指定したFTPサーバのフォルダパスへ送付する。
  - 参照先: 1191ページ SP.FTPPUT

- ■FTPクライアントファイル取得
  - 命令記号: SP.FTPGET
  - 処理内容: (s2)で指定したFTPサーバのファイルを、(s3)で指定したCPUユニットのフォルダパスに取得する。
  - 参照先: 1201ページ SP.FTPGET
</contents>
