### テキスト
#### Point
ワードデバイスのビット指定をした場合、演算結果を格納するビット指定デバイス以外は変化しないことが説明されています。

### 図
2つのビット列が上下に並んでおり、上がビット指定前、下がビット指定後の状態を示しています。

- 上段（ビット指定前）: D10.FからD10.0までのビットが並んでおり、各ビットの値が0または1で示されています。
- 下段（ビット指定後）: 一部のビットが変化していることが矢印で示されており、特に中央付近のビットが0から1、または1から0に変わっています。
- 変化したビットは枠で囲まれて強調されています。

この図は、ビット指定デバイス以外のビットは変化しないことを視覚的に表現しています。
