### テキスト
#### 39.2 文字列の左側、右側からの抽出
- LEFT(_E): 入力された文字列データの左側から指定文字分を出力します。  
- RIGHT(_E): 入力された文字列データの右側から指定文字分を出力します。

#### プログラミング言語別使用例
- ラダー、FBD/LD  
  - [EN/ENOなし]  
    - 入力s、抽出文字数n、出力dの3つの接続があるブロック図  
  - [EN/ENO付き]  
    - 実行条件EN、出力状態ENO、入力s、抽出文字数n、出力dの接続があるブロック図  
- ST言語  
  - EN/ENOなし  
    - d := LEFT(s,n);  
    - d := RIGHT(s,n);  
  - EN/ENO付き  
    - d := LEFT_E(EN,ENO,s,n);  
    - d := RIGHT_E(EN,ENO,s,n);

### 表
#### 設定データ（内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| --- | --- | --- | --- |
| EN | 実行条件(TRUE: 実行、FALSE: 停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | ANYSTRING_SINGLE |
| n(L) | 抽出文字数指定 | 入力変数 | INT |
| ENO | 出力状態(TRUE: 正常、FALSE: 異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANYSTRING_SINGLE |
