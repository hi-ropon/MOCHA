### テキスト
#### 機能
(d)で指定されたデバイスのBIN32ビットデータと、(s)で指定されたデバイスのBIN32ビットデータのビットごとの論理積を求め、その結果を(d)で指定されたデバイスに格納します。

- ビットデバイスの場合、桁指定による点数以降のビットデバイスは0として演算します。

#### エラー
演算エラーはありません。

### 図
図は3段階のビット列の論理積演算を示しています。

1. 最上段は(d)で指定されたデバイスのビット列（b31からb0までの32ビット）を示し、ビットごとの値が1または0で表されています。
2. 中段は(s)で指定されたデバイスのビット列で、同様に32ビットのビット値が示されています。
3. 最下段は(d)のビット列と(s)のビット列の論理積（AND）を取った結果のビット列で、各ビットごとにAND演算の結果が示されています。

この図により、指定された2つのデバイスの32ビットデータのビットごとの論理積の計算過程と結果が視覚的に理解できます。
