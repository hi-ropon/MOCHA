### テキスト
#### +(P)(_U) [オペランドが3個の場合]
指定した2つのBIN16ビットデータを加算する命令です。対応CPUはRnCPU、RnENCPU（プロセス、二重化、一般、安全）です。

#### 実行条件
- 「+」「+_U」命令はパルス信号で実行されます。
- 「+P」「+P_U」命令は立ち上がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー回路では、3つのオペランド(s1, s2, d)があり、s1とs2が加算され、その結果がdに格納されます。

- **FBD/LD図**  
  ブロック図では、EN（実行条件）、s1、s2（加算対象データ）、d（演算結果格納先）、ENO（実行結果）が示されています。  
  （口にはPLUS、PLUSP、PLUS_U、PLUSP_Uのいずれかが入る）
