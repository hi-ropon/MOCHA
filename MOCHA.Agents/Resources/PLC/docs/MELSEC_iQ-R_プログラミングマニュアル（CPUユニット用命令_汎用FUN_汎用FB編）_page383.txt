### テキスト
#### 機能
- 指定したデバイスから(n1)ワード分のデータ範囲内で、左へ(n2)ワード分シフトさせる。
- 最下位から(n2)ワード分は0Hになる。
- (n1)または(n2)で指定した値が0Hの場合は無処理となる。
- (n1)が(n2)より小さい場合は、指定デバイスから(n1)ワード分のデータはすべて0Hになる。

#### エラー
- 演算エラーは発生しない。

### 図（例）
- (n1)=9、(n2)=4の場合のデータシフト例を示す。
- 元のデータは(d)から(d)+8までの9ワードで、各ワードの16進数値が並ぶ。
- 左へ4ワード分シフトした結果、左端4ワード分は元のデータの5番目以降が移動し、右端4ワードは0Hで埋められている。
- 図中に各ワードのアドレスと値が示され、シフト前後のデータの変化が矢印で表現されている。
