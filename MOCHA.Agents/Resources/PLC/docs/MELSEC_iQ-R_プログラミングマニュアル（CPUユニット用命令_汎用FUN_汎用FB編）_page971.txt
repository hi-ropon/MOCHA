### テキスト
#### 倍精度実数常用対数演算 LOG10D(P)
倍精度実数で指定した値の常用対数（底が10の対数）を演算します。

#### 機能
- (s)で指定された値の常用対数を計算し、(d)で指定されたデバイス番号に演算結果を格納します。
- (s)で指定する値は正の数のみ設定可能で、負の数では演算できません。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となります。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

### 図
- ラダー図：LOG10D命令のラダー表現。EN（実行条件）、s（入力値）、d（出力値）を示す。
- FBD/LD図：関数ブロック図でのEN（入力）、ENO（出力）、s（入力値）、d（出力値）の接続を示す。
- 実行条件図：LOG10D命令は立ち上がりエッジで実行、LOG10DP命令は立ち下がりエッジで実行されることを示す。

- 演算結果が0またはアンダーフロー時は結果0。
- エンジニアリングツールからの入力値設定時は丸め誤差に注意。
