### テキスト
#### べき乗(EXPT(_E))
入力値のべき乗を出力する関数です。

#### ラダー、FBD/LDでの使用例
- EN/ENOなし、EN/ENO付きの2種類のブロック形状が示されています。
- ST言語では、EN/ENOなしの場合は `d:=EXPT(s1,s2);`、EN/ENO付きの場合は `d:=EXPT_E(EN,ENO,s1,s2);` と記述します。

#### 設定データ
##### 引数の内容、種別、データ型
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s1(IN1) | 基数 | 入力変数 | ANY_REAL |
| s2(IN2) | 指数 | 入力変数 | ANY_NUM |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANY_REAL |

#### 機能
- s1に入力されたREAL型/LREAL型データを基数、s2で指定したINT型/DINT型/REAL型/LREAL型データを指数としてべき乗を計算し、結果をdから出力します。

#### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、dから演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となります。その場合はdから出力されたデータを使用しないようプログラム処理を行う必要があります。

### 図
- ラダー/FBD/LDの関数ブロック図：EN/ENOなしとEN/ENO付きの2種類のブロック形状が示されている。
- 演算処理イメージ図：基数(s1)が4.0、指数(s2)が2(INT型)の場合、出力(d)は16.0となる例が示されている。

### 表
#### エラーコードと内容
| エラーコード(SD0) | 内容 |
| :--- | :--- |
| 3402H | (s1)または(s2)で指定したデータが-0、非正規化数、非数、±∞のとき。 |
