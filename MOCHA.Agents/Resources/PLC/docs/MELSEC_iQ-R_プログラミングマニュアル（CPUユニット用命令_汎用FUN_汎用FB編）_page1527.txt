### 表
#### 命令処理時間一覧
命令名ごとに、条件別の処理時間（単位：マイクロ秒）がCPUユニット別に示されています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各種論理演算命令（例：OR=、OR<>、AND=など） |
| 条件 | 無実行時、実行時（導通時/非導通時）などの状態 |
| 処理時間（μs） | R00CPU、R01CPU、R02CPU、その他CPUユニットごとの最小・最大処理時間 |

- R00CPU、R01CPUは同じ処理時間で、無実行時は約0.12544μs、実行時は約0.18816μs。
- R02CPUは無実行時約0.01568μs、実行時約0.02352μs。
- 左記以外のCPUユニットは無実行時約0.00392μs、実行時約0.00588μs。

この表は命令の種類と実行条件に応じた各CPUユニットの処理時間の違いを詳細に示しており、性能評価や最適化の参考資料として利用されます。
</contents>
