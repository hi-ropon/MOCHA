### テキスト
#### 注意事項
- PHASE命令は母線への直接接続のみ可能であり、他の命令との並列接続はできません。
- PHASE命令にFB/FUNを接続する場合は、FB/FUNのEN端子への接続のみ可能です。
- PHASE命令に接続したインラインST内で制御構文（IF文、CASE文、FOR文、WHILE文、REPEAT文）を使用する場合、最大127階層までとしてください。

#### エラー
- 演算エラーはありません。

### 図
- PHASE命令の正しい接続例と誤った接続例を示すラダー図
  - 正しい例：PHASE命令が母線に直接接続されている図
  - 誤った例：PHASE命令が他の命令と並列接続されている図に×印
- PHASE命令にFB/FUNを接続する場合の接続例と誤った接続例
  - 正しい例：PHASE命令の出力がFB/FUNのEN端子に接続されている図
  - 誤った例：PHASE命令がFB/FUNの他端子に接続されている図に×印
- インラインST内での制御構文使用例のブロック図
  - PHASE命令に接続されたインラインST内で、IF文のネストが示されており、最大127階層まで可能であることを注記
