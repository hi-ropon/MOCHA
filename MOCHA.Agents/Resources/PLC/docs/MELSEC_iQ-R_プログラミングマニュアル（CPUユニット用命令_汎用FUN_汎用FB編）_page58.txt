### テキスト
#### 同一デバイスのPLS命令を使用した場合
PLS命令は、実行指令のOFF→ON時に指定デバイスをONにし、それ以外の変化（OFF→OFF、ON→ON、ON→OFF）の場合は指定デバイスをOFFにします。
同一デバイスのPLS命令を1スキャンに複数回実行した場合、各PLS命令の実行指令がOFF→ONのときに指定デバイスをONにし、それ以外のときは指定デバイスをOFFにします。
このため、同一デバイスのPLS命令を1スキャンに複数回実行した場合、PLS命令でONしたデバイスが1スキャン内でONしないことがあります。

### 図
#### ラダー図
- X0とX1の入力がそれぞれPLS命令のトリガーとなり、M0が出力される構成を示しています。

#### タイミングチャート
- X0とX1のON/OFFタイミングが異なる場合の動作を示しています。
- X0、X1、M0の状態変化が時間軸に沿って示されており、M0がON/OFFするタイミングが詳細に説明されています。

##### タイミング詳細
1. X0がOFF→ONのため、M0はONになる。
2. X1がOFF→ON以外のため、M0はOFFになる。
3. X0がOFF→ON以外のため、M0はOFFのまま。
4. X1がOFF→ONのため、M0はONになる。

この動作から、同一スキャン内で複数回PLS命令を実行すると、指定デバイスのON状態が期待通りに維持されない場合があることがわかります。
