### テキスト
#### 機能
(s1)で指定された単精度実数と、(s2)で指定された単精度実数の減算を行い、その結果を(d)で指定されたデバイスに格納します。

- (s1)、(s2)、(d)には0、2^-126 ≤ |指定値(格納値)| < 2^128 の範囲の値が指定および格納可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

#### 注意事項
- エンジニアリングツールからの入力値設定時に丸め誤差が発生する可能性があるため、注意が必要です。

### 図
- 単精度実数の減算処理のイメージ図があり、(s1)+1、(s1)、(s2)+1、(s2)、(d)+1、(d)の各デバイスに対応した単精度実数の減算結果が示されています。

| 3403H | (d)が下記範囲を超えるとき（オーバーフローが発生したとき）。<br>|(d)| < 2^128 |
