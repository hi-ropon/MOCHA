### テキスト・表

#### SFC制御命令
- ■ステップ活性チェック  
  指定ステップの活性/非活性を演算結果として判定する命令群。  
  命令記号にはLD、LDI、AND、ANI、OR、ORIがあり、それぞれa接点命令、b接点命令、接点直列・並列接続命令に対応。  
  参照先は1267ページ。

| 命令記号           | 処理内容                                                                                     | 参照先                         |
|--------------------|----------------------------------------------------------------------------------------------|-------------------------------|
| LD [SD/BL□￥SD]    | 指定ステップの活性/非活性を演算結果とします。（a接点命令）                                   | 1267ページ LD、LDI、AND、ANI、OR、ORI [SD/BL□￥SD] |
| LDI [SD/BL□￥SD]   | 指定ステップの活性/非活性を演算結果とします。（b接点命令）                                   | 同上                          |
| AND [SD/BL□￥SD]   | 指定ステップの活性/非活性と、それまでの演算結果のAND演算を演算結果とします。（a接点直列接続命令） | 同上                          |
| ANI [SD/BL□￥SD]   | 指定ステップの活性/非活性と、それまでの演算結果のAND演算を演算結果とします。（b接点直列接続命令） | 同上                          |
| OR [SD/BL□￥SD]    | 指定ステップの活性/非活性と、それまでの演算結果のOR演算を演算結果とします。（a接点1個の並列接続命令） | 同上                          |
| ORI [SD/BL□￥SD]   | 指定ステップの活性/非活性と、それまでの演算結果のOR演算を演算結果とします。（b接点1個の並列接続命令） | 同上                          |

#### 活性ステップ一括読出し
- 指定ブロックのステップ活性状態をビット情報として指定デバイスへ読み出す命令群。  
- 命令記号はMOV、MOVP、DMOV、DMOVP、BMOV、BMOVPがあり、16ビット単位や32ビット単位、ワード単位での読み出しに対応。  
- 参照先は1271～1277ページ。

| 命令記号           | 処理内容                                                                                     | 参照先                         |
|--------------------|----------------------------------------------------------------------------------------------|-------------------------------|
| MOV [K4SD/BL□￥K4SD]   | 指定ブロックのステップ活性状態を、ビット情報として指定デバイスへBIN16ビットデータ単位で読み出します。 | 1271ページ MOV(P) [K4SD/BL□￥K4SD] |
| MOVP [K4SD/BL□￥K4SD]  | 同上                                                                                         | 同上                          |
| DMOV [K8SD/BL□￥K8SD]  | 指定ブロックのステップ活性状態を、ビット情報として指定デバイスへBIN32ビットデータ単位で読み出します。 | 1274ページ DMOV(P) [K8SD/BL□￥K8SD] |
| DMOVP [K8SD/BL□￥K8SD] | 同上                                                                                         | 同上                          |
| BMOV [K4SD/BL□￥K4SD]  | 指定ブロックのステップ活性状態を、指定ステップから指定ワード分一括で読み出します。               | 1277ページ BMOV(P) [K4SD/BL□￥K4SD] |
| BMOVP [K4SD/BL□￥K4SD] | 同上                                                                                         | 同上                          |
