### テキスト
#### BIN32ビットデータテーブルソート2（DSORTTBL2(_U)）
- 対応CPUはR00CPU、R01CPU、R02CPU（ファームウェアバージョン"08"以降）、およびRnENCPU（ファームウェアバージョン"40"以降）で使用可能。
- エンジニアリングツールのバージョンは1.050C以降が必要。
- 本命令は、(n1)×(n2)点のBIN32ビットデータテーブル（並び替え元）を、(n3)列のデータを基準にしてデータ行を昇順または降順に並び替え、指定した((n1)×(n2))点のBIN32ビットデータテーブル（並び替え後）に格納する。

### 図（表）
- 左側の表は並び替え前のBIN32ビットデータテーブルで、(n1)行、(n2)列のデータが示されている。
- 右側の表は並び替え後のテーブルで、(n3)列のデータを基準に昇順/降順に並び替えられている。
- 赤枠で囲まれた部分が並び替えの基準となるデータ列であり、並び替え前後でその列の値に基づいて行全体が入れ替わっていることがわかる。
- 各セルには(s)または(d)で始まるデータ名と数値が記載されている。

### 図（ラダー、ST、FBD/LD）
- ラダー図：DSORTTBL2命令の入力パラメータs, n1, n2, d, n3の接続例を示す。
- ST言語：命令の呼び出し例として、ENO:=DSORTTBL2(EN,s,n1,n2,n3,d);およびENO:=DSORTTBL2_U(EN,s,n1,n2,n3,d);が示されている。
- FBD/LD：命令ブロックの入力（EN, s, n1, n2, n3）と出力（ENO, d）端子の配置を示す。
