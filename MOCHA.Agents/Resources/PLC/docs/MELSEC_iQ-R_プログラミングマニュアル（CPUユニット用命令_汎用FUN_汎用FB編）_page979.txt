### テキスト
#### 倍精度実数最小値検索 EＤＭＩＮ(P)
倍精度実数のブロックデータの最小値を検索する命令です。

#### 実行条件
- EDMIN命令は立ち上がりエッジで実行されます。
- EDMINP命令は立ち下がりエッジで実行されます。

#### 設定データ
##### 内容、範囲、データ型
- (s)：最小値を検索する倍精度実数または、倍精度実数が格納されている元頭デバイス。範囲は0以上2^1022未満の倍精度実数。
- (d)：検索結果を格納する先頭デバイス。dからd+3は最小値、d+4は位置、d+5は個数を格納。
- (n)：倍精度実数ブロックデータの点数。0～65535の符号なしBIN16ビット。
- EN：実行条件（ビット）
- ENO：実行結果（ビット）

##### 使用可能デバイス
- (s)はワードデバイスのT, ST, C, D, W, SD, SW, FD, R, ZR, RDなどが使用可能。
- (d)はワードデバイスのT, ST, C, D, W, SD, SW, FD, R, ZR, RDなどが使用可能。
- (n)はビット、ワード、ダブルワード、間接指定、定数など幅広く使用可能。

### 図
- ラダー形式：命令はST命令で、パラメータとして(s)(d)(n)を指定し、ENO出力があります。
- FBD/LD形式：EN入力、s入力、n入力があり、d出力、ENO出力があるブロック図。
