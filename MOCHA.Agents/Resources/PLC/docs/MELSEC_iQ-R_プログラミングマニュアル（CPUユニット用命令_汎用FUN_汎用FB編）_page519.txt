### テキスト
#### 機能
- (s)で指定されたBIN16ビットデータを、(d)で指定されたデバイスへ転送する。
- (s)がビットデバイスの場合、桁指定で指定されたビットが転送対象となる。転送対象外のビットは0として転送される。

#### エラー
- 演算エラーは存在しない。

### 図
- 上段の図は、(s)で指定された16ビットのビットデータのビット配置を示している。b15からb0までのビットが並び、b7からb0の一部ビットに1や0が設定されている。
- 下段の図は、(d)で指定されたデバイスのビット配置を示し、(s)のビット配置から転送されるビットがどのように配置されるかを示している。b8からb0までのビットがあり、(s)のb7からb0のビットが対応する位置に転送されている。
- 注釈として、(s)が16ビット未満の場合は、0が指定されているものとして転送されることが記載されている。
