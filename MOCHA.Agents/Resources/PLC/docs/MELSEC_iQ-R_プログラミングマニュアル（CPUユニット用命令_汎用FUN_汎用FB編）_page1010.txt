### テキスト
#### 機能
- (s1)で指定されたデバイス入力のOFF→ONの回数を、(s2)で指定された時間だけカウントし、その結果を(d)で指定されたデバイスに格納する。
- SPD命令による計測が完了すると、再度0から計測を開始する。
- SPD命令による計測を中止する場合は、実行指令をOFFにする。
- (s2)=0の場合は無処理となる。

#### エラー
- 演算エラーはない。

#### Point（注意点）
- SPD命令は引数デバイスのデータをCPUユニット内部のワークエリアに登録し、カウント動作はシステム割込みで行う。登録されたデバイスデータは実行指令をOFFまたはSTOP→RUNでクリアされる。
- カウント可能なパルスはONおよびOFF時間がCPUユニットの割込み間隔（1ms）以上必要。
- SPD命令は全プログラムで6回まで使用可能。7回目以降は無処理。
- SPD命令による計測中（指令入力がON中）は設定値の変更不可。変更する場合は指令入力をOFFにしてから行う。

### 図
- 横軸は時間(ms)で、計測開始から(s2)ms間隔で2回計測結果を(d)に格納するタイミングが示されている。
- 実行指令はOFFからONに切り替わり、その後ONのまま維持されている。
- (s1)の信号はOFF→ONを繰り返すパルス波形で、ON/OFFの繰り返し回数をカウントしている様子を表現。
- 計測結果は(s2)の時間経過ごとに(d)に格納されることが示されている。
