### テキスト
#### 使用可能デバイス
オペランドごとに使用可能なビット、ワード、ダブルワード、間接指定、定数、その他のデバイスが表形式で示されています。  
- (U)、(d1)、(s1)、(s2)、(s3)、(s4)、(d2)の各オペランドに対して、どのデバイスが使用可能か「○」「－」で示されています。  
- 例えば、(d1)はワードのT,ST,C,D,W,SD,SW,FD,R,ZR,RDが使用可能で、間接指定のK,Hが使用可能です。

#### コントロールデータ(d1)
- 「使用用途設定エリア」のビットb0、b2の意味と設定内容が説明されています。  
- b0は対象がファイルかフォルダかを指定（0:ファイル指定、1:フォルダ指定）。  
- b2はコピー対象の範囲指定（0:指定フォルダごとコピー、1:指定フォルダ内の全ファイル・サブフォルダをコピー）。  
- b2が0の場合は、指定フォルダをフォルダごと（ファイル、サブフォルダ含む）コピーする動作となります。

### 図（フォルダ構造の説明）
- ルートフォルダ「$MELPRJ$」の下に「folder1」「folder2」があり、さらに「folder1」の中に「folder1a」が存在。  
- 「folder1a」には「user1.csv」「user2.csv」がある。  
- コピー先フォルダ「folder2」には「folder1」「folder1a」がコピーされ、元のフォルダ構成を保持したままコピーされることを示しています。  
- 図中に「指定したフォルダ(s2)」「コピー先のフォルダ(s4)」のラベルが付いています。
