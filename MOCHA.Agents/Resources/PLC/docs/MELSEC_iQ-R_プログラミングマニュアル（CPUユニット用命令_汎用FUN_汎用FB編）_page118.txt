<overview>
本ページはCPUユニット用命令の応用命令に関する説明で構成されています。ページ全体は主に表形式で、各命令の命令記号、処理内容、参照先が整理されています。内容は以下のセクションに分かれています。
- 時刻比較：指定した時刻データと現在時刻の比較に関する命令群
- 時刻比較出力：複数の時刻データを比較し結果を出力する命令
- 時刻帯域比較出力：時刻データの帯域に対する比較と出力命令
- 拡張時計データの読出し：CPU内部の時計素子から詳細な時刻データを読み出す命令
- 拡張時計データの加算：時間データの加算を行う命令
各セクションは命令の機能説明と、必要に応じて時刻データの構成要素を示す表や図が含まれています。
</overview>

#### ■時刻比較出力
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| TCMP | (s1)、(s2)、(s3)で指定した比較する時刻データと(s4)で指定した時刻データを比較し、その結果（小、一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONします。 | 1093ページ TCMP(P) |
| TCMPP | 同上 | 同上 |

#### ■時刻帯域比較出力
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| TZCP | 下限値(s1)の時刻データと上限値(s2)の時刻データの帯域に対し、比較する時刻データ(s3)で比較し、その結果（下、帯域内、上）に応じて(d)、(d)+1、(d)+2のいずれかがONします。 | 1095ページ TZCP(P) |
| TZCPP | 同上 | 同上 |

#### ■拡張時計データの読出し
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| S.DATERD、SP.DATERD | CPU内部の時計素子から1/1000秒を含む時計データを読み出します。 | 1097ページ S(P).DATERD |

- 時計データの構成（(d)からのオフセット）
  - (d): 年
  - (d)+1: 月
  - (d)+2: 日
  - (d)+3: 時
  - (d)+4: 分
  - (d)+5: 秒
  - (d)+6: 曜日
  - (d)+7: 1/1000秒

#### ■拡張時計データの加算
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| S.DATE+、SP.DATE+ | 時間データを加算します。 | 1099ページ S(P).DATE+ |

- 時間データの加算イメージ  
  (s1)の時間データ（hour, minute, second, 1/1000 second）と(s2)の時間データを加算し、(d)に結果を格納します。
