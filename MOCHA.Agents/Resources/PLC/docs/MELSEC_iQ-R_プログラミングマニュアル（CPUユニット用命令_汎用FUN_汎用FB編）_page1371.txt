### テキスト
#### 32.42 REAL型→LREAL型変換
REAL_TO_LREAL(_E)関数は、REAL型データをLREAL型データへ変換します。

##### 機能
- (s)に入力されたREAL型のデータをLREAL型に変換し、(d)から出力します。
- (s)の入力値はREAL型のデータ値です。
- REAL型データは32ビット単精度で処理され、有効桁数は約7桁です。
- 整数値が-16777216～16777215の範囲を超えた場合、変換値に誤差（丸め誤差）が生じます。

##### 演算結果
1. EN/ENOなしファンクション
- 演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション
- 実行条件と演算結果は以下の通りです。

| 実行条件 | 演算結果 |  
| --- | --- |  
| EN | ENO | (d) |  
| TRUE（演算実行） | TRUE | 演算出力値 |  
| FALSE（演算停止） | FALSE*1 | 不定値 |  

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)のデータを使用しないようプログラム処理を行う必要があります。

### 図
- ラダー図、FBD/LD図：REAL_TO_LREAL関数の入力(s)、出力(d)、およびEN/ENO端子の有無を示すブロック図。
- ST言語：EN/ENOなしの場合は「d:=REAL_TO_LREAL(s);」、EN/ENO付きの場合は「d:=REAL_TO_LREAL_E(EN,ENO,s);」という記述例。

### 表
#### 設定データ（引数の内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |  
| --- | --- | --- | --- |  
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |  
| s(IN) | 入力 | 入力変数 | REAL |  
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |  
| d | 出力 | 出力変数 | LREAL |
