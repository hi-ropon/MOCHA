### 表
#### 命令名ごとの基本ステップ数とサブセット可否一覧
この表は、各命令名に対して「基本ステップ数」と「サブセット処理の可否」を示しています。

- **命令名**: 各命令の名称が列挙されています。例として「DBSELECT」「DBDELETE」「RSET」など多数の命令が含まれています。
- **基本ステップ数**: 各命令の基本的なステップ数が数字で示されています。例えば「DBSELECT」は8、「DBDELETE」は6、「RSET」は2などです。一部の命令は「2+ファイル名の文字数」など、文字数に依存するステップ数もあります。
- **サブセット可否**: すべての命令に対して「―」と記載されており、サブセット処理は不可であることを示しています。

この表は命令の処理負荷や構造を理解するための指標として利用されることが想定されます。
</contents>
