### テキスト
- 入出力データエリアで「ライト」となっているデータは、ユーザーのプログラムで書き込む必要があること。
- 「リード」となっているデータはユーザープログラムで読み出して使用すること。
- 「リード/ライト禁止」や「リード」となっているデータに書き込みを行うと正常な演算ができなくなるため、絶対に行わないこと。
- 初期状態から制御を開始する場合は、プログラムでデータクリアが必要。
- 手動モードで手動操作量(MV_MAN)の出力を行う場合でも、PIDCONT(P)命令を毎ステップ実行しなければノンプレスト機能は動作しない。
- PIDCONT(P)命令は、測定値(PV)の取込みを行うA/D変換ユニットと操作量(MV)の出力を行うD/A変換ユニットが正常なときのみ実行されるよう、各ユニットのREADY信号でインタロックを行う必要がある。
- 各ユニットが異常時に実行すると、正常な測定値の取込みや操作量の出力ができず、PID演算も正常に行えない。

### 図（命令のタイミング説明）
- PIDCONT命令の実行タイミングを示す図があり、3つの信号が順に並んでいる。
  1. 制御指令
  2. A/D変換ユニットのREADY信号
  3. D/A変換ユニットのREADY信号
