### 表
#### 命令処理時間一覧表
本表は各命令名ごとに、処理条件別の処理時間（マイクロ秒単位）を示しています。処理時間はCPUユニット別に「R00CPU、R01CPU」「R02CPU」「左記以外のCPUユニット」の3区分に分かれ、それぞれ最小値と最大値が記載されています。

##### 命令名と条件
- 命令名は「ORDT<>」「ORDT>」「ORDT<=」「ORDT<」「ORDT>=」「LDTM=」「LDTM<>」「LDTM>」「LDTM<=」「LDTM<」「LDTM>=」の11種類。
- 条件は「無実行時」「指定の日付と比較:導通時」「指定の日付と比較:非導通時」「現在の日付と比較:導通時」「現在の日付と比較:非導通時」「指定の時刻と比較:導通時」「指定の時刻と比較:非導通時」「現在の時刻と比較:導通時」「現在の時刻と比較:非導通時」など、命令によって異なる比較条件が設定されています。

##### CPUユニット別処理時間
- 「R00CPU、R01CPU」：最小0.18816μs（無実行時）から最大17.400μs程度まで幅がある。
- 「R02CPU」：最小0.02352μs（無実行時）から最大16.400μs程度まで。
- 「左記以外のCPUユニット」：最小0.00588μs（無実行時）から最大15.900μs程度まで。

##### 処理時間の傾向
- 無実行時は全CPUユニットで非常に短い処理時間。
- 指定日付や時刻との比較時は導通時・非導通時で処理時間に若干の差がある。
- 現在の日付や時刻との比較時は処理時間が最も長くなる傾向がある。
