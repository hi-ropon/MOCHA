### テキスト
#### ファイル名／フォルダ名 (オペランド: s2)
ファイル名やフォルダ名を指定する際のルールを説明しています。

- ファイル名は拡張子まで指定可能。
- フォルダパス＋ファイル名（拡張子含む）は最大253文字以内。
- フォルダパスは最大244文字以内（区切り文字は含まない）。
- ファイル名やフォルダパスは区切り文字を除いて1文字以上指定する必要がある。
- 文字列の末尾や区切り文字直前に半角ピリオドや半角スペースを指定しない。
- フォルダパスの階層数は10以内。

##### ファイルを指定する場合の例
- 例示文字列: "/folder1/user1/user1.csv"
- 最大253文字
- 区切り文字は「/」または「\」
- 省略可能な部分があり、省略時は最大252文字となる。

##### フォルダを指定する場合の例
- 例示文字列: "/folder1/user1/subfolder1/"
- 最大244文字
- 区切り文字は「/」または「\」
- 省略可能な部分があり、省略時は最大243文字となる。

#### 移動先フォルダパス (オペランド: s4)
移動先のフォルダパス指定に関する説明。

- フォルダパスは244文字以内（区切り文字は含まない）。
- 区切り文字は「/」または「\」を使用。
- 文字列の末尾や区切り文字直前に半角スペースや半角ピリオドを指定しない。
- 階層数は10以内。

##### 例示文字列
- "/folder1/user1/database/"
- 最大244文字
- 省略可能な部分があり、省略時は最大243文字となる。
