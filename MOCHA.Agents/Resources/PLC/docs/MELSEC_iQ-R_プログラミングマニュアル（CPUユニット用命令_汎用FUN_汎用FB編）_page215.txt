#### 設定データ
内容、範囲、データ型について説明しています。

- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、表に示されたデータ型の安全デバイスおよび安全ラベルのみ使用可能である旨の注意書きがあります。

#### 内容、範囲、データ型

| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
