### テキスト
#### チェックコード命令（CCD(P)）
- 対応CPUはR00CPU、R01CPU、R02CPUで制約なし。RnCPU（R00CPU、R01CPU、R02CPU除く）はファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降が必要。
- 命令は、(s)～(s)+(n)-1に格納されたデータの加算データと水平パリティを算出し、加算データを(d)に、水平方リティを(d)+1に格納する。

#### 命令表記
- ラダー図：3つのオペランド(s)(d)(n)を持つ命令ブロック。
- ST（構造化テキスト）：`ENO:=CCD(EN,s,n,d);` または `ENO:=CCDP(EN,s,n,d);`
- FBD/LD図：EN（実行条件）、s（先頭デバイス）、n（データ数）、d（格納先）、ENO（実行結果）を持つブロック。
