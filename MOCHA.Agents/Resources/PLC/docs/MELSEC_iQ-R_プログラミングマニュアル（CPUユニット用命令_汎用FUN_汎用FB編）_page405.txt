### テキスト
#### 機能
- (d)で指定したデバイスから、倍精度実数(n1)点分のデータ範囲内で右へ(n2)点分シフトする。
- シフト後、(s)から(n2)点分は空きデータ部分に格納される。
- (s)に定数を指定した場合、シフト後の最上位から(n2)点分は指定した値になる。
- (n2)で指定した値が0の場合は無処理となる。

#### 例
- (n1)=4、(n2)=2の場合の動作例を示す。

### 図
- 図はデータのシフト動作を示している。
- 上段に元のデータ範囲(d)+15～(d)の倍精度実数が並び、下段に右へ2点シフトした結果が示されている。
- (n2)点分右へシフトし、空いた左側の部分に元のデータの一部をコピーしている様子が矢印で表現されている。
