<overview>
本ページは主に命令コードとその処理内容を一覧形式で示した表で構成されています。ページ全体は複数のセクションに分かれており、各セクションごとに命令の種類（アナンシェータ、デバイスのセット・リセット、立上り・立下り出力、ビットデバイス出力反転、ダイレクト出力のパルス化、シフト命令、マスタコントロール命令、フェーズ処理命令）がまとめられています。テキストは命令コード、処理内容、参照先ページの3列からなる表形式で記載されています。図やグラフは含まれていません。

### テキスト
#### アナンシェータ
- OUTF: OUTF命令までの演算結果を指定のアナンシェータへ出力する。（参照177ページ）

#### デバイスのセット（アナンシェータを除く）
- SET: 指定ビットをONにする。（参照178ページ）

#### デバイスのリセット（アナンシェータを除く）
- RST: 指定デバイスを0（OFF）にする。タイマやカウンタは現在値を0にし、接点やコイルはOFFにする。（参照180ページ）

#### アナンシェータのセット、リセット
- SET F: 指定されたアナンシェータをONにする。（参照182ページ）
- RST F: 指定されたアナンシェータをOFFにする。（参照184ページ）

#### 立上り、立下り出力
- PLS: PLS指令のOFF→ON時に指定デバイスを1スキャンONさせる。（参照186ページ）
- PLF: PLF指令のON→OFF時に指定デバイスを1スキャンONさせる。（参照188ページ）

#### ビットデバイス出力反転
- FF: 指定されたデバイス状態を反転する。（参照190ページ）

#### ダイレクト出力のパルス化
- DELTA: 指定されたダイレクトアクセス出力（DY）をパルス出力する。（参照192ページ）
- DELTAP: 同上（参照192ページ）

#### シフト命令
##### ビットデバイスシフト
- SFT: 1つ若いデバイスのON/OFF状態を指定したデバイスにシフトし、1つ若いデバイスをOFFにする。（参照194ページ）
- SFTP: 同上（参照194ページ）

#### マスタコントロール命令
##### マスタコントロールのセット、リセット
- MC: マスタコントロールを開始する。（参照196ページ）
- MCR: マスタコントロールを終了する。（参照196ページ）

#### フェーズ処理命令
##### フェーズ処理
- PHASE: フェーズ処理を開始する。（参照201ページ）
- PHASECHG: 実行中のフェーズを終了し、指定したフェーズへ移行する。（参照203ページ）
- PHASEEND: 実行中のフェーズを終了する。（参照205ページ）
</contents>
