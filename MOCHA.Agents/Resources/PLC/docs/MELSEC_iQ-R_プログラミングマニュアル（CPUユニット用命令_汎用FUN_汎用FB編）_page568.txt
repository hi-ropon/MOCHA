### テキスト
#### 割込みプログラムマスク(IMASK)
指定された割込みポインタ番号の割込みプログラムを実行許可状態または実行禁止状態にします。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| IMASK | 常時実行 |

### 図
#### ラダー図
- (s)という割込みマスクデータまたは割込みマスクデータが格納されている先頭デバイスを入力として、ENOにIMASK(EN,s)の結果を出力する構成。

#### FBD/LD図
- EN入力に対してIMASK命令ブロックがあり、ENO出力がある。
