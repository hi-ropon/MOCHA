### テキスト
#### 23.1 データ収集用命令
「DATATRG」命令は、レコーディング機能において指定した設定No.に対してデータを収集するための命令です。

##### 対応CPUユニットと使用条件
- R00CPU、R01CPU、R02CPUでは使用不可。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン"50"以降で使用可能。対応製造情報に制約あり。
- RnSFCPU（一般）はファームウェアバージョン"23"以降で使用可能。
- エンジニアリングツールのバージョン制限も記載あり。

##### 命令の概要
- 指定した設定No.（1～4）に対してデータ収集を行う。
- 実行条件により命令が動作する。

### 図
#### ラダー図
- (s)という設定No.を入力し、命令「DATATRG」を実行するラダー図が示されている。
- ST言語では「ENO:=DATATRG(EN,s);」と記述。

#### FBD/LD図
- EN（実行条件）入力、ENO（実行結果）出力、s（設定No.）入力のブロック図。
