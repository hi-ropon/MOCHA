### 表
#### 基本ステップ数とサブセット処理の可否一覧
命令名ごとに以下の2つの項目が表形式でまとめられています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 比較命令や加減算命令などの命令コードが列挙されています。例: BKCMP=_U、DBKCMP<>、+ (s) (d) など |
| 基本ステップ数 | 各命令の基本的なステップ数が数字で示されています。多くは3〜6の範囲内です。 |
| サブセット可否 | サブセット処理が可能かどうかを示す欄で、「〇」または「―」で表記されています。〇は可、―は不可を意味します。 |

命令名は比較命令（BKCMP、DBKCMP）と加減算命令（+、-、D+など）が中心で、比較命令は基本ステップ数が5または6でサブセット不可、加減算命令は基本ステップ数が3または4でサブセット可能となっています。
