<overview>
本ページは「機能」「エラー」の2つのセクションで構成されています。  
テキスト部分では、デバイスのビット状態をチェックし、結果を別のデバイスに格納する機能の説明があります。  
図部分では、ビットの状態を示すビット列とその対応関係を示した図が掲載されています。  
表部分では、エラーコードとその内容を示すエラー一覧表が掲載されています。
</overview>

<contents>
### テキスト
#### 機能
(s)で指定したデバイスの(n)ビットの状態がONかOFFかをチェックし、その結果を(d)で指定したデバイスへ格納します。  
結果がONであれば(d)はON、OFFであれば(d)はOFFとなります。

### 図
ビット列の図があり、b15からb0までの16ビットが示されています。  
ビット番号の下に対応する数値（15から0）が記載されています。  
(n)はビット位置を示し、例として(n)=5はOFF(0)、(n)=8はON(1)を表しています。  
(s)はビット列全体の範囲を示し、(d)は結果を格納するデバイスを示しています。

### 表
#### エラー
| エラーコード (SD0) | 内容                         |
|--------------------|------------------------------|
| 3405H              | (n)に指定した値が0～15の範囲外のとき。 |

このエラーは、(n)に指定したビット位置が0から15の範囲外の場合に発生します。
</contents>
