### テキスト
#### 符号なしBIN32ビットデータ→倍精度実数変換
命令「UDINT2DBL(P)」は、符号なしBIN32ビットデータを倍精度実数に変換します。対応エンジニアリングツールのバージョンは1.035M以降です。

##### 実行条件
- UDINT2DBL命令は立ち上がりエッジで実行されます。
- UDINT2DBLP命令は立ち下がりエッジで実行されます。

##### 機能
- オペランド(s)で指定した符号なしBIN32ビットデータを倍精度実数に変換し、(d)に格納します。

##### エラー
- 演算エラーはありません。
