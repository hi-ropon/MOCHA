### テキスト
#### ファイル名/フォルダ名（s2）
ファイルやフォルダを指定する際のパス指定ルールについて説明しています。

- ファイル名は拡張子まで指定し、フォルダ名はフォルダパスを指定する。  
- ファイル名＋フォルダ名（拡張子含む）は最大253文字以内、フォルダパスは244文字以内。  
- ファイル名やフォルダパスは区切り文字（"/"または"¥"）を含まず、1文字以上の文字列を指定。  
- 文字列の末尾に区切り文字を指定しないこと。  
- 階層数は10以内。  
- ファイル指定時の例として、"/folder1/user1/user1.csv"の構成を図示。  
  - 最大253文字  
  - 区切り文字は"/"または"¥"  
  - 省略可能な部分がある場合は最大252文字になる。  
- フォルダ指定時の例として、"/folder1/user1/subfolder1/"の構成を図示。  
  - 最大244文字  
  - 区切り文字は"/"または"¥"  
  - 省略可能な部分がある場合は最大243文字になる。
