### テキスト
#### 機能
指定されたデバイス範囲(sからn点まで)のBIN32ビットデータから最小値を検索し、(d)、(d)+1で指定されたデバイスに最小値を格納する機能について説明している。  
具体的には、(s)で指定されたデバイスから検索を開始し、最初に検出した最小値が格納されているデバイス番号が(s)から何点目かを(d)+2に格納し、最小値の個数を(d)+3に格納する。

#### エラー
演算エラーは発生しない。

### 表
| デバイスアドレス | BIN値（2進数） |
|------------------|----------------|
| (s)+1, (s)       | 22342001       |
| (s)+3, (s)+2     | 37282010       |
| (s)+5, (s)+4     | 22342001       |
| (s)+7, (s)+6     | 59872019       |

| デバイスアドレス | 内容       |
|------------------|------------|
| (d)              | 22342001   |
| (d)+1            | 1（最小値）|
| (d)+2            | 1（位置）  |
| (d)+3            | 2（個数）  |

### 図（説明用）
- 左側に複数のデバイスアドレスとBIN値が縦に並び、右側に(d)から始まるデバイスアドレスと最小値、位置、個数が表形式で示されている。
- 矢印で左側のデバイス群から右側のデバイス群へ最小値の検索結果が格納される流れを示している。
