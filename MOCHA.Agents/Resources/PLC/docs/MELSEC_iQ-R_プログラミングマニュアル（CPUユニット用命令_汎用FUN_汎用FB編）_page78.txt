<overview>
本ページはCPUユニット用命令の基本命令に関する説明で構成されています。ページ全体は主に表形式で命令コードごとの処理内容と参照先を示しています。内容は以下のセクションに分かれています。
・符号なしBIN32ビットデータから符号なしBIN16ビットデータへの変換命令
・BIN16ビット/32ビットデータとBIN16ビット/32ビットグレイコードデータ間の変換命令
・BIN16ビットデータブロックからBCD4桁データ変換ブロック命令
テキストは命令名、処理内容、参照先の3列の表で整理されており、グレイコード変換に関しては簡単な図解も含まれています。

### テキスト
#### 符号なしBIN32ビットデータ→符号なしBIN16ビットデータ変換
- **UDINT2UINT / UDINT2UINTP**  
  指定した符号なしBIN32ビットデータを符号なしBIN16ビットデータに変換し、(d)へ格納します。  
  参照先：454ページ UDINT2UINT(P)

#### BIN16ビット/32ビットデータ→BIN16ビット/32ビットグレイコードデータ変換
- **GRY / GRYP**  
  指定したBIN16ビットデータ（-32768～32767）をBIN16ビットグレイコードデータに変換します。  
  処理イメージ：  
  (s) BIN → (d) Gc（グレイコード）  
  参照先：458ページ GRY(P)(_U)

- **GRY_U / GRYP_U**  
  指定したBIN16ビットデータ（0～65535）をBIN16ビットグレイコードデータに変換します。  
  処理イメージは上記と同様。

- **DGRY / DGRYP**  
  指定したBIN32ビットデータ（-2147483648～2147483647）をBIN32ビットグレイコードデータに変換します。  
  処理イメージ：  
  (s)+1, (s) BIN → (d)+1, (d) Gc（グレイコード）  
  参照先：460ページ DGRY(P)(_U)

- **DGRY_U / DGRYP_U**  
  指定したBIN32ビットデータ（0～4294967295）をBIN32ビットグレイコードデータに変換します。  
  処理イメージは上記と同様。

#### BIN16ビット/32ビットグレイコードデータ→BIN16ビット/32ビットデータ変換
- **GBIN / GBINP**  
  指定したBIN16ビットグレイコードデータ（-32768～32767）をBIN16ビットデータに変換します。  
  処理イメージ：  
  (s) Gc（グレイコード） → (d) BIN  
  参照先：462ページ GBIN(P)(_U)

- **GBIN_U / GBINP_U**  
  指定したBIN16ビットグレイコードデータ（0～65535）をBIN16ビットデータに変換します。  
  処理イメージは上記と同様。

- **DGBIN / DGBINP**  
  指定したBIN32ビットグレイコードデータ（-2147483648～2147483647）をBIN32ビットデータに変換します。  
  処理イメージ：  
  (s)+1, (s) Gc → (d)+1, (d) BIN  
  参照先：464ページ DGBIN(P)(_U)

- **DGBIN_U / DGBINP_U**  
  指定したBIN32ビットグレイコードデータ（0～4294967295）をBIN32ビットデータに変換します。  
  処理イメージは上記と同様。

#### BIN16ビットデータブロック→BCD4桁データ変換ブロック
- **BKBSD / BKBCDP**  
  (s)から(n)点分のBINデータを一括でBCDデータに変換し、(d)以降に格納します。  
  参照先：466ページ BKBCD(P)
