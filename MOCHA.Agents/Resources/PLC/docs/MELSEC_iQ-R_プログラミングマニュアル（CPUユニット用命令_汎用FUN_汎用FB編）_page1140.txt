### テキスト
#### 機能
- (H)で指定したユニットから、(n)で指定した点数分のユニット情報を、(d)で指定したデバイス以降に格納する。
- I/O割付でユニット種別と点数を変更しても、実装のユニット状態が読み出される。
- (H)は、ユニット情報を読み出すユニットの先頭入出力番号を16進数4桁で表したときの上3桁で指定する。
- 読み出すユニットの先頭入出力番号を指定しなかった場合、ユニット装着状態のみON、ユニット装着状態以外がOFFのユニット情報を格納する。

### 図
- 図はCPUと複数のRX10ユニット、そしてRJ71EN71ユニットの配置を示している。
- 各ユニットの下には16進数のアドレス（0000H、0010H、0020H、0030H、0040H、0060H）が対応している。
- RJ71EN71ユニットのアドレスは0040Hで、読み出すユニットの先頭入出力番号はK4またはH4を指定することが注記されている。

(1): 読み出すユニットの先頭入出力番号はK4またはH4を指定する。
