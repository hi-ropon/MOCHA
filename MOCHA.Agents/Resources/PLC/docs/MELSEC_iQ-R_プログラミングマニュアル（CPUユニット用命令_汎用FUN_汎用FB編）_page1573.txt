### 表
#### 命令名ごとの基本ステップ数とサブセット処理の可否一覧
- 表は3列構成で、左から「命令名」「基本ステップ数」「サブセット可否」となっています。
- 「命令名」には三角関数や指数関数などの数学関数の命令が並んでいます（例：SIN、COS、TAN、ASIN、ACOS、ATAN、EXP、LOGなど）。
- 「基本ステップ数」は各命令の処理に必要な基本ステップ数を示しており、3または4の数値が記載されています。
- 「サブセット可否」はすべて「―」で記載されており、サブセット処理は不可であることを示しています。

#### 表の例（抜粋）
| 命令名 | 基本ステップ数 | サブセット可否 |
| :--- | :---: | :---: |
| SIN | 3 | ― |
| SINP | 4 | ― |
| COS | 3 | ― |
| COSP | 4 | ― |
| TAN | 3 | ― |
| TANP | 4 | ― |
| ASIN | 3 | ― |
| ASINP | 4 | ― |
| EXP | 3 | ― |
| EXPP | 4 | ― |
| LOG | 3 | ― |

- 命令名の末尾に「P」が付くものは基本ステップ数が4で、それ以外は3となっています。
