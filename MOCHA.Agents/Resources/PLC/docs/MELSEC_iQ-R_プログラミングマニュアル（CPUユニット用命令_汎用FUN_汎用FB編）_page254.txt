### テキスト
#### BIN32ビット除算命令 D/(P)(_U)
指定した2つのBIN32ビットデータを除算する命令です。対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）です。  
バージョン1.035M以降のエンジニアリングツールで対応しています。  
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、安全デバイスおよび安全ラベルのみ使用可能です。

#### 実行条件
- D/、D/_U命令は立ち上がりエッジで実行されます。
- D/P、D/P_U命令は立ち上がりエッジで実行されます。

### 図
#### ラダー図
- (s1)、(s2)の2つの入力と(d)の出力を持つ除算命令のラダー表現が示されています。

#### FBD/LD図
- EN（実行条件）、s1、s2（入力）、ENO（実行結果）、d（出力）を持つブロック図が示されています。
- ブロック内にはDDIVISION、DDIVISIONP、DDIVISION_U、DDIVISIONP_Uのいずれかが入ります。
