### テキスト
#### 活性ステップ一括読出し MOV(P) [K4S□/BL□￥K4S□]
- 対応CPUはR00CPU、R01CPU、R02CPUでバージョン制約なし。その他のCPUはファームウェアバージョンにより使用可能。
- 指定ブロックのステップ活性状態をビット情報として指定デバイスへBIN16ビットデータ単位で読み出す命令。
- エンジニアリングツールのバージョン制約も記載。

### 図
- ラダー図：EN入力、s入力、d出力のMOV命令の基本構造を示す。
- FBD/LD図：EN、ENO、s、dの入出力関係を示す。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| MOV  | 立ち下がりエッジ（下降エッジ） |
| MOVP | 立ち上がりエッジ（上昇エッジ） |
