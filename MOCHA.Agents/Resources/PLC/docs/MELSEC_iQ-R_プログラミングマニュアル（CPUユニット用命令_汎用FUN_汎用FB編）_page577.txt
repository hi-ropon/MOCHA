### テキスト
#### 機能
- FOR～NEXT命令による繰返し処理を強制的に終了し、(P)で指定されたポインタに実行を移すことができます。
- (P)には同一プログラムファイル内のポインタのみ指定可能で、他のプログラムファイルのポインタを指定すると演算エラーになります。
- (d)は強制終了した時点のFOR～NEXT命令における繰返し処理実行回数の残数を格納します。ただし、残数にはBREAK(P)命令実行時の回数も含まれます。
- BREAK(P)命令はFOR～NEXT命令間でのみ使用可能です。
- BREAK(P)命令は1つのネスティングに対してのみ使用可能で、多重ネスティングを強制終了させる場合はネスティング分のBREAK(P)命令を実行する必要があります。

### 図
- ラダーダイアグラム形式でFOR命令とBREAK命令、NEXT命令の関係を示しています。
- (1)は強制終了条件、(2)は強制終了条件成立時の動作を表しています。
- BREAK命令を実行していない場合は、FOR命令で指定した回数分繰返し処理を行い、FOR命令に戻る動作を示しています。
