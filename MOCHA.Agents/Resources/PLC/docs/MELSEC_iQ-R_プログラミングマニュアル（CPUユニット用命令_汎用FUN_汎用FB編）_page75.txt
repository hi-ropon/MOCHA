<overview>
本ページはCPUユニット用命令の基本命令の一部を説明しており、主に以下の3つのセクションで構成されています。
1. テキスト部分：命令の種類ごとの説明見出しと命令コードの一覧。
2. 表部分：命令コード、処理内容、参照先ページをまとめた表が複数あり、命令の詳細を体系的に示している。
3. 図部分：命令の処理内容を示すビット構成やデータ変換の流れを示す簡易図が表内に挿入されている。
</overview>

<contents>
### テキスト
#### ビット処理命令
ワードデバイスのビットセット/リセット、ビットテスト、ビットデバイスの一括リセットに関する命令を説明しています。

#### データ変換命令
BINデータをBCD4桁/8桁に変換する命令について説明しています。

### 表
#### ワードデバイスのビットセット/リセット
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| BSET / BSETP | 指定したワードデバイス内のnビット目をセット(1)します。 | 336ページ BSET(P) |
| BRST / BRSTP | 指定したワードデバイス内のnビット目をリセット(0)します。 | 338ページ BRST(P) |

#### ビットテスト
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| TEST / TESTP | 指定したワードデバイス内のnビット目を取り出します。 | 340ページ TEST(P) |
| DTEST / DTESTP | 指定した2ワードデバイス内のnビット目を取り出します。 | 342ページ DTEST(P) |

#### ビットデバイスの一括リセット
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| BKRST / BKRSTP | 指定したビットデバイスからn点分のビットデバイスをリセットします。 | 344ページ BKRST(P) |

#### BINデータ→BCD4桁/8桁変換
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| BCD / BCDP | 指定したBIN16ビットデータ(0～9999)をBCD4桁データに変換します。 | 410ページ BCD(P) |
| DBCD / DBCDP | 指定したBIN32ビットデータ(0～99999999)をBCD8桁データに変換します。 | 412ページ DBCD(P) |

### 図
- ビットセット/リセット命令の図は、ワードデバイスのビット位置b0～b15の中のnビット目を1または0に設定する様子を示しています。
- ビットテスト命令の図は、ワードデバイスの指定ビットを取り出して別のデバイスに格納する流れを示しています。
- ビットデバイスの一括リセットは、複数のビットデバイスをまとめてOFFにリセットする動作を示す簡易図です。
- BIN→BCD変換命令の図は、BINデータを入力としてBCDデータに変換する処理の流れを矢印で示しています。

</contents>
