### テキスト
#### 機能
(s1)で指定された倍精度実数と、(s2)で指定された倍精度実数の減算を行い、その結果を(d)で指定されたデバイスに格納します。

- (s1)、(s2)、(d)には0から2^1022未満の指定値（格納値）を指定可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。入力値設定時の注意事項は47ページを参照してください。

#### 注意事項
- 入力値の丸め誤差に注意が必要。

### 図
- 浮動小数点数の減算処理のビット配置を示す図
  - (s1)と(s2)の倍精度実数がそれぞれ4つのビットブロックに分かれている様子
  - 減算結果(d)も4つのビットブロックに分かれて格納されることを示している

### 表
#### エラーコード一覧

| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H | (s1)で指定したデータが0、非正規化数、非数、±∞のとき。<br>(s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が指定範囲を超えたとき（オーバーフロー発生時）。<br>|(d)| < 2^1024 |
