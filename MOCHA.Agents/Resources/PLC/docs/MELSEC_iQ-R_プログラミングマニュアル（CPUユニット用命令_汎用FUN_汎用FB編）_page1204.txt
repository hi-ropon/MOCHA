### 表
#### オペランド(s1)のFTPサーバ接続設定デバイス一覧

| デバイス | 項目       | 内容                                                                                   | 設定範囲                  | セット側 |
|----------|------------|----------------------------------------------------------------------------------------|---------------------------|----------|
| +170     | 接続方式   | FTPサーバの接続方式を設定する。                                                        | 0000H: PORTモード         | ユーザ   |
|          |            |                                                                                        | 0001H: PASVモード         |          |
| +171     | ポート番号 | FTPサーバの制御用ポート番号を設定する。                                               | 1～65535 (0001H～FFFFH)   | ユーザ   |

- 「接続方式」はFTPサーバへの接続方法を指定し、PORTモードまたはPASVモードのいずれかを選択可能。
- 「ポート番号」はFTPサーバの制御用ポート番号を1から65535の範囲で設定できる。
