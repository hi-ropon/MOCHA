### テキスト
#### 27.1 他号機CPUユニットアクセス命令 概要
他号機CPUユニットアクセス命令は、自号機CPUユニットから他号機CPUユニットへのデバイスデータの書込み/読出しを行う命令です。この命令により、例えば1号機から2号機への書込み動作を実現します。

#### パラメータ設定
D(P).DDRD命令、D(P).DDWR命令を使用する場合は、システムパラメータ内の定周期通信機能の設定が必要です。

### 図
- 図はCPU間のデバイスデータの書込み動作を示しています。
- 上部に命令の構成（PW、CPU、CPUなど）が示されており、下部に具体的な命令例（MP.DDWR H3E1 D0 D100 "D200" M0）が記載されています。
- 矢印でD0からD100のデータが読み取られ、D200に書き込まれる流れを表現しています。

### 表
| 命令名       | 内容                                                                 | 用途                                                                                   |
|--------------|----------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| D(P).DDRD    | 他号機CPUユニットのデバイスデータを自号機CPUユニットのデバイスへ読み出します。 | CPUユニット間でCPU間定周期通信機能を使用したタイミングでデータ授受を行う場合に使用します。 |
| D(P).DDWR    | 自号機CPUユニットのデバイスデータを他号機CPUユニットのデバイスへ書き込みます。 | 同上                                                                                   |
| M(P).DDRD    | 他号機CPUユニットのデバイスデータを自号機CPUユニットのデバイスへ読み出します。 | それぞれのCPUユニットのタイミングでデータ授受を行う場合に使用します。                   |
| M(P).DDWR    | 自号機CPUユニットのデバイスデータを他号機CPUユニットのデバイスへ書き込みます。 | 同上                                                                                   |
