### テキスト
#### BIN32ビット乗算 D*(P)(_U)
指定した2つのBIN32ビットデータを乗算する命令です。対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）です。  
バージョン1.035M以降のエンジニアリングツールで対応しています。

#### 実行条件
- 命令D*、D*_Uは立ち上がりエッジで実行されます。
- 命令D*P、D*P_Uは立ち上がりレベルで実行されます。

### 図
- **ラダー図**  
  (s1)、(s2)の2つの入力と(d)の出力を持つシンプルな乗算命令のラダー表現。  
- **FBD/LD図**  
  EN（実行条件）、s1、s2（乗算対象）、d（結果出力）、ENO（実行結果）端子を持つブロック図。  
  ブロック名はDMULTI、DMULTIP、DMULTI_U、DMULTIP_Uのいずれかが入る。

### 表

| :--- | :--- |

| D*P、D*P_U | 立ち上がりレベル |
