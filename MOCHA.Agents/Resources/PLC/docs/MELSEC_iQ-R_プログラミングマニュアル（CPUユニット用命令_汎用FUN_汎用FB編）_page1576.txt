### 表
#### 基本ステップ数とサブセット処理の可否一覧
命令名ごとに、以下の2つの項目が表形式でまとめられています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例：MAXP_U、DMAX、WSUMなど） |
| 基本ステップ数 | その命令を実行するのに必要な基本ステップ数（例：4、5、7など） |
| サブセット可否 | サブセット処理が可能かどうかを示す。可能な場合は「〇」、不可の場合は「―」で表記 |

表の内容は、命令ごとに基本ステップ数が4～9の範囲で設定されており、多くの命令はサブセット処理不可（―）ですが、一部の命令（例：WSUM、WSUMP、WSUM_U、WSUMP_U）はサブセット処理が可能（〇）となっています。
