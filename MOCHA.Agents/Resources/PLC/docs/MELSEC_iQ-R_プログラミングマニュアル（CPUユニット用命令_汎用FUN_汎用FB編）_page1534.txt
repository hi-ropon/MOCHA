### 表
#### 命令処理時間一覧表
命令名ごとに、条件別の処理時間（単位：マイクロ秒）がCPUユニット別に最小値と最大値で示されています。

- **列構成**
  - 命令名
  - 条件（例：実行時、(n)=1、(s)=1など）
  - 処理時間（μs）
    - R00CPU、R01CPU（最小・最大）
    - R02CPU（最小・最大）
    - 左記以外のCPUユニット（最小・最大）

- **内容の特徴**
  - 命令名はUDINT2INT、GRY、BKBCD、DABIN、VAL、ASC2INT、EMOD、NEG、DECO、ENCO、SEG、DIS、UNIなど多岐にわたる。
  - 条件欄には実行時やパラメータ(n)、(s)の値が記載されており、処理時間が条件によって異なることがわかる。
  - 処理時間はCPUユニットごとに異なり、R00CPU・R01CPU、R02CPU、その他CPUユニットで最小値と最大値が分けて示されている。
  - 処理時間の値は0.00294μsから最大で27.200μs程度まで幅がある。
