このページは、テキストとグラフ（タイミングチャート）で構成されています。テキストはX0とX1のON→OFFのタイミングが同じ場合の動作説明を行っており、グラフはX0、X1、M0のON/OFF状態の時間変化を示すタイミングチャートです。図はPLC命令の動作タイミングを視覚的に表現しています。
</overview>

<contents>
### テキスト
- 「X0とX1のON→OFFが同一タイミングのとき」の動作について説明しています。
- PLF命令で出力(Y)を指定すると、1スキャンの最後に実行したPLF命令のON/OFF状態が出力されることが記載されています。

### グラフ（タイミングチャート）
- 横軸は時間軸、縦軸は各信号（X0、X1、M0）のON/OFF状態を示しています。
- X0とX1の信号がONからOFFに変わるタイミングが同じ場合の動作を示しています。
- 各信号の状態変化に矢印と番号(1)～(4)が付されており、以下の動作説明が対応しています。

#### 動作説明
1. X0がON→OFFのため、M0はONになる。
2. X1がON→OFFのため、M0はONのまま維持される。
3. X0がON→OFF以外のため、M0はOFFになる。
4. X1がON→OFF以外のため、M0はOFFのまま維持される。

- これにより、M0のON/OFF状態がX0とX1の信号変化に応じてどのように変化するかが示されています。
</contents>
