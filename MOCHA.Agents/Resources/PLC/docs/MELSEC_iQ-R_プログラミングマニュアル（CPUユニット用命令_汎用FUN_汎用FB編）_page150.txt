### テキスト
#### 5.1 接点命令 演算開始、直列接続、並列接続
- LD: a接点演算開始命令 / LDI: b接点演算開始命令  
  指定デバイスのON/OFF情報を演算結果とする。
- AND: a接点直列接続命令 / ANI: b接点直列接続命令  
  指定デバイスのON/OFF情報と、それまでの演算結果のAND演算を演算結果とする。
- OR: a接点1個の並列接続命令 / ORI: b接点1個の並列接続命令  
  指定デバイスのON/OFF情報と、それまでの演算結果のOR演算を演算結果とする。

### 図（ラダー）
- LD、LDI、AND、ANI、OR、ORIの各命令に対応したラダー図が示されている。  
  それぞれの命令は接点の種類（a接点、b接点）や接続方法（直列、並列）を表現している。
