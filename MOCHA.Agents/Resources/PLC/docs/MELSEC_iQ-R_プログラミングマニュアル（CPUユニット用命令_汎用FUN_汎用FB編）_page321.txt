### テキスト
#### 機能
(d)で指定されたデバイスのBIN32ビットデータと、(s)で指定されたデバイスのBIN32ビットデータのビットごとの排他的論理和（XOR）を求め、その結果を(d)で指定されたデバイスに格納します。

- ビットデバイスの場合、桁指定による点数以降のビットデバイスは0として演算します。

#### エラー
演算エラーはありません。

### 図
ビットデータのXOR演算の例を示しています。

- (d)と(s)それぞれのデバイスのビット列（b31からb0まで）が並べられています。
- 各ビットの値（0または1）が示され、(d)のビット列と(s)のビット列の対応するビット同士でXOR演算が行われています。
- 演算結果は(d)のビット列に格納され、XORの結果としてビットが0または1に変化している様子が矢印で示されています。

この図は、32ビットのビットデータに対してビット単位でXOR演算を行い、その結果を指定デバイスに格納する動作を視覚的に説明しています。
