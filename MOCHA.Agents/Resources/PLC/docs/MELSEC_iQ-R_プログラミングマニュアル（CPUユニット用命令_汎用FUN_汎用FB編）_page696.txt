### テキスト
#### BIN16ビットデータゾーン制御
指定した入力値（BIN16ビット値）にバイアス値を付加する命令について説明しています。

#### 実行条件
命令「ZONE」「ZONE_U」「ZONEP」「ZONEP_U」の実行条件が示されています。  
- ZONE、ZONE_Uは立ち上がりエッジで実行  
- ZONEP、ZONEP_Uは立ち下がりエッジで実行
