### テキスト
#### 機能
- 起動接点がONしている時間を1時間単位で計測し、ON時間の累計が指定時間（BIN32ビットデータ）に達した時点で、指定したデバイスをONにします。
- 時間設定は1時間単位で行い、計測中の現在値は1時間単位や1秒単位で格納されますが、一部の値は格納されません。
- アラーム出力がONした後も計測は継続されます。
- 最大値に達し、かつ特定の条件を満たすと計測は停止します。
- 負の値を設定しても動作します。
- 時計データの変更（夏時間機能による時刻補正含む）には影響されません。

#### 注意事項
- 初期値から計測する場合や最大値に達した後も継続して計測する場合は、デバイスやラベルの指定に注意が必要です。
- タイマ計測が正常に行われないため、特定のタイプのプログラム（初期実行タイププログラム、割込みプログラム、定周期実行タイププログラム、イベント実行タイププログラム）では本命令を使用しないこと。
