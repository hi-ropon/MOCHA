### テキスト
#### BIN32ビットデータ帯域比較出力命令（DZCP(P)(_U)）
- R00CPU、R01CPU、R02CPUにはバージョンによる制約なし。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降で対応。
- 下限値(s1)のBIN32ビットデータと上限値(s2)のBIN32ビットデータの帯域に対し、比較データ(s3)のBIN32ビットデータで比較し、その結果（下、帯域内、上）に応じて(d)、(d)+1、(d)+2のいずれかがONになる。

### 図
- **ラダー図**  
  4つの入力（s1, s2, s3, d）を持つ命令ブロックの形状を示す。  
- **ST（構造化テキスト）**  
  ENO:=DZCP(EN,s1,s2,s3,d);  
  ENO:=DZCPP(EN,s1,s2,s3,d);  
  ENO:=DZCP_U(EN,s1,s2,s3,d);  
  ENO:=DZCPP_U(EN,s1,s2,s3,d);  
  の4種類の呼び出し例を示す。  
- **FBD/LD**  
  EN, ENO, s1, s2, s3, dの入出力ポートを持つ命令ブロックの図。
