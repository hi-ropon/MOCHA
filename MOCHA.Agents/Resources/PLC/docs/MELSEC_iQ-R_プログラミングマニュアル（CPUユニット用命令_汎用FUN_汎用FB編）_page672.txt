### テキスト
#### コピー元ファイル名/フォルダ名 (オペランド: s2)
ファイルまたはフォルダを指定する際のコピー元のパス指定方法について説明しています。

- ファイル名は拡張子まで指定可能。
- フォルダパス＋ファイル名（拡張子含む）は最大253文字以内。
- フォルダパスは最大244文字以内（区切り文字は含まない）。
- ファイル名またはフォルダパスは区切り文字を除いて1文字以上指定必須。
- 文字列の末尾に区切り文字や半角ピリオドを指定しないこと。
- フォルダパスの階層数は10以内。

##### ファイル指定時の例
"/folder1/user1/user1.csv"
- 最大253文字
- 区切り文字は「/」または「\」
- 省略可能な部分があり、省略時は最大252文字

##### フォルダ指定時の例
"/folder1/user1/subfolder1/"
- 最大244文字
- 区切り文字は「/」または「\」
- 省略可能な部分があり、省略時は最大243文字

#### コピー先フォルダパス (オペランド: s4)
コピー先のフォルダパス指定方法について説明しています。

- フォルダパスは最大244文字以内（区切り文字は含まない）。
- 区切り文字は「/」または「\」を使用。
- 文字列の末尾に区切り文字や半角ピリオドを指定しないこと。
- フォルダパスの階層数は10以内。

##### フォルダパス指定例
"/folder1/user1/database/"
- 最大244文字
- 区切り文字は「/」または「\」
- 省略可能な部分あり、省略時は最大243文字
