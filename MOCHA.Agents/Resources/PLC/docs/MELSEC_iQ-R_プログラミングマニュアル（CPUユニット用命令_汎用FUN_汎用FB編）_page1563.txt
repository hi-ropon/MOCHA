### 表
#### 基本ステップ数とサブセット処理の可否一覧
この表は各命令名に対して、以下の2つの情報を示しています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例：BKORP、WXOR、DXORなど） |
| 基本ステップ数 | その命令を実行するのに必要な基本的なステップ数（数値） |
| サブセット可否 | その命令がサブセット処理に対応しているかどうかを示す。対応している場合は「〇」、対応していない場合は「―」で表記 |

表の内容は以下の特徴があります。
- 命令名はアルファベットの大文字で表記されており、括弧内にパラメータの種類が示されているものもあります。
- 基本ステップ数は3～6の範囲で設定されている命令が多いです。
- サブセット可否は命令によって対応・非対応が分かれており、例えば「BKORP」は非対応、「WXOR (s) (d)」は対応となっています。

この表は命令の処理負荷や機能対応状況を把握するための一覧表として利用されることが想定されます。
