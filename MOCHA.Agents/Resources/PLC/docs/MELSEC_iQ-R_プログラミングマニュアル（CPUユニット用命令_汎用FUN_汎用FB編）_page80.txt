### テキスト・表
#### BIN16ビット/32ビットデータの補数(符号反転)
- NEG, NEGP: BIN16ビットデバイスの符号を反転する命令。
- DNEG, DNEGP: BIN32ビットデバイスの符号を反転する命令。
- 処理内容は符号反転のイメージ図付きで説明。

#### 8→256ビットデコード
- DECO, DECOP: 指定されたデバイスの下位(n)ビットをデコードする命令。
- 処理内容は(n)ビットのデコードを示す簡単な図付き。

#### 256→8ビットエンコード
- ENCO, ENCOP: 2の(n)乗ビットのデータをエンコードする命令。
- 処理内容は(n)ビットのエンコードを示す簡単な図付き。

#### 7セグメントデコード
- SEG, SEGP: デバイスの下位4ビットで指定された0～Fのデータを7セグメント表示データにデコードする命令。
- 処理内容は7セグメント表示のイメージ図付き。

#### BIN16ビットデータの4ビット分離
- DIS, DISP: 指定した16ビットデータを4ビット単位で分離し、指定デバイスの下位4ビットに格納する命令。

#### BIN16ビットデータの4ビット結合
- UNI, UNIP: 指定デバイスから4ビットデータを結合し、指定デバイスに格納する命令。

#### 任意データのビット分離、結合
- NDIS, NDISP: 指定デバイス以降のデータを指定ビットに分離し、指定デバイスに格納する命令。
- NUNI, NUNIP: 指定デバイス以降のデータを指定ビットごとに結合し、指定デバイスに順に格納する命令。

#### バイト単位データ分離、結合
- WTOB, WTOBP: 指定デバイスから16ビットデータを8ビット単位に分解し、指定デバイスに順に格納する命令。
- BTOW, BTOWP: 指定デバイスから16ビットデータの下位8ビットを16ビットに結合し、指定デバイスに順に格納する命令。

### 参照先
各命令には詳細説明が記載されたページ番号が付記されており、命令ごとに参照可能。
