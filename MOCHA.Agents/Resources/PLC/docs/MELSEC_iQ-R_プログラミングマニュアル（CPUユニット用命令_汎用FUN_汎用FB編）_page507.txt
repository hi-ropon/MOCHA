### テキスト
#### 機能
- (s)で指定されたBIN16ビットデータの下位(n)桁（1桁4ビット）分のデータを、(d)で指定されたデバイスから(n)点分の下位4ビットに格納します。
- (d)で指定されたデバイスから(n)点分の上位12ビットは0になります。
- (n)は1～4が指定可能です。
- (n)=0のときは無処理となり、(d)のデバイスから(n)点の内容は変化しません。

### 図
- 左側にBIN16ビットデータのビット配置（b15～b0）が示されており、(s)で指定されたデータの下位(n)桁分が右側の格納エリアに格納される様子を表しています。
- 右側の格納エリアは(d)で指定されたデバイスの(n)点分の下位4ビットに対応し、上位12ビットは0で埋められます。
- (1)が格納エリア、(2)が0が格納される部分を示しています。

### 表
#### エラー
| エラーコード (SD0) | 内容               |
|--------------------|--------------------|
| 3401H              | (n)が0～4以外のとき。|
