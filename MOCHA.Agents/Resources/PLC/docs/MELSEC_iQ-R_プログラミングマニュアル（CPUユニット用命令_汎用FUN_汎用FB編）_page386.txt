### テキスト
#### nダブルワードデータのnダブルワード右シフト（SFTDWR(P)）
- R00CPU、R01CPU、R02CPUはファームウェアバージョン"06"以降で使用可能。エンジニアリングツールはバージョン"1.047Z"以降で使用可能。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降で使用可能。
- 指定したデバイスから(n1)ダブルワード分のデータ範囲内で、右へ(n2)ダブルワード分シフトする。シフト後の空きデータには0が格納される。

### 図
#### ラダー図
- (d)（シフトする先頭デバイス）、(n1)（シフトするデータの数）、(n2)（シフト数）を入力し、ST命令で右シフトを実行する。

#### FBD/LD図
- EN（入力）、n1、n2を入力し、ENO（出力）、d（出力デバイス）を得るブロック図。

### 表
#### 実行条件
| 命令     | 実行条件               |
| -------- | ---------------------- |
| SFTDWR   | パルス信号             |
| SFTDWRP  | 立ち上がりパルス信号   |

| ---------- | ------------------ | ---------- | ------------------ | ---------------- |

| (n1)       | シフトするデータの数 | 0～65535   | 符号なしBIN16ビット | ANY16            |
| (n2)       | シフト数           | 0～65535   | 符号なしBIN16ビット | ANY16            |
