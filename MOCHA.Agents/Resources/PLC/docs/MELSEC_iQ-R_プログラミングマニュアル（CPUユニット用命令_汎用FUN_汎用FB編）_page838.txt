### テキスト
#### 機能
(s2)で指定されたデバイス番号以降に格納されている文字列データの左の(s3)文字目から、(s1)で指定されたデバイス番号以降に格納されている文字列データを検索します。検索結果は(d)で指定されたデバイスに格納されます。検索結果は、(s2)で指定された文字列データの先頭文字から何文字目かを格納します。

- 一致した文字列データが無い場合は、(d)に0を格納します。

### 図
- 左図は(s2)の文字列データ"ABCDEFGHIJK"の各文字が16進数で示されており、(s3)が3文字目から検索開始されることを示しています。
- 右図は(s1)の文字列データ"EFGH"が16進数で示されており、検索対象文字列です。
- 検索結果(d)は5で、"EFGH"が(s2)の文字列の5文字目から始まることを表しています。
