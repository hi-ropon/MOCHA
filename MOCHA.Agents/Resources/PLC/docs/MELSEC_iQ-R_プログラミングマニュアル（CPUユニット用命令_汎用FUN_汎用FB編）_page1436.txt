### テキスト
#### 34.4 除算 DIV(_E)
DIV(_E)は、入力値の商((s1)÷(s2))を出力する関数です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化、一般）、RnPSFCPU（安全、一般）など多岐にわたります。

##### 入力値と出力値
- s1：被除数（入力変数）
- s2：除数（入力変数）
- EN：実行条件（TRUEで実行、FALSEで停止）
- ENO：出力状態（TRUE正常、FALSE異常）
- d：出力（商）

##### 機能
- (s1)、(s2)にINT型、DINT型、WORD型、DWORD型、REAL型、LREAL型データの除算を行い、演算結果の商を(d)から(s)と同一のデータ型で出力します。
- (s2)への入力値は0以外である必要があります。

##### 演算処理例
INT型の例では、5 ÷ 2 の場合、商は2、余りは1となり、余りは出力されません。

##### 演算結果
1. EN/ENOなしファンクション
- 演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション
- 実行条件ENがTRUEの場合、ENOはTRUEとなり、(d)に演算出力値が出力されます。
- ENがFALSEの場合、ENOはFALSEとなり、(d)の値は不定値となります。
- ENOがFALSEの場合は(d)のデータは不定値となるため、その場合は(d)のデータを使用しないようプログラム処理を行う必要があります。

### 表
#### 設定データ（引数の内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s1(IN1) | 被除数 | 入力変数 | ANY_NUM*1 |
| s2(IN2) | 除数 | 入力変数 | ANY_NUM*1 |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | ANY_NUM*1 |

*1 安全プログラムではREAL型/LREAL型データを指定しないこと。指定するとエラーとなる。

#### 実行条件と演算結果の対応表
| 実行条件 | 演算結果 |
| :--- | :--- |
| EN | ENO | (d) |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOがFALSEの場合、(d)のデータは不定値となる。

### 図
- ラダー回路例（EN/ENOなし、EN/ENO付き）: DIV関数の入力s1、s2と出力dの接続例を示す。
- ST言語例: EN/ENOなしは d:=DIV(s1,s2); EN/ENO付きは d:=DIV_E(EN,ENO,s1,s2); の記述例。
- 除算例（INT型）: 5 ÷ 2 の場合、商は2、余りは1であり、余りは出力されないことを示す図。
