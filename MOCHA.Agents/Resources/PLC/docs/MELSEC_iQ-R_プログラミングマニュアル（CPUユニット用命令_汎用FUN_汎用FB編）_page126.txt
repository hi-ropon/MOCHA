### テキスト
#### プロセス制御命令
プロセス制御命令の詳細については、「MELSEC iQ-R プログラミングマニュアル（プロセス制御FB/命令編）」を参照するよう案内されています。

##### Point
プロセス制御用のプログラムを作成する際には、プロセス制御ファンクションブロックの使用が推奨されます。  
プロセス制御ファンクションブロックの特徴は以下の通りです。  
- FB部品の配置と結線でプログラムが簡単に作成でき、豊富な種類のファンクションブロックが用意されているため、プログラミングが容易になる。  
- ファンクションブロックの初期値はエンジニアリングツールの「FBプロパティ」画面から設定可能で、初期値設定用のプログラムが不要。  
- ファンクションブロックの引数はデバイスのアドレスを意識せず、ラベルで指定できる。  
- エンジニアリングツールのフェースプレートなどからタグデータにアクセスし、タグFBの実行状態の確認や調整が可能。

プロセス制御ファンクションブロックの詳細についても同じく「MELSEC iQ-R プログラミングマニュアル（プロセス制御FB/命令編）」を参照するよう案内されています。
