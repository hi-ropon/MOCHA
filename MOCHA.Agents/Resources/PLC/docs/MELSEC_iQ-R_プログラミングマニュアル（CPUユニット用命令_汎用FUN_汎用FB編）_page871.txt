### テキスト
#### ED-(P)命令（オペランドが3個の場合）
この命令は倍精度実数を減算します。

#### 実行条件
- ED-命令は立下りエッジで実行されます。
- ED-P命令は立上りエッジで実行されます。

### 図
- **ラダー図**  
  3つのオペランド (s1, s2, d) を用いた減算回路が示されています。  
- **FBD/LD図**  
  EN（実行条件）、s1、s2（入力）、d（出力）、ENO（実行結果）を持つブロック図で、EDMINUSまたはEDMINUSPが入ることが示されています。  
- **実行条件の波形図**  
  ED-は下降エッジ、ED-Pは上昇エッジで命令が実行されることを示す波形。
