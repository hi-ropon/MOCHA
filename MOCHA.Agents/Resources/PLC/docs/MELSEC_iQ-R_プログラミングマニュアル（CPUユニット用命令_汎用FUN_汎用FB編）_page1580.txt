### 表
#### 命令名ごとの基本ステップ数とサブセット可否一覧
- 表は3列構成で、左から「命令名」「基本ステップ数」「サブセット可否」となっています。
- 「命令名」にはPID制御関連の命令やデータ転送命令、論理演算命令など多様な命令が列挙されています。
- 「基本ステップ数」は各命令の実行に必要な基本的なステップ数を示しており、2～10の範囲で記載されています。
- 「サブセット可否」はすべて「―」で記載されており、サブセット処理は不可であることを示しています。
