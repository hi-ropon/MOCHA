<overview>
このページは主に表形式で構成されており、CPUユニット用命令のうち「単精度実数」と「倍精度実数」に関する比較出力、加減算、乗除算の命令一覧を示しています。テキスト部分は命令の説明や処理内容の詳細を記述しており、表部分は命令記号、処理内容、参照先ページを整理しています。グラフや図は含まれていません。
</overview>
