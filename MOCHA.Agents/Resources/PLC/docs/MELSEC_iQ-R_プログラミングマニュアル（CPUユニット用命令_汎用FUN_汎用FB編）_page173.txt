### テキスト
#### カウンタ（OUT C）
OUT C命令は、演算結果がOFFからONに変化したときにカウンタの現在値（カウント値）を+1します。カウントアップすると、カウンタのa接点は導通し、b接点は非導通になります。

**Point**
SIL2プロセスCPUおよび安全CPUの安全プログラムで安全カウンタを使用する場合は、特に明記する場合を除き、「OUT C」を「OUT SA C」と読み替えて使用してください。

### 図
- **ラダー図**  
  (d)カウンタ番号とValue（設定値）を示すシンボルが描かれている。

- **ST言語**  
  `ENO:=OUT_C(EN,Coil,Value);`  
  EN（実行条件）、Coil（カウンタ）、Value（設定値）を引数にOUT_C命令を実行し、実行結果をENOに格納する。

- **FBD/LD**  
  EN（入力）、Coil（カウンタ）、Value（設定値）、ENO（出力）を持つ命令ブロックの図。

### 表
#### 実行条件
| 命令  | 実行条件 |
| :---- | :------- |
| OUT C | 常時実行 |

#### 設定データ（内容、範囲、データ型）
| オペランド | 内容           | 範囲       | データ型           | データ型（ラベル） |
| :--------- | :------------- | :--------- | :----------------- | :----------------- |
| (d)        | カウンタ番号   | ―          | ビット             | ANY_BOOL*1         |
| Coil       |               |            |                    |                    |
| Value      | カウンタの設定値 | 0～65535   | 符号なしBIN16ビット*2 | ANY16*2            |
| EN         | 実行条件       | ―          | ビット             | BOOL               |
| ENO        | 実行結果       | ―          | ビット             | BOOL               |

*1 カウンタ型ラベルのみ使用可能。  
*2 ST、FBD/LDの場合、データ型はANY_INTになる。  

**補足**  
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、上記のデータ型の安全デバイスおよび安全ラベルのみ使用可能。
