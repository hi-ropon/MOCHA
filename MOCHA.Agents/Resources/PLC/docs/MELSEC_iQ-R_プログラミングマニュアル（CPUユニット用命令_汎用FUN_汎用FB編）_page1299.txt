### テキスト
#### 29.2 系切替え禁止/許可
- **DCONTSW**: 系切替え（ユーザ切替え）を禁止する命令。
- **ECONTSW**: 系切替え（ユーザ切替え）を許可する命令。

#### ラダー図
- EN入力に対して、ENO出力がDCONTSW(EN)またはECONTSW(EN)の状態を示す。

#### FBD/LD図
- EN入力に対してENO出力があるシンプルなブロック図。

#### 実行条件
| 命令     | 実行条件 |
| -------- | -------- |
| DCONTSW  | 立ち上がりエッジ |
| ECONTSW  | 立ち上がりエッジ |

#### 機能
- 待機系でDCONTSW命令を実行すると、制御系は系切替え禁止状態になる。
- 待機系でECONTSW命令を実行すると、制御系は系切替え許可状態になる。
- 初期状態は系切替え許可状態。
- DCONTSW命令およびECONTSW命令は待機系でのみ実行可能。制御系で実行しても無処理。

#### Point
- バックアップモード時に待機系でこれらの命令を実行するには、両系実行プログラム、または両系実行プログラムから呼び出されるプログラム部品に命令を記述する必要がある。
- 詳細は「MELSEC iQ-R CPUユニットユーザーズマニュアル（応用編）」を参照。
