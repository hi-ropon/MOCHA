### テキスト
#### 機能
- (s)で指定された値の自然対数eを底とした対数を計算し、(d)で指定されたデバイスに演算結果を格納する。
- (s)で指定する値は正の数のみ設定可能（負の数は演算不可）。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

### 表
#### エラー
| エラーコード (SD0) | 内容 |
| :--- | :--- |
| 3402H | (s)に指定した値が0、非正規化数、非数、±∞のとき。 |
| 3405H | (s)に指定可能な範囲外のデータを設定したとき。<br>・指定した値が負の数<br>・指定した値が0 |
