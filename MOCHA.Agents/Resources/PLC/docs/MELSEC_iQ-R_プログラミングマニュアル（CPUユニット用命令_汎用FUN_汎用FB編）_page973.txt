### テキスト
#### 単精度実数最大値検索 EMAX(P)
単精度実数のブロックデータの最大値を検索する命令である。

##### 注意事項
- EMAX命令はST、FBD/LDでは対応していないため、汎用ファンクションのMAXを使用すること。
- 参照：1457ページ MAX(_E)、MIN(_E)

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| EMAX | 立ち上がりエッジ |
| EMAXP | 立ち下がりエッジ |

### 図
- ラダー図：3つの入力(s, d, n)を持つEMAXP命令のシンボルが示されている。
- FBD/LD図：EN入力、s, n入力、d出力、ENO出力を持つ命令ブロックの形状が示されている。

| (d) | 検索結果を格納する先頭デバイス (d)～(d)+1:最大値 (d)+2:位置 (d)+3:個数 | ― | 単精度実数 | ―*2 (ANY_REAL_32_ARRAY) |
