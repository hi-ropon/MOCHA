- 命令記号: DCMP, DCMPP, DCMP_U, DCMPP_U
- 処理内容: (s1)で指定したBIN32ビットデータと(s2)で指定したBIN32ビットデータを比較し、その結果（小、 一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONになる。
