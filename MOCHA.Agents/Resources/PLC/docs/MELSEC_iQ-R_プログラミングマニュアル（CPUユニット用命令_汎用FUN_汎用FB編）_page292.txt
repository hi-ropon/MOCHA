### テキスト
#### 32ビットBINデータインクリメント
「DINC(P)(_U)」命令は、指定したBIN32ビットデータを+1します。

#### 実行条件
- DINC、DINC_U命令は立ち上がりエッジで実行されます。
- DINCP、DINCP_U命令は立ち下がりエッジで実行されます。

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- ラダー図：入力(d)に対して+1インクリメントを行う回路シンボルが示されています。
- ST（構造化テキスト）コード例：
  - ENO:=DINC(EN,d);
  - ENO:=DINCP(EN,d);
  - ENO:=DINC_U(EN,d);
  - ENO:=DINCP_U(EN,d);
- FBD/LD図：EN（入力）、ENO（出力）、d（データ）を持つブロック図が示されています。
- 実行条件波形図：命令ごとの実行タイミングを示す波形（立ち上がりまたは立ち下がりエッジ）
