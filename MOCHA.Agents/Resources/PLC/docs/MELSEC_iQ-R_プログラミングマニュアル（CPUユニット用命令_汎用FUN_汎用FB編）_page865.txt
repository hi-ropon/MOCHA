### テキスト
#### 倍精度実数加算 ED+(P) [オペランドが2個の場合]
倍精度実数を加算する命令です。

- ラダー回路（ST）は対応していません（3個の場合は別ページ参照）。
- FBD/LDも対応していません。

##### ■実行条件
| 命令 | 実行条件 |
|------|----------|
| ED+   | 立ち上がりエッジ |
| ED+P  | 立ち下がりエッジ |

##### ■機能
(d)で指定された倍精度実数と、(s)で指定された倍精度実数の加算を行い、加算結果を(d)で指定されたデバイスに格納します。

- (s)、(d)には0から2^1022以上1024未満の値が指定・格納可能です。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあります。詳細は47ページの注意事項を参照してください。

### 表
#### 内容、範囲、データ型
| オペランド | 内容                                   | 範囲                          | データ型     | データ型(ラベル) |
|------------|--------------------------------------|-------------------------------|--------------|------------------|
| (s)        | 加算データまたは加算データが格納されている先頭デバイス | 0, 2^(-1022) ≤ |s| < 2^1024 | 倍精度実数   | ANYREAL_64       |
| (d)        | 加算されるデータが格納されている先頭デバイス           | 0, 2^(-1022) ≤ |d| < 2^1024 | 倍精度実数   | ANYREAL_64       |

#### 使用可能デバイス
| オペランド | ビット | ワード | ダブルワード | 間接指定 | 定数 | その他 |
|------------|--------|--------|--------------|----------|------|--------|
| (s)        | -      | ○ (T, ST, C, D, W, SD, SW, FD, R, ZR, RD) | - | - | ○ (K, H, E, $) | - |
| (d)        | -      | ○ (T, ST, C, D, W, SD, SW, FD, R, ZR, RD) | - | - | - | - |

### 図・イメージ説明
- ラダー回路の簡単な図示：2つのオペランド(s)と(d)を示す。
- 実行条件の波形：ED+は立ち上がりエッジ、ED+Pは立ち下がりエッジで実行されることを示す。
- 倍精度実数加算のデータ配置イメージ：
  - (d)+3, (d)+2, (d)+1, (d) に倍精度実数が格納されている。
  - (s)+3, (s)+2, (s)+1, (s) に倍精度実数が格納されている。
  - これらを加算し、結果を(d)+3～(d)に格納する流れを矢印で示している。
