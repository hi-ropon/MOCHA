### テキスト
#### シーケンスプログラム終了（END命令）
プログラムの最終を示す命令です。対応CPUはRnCPU、RnENCPU（プロセス）、RnPCPU（二重化）、RnSPFCPU（一般・安全）です。

##### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| END | 常時実行 |

##### 機能
- メインルーチンプログラム、サブルーチンプログラム、割込みプログラムを含むプログラムの最終を示します。
- END命令を実行すると、CPUユニットは実行していたプログラムを終了します。
- プログラム途中でEND処理が必要な場合はFEND命令を使用します。
- エンジニアリングツールの回路編集モードでプログラミングする場合、END命令は自動で入力され編集不可です。
- メインルーチンプログラム、サブルーチンプログラム、割込みプログラムの終了命令の使い分けは以下の通りです。
