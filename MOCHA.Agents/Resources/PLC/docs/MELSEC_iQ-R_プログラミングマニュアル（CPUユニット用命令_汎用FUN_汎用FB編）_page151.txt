### テキスト
#### 設定データ
- 接点として使用するデバイスの種類やデータ型について説明。  
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、特定の安全デバイスおよび安全ラベルのみ使用可能。

#### 機能
- **LD, LDI命令**  
  接点演算開始命令。指定デバイスのON/OFF情報を取り込み、演算結果とする。  
- **AND, ANI命令**  
  接点直列接続命令。指定ビットデバイスのON/OFF情報を取り込み、AND演算結果とする。  
  エンジニアリングツールの回路編集モードでは、24段までの回路作成・表示が可能。  
- **OR, ORI命令**  
  接点並列接続命令。指定ビットデバイスのON/OFF情報を取り込み、OR演算結果とする。  
  回路編集モードでは23個までの連続接続回路作成・表示が可能。  
- **LD, LDI, AND, ANI, OR, ORIの組み合わせによる動作**  
  これら命令の組み合わせによる動作例を示す。LDI、ANI、ORIも同様の動作。

#### エラー
- 演算エラーはなし。
