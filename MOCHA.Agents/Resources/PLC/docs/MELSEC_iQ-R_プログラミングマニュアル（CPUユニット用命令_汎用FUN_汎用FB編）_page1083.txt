### テキスト
#### 機能
(s)で指定したデバイス番号以降に格納されている日時データを秒に換算し、その換算結果を(d)で指定したデバイスに格納します。起点は2000年1月1日0時0分0秒（0秒）です。

- (s)から順にyear、month、day、hour、minute、secondの各データが格納されていることを示す表があり、年のデータ範囲はDATE2SEC(P)で2000～2067年、DATE2SEC(P)_Uでは2000～2099年となっています。
- 例として、2016年11月27日20時51分48秒を指定した場合の秒換算結果（533595108秒）が示されています。
- 本命令の動作はタイムゾーンや夏時間による時刻補正の影響を受けません。夏時間期間中とそれ以外の換算結果は同じになります。

### 図
- (s)から順に年、月、日、時、分、秒のデータを受け取り、秒数に換算して(d)で指定したデバイスに格納する流れを示す簡単なブロック図。
