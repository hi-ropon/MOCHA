### 処理内容の補足
- (s)はソースデータ、(d)はデスティネーション（格納先）を示す。
- 符号付き/符号なしのBIN16/32ビットデータを単精度または倍精度実数に変換する命令が分類されている。
- 文字列やBCDフォーマットから単精度実数への変換命令も含まれている。
- 単精度実数の符号を反転する命令も記載されている。
