### 表
#### GX Works2命令名と互換用命令名一覧
- 表は5つの列で構成されている。
  1. GX Works2命令名
  2. 互換用命令名
  3. 使用プログラム（ST）
  4. 使用プログラム（FBD）
  5. 参照先

- 「使用プログラム」列はST（ステートメントリスト）とFBD（ファンクションブロックダイアグラム）での使用可否を○（使用可）または×（使用不可）で示している。

- 参照先は命令の詳細説明が記載されているマニュアルのページ番号や名称を示している。

- 具体例として、
  - 「STR」命令は互換用命令名「STR_M2」で、ST・FBD両方で使用可能。参照先は799ページのSTR(P)(_U)。
  - 「SWAP」命令は互換用命令名「SWAP_M2」で、STでは使用不可、FBDでは使用可能。参照先は548ページのSWAP(P)。
  - 「TEACH1_P_M」命令は互換用命令名「ZP_TEACH1_M2」で、ST・FBD両方で使用可能。参照先は「MELSEC iQ-R プログラミングマニュアル（ユニット専用命令編）」。

- その他、多数の命令が一覧化されており、命令名の末尾に「_MD」や「_P_MD」などのバリエーションが存在する。

- ページ下部に「付録 付6 他形式プロジェクトからの置換え」と記載されている。

以上より、本ページはGX Works2の命令とその互換命令の対応表として、使用可能なプログラム形式と参照先をまとめた技術資料の一部である。
</contents>
