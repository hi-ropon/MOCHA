#### 注意事項
- (d)+1、(d)+2はシステムで使用するため、値を変更しないこと。
- (d)で指定した領域に格納されるZ、LZ退避種別もシステムで使用するため、値を変更しないこと。
- これらの値の変更は誤動作の原因となる。
