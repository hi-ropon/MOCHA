### テキスト
#### ファイル名/フォルダ名の指定について
ファイルを指定する場合は、削除対象のファイルが格納されているフォルダパス＋ファイル名を指定します。  
フォルダを指定する場合は、削除対象のフォルダのフォルダパスを指定します。

- ファイル名は拡張子まで指定する必要があります。
- フォルダパス＋ファイル名（拡張子含む）は最大253文字以内にしてください。
- フォルダパスは244文字以内にしてください（区切り文字は含まない）。
- ファイル名またはフォルダパスは区切り文字を除いて1文字以上指定する必要があります。
- ファイル指定時は文字列の末尾に区切り文字を指定しないでください。
- 文字列の末尾や区切り文字の直前に半角ピリオドを指定しないでください。
- フォルダパスの階層数は10以内にしてください。

#### ファイルを指定する場合の例
```
"/folder1/user1/user1.csv"
```
- 最大253文字まで指定可能。
- フォルダパスやファイルの区切り文字は「/」または「\」を使用。
- 省略可能な部分があり、省略した場合は最大252文字となる。

#### フォルダを指定する場合の例
```
"/folder1/user1/subfolder1/"
```
- 最大244文字まで指定可能。
- フォルダパスの区切り文字は「/」または「\」を使用。
- 省略可能な部分があり、省略した場合は最大243文字となる。
