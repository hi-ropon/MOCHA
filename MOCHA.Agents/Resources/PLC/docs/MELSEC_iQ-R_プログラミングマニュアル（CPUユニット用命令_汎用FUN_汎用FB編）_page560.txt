### テキスト
#### ENDへジャンプ（GOEND命令）
同一プログラムファイル内のFENDまたはEND命令へジャンプする命令です。

##### 対応CPU
RnCPU、RnENCPU、RnPCPU（プロセス）、RnPCPU（二重化）、RnSPFCPU（一般）、RnSPFCPU（安全）、RnSFCCPU（一般）、RnSFCCPU（安全）

##### ラダー
ラダー回路での使用が可能ですが、ST（ストラクチャードテキスト）では対応していません。

##### FBD/LD
対応していません。

### 表
#### 実行条件

| 命令   | 実行条件       |
|--------|----------------|
| GOEND  | 立ち上がりエッジ |

#### エラーコード一覧

| エラーコード(SD0) | 内容                                                                                   |
|-------------------|----------------------------------------------------------------------------------------|
| 3340H             | FOR命令を実行後、NEXT命令を実行する前にGOEND命令を実行したとき。                     |
| 3381H             | CALL(P)命令、ECALL(P)命令を実行後、RET命令を実行する前にGOEND命令を実行したとき。    |
| 33A1H             | 割込みポインタ(I)による割込みプログラム中で、IRET命令を実行する前にGOEND命令を実行したとき。 |

### 機能
- 同一プログラムファイル内のFENDまたはEND命令へジャンプします。

### 図
- ラダー回路の簡単なジャンプ命令のイメージが示されています。  
- FBD/LDは対応していません。
