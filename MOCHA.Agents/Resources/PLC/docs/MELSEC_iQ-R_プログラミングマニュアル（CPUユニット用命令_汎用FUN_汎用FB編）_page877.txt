### テキスト
#### 倍精度実数乗算 ED*(P)
倍精度実数を乗算する命令です。対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）など複数のCPUに対応しています。

#### 実行条件
- ED*命令は立ち上がりエッジで実行されます。
- ED*P命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー回路の簡単な構成が示されており、(s1)、(s2)、(d)の3つのオペランドが入力されています。  
- **FBD/LD図**  
  ブロック図でEN（実行条件）、s1、s2（乗算対象データ）、d（演算結果）が示されています。  
  （注：図中の□にはEDMULTI、EDMULTIPが入る）
