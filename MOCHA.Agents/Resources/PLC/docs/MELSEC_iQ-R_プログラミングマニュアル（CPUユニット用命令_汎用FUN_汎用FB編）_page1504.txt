### テキスト
#### 43.1 アップカウンタ (CTU(_E))
信号の立ち上がり回数をカウントアップする機能を持つ。

##### 設定データ
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s1(CU) | カウント信号入力 | 入力変数 | BOOL |
| s2(R) | カウント値リセット | 入力変数 | BOOL |
| n(PV) | カウント最大値 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常、FALSE:異常または停止) | 出力変数 | BOOL |
| d1(Q) | カウント完了 | 出力変数 | BOOL |
| d2(CV) | カウント値 | 出力変数 | INT |

##### 機能
1. カウントアップ
- s1がOFFからONに変わると、d2の値を1加算する。
- d2がnに達するとd1がONし、カウントアップは停止する。
- nはカウンタの最大値を設定。s2がONになるとd1はOFF、d2は0にリセットされる。

2. カウント最大値
- nの有効設定範囲は0～32767。
