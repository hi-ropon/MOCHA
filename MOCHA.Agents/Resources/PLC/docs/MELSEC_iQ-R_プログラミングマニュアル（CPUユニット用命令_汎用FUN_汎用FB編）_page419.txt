### テキスト
#### 機能
- (s)で指定された単精度実数を符号付きBIN16ビットデータに変換し、(d)で指定されたデバイスに格納する。  
- 変換後のデータは、(s)で指定された単精度実数の小数点以下1桁目を四捨五入した値となる。  
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。  
- 注意事項は47ページに記載されている。  

#### プログラム例の説明
- M0がONしたとき、D0～D1に格納された単精度実数を符号付きBIN16ビットデータに変換し、D100へ格納する。  

### 図（プログラム例）
- ラダー図でSM402とM0の条件付きで2つの命令が実行される。  
- EMOVP命令でE-1234.5（単精度実数）がD0にセットされる。  
- FLT2INT命令でD0の値を符号付きBIN16ビットデータに変換し、D100に格納する。  
- 変換のビット構成例が示されており、D1,D0の単精度実数（-1234.5）がBIN16ビットデータ（-1235）に変換されている。  

### 表（エラーコード）
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3401H              | (s)で設定する単精度実数が-32768～32767の範囲外のとき。 |
| 3402H              | (s)に特殊な数を設定したとき。<br>・設定した単精度実数が0、2^-126 ≤ |(s)| < 2^128の範囲内でない。<br>・設定したデバイス、ラベルの内容が0、非正規化数、非数、±∞のとき。 |
