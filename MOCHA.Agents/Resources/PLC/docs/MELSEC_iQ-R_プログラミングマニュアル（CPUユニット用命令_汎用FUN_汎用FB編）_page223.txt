### テキスト
#### BIN16ビットデータ帯域比較出力
本命令は、下限値(s1)のBIN16ビットデータと上限値(s2)のBIN16ビットデータの帯域に対し、比較データ(s3)のBIN16ビットデータで比較し、その結果に応じて(d)、(d)+1、(d)+2のいずれかのビットがONになる動作を行います。

対応CPUはR00CPU、R01CPU、R02CPUに制約はなく、RnCPU(R00CPU、R01CPU、R02CPUを除く)はファームウェアバージョン17以降で使用可能です。エンジニアリングツールはバージョン1.020W以降で使用できます。

#### 実行条件
- ZCP、ZCP_U命令は立ち上がりエッジで実行
- ZCPP、ZCPP_U命令は立ち下がりエッジで実行

### 図
#### ラダー図
- (s1)、(s2)、(s3)、(d)の4つのオペランドを持つ命令ブロックが示されており、比較結果に応じて(d)、(d)+1、(d)+2のいずれかがONになることを表現。

#### FBD/LD図
- EN（実行条件）、s1（下限値）、s2（上限値）、s3（比較データ）、d（比較結果出力）を持つ命令ブロックの構造を示す。
