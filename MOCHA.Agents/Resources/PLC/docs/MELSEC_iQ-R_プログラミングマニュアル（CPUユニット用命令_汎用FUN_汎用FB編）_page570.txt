### テキスト
#### 指定割込ポインタの禁止/許可（SIMASK）
指定した割込みポインタ番号の割込みプログラムを、実行許可状態または実行禁止状態に設定する命令です。

##### 機能
- (I)で指定した割込みポインタ番号の割込みプログラムを、(S)で指定したデータに従い、実行許可状態または実行禁止状態にします。
  - (S)が0の場合：割込みプログラムの実行禁止状態
  - (S)が1の場合：割込みプログラムの実行許可状態
- 電源投入時またはCPUユニットリセット時には、I0～I31、I44～I1023の割込みプログラムは実行許可状態、I32～I43の割込みプログラムは実行禁止状態になります。
- 割込みポインタの実行許可状態/実行禁止状態は、SD1400～SD1463（IMASK命令マスクパターン）に格納されます。

##### Point
(I)はインデックス修飾が可能です。インデックス修飾したSIMASK命令を使用することで、I0～I1023の割込みポインタを実行許可状態または実行禁止状態にできます。

### 図
- ラダー図：SIMASK命令のラダー表現。入力(I)、許可/禁止(S)、出力(ENO)の構成。
- FBD/LD図：SIMASK命令のFBD/LD表現。EN、I、S、ENOの端子が示されている。
