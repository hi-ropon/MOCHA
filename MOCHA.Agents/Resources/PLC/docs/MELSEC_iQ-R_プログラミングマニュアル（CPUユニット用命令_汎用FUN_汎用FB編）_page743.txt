### テキスト
#### 動作例
並べ替え元データ（データ行数5、データ列数4）を各条件で並び替えたときの動作例を示しています。

#### Point
1列目に管理番号などの通し番号を入れておくと、その内容で元の行番号が判断できます。

#### 例1
列番(n3)=2、SM774=OFF（昇順）での並び替え結果を示しています。

#### 例2
列番(n3)=3、SM774=ON（降順）での並び替え結果を示しています。

### 解説
- 元データの1列目は管理番号で、これを基に元の行番号を判断可能。  
- 例1は列番2（寸法X）を昇順で並び替えた結果を示し、管理番号の順序が変わっていることがわかる。  
- 例2は列番3（寸法Y）を降順で並び替えた結果を示し、同様に管理番号の順序が変わっている。  
- 並び替えの条件（列番と昇順/降順）によってデータの並びが変化する動作例となっている。
