### 表
#### 命令処理時間一覧
命令名ごとに、条件別の処理時間（μs）がCPUユニット別に示されています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各種命令の名称（例：DWSUM_U、MEAN、CRCなど） |
| 条件 | 命令実行時の条件（例：n=1、n=96、SM772のON/OFF状態、ファームウェアバージョンなど） |
| 処理時間（μs） | CPUユニット別の最小・最大処理時間。R00CPU、R01CPU、R02CPU、左記以外のCPUユニットに分かれている。 |

##### 主な内容
- DWSUM_U、MEAN、MEAN_U、DMEAN、DMEAN_U、SQRT、DSQRTなどの数学系命令の処理時間が条件（n=1、n=96）ごとに記載。
- CRC命令はSM772のON/OFF状態とnの値で処理時間が異なる。
- DBOPEN、DBCLOSE、DBINSERT、DBUPDATE、DBSELECT、DBDELETEなどのデータベース関連命令は、ファームウェアバージョンやs3の値によって処理時間が異なり、左記以外のCPUユニットのみ処理時間が記載されている。
- R00CPU、R01CPU、R02CPUについては多くのDB系命令で処理時間の記載がなく、左記以外のCPUユニットのみ処理時間が示されている。

##### 処理時間の例
- DWSUM_U (n=1) のR00CPU最小処理時間は3.900μs、最大は9.300μs。
- CRC (SM772=OFF, n=96) のR02CPU最大処理時間は19.300μs。
- DBINSERT (s3=16, ファームウェアバージョン"28"より前) の左記以外CPUユニット最大処理時間は50.100μs。

### テキスト
- 条件欄には命令実行時のパラメータや環境条件（例：nの値、SM772の状態、ファームウェアバージョン）が記載されている。
- ファームウェアバージョン"28"より前/以降で処理時間が異なる命令があることが注記されている。
- 一部命令は特定のCPUユニットでのみ処理時間が計測されている。
