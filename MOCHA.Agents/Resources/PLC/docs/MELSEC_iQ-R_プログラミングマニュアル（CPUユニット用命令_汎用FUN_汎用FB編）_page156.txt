### テキスト
#### 設定データ
■内容、範囲、データ型  
オペランドの種類ごとに内容、範囲、データ型（およびラベル）が説明されています。  
- (s): 接点として使用するデバイス（ビット型、ANY_BOOL）  
- EN: 実行条件（ビット型、BOOL）  
- ENO: 実行結果（ビット型、BOOL）  
※SIL2プロセスCPUおよび安全CPUの安全プログラムでは、上記の安全デバイスおよび安全ラベルのみ使用可能。

■使用可能デバイス  
ビット、ワード、ダブルワード、間接指定、定数、その他のデバイスについて、使用可能なデバイス種類が表形式で示されています。  
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、特定の安全デバイスおよび定数のみ使用可能。

#### 機能
■LDPI、LDIF  
- LDPI命令は立上りパルス否定演算開始命令で、指定ビットデバイスのOFF→ON、ON→OFFの変化に応じて導通する。  
- LDIF命令は立下りパルス否定演算開始命令で、指定ビットデバイスのOFF→ON、ON→OFFの変化に応じて導通する。  
- ワードデバイスのビット指定時は、指定ビットの0→1や1→0の変化に応じて導通。  
- STおよびFBD/LDで使用時のEN/ENOの動作について説明。  
- ENは常時ON指定が必要。  
- FBD/LD使用時はENに左母線または常時ONの変数部品/定数部品を指定。

■ANDPI、ANDFI  
- ANDPI命令は立上りパルス否定直列接続命令、ANDFI命令は立下りパルス否定直列接続命令で、AND演算結果を出力。  
- STおよびFBD/LD使用時のEN/ENOの動作について説明。  
- ENは実行条件ではない。
