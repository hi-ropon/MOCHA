### テキスト
#### 符号付きBIN32ビットデータ→単精度実数変換
命令DINT2FLT(P)は、符号付きBIN32ビットデータを単精度実数に変換します。  
対応するエンジニアリングツールのバージョンは1.035M以降です。

#### 実行条件
- DINT2FLT命令は立ち上がりエッジで実行されます。
- DINT2FLTP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー命令として、(s)に符号付きBIN32ビットデータ、(d)に変換後の単精度実数を指定します。  
- **FBD/LD図**  
  入力ENに実行条件、sに符号付きBIN32ビットデータ、dに単精度実数の出力を接続します。出力ENOは実行結果を示します。
