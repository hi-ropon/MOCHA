### テキスト
#### 機能
- (s1)で指定したループNo.の演算パラメータを、(s2)で指定したデバイス番号以降に格納されているPID制御用データに変更する。
- (s2)で指定したデバイス番号以降のPID制御用データの構成について説明。

#### エラー
- エラーコード(SD0)の説明。指定範囲外のデータ設定時やPID制御用データの設定範囲外、命令実行前の条件不足などのエラー内容を記載。

### 表
#### PID制御用データ
| 項目 | 内容 |
| :--- | :--- |
| (s2)+0 | 演算式選択 |
| (s2)+1 | サンプリング周期(Ts) |
| (s2)+2 | 比例定数(Kp) |
| (s2)+3 | 積分定数(Ti) |
| (s2)+4 | 微分定数(Td) |
| (s2)+5 | フィルタ係数(α) |
| (s2)+6 | 操作量下限値(MVLL) |
| (s2)+7 | 操作量上限値(MVHL) |
| (s2)+8 | 操作量変化率リミット値(ΔMVL) |
| (s2)+9 | 測定値変化率リミット値(ΔPVL) |
| (s2)+10 | 0 |
| (s2)+11 | 微分ゲイン(KD) |
| (s2)+12 | 0 |
| (s2)+13 | 0 |

#### エラーコード(SD0)
| エラーコード | 内容 |
| :--- | :--- |
| 3405H | (s1)に指定可能な範囲外のデータを設定したとき。<br>・指定したループNo.が存在しない<br>・指定した値が1～32以外<br>(s2)に指定可能な範囲外のデータを設定したとき。<br>・PID制御用データが設定可能範囲外<br>・PID制御用データの(s2)+10、(s2)+12、(s2)+13が0でない |
| 3422H | S(P).PIDPRMW命令実行前に、S(P).PIDINIT命令を実行していないとき。 |
