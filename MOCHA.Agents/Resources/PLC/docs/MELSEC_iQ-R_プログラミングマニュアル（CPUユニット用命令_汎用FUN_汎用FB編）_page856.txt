### テキスト
#### 機能
- 下限値(s1)の倍精度実数と上限値(s2)の倍精度実数の帯域に対し、比較データ(s3)の倍精度実数で比較します。
- 比較結果により、下限帯域内、上限帯域内、またはその外側に応じて(d)、(d)+1、(d)+2のいずれかのビットがONします。

#### 注意事項
- 下限値(s1)は上限値(s2)より小さい値に設定してください。
- もし下限値(s1)を上限値(s2)より大きな値に設定した場合、上限値(s2)は下限値(s1)と同じ値として扱われます。

#### エラー
| エラーコード(SD0) | 内容 |
| :--- | :--- |
| 3402H | (s1)～(s3)で指定したデータが0、非正規化数、非数、±∞のとき。 |

### 図（ラダー回路図）
- X0入力がONのとき、EDZCP命令が実行されます。
- (d)、(d)+1、(d)+2の3つのビットがあり、それぞれ以下の条件でONします。
  - (d): [(s1)～(s1)+3] > [(s3)～(s3)+3] + 3 のときON
  - (d)+1: [(s1)～(s1)+3] ≤ [(s3)～(s3)+3] + 3 かつ [(s2)～(s2)+3] の範囲内のときON
  - (d)+2: [(s3)～(s3)+3] > [(s2)～(s2)+3] のときON

- つまり、比較データ(s3)が下限値(s1)より小さい場合、(d)がON、帯域内なら(d)+1がON、上限値(s2)より大きい場合は(d)+2がONします。
