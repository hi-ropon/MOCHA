### テキスト
#### BIN32ビットブロックデータ減算 DBK-(P)(_U)
指定した2つのBIN32ビットデータブロックを減算する命令です。  
バージョン"1.035M"以降のエンジニアリングツールで対応しています。

#### ■実行条件
- DBK-、DBK-_U命令は立ち下がりエッジで実行
- DBK-P、DBK-P_U命令は立ち上がりエッジで実行

#### ■内容、範囲、データ型
- (s1) 減算されるデータまたは減算されるデータが格納されている先頭デバイス
  - 符号付きBIN32ビット：範囲 -2147483648～2147483647、データ型ANY32_S
  - 符号なしBIN32ビット：範囲 0～4294967295、データ型ANY32_U
- (s2) 減算データまたは減算データが格納されている先頭デバイス
  - 符号付きBIN32ビット：範囲 -2147483648～2147483647、データ型ANY32_S
  - 符号なしBIN32ビット：範囲 0～4294967295、データ型ANY32_U
- (d) 演算結果を格納する先頭デバイス
  - 符号付きBIN32ビットまたは符号なしBIN32ビット
- (n) 減算データ個数（0～65535）、符号なしBIN16ビット
- EN 実行条件（ビット）
- ENO 実行結果（ビット）

※ラベルで設定する場合は、動作に必要な領域が確保できる配列を定義し、その配列型ラベルの要素を指定する必要があります。

### 図
- ラダー図：s1, s2, d, nの4つの入力を持ち、ENOを出力する命令ブロックの構造を示す。
- FBD/LD図：EN入力、ENO出力、s1, s2, nの入力、dの出力を持つ命令ブロックの構造図。  
（DBKMINUS、DBKMINUSP、DBKMINUS_U、DBKMINUSP_Uのいずれかが入る）
