### テキスト
#### 入力フィルタ: (s3)+2
- 設定範囲は0～99[%]。
- PID制御の比例動作、積分動作、微分動作に影響を与えるソフトウェアフィルタ。
- 測定値（PV）のノイズによる変動を低減するためのフィルタで、入力フィルタのパラメータαを制御対象の特性やノイズレベルに合わせて設定することでノイズの影響を抑制可能。
- αが小さいとフィルタ効果が小さくなり、ノイズ除去効果が減少。
- αが大きすぎると入力応答が悪化する。
- 入力フィルタは目標値（SV）に作用し、比例動作、積分動作、微分動作に影響を与える。

#### 比例ゲイン: (s3)+3
- 設定範囲は1～32767[%]。
- PID制御の比例動作に関するパラメータ。
- 出力値（MV）は比例動作で偏差（目標値SVと測定値PVの差）に比例して増加。
- 比例ゲイン(Kp)はこの比例の割合を示し、出力値は「MV = 比例ゲイン(Kp) × 偏差(EV)」で表される。
- 比例ゲインの逆数を比例帯と呼ぶ。
- 比例ゲインが大きいほど、目標値に測定値を近づける動きが強くなる。

### グラフ
#### 入力フィルタの効果を示す図
- 左上：実際の測定値（PV）のステップ変化を示すグラフ。
- 左下：入力フィルタを通した後の測定値の応答曲線。ステップ応答が滑らかになる様子を示す。
- 右上：ノイズによるパルス入力の例。
- 右下：入力振幅と入力フィルタを通した後の振幅の比較。フィルタ通過後の振幅が小さくなっていることを示す。

#### 比例ゲインの違いによる制御動作の例（冷房の正動作）
- 上段グラフ：温度（℃）の時間変化。目標値（SV）に対する測定値（PV）の推移を示す。比例ゲインKp1 < Kp2 < Kp3の順に残留偏差が小さくなる様子を描写。
- 下段グラフ：出力値（MV）の時間変化。比例ゲインが大きいほど出力の初期値が高く、制御が強く働く様子を示す。
- グラフの横軸は時間(t)、縦軸は温度(℃)または出力(MV)。

### 用語説明
- ℃: 温度
- SV: 目標値
- PV: 測定値
- MV: 出力
- t: 時間
