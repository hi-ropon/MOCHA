### テキスト
#### BIN16ビット/32ビットデータゾーン制御
- ZONE、ZONEP、ZONE_U、ZONEP_U：指定したBIN16ビット値にバイアス値を付加する命令。
- DZONE、DZONEP、DZONE_U、DZONEP_U：指定したBIN32ビット値にバイアス値を付加する命令。
- 参照先はそれぞれ696ページ、698ページ。

#### BIN16ビット/32ビット単位スケーリング（ポイント別座標データ）
- SCL、SCLP、SCL_U、SCLP_U：16ビットデータ単位のスケーリング用変換データを用いて、指定入力値でスケーリングを行い、演算結果を指定デバイスに格納。
- DSCL、DSCLP、DSCL_U、DSCLP_U：32ビットデータ単位のスケーリング用変換データを用いて同様の処理を行う。
- スケーリング変換は、(s2)で指定したデバイス以降に格納したスケーリング用変換データに基づいて実施。
- 参照先は700ページ、703ページ。

#### BIN16ビット/32ビット単位スケーリング（X/Y別座標データ）
- SCL2、SCL2P、SCL2_U、SCL2P_U：16ビットデータ単位のX/Y別座標スケーリング用変換データを用いてスケーリングを実施。
- DSCL2、DSCL2P、DSCL2_U、DSCL2P_U：32ビットデータ単位のX/Y別座標スケーリング用変換データを用いて同様の処理を行う。
- 参照先は706ページ、708ページ。

#### データ処理命令
##### BIN16ビット/32ビットデータサーチ
- SERDATA、SERDATAP：BIN16ビットデータを対象に、(s2)で指定したBIN16ビットデータから(n)点分をサーチする命令。
- DSERDATA、DSERDATAP：BIN32ビットデータを対象に、(s2)で指定したBIN32ビットデータから(n)点分をサーチする命令。

### 図
- BIN16ビットデータサーチ命令の動作図：
  - (s1)で指定したデータを、(s2)で指定したデータ群から(n)点分サーチ。
  - (d)は一致した位置、(d)+1は一致個数を示す。
- BIN32ビットデータサーチ命令の動作図：
  - 32ビット単位で同様のサーチを行う。
  - (d)は一致した位置、(d)+1は一致個数。
