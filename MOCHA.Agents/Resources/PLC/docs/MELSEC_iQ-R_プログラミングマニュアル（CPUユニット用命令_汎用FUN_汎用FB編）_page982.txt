### テキスト
#### 系列変更 SRND(P)
指定したデバイスに格納されているBIN16ビットデータの内容に従って乱数系列を変更する命令です。SRND(P)命令は乱数の算出パターンを変更できます。

#### 機能
(s)で指定されたデバイスに格納されているBIN16ビットデータの内容に従って、乱数系列を変更します。

#### エラー
演算エラーはありません。

### 図
#### ラダー図
- (s)入力を受けてSRND命令を実行するラダー回路が示されている。

#### FBD/LD図
- EN入力を受けて、ENO出力を返すSRND命令のブロック図。

#### 実行条件
- SRND命令は立ち下がりエッジで実行される。
- SRNDP命令は立ち上がりエッジで実行される。
