### テキスト
#### 機能
(d)で指定したデバイスから(n1)ビット分のデータ範囲内で、右へ(n2)ビット分シフトさせる命令です。SM700では、シフト範囲から一つ右の1ビットが格納されます。

- (n1)は1～64で設定可能。
- (n1)と(n2)の関係は、(n1)>(n2)で指定し、(n1)≤(n2)の場合は(n2)÷(n1)の余りの値でシフトされる。ただし余りが0のときは無処理。
- 最上位から(n2)ビット分は0になる。
- (n1)または(n2)が0の場合は無処理。

#### 注意点
- (n1)と(n2)の値の範囲に注意が必要。

### 図
#### ビットシフトの例
- (n1)=10、(n2)=4の場合のビットシフト動作を示す。
- 上段は元のビット列(d)から(d)+9までの10ビットを示し、各ビットの値が1または0で表されている。
- 下段は右に4ビットシフトした結果を示し、シフト範囲外のビットは0に置き換わっている。
- SM700の場合は、シフト範囲の右隣の1ビットが別に格納されることを示す。
