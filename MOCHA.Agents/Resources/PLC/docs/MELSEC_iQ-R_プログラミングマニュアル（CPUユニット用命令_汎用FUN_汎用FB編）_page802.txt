### テキスト
#### BIN32ビットデータ→文字列変換
命令「DSTR(P)(_U)」は、BIN32ビットデータを指定した位置に小数点を付加して文字列に変換します。

#### 実行条件
命令ごとに実行条件の波形が示されており、DSTRおよびDSTR_Uは立ち下がりエッジ、DSTRPおよびDSTRP_Uは立ち上がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー回路図では、(s1)、(s2)、(d)の3つのオペランドが示されており、ST言語の命令形式も併記されています。  
  例：ENO:=DSTR(EN,s1,s2,d); ENO:=DSTR_U(EN,s1,s2,d); ENO:=DSTRP_U(EN,s1,s2,d);

- **FBD/LD図**  
  ブロック図では、EN（入力）、s1、s2、dの入力とENO（出力）が示されています。

| DSTRP      | 立ち上がりエッジ |
