### テキスト
#### 機能
(s)で指定されたBIN16ビットデータを(d)で指定されたデータテーブルの(n)番目に挿入します。FINS(P)命令実行後は、データテーブルの(n)番目からのデータが1個ずつ繰り下がります。
- N：データ数
- D_Tr：データテーブル範囲（ユーザで管理）
- (n)=2の場合、(d)+2に挿入されます。
- データ数(d)には0～65535が格納されます。

#### エラー
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3405H             | (n)の値が0でFINS(P)命令を実行したとき。<br>(d)の値がFFFFHでFINS(P)命令を実行したとき。 |

### 図
- 左側の図は、(d)の3番目の位置にデータがあり、その後ろに5432、1234、-123、0、0が続いているデータテーブルを示しています。
- (s)には4444が入っています。
- 右側の図は、(d)の4番目の位置に4444が挿入され、元のデータが1つずつ下にずれている様子を示しています。
- 矢印でデータの移動方向が示されており、データテーブルの(n)番目に新しいデータが挿入される動作を視覚的に説明しています。
