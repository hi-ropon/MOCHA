### テキスト
#### 倍精度実数n点の1点右シフト（EDSFR(P)）
この命令は、指定したデバイスから倍精度実数(n)点分を右へ1点シフトします。シフト後の空きデータには0が格納されます。

対応CPUは以下の通りです。
- R00CPU、R01CPU、R02CPUはファームウェアバージョン"06"以降で使用可能。エンジニアリングツールはバージョン"1.047Z"以降で使用可能。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン"31"以降で使用可能。エンジニアリングツールはバージョン"1.040S"以降で使用可能。

#### 命令の書式（ST）
```
ENO := EDSFR(EN, n, d);
ENO := EDSFRP(EN, n, d);
```

### 図
- ラダー図：命令のシンボルとパラメータ(d, n)の配置を示す。
- FBD/LD図：EN（入力）、ENO（出力）、n、dのポートを持つ命令ブロックの図。
- 実行条件：EDSFRは立ち上がりエッジ、EDSFRPは立ち下がりエッジで命令が実行されることを示す簡易波形図。
