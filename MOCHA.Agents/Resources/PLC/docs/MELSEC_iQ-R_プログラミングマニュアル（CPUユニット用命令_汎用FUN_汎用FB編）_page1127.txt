### テキスト
#### 使用可能デバイス
オペランドごとに使用可能なビット、ワード、ダブルワード、間接指定、定数、その他のデバイスが一覧表形式で示されています。  
- (U/H)、(s1)、(s2)、(n)の各モードにおける使用可否が○や―で表記されています。  
- 注釈として、DFROMD(P)命令は特定の条件下でのみ使用可能であることが記載されています。

#### 機能説明
(U/H)はユニットおよびCPUユニットの先頭入出力番号を16進数4桁で表したときの上3桁で指定することが説明されています。

### 図
ユニット構成図が示されており、PW、CPU、複数のRX10ユニットが並んでいます。  
- 各ユニットのアドレスが16進数で示されており、RJ71EN71ユニットは0040Hの位置にあります。  
- 図中の矢印(1)は読み出すユニットの先頭入出力番号がK4またはH4で指定されることを示しています。
