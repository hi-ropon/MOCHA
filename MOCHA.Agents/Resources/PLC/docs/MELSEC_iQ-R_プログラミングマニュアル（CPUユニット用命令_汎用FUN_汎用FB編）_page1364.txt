### テキスト
#### エラー
エラーコード(SD0)とその内容について説明しています。

##### エラーコード一覧
| エラーコード(SD0) | 内容                         |
|-------------------|------------------------------|
| 3401H             | (s)の各桁に0～9以外の値があるとき。 |

##### エラー発生時の対応
- SM754をONにすることで、上記エラーを出さないようにできます。
- ただし、範囲外の数値を設定した場合は、SM754のON/OFFに関わらず命令は実行されません。

このページは、BCD型からDINT型への変換に関するエラー処理の説明の一部であり、エラーコード3401Hの意味と対処方法を示しています。
