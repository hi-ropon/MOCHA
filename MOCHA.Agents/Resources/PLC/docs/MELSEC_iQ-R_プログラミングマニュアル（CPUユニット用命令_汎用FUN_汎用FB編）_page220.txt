### テキスト
#### 使用可能デバイス
比較演算命令で使用可能なデバイスの種類を示しています。オペランドごとにビット、ワード、ダブルワード、間接指定、定数、その他の対応状況が表形式でまとめられています。注釈として、FX、FYは使用不可、T、ST、Cも一部使用不可であることが記載されています。

#### 機能
(s1)で指定したBIN16ビットデータと(s2)で指定したBIN16ビットデータを比較し、その結果（小さい、一致、大きい）に応じて(d)、(d)+1、(d)+2のいずれかがONになる動作を説明しています。

#### エラー
演算エラーは発生しないことが明記されています。

### 図（グラフ）
- ラダー図：X0接点を起点にCMP命令が(s1)、(s2)、(d)のオペランドで構成されている。比較結果に応じて(d)、(d)+1、(d)+2がONになる条件が示されている。
- タイミングチャート：X0のON/OFFに連動して(s1)、(s2)、(d)、(d)+1、(d)+2の状態変化が時間軸で示されている。比較結果により(d)、(d)+1、(d)+2が一定時間保持される動作が視覚的に表現されている。
