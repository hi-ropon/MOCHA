### テキスト
#### 32.66 ワードラベルの指定ビット読出し
関数名：GET_BIT_OF_INT(_E)

この関数はワードラベルの指定ビットを読み出すためのものです。対応CPUはRnCPU、RnENCPU、各種プロセスCPU、一般CPU、安全CPUなど複数に対応しています。

##### 機能
- (s)の(n)ビット目を出力します。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
|-------------|--------------|-----|
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、その場合は(d)から出力されたデータを使用しないようプログラム処理を行う必要があります。

##### エラー
演算エラーはありません。

### 図
- ラダー回路図（EN/ENOなし）  
  入力s、n、出力dのシンプルな構成。

- ラダー回路図（EN/ENO付き）  
  入力EN、s、n、出力d、ENOを持つ構成。

### 表
#### 設定データ：内容、種別、データ型

| 引数 | 内容 | 種別 | データ型 |
|------|------|------|----------|
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s | 入力 | 入力変数 | INT |
| n | 0から15の定数のみ指定可能 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常実行、FALSE:異常または停止) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL |
