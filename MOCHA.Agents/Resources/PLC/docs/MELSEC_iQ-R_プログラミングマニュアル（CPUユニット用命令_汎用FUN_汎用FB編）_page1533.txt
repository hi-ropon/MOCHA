### 表
#### 命令処理時間一覧表
- 表は「命令名」「条件」「処理時間（μs）」の3つの大項目で構成されています。
- 処理時間はCPUユニット別に「R00CPU、R01CPU」「R02CPU」「左記以外のCPUユニット」の3区分に分かれ、それぞれ最小値と最大値が記載されています。
- 命令名はDWFSRTR、SFTDWL、DWSFTLなどの略称で示されており、条件欄には(n1)=16、(n2)=1や(n1)=16、(n2)=15などのパラメータが記載されています。
- 一部の命令は「実行時」や(s)=数値の条件が付いています。
- 処理時間は命令や条件によって異なり、例えばDWFSRTR命令の(n1)=16、(n2)=1条件ではR00CPUの処理時間は最小6.300μs、最大11.200μsとなっています。
- 表の下部にはBCD、DBCD、BINなどの命令もあり、こちらは実行時の処理時間が示されています。
- 数値は小数点以下5桁まで記載されており、非常に詳細な処理時間データとなっています。

以上より、本ページは各命令の処理時間をCPUユニット別・条件別に詳細に示した技術資料の一部であることがわかります。
</contents>
