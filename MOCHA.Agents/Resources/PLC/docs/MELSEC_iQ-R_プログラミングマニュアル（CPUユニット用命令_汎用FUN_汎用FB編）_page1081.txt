### テキスト
#### 機能
(s)で指定されたデバイス番号以降に格納されている秒のデータを、時・分・秒に換算し、その換算結果を(d)で指定されたデバイス以降に格納します。

#### 例
(s)に45325秒が指定された場合、これを時・分・秒に換算すると以下のようになります。
- (d)に12（時）
- (d)+1に35（分）
- (d)+2に25（秒）

### 図
- 秒のデータが(s)に格納されている様子を示す図
- 秒のデータを時・分・秒に分解し、それぞれ(d)、(d)+1、(d)+2に格納する流れを示す図
