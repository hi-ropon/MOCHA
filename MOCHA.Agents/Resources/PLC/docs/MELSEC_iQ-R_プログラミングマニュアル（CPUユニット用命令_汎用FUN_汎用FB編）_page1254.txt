### テキスト
#### 指定ループNo.の演算開始 PIDRUN(P)
プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります（詳細はMELSEC iQ-R CPUユニットユーザーマニュアル応用編参照）。

指定したループNo.の演算を開始する命令です。

#### 機能
- (s)で指定したループNo.のPID演算を開始します。
- PIDSTOP(P)命令で停止したループNo.を再度実行させるための命令です。
- すでにPID演算実行中のループNo.に対してPIDRUN(P)命令を実行した場合は無処理となります。

### 図
#### ラダー
- ラダー図では、(s)の値を入力し、ENOにPIDRUNの実行結果を出力する形で表現されています。

#### ST（構造化テキスト）
- ENO := PIDRUN(EN, s);
- ENO := PIDRUNP(EN, s);

#### FBD/LD（ファンクションブロック図/ラダーダイアグラム）
- EN（入力）、s（ループNo.）、ENO（出力）を持つブロックで表現。
