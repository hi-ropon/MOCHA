### テキスト
#### 機能
- 角度の大きさの単位を、(s)で指定された度単位（DEG.単位）からラジアン単位に変換し、(d)で指定されたデバイス番号に格納する。
- 度単位からラジアン単位への変換は以下の式で行う。
  - ラジアン単位 = 度単位 × (π / 180)
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細な注意事項は47ページを参照。

#### 注意点
- 角度の単位変換に関する簡単な図解があり、(s)+3、(s)+2、(s)+1、(s)の4つの度単位の値から、(d)+3、(d)+2、(d)+1、(d)の4つのラジアン単位の値に変換されるイメージが示されている。

### 表
#### エラーコード一覧
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)に指定した値が0、非正規化数、非数、±∞のとき。 |
| 3403H              | (d)が下記範囲を超えるとき。（オーバーフローが発生したとき）<br> |(d)| < 2^1024 |
