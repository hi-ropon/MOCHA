### テキスト
#### BIN32ビットブロックデータ比較(DBKCMP□(P)(_U))
指定した2つのBIN32ビットブロックデータを比較する命令です。  
括弧内にはDBKCMP=(P)(_U)、DBKCMP<>(P)(_U)、DBKCMP>(P)(_U)、DBKCMP>=(P)(_U)などの比較演算子が入ります。

##### 実行条件
- DBKCMP□(P)(_U)は立ち上がりエッジで実行
- DBKCMP□P(_U)は立ち下がりエッジで実行

##### 注釈
- バージョンが"1.035M"以降のエンジニアリングツールで対応
- 比較演算子はEQ(=)、NE(≠)、GT(>)、LE(<=)、LT(<)、GE(>=)が使用可能

### 図
#### ラダー図
- s1, s2, d, nの4つの入力を持ち、比較結果を出力する構造を示す
- EN（Enable）入力、ENO（Enable Output）出力あり

#### FBD/LD図
- EN入力、s1, s2, nの比較データ入力、dの比較結果出力、ENO出力を持つ命令ブロックの形状を示す

#### 設定データ：内容、範囲、データ型
| オペランド | 内容                             | 範囲                     | データ型           | データ型(ラベル) |
| --------- | -------------------------------- | ------------------------ | ------------------ | ---------------- |
| (s1)      | 比較データまたは格納先先頭デバイス | -2147483648～2147483647 (符号付きBIN32ビット) / 0～4294967295 (符号なしBIN32ビット) | 符号付き/符号なしBIN32ビット | ANY32_S*1 / ANY32_U*1 |
| (s2)      | 比較データが格納されている先頭デバイス | —                        | 符号付き/符号なしBIN32ビット | ANY32_S*1 / ANY32_U*1 |
| (d)       | 比較演算結果を格納する先頭デバイス   | —                        | ビット             | ANY_BOOL*1       |
| (n)       | 比較するデータ数                   | 0～65535                 | 符号なしBIN16ビット | ANY16            |
| EN        | 実行条件                         | —                        | ビット             | BOOL             |

*1 ラベルで設定する場合は、動作に必要な領域が確保できる配列を定義し、その配列型ラベルの要素を指定する必要あり。
