### テキスト
#### 単精度実数指数演算 EXP(P)
単精度実数で指定した値の指数を演算します。  
対応CPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）です。  
EXP命令はST（ステップ言語）で使用可能ですが、FBD/LDでは対応していません。FBD/LDでは汎用ファンクションのEXPを使用してください（1421ページ参照）。

#### 実行条件
- EXP命令はパルス信号で実行されます。
- EXPP命令は立ち上がりエッジで実行されます。

### 図
- ラダー回路：EXP命令の構文は「ENO:=EXPP(EN,s,d);」で、sが指数演算対象の単精度実数、dが演算結果の格納先です。
- FBD/LD：EN（実行条件）、ENO（実行結果）、s（入力値）、d（出力値）の端子を持つブロック図が示されています。
