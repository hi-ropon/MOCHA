### テキスト
#### 機能
- (s)で指定された角度のTAN（正接）値から角度を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- 演算結果の角度はラジアン単位で格納される。
- 演算結果が0、またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### 注意事項
- ラジアンと角度変換については、RAD(P)命令、DEG(P)命令を参照すること。
- 参照ページは941ページ（RAD(P)）、943ページ（DEG(P)）。

### 図解
- (s)+1、(s)の単精度実数からTAN⁻¹（逆正接）を計算し、(d)+1、(d)の単精度実数に結果を格納する流れを示す簡単な図。
