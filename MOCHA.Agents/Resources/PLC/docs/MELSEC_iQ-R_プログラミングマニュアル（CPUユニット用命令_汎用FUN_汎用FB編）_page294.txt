### テキスト
#### 32ビットBINデータデクリメント DDEC(P)(_U)
指定したBIN32ビットデータを-1します。

#### 実行条件
- DDEC、DDEC_U：立ち下がりエッジで実行
- DDECP、DDECP_U：立ち上がりエッジで実行

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、記載されている安全デバイスおよび安全ラベルのみ使用可能。

### 図
#### ラダー図
- (d)デバイスに対して-1を行う命令ブロックの図示。

#### ST（構造化テキスト）
- ENO := DDEC(EN,d);
- ENO := DDECP(EN,d);
- ENO := DDEC_U(EN,d);
- ENO := DDECP_U(EN,d);

#### FBD/LD（ファンクションブロック図/ラダーダイアグラム）
- EN（入力）、ENO（出力）、d（デバイス）を持つ命令ブロックの図示。
