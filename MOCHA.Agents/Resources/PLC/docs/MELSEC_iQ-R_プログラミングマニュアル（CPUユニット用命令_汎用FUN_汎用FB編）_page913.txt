### テキスト
#### 単精度実数COS⁻¹演算（ACOS(P)）
単精度実数で指定したCOS値から角度を演算します。

##### ■命令対応CPU
RnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）

##### ■実行条件
- ACOS命令はラダー（ST）やFBD/LDでは対応していません。汎用ファンクションのACOSを使用してください。
- ACOS命令は立下りエッジで実行されます。
- ACOSP命令は立上りエッジで実行されます。

### 図
- ラダー回路図：COS⁻¹演算の入力(s)と出力(d)を示すシンボル
- FBD/LD図：EN（入力）、ENO（出力）、s（COS値入力）、d（演算結果出力）のブロック図
