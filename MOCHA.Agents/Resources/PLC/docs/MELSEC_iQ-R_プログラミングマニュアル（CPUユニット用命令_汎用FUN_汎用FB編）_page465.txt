### テキスト
#### 機能
(s)で指定されたデバイスに格納されているBIN32ビットグレイコードデータを、BIN32ビットデータに変換し、(d)で指定されたデバイスに格納します。

##### ビット配置の説明
- (s)のデバイスに格納されているデータは、ビット31からビット0までの32ビットで構成されています。
- (s)+1のデバイスも同様に32ビットのデータを持ちます。
- 変換後の(d)のデバイスも同様に32ビットのデータで、(d)+1のデバイスも32ビットのデータを持ちます。
- (s)の上位16ビットと下位16ビットのビット配置を示し、(d)の対応するビット配置に変換される様子が図で表されています。

### 表
#### エラー
| エラーコード (SD0) | 内容 |
| :--- | :--- |
| 3401H | DGBIN(P)命令使用時、(s)で指定した値が0～2147483647以外のとき。 |

このエラーは、DGBIN(P)命令を使用する際に、(s)で指定した値が許容範囲外の場合に発生します。
