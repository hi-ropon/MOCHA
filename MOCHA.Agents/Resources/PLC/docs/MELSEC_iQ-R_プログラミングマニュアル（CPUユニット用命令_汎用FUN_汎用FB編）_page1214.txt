### テキスト
#### 動作設定: (s3)+1
- ■動作方向(正動作/逆動作): (s3)+1 ビット0  
  設定範囲はOFFが正動作、ONが逆動作。  
  正動作または逆動作を選択する。  
  - オートチューニング(リミットサイクル法)の場合は、PID制御したい方向を設定する必要がある。  
  - オートチューニング(ステップ応答法)の場合は、正動作または逆動作のいずれで実行しても完了時に自動で設定される。  
  設定例: [正動作((s3)+1 ビット0=OFF)]、[逆動作((s3)+1 ビット0=ON)]

- ■警報設定(入力変化量、出力変化量): (s3)+1 ビット1、2  
  設定範囲はOFFが警報なし、ONが警報有効。  
  入力変化量、出力変化量のチェックが可能。チェック結果は(s3)+24で確認できる。  
  入力変化量警報を使用する場合は、該当ビットをONにし、警報値を設定する必要がある。

### 表
#### 入力変化量警報設定
| 設定項目           | 内容                     | 設定範囲               |
|--------------------|--------------------------|------------------------|
| 動作設定(ACT)      | (s3)+1 ビット1           | ON: 使用する / OFF: 使用しない |
| 入力変化量警報設定値 | (s3)+20 入力変化量(増側)警報設定値 | 0～32767               |
|                    | (s3)+21 入力変化量(減側)警報設定値 |                        |

#### 出力変化量警報設定
| 設定項目           | 内容                     | 設定範囲               |
|--------------------|--------------------------|------------------------|
| 動作設定(ACT)      | (s3)+1 ビット2           | ON: 使用する / OFF: 使用しない |
|                    | (s3)+1 ビット5           | 出力値上下限設定（必ずOFFに設定） |
| 出力変化量警報設定値 | (s3)+22 出力変化量(増側)警報設定値 | 0～32767               |
|                    | (s3)+23 出力変化量(減側)警報設定値 |                        |

### 補足
- 入力変化量警報は(s3)+1のビット1をONにして使用し、増側・減側の警報値をそれぞれ設定する。  
- 出力変化量警報は(s3)+1のビット2をONにして使用し、ビット5は必ずOFFに設定する必要がある。  
- 警報の設定値は0から32767の範囲で設定可能。
