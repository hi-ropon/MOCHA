### テキスト
#### 1.3 実行条件
命令の実行条件には複数の種類があり、それぞれの条件に応じて命令の実行タイミングや動作が異なります。

#### 各命令の実行条件
命令ごとに実行条件が異なり、ST言語、FBD/LD言語ではENが実行条件となり、ENがTRUEの場合のみ命令が実行されます。ENはEN0と同じ状態です。汎用ファンクションやファンクションブロックではENの有無で実行条件が異なり、ENなしの場合は常時実行となります。EN付きの場合は該当ページを参照する必要があります。

### 表
#### 実行条件の種類

| 実行条件     | 説明                                                                                              |
|--------------|-------------------------------------------------------------------------------------------------|
| ON時実行     | ON中実行形の命令で、命令の前条件がONの間だけ命令を実行。前条件がOFFの場合は実行せず処理しない。   |
| 立上り実行   | ON時1回実行形の命令で、前条件の立上り時(OFF→ON)だけ命令を実行。以降条件がONでも実行しない。       |
| OFF時実行    | OFF中実行形の命令で、前条件がOFFの間だけ命令を実行。前条件がONの場合は実行せず処理しない。          |
| 立下り実行   | OFF時1回実行形の命令で、前条件の立下り時(ON→OFF)だけ命令を実行。以降条件がOFFでも実行しない。       |
| 常時実行     | 前条件のON/OFFに関係なく常に実行。前条件がOFFの場合はOFF処理を行う。                              |

*注記*  
ST言語、FBD/LD言語では命令中のENが命令の前条件となります。
