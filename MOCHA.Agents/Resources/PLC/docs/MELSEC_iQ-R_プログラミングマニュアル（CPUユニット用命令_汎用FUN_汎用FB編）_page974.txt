### テキスト
#### 機能
- (s)から(n)点分の単精度実数のブロックデータに対して最大値を検索し、(d)で指定したデバイスへ格納する。
- 最初に最大値を検出したデータを(d)+2で指定したデバイスへ格納し、最大値と同じ値の個数を(d)+3で指定したデバイスへ格納する。
- 検索結果（位置）の値は、(s)で指定したブロックデータの先頭を1点目として扱う。
- (n)で指定した値が0の場合は無処理となる。

#### エラー
- エラーコード「3402H」： (s)で指定したブロックデータに単精度実数以外の値が存在する場合に発生。

### 図
- 左側の表は、(s)から(n)点分のブロックデータの例を示している。数値は1.2345、123.45、-1.2345、-12.345、-123.45の5点。
- 右側の表は、(d)で指定したデバイスに格納される値を示す。最大値123.45が(d)+1に格納され、最大値の位置2が(d)+2に、最大値と同じ値の個数1が(d)+3に格納される。
