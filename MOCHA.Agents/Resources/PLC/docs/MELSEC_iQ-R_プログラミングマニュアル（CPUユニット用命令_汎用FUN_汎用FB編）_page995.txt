### テキスト
#### ファイルレジスタの1バイトデータリード命令 ZRRDB(P)
指定した通しバイトNo.のファイルレジスタの内容を読み出す命令です。

#### 実行条件
- ZRRDB命令は立下りエッジで実行されます。
- ZRRDBP命令は立上りエッジで実行されます。

### 図
#### ラダー図
- (s)に実行条件を入力し、(d)に読み出したデータを格納します。
- ST言語では、ENO:=ZRRDB(EN,s,d); または ENO:=ZRRDBP(EN,s,d); と記述します。

#### FBD/LD図
- ブロック内にEN（実行条件）、ENO（実行結果）、s（通しバイトNo.）、d（データ格納先）があります。
