### テキスト
#### BIN32ビット減算
指定した2つのBIN32ビットデータを減算する命令について説明しています。

#### 対応CPU
- RnCPU、RnENCPU（プロセス、三重化、一般、安全）に対応。

#### 実行条件
- 命令D-、D-_Uは立ち上がりエッジで実行。
- 命令D-P、D-P_Uは立ち下がりエッジで実行。

#### 注意事項
- ST命令は対応していません（3オペランドの場合は別ページ参照）。
- FBD/LD命令も対応していません。

#### 安全プログラムでの使用制限
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能。

### 図
- ラダー回路の簡単な図があり、(s)と(d)の2つのオペランドを示す形で減算命令の構成を表現。
