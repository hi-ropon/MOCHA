### テキスト
#### ORP、ORF命令について
- ORP命令は立上りパルス並列接続命令、ORF命令は立下りパルス並列接続命令であり、それまでの演算結果とOR演算を行い演算結果とします。
- ORP命令、ORF命令で使用するON/OFF情報は表に示されています。
- ORP命令をSTおよびFBD/LDで使用した場合、ENと指定ビットデバイスの立上りのOR演算結果がONの時、ENOはONとなるが、EN自体は実行条件にはならない。
- ORF命令をSTおよびFBD/LDで使用した場合、ENと指定ビットデバイスの立下りのOR演算結果がONの時、ENOはONとなるが、EN自体は実行条件にはならない。

#### LDP、LDF、ANDP、ANDF、ORP、ORFの組み合わせによる動作
- これらの命令の組み合わせによる動作例は、LD、AND、ORの組み合わせによる動作例と同様である。
- 詳細は150ページのLD、LDI、AND、ANI、OR、ORIの説明を参照。

#### エラー
- 演算エラーはありません。
