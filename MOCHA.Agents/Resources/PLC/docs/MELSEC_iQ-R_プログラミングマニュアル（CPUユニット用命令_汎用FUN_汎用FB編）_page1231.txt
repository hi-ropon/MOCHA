### テキスト
#### 入出力データ
入出力データとは、PID演算を行うために設定する入力データ（設定値SV、測定値PVなど）と、演算結果の出力データのことです。ワードデバイスの任意の番号を指定可能ですが、使用ループ分のすべてのデータを連続したデバイス番号に設定する必要があります。入出力データの割付けは、関連ページ（不完全微分1239ページ、完全微分1250ページ）を参照してください。各ループに割り付けられた項目とPID演算用のワークエリアが存在します。

### 表
#### 入出力データ項目詳細

| 項目           | 内容                                                                                   | 設定範囲                     | 備考                                                                                         |
|----------------|----------------------------------------------------------------------------------------|------------------------------|----------------------------------------------------------------------------------------------|
| 設定値 (SV)    | PID制御の目標値です。                                                                   | 0～2000*1<br>-32768～32767*2 | "PIDリミット制限あり"では、範囲外の値は0または2000に変換されます。                          |
| 測定値 (PV)    | 制御対象からA/D変換ユニットにフィードバックされたデータです。                           | -50～2050*1<br>-32768～32767*2| "PIDリミット制限あり"では、範囲外の値は-50または2050に変換されます。                        |
| 自動操作量 (MV)| PID演算で算出した操作量で、D/A変換ユニットから制御対象へ出力します。                   | -50～2050*1<br>-32768～32767*2| なし                                                                                         |
| フィルタ後の測定値 (PVf) | 演算式で算出した測定値。PVf = PVn + α(PVn-1 - PVn)                                  | -50～2050*1<br>-32768～32767*2| なし                                                                                         |
| 手動操作量 (MV_MAN) | 手動時にD/A変換ユニットから出力するデータを格納します。                              | -50～2050*1<br>-32768～32767*2| "PIDリミット制限あり"では、範囲外の値は-50または2050に変換されます。                        |
| 手動/自動選択 (MAN/AUTO) | D/A変換ユニットへの出力データを手動操作量か自動操作量か選択します。             | 0:自動操作量<br>1:手動操作量 | 0以外はエラーとなり、該当ループのPID演算は実行されません。                                  |
| アラーム (ALARM) | 手動操作量(MV)、測定値(PV)の変化率がリミット値の範囲外かの判別用。1度セットするとリセットまで保持。 | なし                         | PVが範囲外のときb0が1、MVが範囲外のときb1が1になります。                                   |

*1 "PIDリミット制限あり"設定時の範囲  
*2 "PIDリミット制限なし"設定時の範囲

### 図（表内説明）
- アラームのビット説明：  
  b0ビットは測定値(PV)がリミット範囲外のとき1になる。  
  b1ビットは自動操作量(MV)がリミット範囲外のとき1になる。
