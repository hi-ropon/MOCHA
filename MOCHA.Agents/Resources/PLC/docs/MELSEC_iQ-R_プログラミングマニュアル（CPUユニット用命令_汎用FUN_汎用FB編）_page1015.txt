### テキスト
#### 15.1 近回り制御命令
ロータリテーブルの近回り制御について説明しています。指定した値で等分割されたロータリテーブル上で、ロータリテーブルを近回りで回転させる制御を行う命令「ROTC」について記載されています。

### 図
- **ラダー図**  
  ROTC命令のラダー図が示されており、パラメータとして(s)、(n1)、(n2)、(d)が入力されている様子が描かれています。  
  ST言語では「ENO:=ROTC(EN,s,n1,n2,d);」と表記されます。

- **FBD/LD図**  
  ROTC命令のFBD/LD図が示されており、EN（実行条件）、s、n1、n2が入力、ENO（実行結果）、dが出力として表現されています。

### 表
#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| ROTC | 立ち上がりエッジ |

#### 設定データ（内容、範囲、データ型）
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (s) | (s)+0: テーブル回転数測定用(システム用) | — | 符号付きBIN16ビット | ANY16_ARRAY (要素数:3) |
|  | (s)+1: 呼出し窓口番号 |  | 符号なしBIN16ビット |  |

| (n1) | テーブル分割数 | 2〜65535 | 符号なしBIN16ビット | ANY16 |
| (n2) | 低速区間数 | 0〜(n1)未満 | 符号なしBIN16ビット | ANY16 |
| (d) | (d)+0: A相入力信号 | — | ビット | ANYBIT_ARRAY (要素数:8) |
|  | (d)+1: B相入力信号 |  |  |  |

以上の内容から、ROTC命令はロータリテーブルの回転を近回りで制御するための命令であり、複数のパラメータを用いて回転数や分割数、入力信号などを指定し、実行条件に基づいて動作します。
