### テキスト
#### BIN16ビットデータサーチ（最小・同一・最大）
SERMM(P)命令は、(s1)で指定したBIN16ビットデータの範囲から、(s2)で指定したBIN16ビットデータと同一のデータおよび最大値、最小値を検索します。

- 対応CPUはR00CPU、R01CPU、R02CPUで、バージョンによる制約はありません。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョンが17以降で使用可能です。
- エンジニアリングツールはバージョン1.020W以降で使用可能です。

#### 命令の動作
- EN（実行条件）がONのときに命令が実行されます。
- SERMMは立ち下がりエッジで実行。
- SERMMPは立ち上がりエッジで実行。

### 図
#### ラダー図
- (s1)、(s2)、(d)、(n)の4つの入力を持つ命令ブロックが示されています。
- ST言語では、`ENO:=SERMM(EN,s1,s2,n,d);` または `ENO:=SERMMP(EN,s1,s2,n,d);` と記述。

#### FBD/LD図
- EN、s1、s2、nを入力、ENOとdを出力とする命令ブロックの図。
