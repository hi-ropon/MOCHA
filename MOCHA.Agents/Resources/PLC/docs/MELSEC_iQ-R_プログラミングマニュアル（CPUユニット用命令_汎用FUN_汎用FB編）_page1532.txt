### 表
#### 命令処理時間一覧表
この表は各命令名ごとに、条件別の処理時間をCPUユニット別に最小値と最大値で示しています。CPUユニットは「R00CPU、R01CPU」「R02CPU」「左記以外のCPUユニット」の3種類に分かれています。

| 項目 | 内容 |
| :--- | :--- |
| 命令名 | 各命令の名称（例：BSET、BRST、TESTなど） |
| 条件 | 命令の実行条件。例えば、(n)=1や(n)=15などの繰り返し回数や「実行時」など |
| 処理時間(μs) | CPUユニットごとの処理時間。最小値と最大値が記載されている。 |

##### 主な命令例と処理時間
- BSET、BRST：(n)=1および(n)=15の条件で、R00CPU、R01CPUは約0.094μs、R02CPUは約0.012μs、その他CPUは約0.003μs。
- TEST：実行時にR00CPU、R01CPUで約0.384μs、R02CPUで約0.048μs、その他CPUで約0.012μs。
- BKRST：(n)=1でR00CPU、R01CPUは3.0～3.5μs、R02CPUは1.1～2.3μs、その他CPUは0.9～2.0μs。
- SFR、SFL：(n)=1および(n)=15で約0.28μs（R00CPU、R01CPU）、約0.035μs（R02CPU）、約0.0088μs（その他CPU）。
- BSFR、BSFL、DSFR、DSFLなどの命令は(n)=1と(n)=96の条件で処理時間が大幅に異なり、最大で10μs以上かかるものもある。
- SFTBR、SFTR、SFTBL、SFTLなどの命令は(n1)=16、(n2)=1または15の条件で処理時間が示されている。
- WSFR、WSFL、SFTWR、SFTWL、SFTDWRなども同様に条件別の処理時間が記載されている。

この表はCPUの種類や命令の繰り返し回数によって処理時間が大きく変動することを示しており、リアルタイム制御や性能評価に重要な情報となっています。
