### テキスト
#### 機能
(d)で指定したデバイスから(n)点分のデータを、左へ1ワードシフトさせる動作について説明しています。シフト後、最下位の1ワード分は0になります。

#### エラー
演算エラーは発生しないことが明記されています。

### 図
データシフトの動作を示す図があり、上段に元のデータ配列が(d)+(n)-1から(d)まで並び、下段に左へ1ワードシフトした結果が示されています。具体例として、元データの一部に「555」「120」「325」「100」「50」「40」があり、シフト後は「120」「325」「100」「50」「40」「0」となっています。これにより、最下位のデータが0に置き換わることが視覚的に理解できます。
