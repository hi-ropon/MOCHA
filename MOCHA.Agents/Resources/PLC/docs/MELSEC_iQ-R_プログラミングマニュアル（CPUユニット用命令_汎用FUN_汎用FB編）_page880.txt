### テキスト
#### 機能
(s1)で指定された倍精度実数を(s2)で指定された倍精度実数で除算し、その結果を(d)で指定されたデバイスに格納します。  
- (s1)、(s2)、(d)にはそれぞれ0から2^1024未満の値が指定・格納可能です。  
- 演算結果が0未満またはアンダーフローが発生した場合、結果は0となります。  
- エンジニアリングツールから入力値を設定する際、丸め誤差が発生する場合があります。詳細は47ページの注意事項を参照してください。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3400H | (s2)で指定したデータ（除数）が0のとき。 |
| 3402H | (s1)で指定したデータが0、非正規化数、非数、±∞のとき。<br>(s2)で指定したデータが0、非正規化数、非数、±∞のとき。 |
| 3403H | (d)が指定範囲を超えるとき（オーバーフローが発生したとき）。<br>|d| < 2^1024 |

### 図
浮動小数点数の除算におけるデバイス配置を示す図。  
- (s1)の倍精度実数は4つの連続したデバイス(s1)、(s1)+1、(s1)+2、(s1)+3に格納されている。  
- (s2)も同様に4つの連続したデバイスに格納。  
- 除算結果は(d)、(d)+1、(d)+2、(d)+3の4つのデバイスに格納される。  
この図は、倍精度実数の各部分がどのデバイスに対応しているかを視覚的に示しています。
