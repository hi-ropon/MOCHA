### テキスト
#### データシフト命令
BIN16ビットデータやnビットデータ、nワードデータの右シフトおよび左シフトを行う命令について説明しています。

##### ■BIN16ビットデータのnビット右シフト、左シフト
- 指定したデバイスのBIN16ビットデータを右または左にシフトします。
- シフト後の空きビットには0が格納されます。

##### ■nビットデータの1ビット右シフト、左シフト
- 指定したデバイスからn点分を右または左に1ビットシフトします。
- シフト後の空きビットには0が格納されます。

##### ■nワードデータの1ワード右シフト、左シフト
- 指定したデバイスからn点分を右または左に1ワードシフトします。
- シフト後の空きワードには0が格納されます。

### 表
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| SFR / SFRP | 指定したデバイスのBIN16ビットデータを右シフト。空きビットは0。 | 346ページ SFR(P) |
| SFL / SFLP | 指定したデバイスのBIN16ビットデータを左シフト。空きビットは0。 | 348ページ SFL(P) |
| BSFR / BSFRP | 指定したデバイスからn点分を右へ1ビットシフト。空きビットは0。 | 350ページ BSFR(P) |
| BSFL / BSFLP | 指定したデバイスからn点分を左へ1ビットシフト。空きビットは0。 | 352ページ BSFL(P) |
| DSFR / DSFRP | 指定したデバイスからn点分を右へ1ワードシフト。空きワードは0。 | 354ページ DSFR(P) |
| DSFL / DSFLP | 指定したデバイスからn点分を左へ1ワードシフト。空きワードは0。 | 356ページ DSFL(P) |

### 図
- BIN16ビットデータのシフト図：16ビットのビット列が右または左にシフトし、空いたビットは0で埋められる様子を示す。
- nビットデータの1ビットシフト図：n点分のビット列が右または左に1ビットずつシフトし、空いたビットは0で埋められる様子を示す。
- nワードデータの1ワードシフト図：n点分のワード単位のデータが右または左に1ワードずつシフトし、空いたワードは0で埋められる様子を示す。

以上の内容により、各命令のシフト動作とその結果のビット/ワードの状態が理解できます。
