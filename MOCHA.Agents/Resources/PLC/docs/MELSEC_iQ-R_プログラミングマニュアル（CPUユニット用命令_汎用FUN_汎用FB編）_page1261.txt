### テキスト
#### 他号機CPUユニットからのデバイス読出し
- 対象命令はD(P).DDRDおよびM(P).DDRD。
- RnENCPIUはファームウェアバージョン"25"以降、エンジニアリングツールはバージョン"1.032J"以降で使用可能。
- RnSFCPU(一般)はファームウェアバージョン"17"以降、エンジニアリングツールはバージョン"1.055H"以降で使用可能。
- マルチCPUシステム構成時に、他号機CPUユニットからデバイスデータを読み出すために使用される。

### 図
#### ラダー図
- (U/H)、(s1)、(s2)、(d1)、(d2)の各パラメータを持つ命令ブロックの構造を示す。
- EN（実行条件入力）、ENO（実行結果出力）端子がある。

#### FBD/LD図
- EN、U/H、s1、s2を入力、ENO、d1、d2を出力とする命令ブロックの構造を示す。

### 表
#### 実行条件
| 命令       | 実行条件の波形例               |
|------------|-------------------------------|
| D.DDRD     | 立ち下がりエッジで実行        |
| M.DDRD     | 立ち下がりエッジで実行        |
| DP.DDRD    | 立ち上がりエッジで実行        |
| MP.DDRD    | 立ち上がりエッジで実行        |

#### 設定データ（内容、範囲、データ型）
| オペランド | 内容                                         | 範囲          | データ型           | データ型（ラベル）           |
|------------|----------------------------------------------|---------------|--------------------|------------------------------|
| (U/H)      | 他号機CPUユニットの先頭入出力番号（16進数4桁の上3桁） | 3E0H～3E3H    | 符号なしBIN16ビット | ANY16                        |
| (s1)       | コントロールデータを格納している自号機CPUユニットの先頭デバイス | コントロールデータ参照 | ワード               | ANY16_ARRAY（要素数:2）       |
| (s2)       | 読み出すデータを格納する他号機CPUユニットの先頭デバイス | なし          | 文字列             | ANYSTRING_SINGLE             |
| (d1)       | 読み出したデータを格納する自号機CPUユニットの先頭デバイス | なし          | ワード             | ANY16*1                      |
| (d2)       | 完了デバイス                                   | なし          | ビット             | ANYBIT_ARRAY（要素数:2）      |
| EN         | 実行条件                                     | なし          | ビット             | BOOL                         |
| ENO        | 実行結果                                     | なし          | ビット             | BOOL                         |

*1 ラベルで設定する場合は、動作に必要な領域が確保できるよう配列を定義し、その配列型ラベルの要素を指定する必要がある。
