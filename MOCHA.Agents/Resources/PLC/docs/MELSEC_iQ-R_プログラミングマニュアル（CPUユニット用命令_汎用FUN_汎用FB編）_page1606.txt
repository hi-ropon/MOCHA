### 表
#### GX Works2命令と互換用命令の対応一覧
- 表は4つの列で構成されている。
  1. **GX Works2命令名**: 元の命令名が記載されている。
  2. **互換用命令名**: GX Works2命令に対応する互換用の命令名が示されている。
  3. **使用プログラム**: ST（Structured Text）とFBD（Function Block Diagram）での使用可否を「〇（使用可）」「×（使用不可）」で示している。
  4. **参照先**: 各命令の詳細説明が記載されているページ番号と該当箇所が記載されている。

- 命令の例としては「D/」「D/P」「DABIN」「DB*」「DBK+」などがあり、それぞれに対応する互換用命令名が設定されている。
- 使用プログラムの欄では、STで使用不可でもFBDでは使用可能な命令が多く見られる。
- 参照先はページ番号と命令名の略称が記載されており、詳細な説明は該当ページを参照する形式。
