### テキスト
#### BIN32ビットデータの左ローテーション
- DROL(P)命令は、BIN32ビットデータをキャリフラグを含めずに(n)ビット左へ回転させます。
- DRCL(P)命令は、BIN32ビットデータをキャリフラグを含めて(n)ビット左へ回転させます。

#### 注意事項
- DROL命令はST、FBD/LDでは対応していません。汎用ファンクションのROLを使用してください。（1447ページ参照）

#### 実行条件
| 命令 | 実行条件 |
| --- | --- |
| DROL, DRCL | 立ち上がりエッジ（パルス） |
| DROLP, DRCLP | 立ち下がりエッジ（パルス） |

### 図
- ラダー図：DROL(P)、DRCL(P)命令のシンボルが示されており、(d)がローテーションする先頭デバイス、(n)がローテーションさせるビット数を表す。
- FBD/LD図：EN（入力）、ENO（出力）、n、dのポートを持つブロック図が示されている。
