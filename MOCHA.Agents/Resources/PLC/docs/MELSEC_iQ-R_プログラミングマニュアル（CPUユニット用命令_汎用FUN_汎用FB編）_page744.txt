### テキスト
#### 注意事項
- 動作中はオペランドやデータの内容を変化させないこと。並び替え完了時に正しい順序にならない場合がある。
- (s)と(d)に同一デバイスを指定した場合、並び替え元のデータは並び替え後のデータ順に書き換えられる。そのため、実行完了（SD774のビット6がONする）まで(s)の内容を変化させてはいけない。
- (s)と(d)を指定する際は、同一デバイス範囲を指定するか、範囲が重複しないデバイスを指定すること。

### 図
- 3つの図はそれぞれ、元データと並び替え後のデータの範囲関係を示している。
  1. 左図：元データD10～D30、並び替え後のデータもD10～D30で範囲が同じ。○（正しい）
  2. 中図：元データD10～D30、並び替え後のデータD35～D55で範囲が異なるが重複しない。○（正しい）
  3. 右図：元データD10～D30、並び替え後のデータD2～D22で範囲が重複している。×（誤り）
