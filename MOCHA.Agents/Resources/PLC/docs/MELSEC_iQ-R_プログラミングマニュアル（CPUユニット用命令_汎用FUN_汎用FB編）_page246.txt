### テキスト
#### D-(P)(_U)命令（オペランドが3個の場合）
指定した2つのBIN32ビットデータを減算する命令です。

#### 実行条件
- D-、D-_U命令は立ち下がりエッジで実行されます。
- D-P、D-P_U命令は立ち上がりエッジで実行されます。

#### 注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**  
  ラダー回路図では、(s1)、(s2)の2つの入力ビットと(d)の出力ビットが示されており、D-(P)(_U)命令の動作を表しています。

- **FBD/LD図**  
  FBD/LDの命令ブロック図には、EN（実行条件）、s1、s2（入力）、d（出力）、ENO（実行結果）が示されています。  
  （口には、DMINUS、DMINUSP、DMINUS_U、DMINUSP_Uのいずれかが入ることが記載されています）
