### テキスト
#### コネクションの確立 - SP.SOCOPEN命令
指定したコネクションのオープン処理を行う命令です。

- ラダー図では、(U)、(s1)、(s2)、(d)の各オペランドが示されています。
- ST言語では、`ENO:=SP_SOCOPEN(EN,U,s1,s2,d);`と記述します。
- FBD/LD図では、EN、ENO、U、s1、s2、dの各端子が示されています。

#### 実行条件
- 命令：SP.SOCOPEN
- 実行条件は特に指定なし（図示は空欄）
