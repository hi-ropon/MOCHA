### テキスト
#### BCDフォーマットデータ→単精度実数データ変換
命令名：EREXP(P)

BCD型浮動小数点フォーマットデータを指定された小数部桁数に基づいて単精度実数データに変換します。

#### 実行条件
- EREXP命令は立ち上がりエッジで実行されます。
- EREXPP命令は立ち下がりエッジで実行されます。

### 図
#### ラダー図
- (s1)、(s2)、(d)の3つのオペランドを持つ命令ブロックの形状が示されています。
- ST言語では、ENO:=EREXP(EN,s1,s2,d); または ENO:=EREXP(EN,s1,s2,d); の形式で記述されます。

#### FBD/LD図
- 入力EN、s1、s2、出力ENO、dの接続を示す命令ブロックの図が描かれています。
