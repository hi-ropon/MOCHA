### テキスト
#### ビットデバイス出力反転（FF命令）
指定されたデバイスの状態を反転させる命令です。  
対応CPUはRnCPU、RnENCPU（プロセス・二重化）、RnSFCPU（一般・安全）となっています。

#### 実行条件
- 命令：FF
- 実行条件は特に指定なし（常に実行）

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定されたデータ型の安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**  
  ラダー図では、(d)のビットデバイスを反転する命令として表現されています。  
- **ST（構造化テキスト）**  
  `ENO := FF(EN, d);` の形式で記述されます。  
- **FBD/LD**  
  ファンクションブロック図では、EN（実行条件）、ENO（実行結果）、d（反転対象デバイス）を持つブロックとして表現されます。
