### テキスト
#### 演算結果反転（INV命令）
INV命令は、INV命令の直前までの演算結果を反転します。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| INV | 常時実行 |

#### 機能
- INV命令は、命令直前までの演算結果を反転させます。

| INV命令直前までの演算結果 | INV命令実行後の演算結果 |
| :------------------------- | :----------------------- |
| OFF                        | ON                       |
| ON                         | OFF                      |

#### エラー
演算エラーはありません。

#### Point（注意点）
- INV命令は、INV命令の直前までの演算結果で動作するため、AND（演算開始、直列接続、並列接続）と同一位置で使用してください。LD、OR（演算開始、直列接続、並列接続）の位置では使用できません。
- 回路ブロックを使用している場合は、回路ブロックの範囲で演算結果を反転します。INV命令とANB命令を併用して回路を動作させる場合は、反転する範囲に注意してください。

### 図
- ラダー図：INV命令のラダー回路表現。接点の直後に斜線が入った反転記号がある。
- ST（構造化テキスト）表現：`ENO := INV(EN);`
- FBD/LD図：EN入力とENO出力の反転ブロック図。
- 回路ブロックの反転範囲を示すラダー図：M0、M1、M2の接点がANB命令でまとめられ、M10、M20の接点と並列接続されている。破線で囲まれた範囲が反転範囲として示されている。

- 演算結果反転前後の状態表（OFF→ON、ON→OFF）
