### テキスト
#### 機能
(s)で指定されたデバイスのBCD8桁データ（0～99999999）をBIN32ビットデータに変換し、(d)で指定されたデバイスに格納します。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3401H              | (s)の各桁に0～9以外の値があるとき。*1 |

*1 SM754（BIN、DBIN命令エラー不可フラグ）をONにすると本エラーを検出しないようにできます。範囲外の数値を設定した場合、SM754のON/OFFに関わらず命令は実行されません。また、DBIN(P)命令の場合、エラーの有無に関わらず指令（実行条件）をOFF→ONさせるまで次の演算は実行されません。

### 図
#### BCD8桁データからBIN32ビットデータへの変換イメージ
- 上段にBCD8桁の各桁のビット配置（8,4,2,1の重み付け）が示されている。
- 各桁は10^7から10^0までの桁位置が示され、(s)のビットが並ぶ。
- 矢印でBIN変換を示し、下段にBIN32ビットデータのビット配置が示されている。
- BINデータは2^31から2^0までのビットが並び、(d)のビットとして格納される。
- 最上位ビットの一部は必ず0になることが注記されている。

### 表
#### エラーコード詳細
| エラーコード | 内容説明 |
|--------------|----------|
| 3401H        | (s)の各桁に0～9以外の値がある場合に発生するエラー。|

以上の内容により、BCD形式の8桁データを32ビットの2進数データに変換し格納する機能の動作と、入力値の範囲外によるエラー発生条件が説明されています。
