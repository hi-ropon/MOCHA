### 表
#### 命令処理時間一覧
この表は各命令名に対して、条件別の処理時間（単位はマイクロ秒）をCPUユニット別に示しています。

- **命令名**: 各種命令の名称が記載されています。
- **条件**: 命令の実行条件やパラメータ（例: (n)=1、(n)=96、実行時など）が示されています。
- **処理時間(μs)**: CPUユニットごとに最小値と最大値が記載されています。
  - R00CPU, R01CPU
  - R02CPU
  - 左記以外のCPUユニット

##### 処理時間の例
- DBKCMP>_U 命令は、(n)=1 の場合、R00CPUで最小5.2μs、最大11.7μs、R02CPUで最小4.2μs、最大10.7μs、その他CPUユニットで最小3.7μs、最大10.2μs。
- 演算子「+ (s) (d)」は実行時にR00CPUで0.18816μs、R02CPUで0.02352μs、その他CPUユニットで0.00588μs。
- 複雑な命令（例: B+ (s1) (s2) (d)）はR00CPUで最小3.3μs、最大6.6μs、R02CPUで最小2.3μs、最大5.6μs、その他CPUユニットで最小1.8μs、最大5.1μs。

この表は命令の種類や条件により処理時間が異なることを示しており、CPUユニットごとの性能差も把握できます。
