### テキスト
#### 機能
(s1)で指定したBIN32ビットデータと、(s2)で指定したBIN32ビットデータの加算を行い、その結果を(d)で指定したデバイスに格納します。

- 加算結果にオーバーフローが発生した場合、桁上げは無視されます。この場合、SM700（キャリフラグ）はONしません。

#### エラー
演算エラーはありません。

### 図
- BIN32ビットデータの加算例：
  - 例1：567890 (BIN) + 123456 (BIN) = 691346 (BIN)
  - 例2（オーバーフロー発生時）：1234567890 (BIN) + 987654321 (BIN) = -2072745085 (BIN)
  - 例3（オーバーフロー発生時）：-1234567890 (BIN) + -987654321 (BIN) = 2072745085 (BIN)
  - 例4（D+(P)_Uの場合）：3456789012 (BIN) + 1234567890 (BIN) = 396389606 (BIN)

各図は、(s1)、(s2)、(d)の各デバイスのビット構成（b31～b0）を示し、加算前後の値を具体的なBIN数値で表しています。
