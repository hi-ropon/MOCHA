### テキスト
#### 演算結果
1. EN/ENOなしファンクションブロック  
演算結果は以下の通りです。  
- 演算エラーなしの場合は演算出力値が(d1)、(d2)に出力されます。  
- 演算エラーありの場合は不定値となります。

2. EN/ENO付きファンクションブロック  
実行条件と演算結果は以下の通りです。  
- ENがTRUE（演算実行）かつ ENOがTRUE（演算エラーなし）の場合、演算出力値が(d1)、(d2)に出力されます。  
- ENがTRUEかつ ENOがFALSE（演算エラーあり）の場合、出力は前回の値を保持します。  
- ENがFALSE（演算停止）の場合も前回の値を保持します。

### 表
#### 演算結果（EN/ENOなし）
| 演算結果       | (d1)、(d2)       |
| -------------- | ---------------- |
| 演算エラーなし | 演算出力値       |
| 演算エラーあり | 不定値           |

#### 実行条件と演算結果（EN/ENO付き）
| EN   | ENO               | (d1)、(d2)       |
| -----|-------------------|------------------|
| TRUE | TRUE（演算エラーなし） | 演算出力値       |
| TRUE | FALSE（演算エラーあり）| 前回出力値       |
| FALSE| FALSE             | 前回出力値       |

### グラフ（タイミングチャート）
#### 1. EN/ENOなしファンクションブロックのタイミングチャート（n=T#5sの場合）
- (s)信号がOFFからONになると(d2)の時間計測を開始。  
- (d2)が指定時間に達すると(d1)がONになる。  
- (s)がONからOFFに変わり、(d1)がONからOFFになると(d2)を初期化する。  
- (d2)は0から5までカウントアップし、時間経過を示す。

#### 2. EN/ENO付きファンクションブロックのタイミングチャート（n=T#5sの場合）
- ENがONかつ(s)がOFFからONになると(d2)の時間計測を開始。  
- (d2)が指定時間に達すると(d1)がONになる。  
- ENがONかつ(s)がONからOFFに変わり、(d1)がONからOFFになると(d2)を初期化する。  
- EN、(s)、ENO、(d1)、(d2)の信号状態が時間軸に沿って変化し、動作の流れを示す。

(1)～(3)の番号はタイミングチャート内のポイントを示し、それぞれの動作開始や終了の条件を説明しています。
