本ページは主に命令コードとその処理内容を表形式でまとめた技術資料の一部である。ページ構成は以下の通り。

<contents>
### テキスト
#### 拡張時計データの減算
時間データを減算する命令について説明している。

#### タイミング計測命令
タイミングパルス発生や指定データの時間計測に関する命令を解説。

#### ユニットアクセス命令
I/Oリフレッシュ、選択リフレッシュ、ユニットリフレッシュの各命令について説明。

### 表
#### 拡張時計データの減算命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| S.DATE- / SP.DATE- | 時間データを減算する。 | 1101ページ S(P).DATE- |

#### タイミングパルス発生命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| DUTY | ユーザ用タイミングクロックを指定したスキャン分ONにし、指定したスキャン分OFFにする。 | 1103ページ DUTY |

#### 指定データの時間計測命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| TIMCHK | 入力条件のON時間を計測し、設定した時間以上連続でONなら(d)で指定したデバイスをONにする。 | 1105ページ TIMCHK |

#### アワーメータ命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| HOURM | 起動接点ON時間を1時間単位で計測し、累計時間が指定時間に達したら(d2)のデバイスをONにする。 | 1107ページ HOURM |
| DHOURM | 起動接点ON時間を1時間単位で計測し、累計時間が指定時間に達したら(d2)のデバイスをONにする（32ビットデータ対応）。 | 1109ページ DHOURM |

#### ユニットアクセス命令
| 命令記号 | 処理内容 | 参照先 |
| :--- | :--- | :--- |
| RFS / RFSP | 1スキャン途中で該当入出力部分のリフレッシュを行う。 | 1111ページ RFS(P) |
| COM / COMP | 入力条件成立で各種ユニットのリフレッシュ処理やサービス処理を行う。 | 1113ページ COM(P) |
| S.ZCOM / SP.ZCOM | 指定された各種ユニットのリフレッシュ処理を行う。 | 1115ページ S(P).ZCOM |

### 図
#### 拡張時計データの減算のイメージ
時間データの減算は、hour、minute、second、1/1000 secondの各単位で行われる。  
(s1)から(s2)を引き、結果(d)を得る構造を示す。

#### タイミングパルス発生のタイミング図
スキャンのON時間(n1)とOFF時間(n2)を示すタイミングチャート。  
(d)はSM420～SM424の範囲を示す。
