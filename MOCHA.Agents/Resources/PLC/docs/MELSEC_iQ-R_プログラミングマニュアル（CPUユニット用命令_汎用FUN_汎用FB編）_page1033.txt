### テキスト
#### データベース切断(DBCLOSE(P))
指定したデータベースとの接続を解除する命令です。  
使用できないCPUはR00CPU、R01CPU、R02CPUです。

#### 実行条件
- DBCLOSE命令は立ち下がりエッジで実行されます。
- DBCLOSEP命令は立ち上がりエッジで実行されます。

### 図
#### ラダー回路図
- (s)、(d1)、(d2)の3つのオペランドを持つ命令ブロックが描かれています。
- EN入力に対して、ENO出力が返される構造です。

#### FBD/LD回路図
- EN入力、ENO出力、s、d1、d2の5つの端子を持つ命令ブロックが示されています。
