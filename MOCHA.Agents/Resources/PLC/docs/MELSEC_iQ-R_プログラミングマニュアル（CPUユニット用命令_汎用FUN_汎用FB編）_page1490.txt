### テキスト
#### 40.3 乗算
関数名：MUL_TIME(_E)  
対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化、一般）、RnPSFCPU（安全、一般）  
機能：入力値（TIME型）の積((s1)×(s2))を出力します。

#### 使用例
- ラダー、FBD/LDでの使用例が2種類示されています。
  - EN/ENOなしの場合
  - EN/ENO付きの場合
- ST言語での使用例も記載されています。
  - EN/ENOなし：d:=MUL_TIME(s1,s2);
  - EN/ENO付き：d:=MUL_TIME_E(EN,ENO,s1,s2);

### 図
- ラダー、FBD/LDの回路図例
  - EN/ENOなし：s1、s2を入力し、dを出力するシンプルな乗算ブロック
  - EN/ENO付き：EN入力、ENO出力が追加されている乗算ブロック

### 表
#### 設定データ（内容、種別、データ型）

| 引数   | 内容                          | 種別     | データ型  |
|--------|-------------------------------|----------|-----------|
| EN     | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL      |
| s1(IN1)| 入力                          | 入力変数 | TIME      |
| s2(IN2)| 入力                          | 入力変数 | ANY_NUM*1 |
| ENO    | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL      |
| d      | 出力                          | 出力変数 | TIME      |

*1 安全プログラムではINT型/REAL型/LREAL型データを指定しないこと。指定した場合はエラーとなる。
