### テキスト
#### 32.65 ビット配列のコピー (CPY_BITARR(_E))
ビット配列を指定したビットサイズ分コピーする関数です。

##### 機能
- (s)に指定されたビット配列の(n)ビットを(d)に出力します。

##### 演算処理
- (s)に指定されたビット配列の(n)ビットを(d)に出力します。

##### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
|-------------|--------------|-----|
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となります。その場合は(d)から出力されたデータを使用しないようプログラムの処理を行ってください。

##### エラー
- 演算エラーはありません。

### 図
- ラダー、FBD/LD言語での関数ブロック図  
  - EN/ENOなし：s, n, dの3つの端子を持つシンプルなブロック  
  - EN/ENO付き：EN, ENO, s, n, dの5つの端子を持つブロック

- ST言語での使用例  
  - EN/ENOなし: d := CPY_BITARR(s, n);  
  - EN/ENO付き: d := CPY_BITARR_E(EN, ENO, s, n);

### 表
#### 設定データ：内容、種別、データ型

| 引数 | 内容 | 種別 | データ型 |
|------|------|------|----------|
| EN | 実行条件(TRUE: 実行、FALSE: 停止) | 入力変数 | BOOL |
| s(BitArrIn) | 入力 | 入力変数 | BOOL配列要素 |
| n | 4, 8, 12, 16, 20, 24, 28, 32の定数のみ指定可能 | 入力変数 | INT |
| ENO | 出力状態(TRUE: 正常実行、FALSE: 異常または停止) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL配列要素 |

#### 演算結果（EN/ENO付きファンクション）

| 実行条件 EN | 演算結果 ENO | (d) |
|-------------|--------------|-----|
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOがFALSEの場合、(d)のデータは不定値となるため使用しないこと。
