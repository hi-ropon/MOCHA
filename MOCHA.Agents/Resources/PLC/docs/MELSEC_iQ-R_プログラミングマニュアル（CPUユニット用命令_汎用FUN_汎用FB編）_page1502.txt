### テキスト
#### 42.2 立ち下りエッジ検出
「F_TRIG(_E)」は信号の立ち下がりを検出してパルス信号を出力する機能です。対応CPUはRnCPU、RnENCPU（プロセス、三重化、一般、安全）となっています。

### 図
- ラダー図およびFBD/LDの回路図例が示されています。
  - [EN/ENOなし]の場合は、s（入力）からd（出力）へ信号が流れる簡単な図。
  - [EN/ENO付き]の場合は、EN（実行条件）、ENO（出力状態）、s（入力）、d（出力）が示された図。
- ST言語では、EN/ENOなしの場合はインスタンス名（CLK:=s, Q:=d）を指定し、EN/ENO付きの場合はEN:=en、ENO:=eno、CLK:=s、Q:=dのように指定します。

### 表
#### 設定データ：内容、種別、データ型
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(CLK) | 立ち下がりエッジ検出入力 | 入力変数 | BOOL |
| ENO | 出力状態(TRUE:正常、FALSE:異常または停止) | 出力変数 | BOOL |
| d(Q) | 出力 | 出力変数 | BOOL |

この表は、各引数の役割とデータ型を明確に示しています。
