### テキスト
#### DAND(P) [オペランドが3個の場合]
指定した2つのBIN32ビットデータの論理積を行う命令です。

#### ■実行条件
- DAND命令は立ち上がりエッジで実行されます。
- DANDP命令は立ち下がりエッジで実行されます。

#### ■注意事項
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**  
  3つのオペランド（s1, s2, d）を持つDAND命令のラダー表現。s1とs2が入力、dが出力。

- **ST（構造化テキスト）**  
  ENO:=DAND(EN,s1,s2,d);  
  ENO:=DANDP(EN,s1,s2,d);

- **FBD/LD（ファンクションブロック図/ラダーダイアグラム）**  
  EN, ENO, s1, s2, dのポートを持つDAND命令のブロック図。
