このページはテキストと簡単な図で構成されています。テキスト部分では、指定されたデバイスから複数点のBIN32ビットデータの平均値を計算し、別のデバイスに格納する機能について説明しています。図はその処理のイメージを示しています。エラーに関する記述もあり、演算エラーがないことが明記されています。表やグラフは含まれていません。

<contents>
### テキスト
#### 機能
・(s)で指定されたデバイスから(n)点分のBIN32ビットデータの平均値を算出し、(d)で指定されたデバイスに格納します。  
・平均値の算出は、(s)+1、(s)+2、…、(s)+2(n)-1、(s)+2(n)-2のデータを対象とします。  
・算出結果が整数値にならない場合、小数点以下は切り捨てられます。  
・(n)で指定した値が0の場合は処理されません。

#### エラー
演算エラーはありません。

### 図
・左側に縦に並んだ複数のデバイスデータを示す図があり、(s)から始まる連続したデバイスの位置を示しています。  
・右側に1つのデバイスデータがあり、(d)で指定されたデバイスに平均値が格納されることを示しています。  
・矢印で左の複数デバイスデータから右の1つのデバイスデータへ平均値が計算されて渡される流れを表現しています。
</contents>
