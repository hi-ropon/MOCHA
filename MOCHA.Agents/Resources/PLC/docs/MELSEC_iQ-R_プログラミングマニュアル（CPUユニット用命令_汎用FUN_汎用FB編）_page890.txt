### テキスト
#### 符号なしBIN16ビットデータ→倍精度実数変換
命令名：UINT2DBL(P)

符号なしBIN16ビットデータを倍精度実数に変換する命令です。

##### 実行条件
- UINT2DBL：立ち上がりエッジで実行
- UINT2DBLP：立ち下がりエッジで実行

##### 機能
(s)で指定した符号なしBIN16ビットデータを倍精度実数に変換し、(d)へ格納します。

##### エラー
演算エラーはありません。

### 図
- ラダー図：命令のラダー回路表現。入力(s)、出力(d)を持つ。
- FBD/LD図：命令のファンクションブロック図。入力EN、出力ENO、入力s、出力dを示す。
- ビットデータ変換イメージ図：16ビットの符号なしBINデータ（例：10E1H）を倍精度実数（例：4321.0）に変換し、D103,D102,D101,D100に格納する様子を示す。
