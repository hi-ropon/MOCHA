### テキスト
#### 機能
- (s)で指定された倍精度実数を符号付きBIN16ビットデータに変換し、(d)で指定されたデバイスに格納する。
- 変換後のデータは、(s)で指定された倍精度実数の小数点以下1桁目を四捨五入した値となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### 注意事項
- エンジニアリングツールから入力値を設定する際の注意事項は47ページを参照。

#### プログラム例の説明
- M0がONしたとき、D0～D3に格納された倍精度実数を符号付きBIN16ビットデータに変換し、D100へ格納する。

### 図
- プログラム例の図解
  - SM402とM0の接点があり、M0がONするとEDMOVP命令でE-5432.1の値をD0に格納。
  - DBL2INT命令でD0の倍精度実数をD100に符号付きBIN16ビットデータに変換。
  - (s)側はD3, D2, D1, D0のデータを16進数で示し、(d)側はD100の16進数データを示している。

### 表
#### エラーコード（SD0）
| エラーコード | 内容 |
| :--- | :--- |
| 3401H | (s)で設定する倍精度実数が-32768～32767の範囲外のとき。 |
