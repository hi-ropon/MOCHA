### テキスト
#### 単精度実数→符号付きBIN32ビットデータ
命令名：FLT2DINT(P)  
指定した単精度実数を符号付きBIN32ビットデータに変換します。  
対応ツールバージョンは1.035M以降のエンジニアリングツールです。

#### 実行条件
- FLT2DINT命令は立ち上がりエッジで実行されます。  
- FLT2DINTP命令は立ち下がりエッジで実行されます。

#### 設定データ
##### 内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (s) | 単精度実数または単精度実数が格納されている先頭デバイス | -2147483648～2147483647 | 単精度実数 | ANYREAL_32 |
| (d) | BINデータを格納する先頭デバイス | ― | 符号付きBIN32ビット | ANY32_S |
| EN | 実行条件 | ― | ビット | BOOL |
| ENO | 実行結果 | ― | ビット | BOOL |
