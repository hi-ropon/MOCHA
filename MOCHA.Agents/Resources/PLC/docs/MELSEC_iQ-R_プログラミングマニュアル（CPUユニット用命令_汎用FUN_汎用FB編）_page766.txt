### テキスト
#### 運転継続異常命令 PALERT(P)
CPUユニットを続行エラー状態にします。PALERT命令を実行すると、SM0（最新自己診断エラー、アナンシエータON含む）がONし、CPUユニットのERROR LEDが点灯します。ただし、SM1（最新自己診断エラー、アナンシエータON含まない）はONしません。

#### 機能
- CPUユニットを続行エラー（エラーコード：1810H）に設定します。
- (s)で指定したBIN16ビットデータを、エラーコード1810Hの詳細情報2に格納します。

### 図
- **ラダー図**  
  PALERT命令のラダー表現。入力(s)があり、ST命令ではENO:=PALERT(EN,s); ENO:=PALERTP(EN,s);と記述されています。

- **FBD/LD図**  
  入力EN、出力ENO、パラメータsを持つ命令ブロックの図。

- **実行条件図**  
  PALERT命令は立ち上がりエッジで実行されることを示す波形図。
