### テキスト
#### 単精度実数常用対数演算（LOG10(P)）
単精度実数で指定した値の常用対数（底が10の対数）を演算します。

#### 実行条件
- LOG10命令は立ち上がりエッジで実行
- LOG10P命令は立ち下がりエッジで実行

#### 機能
- オペランド(s)で指定した値の常用対数を計算し、(d)で指定したデバイス番号に演算結果を格納します。
- (s)に指定できる値は正の数のみ（負の数は演算不可）
- 演算結果が0またはアンダーフロー発生時は結果は0になる
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生する可能性あり（詳細は47ページ参照）

### 図
- ラダー図：LOG10命令のラダー表現。入力(s)、出力(d)を示す。
- FBD/LD図：命令ブロックの入力EN、出力ENO、入力s、出力dの接続例を示す。
