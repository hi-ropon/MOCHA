### テキスト
#### 機能
・(d)で指定したワードデバイスの(n)ビット目をセット(1)します。
・(n)が15を超えた場合は、下位4ビットのデータで実行されます。

#### エラー
演算エラーはありません。

### 図
・ラダー図形式で「BSETP D10 K6」という命令が示されています。
・D10のビット配置が上下2段で示され、b15からb0までのビットが並んでいます。
・上段は命令実行前のビット状態、下段は6ビット目(b6)がセットされた後の状態を表しています。
・矢印と(1)の注釈で、D10の6ビット目がセットされることを示しています。
