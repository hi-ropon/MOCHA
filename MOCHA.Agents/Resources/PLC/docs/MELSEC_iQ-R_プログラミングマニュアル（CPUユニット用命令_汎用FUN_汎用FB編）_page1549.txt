### 表
#### 命令処理時間一覧表
命令名ごとに、以下の条件における処理時間（マイクロ秒）がCPUユニット別に最小値・最大値で示されています。

- 条件
  - 無実行時
  - 指定の時刻と比較：導通時
  - 指定の時刻と比較：非導通時
  - 現在の時刻と比較：導通時
  - 現在の時刻と比較：非導通時

- CPUユニット
  - R00CPU、R01CPU
  - R02CPU
  - 左記以外のCPUユニット

各命令（例：ANDTM=、ANDTM<>、ANDTM>、ANDTM<=、ANDTM<、ANDTM>=、ORTM=、ORTM<>、ORTM>、ORTM<=）について、上記条件とCPUユニットごとに処理時間の最小値と最大値が詳細に記載されています。

例えば、命令ANDTM=の無実行時の処理時間はR00CPU、R01CPUで0.18816μs、R02CPUで0.02352μs、その他CPUユニットで0.00588μsとなっています。指定の時刻と比較した場合や現在の時刻と比較した場合は、導通時・非導通時で処理時間が異なり、最大で16μs台までの幅があります。

この表は命令の実行条件別に処理時間の目安を示しており、CPUユニットごとの性能差や条件による処理時間の変動を把握するための資料です。
</contents>
