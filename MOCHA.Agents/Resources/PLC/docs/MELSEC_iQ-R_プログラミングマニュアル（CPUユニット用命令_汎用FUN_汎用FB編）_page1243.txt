### テキスト
#### 指定ループNo.の演算開始
この命令は指定したループNo.のPID演算を開始します。プロセスCPU（二重化モード）およびSIL2プロセスCPUの両系実行プログラムで使用した場合、バックアップモード時の待機系で動作に制約があります。

#### 機能
- (s)で指定したループNo.のPID演算を開始します。
- S(P).PIDSTOP命令でPID演算を停止したループNo.を再度実行させるための命令です。
- すでにPID演算実行中のループNo.に対してS(P).PIDRUN命令を実行した場合は無処理となります。

### 図
#### ラダー図
- 停止するループNo.を指定する入力(s)があり、ENOにS_PIDRUN(EN,s)、SP_PIDRUN(EN,s)が出力される構造。

#### FBD/LD図
- EN入力に対してENO出力があり、s入力があるブロックの図。
