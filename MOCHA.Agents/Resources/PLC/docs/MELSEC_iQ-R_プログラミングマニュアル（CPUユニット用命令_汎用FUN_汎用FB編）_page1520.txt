### テキスト
#### 演算結果
1. EN/ENOなしファンクションブロック
- 演算結果は表に示されており、演算エラーなしの場合は演算出力値(d1),(d2)が出力され、演算エラーありの場合は不定値となる。

2. EN/ENO付きファンクションブロック
- 実行条件（EN）と演算結果（ENO、(d)）の関係が表で示されている。
  - ENがTRUE（演算実行）の場合、ENOがTRUEなら演算エラーなしで演算出力値を出力、ENOがFALSEなら演算エラーありで前回出力値を保持。
  - ENがFALSE（演算停止）の場合、ENOはFALSEで前回出力値を保持。

#### エラー
- エラーコード(SD0)と内容の表があり、3401Hは「出力時間設定値が有効な範囲を超えているとき」のエラーを示す。

### 図（タイミングチャート）
1. EN/ENOなしファンクションブロックのタイミングチャート（n=T#5sの場合）
- (s)のON→OFFで(d2)の時間計測開始。
- (d2)が指定時間に達すると(d1)をON。
- (s)のOFF→ONで(d2)を初期化。

2. EN/ENO付きファンクションブロックのタイミングチャート（n=T#5sの場合）
- EN=ONかつ(s)のON→OFFで(d2)の時間計測開始。
- EN=ONかつ(s)のOFF→ONで(d2)を初期化。
