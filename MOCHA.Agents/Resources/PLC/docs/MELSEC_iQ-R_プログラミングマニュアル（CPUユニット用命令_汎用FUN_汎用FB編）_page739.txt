### テキスト
#### 動作例の説明
データ行数(n1)=5、データ列数(n2)=4の並べ替え元データを各条件で並び替えたときの動作例を示しています。  
1列目に管理番号などの通し番号を入れておくと、その内容で元の行番号が判断できることがポイントとして説明されています。

#### ポイント
1列目に管理番号などの通し番号を入れておくことで、並び替え後に元の行番号を判別可能。
