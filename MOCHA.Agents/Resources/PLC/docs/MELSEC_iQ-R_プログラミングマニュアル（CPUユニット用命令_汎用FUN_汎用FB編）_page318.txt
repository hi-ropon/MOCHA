### テキスト
#### WXOR(P)命令の概要
指定した2つのBIN16ビットデータの排他的論理和（XOR）を行う命令です。オペランドが3個の場合の使い方を示しています。

#### 実行条件
- WXOR命令は立ち下がりエッジで実行されます。
- WXORP命令は立ち上がりエッジで実行されます。

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**：3つのオペランド(s1, s2, d)を使ったWXOR命令のラダー回路例が示されています。
- **ST言語**：`ENO:=WXOR(EN,s1,s2,d);`および`ENO:=WXORP(EN,s1,s2,d);`の記述例が示されています。
- **FBD/LD**：EN, s1, s2を入力、ENO, dを出力とするブロック図が描かれています。
