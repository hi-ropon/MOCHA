### テキスト
#### DHOURM命令の概要
- R00CPU、R01CPU、R02CPUには制約なし。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン17以降で使用可能。
- エンジニアリングツールはバージョン1.020W以降で対応。
- 起動接点がONしている時間を1時間単位で計測し、累計ON時間が指定時間（BIN32ビットデータ）に達した時点で、指定デバイス(d2)をONにする。

### 図
- ラダー図：起動接点(s)、時間(d1)、出力デバイス(d2)の関係を示す。
- ST言語：ENO:=DHOURM(EN,s,d1,d2);
- FBD/LD：EN入力、s入力、d1、d2出力のブロック図。
