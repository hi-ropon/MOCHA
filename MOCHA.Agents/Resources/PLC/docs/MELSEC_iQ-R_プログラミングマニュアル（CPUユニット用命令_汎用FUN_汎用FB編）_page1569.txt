### 表
#### 基本ステップ数とサブセット処理の可否一覧
この表は各命令の名称に対して、基本ステップ数とサブセット処理が可能かどうかを示しています。

- **命令名**: 各命令の名称が記載されています。例として「EFCALLP (ファイル名) (P)」や「XCALL (P)」などがあります。
- **基本ステップ数**: 命令の基本的なステップ数が記載されています。ファイル名の文字数に依存するものや固定の数値（例: 3、4、5など）があります。
- **サブセット可否**: サブセット処理が可能かどうかを「〇」または「―」で示しています。〇は可能、―は不可を意味します。

表の内容例：
- 「EFCALLP (ファイル名) (P)」は「5＋ファイル名の文字数」の基本ステップ数で、サブセット処理は不可。
- 「LD$=」は基本ステップ数3で、サブセット処理は可能（〇）。
- 「SP.FREAD」は基本ステップ数8で、サブセット処理は不可。
