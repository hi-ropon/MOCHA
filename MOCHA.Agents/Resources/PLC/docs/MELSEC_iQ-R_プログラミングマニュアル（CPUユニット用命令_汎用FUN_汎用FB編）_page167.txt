### テキスト
#### タイマ命令（OUT T, OUTH T, OUT ST, OUTH ST）
- 対応CPU: プロセスCPU（二重化モード）、SIL2プロセスCPUの両系実行プログラムで使用時に制約あり（詳細はMELSEC iQ-R CPUユニットユーザーマニュアル参照）。
- SIL2プロセスCPUおよび安全CPUの安全プログラムで安全タイマを使用する場合は、以下のように読み替える必要がある。
  - 「タイマ」→「安全タイマ」
  - 「積算タイマ」→「安全積算タイマ」
  - 「T」→「SA¥T」、「ST」→「SA¥ST」、「M0」→「SA¥M0」
  - 「スキャン」→「安全サイクル処理」
  - 「スキャンタイム」→「安全サイクル時間」
- 命令の種類と概要：
  - OUT T：低速タイマ命令
  - OUTH T：高速タイマ命令
  - OUT ST：低速積算タイマ命令
  - OUTH ST：高速積算タイマ命令
- OUT命令は演算結果がONのとき、コイルがONしてタイマの計測を行う。タイムアップするとa接点は導通し、b接点は非導通になる。

### 図
#### ラダー図
- タイマ命令のラダー図例が示されている。
- 設定値(Value)を指定し、コイルとタイマ命令が接続されている。
- ST（ステートメント）欄には以下の記述：
  - ENO:=OUT_T(EN,Coil,Value);
  - ENO:=OUTH(EN,Coil,Value);

#### FBD/LD図
- タイマ命令のFBD/LD図例が示されている。
- ブロックにEN、Coil、Valueの入力があり、ENOが出力される。
- ValueにはOUT_T、OUTHが入る。

### 表
#### 実行条件
| 命令       | 実行条件 |
|------------|----------|
| OUT T      | 常時実行 |
| OUTH T     | 常時実行 |
| OUT ST     | 常時実行 |
| OUTH ST    | 常時実行 |

- すべての命令は常時実行されることが示されている。
