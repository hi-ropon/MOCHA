### テキスト
#### 機能
- (s)で指定された割込み優先度以下の割込みポインタ番号の割込みプログラムを禁止する機能について説明しています。

#### 割込み優先度設定
- 割込み優先度はI No.（割込み番号）ごとに設定され、優先度に応じて割込みプログラムの実行可否が決まります。

### 表
#### 割込み優先度設定一覧
| I No. | 優先度 |
| :---- | :---- |
| I50   | 5     |
| I51   | 6     |

### 図
- ラダー図でI50とI51の割込みプログラムの実行制御を示しています。
- DI K6の入力条件でシーケンスプログラムAが動作し、その後FENDで終了。
- I50割込みプログラムBは優先度5の割込みのため実行可能。
- I51割込みプログラムBは優先度6の割込みのため実行禁止。

#### 図中の説明
- A：シーケンスプログラム
- B：割込みプログラム
- (1)：優先度6～8の割込み禁止区間（優先度1～5は許可区間）
- (2)：優先度5の割込みのため実行可能
- (3)：優先度6の割込みのため実行禁止
