### 表
#### 命令処理時間一覧表
この表は、各命令名に対して、条件別の処理時間をCPUユニットごとに最小値と最大値で示しています。CPUユニットはR00CPU、R01CPU、R02CPU、及びそれ以外のCPUユニットに分類されています。

- **命令名**: 各行に命令名が記載されています（例：S.PIDSTOP、PIDINIT、D.DDRDなど）。
- **条件**: 命令実行時の条件が記載されています（例：1ループ、32ループ、読出しデータ点数など）。
- **処理時間（μs）**: CPUユニットごとに最小値と最大値が示されています。
  - R00CPU、R01CPU
  - R02CPU
  - 左記以外のCPUユニット

##### 主な命令と条件例
- S.PIDSTOP、S.PIDRUN、S.PIDPRMW：1ループの処理時間
- PIDINIT：1ループおよび32ループの処理時間
- PIDCONT：1ループ（初回、2回目以降）、32ループ（初回、2回目以降）
- D.DDRD、D.DDWR：読出しデータ点数1、16、96、8192の場合
- M.DDRD、M.DDWR：書込みデータ点数1、16、96、8192の場合
- 論理演算命令（AND、OR、LD、LDI、ANI、ORIなど）：導通時/非導通時の処理時間
- MOV(P)命令：条件なしでの処理時間

##### 処理時間の特徴
- ループ回数やデータ点数が増えると処理時間も増加する傾向にあります。
- CPUユニットによって処理時間に差があり、R02CPUは一般的にR00CPU、R01CPUよりも処理時間が短い場合が多いです。
- 論理演算命令は導通時と非導通時で処理時間が異なります。
