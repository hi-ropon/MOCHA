### テキスト
#### 符号なしBIN32ビットデータ→符号付きBIN16ビットデータ変換
指定した符号なしBIN32ビットデータを符号付きBIN16ビットデータに変換する命令「UDINT2INT(P)」について説明しています。

#### 実行条件
- 命令「UDINT2INT」は立下りエッジで実行されます。
- 命令「UDINT2INTP」は立上りエッジで実行されます。

#### 設定データ
##### 内容、範囲、データ型
| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |
| (s) | BINデータまたはBINデータが格納されている先頭デバイス | 0～4294967295 | 符号なしBIN32ビット | ANY32_U |
| (d) | BINデータを格納するデバイス | ― | 符号付きBIN16ビット | ANY16_S |
| EN | 実行条件 | ― | ビット | BOOL |
| ENO | 実行結果 | ― | ビット | BOOL |

### 図
- **ラダー図**：命令のラダー表現。入力(s)、出力(d)が示されている。
- **FBD/LD図**：命令のファンクションブロック図。EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）の接続が示されている。
