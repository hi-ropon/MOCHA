### テキスト
#### 注意事項
- MTR命令は実入出力を直接操作するため注意が必要。MTR命令の指令がOFFでも、MTR命令でONにした出力はOFFにならない。プログラムで(d1)に指定した出力をOFFにする必要がある。
- MTR命令の実行間隔は、入力ユニットと出力ユニットの応答時間を加えた値以上に設定すること。実行間隔が短すぎると正常に入力取り込みができない。
- プログラムのスキャンタイムが短い場合は、コンスタントスキャンでスキャンタイムと応答時間の合計を長くする必要がある。
