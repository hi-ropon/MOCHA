### テキスト
#### 機能
- (s1)で指定したループNo.の演算パラメータを、(s2)で指定したデバイス番号以降に格納されているPID制御用データに変更する。
- (s2)で指定したデバイス番号以降のPID制御用データの構成について説明。

### 表
#### PID制御用データ構成
| アドレス | 内容 |
| :--- | :--- |
| (s2)+0 | 演算式選択 |
| (s2)+1 | サンプリング周期(Ts) |
| (s2)+2 | 比例定数(Kp) |
| (s2)+3 | 積分定数(Ti) |
| (s2)+4 | 微分定数(Td) |
| (s2)+5 | フィルタ係数(α) |
| (s2)+6 | 操作量下限値(MVLL) |
| (s2)+7 | 操作量上限値(MVHL) |
| (s2)+8 | 操作量変化率リミット値(ΔMVL) |
| (s2)+9 | 測定値変化率リミット値(ΔPVL) |

#### エラーコード一覧
| エラーコード (SD0) | 内容 |
| :--- | :--- |
| 3405H | (s1)に指定可能な範囲外のデータを設定した場合。<br>・指定したループNo.が存在しない<br>・指定した値が1～32以外<br>・(s2)に指定したPID制御用データが設定可能範囲外のとき |
| 3422H | PIDPRMW(P)命令実行前に、PIDINIT(P)命令を実行していない場合 |
