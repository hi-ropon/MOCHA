### テキスト
#### パルス演算開始、パルス直列接続、パルス並列接続
- **LDP**: 立上りパルス演算開始命令。指定ビットデバイスの立上り時（OFF→ON）にのみ導通する。
- **LDF**: 立下りパルス演算開始命令。指定ビットデバイスの立下り時（ON→OFF）にのみ導通する。
- **ANDP / ANDF**: 立上り/立下りパルス直列接続命令。これまでの演算結果とAND演算を行う。
- **ORP / ORF**: 立上り/立下りパルス並列接続命令。これまでの演算結果とOR演算を行う。

### 図
#### ラダー図
- LDP、LDFはそれぞれ立上り、立下りのパルス検出を示す接点記号が描かれている。
- ANDP、ANDFは直列接続のパルス演算を示す接点が並列に配置されている。
- ORP、ORFは並列接続のパルス演算を示す接点が並列に配置されている。
- 各命令の動作タイミングを示す(s)マークが付いている。

#### FBD/LD図
- EN（入力）、ENO（出力）、s（パルス信号）を持つブロック図が示されている。

### 表

| LDF | 常時実行 |
