### テキスト
#### 演算結果パルス化
- MEP命令（立上り）：演算結果がOFFからONに変わる時に導通する。
- MEF命令（立下り）：演算結果がONからOFFに変わる時に導通する。

#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| MEP | 常時実行 |
| MEF | 常時実行 |

#### 機能
- **MEP**
  - 演算結果がOFF→ONに変わる時にON（導通状態）になる。
  - それ以外の時はOFF（非導通状態）。
  - 複数の接点を直列に接続した場合のパルス化処理が容易になる。
- **MEF**
  - 演算結果がON→OFFに変わる時にON（導通状態）になる。
  - それ以外の時はOFF（非導通状態）。
  - 複数の接点を直列に接続した場合のパルス化処理が容易になる。

#### エラー
- 演算エラーはなし。

#### Point（注意点）
- MEP命令、MEF命令はサブルーチンプログラムやFOR～NEXT命令などのインデックス修飾した接点のパルス化では正常に動作しない場合がある。詳細は163ページのEGP、EGFを参照。
- MEP命令、MEF命令は直前にあるLD命令からMEP/MEF命令の直前までの演算結果で動作するため、AND命令（150ページ 演算開始、直列接続、並列接続）と同一位置で使用すること。
- MEP命令、MEF命令はLD、OR命令（150ページ 演算開始、直列接続、並列接続）の位置では使用不可。

### 図
- **ラダー**
  - MEP接点は立上りパルスを表す矢印付き接点。
  - MEF接点は立下りパルスを表す矢印付き接点。
- **ST**
  - `ENO := MEP(EN);`
  - `ENO := MEF(EN);`
- **FBD/LD**
  - EN入力、ENO出力のシンプルなブロック図。
