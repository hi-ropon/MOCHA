### テキスト
#### 立下り出力（PLF）
PLF命令は、ONからOFFに変わる際に指定したデバイスを1スキャンだけONにする命令です。

**Point**
- SIL2プロセスCPUおよび安全CPUの安全プログラムで本命令を使用する場合は、特に明記がない限り以下のように読み替えます。
  - 「X5」→「SA￥X5」
  - 「スキャン」→「安全サイクル処理」

#### 実行条件
PLF命令は、指定された実行条件が成立したときに動作します。

### 図
- **ラダー図**  
  - (d)のパルス化するデバイス番号を入力し、ENO:=PLF(EN,d);の形で記述されます。
- **FBD/LD図**  
  - EN（実行条件）入力、ENO（実行結果）出力、d（デバイス番号）入力のブロックで表現されます。

### 表
#### 設定データ：内容、範囲、データ型

| オペランド | 内容                   | 範囲 | データ型 | データ型（ラベル） |
| :--------- | :--------------------- | :--- | :------- | :----------------- |
| (d)        | パルス化するデバイス番号 | —    | ビット   | ANY_BOOL           |
| EN         | 実行条件               | —    | ビット   | BOOL               |
| ENO        | 実行結果               | —    | ビット   | BOOL               |

- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、上記のデータ型の安全デバイスおよび安全ラベルのみ使用可能。
