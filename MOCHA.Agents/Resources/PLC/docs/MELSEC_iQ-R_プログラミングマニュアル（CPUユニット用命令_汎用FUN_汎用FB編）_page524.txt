### テキスト
#### BIN32ビットデータ否定転送（DCML(P)）
指定されたBIN32ビットデータをビットごとに反転し、転送する命令です。

#### 実行条件
- DCML命令は立ち下がりエッジで実行されます。
- DCMLP命令は立ち上がりエッジで実行されます。

#### 注意事項
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- ラダー図：s（入力デバイス）、d（出力デバイス）を指定し、DCML命令を実行する回路図が示されています。
- ST言語：ENO:=DCML(EN,s,d); ENO:=DCMLP(EN,s,d); の形式で記述されます。
- FBD/LD：EN（実行条件入力）、ENO（実行結果出力）、s（入力）、d（出力）を持つ命令ブロックの図が示されています。
