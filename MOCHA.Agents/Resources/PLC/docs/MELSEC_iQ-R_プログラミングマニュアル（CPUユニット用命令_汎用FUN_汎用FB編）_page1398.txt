### テキスト
#### 32.59 STRING型→REAL型変換
関数名：STRING_TO_REAL(_E)  
この関数はSTRING型データをREAL型データへ変換します。

#### ラダー、FBD/LD、STでの使用例
- ラダー/FBD/LDでは、EN/ENO付き・なしの2種類の形態が示されています。
- ST言語では、EN/ENOなしの場合は `d:=STRING_TO_REAL(s);`、EN/ENO付きの場合は `d:=STRING_TO_REAL_E(EN,ENO,s);` と記述します。

#### 設定データ
| 引数 | 内容 | 種別 | データ型 |
| --- | --- | --- | --- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | STRING(24) |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | REAL |

#### 機能
##### 演算処理
- 入力されたSTRING型（小数点型式/指数型式）のデータをREAL型のデータに変換し、出力(d)します。

### 表
#### STRING型データのバイト構成
| オフセット | 上位バイト | 下位バイト |
| --- | --- | --- |
| +0 | 1文字目のアスキーコード | 符号のアスキーコード |
| +1 | 3文字目のアスキーコード | 2文字目のアスキーコード |
| +2 | 5文字目のアスキーコード | 4文字目のアスキーコード |
| +3 | 7文字目のアスキーコード | 6文字目のアスキーコード |
| +4 | 9文字目のアスキーコード | 8文字目のアスキーコード |
| +5 | 11文字目のアスキーコード | 10文字目のアスキーコード |
| +6 | 00H（文字列の終端を示す） |  |

このバイト配列のSTRING型データをREAL型に変換して出力します。
