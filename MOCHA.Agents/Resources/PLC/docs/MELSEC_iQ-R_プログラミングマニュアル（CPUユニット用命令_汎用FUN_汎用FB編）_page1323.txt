### テキスト
#### 32.9 WORD型→INT型変換
WORD_TO_INT(_E)関数はWORD型データをINT型データへ変換します。

#### 対応CPU
RnCPU、RnENCPU（プロセス・三重化・一般・安全）に対応しています。

#### 機能
- 演算処理  
  入力されたWORD型のデータ(s)をINT型のデータ(d)に変換して出力します。  
  sへの入力値はWORD型のデータ値です。

- 演算結果  
  1. EN/ENOなしファンクション  
  演算処理を実行し、dから演算出力値が出力されます。  
  2. EN/ENO付きファンクション  
  実行条件と演算結果は表の通りです。

#### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDの関数ブロック図  
  - EN/ENOなし: 入力s、出力dの2端子  
  - EN/ENO付き: 入力EN、入力s、出力ENO、出力dの4端子  
- ST言語例  
  - EN/ENOなし: d := WORD_TO_INT(s);  
  - EN/ENO付き: d := WORD_TO_INT_E(EN, ENO, s);

### 表
#### 設定データ（引数、内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | WORD |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | INT |

#### 実行条件と演算結果
| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となるため、そのデータを使用しないようプログラム処理を行う必要があります。

### 演算例
- 入力WORD値「5678H」がINT型に変換され「22136」として出力される例が示されています。
