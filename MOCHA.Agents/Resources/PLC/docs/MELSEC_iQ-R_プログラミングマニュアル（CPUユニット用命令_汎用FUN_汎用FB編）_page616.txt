### テキスト
#### 機能
■DROL(P)
- (d)で指定されたデバイスのBIN32ビットデータを、SM700（キャリフラグ）を含めないで(n)ビット左へ回転させる命令。
- SM700（キャリフラグ）はDROL(P)命令の実行前の状態によりONまたはOFFになる。
- (d)にビットデバイスが指定された場合は、桁指定で指定されたデバイス範囲で回転する。
- 実際に回転するビット数は、(n)÷（桁指定で指定された点数）の余りとなる。
  - 例：n=31、点数=24ビットの場合、31÷24=1余り7なので7ビット左へ回転。
- (n)は0～31を指定可能。32以上の値が指定された場合は、(n)÷32の余りの値で回転。
  - 例：n=34の場合、34÷32=1余り2なので2ビット左へ回転。

■DRCL(P)
- DROL(P)と同様に(d)で指定されたデバイスのBIN32ビットデータを、SM700（キャリフラグ）を含めて(n)ビット左へ回転させる命令。
- SM700（キャリフラグ）はDRCL(P)命令の実行前の状態によりONまたはOFFになる。
- (d)にビットデバイスが指定された場合は、桁指定で指定されたデバイス範囲で回転。
- 実際に回転するビット数は、(n)÷（桁指定で指定された点数）の余りとなる。
  - 例：n=31、点数=24ビットの場合、31÷24=1余り7なので7ビット左へ回転。
- (n)は0～31を指定可能。32以上の値が指定された場合は、(n)÷32の余りの値で回転。
  - 例：n=34の場合、34÷32=1余り2なので2ビット左へ回転。

#### エラー
- 演算エラーはありません。
