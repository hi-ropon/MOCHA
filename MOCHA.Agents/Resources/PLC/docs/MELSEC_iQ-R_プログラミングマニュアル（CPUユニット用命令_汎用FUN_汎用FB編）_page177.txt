### テキスト
#### アナンシェータ OUT F
OUT F命令は、OUT F命令までの演算結果を指定のアナンシェータへ出力します。

##### 実行条件
- 命令：OUT F
- 実行条件：常時実行

##### 機能
- OUT F命令までの演算結果を指定のアナンシェータへ出力します。
- OUT F命令でアナンシェータ(F)をONする場合は以下の動作となります。
  - CPUユニット前面のUSER LEDが点灯します。
  - ONするアナンシェータ番号(F番号)が特殊レジスタ(SD64～SD79)に格納されます。
  - SD63の内容が+1されます。
- SD63の内容が16の場合（アナンシェータがすでに16個ONしている場合）は、新たにアナンシェータがONしても、ONするアナンシェータ番号がSD64～SD79には格納されません。
- OUT F命令でアナンシェータをOFFする場合は以下の動作となります。
  - コイルはOFFしますが、USER LEDの状態、SD63～SD79の内容は変化しません。
  - USER LEDの消灯、SD63～SD79からOUT F命令でOFFしたアナンシェータを削除する場合は、RST F命令で行います。

##### エラー
- 演算エラーはありません。
