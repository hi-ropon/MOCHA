### テキスト
#### 機能
- (s)で指定された単精度実数を符号なしBIN16ビットデータに変換し、(d)で指定されたデバイスに格納する。  
- 変換後のデータは、(s)で指定された単精度実数の小数点以下1桁目を四捨五入した値となる。  
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。  
- 注意事項は47ページを参照。  

#### プログラム例の説明
- M0がONしたとき、D0～D1に格納された単精度実数を符号なしBIN16ビットデータに変換し、D100へ格納する。

### 図
- プログラム例のラダー図：SM402、M0の条件でEMOVP命令によりE1234.5をD0にセットし、FLT2UINT命令でD0の値をD100に変換格納する流れを示す。  
- ビット構成図：  
  - D1、D0の16ビットずつのビット列（b31～b16、b15～b0）を示し、値はそれぞれ449AH（十六進数）、5000H。  
  - これを符号なしBIN16ビットデータに変換し、D100に04D3H（1235）として格納する過程を表現。
