### テキスト
#### DWORD型→BOOL型変換
DWORD型データをBOOL型データへ変換する関数「DWORD_TO_BOOL(_E)」について説明しています。

##### 機能
- (s)に入力されたDWORD型のデータをBOOL型のデータに変換し、(d)から出力します。
- 入力値が0Hの場合はFALSEを出力し、0H以外の場合はTRUEを出力します。
- (s)への入力値はDWORD型のデータ値です。

##### 演算結果
1. EN/ENOなしファンクション  
　演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
　実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、(d)から出力されたデータを使用しないようプログラム処理を行う必要があります。

##### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LD言語での関数ブロック図  
　EN/ENOなしとEN/ENO付きの2種類のブロック図が示されています。  
　EN/ENOなしは単純にs入力からd出力へ変換。  
　EN/ENO付きはEN入力、ENO出力が追加されています。

- ST言語での記述例  
　EN/ENOなし：d := DWORD_TO_BOOL(s);  
　EN/ENO付き：d := DWORD_TO_BOOL_E(EN, ENO, s);

- DWORD型→BOOL型変換のイメージ図  
　入力(s)が0Hの場合はFALSE、0H以外の場合はTRUEを出力(d)する流れを矢印で示しています。

### 表
#### 設定データ（引数の内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | DWORD |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL |

#### 実行条件と演算結果の対応表

| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE*1 | 不定値 |

*1 ENOがFALSEの場合、(d)のデータは不定値となるため注意が必要。
