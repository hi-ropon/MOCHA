### テキスト
#### 時計データの減算
「DATE-(P)」命令は時間データを減算する命令です。  
対応するCPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化）、RnPSFCPU（一般、安全）、RnSFCPU（一般、安全）です。  

##### ラダー図
ラダー図では、(s1)、(s2)、(d)の3つのオペランドが示されています。  

##### FBD/LD図
FBD/LD図では、EN（実行条件）、s1、s2（入力）、ENO（実行結果）、d（出力）端子が示されています。  
（図中の□にはDATEMINUS、DATEMINUSPが入ります）  

##### 実行条件
| 命令   | 実行条件の波形 |
|--------|----------------|
| DATE-  | 立下りエッジ    |
| DATE-P | 立上りエッジ    |
