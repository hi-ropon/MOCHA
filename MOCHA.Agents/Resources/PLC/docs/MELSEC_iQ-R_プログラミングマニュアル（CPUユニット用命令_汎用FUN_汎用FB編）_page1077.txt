### テキスト
#### 機能
(s1)で指定された時刻データから、(s2)で指定された時間データを減算し、その結果を(d)で指定されたデバイス番号以降に格納する機能について説明しています。

#### 例
10時40分20秒から3時50分10秒を減算した場合の具体例が示されています。  
- (s1)の時刻データ：10時40分20秒  
- (s2)の時間データ：3時50分10秒  
- 減算結果(d)：6時50分10秒  

また、減算結果が負の数になった場合は、そのデータに24を加えた値が演算結果となることが説明されています。  
例として、4時50分32秒から10時42分12秒を減算すると、-6時8分20秒ではなく18時8分20秒になると記載されています。

#### エラー
| エラーコード (SD0) | 内容                         |
|--------------------|------------------------------|
| 3405H              | (s1)、(s2)のデータが範囲外のとき。 |

### 図（数値配置例）
- (s1)、(s2)、(d)の3つの列にそれぞれhour（0～23）、minute（0～59）、second（0～59）の時刻データが配置されている。
- 減算は(s1)の時刻から(s2)の時刻を引き、結果を(d)に格納する。
- 例示では具体的な数値が表形式で示され、減算の結果がどのように(d)に格納されるかが視覚的に理解できる。
