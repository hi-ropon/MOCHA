### テキスト
#### PID演算命令およびPID制御命令の使い分け
PID制御を行うための命令には「PID演算命令」と「PID制御命令」の2種類があり、それぞれの使用条件が示されています。

#### PID演算命令の概要
PID演算命令は、目標値（SV）に近づけるために、測定値（PV）から比例動作（P動作）、積分動作（I動作）、微分動作（D動作）を組み合わせて出力値（MV）を演算する命令です。

- 警報出力機能：入力（測定値）や出力（値）の変化量に応じて警報出力をONにできます。
- 出力値の上下限設定：PID制御の積分項の増大を抑制するために出力の上下限値を設定可能。
- オートチューニング機能：比例ゲイン（Kp）、積分時間（Ti）、微分時間（Td）を自動設定可能。リミットサイクル法またはステップ応答法を選択できる。
- 演算方式：速度形・測定値微分形で演算を行う。
