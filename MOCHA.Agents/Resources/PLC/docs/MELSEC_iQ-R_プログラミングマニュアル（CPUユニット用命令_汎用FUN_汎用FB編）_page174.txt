### テキスト
#### 機能
- OUT命令までの演算結果がOFFからONに変化したとき、(d)(ST, FBD/LDの場合はCoil)で指定したカウンタの現在値（カウント値）を+1する。
- カウントアップ（現在値＝設定値）すると、a接点は導通し、b接点は非導通になる。
- 演算結果がONのままではカウントされない（カウント入力はパルス化する必要なし）。
- カウントアップ後は、RST命令を実行するまでカウント値および接点の状態は変化しない。
- 設定値が0の場合は、1と同一処理となる。

#### エラー
- 演算エラーはありません。
