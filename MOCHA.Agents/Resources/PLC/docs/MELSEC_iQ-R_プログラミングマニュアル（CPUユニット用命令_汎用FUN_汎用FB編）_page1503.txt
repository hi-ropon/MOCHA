### テキスト
#### 機能
■演算処理  
(s)がOFFすると、(d)を1スキャンのみONします。

■演算結果  
1. EN/ENOなしファンクションブロック  
演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクションブロック  
実行条件と演算結果は表に示す通りです。

#### エラー  
演算エラーはありません。

### 図（タイミングチャート）
1. EN/ENOなしファンクションブロックのタイミングチャート  
- (s)信号がON/OFFを繰り返す中で、(s)の立下りで(d)がONになる。  
- 次のスキャンで(d)はOFFになる。

2. EN/ENO付きファンクションブロックのタイミングチャート  
- EN信号がONかつ(s)の立下りで(d)がONになる。  
- 次スキャンで(d)はOFFになる。  
- ENがOFFの場合、(d)は前回のスキャンの出力を保持する。

### 表
#### 実行条件と演算結果

| 実行条件 | 演算結果 | 内容 |
| :--- | :--- | :--- |
| EN | ENO | (d) |
| TRUE（演算実行） | TRUE | 演算出力値 |
| FALSE（演算停止） | FALSE | 前回出力値 |

### 注釈
(1) EN=ONかつ(s)の立下りで(d)をONします。  
(2) 次スキャンで(d)をOFFします。  
(3) EN=OFFの場合は、(d)は前回のスキャンの出力を保持します。
