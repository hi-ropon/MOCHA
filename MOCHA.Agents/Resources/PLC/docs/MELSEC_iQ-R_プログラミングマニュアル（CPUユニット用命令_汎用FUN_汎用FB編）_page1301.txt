### テキスト
#### 待機系から制御系へのデータ書込み（CONTWR(P)）
- プロセスCPU（二重化モード）において、ファームウェアバージョンが18以降、エンジニアリングツールが1.050C以降で使用可能。
- 両系実行プログラムで、待機系から制御系へデータを書き込むための命令。

#### 実行条件
- CONTWR命令は立下りエッジで実行。
- CONTWRP命令は立上りエッジで実行。

### 図
#### ラダー図
- (s1)(s2)(d1)(d2)の4つのオペランドを持つ命令ブロックが示されている。

#### FBD/LD図
- EN（実行条件）、ENO（実行結果）、s1、s2、d1、d2の入出力ポートを持つ命令ブロックの構成図。
