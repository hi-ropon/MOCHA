### テキスト
#### デバッグ，故障診断命令「LEDR」
CPUユニットのアナンシェータ表示や運転を継続できる自己診断エラーの表示をリセットする命令です。1回の命令実行でエラー表示とアナンシェータの両方をリセットします。

##### 機能詳細
- 自己診断エラー発生時の動作：
  - 運転を継続できる自己診断エラーが発生している場合、CPUユニット前面のERROR LEDが消灯します。
  - ただし、SM0、SM1（最新自己診断エラー）、SD0（最新自己診断エラーコード）の内容はリセットされず、プログラムでのリセットが必要です。
  - バッテリエラー発生時は、バッテリ交換後にLEDR命令を実行するとCPUユニット前面のBATTERY LEDが消灯し、SM51（バッテリ低下ラッチ）もOFFになります。
- アナンシェータ(F)がONしているときの動作：
  - USER LEDが消灯します。
  - SD62（アナンシェータ番号）、SD63（アナンシェータ個数）、SD64～79（アナンシェータ検出番号テーブル）をすべてクリアします。

### 図
- ラダー図：LEDR命令のラダー回路例。接点がONのときにLEDR命令が実行される。
- FBD/LD図：EN入力で命令が有効になり、ENO出力で命令の実行結果を返すブロック図。
