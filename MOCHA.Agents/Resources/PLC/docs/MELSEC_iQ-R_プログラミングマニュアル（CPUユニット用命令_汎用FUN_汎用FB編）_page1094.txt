#### 使用可能デバイス
オペランドごとに使用可能なビット、ワード、ダブルワード、間接指定、定数、その他のデバイスが一覧表で示されています。  
- (s1), (s2), (s3)は使用可能だが、FX、FYは使用不可。  
- (s4)は一部のデバイスが使用不可。  
- (d)は一部のワードデバイスが使用不可。  
注釈として、(s1), (s2), (s3)で指定した時刻データと(s4)で指定した時刻データを比較し、その結果に応じて(d), (d)+1, (d)+2のいずれかがONになることが説明されています。

#### 機能
(s1), (s2), (s3)で指定した時刻データと(s4)で指定した時刻データを比較し、比較結果（小、一致、大）に応じて(d), (d)+1, (d)+2のいずれかがONになります。  
- (s1): 時  
- (s2): 分  
- (s3): 秒  
- (s4): 時  
- (s4)+1: 分  
- (s4)+2: 秒

#### 時刻データ比較回路図
- X0の入力に対して、TCMP命令で(s1), (s2), (s3), (s4), (d)の各オペランドを比較。  
- (d), (d)+1, (d)+2のいずれかがONになる条件を示す。  
- 比較条件は以下の通り:  
  - (d)がON: (s1), (s2), (s3) > (s4), (s4)+1, (s4)+2  
  - (d)+1がON: (s1), (s2), (s3) = (s4), (s4)+1, (s4)+2  
  - (d)+2がON: (s1), (s2), (s3) < (s4), (s4)+1, (s4)+2
