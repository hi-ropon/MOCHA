### 表
#### 命令処理時間一覧
この表は各命令の処理時間を示しています。処理時間はマイクロ秒（μs）単位で、CPUユニット別に最小値と最大値が記載されています。

- **命令名**: 各命令の名称が記載されています。例としてORD>=_U、CMP、DCMP、ZCP、BKCMP=など多数の命令が含まれています。
- **条件**: 命令の実行条件が記載されています。多くは「—」で特に条件がないことを示していますが、一部命令は(n)=1や(n)=96などの条件が指定されています。
- **処理時間（μs）**:
  - R00CPU、R01CPUの最小値と最大値
  - R02CPUの最小値と最大値
  - 左記以外のCPUユニットの最小値と最大値

例えば、ORD>=_U命令は無実行時にR00CPUで0.12544μs、実行時（導通時/非導通時）に0.18816μsの処理時間がかかります。CMP命令はR00CPUで最小4.4μs、最大6.9μsの処理時間です。

この表はCPUの種類や命令の条件によって処理時間が異なることを示しており、性能評価や最適化に役立つ情報となっています。
</contents>
