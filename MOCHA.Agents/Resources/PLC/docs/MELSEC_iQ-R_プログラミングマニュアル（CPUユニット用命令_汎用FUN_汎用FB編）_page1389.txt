### テキスト
#### 32.53 STRING型→BOOL型変換
関数名：STRING_TO_BOOL(_E)  
この関数はSTRING型データをBOOL型データへ変換します。

#### 設定データ
■内容、種別、データ型  
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE: 実行、FALSE: 停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | STRING(1) |
| ENO | 出力状態(TRUE: 正常、FALSE: 異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | BOOL |

#### 機能
■演算処理  
- 入力されたSTRING型（小数点型式/指数型式）のデータをBOOL型のデータに変換し、出力(d)します。  
- 例として、sが"0"の場合はFALSE、"12"の場合はTRUEに変換されます。

■演算結果  
1. EN/ENOなしファンクション  
- 演算処理を実行し、dから演算出力値が出力されます。  
2. EN/ENO付きファンクション  
- 実行条件と演算結果は以下の通りです。

| 実行条件 EN | 演算結果 ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となるため、その場合はdから出力されたデータを使用しないようプログラム処理を行う必要があります。

#### エラー
演算エラーはありません。

### 図
- ラダー図、FBD/LD図では、EN/ENO付き・なしの関数ブロックの形状が示されています。  
- ST言語では、EN/ENOなしの場合は「d:=STRING_TO_BOOL(s);」、EN/ENO付きの場合は「d:=STRING_TO_BOOL_E(EN,ENO,s);」と記述します。
