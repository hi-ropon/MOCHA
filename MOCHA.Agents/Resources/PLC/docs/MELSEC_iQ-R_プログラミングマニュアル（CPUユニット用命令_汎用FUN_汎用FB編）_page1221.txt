### テキスト
#### ステップ応答法
ステップ応答法の概要は1583ページを参照するよう案内されています。

##### ■オートチューニング（ステップ応答法）で設定される制御用データ
- 動作方向（正動作/逆動作）: (s3)+1 ビット0
- 比例ゲイン(Kp): (s3)+3
- 積分時間(Ti): (s3)+4
- 微分時間(Td): (s3)+6

##### ■オートチューニング手順
1. **オートチューニング用出力値の設定**  
   出力値(MV)にオートチューニング用出力値を転送する。出力機器に対して出力可能最大値の0.5～1.0倍に設定する。

2. **オートチューニングで設定されないデータの設定**  
   下記の設定項目をシステムに応じて設定する。

| 設定項目 | 備考 |
| :--- | :--- |
| (s1) 目標値(SV) | 測定値(PV)との差が150以上になるように設定する。 |
| (s3) サンプリングタイム(Ts) | 1000ms以上に設定する。 |
| (s3)+2 入力フィルタ(α) | 設定なし |
| (s3)+5 微分ゲイン(KD) | 入力フィルタを設定する場合、微分ゲインは通常"0"に設定する。 |
| その他 | 必要に応じて設定する。 |

*備考*  
- 目標値(SV)は測定値(PV)との差が150以上でないと正しくオートチューニングできないため、150未満の場合は目標値を設定し直す必要がある。  
- サンプリングタイム(Ts)は必ず1000ms以上に設定し、出力変化周期より十分長い時間にする。

3. **オートチューニングの開始**  
   (s3)+1(動作設定ACT)のビット4をONにするとオートチューニングが開始される。  
   測定値から目標値への変化量が1/3以上変化するとオートチューニング完了となり、ビット4は自動的にOFFになる。

##### ■オートチューニング時の注意事項
- 測定値(PV)が変化しない場合のプログラム対策  
  アナログ入力断線などにより測定値が正常に変化しない場合、オートチューニングが終了しない。  
  入力値やチューニング開始からの経過時間を監視するシーケンスを導入し、検知・回避することが推奨される。
