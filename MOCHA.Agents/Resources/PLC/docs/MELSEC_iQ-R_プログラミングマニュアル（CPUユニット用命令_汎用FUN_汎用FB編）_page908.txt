### テキスト
#### 機能
- (s)で指定された角度のCOS（余弦）値を演算し、(d)で指定されたデバイス番号に演算結果を格納する。
- COS関数の引数(s)は単精度実数で、角度はラジアン単位（角度×π÷180）で設定する。
- 演算結果が0またはアンダーフローが発生した場合、演算結果は0となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。
- 詳細は47ページの注意事項を参照。

#### エラー
- エラーコード「3402H」：(s)に指定した値が-0、非正規化数、非数、±∞のときに発生。

#### Point
- 角度とラジアンの変換については、RAD(P)命令、DEG(P)命令を参照すること。
- 参照ページ：941ページ（RAD(P)）、943ページ（DEG(P)）
