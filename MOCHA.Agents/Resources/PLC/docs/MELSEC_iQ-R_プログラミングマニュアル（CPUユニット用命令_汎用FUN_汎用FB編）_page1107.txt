### テキスト
#### アワーメータ（HOURM）
- 対応CPU: R00CPU、R01CPU、R02CPUは制約なし。RnCPU（R00CPU、R01CPU、R02CPU除く）はファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降で使用可能。
- 機能: 起動接点がONしている時間を1時間単位で計測し、累計ON時間が指定時間（BIN16ビットデータ）に達した時点で、指定デバイスをONにする。

#### ラダー図
- 入力: (s)（アラーム時間）、(d1)（計測値格納デバイス）、(d2)（タイムアップ時ONデバイス）
- 出力: なし（内部処理）
- ラダー命令: ENO := HOURM(EN, s, d1, d2)

#### ST言語
- ENO := HOURM(EN, s, d1, d2)

#### FBD/LD図
- ブロックにEN（入力）、ENO（出力）、s、d1、d2の端子がある。
