### テキスト
#### データベーストランザクション開始(DBTRANS(P))
指定した識別番号に対するデータベースでのトランザクション開始を宣言する命令です。R00CPU、R01CPU、R02CPUでは使用できません。

### 図
#### ラダー図
- ラダー図では、(s)、(d1)、(d2)の3つのオペランドを持つ命令ブロックが示されています。

#### FBD/LD図
- FBD/LD図では、EN、ENO、s、d1、d2のポートを持つ命令ブロックが示されています。
