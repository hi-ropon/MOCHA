### テキスト
#### 機能
- 指定したデバイス(d)から、(n)点分の倍精度実数を左へ1点シフトする。
- 最下位から1倍精度実数分は0になる。
- (n)で指定された値が0の場合は無処理となる。

#### エラー
- 演算エラーは発生しない。

### 図（例）
- (n)=6の場合のデータシフトの動作を示す。
- 上段に元のデータ列（12桁以上の倍精度実数）が並び、(d)+23～(d)+20、(d)+19～(d)+16、…のようにデバイスアドレスが示されている。
- 下段は左へ1点シフトした結果で、最下位のデバイスには0が入る。
- 具体的には、元のデータのうち最も右側の6点分が左に1点ずつずれ、最も右端の位置は0で埋められている。
