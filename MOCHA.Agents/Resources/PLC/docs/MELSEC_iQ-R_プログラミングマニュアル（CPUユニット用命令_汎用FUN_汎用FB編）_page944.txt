### テキスト
#### 機能
- 角度の大きさの単位を、(s)で指定されたラジアン単位から度単位（DEG.単位）に変換し、(d)で指定されたデバイス番号に格納する機能について説明しています。
- ラジアン単位から度単位への変換は以下の式で行います。
  \[
  \text{度単位} = \text{ラジアン単位} \times \frac{180}{\pi}
  \]
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあり、その注意事項については47ページを参照するよう案内しています。

### 表
#### エラーコード一覧
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3402H              | (s)に指定した値が-0、非正規化数、非数、±∞のとき。 |
| 3403H              | (d)が下記範囲を超えるとき。（オーバーフローが発生したとき。）<br> \(|d| < 2^{128}\) |

- 表はエラーコードとその発生条件を示しており、角度変換処理における異常状態を説明しています。
