### テキスト
#### 機能
- (d)で指定されたSM420～SM424（ユーザ用タイミングクロック）を、(n1)で指定されたスキャン分ONし、(n2)で指定されたスキャン分OFFします。
- タイミング図では、SM420～SM424のON/OFF状態が(n1)の期間ON、(n2)の期間OFFとして示されています。
- (n1)スキャン、(n2)スキャンの意味はそれぞれ(n1)スキャン、(n2)スキャンと定義されています。
- スキャン実行タイププログラムではSM420～SM424を使用します。
- (n1)、(n2)が0に設定された場合の動作は以下の通りです。
  - (n1)=0、(n2)≥0の場合はSM420～SM424はOFFのまま。
  - (n1)>0、(n2)=0の場合はSM420～SM424はONのまま。
- DUTY命令実行時に(n1)、(n2)、(d)で指定されたデータをシステムに登録し、タイミングパルスのON/OFFはEND処理で行います。

### 図
- SM420～SM424のON/OFFタイミングを示す図
  - 横軸にスキャン数を示し、(n1)の期間がON、(n2)の期間がOFFとして棒グラフで表現。
  - ONとOFFの状態が交互に繰り返されるイメージ。
