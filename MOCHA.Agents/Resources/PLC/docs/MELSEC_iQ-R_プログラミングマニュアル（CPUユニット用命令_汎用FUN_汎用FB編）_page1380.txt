### テキスト
#### 32.46 LREAL型→REAL型変換
LREAL_TO_REAL(_E)関数は、LREAL型データをREAL型データへ変換します。

#### 機能
- 演算処理  
  入力されたLREAL型のデータ(s)をREAL型のデータ(d)に変換して出力します。  
  入力値はLREAL型のデータ値です。

- 演算結果  
  1. EN/ENOなしファンクション  
     演算処理を実行し、dから演算出力値が出力されます。  
  2. EN/ENO付きファンクション  
     実行条件と演算結果は以下の通りです。

### 図
- ラダー、FBD/LDの関数ブロック図  
  - EN/ENOなし：s入力からd出力へ変換するシンプルなブロック  
  - EN/ENO付き：EN入力、ENO出力を持ち、s入力からd出力へ変換するブロック

- ST言語での使用例  
  - EN/ENOなし：d:=LREAL_TO_REAL(s);  
  - EN/ENO付き：d:=LREAL_TO_REAL_E(EN,ENO,s);

### 表
#### 設定データ（引数の内容、種別、データ型）

| 引数  | 内容                          | 種別     | データ型 |
|-------|-------------------------------|----------|----------|
| EN    | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL     |
| s(IN) | 入力                          | 入力変数 | LREAL    |
| ENO   | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL     |
| d     | 出力                          | 出力変数 | REAL     |

#### 実行条件と演算結果

| EN                  | ENO                  | (d)の値       |
|---------------------|----------------------|---------------|
| TRUE (演算実行)      | TRUE                 | 演算出力値    |
| FALSE (演算停止)     | FALSE*1              | 不定値        |

*1 ENOからFALSEが出力された場合、dから出力されるデータは不定値となるため、その場合はdから出力されたデータを使用しないようプログラム処理を行う必要があります。
