### テキスト
#### BIN16ビットデータ比較出力 CMP(P)(_U)
- 対応CPUはR00CPU、R01CPU、R02CPUでバージョン制約なし。RnCPU（R00CPU、R01CPU、R02CPU除く）とRnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降で使用可能。
- (s1)で指定したBIN16ビットデータと(s2)で指定したBIN16ビットデータを比較し、結果（小、一致、大）に応じて(d)、(d)+1、(d)+2のいずれかがONになる。

### 図
- ラダー図：3つの入力(s1)(s2)(d)を持つ比較命令のラダー表現。
- ST言語例：ENO:=CMP(EN,s1,s2,d); ENO:=CMPP(EN,s1,s2,d); ENO:=CMP_U(EN,s1,s2,d); ENO:=CMPP_U(EN,s1,s2,d);
- FBD/LD図：EN、s1、s2を入力、ENOとdを出力とする機能ブロック図。
- 実行条件タイミング図：CMP/CMP_Uは立下りエッジ、CMPP/CMPP_Uは立上りエッジで実行。

### 表
#### 実行条件
| 命令       | 実行条件       |
|------------|----------------|
| CMP, CMP_U | 立下りエッジ    |
| CMPP, CMPP_U | 立上りエッジ  |

#### 設定データ（内容、範囲、データ型）
| オペランド | 内容                         | 範囲           | データ型               | データ型(ラベル)    |
|------------|------------------------------|----------------|------------------------|---------------------|
| (s1)       | 比較データまたは格納デバイス | -32768～32767 (符号付き) / 0～65535 (符号なし) | 符号付きBIN16ビット / 符号なしBIN16ビット | ANY16_S / ANY16_U |
| (s2)       | 比較データまたは格納デバイス | -32768～32767 (符号付き) / 0～65535 (符号なし) | 符号付きBIN16ビット / 符号なしBIN16ビット | ANY16_S / ANY16_U |
| (d)        | 比較結果格納先先頭デバイス   | なし           | ビット配列（要素数3）  | ANYBIT_ARRAY        |
| EN         | 実行条件                     | なし           | ビット                  | BOOL                |
| ENO        | 実行結果                     | なし           | ビット                  | BOOL                |
