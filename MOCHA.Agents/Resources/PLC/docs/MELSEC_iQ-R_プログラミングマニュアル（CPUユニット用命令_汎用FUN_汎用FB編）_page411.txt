### テキスト
#### 機能
(s)で指定されたデバイスのBIN16ビットデータ（0～9999）をBCD4桁データに変換し、(d)で指定されたデバイスに格納します。

#### エラー
| エラーコード (SD0) | 内容                         |
|--------------------|------------------------------|
| 3401H              | (s)のデータが0～9999以外のとき。 |

### 図
BIN16ビットデータ（0～9999）の各ビットがどのようにBCD4桁データに変換されるかを示しています。

- BIN9999のビット配列（2^15～2^0）が示されており、特定のビットが1または0に設定されています。
- 変換後のBCD9999は4桁のBCDコードで表され、それぞれの桁（×10^3、×10^2、×10^1、×10^0）に対応するビットが示されています。
- (1)の部分は必ず0にする必要があることが注記されています。

この図はBIN16ビットデータをBCD4桁に正確に変換するためのビット対応関係を視覚的に説明しています。
