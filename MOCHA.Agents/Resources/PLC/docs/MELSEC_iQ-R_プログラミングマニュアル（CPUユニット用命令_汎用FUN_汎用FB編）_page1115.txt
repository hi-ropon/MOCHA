### テキスト
#### ユニットリフレッシュ S(P).ZCOM
指定されたユニットのリフレッシュ処理を行う命令です。シーケンスプログラムの任意のタイミングでリフレッシュを実施する際に使用します。

#### 実行条件
- S.ZCOM命令は立下りエッジで実行
- SP.ZCOM命令は立上りエッジで実行

#### 注意事項
- CC-Link IE TSN Plusマスタ・ローカルユニットでは、(U)指定時にRJ71GN11-EIPのリンクリフレッシュが動作
- シンプルモーションユニット(RD77GF)やモーションユニットのバッファメモリリフレッシュも動作対象

### 図
#### ラダー図
- (J/U)入力によりリフレッシュ処理を実行するラダー命令の構造を示す
- ST命令ではENO出力により各種ZCOM命令の実行結果を示す

#### FBD/LD図
- EN入力、ENO出力、J/U入力を持つ命令ブロックの構造を示す

| :--- | :--- | :--- | :--- | :--- |
| (J)*1*2 | 自局のネットワークNo. | 1～239 | デバイス名 | ANY16 |
| (U)*1 | ユニットの先頭入出力番号(16進4桁の上3桁) | 0H～FFH | － | － |
| EN | 実行条件 | － | ビット | BOOL |

*注釈*
- *1 CC-Link IE TSN Plusマスタ・ローカルユニットでの動作詳細
- *2 シンプルモーションユニット(RD77GF)のリフレッシュ動作も含む

- S(P).ZCOM命令は任意のタイミングでユニットのリフレッシュを行うために使用され、対象はCC-Link IE TSNマスタ・ローカルユニット、コントローラネットワークユニット、フィールドネットワークマスタ・ローカルユニット、MELSECNET/Hユニット、CC-Linkユニット、インテリジェント機能ユニットなど多岐にわたります。
