### テキスト
#### 注意事項
- SET_BIT_OF_INT(_E)をSTで使用する場合は、戻り値を変数に代入するプログラムを作成する必要がある。
  - 例として、g_int1 := SET_BIT_OF_INT(TRUE, 0); g_bool1 := GET_BIT_OF_INT(g_int1, 0); のコードが示されている。
- SET_BIT_OF_INT(_E)の戻り値は入出力を兼ねているため、他の命令やファンクション、ファンクションブロックの入力引数に直接代入しないこと。
  - 例として、GET_BIT_OF_INTの第一引数にSET_BIT_OF_INTを直接入れると不定値になる旨が記載されている。

#### エラー
- 演算エラーは存在しない。

#### Point（ポイント）
- ラベルのビット指定を使用すると、SET_BIT_OF_INTと同じ動作のプログラムを簡潔に作成可能。
- 例として、SET_BIT_OF_INTを使用した場合と同様に、g_int1のビット5(b5)をg_bool1の値に変更する方法が示されている。

### 図
- ラダーの場合の例：g_int1.5 := g_bool1;
- FBD/LDの場合の例：g_bool1からg_int1.5へ接続されたブロック図が示されている。

これにより、ラベルのビット指定を使った簡潔なプログラム作成方法が理解できる。
