### テキスト
#### 機能
(s1)で指定したBCD8桁データと、(s2)で指定したBCD8桁データの加算を行い、その加算結果を(d)で指定したデバイスに格納します。

- 加算の例として、(s1)のBCDデータ「56789123」と(s2)のBCDデータ「01234567」を加算すると、結果は「58023690」となり、(d)に格納されます。
- 加算結果が99999999を超えた場合は桁上げは無視され、SM700のキャリフラグはONになりません。
- 例として「99900000」と「01654321」を加算すると、「00654321」となります。

#### エラー
| エラーコード (SD0) | 内容 |
|--------------------|------|
| 3405H              | (s1)のBCDデータが0～99999999以外の場合。<br>(s2)のBCDデータが0～99999999以外の場合。 |

### 図
- BCD8桁データの加算例を示す図があり、(s1)+1、(s1)、(s2)+1、(s2)、(d)+1、(d)の位置関係が示されています。加算結果が(d)に格納されることを視覚的に表現しています。
