### テキスト
#### BIN16ビットグレイコードデータ→BIN16ビットデータ変換
「GBIN(P)(_U)」命令は、指定したBIN16ビットグレイコードデータをBIN16ビットデータに変換します。

#### 実行条件
- GBIN、GBIN_U命令は立下りエッジで実行
- GBINP、GBINP_U命令は立上りエッジで実行

#### 機能
(s)で指定されたデバイスに格納されているBIN16ビットグレイコードデータを、BIN16ビットデータに変換し、(d)で指定されたデバイスに格納します。

### 図
- ラダー図：入力(s)と出力(d)の接続イメージを示す。
- FBD/LD図：EN（実行条件）、ENO（実行結果）、s（入力）、d（出力）の接続を示す。
- ビット列変換図：グレイコードのビット列からBINコードのビット列への変換例を示す。  
  例として、sのビット列「0000011011110111」がdのビット列「0000010011010010」に変換される様子を示している。
