| +2 | 書込み結果データ数 | (s3)で指定したデータに対し、実際に書き込んだデータ数。単位は(d1)+7のデータタイプによる | － | システム |
| +3 | 未使用 | － | － | － |

| +6 | 列数指定 | バイナリ書込み時は必ず0指定。CSV書込み時は書込み列数指定。<br>0: 列なし（1行）<br>0以外: 指定数の列 | 0000H～FFFFH (0～65535) | ユーザ |

#### 書込みデータ(s3)
- **オペランド**: (s3)
- **項目説明**

| デバイス | 項目 | 内容 | 設定範囲 | セット側 |
| :--- | :--- | :--- | :--- | :--- |
