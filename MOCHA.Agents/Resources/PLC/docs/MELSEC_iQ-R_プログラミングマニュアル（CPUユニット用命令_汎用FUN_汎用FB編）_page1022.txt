### テキスト
#### 17.1 マトリクス入力命令
マトリクス入力命令「MTR」は、指定した入力番号以降に接続されている16点×n列の入力を順次読み込みます。

### 図
- **ラダー図**  
  ラダー回路では、MTR命令が「(s)(d1)(d2)(n)」のパラメータを持つ形で表現されています。  
- **FBD/LD図**  
  FBD/LDでは、EN（イネーブル）入力に対してMTR命令ブロックがあり、出力はENOとなっています。パラメータはs、n、d1、d2です。  
- **実行条件図**  
  実行条件はパルス状の信号で、MTR命令がこの信号の立ち上がりで実行されることを示しています。

### 表
| 命令 | 実行条件 |
| :--- | :--- |
| MTR | パルス信号（立ち上がり） |

この表は、MTR命令がパルス信号の立ち上がりで実行されることを示しています。
