### テキスト
#### デバイスで扱うデータ
設定データをデバイスで指定する場合について、指定可能なデバイスや定数の種類を示しています。

### 表
#### データ型ごとの内容と指定可能デバイス/定数

| データ型           | 内容                                                                                     | 指定可能デバイス/定数                                                                                   |
|--------------------|------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| ビット             | ビットデータを取り扱うことができる。                                                    | ・ビットデバイス<br>・ワードデバイスのビット指定                                                        |
| ワード             | ワードデータを取り扱うことができる。                                                    | ・ワードデバイス<br>・ビットデバイスの桁指定(K1～K4)                                                  |
| 符号付きBIN16ビット | 16ビットデータを取り扱う。符号付きと符号なしで値の範囲が異なる。                        | ・10進定数<br>・16進定数                                                                                 |
| 符号なしBIN16ビット | 16ビットデータを取り扱う。符号付きと符号なしで値の範囲が異なる。                        | ・10進定数<br>・16進定数                                                                                 |
| ダブルワード       | ダブルワードデータを取り扱うことができる。                                            | ・ワードデバイス<br>・ダブルワードデバイス<br>・ビットデバイスの桁指定(K1～K8)<br>・10進定数<br>・16進定数 |
| 符号付きBIN32ビット | 32ビットデータまたは16ビットデータが2個連続しているデータを取り扱うことができる。符号付きと符号なしで値の範囲が異なる。 | ・ワードデバイス<br>・ダブルワードデバイス<br>・ビットデバイスの桁指定(K1～K8)<br>・10進定数<br>・16進定数 |
| 符号なしBIN32ビット | 32ビットデータまたは16ビットデータが2個連続しているデータを取り扱うことができる。符号付きと符号なしで値の範囲が異なる。 | ・ワードデバイス<br>・ダブルワードデバイス<br>・ビットデバイスの桁指定(K1～K8)<br>・10進定数<br>・16進定数 |
| BCD4桁            | BCD4桁データを取り扱う。16ビットデータを4桁で区切り、各桁を0～9で指定。               | ・ワードデバイス<br>・ビットデバイスの桁指定(K1～K4)<br>・10進定数<br>・16進定数                       |
| BCD8桁            | BCD8桁データを取り扱う。32ビットデータを8桁で区切り、各桁を0～9で指定。               | ・ワードデバイス<br>・ダブルワードデバイス<br>・ビットデバイスの桁指定(K1～K8)<br>・10進定数<br>・16進定数 |
| 単精度実数         | 単精度浮動小数点データを取り扱うことができる。                                         | ・ワードデバイス<br>・ダブルワードデバイス<br>・実数定数                                               |
| 倍精度実数         | 倍精度浮動小数点データを取り扱うことができる。                                         | ・ワードデバイス<br>・ダブルワードデバイス<br>・実数定数                                               |
| 文字列             | ASCIIコード、シフトJISコードの文字列データを取り扱うことができる。                     | ・ワードデバイス<br>・文字列定数                                                                     |
| Unicode文字列      | Unicodeの文字列データを取り扱うことができる。                                          | ・ワードデバイス<br>・文字列定数                                                                     |
| デバイス名         | デバイスを直接指定できる。                                                             | ・使用可能デバイスで対応のデバイス名                                                                   |

### 注釈
- *1 命令によってソース(s)や数値データ(n)の指定するデータに、定数が使用できる。
- *2 指定方法については、各データ型の詳細ページを参照すること。
