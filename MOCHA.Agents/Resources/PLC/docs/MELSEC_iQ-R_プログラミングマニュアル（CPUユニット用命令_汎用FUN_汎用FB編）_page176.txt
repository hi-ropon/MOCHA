### テキスト
#### 機能
- OUT命令の演算結果がOFFからONに変化したとき、(d)(ST, FBD/LDの場合はCoil)で指定したロングカウンタの現在値（カウント値）を+1する。
- カウントアップは現在値が設定値に達すると、a接点は導通し、b接点は非導通になる。
- 演算結果がONのままではカウントされない（カウント入力をパルス化する必要はない）。
- カウントアップ後は、RST命令を実行するまでカウント値および接点の状態は変化しない。
- 設定値が0の場合は、1と同じ処理となる。

#### エラー
- 演算エラーは存在しない。
