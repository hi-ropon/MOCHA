### テキスト
#### 28 SFCプログラム用命令
本章では、SFCプログラムの制御を行う命令について説明しています。詳細は「MELSEC iQ-R プログラミングマニュアル（プログラム設計編）」を参照してください。

#### 28.1 SFC制御命令 - ステップ活性チェック
使用可能な命令は以下の通りです。
- LD, LDI, AND, ANI, OR, ORI [S□/BL□ ¥ S□]

対応するCPUは、RnCPU、RnENCPU、RnPCPU（プロセス）、RnPCPU（二重化）、RnPSECPIU（一般）、RnPSECPIU（安全）、RnSFCPIU（一般）、RnSFCPIU（安全）です。ただし、バージョンによる制約があります。
- RnCPU（R00CPU、R01CPU、R02CPUなど）は制約なし。
- RnCPU（R00CPU、R01CPU、R02CPU除く）、RnENCPUはファームウェアバージョン12以降で使用可能。
- プロセスCPU（プロセスモード）はファームウェアバージョン03以降で使用可能。
- プロセスCPU（二重化モード）はファームウェアバージョン18以降で使用可能。

命令の意味：
- LD: a接点命令
- LDI: b接点命令
指定ステップの活性/非活性を演算結果とします。

- AND: a接点直列接続命令
- ANI: b接点直列接続命令
指定ステップの活性/非活性と、それまでの演算結果のAND演算を行います。

- OR: a接点1個の並列接続命令
- ORI: b接点1個の並列接続命令
指定ステップの活性/非活性と、それまでの演算結果のOR演算を行います。
