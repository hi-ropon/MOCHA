### テキスト
#### データ転送命令
BIN16ビットおよび32ビットデータの転送に関する命令群を説明しています。

##### ■BIN16ビット/32ビットデータ転送
- MOV / MOVP  
  指定されたデバイスのBIN16ビットデータを転送する命令。  
- DMOV / DMOVP  
  指定されたデバイスのBIN32ビットデータを転送する命令。  

##### ■BIN16ビット/32ビットデータ否定転送
- CML / CMLP  
  指定されたBIN16ビットデータをビットごとに反転して転送する命令。  
- DCML / DCMLP  
  指定されたBIN32ビットデータをビットごとに反転して転送する命令。  

##### ■桁移動
- SMOV / SMOVP  
  データを桁単位（4ビット）で分配または合成する命令。  

##### ■1ビットデータ反転転送
- CMLB / CMLBP  
  指定したビットデータを反転し、指定先に格納する命令。  

##### ■BIN16ビットデータブロック転送（16ビット）
- BMOV / BMOVP  
  指定されたデバイスから(n)点のBIN16ビットデータを一括転送する命令。  
  (n)は1～65535の範囲。  

##### ■BIN16ビットデータブロック転送（32ビット）
- BMOVL / BMOVLP  
  指定されたデバイスから(n)点のBIN16ビットデータを一括転送する命令。  
  (n)は1～4294967295の範囲。  

##### ■同一BIN16ビットデータブロック転送（16ビット）
- FMOV / FMOVP  
  BIN16ビットデータを指定されたデバイスから(n)点に転送する命令。  
  (n)は1～65535の範囲。
