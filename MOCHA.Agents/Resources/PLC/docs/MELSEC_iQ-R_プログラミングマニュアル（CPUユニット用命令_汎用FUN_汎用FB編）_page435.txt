### テキスト
#### 機能
- (s)で指定した符号付きBIN16ビットデータ(ANY16_S)を、符号なしBIN16ビットデータ(ANY16_U)に変換し、(d)で指定したラベルに格納する。
- INT2UINT命令の使用例を示す。

#### 例
- +_U命令はオペランドにANY16_Uを指定する必要があるため、+_U命令を実行する前にINT2UINT命令を使用して、ANY16_SのwLabel0をANY16_UのuLabel1へ変換する。
- 値としてはwLabel0をそのままuLabel1へ格納する。

#### エラー
- 演算エラーはありません。

### 図
- bSwitchAというビット信号があり、これがONのときINT2UINT命令が実行される。
- INT2UINT命令はwLabel0（符号付きワード）を入力として受け取り、uLabel1（符号なしワード）に変換して格納する。
- +_U命令はuLabel1をオペランドとして使用する。
- 図中の(1)はwLabel0の値をそのままuLabel1に格納することを示す。
- 図中の(2)は+_U命令のオペランドのデータ型に合わせて演算を行うことを示す。

### 用語説明
- bSwitchA：ビット信号
- wLabel0：符号付きワードデータ（16ビット）
- uLabel0、uLabel1：符号なしワードデータ（16ビット）
