### テキスト
#### D+(P)(_U) [オペランドが3個の場合]
指定した2つのBIN32ビットデータを加算する命令です。

#### 実行条件
- D+、D+_U命令は立ち上がりエッジで実行されます。
- D+P、D+P_U命令は立ち上がりレベルで実行されます。

#### 使用可能デバイス
- SIL2プロセスCPUおよび安全CPUの安全プログラムでは、特定の安全デバイスおよび安全ラベルのみ使用可能です。

### 図
- **ラダー図**  
  EN（実行条件）、s1、s2（加算する2つのデータ）、d（演算結果格納先）を示すラダー回路の簡単な図が掲載されています。

- **FBD/LD図**  
  EN（入力）、s1、s2（入力）、d（出力）、ENO（実行結果出力）を持つブロック図が示されています。  
  ブロック内にはDPLUS、DPLUSP、DPLUS_U、DPLUSP_Uのいずれかが入ることが記載されています。
