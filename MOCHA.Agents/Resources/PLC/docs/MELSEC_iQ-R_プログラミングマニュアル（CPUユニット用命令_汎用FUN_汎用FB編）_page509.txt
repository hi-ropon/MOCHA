### テキスト
#### 機能
(s)で指定されたデバイスから(n)点分のBIN16ビットデータの下位4ビットを、(d)で指定されたBIN16ビットデバイスに結合します。

- (d)で指定されたデバイスの上位(4-n)の桁のビットは0になります。  
- (n)は1～4が指定可能です。  
- (n)=0のときは無処理となり、(d)のデバイスの内容は変化しません。

#### 注意事項
- (1)の部分は無視されます。  
- (2)は結合データを示しています。

### 図
ビット結合のイメージ図があり、以下のように示されています。

- (s), (s)+1, (s)+2, (s)+3 の4つのデバイスのビットb15～b0が並んでいる。  
- それぞれのデバイスの下位4ビット(b3～b0)が抽出され、(d)のデバイスの下位16ビットの特定ビット位置に結合される。  
- (d)のデバイスの上位ビット(b15～b12)は0に設定される。
