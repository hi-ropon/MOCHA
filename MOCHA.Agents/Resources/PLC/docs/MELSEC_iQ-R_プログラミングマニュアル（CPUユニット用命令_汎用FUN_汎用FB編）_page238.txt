### テキスト
#### -(P)(_U) [オペランドが3個の場合]
指定した2つのBIN16ビットデータを減算する命令です。対応CPUはRnCPU、RnFNCU（プロセス、三重化、一般、安全）です。

#### 実行条件
命令には4種類あり、それぞれ実行条件が異なります。
- 「-」「-_U」：パルス実行
- 「-P」「-P_U」：立ち上がりエッジ実行

#### 使用可能デバイス
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、指定された安全デバイスおよび安全ラベルのみ使用可能です。

### 図
#### ラダー図
- ラダー図では、3つのオペランド（s1、s2、d）を持つ減算命令の構造が示されています。s1、s2が減算され、その結果がdに格納されます。

#### FBD/LD図
- FBD/LD図では、EN（実行条件）、s1、s2（入力データ）、d（出力データ）、ENO（実行結果）というポートを持つ命令ブロックの形状が示されています。
- ブロック内にはMINUS、MINUSP、MINUS_U、MINUSP_Uのいずれかが入ります。

### 表
#### 実行条件一覧

| 命令   | 実行条件       |
|--------|----------------|
| -      | パルス         |
| -_U    | パルス         |
| -P     | 立ち上がりエッジ |
| -P_U   | 立ち上がりエッジ |
