### テキスト
#### 32.63 INT型→ビット配列変換
関数名：INT_TO_BITARR(_E)  
INT型データの下位nビットをビット配列に出力する関数です。

#### 機能
- 演算処理  
  指定されたANY16型の入力(s)の下位nビットを出力(d)に出力します。  
  指定されたビット数以上の出力ビットは変更されません。

- 演算結果  
  1. EN/ENOなしファンクション  
  演算処理を実行し、(d)から演算出力値が出力されます。  
  2. EN/ENO付きファンクション  
  実行条件と演算結果は表の通りです。

#### エラー
演算エラーはありません。

### 図
- ラダー(FBD/LD)  
  EN/ENOなしとEN/ENO付きの2種類の関数ブロック図が示されています。  
  - EN/ENOなし：入力s、n、出力d  
  - EN/ENO付き：入力EN、s、n、出力ENO、d

- ST言語  
  EN/ENOなし：d := INT_TO_BITARR(s,n);  
  EN/ENO付き：d := INT_TO_BITARR_E(EN,ENO,s,n);

### 表
#### 設定データ（引数の内容、種別、データ型）

| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s | 入力 | 入力変数 | ANY16 |
| n | 4, 8, 12, 16の定数のみ指定可能 | 入力変数 | INT |
| ENO | 出力状態(TRUE:正常実行、FALSE:異常または停止) | 出力変数 | BOOL |
| d | 出力(要素指定は変数も可) | 出力変数 | BOOL配列要素 |

#### 実行条件と演算結果

| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE(演算実行) | TRUE | 演算出力値 |
| FALSE(演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、そのデータを使用しないようプログラムで処理してください。
