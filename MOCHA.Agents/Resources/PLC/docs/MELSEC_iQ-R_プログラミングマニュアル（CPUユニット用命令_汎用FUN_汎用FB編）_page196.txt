### テキスト
#### 5.5 マスタコントロール命令
マスタコントロール命令は、マスタコントロールのセットおよびリセットを行う命令です。  
対象CPUはRnCPU、RnENCPU、RnPCPU（プロセス・二重化・一般・安全）など複数の種類に対応しています。

##### Point
SIL2プロセスCPUおよび安全CPUの安全プログラムで本命令を使用する場合は、特に明記がない限り以下のように読み替えて使用します。  
- 「タイマ」→「安全タイマ」  
- 「積算タイマ」→「安全積算タイマ」  
- 「カウンタ」→「安全カウンタ」  
- デバイス「X」「M」には「SA￥」を付ける（例：「X0」→「SA￥X0」、「M0」→「SA￥M0」）  
- 「スキャンタイム」→「安全サイクル時間」

- MC命令はマスタコントロールを開始します。  
- MCR命令はマスタコントロールを終了します。

### 図
#### ラダー図
- MC命令はネスティング(N)とONにするデバイス(d)を入力として受け取り、マスタコントロール回路を形成しています。  
- MCR命令はネスティング(N)のみを入力とします。

#### ST言語
- MC命令: ENO := MC(EN, N, d);  
- MCR命令: ENO := MCR(EN, N);

#### FBD/LD図
- MC命令はEN、N、dを入力とし、ENOを出力するブロックで表現されます。  
- MCR命令はEN、Nを入力とし、ENOを出力するブロックで表現されます。

### 表
#### 実行条件
| 命令 | 実行条件 |
| :--- | :--- |
| MC | 常時実行 |
| MCR | 常時実行 |

#### 設定データ（内容、範囲、データ型）
| オペランド | 内容 | 範囲 | データ型 | データ型（ラベル） |
| :--- | :--- | :--- | :--- | :--- |
| (N) | ネスティング | N0～N14 | デバイス名 | ANY16_S*1 |
| (d) | ONさせるデバイス番号 | ― | ビット | ANY_BOOL |
| EN | 実行条件 | ― | ビット | BOOL |
| ENO | 実行結果 | ― | ビット | BOOL |

*1 デバイス(N)に割り付けたラベル、または定数を割り付けたラベルのみ使用可能。  
SIL2プロセスCPUおよび安全CPUの安全プログラムでは、上記のデータ型の安全デバイスおよび安全ラベルのみ使用可能。
