### テキスト
#### 機能
- 起動接点がONしている時間を1時間単位で計測し、ON時間の累計が指定時間（16ビットデータ）に達した時点で、指定デバイスをONにします。
- 時間設定は1時間単位で行い、現在値は1時間単位または1秒単位で格納されます。
- アラーム出力がONした後も計測は継続されます。
- 最大値に達した場合は計測を停止しますが、負の値設定でも動作可能です。
- 時計データの変更（夏時間機能など）には影響されません。

#### 注意事項
- 計測値が16ビットの最大値に達した後も継続して計測する場合は、デバイス指定に注意が必要で、2要素分をクリアする必要があります。
- タイマ計測が正常に行われないため、特定のプログラムタイプ（初期実行タイプ、割込み、定周期実行、イベント実行）では本命令を使用しないこと。
