### テキスト
#### ルーチング情報命令の読出し
S(P).RTREAD命令は、パラメータで設定したルーチング情報から指定した転送先ネットワークNo.のデータを読み出す命令です。

#### 実行条件
- S.RTREAD命令は立下りエッジで実行されます。
- SP.RTREAD命令は立上りエッジで実行されます。

### 図
- **ラダー図**  
  (s)と(d)の2つのパラメータを持つ命令ブロックが示されています。  
- **FBD/LD図**  
  EN（実行条件入力）、ENO（実行結果出力）、s（転送先ネットワークNo.）、d（読み出し先デバイス）を持つ命令ブロックが示されています。

#### 実行条件
| 命令       | 実行条件           |
|------------|--------------------|
| S.RTREAD   | 立下りエッジ        |
| SP.RTREAD  | 立上りエッジ        |
