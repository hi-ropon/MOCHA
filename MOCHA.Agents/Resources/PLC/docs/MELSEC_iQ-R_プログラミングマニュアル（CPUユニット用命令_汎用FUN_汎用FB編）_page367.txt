### テキスト
#### 機能
- (d)で指定したデバイスから、(n)点分の倍精度実数を右へ1点シフトする。
- 最上位から1倍精度実数分は0になる。
- (n)で指定された値が0のときは無処理となる。

#### エラー
- 演算エラーはありません。

### 図（例）
- (n)=6の場合のデータシフトの動作を示している。
- 上段の数値列は(d)+23～(d)+20から(d)+3～(d)までの倍精度実数の値を示す。
- 下段はシフト後の値で、最上位の(d)+23～(d)+20は0になり、以降の値は1点右にシフトされた値が入る。
- 右端の(d)+3～(d)の値はシフトにより消失していることを矢印と×印で示している。
