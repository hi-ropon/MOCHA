### テキスト
#### 比較演算命令 - BIN16ビットデータ比較
LD□(_U)、AND□(_U)、OR□(_U)命令は、指定した2つのBIN16ビットデータをa接点扱いで比較します。  
対応するCPUはRnCPU、RnENCPU、RnPCPU（プロセス、二重化、一般）、RnPSFCPU（安全、一般）、RnSFCPU（一般、安全）です。  
命令の比較演算子には、EQ（等しい）、NE（等しくない）、GT（より大きい）、LE（以下）、LT（より小さい）、GE（以上）が含まれます。  
バージョン1.035M以降のエンジニアリングツールで対応しています。

#### 実行条件
| 命令                     | 実行条件   |
|--------------------------|------------|
| LD□(_U)、AND□(_U)、OR□(_U) | 常時実行   |

### 図
#### ラダー図
- LD命令、AND命令、OR命令のラダー図が示されている。  
- 各命令は2つの入力(s1, s2)を持ち、a接点として扱われる。  
- 比較演算子は命令の口に入る形で指定される（=(_U), <>(_U), >(_U), <=(_U), <(_U), >=(_U)）。

#### FBD/LD命令ブロック図
- EN（イネーブル）、s1、s2の入力とENO（イネーブル出力）を持つ命令ブロックの図。  
- 比較演算子はLD、AND、ORと、EQ(_U)、NE(_U)、GT(_U)、LE(_U)、LT(_U)、GE(_U)の組み合わせで指定可能。
