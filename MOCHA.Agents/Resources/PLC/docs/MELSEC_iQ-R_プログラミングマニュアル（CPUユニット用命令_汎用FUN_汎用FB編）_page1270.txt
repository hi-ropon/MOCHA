### テキスト
#### 設定データの内容、範囲、データ型
- 「オペランド」「内容」「範囲」「データ型」「データ型（ラベル）」の項目で構成されている。
- 接点として使用するデバイスはビット型で、ラベルはANY_BOOL。

#### 機能
- 指定したブロックが活性状態かどうかをチェックする。
- ブロックの活性/非活性状態により、接点のON/OFFが変わる。
- ブロックNo.はCPUユニットごとに指定範囲が決まっている。
- 範囲外のブロックNo.を指定すると、a接点命令もb接点命令もOFFになる。
- SFCプログラムの有無や状態により、a接点命令とb接点命令のON/OFFが変わる。

#### エラー
- 演算エラーはない。
