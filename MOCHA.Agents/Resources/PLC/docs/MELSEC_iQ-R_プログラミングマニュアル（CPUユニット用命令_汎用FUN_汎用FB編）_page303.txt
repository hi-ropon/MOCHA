<contents>
### テキスト
#### 機能
- (s1)で指定されたデバイスのBIN32ビットデータと、(s2)で指定されたデバイスのBIN32ビットデータのビットごとの論理積（AND）を求め、その結果を(d)で指定されたデバイスに格納する。
- ビットデバイスの場合、桁指定による点数以降のビットデバイスは0として演算する。

#### エラー
- 演算エラーは発生しない。

### 図
- 図は3段構成で、上段が(s1)のビットデータ、中央が(s2)のビットデータ、下段が(d)の結果ビットデータを示している。
- 各ビットはb31からb0までの32ビットで表されており、ビットごとにAND演算が行われている。
- 例えば、(s1)のb31ビットが1、(s2)のb31ビットが0の場合、結果(d)のb31ビットは0となる。
- 図中にビットの並びとAND演算の矢印が示され、ビット単位の論理積の動作を視覚的に理解できる。

</contents>
