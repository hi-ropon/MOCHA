### テキスト
#### 機能
(s1)で指定したBCD4桁データと、(s2)で指定したBCD4桁データの乗算を行い、その乗算結果を(d)で指定したデバイスに格納します。  
乗算結果は(d)+1が上位4桁、(d)が下位4桁となります。  
図では、(s1)に「5678」、(s2)に「0876」（先頭に0が入る）を指定し、乗算結果「0497 3928」が(d)+1と(d)に分けて格納される様子を示しています。

### 表
#### エラー
| エラーコード (SD0) | 内容 |
| :--- | :--- |
| 3405H | (s1)のBCDデータが0～9999以外のとき。<br>(s2)のBCDデータが0～9999以外のとき。 |

エラーコード3405Hは、(s1)または(s2)のBCDデータが有効範囲外（0～9999以外）である場合に発生します。
