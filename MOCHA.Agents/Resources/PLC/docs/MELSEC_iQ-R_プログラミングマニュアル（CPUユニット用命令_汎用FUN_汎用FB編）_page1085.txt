### テキスト
#### 機能
(s)で指定したデバイス番号以降に格納されている秒データを日時に換算し、その換算結果が(d)で指定したデバイス以降に格納されます。起点は2000年1月1日0時0分0秒（0秒）です。

曜日は(d)+6に「日～土」が0～6で格納されます。

夏時間やタイムゾーンによる時刻補正を含む時計データの変更は、本命令の動作に影響しません。例えば夏時間期間中の換算結果と夏時間期間外の換算結果は同じになります。

#### エラー
| エラーコード(SD0) | 内容                     |
|-------------------|--------------------------|
| 3405H             | (s)のデータが範囲外のとき。 |

### 図
- 秒データ(s)が(d)で指定したデバイス以降に日時データとして変換される流れを示す図。
- 具体例として1880337074秒を指定した場合、年：2059、月：8、日：2、時：3、分：51、秒：14、曜日：6と換算される例を示す図。
