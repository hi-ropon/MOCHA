### テキスト
#### BIN16ビットブロックデータ加算命令（BK+(P)(_U)）
指定した2つのBIN16ビットデータブロックを加算する命令です。

##### 実行条件
- BK+、BK+_U命令は立ち上がりエッジで実行されます。
- BK+P、BK+P_U命令はレベル条件で実行されます。

##### 注意事項
- バージョン"1.035M"以降のエンジニアリングツールで対応しています。
- ラベルで設定する場合は、動作に必要な領域が確保できるよう配列を定義し、その配列型ラベルの要素を指定してください。

### 図
#### ラダー図
- ラダー図はs1、s2、d、nの4つのパラメータを入力として示しています。
- ST命令形式の例として、BKPLUS、BKPLUSP、BKPLUS_U、BKPLUSP_Uの呼び出し例が記載されています。

#### FBD/LD図
- EN（実行条件）、s1、s2、nを入力し、dを出力するブロック図が示されています。
- ブロック内にはBKPLUS、BKPLUSP、BKPLUS_U、BKPLUSP_Uが入ることが示されています。
