### テキスト
#### 機能
- (s)で指定された単精度実数を符号なしBIN32ビットデータに変換し、(d)で指定されたデバイスに格納する。
- 変換後のデータは、(s)で指定された単精度実数の小数点以下1桁目を四捨五入した値となる。
- エンジニアリングツールから入力値を設定する場合、丸め誤差が発生することがあるため注意が必要。

#### 注意事項
- 47ページに注意事項が記載されている。
- プログラム例では、M0がONしたときにD0～D1に格納された単精度実数を符号なしBIN32ビットデータに変換し、D100～D101に格納する。

### 図（プログラム例）
- SM402がONのとき、EMOVP命令でE123456.7をD0に格納。
- M0がONのとき、FLT2UDINT命令でD0の単精度実数を符号なしBIN32ビットデータに変換し、D100に格納。
- ビット構成として、(s)側のD1,D0はb31～b16、b15～b0に分かれ、16進数で47F1H、205AH（123456.7）を表す。
- (d)側のD101,D100はb31～b16、b15～b0に分かれ、16進数で0001H、E241H（123457）を表す。

### 表（エラーコード）
| エラーコード(SD0) | 内容 |
|-------------------|------|
| 3401H             | (s)で設定する単精度実数が0～4294967295の範囲外のとき。 |
| 3402H             | (s)に特殊な数を設定したとき。<br>・設定した単精度実数が指定範囲内でない（0、2^-126 ≤ |(s)| < 128）<br>・設定したデバイス、ラベルの内容が0、非正規化数、非数、±∞のとき。 |
