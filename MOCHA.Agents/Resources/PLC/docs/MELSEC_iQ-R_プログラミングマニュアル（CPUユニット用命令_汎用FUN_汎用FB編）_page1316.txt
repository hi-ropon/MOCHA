### テキスト
#### BOOL型→DWORD型変換
関数名：BOOL_TO_DWORD(_E)  
BOOL型データをDWORD型データへ変換する関数です。  
対応CPU：RnCPU、RnENCPU、RnPCPU（プロセス、二重化、一般、安全）など。

#### 機能
- (s)に入力されたBOOL型データをDWORD型データに変換し、(d)から出力します。
- 入力値がFALSEの場合はDWORD型のデータ値で0Hを出力します。
- 入力値がTRUEの場合はDWORD型のデータ値で1Hを出力します。
- (s)への入力値はBOOL型のデータ値です。

#### 演算結果
1. EN/ENOなしファンクション  
   演算処理を実行し、(d)から演算出力値が出力されます。

2. EN/ENO付きファンクション  
   実行条件と演算結果は以下の通りです。

#### エラー
演算エラーはありません。

### 図
- ラダー、FBD/LDの関数ブロック図  
  EN/ENOなしとEN/ENO付きの2種類のブロック図が示されています。  
  入力(s)、出力(d)、EN、ENOの接続が視覚的に表現されています。

- ST言語の使用例  
  EN/ENOなし：d:=BOOL_TO_DWORD(s);  
  EN/ENO付き：d:=BOOL_TO_DWORD_E(EN,ENO,s);

- BOOL型→DWORD型変換イメージ図  
  FALSE → 0H  
  TRUE → 1H  
  と変換される様子が矢印で示されています。

### 表
#### 設定データ（引数、内容、種別、データ型）
| 引数 | 内容 | 種別 | データ型 |
| :--- | :--- | :--- | :--- |
| EN | 実行条件(TRUE:実行、FALSE:停止) | 入力変数 | BOOL |
| s(IN) | 入力 | 入力変数 | BOOL |
| ENO | 出力状態(TRUE:正常、FALSE:異常) | 出力変数 | BOOL |
| d | 出力 | 出力変数 | DWORD |

#### 実行条件と演算結果
| EN | ENO | (d) |
| :--- | :--- | :--- |
| TRUE (演算実行) | TRUE | 演算出力値 |
| FALSE (演算停止) | FALSE*1 | 不定値 |

*1 ENOからFALSEが出力された場合、(d)から出力されるデータは不定値となるため、そのデータを使用しないようプログラムで処理してください。
