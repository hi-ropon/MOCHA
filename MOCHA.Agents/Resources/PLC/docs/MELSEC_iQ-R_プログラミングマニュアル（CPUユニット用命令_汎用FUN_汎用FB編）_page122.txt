### テキスト
#### 内蔵Ethernet機能用命令
Ethernet機能に関連する命令群の説明。

##### オープン/クローズ処理命令
- **コネクションの確立**
  - 命令記号: SP.SOCOPEN
  - 処理内容: (s1)で指定したコネクションのオープン処理を行う。
  - 参照先: 1154ページ SP.SOCOPEN

- **コネクションの切断**
  - 命令記号: SP.SOCCLOSE
  - 処理内容: (s1)で指定したコネクションに対するクローズ処理を行う（コネクションの切断）。
  - 参照先: 1157ページ SP.SOCCLOSE

##### ソケット通信用命令
- **受信データのEND処理時読出し**
  - 命令記号: SP.SOCRCV
  - 処理内容: END処理時に(s1)で指定したコネクションの受信データをソケット通信受信データエリアから読み出す。
  - 参照先: 1159ページ SP.SOCRCV

- **受信データの命令実行時読出し**
  - 命令記号: S.SOCRCVS
  - 処理内容: 命令実行時に(s)で指定したコネクションの受信データをソケット通信受信データエリアから読み出す。
  - 参照先: 1162ページ S.SOCRCVS

- **データ送信**
  - 命令記号: SP.SOCSND
  - 処理内容: (s1)で指定したコネクションの相手機器へ、(s3)で設定したデータを送信する。
  - 参照先: 1164ページ SP.SOCSND

- **コネクション情報の読出し**
  - 命令記号: SP.SOCCINF
  - 処理内容: (s1)で指定したコネクションのコネクション情報を読み出す。
  - 参照先: 1167ページ SP.SOCCINF

- **コネクションの交信先変更(UDP/IP)**
  - 命令記号: SP.SOCCSET
  - 処理内容: (s1)で指定したコネクションの交信相手IPアドレス、交信相手ポート番号を変更する（UDP/IP通信時のみ）。
  - 参照先: 1169ページ SP.SOCCSET

- **コネクションの受信モード変更**
  - 命令記号: SP.SOCRMODE
  - 処理内容: (s1)で指定したコネクションに対して、TCP受信モードおよび受信データサイズを変更する。
  - 参照先: 1171ページ SP.SOCRMODE

- **ソケット通信受信データリード**
  - 命令記号: S.SOCRDATA / SP.SOCRDATA
  - 処理内容: (s1)で指定したコネクションのソケット通信受信データエリアから、(n)で指定したワード分のデータを読み出し、(d)で指定されたデバイス以降に格納する。
  - 参照先: 1175ページ S(P).SOCRDATA
