### テキスト
#### 19.2 タイミング計測命令 タイミングパルス発生
ユーザ用タイミングクロックを指定したスキャン分ONにし、指定したスキャン分OFFにする命令「DUTY」について説明しています。

#### 命令概要
- DUTY命令はユーザ用タイミングクロックを指定し、スキャンのON時間とOFF時間を制御します。

### 図
#### ラダー図
- DUTY命令のラダー図は、(n1)、(n2)、(d)の3つのパラメータを持つ命令ブロックとして示されています。
- ST言語では `ENO:=DUTY(EN,n1,n2,d);` と記述されます。

#### FBD/LD図
- 入力EN、n1、n2があり、出力ENOとdがあるブロック図で表現されています。

| オペランド | 内容 | 範囲 | データ型 | データ型(ラベル) |
| :--- | :--- | :--- | :--- | :--- |

| (d) | 動作させるユーザ用タイミングクロック特殊リレーデバイス番号 | SM420～SM424 | ビット | ANY_BOOL *1 |
