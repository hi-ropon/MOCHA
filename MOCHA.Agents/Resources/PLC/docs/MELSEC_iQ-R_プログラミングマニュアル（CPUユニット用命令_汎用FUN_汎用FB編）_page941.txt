### テキスト
#### 単精度実数角度→ラジアン変換（RAD(P)）
角度の大きさの単位を、単精度実数で指定した度単位からラジアン単位に変換する命令です。

#### 実行条件
- RAD命令は立ち上がりエッジで実行されます。
- RADP命令は立ち下がりエッジで実行されます。

### 図
- **ラダー図**  
  ラダー図では、(s)に角度の値を入力し、(d)にラジアン変換後の値を出力します。  
  ST言語では、`ENO:=RAD(EN,s,d);` または `ENO:=RADP(EN,s,d);` と記述します。

- **FBD/LD図**  
  ブロック図では、EN（実行条件）、s（入力角度）、d（出力ラジアン）、ENO（実行結果）端子が示されています。
