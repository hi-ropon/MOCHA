### テキスト
#### 機能
(d)で指定したデバイスから(n1)ワード分のデータ範囲内で、左へ(n2)ワード分シフトさせます。シフト後、(s)から(n2)点分を空きデータ部分に格納します。
- (s)に定数を指定した場合、シフト後の最下位から(n2)点分は指定した値になります。
- (n2)で指定した値が00の場合は無処理となります。

#### 注意事項
(s)と(d)をビットの桁指定で指定する場合、(s)と(d)の桁数を合わせる必要があります。

### 図
#### データシフトの例（(n1)=9、(n2)=4の場合）
- (d)から始まる9ワードのデータが左へ4ワード分シフトされる様子を示しています。
- シフト前のデータは「1FFH、10H、0H、7FFH、3AH、1FH、30H、0H、FFH」。
- 左へ4ワードシフト後、左端の4ワード分は捨てられ、右端4ワード分は空きデータ部分にコピーされます。
- 空き部分には(s)の値が入るか、指定があればその値が入ります。

| 3405H              | (n1)、(n2)に指定した値が(n1) < (n2)のとき。 |
