### テキスト
#### 機能
- (d)で指定したデバイスから(n1)ビット分のデータ範囲内で、左へ(n2)ビット分シフトする。
- シフト後、(s)から(n2)点分を空きデータ部分に格納する。
- (s)に定数0を指定した場合、シフト後の最下位から(n2)点分は0になる。
- (s)に定数1を指定した場合、シフト後の最下位から(n2)点分は1になる。
- (n2)で指定した値が0の場合は無処理となる。

#### 例
- (n1)=10、(n2)=4の場合の動作を示す。

### 図
- ビットシフトの動作を示す図解。
- (d)から(n1)ビット分のデータが左へ(n2)ビット分シフトされる様子をビット単位で示している。
- シフト後の空き部分には(s)からのデータがコピーされる。
- 図中に「① (n2)ビット分、左へシフト」「② コピー」の動作説明がある。
