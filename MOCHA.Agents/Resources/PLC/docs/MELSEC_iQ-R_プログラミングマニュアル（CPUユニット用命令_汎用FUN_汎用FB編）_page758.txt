### テキスト
#### BIN32ビット平方根算出 DSQRT(P)
- R00CPU、R01CPU、R02CPUはバージョンによる制約なし。
- RnCPU（R00CPU、R01CPU、R02CPUを除く）、RnENCPUはファームウェアバージョン17以降で使用可能。エンジニアリングツールはバージョン1.020W以降対応。
- 指定したBIN32ビットデータの平方根を算出する命令。

#### 機能
- (s)で指定したBIN32ビットデータの平方根を計算し、結果を(d)に格納。
- 計算結果は小数点以下切り捨ての整数。
- 計算式：√(s)+1、(s)→(d)

#### エラー
- 演算エラーはなし。

### 図
- **ラダー図**：ラダー命令の形状が示されており、(s)と(d)のオペランドがある。
- **STコード**：
  ```
  ENO:=DSQRT(EN,s,d);
  ENO:=DSQRTP(EN,s,d);
  ```
- **FBD/LD図**：ファンクションブロックの形状で、EN（実行条件）、ENO（実行結果）、s（入力データ）、d（出力データ）の端子がある。
