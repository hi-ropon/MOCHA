### テキスト
#### 機能
- (s)で指定されたデバイスのBIN32ビットデータと同一のデータを、(d)で指定されたデバイスへ(n)点分転送する。
- (s)に桁指定がある場合、転送されるデータは桁指定分のみとなる。
  - 例として、(s)にK5Y0が指定された場合は、(s)のワードデバイスの下位20ビット（5桁分）が対象。
- (d)に桁指定がある場合、(d)で指定された桁指定分のデータを転送する。
  - (d)にK5Y0が指定された場合は、(s)のデバイスの下位20ビットが対象。
- (s)、(d)の両方で桁指定がある場合も、桁数に関係なく(d)で指定された桁指定分のデータを転送する。
- (n)で指定された値が0の場合は無処理となる。

#### エラー
- 演算エラーはありません。

### 図
1. **BIN32ビットデータの転送イメージ**
   - (s)で指定されたデバイスのビットb31からb0までの32ビットデータ（例: 1234567H）を、(d)で指定されたデバイスの連続したアドレスに順次転送する様子を示す。
   - 転送先は(d)+1、(d)+3、(d)+5…と2ずつ増加し、(n)点分繰り返す。

   - (s)に桁指定がある場合、転送されるのは桁指定分のみ。
   - 例として、(s)にK5Y0が指定された場合は、ワードデバイスの下位20ビット（5桁分）が対象。

3. **(d)に桁指定がある場合の転送**
   - (d)に桁指定がある場合、(d)で指定された桁指定分のデータを転送。
