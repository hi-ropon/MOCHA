### テキスト
#### 注意事項
- 動作中はオペランドやデータの内容を変更しないこと。並び替え完了時に正しい順序にならない場合がある。
- (s)と(d)に同一デバイスを指定した場合、並び替え元のデータは並び替え後のデータ順に書き換えられるため、実行完了（SD774のビット0がONするまで）(s)の内容を変更しないこと。
- (s)と(d)を指定する際は、同一デバイス範囲を指定するか、範囲が重複しないデバイスを指定すること。

### 図
3つの図は元データと並び替え後のデータの範囲関係を示している。
- 左図：元データD10～D30、並び替え後のデータもD10～D30で範囲が同じ → ○（正しい）
- 中図：元データD10～D30、並び替え後のデータがD35～D55で範囲が重ならない → ○（正しい）
- 右図：元データD10～D30、並び替え後のデータがD2～D22で範囲が重複 → ×（誤り）
