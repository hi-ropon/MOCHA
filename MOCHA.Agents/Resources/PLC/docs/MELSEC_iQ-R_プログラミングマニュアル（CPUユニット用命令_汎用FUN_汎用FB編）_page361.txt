### テキスト
#### 機能
- 指定したデバイス(d)から、(n)ダブルワード分を左へ1ダブルワードシフトする。
- 最下位から1ダブルワード分は0になる。
- (n)で指定された値が0のときは無処理となる。

#### エラー
- 演算エラーはありません。

### 図
- 図は(n)=6の場合のデータシフトの例を示している。
- 上段に元のデバイス値が並び、下段にシフト後の値が示されている。
- シフト前は(d)+11から(d)までの7つのデバイスにそれぞれ数値が入っている。
- シフト後は左に1ダブルワード分ずれており、最下位の(d)は0になっている。
- 具体的には、元の(d)+11の値555555は消え、(d)+10の212121が(d)+11に移動し、以降同様にシフトされている。
