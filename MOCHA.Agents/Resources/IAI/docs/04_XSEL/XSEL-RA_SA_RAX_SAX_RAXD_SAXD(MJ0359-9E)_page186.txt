### テキスト
#### 3.9.2 汎用RS-232Cポートの配線
SEL言語から制御でき、さまざまな上位コントローラーに合わせて伝文フォーマットを設定可能な1チャンネルの通信ポートです。  
詳細は「別冊 SEL言語プログラミングマニュアル（管理番号：MJ0224）」を参照してください。

### 図
```mermaid
flowchart LR
    A[上位コントローラー] -- RS232C --> B[汎用RS232Cポートコネクター]
```
**図の説明:**  
上位コントローラーと汎用RS-232CポートコネクターがRS232Cケーブルで接続されているイメージ図。

| 項目               | 内容                             | 備考                         |
|--------------------|--------------------------------|------------------------------|
| 使用コネクター     | D-sub9ピン（DTE）               | コントローラー側はメスコネクター |
| 型番               | XM2C-0942-502L（OMRON）         |                              |
| コネクター名称     | S1                             |                              |

| チャンネル数       | 1                              |                              |
