### テキスト
#### (19) 汎用RS-232Cポートコネクター
外部のRS-232C機器との接続用ポートです。
詳細は[3.9.1項]参照。

#### (20) フィールドネットワークボード（オプション）装着位置2
CC-Link、DeviceNet、PROFIBUS-DP、またはCC-Link IE Field用のフィールドネットワークボード（オプション）が装着されます。
詳細は別冊の各フィールドネットワークの取扱説明書を参照。

#### (21) ティーチングコネクター
ティーチングボックス、またはパソコン（パソコン対応ソフト）を接続し、アクチュエーターの操作および設定などを行うティーチングツール用のコネクターです。
詳細は[3.10.1項]参照。

#### (22) ブレーキ解除スイッチ
ブレーキ付アクチュエーターのブレーキを強制的に解除（励磁開放）するためのスイッチです。
装置の立ち上げ時、または表示や異常時にアクチュエーターを手で動かしたい場合に、スイッチをRLS側にするとブレーキを強制的に解除できます。
必要のないときはスイッチをNOM側にしてください。
（注）IX-N□□1205/1505/1805を除くスカラロボットのブレーキ軸には本スイッチは搭載されません。

#### 表1.4-9 モードスイッチの機能

| スイッチ位置               | 機能                                                                                   |
|----------------------------|----------------------------------------------------------------------------------------|
| MANU（手動モード） 左側    | ブレーキが強制的に解除されます。                                                       |
| AUTO（自動モード） 右側    | ブレーキはコントローラーにより自動的に制御されます。                                   |
|                            | ・サーボON ：ブレーキ解除                                                             |
|                            | ・サーボOFF：ブレーキ有効                                                             |
