### テキスト
#### 3.2.3 電源回路の配線（XSEL-SAX4・三相電源高容量仕様）
三相電源高容量仕様のXSELコントローラーには、以下の電源を供給してください。

コントローラーの消費電力と発熱量はドライバーボードなどの構成により異なるため、消費電力を計算の上、サーキットブレーカーを選定してください。詳細は「2.2.4 サーキットブレーカーの選定」を参照してください。

注1：電源はお客様にて用意してください。

### 表
#### 電源種類と仕様

| 電源種類     | 仕様                   | 備考                          |
| :----------- | :--------------------- | :---------------------------- |
| モーター電源 | 三相 AC200～230V±10%   |                               |
| 制御電源     | 単相 AC200～230V±10%   |                               |
| I/O電源      | DC24V±10%              | I/OスロットにPIOボードを搭載する場合 |
| ブレーキ電源 | DC24V±10%              | ブレーキ付きの場合             |

```mermaid

    ノイズフィルター --> クランプフィルター3[(5) クランプフィルター]
    サーキットブレーカー --> サージプロテクター[(6) サージプロテクター]
    サージプロテクター --> D種接地[D種接地<br>(旧第3種設置：接地抵抗100Ω以下)]

    subgraph XSELコントローラー端子

    end

    XSELコントローラー端子 --> PIOボード[PIOボード（オプション）]

    BK_PWR[ブレーキ電源入力コネクター] --> ブレーキ電源1[DC24V電源 + 24V / 0V]
    BK_PWR --> ブレーキ電源2[DC24V電源 + 24V / 0V]

    ブレーキ電源1 --> BK_PWR
    ブレーキ電源2 --> PIOボード
```

**図の説明:**  
- 3相200Vの電源が漏電ブレーカー、サーキットブレーカーを経由し、ノイズフィルター、リングコア、複数のクランプフィルター、サージプロテクターを通ってXSELコントローラーのAC電源入力コネクターに接続される。  
- XSELコントローラーの端子からは制御電源、モーター電源、PE端子に分かれ、オプションのPIOボードにも接続される。  
- ブレーキ電源はDC24V±10%で、ブレーキ電源入力コネクターを介して供給される。  
- D種接地は接地抵抗100Ω以下で設置する必要がある。
