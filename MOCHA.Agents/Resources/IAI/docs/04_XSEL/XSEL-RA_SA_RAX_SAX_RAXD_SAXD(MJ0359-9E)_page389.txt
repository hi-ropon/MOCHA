### テキスト
#### 6.5.20 軸が移動中であることを表す信号を出力したい
軸が移動指令（MOVP命令など）によって移動中であることを表す信号を出力できます。  
「移動中」とは以下の条件をすべて満たす状態を指します。  
- サーボON状態  
- サーボ軸使用中  
- 移動指令未完了  
- 一時停止中でない  

（※）本機能はメインアプリ部V1.40以降でサポート。  
（※）一つのSEL命令で複数軸を動作させる場合、全軸の移動指令が完了するか動作が打切られるまで移動中とみなす。

### 表
#### 状態と出力ポート設定
| 状態   | 出力ポート：軸別パラメーターNo.238に設定 |
| :----- | :--------------------------------------- |
| 移動中 | ON                                       |
| 停止中 | OFF                                      |

#### パラメーター設定
| パラメーターNo.       | 設定値         | 内容           |
| :-------------------- | :------------- | :------------- |
| 軸別パラメーターNo.238 | 出力ポートNo.  |                |
