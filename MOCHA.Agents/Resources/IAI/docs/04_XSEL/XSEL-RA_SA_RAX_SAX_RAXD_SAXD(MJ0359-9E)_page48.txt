### テキスト
#### I/Oスロット1および2のPIOボード装着
- I/Oスロット1および2にはPIOボード（オプション）が装着可能。
- それぞれのスロットに装着可能なPIOボードの種類と仕様は表1.4-5、表1.4-6に示されている。

#### システム動作ステータスLED2
- メインCPUおよびネットワークインターフェイス2の動作状態を示すLED。
- 状態表示はフィールドバスによって異なるため、詳細は各フィールドバス取扱説明書を参照。

### 表
#### 表1.4-5 装着可能PIOボード（I/Oスロット1）
| 名称           | XSEL型式表示 | 入力点数 | 出力点数 | 極性 |
| :------------- | :----------- | :------- | :------- | :--- |
| PIOボード      | N1           | 32       | 16       | NPN  |
|                | P1           | 32       | 16       | PNP  |
|                | N2           | 16       | 32       | NPN  |
|                | P2           | 16       | 32       | PNP  |
| 多点PIOボード  | N3           | 48       | 48       | NPN  |
|                | P3           | 48       | 48       | PNP  |

#### 表1.4-7 システム動作ステータスLED2の状態表示
| 項目          | 内容                                                                                     |
| :------------ | :--------------------------------------------------------------------------------------- |
| －            | ネットワーク2の状態表示。フィールドバスによって表示内容が異なる。各フィールドバス取扱説明書参照。 |
| CPU STATUS2   | モーション制御マスターボードの通信異常時点灯（橙）                                       |
| CPU STATUS1   | メインCPU（プロセッサーユニット部）正常起動時点灯（緑）<br>メインCPU（プロセッサーユニット部）動作異常時（消灯または橙点灯） |
| CPU STATUS0   | メインCPU（デバイス）正常起動時点灯（緑）<br>電源投入後のメインCPUの初期処理中（消灯または橙点灯） |
