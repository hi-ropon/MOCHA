### テキスト
#### 5.2.3 インターフェイス仕様
コントローラー部とビジョンシステムとの接続は、パラレル通信とシリアル通信の両方を使用して接続します。

### 表
| 項目       | 用途                                   | 備考                                           |
|------------|--------------------------------------|------------------------------------------------|
| パラレル   | PIO入力1点                           | ビジョンシステムの初期化完了ステータス入力（必要に応じて使用）<br>ビジョンシステムとのハンドシェイク用（PIOボード選択が必要） |
|            | PIO出力1点                          | ビジョンシステムへの撮像指令出力（必須）       |
| シリアル   | Ethernet（TCP）                     | 撮像ワークデータの受信（必須）                  | Ethernetに標準対応（標準搭載のEthernetポートを使用）<br>TCP/IP通信パラメーターNo.要変更 |
|            | RS-232C                            | 撮像ワークデータの受信（必須）                  | 汎用SIOチャンネル数変更（2チャンネル→1チャンネル） |

### 図
コントローラー背面のインターフェイス部の写真に以下の注釈が付されています。

- 標準 Ethernet
- 汎用 SIO チャンネル 1
- PIO ボード
