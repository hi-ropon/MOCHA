### テキスト
#### 7.1 トラブル発生時の処理
迅速な復旧と再発防止のため、トラブル発生時には以下の点を確認してください。

1. コントローラーの7セグメント表示器とLEDの確認  
　・7セグメント表示器の表示内容は表7.1-2「パネルウィンドウの表示一覧」を参照  
　・LEDの状態を確認

2. 上位コントローラー（PLCなど）のアラームの有無確認

3. 制御電源・モーター電源・フィールドネットワークやPIOの電源電圧の確認  
　瞬時停電・電圧降下・電源異常が発生していないか確認

4. ブレーキ電源の電圧確認（ブレーキ付アクチュエーターの場合）

5. アラームの詳細確認  
　ティーチングツールでアラーム情報を確認

6. コネクター類の脱落または不完全接続の確認

### 表
#### 表7.1-1 コントローラーのLEDの表示内容

| 番号 | 名称   | LED点灯時の状態                                    |
| :--- | :----- | :------------------------------------------------ |
| (1)  | RDY    | CPUレディー（プログラム運転可能）                  |
| (2)  | ALM    | CPUアラーム（メッセージレベル以上エラー）、CPUハード異常 |
| (3)  | EMG    | 非常停止状態、CPUハード異常、電源系ハード異常      |
| (4)  | PSE    | 電源系ハード異常                                  |
| (5)  | CLK    | システムクロック異常、CPUハード異常                |
| (6)  | CP_ON  | 制御電源正常                                      |
