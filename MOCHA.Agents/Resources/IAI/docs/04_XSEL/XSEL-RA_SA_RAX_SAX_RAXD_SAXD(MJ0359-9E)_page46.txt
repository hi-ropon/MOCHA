### テキスト
#### (9) システム動作ステータスLED1
システム動作状態（モーション制御マスター）およびネットワークインターフェイス1の動作状態を示すLEDです。

#### (10) 拡張モーション制御マスターコネクター
SCON-CB、SCON-CA、PCON-CB、ACON-CB、DCON-CB、MCON-CのMECHATROLINK-Ⅲ対応コントローラーをモーション制御する場合に接続するコネクターです。  
MECHATROLINK-Ⅲのケーブルで接続し、最大32軸のスレーブ軸としてSELプログラムで動作可能です。  
[別冊の拡張モーション制御取扱説明書（管理番号：MJ0364）]参照。

#### (11) USBコネクター
パソコン（専用ティーチングソフト）を接続するUSBコネクターです。Mini-Bタイプ。  
パソコン対応ソフトを使用し、アクチュエーターの操作や設定が可能。  
パソコン対応ソフト使用時は、（21）ティーチングコネクターにダミープラグDP-2を挿入してください。

#### (12) パネルウィンドウ
4桁の7セグメント表示器と6個のLEDでコントローラーの状態を表示します。  
[4.3.2項または7.1節]参照。

#### 表1.4-2 システム動作ステータスLED1の状態表示

| 項目 | 説明 |
| :--- | :--- |
| - | ネットワーク1の状態表示。フィールドバスにより表示内容が異なる。各フィールドバス取扱説明書参照。 |
| LK | モーション制御マスターボードのリンク確立時点灯（緑） |
| ER | モーション制御マスターボードの通信異常時点灯（橙） |
| SD | 未使用 |
| RUN | モーション制御マスター機能有効時点灯（緑） |
