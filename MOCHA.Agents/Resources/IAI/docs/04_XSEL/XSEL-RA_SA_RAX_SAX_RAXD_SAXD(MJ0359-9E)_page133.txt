### テキスト
#### 3.2.4 電源回路の配線（単相電源仕様）
単相電源仕様のXSELコントローラーには、以下の電源を供給してください。

コントローラーの消費電力と発熱量は、ドライバーボードなどの構成により異なります。  
消費電力を計算の上、サーキットブレーカーを選定してください。  
詳細は「2.2.4 サーキットブレーカーの選定」を参照してください。

注1 お客様にて用意をしてください。

### 表
#### 電源種類と仕様

| 電源種類     | 仕様                     | 備考                         |
| :----------- | :----------------------- | :--------------------------- |
| モーター電源 | 三相 AC200～230V±10%    |                              |
| 制御電源     | 単相 AC200～230V±10%    |                              |
| I/O電源      | DC24V±10%                | I/OスロットにPIOボードを搭載する場合 |
| ブレーキ電源 | DC24V±10%                | ブレーキ付きの場合           |

### 図
#### 単相電源の配線例

```mermaid
flowchart LR
    A[単相200V] --> B[漏電ブレーカー]
    B --> C[サーキットブレーカー]
    C --> D1[ノイズフィルター (1)]
    D1 --> D2[リングコア (2)]
    D2 --> D3[クランプフィルター (3)]
    D2 --> D4[クランプフィルター (4)]
    D3 & D4 --> E[XSELコントローラー AC電源入力コネクター]
    E --> F[制御電源 (A端子)]
    E --> G[モーター電源 (B端子)]
    C --> H[サージプロテクター (5)]
    E --> I[ブレーキ電源入力コネクター]
    I --> J[DC24V電源]
    E --> K[PIOボード (オプション)]
    J --> L[DC24V電源]
    subgraph 接地
        M[D種接地 (旧第3種設置:接地抵抗100Ω以下)]
    end
    D2 --> M
```

**図の説明:**  
- 単相200Vの電源が漏電ブレーカー、サーキットブレーカーを経由し、ノイズフィルター、リングコア、クランプフィルターを通ってXSELコントローラーのAC電源入力に接続される。  
- ブレーキ電源はDC24V電源から供給され、ブレーキ電源入力コネクターに接続される。  
- I/OスロットにPIOボードを搭載する場合は、別途DC24V電源が供給される。  
- 接地はD種接地（接地抵抗100Ω以下）を推奨。
