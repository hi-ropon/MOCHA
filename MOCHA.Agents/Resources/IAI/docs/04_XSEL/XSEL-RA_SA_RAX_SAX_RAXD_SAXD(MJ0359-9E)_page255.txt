### テキスト
#### IXAスカラロボットのコンプライアンス制御機能
- SCLO 0（スカラ専用命令-コンプライアンスモードオプション機能設定（探索動作設定））について説明。
- コンプライアンスモード（コンプライアンス制御）の探索動作設定を行い、操作2で指定した変数（連続した2個）に格納したデータを探索動作条件として設定する。

#### 機能説明
- 操作2で指定する整数変数内容は、探索動作有効軸パターンと探索動作無効Z軸押付け動作エリアの2つの値で構成される。
- 探索動作無効Z軸押付け動作エリアは、指定区間で探索動作を無効化する機能であり、+Zb方向へのピン挿入時に一定の挿入量を超えれば探索動作を省略可能。

### 表
#### SCLO 0 命令設定（拡張条件・入力条件・命令・出力部）

| 拡張条件 (LD,A,O,AB,OB) | 入力条件 (入出力・フラグ) | 命令・宣言 | 操作1 | 操作2 | 出力部 (出力・フラグ) |
| :---------------------- | :----------------------- | :--------- | :---- | :---- | :-------------------- |

#### 対応機種
- XSEL-RAX/SAX/RAXD/SAXD（メインアプリ部 V1.14以降）

#### 操作2 指定整数変数内容

| 設定値 | 内容                                                                                          |
| :----- | :-------------------------------------------------------------------------------------------- |
| n      | 探索動作有効軸パターン<br>指定した関節軸単位で探索動作を行う。第1軸、または第2軸（2台目は第5軸、第6軸）が指定可能。 |
| n+1    | 探索動作無効Z軸押付け動作エリア（0.001mm単位）<br>Z軸押付け動作（PUSH命令）目標ポジションから手前の距離を指定。<br>指定距離のエリアに到達すると探索動作を無効化。0以下は無効。 |

```mermaid
    A[ロボットハンド] --> B[ピン]
    B --> C[探索不要位置]
    C --> D[PUSH命令目標位置]
    D --> E[探索動作無効 Z軸押付け動作エリア]
    E --> F[+Zb方向]
```

**図の説明:**  
ロボットハンドがピンを持ち、+Zb方向にピンを挿入する動作を示す。  
PUSH命令の目標位置から手前の範囲が「探索動作無効Z軸押付け動作エリア」として設定され、この範囲内では探索動作が無効化される。  
これにより、ピンがある程度挿入されると以降の探索動作が不要となる。
