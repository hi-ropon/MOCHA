### テキスト
#### 6.5.4 非常用プログラムを起動したい
非常停止の入力やセーフティーゲートオープン時に非常用プログラムを動作させるには、非常用プログラムNo.と使用する出力ポートの範囲を設定する必要があります。動作可能なプログラムはアクチュエーターの動作を伴わないI/O処理のプログラムのみです。

また、非常用プログラムは「その他パラメーターNo.5 “動作・プログラム打切時I/O処理プログラム起動種類”」の設定により起動条件が決まります。

#### 6.5.5 非常停止が解除されたら、自動で復旧（再起動）したい
非常停止解除時に自動でエラーリセットを行い、プログラムを実行させることが可能です。

### 表
#### 非常用プログラム起動に関するパラメーター設定

| パラメーターNo.           | 設定値               | 内容                         |
| :------------------------ | :------------------- | :--------------------------- |
| その他パラメーター No.2    | 非常用プログラムNo.  |                              |
| I/Oパラメーター No.70      | 下限出力ポートNo.    | 使用する出力ポートの下限側を設定 |
| I/Oパラメーター No.71      | 上限出力ポートNo.    | 使用する出力ポートの上限側を設定 |

#### その他パラメーターNo.5（動作・プログラム打切時I/O処理プログラム起動種類）

| パラメーターNo.           | 内容                                                                                   |
| :------------------------ | :------------------------------------------------------------------------------------- |
| その他パラメーター No.5    | 0：プログラム実行中、全動作解除要因発生時<br>1：全動作解除要因発生時（プログラム実行の有無と無関係に起動）<br>2：プログラム実行中、動作レベル以上のエラー発生時に、全動作解除要因発生時<br>3：動作レベル以上のエラー発生時に、全動作解除要因発生時（プログラム実行の有無と無関係に起動） |

#### 非常停止解除後の自動復旧に関するパラメーター設定

| パラメーターNo.           | 設定値           | 内容                                         |
| :------------------------ | :--------------- | :------------------------------------------- |
| その他パラメーター No.1    | 実行プログラムNo.|                                              |
| その他パラメーター No.10   | 3                | 非常停止解除で再起動（ソフトウェアリセット）を行う |
| I/Oパラメーター No.33      | 1                | AUTOモード時、再起動後、プログラムを実行する       |
