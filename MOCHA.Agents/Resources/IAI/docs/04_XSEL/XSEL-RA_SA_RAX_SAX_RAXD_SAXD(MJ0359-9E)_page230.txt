### テキスト
#### 5.3 シンクロ機能

2) 以下の計算を行う手順が示されています。

スレーブ軸の軸別パラメーターNo.12「原点プリセット値（0.001mm）」に対して、以下の計算式を適用します。

```
原点プリセット値（0.001mm）
+ ((マスター軸現在位置表示値[mm] - スレーブ軸現在位置表示値[mm]) × 1000)
```

本例では、

```
0 + ((-0.002 - 1.184) × 1000) = -1,186
```

となります。

3) 計算結果をスレーブ軸の軸別パラメーターNo.12「原点プリセット値」に入力し、以下の操作を行います。

- 「コントローラーへ転送」
- 「フラッシュROM書込み」
- 「コントローラー再起動（ソフトウェアリセット）」

これにより設定が反映されます。
