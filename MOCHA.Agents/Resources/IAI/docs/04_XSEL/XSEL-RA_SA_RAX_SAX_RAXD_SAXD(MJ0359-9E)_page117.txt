### テキスト
#### 3.1.2 XSEL-SA

##### 注釈
1. お客様で用意してください。
2. ブレーキ付きアクチュエーターは、コントローラーにブレーキ用電源+24Vの供給が必要です。
3. パソコンなどのティーチングツールをティーチングコネクターに接続しない場合は、ダミープラグ（DP-2）を接続してください。
4. 安全カテゴリーの対応は、「9.1 SA/SAX/SAXDタイプの安全回路例（安全カテゴリー対応）」を確認してください。

##### 警告
- XSEL-SAにパソコンを接続する場合、パソコン対応ソフトIA-101-X-MWに付属の標準パソコンケーブルCB-ST-E1MW050（-EB）：黒色は使用できません。
- パソコン対応ソフトは使用できますが、ケーブルはCB-ST-A1MW050（-EB）：灰色またはCB-ST-A2MW050（-EB）：黒色・黄帯付を使用してください。

##### 注意
- ICSA、ICSPA（直交ロボット）の場合は、ケーブルに番号が表示されています。コントローラーのコネクター番号に合わせて接続してください。
- 単軸扱いで納入されたアクチュエーターは、コネクター番号が表示されていません。コネクター番号などを表示して間違いのないようにしてください。
- 接続を間違えると、モーター／基板の損傷、誤作動の原因になります。

### 図
```mermaid
flowchart LR
    電源[電源 三相または単相AC200V] --> 駆動源遮断回路
    駆動源遮断回路 --> 回生抵抗ユニット(オプション)
    駆動源遮断回路 --> XSEL-SA本体
    XSEL-SA本体 --> ブレーキ用電源[ブレーキ用電源 24V電源]
    XSEL-SA本体 --> 安全回路
    XSEL-SA本体 --> フラットケーブル --> 上位システムPLC
    XSEL-SA本体 --> ティーチングボックス(オプション)
    XSEL-SA本体 --> ダミープラグ(DP-2)
    XSEL-SA本体 --> モーターケーブル/エンコーダーケーブル --> モーター
    ティーチングボックス --> パソコン(パソコン対応ソフト オプション)
```

**図の説明:**
- 電源は三相または単相AC200Vで、駆動源遮断回路を経由し、回生抵抗ユニット（オプション）やXSEL-SA本体に供給される。
- XSEL-SA本体はブレーキ用電源（24V）と安全回路を内蔵。
- 本体からはフラットケーブルで上位システムPLCに接続。
- ティーチングボックス（オプション）やパソコン対応ソフト（オプション）も接続可能。
- ティーチングツールを接続しない場合はダミープラグ（DP-2）を接続する。
- モーターケーブルとエンコーダーケーブルはモーターに接続される。
