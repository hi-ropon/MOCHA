### テキスト
#### 4.4 プログラム運転
プログラム運転には、設定されたプログラムNo.の自動起動と外部からプログラムNo.を指定して起動する二つの方法がある。

#### 4.4.1 電源投入による自動起動
電源投入後、XSELコントローラーに登録したプログラムNo.を自動起動できる。手順は以下の通り。

1. パソコン対応ソフトを接続し、I/OパラメーターNo.33を「1」に設定し、その他パラメーターNo.1に「起動するプログラムNo.」を設定する。

2. XSELコントローラーのモードスイッチをAUTO側に設定する。

3. 電源を再投入する。

4. XSELコントローラーが再起動後（正常起動＝RDY信号ON後）、設定したプログラムが起動する。

### 表
#### 表4.4-1 I/OパラメーターNo.33の設定値と機能

| パラメーター種類、およびNo. | 設定値 | 内容 |
| :---------------------------- | :----- | :--- |
| I/OパラメーターNo.33          | 0      | 入力ポート003を汎用入力にする。 |
|                               | 1 (出荷時) | AUTOモード時、電源投入またはソフトウェアリセットを行うと、その他パラメーターNo.1に設定した番号のプログラムを実行する。入力ポート003を汎用入力として使用可能。 |
|                               | 2      | 入力ポート003をONすると、その他パラメーターNo.1に設定した番号のプログラムを実行。OFFすると実行中の全プログラムを停止。 |
| その他のパラメーターNo.1      | 1～255 | 自動実行させるプログラム番号を設定。 |
