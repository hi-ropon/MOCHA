### テキスト
#### 3.3 非常停止回路（システムI/O）の配線

- DETIN、DETOUT信号かAUTO1IN、AUTO1OUT信号のいずれかをパラメーターで選択可能。  
  - IOパラメーターNo.24 bit24-27: DET_RAUTO1機能選択(0:DET、1:RAUTO1)で設定。  
  - 出荷時はDETIN、DETOUT信号設定。  
  - XSEL-RA/RAX/RAXDコントローラーではDETIN、DETOUT信号は未使用信号。

- AUTO1IN、AUTO1OUT信号およびAUTO2IN、AUTO2OUT信号  
  - モードスイッチがAUTOの時、AUTO1IN、AUTO1OUT信号間およびAUTO2IN、AUTO2OUT信号間に外部スイッチを接続し、外部スイッチでAUTO/MANUモードの切替が可能。  
  - スイッチで短絡するとAUTOモード、開放するとMANUモードになる。  
  - 外部スイッチ未接続時は9ピンと10ピン間、19ピンと20ピン間を短絡する。

- 各信号の説明  
  1. EMGin (IN) / EMGin (+24V)間が切断されると、アクチュエーターの減速停止、サーボOFF動作を実施し、モーター駆動用電源が切断される。  
  2. EMG1 (line+) / (line-) は非常停止の接点出力。  
  3. SDN (OUT+) / SDN (OUT-) 接点出力は、コントローラーが外部からのモーター電源供給を禁止している場合のブレーキ状態の接点。  
  4. ENBin (IN) / ENBin (+24V)間が切断されると、アクチュエーターの減速停止、サーボOFF動作を実施し、モーター駆動用電源が切断される。  
  5. ENB1 (line+) / (line-) はイネーブル信号の接点出力。  
  6. RDY (OUT+) / RDY (OUT-) 接点出力は、コントローラーが正常動作中にメークする接点で、システム全体の健全性確認に使用可能。
