### テキスト
#### パラメーター設定（応用編）
【設定例】
ゾーン1に設定する例として、以下の領域（ゾーン）を示す。
- 1軸目：150～200mmに進入したら出力ポートNo.311がON
- 2軸目：75～125mmに進入したら出力ポートNo.312がON

#### 6.5.22 エラーのレベルを出力したい
全単軸アクチュエーターが原点復帰完了であることを確認可能。
（スカラロボットの場合は原点復帰完了であることを出力できない）

### 表
#### 軸別パラメーター設定値
| 軸別パラメーター | 1軸目   | 2軸目   |
| :--------------- | ------: | ------: |
| No.86            | 200,000 | 125,000 |
| No.87            | 150,000 |  75,000 |
| No.88            |     311 |     312 |

※MAX値（上限値）、MIN値（下限値）の設定は0.001mm単位。

#### エラーレベルと出力ポート対応
| エラーレベル       | 出力ポート300 | 出力ポート301 |
| :----------------- | :-----------: | :-----------: |
| メッセージレベル以下 | ON           | ON            |
| 動作解除レベル       | OFF          | ON            |
| コールドスタートレベル | OFF          | OFF           |

#### パラメーターNo.と設定内容
| パラメーターNo.       | 設定値 | 内容                                                         |
| :-------------------- | -----: | :----------------------------------------------------------- |
| I/Oパラメーター No.46 | 2      | メッセージレベルの場合だけ、出力ポート300をONする設定       |
| I/Oパラメーター No.47 | 3      | メッセージレベル、および動作解除レベルの場合、出力ポート301をONする設定 |
