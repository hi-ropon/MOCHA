### テキスト
#### 6.5.28 ティーチングツールを接続しなくても現在の電流値を知りたい
パネルウィンドウで電流モニターを行うことが可能です。  
連続した軸No.で4軸まで電流モニターが可能です。

例として、6軸コントローラーにおいて、その他のパラメーターNo.49を1、No.50を3と設定した場合、向かって一番右のセグメントが3軸目を表示します。  

パラメーター入力後、フラッシュROM書込み、ソフトウェアリセット（再立上げ）を行うと、  
“レディーステータス表示”・“プログラム実行中 No.表示”の代わりに、モーター電流定格比[%]をセグメントのパターンで表示します。  

セグメントの表示パターンとモーター電流定格比[%]の関係は以下の通りです。  
（※ 赤線が点灯部分になります。）

### 表
| パラメーターNo. | 設定値                         | 内容 |
| :-------------- | :----------------------------- | :--- |
| その他のパラメーターNo.49 | 1                             |       |
| その他のパラメーターNo.50 | モニタリングする軸No.の最小値 |       |

### 図（7セグメント表示例）
6軸、5軸、4軸、3軸の7セグメント表示が並んでいる図

### 表（モーター電流定格比[%]と7セグメント表示の対応）
| 7セグメント表示 | モーター電流定格比[%]の範囲          |
| :-------------- | :---------------------------------- |
| 0（灰色）       | 0 < モーター電流定格比[%] ≤ 25      |
| 0（赤色）       | 100 < モーター電流定格比[%] ≤ 150   |
| 1（灰色）       | 25 < モーター電流定格比[%] ≤ 50     |
| 0（赤色）       | 150 < モーター電流定格比[%] ≤ 200   |
| 1（灰色）       | 50 < モーター電流定格比[%] ≤ 75     |
| 8（赤色）       | 200 < モーター電流定格比[%]          |
| 1（赤色）       | 75 < モーター電流定格比[%] ≤ 100    |

（※ 赤線が点灯部分）
