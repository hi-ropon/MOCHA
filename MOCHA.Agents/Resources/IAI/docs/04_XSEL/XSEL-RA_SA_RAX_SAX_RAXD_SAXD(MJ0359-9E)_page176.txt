### テキスト
#### ブレーキボックスの配線（コネクターについて）
1) ブレーキ解除スイッチコネクター1、ブレーキ解除スイッチコネクター2について説明しています。  
スイッチは定格DC30V 1.5mA以上のものを使用するよう注意喚起があります。

2) ブレーキ電源入力コネクターについて説明しています。  
ケーブル側とブレーキボックス側のコネクター型式が示され、ブレーキボックスに付属していることが記載されています。

### 表
#### ブレーキ解除スイッチコネクター1、2

| コネクター   | 型式                                   | 備考           |
| :----------- | :------------------------------------ | :------------- |
| ケーブル側   | XAP-02V-1<br>コンタクト BXA-001T-P0.6<br>（日本圧着端子） |                |
| 使用コネクター | S02B-XASS-1（日本圧着端子）           |                |

| ピン番号 | 信号名 | 内容                     | 適合電線径               |
| :------- | :----- | :----------------------- | :----------------------- |
| 1        | BKMRL  | ブレーキ解除スイッチ入力 | 0.08～0.3mm² (AWG28～22) |
| 2        | COM    | ブレーキ解除スイッチ入力用コモン |                          |

#### ブレーキ電源入力コネクター

| コネクター       | 型式                              | 備考               |
| :--------------- | :-------------------------------- | :----------------- |
| ケーブル側       | MC1.5/2-ST-3.5<br>（フェニックスコンタクト） | ブレーキボックスに付属 |
| ブレーキボックス側 | MC1.5/2-GF-3.5<br>（フェニックスコンタクト） |                    |

| ピン番号 | 信号名 | 内容           | 適合電線径               |
| :------- | :----- | :------------- | :----------------------- |
| 1        | 0V     | 電源グラウンド | 0.08～1.25mm² (AWG28～16) |
| 2        | 24VIN  | +24V電源      |                          |
