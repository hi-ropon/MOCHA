### テキスト
#### 3.2.2 電源回路の配線（三相電源仕様）
三相電源仕様のXSELコントローラーには、以下の電源を供給してください。

コントローラーの消費電力と発熱量は、ドライバーボードなどの構成により異なります。  
消費電力を計算の上、サーキットブレーカーを選定してください。  
詳細は「2.2.4 サーキットブレーカーの選定」を参照してください。

注1 ブレーキ付きアクチュエーターがなければ不要です。

### 表
| 電源種類       | 仕様                     | 備考                          |
| :------------- | :----------------------- | :---------------------------- |
| モーター電源   | 三相 AC200～230V±10%    |                               |
| 制御電源       | 単相 AC200～230V±10%    |                               |
| I/O電源        | DC24V±10%                | I/OスロットにPIOボードを搭載する場合 |
| ブレーキ電源   | DC24V±10%                | ブレーキ付きの場合             |

**図の説明:**  
- 3相200Vの電源が漏電ブレーカー、サーキットブレーカーを経由し、ノイズフィルター、リングコア、クランプフィルターを通ってXSELコントローラーのAC電源入力コネクターに接続されている。  
- サージプロテクターはD種接地（接地抵抗100Ω以下）に接続されている。  
- DC24V電源はブレーキ電源入力コネクターおよびPIOボード（オプション）に供給されている。  
- ブレーキ付きの場合はブレーキ電源が必要。
