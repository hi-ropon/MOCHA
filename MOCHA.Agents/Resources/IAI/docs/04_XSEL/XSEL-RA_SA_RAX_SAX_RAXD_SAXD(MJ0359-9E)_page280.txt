### テキスト
#### IXAスカラロボットの衝突検知機能（参考プログラムと読み取りの例）
- 例として、コマンド「ECMD」にオペランド1が5、オペランド2が2の命令が示されています。
- この命令は2軸目のステータスを変数99に格納します。
- 命令実行後、変数99に10進数の28が格納されている場合、2軸目のステータスは以下のようになります。

### 表
| Cmnd | Operand 1 | Operand 2 |
| :--- | :-------- | :-------- |
| ECMD | 5         | 2         |

### 図
```mermaid
flowchart TD

    B --> D[ビット6: 0 原点復帰完了]
    B --> E[ビット5: 0 サーボON]
    B --> F[ビット4: 1 動作系コマンド正常終了]
    B --> G[ビット3: 1 押付け空振り未検出]
    B --> H[ビット2: 0 衝突検知機能未使用]
```

### 図の説明
- 変数99の値28は2進数で11100bに変換される。
- 各ビットの意味は以下の通り：
  - ビット7: サーボ軸未使用（0）
  - ビット6: 原点復帰完了（0）
  - ビット5: サーボON（0）
  - ビット4: 動作系コマンド正常終了（1）
  - ビット3: 押付け空振り未検出（1）
  - ビット2: 衝突検知機能未使用（0）
