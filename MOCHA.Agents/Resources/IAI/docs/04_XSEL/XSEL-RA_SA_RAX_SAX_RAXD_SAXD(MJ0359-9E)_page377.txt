### テキスト
#### 6.3.3 回転軸移動における近回り制御の運転例
近回り制御選択は軸別パラメーターNo.67「回転移動軸近回り制御選択」で有効/無効の設定が可能です。  
近回り選択を有効にすると、同一方向にだけ移動を行うことが可能となります。

ポジションNo.1 → 2 → 3 → 4 と順番に移動を行ったとき、近回り選択が「0：無効」の場合と「1：有効」の場合では動作が異なります。

- 「0：無効」の場合  
  移動は順方向だけでなく逆方向にも回る可能性があります。

- 「1：有効」の場合  
  移動は常に同一方向（順方向）にのみ行われます。

### 表
| ポジションNo. | 位置（度） |
| :------------ | :--------- |
| 1             | 0          |
| 2             | 90         |
| 3             | 180        |
| 4             | 270        |

※位置データは1度＝1mmとして入力。例：1.2は1.2度として扱う。

```mermaid
flowchart LR
    A[ポジション No.1] --- B[ポジション No.2]
    B --- C[ポジション No.3]

```

#### 近回り制御「0：無効」の動作例
```mermaid
flowchart LR
    P1((1)) -->|順方向| P2((2))
    P2 -->|逆方向| P3((3))
    P3 -->|順方向| P4((4))
    P4 -->|逆方向| P1((1))
```

#### 近回り制御「1：有効」の動作例
```mermaid
flowchart LR
    P1((1)) -->|順方向| P2((2))
    P2 -->|逆方向| P3((3))
    P3 -->|順方向| P4((4))
    P4 -->|順方向| P1((1))
```

**図の説明:**  
- 「0：無効」の場合は、最後の移動で逆方向に回る動作が発生する。  
- 「1：有効」の場合は、常に同一方向（順方向）に回る動作となる。
