### テキスト
#### 4.3.2 パネルウィンドウの表示
4桁の7セグメントLEDにコントローラーの状態を示します。  
正常に起動すると、イニシャル処理表示の後に「rdy」が表示されます。  
「E＊＊＊」が表示された場合は、7章のトラブルシューティングを確認してください。  
その他の表示については表4.3-1を参照してください。  
CODEの7セグメント表示については表4.3-2を参照してください。

### 表
#### 表4.3-1 コントローラーのLEDの表示内容

| 番号 | 名称   | LED点灯時の状態                             |
| :--- | :----- | :----------------------------------------- |
| (1)  | RDY    | CPUレディー（プログラム運転可能）          |
| (2)  | ALM    | CPUアラーム（メッセージレベル以上エラー）<br>CPUハード異常 |
| (3)  | EMG    | 非常停止状態、CPUハード異常、電源系ハード異常 |
| (4)  | PSE    | 電源系ハード異常                           |
| (5)  | CLK    | システムクロック異常、CPUハード異常       |
| (6)  | CP_ON  | 制御電源正常                               |
