### テキスト
#### (13) システム動作設定スイッチ
システムの動作モードを設定するスイッチです。通常は表の設定にしてください。

#### (14) Ethernetコネクター
外部のEthernet通信機器を接続するコネクターです。  
詳細は別冊のXSEL Ethernet取扱説明書（管理番号：MJ0140）を参照してください。

#### (15) モードスイッチ
コントローラーの動作モードを指定するためのスイッチです。  
誤操作防止のレバーロック式トグルスイッチで、操作時は手前に引いて操作してください。

### 図
#### システム動作設定スイッチのイラスト
```mermaid
flowchart LR
    ON -->|スイッチ番号1| 1
    OFF -->|スイッチ番号2| 2
    ON -->|スイッチ番号3| 3
    OFF -->|スイッチ番号4| 4
```
（イラストは4つのスイッチが縦に並び、ON/OFFの切り替えを示す）

#### モードスイッチのイラスト
```mermaid
flowchart LR
    MODE --> MANU[MANU (手動モード)]
    MODE --> AUTO[AUTO (自動モード)]
```
（丸いスイッチの図で、MANUが上側、AUTOが下側に位置）

### 表
#### 表1.4-3 システム動作設定スイッチの出荷時設定
| スイッチ番号 | 設定 |
| :--- | :--- |
| 1 | ON |
| 2 | OFF |
| 3 | OFF |
| 4 | ON |

#### 表1.4-4 モードスイッチの機能
| スイッチ番号 | 設定 | 内容 |
| :--- | :--- | :--- |
| MANU（手動モード） | 上側 | ティーチングツールが有効です。 |
| AUTO（自動モード） | 下側 | ティーチングツールが無効です。<br>（注）付属のダミープラグを（21）のティーチングコネクターに装着してください。装着されていない場合は、非常停止が解除できません。 |
