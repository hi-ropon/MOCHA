#### IXAスカラロボットのコンプライアンス制御機能

##### 注意
- コンプライアンス制御機能は、ワークの挿入などを支援する補助的な機能であり、100%の挿入成功を保証するものではない。
- ロボットの機種、位置・姿勢、ワークの条件によってはワークの挿入が行えない場合がある。
- ワークや挿入する穴の材質や硬さによっては、ワークや穴を傷付ける場合があるため注意が必要。

##### 対応機種
- IXA-3NNN3015 / 4NNN3015 / 3NSN3015 / 4NSN3015
- IXA-3NNN45□□ / 4NNN45□□ / 3NSN45□□ / 4NSN45□□
- IXA-3NNN60□□ / 4NNN60□□ / 3NSN60□□ / 4NSN60□□
- IXA-4NSC3015 / 4NSC45□□ / 4NSC60□□

##### 対応するXSELコントローラーメインアプリバージョン
- XSELコントローラーメインアプリ部：V1.14以降

##### 対応ティーチングツール
- XSELパソコン対応ソフト：V13.02.16.00以降
- ティーチングペンダント（TB-02/TB-03）：V2.00以降

##### 注意事項
- コンプライアンス制御機能は、衝突検知機能、コンベアートラッキング機能と同時に使用できない。
- 同時使用しないようにSELプログラムなどを作成する必要がある。
- 同時使用時はエラーNo.412「排他モード指定エラー」が発生する。
