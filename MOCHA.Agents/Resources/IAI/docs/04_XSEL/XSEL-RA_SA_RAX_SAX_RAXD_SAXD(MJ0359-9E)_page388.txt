### テキスト
#### 6.5.18 全単軸アクチュエーターが原点位置であることを出力したい
全単軸アクチュエーターが原点位置であることを確認できます。
※スカラロボットの場合は、原点位置であることを出力できません。

#### 6.5.19 全単軸アクチュエーターが原点復帰完了であることを出力したい
全単軸アクチュエーターが原点復帰完了であることを確認できます。
※スカラロボットの場合は、原点復帰完了であることを出力できません。

### 表
#### 6.5.18 全単軸アクチュエーターが原点位置であることを出力したい - 現在位置と出力ポート対応

| 現在位置 | 出力ポート 304 |
| :------- | :------------- |
| 原点     | ON             |
| 原点以外 | OFF            |

#### 6.5.18 パラメーター設定

| パラメーターNo.       | 設定値 | 内容                             |
| :-------------------- | :----- | :------------------------------ |
| I/Oパラメーター No.50 | 1      | 全単軸アクチュエーター原点位置で出力に設定 |

#### 6.5.19 全単軸アクチュエーターが原点復帰完了であることを出力したい - 現在位置と出力ポート対応

| 現在位置       | 出力ポート 304 |
| :------------- | :------------- |
| 原点復帰完了   | ON             |
| 原点復帰未完了 | OFF            |

#### 6.5.19 パラメーター設定

| :-------------------- | :----- | :------------------------------------ |
| I/Oパラメーター No.50 | 2      | 全単軸アクチュエーター原点復帰完了状態で出力に設定 |
