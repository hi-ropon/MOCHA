### 表
#### 動作解除レベルエラー一覧

| No.  | エラー名称                   | 内容・対処                                                                                                                         |
| :--- | :--------------------------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| B4A  | 軸移動中指令エラー           | 【内容・原因】<br>軸移動中に不正な操作が行われた。軸移動中にコンプライアンスモードを設定（COMP命令）<br>【対処方法】<br>軸移動中にコンプライアンスモードの設定をしないように、SELプログラムなどを変更する。 |
| B4E  | 位置ゲイン切替えエラー       | 【内容・原因】<br>位置ゲインを切替えできなかった。以下の要因が考えられる。<br>1. 動作中・押付け中軸などに対して位置ゲイン切替えを試みた。<br>2. 位置ゲイン切替え中にプログラムが終了した。<br>3. 振動が発生して位置決め幅内に留まらない。<br>4. コンプライアンスモード中に位置ゲイン切替えを試みた。<br>など<br>【対処方法】<br>1～4のタイミングを避けて、WGHT・WGT2命令を実行する。 |
| B71  | 座標系 No.エラー             | 【内容・原因】<br>座標系No.が異常。<br>・ワーク座標系No.0～31以外を指定<br>・ツール座標系No.0～127以外を指定<br>・ワーク座標系No.0・ツール座標系No.0のオフセット量を変更<br>・COMP1命令実行時にツール座標系No.0以外を選択<br>【対処方法】<br>SEL命令・IAIプロトコルコマンドの座標系No.指定、選択座標系No.の値を確認し修正する。 |
| B73  | サーボ使用中座標系データ変更禁止エラー | 【内容・原因】<br>1. サーボ系使用中の選択ワーク座標系No.・ツール座標系No.を変更しようとした。<br>2. サーボ系使用中のワーク座標系・ツール座標系オフセット量を変更しようとした。<br>3. コンプライアンスモード中にツール座標系選択No.を変更しようとした。<br>【対処方法】<br>1. 動作中などに選択座標系No.を変更しないようSELプログラムを修正。<br>2. 動作中などに現在選択中のワーク座標系・ツール座標系オフセット量を変更しないよう修正。<br>3. コンプライアンスモード中に選択ツール座標系No.を変更しないよう修正。 |
| B74  | CP動作制限帯侵入エラー       | 【内容・原因】<br>1. スカラロボットのCP動作中にロボットの現在位置がCP動作制限帯（特異点近傍）内に入った。<br>2. コンプライアンス制御中にロボットの指令位置・現在位置がCP動作制限帯内に入った。<br>【対処方法】<br>1. CP移動経路が特異点近傍にならないようロボット・ワークの位置を調整。<br>2. コンプライアンス制御中に特異点近傍位置にならないよう位置を調整。 |
| B80  | 指定禁止軸エラー             | 【内容・原因】<br>SEL命令・IAIプロトコルコマンドなどの軸パターン・軸No.に指定不可な軸が含まれている。<br>【対処方法】<br>軸パターン・軸No.指定値を確認し適切な値に変更する。 |
