### テキスト
#### 4.4.2 プログラム No.を指定して起動
外部から起動したいプログラム番号を指定し、起動信号の入力により運転を行うことができます。  
以下にその手順を説明します。

1) パソコン対応ソフトを接続して、表の設定値を参考にして設定を行います。

### 表
#### 表 4.4-2 プログラム起動用パラメーター設定例

| パラメーター種類、および No. | 設定値 | 内容 | 設定 |
| :---------------------------- | :----- | :--- | :--- |
| I/Oパラメーター No.30 | 0 | 入力ポート000を汎用入力にします。 |  |
| 〃 | 1 (出荷時) | 起動するプログラム番号を、入力ポート007(LSB)～014(MSB)にBCDで指定後、入力ポート000をONするとプログラムを実行します。入力ポート000は0.1秒以上ONしてください。 | 1～4のいずれか |
| 〃 | 2 | 起動するプログラム番号を、入力ポート007(LSB)～014(MSB)にバイナリーで指定後、入力ポート000をONするとプログラムを実行します。入力ポート000は0.1秒以上ONしてください。 |  |
| 〃 | 3 | 起動するプログラム番号を、入力ポート008(LSB)～014(MSB)にBCDで指定後、入力ポート000をONするとプログラムを実行します。入力ポート000は0.1秒以上ONしてください。※入力ポート014の機能はポート007に移動します。 |  |
| 〃 | 4 | 起動するプログラム番号を、入力ポート008(LSB)～014(MSB)にバイナリーで指定後、入力ポート000をONするとプログラムを実行します。入力ポート000は0.1秒以上ONしてください。※入力ポート014の機能はポート007に移動します。 |  |
| No.30=1または2の場合 | I/Oパラメーター No.37～43 | 0 | 入力ポート007～013を汎用入力に設定します。 | 1 |
| 〃 | 〃 | 1 | 入力ポート007～013を、起動プログラム番号指定ポートに設定します。 |  |
| No.30=3または4の場合 | I/Oパラメーター No.38～44 | 0 | 入力ポート008～014を汎用入力に設定します。 | 1 |
| 〃 | 〃 | 1 | 入力ポート008～014を、起動プログラム番号指定ポートに設定します。 |  |
