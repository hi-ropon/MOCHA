### テキスト
#### 注意事項
- 最大プログラム数は255。ただしBCDコード指定で起動できるプログラムはNo.1～No.99まで。
- BCDコードを使用する場合、No.100～No.255の起動はオートスタートプログラム起動またはプログラム起動命令“EXPG”で行う。

#### プログラム起動手順
1. プログラムNo.の入力は、XSELコントローラーが起動後（正常起動＝RDY信号ON後）、0.01秒以上経過してから行う。  
2. 起動するプログラムNo.は、I/OパラメーターNo.30で設定した入力ポートにセットする。  
3. プログラムNo.をセットして0.05秒以上経過後、入力ポート000（プログラムスタート）をONにする。  
4. 入力ポート000は0.1秒以上ONにする。  
5. 設定したプログラムが起動する。

```mermaid
flowchart TD
    A[RDY出力Z軸 (XSELの出力)] -->|2) Min.0.01s| B[プログラムNo. (上位から出力)]
    B -->|3)| C[プログラムスタート (上位から出力)]
    C -->|4) Min.0.05s| D[プログラム]
    C -->|5) Min.0.1s| D
    D -->|6) プログラム実行| E[プログラム実行状態]
```

**図の説明:**  
- RDY信号がONになってから0.01秒以上経過後にプログラムNo.が入力される。  
- プログラムNo.セット後、0.05秒以上経過してからプログラムスタート信号がONになる。  
- プログラムスタート信号は0.1秒以上ONに保たれ、その後プログラムが実行される。
