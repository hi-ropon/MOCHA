### テキスト
#### 7.4.4 PSA-24 ステータスデータモニター
当社製DC24V電源ユニット「PSA-24」は、装置の運用・保守の効率向上を目的とした予防保全機能として、上位からステータスデータをモニターする機能を搭載しています。  
ステータスデータモニター以外のPSA-24搭載予防保全機能については、「PSA-24取扱説明書（MJ0379）」を参照してください。

##### 【接続方法】
- RCONのゲートウェイボードとPSA-24を接続し、RCON経由で電源のステータスデータをモニターします。
- パソコン専用ティーチングソフト（ソフトバージョン RCM-101：V13.00.00以降、IA-OS：Ver.1.00.00.00以降）を使ってステータスデータをモニターする場合は、RCONのUSBインターフェイスを使って接続します。
- フィールドネットワーク上にも電源のステータスデータを上げているため、PLC経由で上位のネットワークからステータスデータをモニターすることも可能です。  
（詳細は「6.2.1 固定領域の制御・状態信号【2】電源ユニット（PSA-24）状態信号」を参照）

### 図
```mermaid
flowchart LR
    PSA1[PSA-24 電源ユニット] -- 最大5台まで接続 --> PSA2[PSA-24 電源ユニット]
    PSA2 -- 電源付属コネクター --> RCON[RCON ゲートウェイボード]
    PSA2 -- RCON付属コネクター --> RCON
    RCON -- USB --> PC1[PC (USB経由でPCソフトにてモニター)]
    RCON -- Fieldbus --> PLC[PLC]
    PLC -- Ethernet --> PC2[PC (PLC経由にて遠隔監視)]
```

**図の説明:**  
- PSA-24電源ユニットは最大5台まで直列接続可能  
- PSA-24からRCONへは専用コネクターで接続  
- RCONはUSB経由でPCに接続し、専用ソフトでステータスデータをモニター可能  
- また、RCONはFieldbus経由でPLCに接続し、PLC経由でEthernetネットワーク上のPCから遠隔監視が可能
