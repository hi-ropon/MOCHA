### テキスト
#### ブレーキリリーススイッチ
ブレーキ強制解除用のスイッチです。  
通常使用時はNOM側にしておきます。NOM側では、サーボONするとブレーキ解除、サーボOFFでブレーキロックとなります。  
RLS側では、サーボON/OFFに関係なく強制解除となります（制御電源OFF時を除く）。

#### 警告
- ブレーキの解除は十分に注意して行ってください。不用意に行うと、スライダーあるいはロッドの落下により、けがやアクチュエーター本体、ワークまたは装置などの破損の原因となります。  
- ブレーキ解除後は必ずブレーキを有効状態に戻してください。ブレーキ解除状態のまま運転を行うと非常に危険です。スライダーあるいはロッドの落下により、けがやアクチュエーター本体、ワークまたは装置などの破損の原因となります。

### 画像
（ブレーキリリーススイッチの写真）
- スイッチは「RLS」と「NOM」の2つの位置があり、切り替え可能。

### 表
| 記号 | 説明                         |
| :--- | :--------------------------- |
| RLS  | ブレーキリリース（Brake Release） |
| NOM  | ブレーキロック（Normal）          |
