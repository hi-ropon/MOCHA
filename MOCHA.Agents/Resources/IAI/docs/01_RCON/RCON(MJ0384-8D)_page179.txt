### テキスト
#### ジョグスイッチ
ジョグ運転用のスイッチです。  
パラメーターNo.194「JOGスイッチ」を“0”に設定すると本スイッチが有効になります（初期値は“0：有効”）。  

JOG+側にスイッチを傾けると＋方向に、JOG−側に傾けると−方向にジョグ動作を行います。  
スイッチを傾け続けると段階的にジョグ速度が速くなります。ただし、原点復帰未完了の場合はジョグ速度は原点復帰速度となります。  

サーボON中にジョグスイッチでジョグ動作をすると完了後もサーボONのままですが、サーボOFF中の場合はサーボONしてからジョグ動作を行い、完了後にサーボOFFします。  

ジョグスイッチの操作はMANUモード時のみ有効で、AUTOモード時は無効です。  
また、ティーチングツールでアクチュエーターを操作できる画面を開いている場合もジョグスイッチは無効となり、ジョグスイッチによる動作中に操作画面を開くとアクチュエーターは減速停止します。  

### 画像
ジョグスイッチの実物写真（青色のスイッチでJOG−とJOG+の表示あり）

### 表
| 記号  | 説明                         |
| :---- | :--------------------------- |
| JOG+  | ＋方向にジョグ動作（原点の反対方向） |
| JOG−  | －方向にジョグ動作（原点方向）       |

### 注意事項
- ティーチングツールでアクチュエーター操作画面を開いているときに通信が切断されると、ジョグスイッチ操作が無効のままになる。  
- ジョグスイッチ操作を有効に戻すには、RCONシステムの電源再投入かソフトウェアリセットを行う必要がある。
