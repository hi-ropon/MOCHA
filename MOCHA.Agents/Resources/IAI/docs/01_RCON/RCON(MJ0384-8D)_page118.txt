### テキスト
#### SIOコネクター
ティーチングボックスやパソコン専用ティーチングソフトと接続するためのコネクターです。パソコン専用ティーチングソフトについてはUSB（mini-B）でも接続可能です。

### 画像
SIOコネクターの写真が掲載されています。

### 表
#### 基板側コネクター仕様（TCS7587-0121077 ホシデン）

| ピン番号 | 信号名   | 説明                                   |
| :------- | :------- | :------------------------------------ |
| 1        | TP_SD+   | ティーチングボックス/PC用 RS-485 差動信号＋側 |
| 2        | TP_SD−   | ティーチングボックス/PC用 RS-485 差動信号−側 |
| 3        | T5V      | ティーチングボックス用 5V 出力         |
| 4        | ENB      | イネーブル信号入力                     |
| 5        | STOPA    | 停止ライン A                          |
| 6        | T24V     | ティーチングボックス用 24V 出力        |
| 7        | GND      | 0V                                    |
| 8        | STOPB    | 停止ライン B                          |
| 9（シェル） | GND      | 0V                                    |
