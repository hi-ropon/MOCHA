### テキスト
#### ポジションデータ編集画面の切替え
ポジションデータ編集画面は、簡易、標準、全項目、カスタマイズの4種類から表示切替が選択できます。

### 図
```mermaid
flowchart LR
    A[ポジションデータ編集画面] --> B{表示切替選択}
    B -->|簡易| C[簡易入力画面]
    B -->|標準| D[必要最小限表示画面]
    B -->|全項目| E[全表示画面]
    B -->|カスタマイズ| F[カスタマイズ画面]
    F --> G[表示させたい項目を選択可能]
```

**図の説明:**  
- 「簡易」：簡単入力画面  
- 「標準」：必要最小限の表示画面  
- 「全項目」：全ての項目を表示する画面  
- 「カスタマイズ」：表示させたい項目を選択できる画面（加減速モード、搬送負荷、ポジションゾーン信号、制振制御、停止モード、しきい等の項目）

### チェックボックスリスト（カスタマイズ項目）
| 項目名           | 説明                     |
|------------------|--------------------------|
| 加減速モード     | 加減速のモード設定       |
| 搬送負荷         | 搬送負荷の表示           |
| ポジションゾーン信号 | ポジションゾーンの信号表示 |
| 制振制御         | 制振制御の表示           |
| 停止モード       | 停止モードの表示         |
| しきい           | しきい値の表示           |
</contents>
