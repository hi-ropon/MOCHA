### テキスト
#### 4.8 EC接続ユニットへの配線
##### （1）モーター駆動用DC電源の電源配線
- 電源コネクター型式：MSTB2.5/6-STF-5.08
- 配線箇所はモーター電源（L1とL2）、制御電源（L1CとL2C）、アース（PE）の計5か所
- 配線手順：
  1. 各配線径は別表を参照
  2. 配線のストリップ部長さは7mm
  3. 電線を電源コネクターの電線挿込み口に奥まで挿入
  4. マイナスドライバーで端子ねじを締める

### 図
- 電源コネクターの写真に各部名称（端子ねじ、挿込み口、電線、マイナスドライバー）を示し、配線方法を解説
- 電線のストリップ長さ7mmを示す写真
- 配線の良い例・悪い例のイラスト（配線が端子に正しく接触しているか否か）

```mermaid
flowchart TD
    A[電線のストリップ長さ7mmに剥く] --> B[電線を電源コネクターの挿込み口に奥まで挿入]
    B --> C[マイナスドライバーで端子ねじを締める]
    C --> D{配線の良否}
    D -->|良い例| E[端子にしっかり接触]
    D -->|悪い例| F[端子に接触不良]
```

**図の説明:**  
- 電線は7mmのストリップ長さに剥き、電源コネクターの挿込み口に奥まで挿入する。  
- マイナスドライバーで端子ねじを締めて固定する。  
- 配線が端子にしっかり接触していることが重要で、接触不良は避ける。
