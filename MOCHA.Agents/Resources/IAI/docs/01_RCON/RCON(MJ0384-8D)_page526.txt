### テキスト
#### 6.3 ドライバーユニットの入出力信号
24Vドライバーユニットは、PCONコントローラーなどのCON系と同じ押付け方法、またはPSEPなどのSEP系コントローラーと同じ押付け方法をパラメーターNo.181「押付け方式」で選択可能です。200VドライバーユニットはCON方式のみ対応しています。

##### (1) CON方式押付け
現在位置から目標位置へ到達後、位置決め幅に設定した距離だけ押付け速度で移動します。押付け移動中にワークに押当たり押付け完了と判断すると、位置決め完了信号PENDがONします。

##### (2) SEP方式押付け
目標位置から位置決め幅（直接数値指定モード）に設定した距離分手前をスタート位置として押付け動作を行います。引張り動作はありません。押付け移動中にワークに押当たり押付け完了と判断するとPEND信号がONします。

##### 注釈
- 直接数値指定モード、フル直値モード、フル直値2モード、簡易直値モードでは位置データ指定レジスターに入力した値が使用されます。
- 位置決め幅指定レジスターに入力した値が位置決め幅として使用されます。

### 図
#### CON方式押付けの速度と位置の関係
```mermaid
flowchart LR
    A[速度] --> B[移動距離]
    B --> C[目標位置]
    C --> D[位置決め幅（押付け動作による最大移動量）]
    D --> E[ワークに押当たり、押付け完了と判断して位置決め完了信号PENDがONする位置]
```

#### SEP方式押付けの速度と位置の関係
```mermaid
flowchart LR
    A[速度] --> B[押付け開始位置（現在位置）]
    B --> C[押付け幅（位置決め幅）]
    C --> D[目標位置]
    D --> E[ワークに押当たり、押付け完了と判断して位置決め完了信号PENDがONする位置]
```

**図の説明:**
- CON方式は目標位置に到達後、位置決め幅分だけ押付け速度で移動し押付け完了を検出。
- SEP方式は目標位置の手前から押付け動作を開始し、押付け完了を検出。
