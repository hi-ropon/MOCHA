### テキスト
#### 制振制御モード選択（NTC0、NTC1）PLC出力信号
- 制振制御機能は、アクチュエーターによって引き起こされる負荷の振動を抑制する機能です。  
- 振動数を測定し、最大3種類のパラメーターセットに設定します。  
- 登録されたパラメーターセットの中から1種類を選択して使用します。  
- 詳細は「7.1 制振制御機能」を参照してください。

### 表
#### NTC1、NTC0の状態と機能対応表

| NTC1 | NTC0 | 機能               | 備考           |
| :--- | :--- | :----------------- | :------------- |
| OFF  | OFF  | パラメーターセット0選択 | 制振制御未設定 |
| OFF  | ON   | パラメーターセット1選択 |                |
| ON   | OFF  | パラメーターセット2選択 |                |
| ON   | ON   | パラメーターセット3選択 |                |

### グラフ
#### NTC0、NTC1信号入力タイミング（フル直値動作）

```mermaid
sequenceDiagram
    participant PLC
    participant RCON

    PLC->>RCON: 目標位置データ送信
    PLC->>RCON: NTC0信号送信
    PLC->>RCON: NTC1信号送信
    PLC->>RCON: DSTR信号送信

    Note right of RCON: T1 = 0ms以上に設定すること
```

**図の説明:**  
- 目標位置データがPLCからRCONへ送信される。  
- NTC0、NTC1信号は目標位置データの後に送信される。  
- DSTR信号はNTC0、NTC1信号の後に送信され、T1（0ms以上）遅延がある。  

### 注意事項
- フル直値モードおよびフル直値2モードでは、移動指令（DSTR）認識時にNTC0、NTC1信号状態を取り込むため、移動中にNTC0、NTC1信号のON/OFF操作を行っても無視されます。
