### グラフ
#### LED点滅タイミング

4種類のLED点滅パターンが時間軸に沿って示されています。  
各パターンはON/OFFの状態変化を秒単位で表現しています。

```mermaid
flowchart TD
    subgraph 点滅1
        A1[OFF 0秒] --> A2[ON 0.75秒] --> A3[OFF 1秒]
    end
    subgraph 点滅2
        B1[OFF 0秒] --> B2[ON 0.75秒] --> B3[OFF 1秒] --> B4[ON 1.25秒] --> B5[OFF 1.5秒]
    end
    subgraph 点滅3
        C1[OFF 0秒] --> C2[ON 0.75秒] --> C3[OFF 1秒] --> C4[ON 1.25秒] --> C5[OFF 1.5秒] --> C6[ON 1.75秒] --> C7[OFF 2秒]
    end
    subgraph 点滅4
        D1[OFF 0秒] --> D2[ON 0.75秒] --> D3[OFF 1秒] --> D4[ON 1.25秒] --> D5[OFF 1.5秒] --> D6[ON 1.75秒] --> D7[OFF 2秒] --> D8[ON 2.25秒] --> D9[OFF 2.5秒]
    end
```

**図の説明:**  
- 点滅1は0秒から0.75秒までOFF、0.75秒から1秒までON、その後OFF。  
- 点滅2は点滅1のパターンに加え、1.25秒から1.5秒までONの追加点滅。  
- 点滅3は点滅2のパターンにさらに1.75秒から2秒までONの追加点滅。  
- 点滅4は点滅3のパターンに2.25秒から2.5秒までONの追加点滅。
