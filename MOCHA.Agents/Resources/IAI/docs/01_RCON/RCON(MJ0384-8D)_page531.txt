### テキスト
#### 【26】加減速モード（MOD0、MOD1） PLC出力信号
加減速パターン特性を選択するための信号です。いずれかをアクチュエーターの移動指令前に選択してください。

- リモートI/Oモード1～3、5、簡易直値モードの場合は、ポジションデータの「加速度」、「減速度」欄で設定します。
- フル直値モードでは、加速度と減速度の各レジスターに設定します。
- 直接数値指定モードでは、以下の表の通りMOD0、MOD1の組み合わせで機能が決まります。

#### 注意
- 移動指令（DSTR）認識時にMOD0、MOD1信号状態を取り込むため、移動中にMOD0、MOD1信号をON/OFF操作しても無視されます。

### 表
#### MOD0、MOD1の組み合わせと機能

| MOD1 | MOD0 | 機能           |
| :--- | :--- | :------------- |
| OFF  | OFF  | 台形パターン   |
| OFF  | ON   | S字モーション  |
| ON   | OFF  | 一次遅れフィルター |
| ON   | ON   | 使用できません |

### 図
#### フル直値動作でのMOD0、MOD1信号入力タイミング

```mermaid
flowchart TD
    A[目標位置データ (PLC → RCON)] -->|交差| B[MOD0 (PLC → RCON)]
    B --> C[MOD1 (PLC → RCON)]
    C --> D[DSTR (PLC → RCON)]
    D -->|T1 (≥0ms)| E[信号認識完了]
```

**図の説明:**  
- 目標位置データがPLCからRCONへ送られた後、MOD0、MOD1信号が順に入力される。  
- DSTR信号はMOD0、MOD1信号の入力後に一定時間（T1 ≥ 0ms）遅れて入力される。  
- このタイミングで信号が認識され、加減速モードが決定される。
