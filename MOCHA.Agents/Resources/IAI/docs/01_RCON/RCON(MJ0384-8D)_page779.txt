### テキスト
#### GWパラメーター3の設定項目説明

- **ドライバーシャットダウン解除遅延時間**  
  ドライバーユニットの各軸に電源を順次供給する際の遅延時間（インターバル）を設定。  
  各軸に電源を供給するタイミングをずらすことで突入電流を低減する目的で使用。  
  初期値は1軸あたり5ms。  
  ただし、200Vドライバーユニットの遅延時間の最小値は50msで、設定値が50ms未満の場合は強制的に50msとなる。  
  EC接続ユニットの場合は、1ユニット（4軸）ごとの駆動源遮断回路の解除タイミングをずらす。

- **電源ユニット接続台数**  
  ゲートウェイユニットに接続する当社製電源ユニット（型式：PSA-24）の台数を設定。  
  接続可能な最大台数は5台。

- **電源ユニットモニタ種別**  
  当社製電源ユニット（PSA-24）のモニター項目を選定。  
  モニターできる項目は次ページの表のいずれか1つ。

### 画像（設定画面）
- 「特殊パラメータ設定」ウィンドウの「GWパラメータ3」タブのスクリーンショット。  
  以下の項目が入力・選択可能になっている。

| 項目名                         | 内容（画面上の表示）                |
|------------------------------|----------------------------------|
| ドライバーシャットダウン解除遅延時間[msec] | 20                               |
| 電源ユニット接続台数               | 2                                |
| 電源ユニットモニタ種別             | 出力電圧                         |
| 電源ユニット通信リトライ回数         | 2                                |
| 初期化内部通信待ち時間[msec]         | 0                                |
