### テキスト
#### モーター駆動用DC電源の状態出力コネクターの接続
モーター駆動用DC電源の状態を出力します。出力はPWR、MP、ALMのLEDと連動します。  
状態出力コネクターの取付けはマイナスドライバーで締め込む方式です。

### 画像説明
- 状態出力コネクター（型式：DFMC1.5/3-STF-3.5）  
- モーター駆動用DC電源（型式：PSA-200-*）  
- マイナスドライバーでコネクターを締め込む様子を示す写真

### 表1：コネクター信号一覧

| ピン番号 | 信号名  | 内容               | 備考                             |
| :------- | :------ | :----------------- | :------------------------------- |
| 1        | *ALM-   | アラーム接点出力   |                                 |
| 2        | *ALM+   | アラーム接点出力   |                                 |
| 3        | MP-     | モーター電源接点出力 | フォトカプラー絶縁オープンコレクター出力 |
| 4        | MP+     | モーター電源接点出力 | フォトカプラー絶縁オープンコレクター出力 |
| 5        | PWR-    | 制御電源接点出力   |                                 |
| 6        | PWR+    | 制御電源接点出力   |                                 |

### 表2：適合電線径仕様

| 項目         | 仕様       |
| :----------- | :--------- |
| 適合電線     | AWG24～16  |
| 最大ケーブル長 | 10m        |
| ストリップ長 | 10.0mm     |

※絶縁被覆の温度定格が60℃以上の電線を使用してください。
