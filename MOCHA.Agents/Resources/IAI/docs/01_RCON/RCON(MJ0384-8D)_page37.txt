### テキスト
#### アクチュエーターの座標系
(2) スライダータイプ  
(3) テーブルタイプ  
(4) アームタイプ  

各タイプのアクチュエーターにおける座標系の説明があり、動作方向の正負の向きとモーター側の位置関係について記載されています。

### 図
- 各アクチュエーターの3D図に、動作方向の座標軸が矢印で示されている。
- 矢印の片側に「0（＋）」、反対側に「＋（0）」と表記されている。
- モーター側の位置がラベルで示されている。

```mermaid
flowchart LR
    subgraph スライダータイプ
        A1[モーター側] -->|動作方向| A2[0 (+)]
        A2 --> A3[+ (0)]
    end

    subgraph テーブルタイプ
        B1[モーター側] -->|動作方向| B2[0 (+)]
        B2 --> B3[+ (0)]
    end

    subgraph アームタイプ
        C1[モーター側] -->|動作方向| C2[0 (+)]
        C2 --> C3[+ (0)]
    end
```

**図の説明:**  
- 各タイプのアクチュエーターはモーター側から見て動作方向が決まっている。  
- 座標系は「0（＋）」から「＋（0）」の方向に動くことを示している。  
- モーター側の位置は動作方向の基準点として重要である。
