### テキスト
#### 入出力信号のタイミングと運転手順
- 目標位置データ設定（PLC→RCON）
- 指令ポジションNo.（PLC→RCON）
- 位置決めスタート CSTR（PLC→RCON）
- 位置決め完了 PEND（RCON→PLC）
- 現在位置（RCON→PLC）
- 移動中 MOVE（RCON→PLC）
- アクチュエーター移動

注1 Tdpf：応答時間  
応答時間の詳細は「6.5.1 入出力信号のタイミング」を参照。

### 図（タイミングチャート）
```mermaid
flowchart TD
    subgraph PLC_to_RCON
        A1[目標位置データ設定 (n1)] --> A2[n2]
        A3[n2] -.-> A4[n3]
        B1[指令ポジションNo. (v1)] --> B2[v2]
        B3[v2] -.-> B4[v3]
    end

    subgraph Timing
        C1[位置決めスタート CSTR] -->|10ms以上| C2[位置決め完了 PEND]
        C2 --> C3
    end

    subgraph RCON_to_PLC
        D1[現在位置 (n1)] --> D2[n2]
        E1[移動中 MOVE] --> E2
    end

    subgraph Actuator
        F1[アクチュエーター移動] --> F2
    end

    A2 -->|0ms以上| C1
    C2 --> D1
    D1 --> E1
    E1 --> F1

    %% 応答時間 Tdpf 表示
    C1 -. Tdpf (注1) .-> C2
    C2 -. Tdpf (注1) .-> D2
```

**図の説明:**  
- PLCからRCONへ目標位置データと指令ポジションNo.が送信される。  
- 位置決めスタート信号（CSTR）が送られ、10ms以上の期間を経て位置決め完了信号（PEND）が返される。  
- 現在位置情報と移動中信号がRCONからPLCへ送られ、アクチュエーターが移動する。  
- Tdpfは応答時間を示し、詳細は別章で説明されている。
