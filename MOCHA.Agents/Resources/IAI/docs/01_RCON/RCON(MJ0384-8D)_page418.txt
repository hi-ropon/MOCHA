### テキスト
#### アクチュエーターを登録したポジション（目標位置）へ移動させる手順

1) 移動させたいポジションNo.の“位置”欄をクリックして選択する。

2) “ポジション移動”欄の「▶」ボタンをクリックする。  
停止させたい場合は「■」ボタンをクリックする。

### 表
| No. | 位置 [mm] |
| :-- | :-------- |
| 0   |           |
| 1   | 0.00      |
| 2   | 100.00    |

### 図
```mermaid
flowchart TD
    A[ポジションNo.の位置欄をクリック] --> B[ポジション移動欄の▶ボタンをクリック]
    B --> C{停止させたい場合}
    C -->|はい| D[■ボタンをクリック]
    C -->|いいえ| E[ステップ移動開始]
    E --> F[選択したポジションへ移動実行]

    subgraph 操作画面
        B
        D
        E
    end
```

**図の説明:**  
- 「停止」ボタン（■）をクリックすると動作を停止できる。  
- 「ステップ移動」ボタン（▶）をクリックすると選択したポジションへアクチュエーターが移動する。  
- 移動イメージは原点（座標0mm）からポジションNo.1の100mmまで動作する様子を示している。
