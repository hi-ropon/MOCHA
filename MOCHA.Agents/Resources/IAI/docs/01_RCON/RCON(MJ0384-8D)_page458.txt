### テキスト
#### 入出力信号一覧
PLC入力の電源ユニット状態信号0～4に関するビットごとの信号内容が説明されています。  
特に、各ビットの役割や発生条件、動作内容が記載されています。

#### モニター対象項目
電源ユニットの各種パラメータ（出力電圧、補助巻線電圧、出力電流など）について、モニター時の意味や動作内容が説明されています。

### 表
#### 入出力信号一覧

| 信号種類 | ビット | 記号名 | 内容 |
| :--- | :--- | :--- | :--- |
| PLC入力 電源ユニット状態信号0～4 | b15 | LNK | ゲートウェイユニットと電源ユニットの通信が正常に確立し、ゲートウェイユニットに有効と認識されているときにONします。 |
|  | b14～11 | - | 使用できません |
|  | b10 | OPMV | モニター対象のデータ読取異常が発生した場合にONします。 |
|  | b9 | FANW | ファンの回転数が30%低下した時点で、メッセージレベルのアラーム（アラームコード04C “FAN回転数低下”）を発生します。アラームが発生するとONし、解除されるとOFFします。 |
|  | b8 | FANA | ファンの回転数が50%低下してしまった場合、動作解除レベルのアラーム（アラームコード0D6 “FAN異常検出”）を発生し、アクチュエーターが停止します。アラームが発生するとONし、解除されるとOFFします。 |
|  | b7～b0 | PSMV | ゲートウェイパラメーターの特殊パラメーターで選択した項目をモニターします。<br>設定方法は[5.5.5 Ethernetコネクター（オプション）を使用した接続設定]を参照。<br>以下7項目の内1項目をモニター可能。<br> (1) 出力電圧：0～255V<br> (2) 補助巻線電圧：0～255V<br> (3) 出力電流：0～25.5A（0.1A単位）<br> (4) ピークホールド電流：0～25.5A（0.1A単位）<br> (5) 負荷率：0～255%<br> (6) FAN回転速度：0～25,500r/min（100r/min単位）<br> (7) PCB温度：0～255℃ |

#### モニター対象項目

| 項目 | 内容 |
| :--- | :--- |
| (1) 出力電圧 | 本電源は負荷に応じて、出力電圧を変動させています。出力電圧モニター値が変化しますが、異常ではありません。 |
| (2) 補助巻線電圧 | 電源内部の制御電源電圧です。出力電圧側の負荷に応じて、出力電圧と同様に変化します。 |
| (3) 出力電流 | 出力電流の瞬時値です。 |
| (4) ピークホールド電流 | 出力電流のピーク値です。 |
| (5) 負荷率 | 出力電流の積分値と定格出力電流の比率です。この値が100%を超えると過負荷エラーが発生し出力電圧が遮断されます。 |
| (6) FAN回転速度 | ファンの回転速度です。 |
| (7) PCB温度 | 内部温度：二次側の出力コンデンサー近傍の温度です。 |
