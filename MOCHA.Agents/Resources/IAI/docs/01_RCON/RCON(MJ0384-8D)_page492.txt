### テキスト
#### 各アドレスの割付け内容
- 「m」は各軸の先頭レジスターアドレス、「n」は各軸の先頭の相対アドレスを示す。
- 「p」は各軸の先頭のモジュールアドレス。
- CC-Link、CC-Link IE Field、CC-Link IE TSN、DeviceNetはワードアドレス。
- PROFIBUS-DP、EtherNet/IP、EtherCATはバイトアドレス。
- PROFINET-IOは4ワードモジュールのアドレス。
- 現在電流値はパルスモーターの場合は指令電流値、ACサーボモーター（SCON接続含む）の場合はフィードバック電流値。
- LOAD、TRQSは特定のコントローラ（RCON-PC/PCF、RCON-SC、SCON-CB-RC、SCON2-CG-RC）のみ対応。

### 図（ビット割付け構成）
1ワード＝16ビット（15b〜0b）で構成され、以下の信号が割り当てられている。

| アドレス | 内容（ビット単位） |
| --- | --- |
| p+2 (m+8〜m+F相当) | 現在指令値（L・H）、推定回生放電電力量（L・H）、通算走行距離（L・H） |
| p+3 | 状態信号1（16ビット中一部ビットにMEND、ALMLなどの信号名） |
| p+4以降 | 状態信号2（EMGS、CRDY、ZONE2、ZONE1、PZONE、LOAD、TRQS、GHMS、PUSHS、PSFL、SV、ALM、MOVE、HEND、PENDなどの信号名） |

**図の説明:**
- 各ワードは16ビットで構成され、ビットごとに異なる信号が割り当てられている。
- 状態信号1、状態信号2は複数のフラグや状態を示すビット群である。
- アドレスは通信プロトコルによりワードまたはバイト単位で異なるが、ここでは共通のビット割付けを示している。
