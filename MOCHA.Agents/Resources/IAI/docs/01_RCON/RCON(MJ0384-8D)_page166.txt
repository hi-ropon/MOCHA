### テキスト
#### AC200V入力コネクター
三相、単相AC200V入力用コネクターです。  
三相AC200Vの場合は接続できる合計W数は2,400W、単相AC200Vの場合は合計W数は1,600Wまでとなります。

### 写真
- AC200V入力コネクターの写真（緑色の4ピンコネクター）

| 種類           | 型式                             |
| :------------- | :------------------------------- |
| 基板側コネクター | PC5/4-GF-7.62（フェニックス・コンタクト） |
| ケーブル側コネクター | SPC5/4-STF-7.62（フェニックス・コンタクト） |
