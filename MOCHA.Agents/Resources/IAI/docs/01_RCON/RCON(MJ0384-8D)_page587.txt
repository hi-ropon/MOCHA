### テキスト
#### 衝突検出機能の設定

【設定】  
(1) 使用する機能の選択  
パラメーターNo.168「衝突検出機能」を設定する。  

(2) 検出電流値の設定  
ポジションテーブルの「しきい」の欄に0～100[%]で設定。0を設定すると検出しない。  

(3) 判定時間の設定  
パラメーターNo.50「負荷出力判定時間」を設定。  
設定範囲：0～9,999ms（初期値255ms）  

(4) 判定範囲（ポジションゾーン）の設定  
ポジションテーブルの「ゾーン＋」「ゾーン－」に範囲を設定。  
設定時は「ゾーン＋」より「ゾーン－」を小さな値に設定する。  
設定範囲：0.00～アクチュエーターストローク長（mm）  

### 表
#### 使用する機能の選択（パラメーターNo.168）

| 設定値 | 内容                                                                                     | アラームレベル                          |
| :----- | :--------------------------------------------------------------------------------------- | :------------------------------------ |
| 0      | 検出を行いません（2、4、6を設定した場合も同様）                                         | －                                    |
| 1      | ポジションゾーン設定範囲で検出を行います。                                             |                                       |
| 3 (注1) | ポジションゾーン設定範囲で検出を行いますが、次の場合は検出を行いません。<br>・一時停止解除後の最初の移動<br>・ポジションゾーン範囲内で停止した状態からの移動 | 動作解除レベル（サーボOFF）<br>（アラーム0DF） |
| 5      | ポジションゾーン設定範囲で検出を行います。                                             |                                       |
| 7 (注1) | ポジションゾーン設定範囲で検出を行いますが、次の場合は検出を行いません。<br>・一時停止解除後の最初の移動<br>・ポジションゾーン範囲内で停止した状態からの移動 | メッセージレベル（アラーム049）       |

注1：この設定で、加速時の電流値増大による誤検出を回避できる。
