### テキスト・表
#### 【入出力信号一覧】直接数値指定モード（ON=該当ビットが"1"、OFF=該当ビットが"0"）

| 信号種類       | ビット       | 信号名 | 内容                                                                                                                                                                                                                                     | 詳細   |
| -------------- | ------------ | ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| 位置データ指定 | 32ビットデータ | -      | 32ビット符号付整数（単位：0.01mm、ただしDDモーターの場合は0.001度）<br>例）+25.40mmならば2540と指定します。<br>●設定最大値は+9,999.99mm=999999です。<br>●位置データはソフトストロークの範囲内で設定してください。<br>●16進数で入力する場合、負数は2の補数で指定してください。 | 6.5.2  |
| 位置決め幅     | 32ビットデータ | -      | 32ビット整数（単位：0.01mm、ただしDDモーターの場合は0.001度）<br>例）+25.40mmならば2540と指定します。<br>●位置データはソフトストロークの範囲内で設定してください。<br>●押付け動作の方向はDIRで指定します。<br>●位置決め幅指定データの設定がされていない場合、パラメーターNo.10「位置決め幅初期値」の設定は適用されませんので注意してください。 | 6.5.2  |
| 速度           | 16ビットデータ | -      | 16ビット整数（単位：1.0mm/s、または0.1mm/s）<br>例）単位：1.0mm/sの場合、200mm/sなら200と指定します。<br>●速度の設定がされていない場合、または設定が"0"の場合は停止したままとなり、アラームにはなりません。<br>移動中に設定を"0"にして速度変更を行った場合は減速停止します。<br>●単位の切替はゲートウェイパラメーター設定ツールで行います。（出荷時は1.0mm/sに設定） | 6.5.2  |
| 加減速度       | 16ビットデータ | -      | 16ビット整数（単位：0.01G）<br>例）0.20Gに設定する場合、20と指定します。<br>●加減速度の設定がされていない場合、パラメーターNo.9「加減速度初期値」の設定は適用されませんのでご注意ください。<br>●加速度、減速度の個別設定はできません。加減速度の設定となります。 | 6.5.2  |
| 押付け時電流制限値 | 16ビットデータ | -      | 押付け動作時の電流制限値を設定します。<br>設定の範囲は0～200%で、255=100%、510=200%となります（注1）。<br>例）50%に設定する場合は、255×50%=127と設定します。                                                                                   | 6.5.2  |

**注1**: 電流制限値の設定範囲と対応するパーセンテージの関係を示しています。
