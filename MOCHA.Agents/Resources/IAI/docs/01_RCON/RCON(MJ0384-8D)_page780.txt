### 表
#### ゲートウェイパラメーター設定ツールの項目説明

| 項目               | 内容                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 出力電圧           | 本電源は負荷に応じて、出力電圧を変動させています。出力電圧モニター値が変化しますが、異常ではありません。 |
| 補助巻線電圧       | 電源内部の制御電源電圧です。出力電圧側の負荷に応じて、出力電圧と同様に変化します。                     |
| 出力電流           | 出力電流の瞬時値です。                                                                   |
| ピークホールド電流 | 出力電流のピーク値です。                                                                 |
| 負荷率             | 出力電流の積分値と、定格出力電流の比率です。この値が100%を超えると、過負荷エラーが発生し出力電圧が遮断されます。 |
| FAN回転速度        | ファンの回転速度です。                                                                   |
| PCB温度            | 内部温度：二次側の出力コンデンサー近傍の温度です。                                       |

### テキスト
- 電源ユニット通信リトライ回数：通信エラーが発生した場合のリトライ回数を設定します。  
- 初期化内部通信待ち時間：ドライバーユニットが内部通信を開始するまでの追加の待ち時間を設定します。

### 画像説明
【GWモードセレクト】設定画面のスクリーンショットが掲載されています。  
設定項目は以下の3つで、それぞれ「無効」または「有効」を選択可能です。

### テキスト
- イネーブル有効  
  TPのイネーブルスイッチの有効/無効を選択します。  
- バイトスワップ  
  バイトスワップを設定します。詳細は「バイトスワップ（C7-73ページ）」を参照。  
- Wワードデータワードスワップ  
  Wワードサイズのデータをワードサイズでスワップするか設定します。詳細は「Wワードデータワードスワップ（C7-73ページ）」を参照。
</contents>
