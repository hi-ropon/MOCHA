### テキスト
#### 2.3 フィールドネットワーク仕様
RCONシステムのフィールドネットワークインターフェイス仕様について説明しています。

---

#### 2.3.1 CC-Linkインターフェイス仕様
CC-Linkの通信規格や速度、通信方式、ケーブル長などの仕様を示しています。

---

#### 2.3.2 CC-Link IE Fieldインターフェイス仕様
IEEE802.3ab（1000BASE-T）準拠のCC-Link IE Fieldの通信規格や速度、制御方式、ケーブル長、接続コネクター、サイクリック通信のビット・ワード数などの仕様を示しています。

---

### 表

#### CC-Linkインターフェイス仕様

| 項目           | 仕様                                                                                   |
| :------------- | :------------------------------------------------------------------------------------- |
| 通信規格       | CC-Link Ver1.10 または Ver2.00                                                        |
| 局種別         | リモートデバイス局（最大4局占有）                                                    |
| 通信速度       | 10Mbps / 5Mbps / 2.5Mbps / 625kbps / 156kbps                                         |
| 通信方式       | ブロードキャストポーリング方式                                                        |
| 占有局数       | [6.1.4 全体アドレス構成例]参照                                                        |
| 通信ケーブル長 | 通信速度（bps）: 10M / 5M / 2.5M / 625k / 156k<br>総ケーブル長（m）: 100 / 160 / 400 / 900 / 1200 |
| 通信ケーブル   | 専用ケーブルを使用してください                                                        |
| コネクター     | MSTB2.5/5-STF-5.08 AU（フェニックス・コンタクト）                                   |

注1: T分岐通信の場合は、マスターユニットおよび搭載されるPLCの取扱説明書を参照。  
注2: ケーブル側コネクターは標準付属品。製品の確認は[4.2.1]参照。

---

#### CC-Link IE Fieldインターフェイス仕様

| 項目           | 仕様                                                                                   |
| :------------- | :------------------------------------------------------------------------------------- |
| 通信規格       | IEEE802.3ab（1000BASE-T）準拠                                                        |
| 通信速度       | 1Gbps                                                                                  |
| 通信制御方式   | トークンパッシング方式                                                                 |
| 通信ケーブル長 | 各セグメント間の距離：100m以内                                                        |
| 最大接続台数   | 254台（マスター局とスレーブ局の合計）                                                |
| 最大ネットワーク数 | 239                                                                                  |
| 通信ケーブル   | カテゴリー5e以上（アルミテープと編組の二重遮蔽シールドケーブル推奨）                   |
| 接続コネクター | RJ-45コネクター2個                                                                    |
| サイクリック通信 | RX（スレーブ→マスター）：16384ビット<br>RY（マスター→スレーブ）：16384ビット<br>RWr（スレーブ→マスター）：8192ワード<br>RWw（マスター→スレーブ）：8192ワード |

注1: 通信ケーブルは別途用意してください。
