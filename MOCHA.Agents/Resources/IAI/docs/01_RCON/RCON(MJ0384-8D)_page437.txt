### テキスト
#### RCONシステムの割付け順序
前ページのRCONシステム構成例に基づき、PLCからRCONシステム、及びRCONシステムからPLCへの割付け順序を示しています。

#### 補足
ドライバーユニットの割付け順はゲートウェイパラメーターの設定で変更可能です。
- 詳細は「5.5.3 軸番号の割付変更」を参照してください。
- ドライバーユニットの動作モードは軸ごとに設定可能で、設定方法は「5.5.2 動作モードの個別設定」を参照してください。

#### 注意
オプションユニットはRCONゲートウェイユニットに近い順に割付けられ、ソフトウェア上での入替えはできません。

### 表
#### PLC → RCONシステム と RCONシステム → PLC の割付け順序

| PLC → RCONシステム           | RCONシステム → PLC                          |
|-----------------------------|---------------------------------------------|
| **GW 制御（2W）**            | **GW 制御（2W）**                            |
| **予約（6W）**               | 電源ユニット状態 or 軽故障メンテナンス信号（5W） |
| 軸番号1 制御（RCON-PC-1）    | 軸番号1 状態（RCON-PC-1）                    |
| 軸番号2 制御（RCON-SC-1）    | 軸番号2 状態（RCON-SC-1）                    |
| 軸番号3 制御（SCON-CB-*-RC） | 軸番号3 状態（SCON-CB-*-RC）                 |
| オプションユニット0 制御（RCON-EXT-NP） | オプションユニット0 状態（RCON-EXT-NP）       |
| オプションユニット1 制御（RCON-NP）      | オプションユニット1 状態（RCON-NP）            |
| オプションユニット2 制御（RCON-EC-4_1）  | オプションユニット2 状態（RCON-EC-4_1）        |
| オプションユニット3 制御（RCON-EC-4_2）  | オプションユニット3 状態（RCON-EC-4_2）        |

※オプションユニットはRCONゲートウェイユニットに近い順に割付けられます。
</contents>
