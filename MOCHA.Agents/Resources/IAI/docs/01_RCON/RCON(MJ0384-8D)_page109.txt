### テキスト
#### 3.1.2 型式
##### （1）型式確認
ゲートウェイユニットの型式は以下のように構成されています。

```
RCON - タイプ - I/O種類 - オプション
```

- タイプ  
  - GW：標準タイプ  
  - GWG：安全カテゴリ対応タイプ  

- I/O種類  
  - CC：CC-Link接続仕様  
  - CIE：CC-Link IE Field接続仕様  
  - CIT：CC-Link IE TSN接続仕様  
  - DV：DeviceNet接続仕様  
  - EC：EtherCAT接続仕様  
  - EP：EtherNet/IP接続仕様  
  - PR：PROFIBUS-DP接続仕様  
  - PRT：PROFINET IO接続仕様  
  - ECM：EtherCATモーション接続仕様  
  - ML3：MECHATROLINK-III接続仕様  
  - SSN：SSCNETⅢ/H接続仕様  
  - N：I/Oなし（※1）  

- オプション  
  - EP：EtherNet/IP接続（※1、※2）  
  - ET：Ethernet搭載（※2）  
  - FU*：ファンユニット装着（*は装着数を指定、1～8）  
  - TRN：ターミナルユニットなし  

※1 I/O種類「N」とオプション「EP」は必ずセットで選択してください。  
※2 オプション「EP」と「ET」は同時に選択できません。

```mermaid
flowchart LR
    A[ゲートウェイユニット] --> B[フィールドネットワークコネクター（I/O種類）]
    A --> C[Ethernetポート（オプション）]

```

**図の説明:**  
- ゲートウェイユニット本体にフィールドネットワークコネクターがあり、I/O種類に対応。  
- オプションとしてEthernetポートが装備可能。  
- ファンユニットは単品型式RCON-FUで装着可能。  
- ターミナルユニットは単品型式RCON-GW-TRで装着される。
