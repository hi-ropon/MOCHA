### テキスト
#### 占有情報（マスター登録情報）
占有情報は、接続するドライバーユニットやオプションユニットの接続台数、各ドライバーユニットの動作モード設定により変化します。  
以下に各ネットワークの占有情報の例を示します。

##### 設定条件
- ドライバーユニット　：7軸  
- 動作モード　　　　：全軸_直接数値指定（位置決め）（サイズ：8W）  
- オプションユニット：4台  

### 表
#### 各ネットワークの占有情報例

| ネットワーク名       | 内容                                                                                   |
|----------------------|----------------------------------------------------------------------------------------|
| **CC-Link**          | 局種別：Ver.2 リモートデバイス局<br>拡張サイクルック：8倍<br>占有局数：2局           |
| **CC-Link IE Field** | 局種別：インテリジェントデバイス局<br>RX/RY：128ビット（16バイト）<br>RWw/RWr：64ワード（128バイト） |
| **CC-Link IE TSN**   | 局種別：リモート局<br>RX/RY：128ビット（16バイト）<br>RWw/RWr：60ワード（120バイト）  |
| **DeviceNet**        | 出力サイズ：136バイト<br>入力サイズ：136バイト                                      |
