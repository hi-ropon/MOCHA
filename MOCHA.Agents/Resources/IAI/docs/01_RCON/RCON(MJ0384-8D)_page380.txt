### テキスト
#### 4) 軸構成変更画面が開きます。
軸番号割付設定画面で、追加するユニットの軸番号の設定方法を選択します。設定内容を反映する場合は「進む」ボタンをクリックします。

#### 予約ユニット追加時の軸番号設定例
- 「未割当番号」を選択した場合、追加した分のドライバーの軸番号は未割当の番号が設定されます。  
- 「シフト」を選択した場合、追加した分のドライバーの軸番号は前からシフトされて割り当てられます。

### 表
| 選択項目       | 軸番号の設定変更結果                         |
| :------------- | :------------------------------------------ |
| 未割当番号     | 未割当の番号を設定する。                     |
| シフト         | 追加した分のドライバーの軸番号を前からシフトする。 |

### 図
#### 軸番号割付設定画面の例
- 左側にUnit0の軸番号が0,1と設定されている状態から、右側の画面でUnit0の軸番号が9,10に変更されている様子が示されている。
