### テキスト
#### S字モーション比率設定（パラメーターNo.56）
- パラメーターNo.56は「S字モーション比率設定」で、単位は％、入力範囲は0～100、工場出荷時の初期値は0です。  
- ポジションテーブルの「加減速モード」欄の値を1（S字モーション）に設定した場合に使用します。  
- サイクルタイムを延ばすことなく加減速時のショックを和らげることが可能です。  
- S字モーションは加速時間を1周期とする正弦波形であり、本パラメーターで振幅の度合いを指定します。

### 表
#### 振幅の度合い設定表

| 設定[%] | 振幅の度合い                                   |
| :------ | :--------------------------------------------- |
| 0       | S字モーションなし（イメージ図の点線）         |
| 10      | 正弦波の振幅×0.1（イメージ図の実線）          |
| 50      | 正弦波の振幅×0.5（イメージ図の1点鎖線）       |
| 100     | 正弦波の振幅×1（イメージ図の2点鎖線）         |

### グラフ
#### 加速時間中の速度変化イメージ
```mermaid
flowchart LR
    A[時間] --> B[速度]
    B --> C[加速時間]
    C --> D[振幅]
```
- 横軸は時間、縦軸は速度を示す。  
- 加速時間内に振幅があり、S字モーションの正弦波形の振幅を示す。  

#### 設定値ごとの速度変化曲線
```mermaid
flowchart LR
    A[時間(0～2秒以内)] --> B[速度]
    B --> C{設定値}
    C -->|10| D[振幅小（実線）]
    C -->|50| E[振幅中（1点鎖線）]
    C -->|100| F[振幅大（2点鎖線）]
```
- 横軸は時間（0～2秒以内）、縦軸は速度。  
- 設定値10、50、100に応じて速度の上昇曲線の振幅が異なる。  
- 振幅が大きいほど加速時のショックが和らぐが、速度変化の曲線がより滑らかになる。
