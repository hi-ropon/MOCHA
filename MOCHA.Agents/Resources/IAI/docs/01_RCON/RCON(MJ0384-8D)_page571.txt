### テキスト
#### 無限回転制御
パラメーターNo.79「回転軸モード選択」を「1：インデックスモード」に、パラメーターNo.80「回転軸近回り選択」を「1：有効」に設定することで、同一方向に連続的に運転することが可能となり、モーターのように連続回転を行えます。  
連続運転の設定方法について説明しています。

##### 運転例
2回転させて、最後にポジションNo.4に停止させる例を示しています。

**図の説明:**  
- 円グラフは4つのポジション（No.1～No.4）を示し、回転軸の位置を表しています。  
- 位置決めタイミングの図は、ポジションNo.1からNo.4までの位置決め動作が連続して行われる様子を示し、複数回の回転を表現しています。

### 表
| ポジションNo. | 位置   |
| :------------ | :----- |
| 1             | 0.00   |
| 2             | 120.00 |
| 3             | 240.00 |
| 4             | 90.00  |

- 位置データは1度=1mmとして入力します。例：1.2は1.2度として扱います。

### 詳細説明
1. ポジションNo.1～3の位置決め幅の設定は、減速開始位置より手前になるように広げます。  
2. ポジションNo.1の位置決め完了信号（PEND）がONになると、ポジションNo.2の位置決めを開始します。同様にNo.3、No.4へと連続して位置決めを実行します。  
3. ポジションNo.1～4の速度設定を同じにすると、同一速度で回転し、最後にポジションNo.4で停止します。何回転させるかは、ポジションNo.1～3の繰り返し回数で決定します。
