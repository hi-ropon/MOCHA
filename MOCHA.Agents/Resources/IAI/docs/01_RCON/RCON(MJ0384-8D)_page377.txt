### テキスト
#### 軸番号設定の追加・削除後の設定手順
5) 次に、追加/削除後の軸番号設定を行います。  
以下の選択項目から選択して、「進む」ボタンを押します。

### 表
| 軸数 | 選択項目 | 軸番号の設定変更結果 |
| :--- | :--- | :--- |
| 追加時 | シフト | 追加した分のドライバーの軸番号を前からシフトする。 |
| 追加時 | 未割当番号 | 追加したドライバーに未割当の番号を設定する。 |
| 削除時 | シフト | 削除した分のドライバーの軸番号を前からシフトする。 |
| 削除時 | 予約ユニットとして残す | 編集したユニットの後ろに予約ユニットを追加し軸番号を残す。 |

### 図（画像説明）
- 上段：軸番号割付設定画面のスクリーンショット。  
  - 「軸数」「軸番号設定方法」の選択肢があり、「進む」ボタンで次に進む操作を示す。  
  - 左は「予約して残す」、右は「シフト」を選択した例。  

- 下段：軸番号設定変更の具体例  
  - ＜Unit1にDrv1を追加し、「シフト」を選択した場合＞  
    Unit1のDrv1が追加され、以降の軸番号が前にシフトされている。  
  - ＜Unit1にDrv1を追加し、未割当番号を選択した場合＞  
    Unit1のDrv1が追加されるが、既存の軸番号は変わらず、新規ドライバーに未割当番号が割り当てられている。
