### テキスト
#### 3.4.2 型式
##### （1）型式の見方

【200Vドライバーユニット】  
200Vドライバーユニットの型式は以下のように構成される。

- RCON - [タイプ] - [接続軸数] - [オプション]

タイプ、接続軸数、オプションの意味は以下の通り。

| 項目       | 内容                         |
|------------|------------------------------|
| タイプ     | SC：200V ACサーボモータータイプ |
| 接続軸数   | 1：1軸仕様                   |
| オプション | 無記入：標準タイプ           |

##### 200V AC 60-750Wモーターの型式一覧

| 型式   | モーター出力・備考                |
|--------|---------------------------------|
| 60     | 60Wサーボモーター               |
| 100    | 100Wサーボモーター              |
| 100S   | 100Wサーボモーター（LSA用）     |
| 150    | 150Wサーボモーター              |
| 200    | 200Wサーボモーター              |
| 200S   | 200Wサーボモーター（LSA、DD用） |
| 300S   | 300Wサーボモーター（LSA用）     |
| 400    | 400Wサーボモーター              |
| 600    | 600Wサーボモーター              |
| 750    | 750Wサーボモーター              |

【200Vドライバー用ファンユニット】  
型式は「RCON-FUH」となる。
