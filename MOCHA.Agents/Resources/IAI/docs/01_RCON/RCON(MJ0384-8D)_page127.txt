#### LED表示

| 名称      | パネル表記 | 表示色 | 状態       | 説明                                                                                   |
|-----------|------------|--------|------------|----------------------------------------------------------------------------------------|
| STATUS 0  | RUN        | 緑     | 点灯       | ネットワーク加入後のリフレッシュ＆ポーリング正常受信、またはリフレッシュ正常受信       |
|           |            |        | 消灯       | 1. ネットワーク未加入<br>2. チャネルのキャリア検出<br>3. タイムオーバー<br>4. ハードウェアリセット中 |
| STATUS 1  | ERR        | 橙     | 点灯       | 1. CRCエラー<br>2. リセット解除時、局番設定エラー（0または占有局数を含んで65局以上）<br>3. リセット解除時、ボーレート設定エラー |
|           |            |        | 点滅 (0.4s点滅) | リセット解除時の局番およびボーレート設定値から値が変化した                                   |
|           |            |        | 消灯       | 1. 正常交信<br>2. ハードウェアリセット中                                                 |
