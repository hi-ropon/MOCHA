#### ユニット種別変更画面の操作
3) “ユニット種別変更画面”が開きます。  
以下の6種類からユニット種別を選択し、「次へ」ボタンを押してください。

#### ユニット種別一覧
| ユニット種別 |
| :----------- |
| PC、PCF、AC、DC、SC、SCON、予約 |

#### 軸構成変更画面の操作
4) “軸構成変更画面”が開きます。  
まず、ドライバーユニットで使用する軸数を選択してください。

#### 選択したドライバーユニットの軸数と選択項目

| 選択したドライバーユニットの軸数 | 選択項目                     |
| :------------------------------ | :--------------------------- |
| 1軸                             | 1 ※1                       |
|                                | 2 (Drv0 に追加)             |
|                                | 2 (Drv1 に追加)             |
| 2軸                             | 1 (Drv0 を削除)             |
|                                | 1 (Drv1 を削除)             |
|                                | 2                          |

※1：PCF、SC、SCONは1ユニット1軸のため、選択項目は1のみとなります。

### 画像説明
- ユニット種別変更画面のスクリーンショットには、ユニット種別の選択ドロップダウンと「次へ」ボタンが表示されています。  
- 軸構成変更画面のスクリーンショットは2枚あり、軸数を「追加」または「削除」する操作が矢印で示されています。  
- それぞれの画面で現在値とプレビューが表示され、軸数の変更内容が視覚的にわかるようになっています。
