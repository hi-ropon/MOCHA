### テキスト
#### AUTO/MANUスイッチ
自動運転/手動運転切替えスイッチです。システムI/OコネクターのAUTO/MANU入力とは直列接続となっています。

| 記号 | 説明 |

| AUTO | PLCなどの上位機器からの指令を受付け可能なオンライン動作モード |
| MANU | PLCなどの上位機器からの指令を受付けない教示動作モード |
