### テキスト
#### 停止モード
位置決め完了後の節電のため、一定時間後に自動的にサーボOFFを行うことができます。  
時間の設定はパラメーターNo.36～38の自動サーボOFF遅延時間1～3で行い、3種類の時間を選択可能です。  
サーボモーター仕様、DCブラシレスモーター仕様は0～3から選択可能、パルスモーター仕様は0～7から選択可能です。

#### 注意
- 自動サーボOFF中は保持トルクがありません。外力が加わるとアクチュエーターが動くため、設定時は十分注意してください。  
- 次の移動指令が相対量指定（ピッチ送り）の場合、自動サーボOFFは使用しないでください。位置ずれの可能性があります。  
- 押付け動作では自動サーボOFFを使用しないでください。押付け力が失われます。  
- パソコン専用ティーチングソフトのティーチモードでの運転では自動サーボOFFは機能しません。

### 表
#### 停止モード設定値一覧

| 設定値 | 位置決め完了後の動作                                   | 選択可能な仕様       |
| :----- | :---------------------------------------------------- | :------------------- |
| 0      | サーボONのまま                                        | 全仕様               |
| 1      | 一定時間（パラメーターNo.36設定値）後自動サーボOFF   | 全仕様               |
| 2      | 一定時間（パラメーターNo.37設定値）後自動サーボOFF   | 全仕様               |
| 3      | 一定時間（パラメーターNo.38設定値）後自動サーボOFF   | 全仕様               |
| 4      | フルサーボ制御                                       | パルスモーター仕様   |
| 5      | 一定時間（パラメーターNo.36設定値）フルサーボ制御後、自動サーボOFF | パルスモーター仕様   |
| 6      | 一定時間（パラメーターNo.37設定値）フルサーボ制御後、自動サーボOFF | パルスモーター仕様   |
| 7      | 一定時間（パラメーターNo.38設定値）フルサーボ制御後、自動サーボOFF | パルスモーター仕様   |
