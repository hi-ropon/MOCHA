### テキスト
#### 5.2.1 フィールドネットワーク動作モード
フィールドネットワークは、以下の動作モードから選択して動作させることが可能です。（モーションネットワークおよびEC接続ユニット接続時を除く）  
上位に接続したPLCなどから、動作に必要なデータ（目標位置、速度、加速度、押付け電流値など）を決められたアドレスに書き込んで動作させます。

### 表
| 動作モード           | 内容                                                                                         | 概要                                                                                         |
|----------------------|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| 直接数値指定モード   | 目標位置、速度、加減速度、押付け電流制限値を数値指定可能。0.01mm単位で現在位置のほか、現在速度、現在電流値もモニター可能。 | PLCから目標位置、速度、加速度、押付け％、現在位置、現在電流値、現在速度、アラームコード、状態信号を通信。アクチュエーター制御。 |
| 簡易直値モード       | 目標位置を直接数値で指定可能。0.01mm単位で現在位置のモニターも可能。                         | PLCから目標位置、制御信号、現在位置、完了ポジションNo.、状態信号を通信。アクチュエーター制御。 |
| ポジショナー1モード  | 最大128点の位置データを登録し、登録位置に停止可能。0.01mm単位で現在位置のモニターも可能。   | PLCから目標ポジションNo.、制御信号、現在位置、完了ポジションNo.、状態信号を通信。アクチュエーター制御。 |
| ポジショナー2モード  | 最大128点の位置データを登録し、登録位置に停止可能。現在位置のモニター不可。送受信データ量を減らしたモード。 | PLCから目標ポジションNo.、制御信号、完了ポジションNo.、状態信号を通信。アクチュエーター制御。 |
| ポジショナー3モード  | 最大128点の位置データを登録し、登録位置に停止可能。現在位置のモニター不可。送受信データ量をさらに減らし、最低限の信号で制御。 | PLCから目標ポジションNo.、制御信号、完了ポジションNo.、状態信号を通信。アクチュエーター制御。 |
| ポジショナー5モード  | 最大16点の位置データを登録し、登録位置に停止可能。送受信データ量、ポジションテーブルを減らし、0.1mm単位で現在位置のモニター可能。 | PLCから目標ポジションNo.、制御信号、現在位置、完了ポジションNo.、状態信号を通信。アクチュエーター制御。 |

### 図
各動作モードに対応したPLCとアクチュエーター間のフィールドネットワーク通信のイメージ図がそれぞれ示されています。  
PLCからアクチュエーターへ目標位置や制御信号を送り、アクチュエーターから現在位置や状態信号を返す構成です。

（例：直接数値指定モードの通信イメージ）
```mermaid
flowchart LR
    PLC["PLC"]
    Actuator["アクチュエーター"]
    PLC -->|目標位置、位置決め幅、速度、加減速度、押付け％、制御信号| Actuator
    Actuator -->|現在位置、現在電流値、現在速度（指令値）、アラームコード、状態信号| PLC
    PLC ---|フィールドネットワークによる通信| Actuator
```

（例：簡易直値モードの通信イメージ）
```mermaid
flowchart LR
```
