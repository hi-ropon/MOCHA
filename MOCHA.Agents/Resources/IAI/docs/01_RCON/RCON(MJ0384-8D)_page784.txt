### テキスト
#### I/Oデータ（レジスターモニター）

本レジスターモニター画面では、ゲートウェイユニットが上位（マスター）から受信したデータおよび上位（マスター）側に返す送信データを表示します。

- **データ読込み周期**  
  表示データの更新周期を100～500msの間で選択可能。

- **表示切替**  
  表示を2進数と16進数から選択可能。

- **同期スクロール**  
  チェックを入れると送受信データが同時にスクロールします。

### 画像説明
レジスターモニター画面のスクリーンショットには、左側に「マスター→ゲートウェイ」、右側に「ゲートウェイ→マスター」のデータアドレスとデータが一覧表示されています。  
画面下部には以下の操作項目があり、それぞれ矢印で指示されています。

| ラベル         | 説明                           |
| -------------- | ------------------------------ |
| データ読込み周期 | 表示データの更新周期を設定     |
| 表示切替       | 2進数/16進数の表示切替         |
| 同期スクロール | 送受信データの同期スクロール設定 |
