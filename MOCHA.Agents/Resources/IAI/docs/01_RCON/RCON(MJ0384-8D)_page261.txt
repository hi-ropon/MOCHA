### テキスト
#### 4.3 ユニットの連結と据付け
##### 4.3.1 ユニット連結
DINレールに取り付ける前に、ユニットを連結します。

1. 連結アッパーパーツ・連結ボトムパーツの操作部をパネル側へ回し、クリック感があるところに位置させる。  
2. 破線の○部2か所に連結アッパーパーツ・連結ボトムパーツ、点線の□部4か所に位置決めボス、計6か所が嵌合するように2つのユニットを位置合わせする。  
3. 位置合わせができたら、接続コネクター×2がしっかり接続されるよう挿入する。  
4. 連結アッパーパーツ・連結ボトムパーツの操作部を背面側に向かってクリック感があるところまでしっかり回す。

### 図
ユニットの上面・背面の図に、連結アッパーパーツ操作部と連結ボトムパーツ操作部の位置が示されており、  
それぞれの操作部の回転方向が「パネル側」から「背面側」へ回す矢印で表現されている。

```mermaid
flowchart LR
    A[連結アッパーパーツ操作部] -->|パネル側から背面側へ回す| B[クリック感がある位置で停止]
    C[連結ボトムパーツ操作部] -->|パネル側から背面側へ回す| D[クリック感がある位置で停止]
```

**図の説明:**  
- 連結アッパーパーツ操作部はパネル側から背面側へ回す。  
- 連結ボトムパーツ操作部も同様にパネル側から背面側へ回す。  
- 破線の○部2か所と点線の□部4か所でユニット同士を位置合わせし、接続コネクター×2を確実に接続する。
