### テキスト
#### 押付け方式（パラメーターNo.181）
押付け方式はCON方式またはSEP方式から選択可能です。  
※200Vドライバーユニットの場合はCON方式のみ対応。

(1) CON方式押付け  
現在位置から目標位置へ移動し、位置決め幅に設定した距離を押付け速度で移動します。  
押付け移動中にワークに押当たり、押付け完了と判断すると位置決め完了信号PENDがONします。

(2) SEP方式押付け  
目標位置から位置決め幅に設定した距離を差し引いた位置をスタート位置として押付け動作を行います。  
引張り動作はできません。  
押付け移動中にワークに押当たり、押付け完了と判断するとPEND信号がONします。

注1：直接数値指定、簡易直値モードでは位置データ指定レジスターに入力した値。  
注2：直接数値指定モードでは位置決め幅指定レジスターに入力した値。

### 図
#### CON方式押付けの動作イメージ
```mermaid
flowchart LR
    A[現在位置] --> B[目標位置へ移動]
    B --> C[位置決め幅内を押付け速度で移動]
    C --> D[ワークに押当たり押付け完了判定]
    D --> E[位置決め完了信号PEND ON]
```
- 横軸：位置、縦軸：速度のグラフで、目標位置まで速度一定で移動し、位置決め幅内で速度が減速し押付け完了位置でPEND信号がONする。

#### SEP方式押付けの動作イメージ
```mermaid
flowchart LR
    A[現在位置] --> B[押付け開始位置（目標位置－位置決め幅）]
    B --> C[押付け幅（位置決め幅）を押付け動作]
    C --> D[ワークに押当たり押付け完了判定]
    D --> E[位置決め完了信号PEND ON]
```
- 横軸：位置、縦軸：速度のグラフで、押付け開始位置から押付け幅分押付け動作を行い、押付け完了位置でPEND信号がONする。
