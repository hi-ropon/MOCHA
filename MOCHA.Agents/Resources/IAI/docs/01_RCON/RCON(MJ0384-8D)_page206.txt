### テキスト
#### 外部入力仕様
SCON拡張ユニット、PIO/SIO/SCON拡張ユニット、PIOユニットの外部入力仕様について説明しています。

### 表
#### 外部入力仕様の項目と仕様

| 項目       | 仕様               |
|------------|--------------------|
| 入力点数   | 16点               |
| 入力電圧   | DC24V±10%          |
| 入力電流   | 4mA/1回路          |
| ON電圧    | 最小 DC18V（3.5mA） |
| OFF電圧   | 最大 DC6V（1mA）    |
| 絶縁方式   | フォトカプラー      |

### 図
#### NPN仕様回路図
```mermaid
flowchart LR
    A[+24V] -->|抵抗| B[5.6kΩ]
    B --> C[各入力]
    C --> D[ロジック回路]
    D --> E[フォトカプラー]
    E --> F[内部回路]
    C --> G[0V]
```

#### PNP仕様回路図
```mermaid
flowchart LR
    A[+24V] --> C[各入力]
    C --> B[5.6kΩ]
    B --> D[ロジック回路]
    D --> E[フォトカプラー]
    E --> F[内部回路]
    C --> G[0V]
```

**図の説明:**
- NPN仕様は入力側に抵抗があり、+24Vから入力を経てロジック回路へ信号が送られます。
- PNP仕様は入力側に抵抗があり、入力から+24Vへ接続される形で回路が構成されています。
- 両仕様ともフォトカプラーを介して内部回路に信号を伝達しています。
