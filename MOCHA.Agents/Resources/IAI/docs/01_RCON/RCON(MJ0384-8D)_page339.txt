### テキスト
#### 4.12.5 EtherCAT
EtherCAT接続に使用するコネクターとケーブルの仕様について説明しています。

- コントローラー側コネクターは、写真で上面からの形状が示されています。
- ケーブル側、コントローラー側ともにEthernet ANSI/TIA/EIA-568-Bカテゴリー5以上のシールド付き8P8Cモジュラープラグ（RJ-45）を使用する必要があります。
- Ethernetケーブルはカテゴリー5以上のストレートSTPケーブルを推奨しています。
- 終端抵抗は不要です。

### 表
#### EtherCAT接続コネクター仕様

| コネクター名称 | EtherCAT接続コネクター | 備考 |
| :--- | :--- | :--- |
| ケーブル側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上 シールド付 8P8C モジュラープラグ（RJ-45） | 用意してください |
| コントローラー側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上 シールド付 8P8C モジュラージャック（RJ-45） |  |

| 5 | － | 未使用 |  |
| 6 | RD- | 受信データ－ |  |

- Ethernetケーブルはカテゴリー5以上のストレートSTPケーブルを使用してください。
