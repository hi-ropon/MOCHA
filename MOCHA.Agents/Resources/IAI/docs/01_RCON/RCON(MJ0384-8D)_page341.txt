### テキスト
#### 4.12.7 EtherNet/IP（フロントポート）
- コントローラー側のコネクター正面図を示し、EtherNet/IP接続に使用するコネクターの種類を説明。
- ケーブル側、コントローラー側ともに「Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上 シールド付 8P8C モジュラープラグ（RJ-45）」または「モジュラージャック（RJ-45）」を使用すること。
- Ethernetケーブルはカテゴリー5以上のストレートSTPケーブルを使用することを推奨。

| ケーブル側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上 シールド付 8P8C モジュラープラグ（RJ-45） | 用意してください |
| コントローラー側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上 シールド付 8P8C モジュラージャック（RJ-45） |  |

### 図
#### コントローラー側コネクター正面図と接続イメージ

- コントローラー側コネクターの写真に矢印で接続位置を示す。
- ネットワーク構成図はスイッチングハブを中心に、マスターユニット（パソコンなど）、スレーブ機器、RCON-GW EtherNet/IP（フロントポート）オプションが接続されている。
- ケーブルはEthernetストレートケーブル（カテゴリー5以上、アルミテープと編組の二重遮断シールドケーブル推奨）を使用。
