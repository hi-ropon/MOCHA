### テキスト
#### 押付け方向指定（DIR）PLC出力信号
押付けを行う方向を指定します。  
- 本信号をOFFにすると目標位置から位置決め幅を減算した値に向かって押付けを行う。  
- 本信号をONにすると目標位置に位置決め幅を加算した値に向かって押付けを行う。  
- 通常位置決め動作を選択した場合は本信号は無効。

##### 押付け動作の方向設定例
1) 始点からプラス方向に押付け動作を行う場合  
2) 始点からマイナス方向に押付け動作を行う場合  

上記のように、座標値が増える方向に押付ける場合は押付け方向プラス（＋）に設定し、座標値が減る方向に押付ける場合は押付け方向マイナス（－）に設定する。  
押付け方向の設定を誤ると正しい動作ができず、位置決め幅×2の距離だけ始点側で押付け動作してしまうため注意が必要。

#### 押付け動作の速度・位置関係（始点からマイナス方向の場合）
```mermaid
flowchart LR
    F[現在位置] --> E[押付け動作最大移動位置]
    E --> D[正しい動作]
    D --> C[設定速度に達する]
    C --> B[押付け速度上昇]
    B --> A[始点]

    subgraph 説明
        DIRプラスの場合は点線で示す
        位置決め幅、目標位置、押付け動作開始位置を示す
    end
```

**図の説明:**  
- 縦軸は速度（現在速度、設定速度、押付け速度）を示し、横軸は位置（始点から現在位置まで）を示す。  
- 押付け方向がプラスの場合は始点から目標位置に向かって速度が変化し押付け動作を行う。  
- 押付け方向がマイナスの場合は逆方向に速度が変化し押付け動作を行う。  
- DIR信号の設定により押付け動作の方向が決定される。
