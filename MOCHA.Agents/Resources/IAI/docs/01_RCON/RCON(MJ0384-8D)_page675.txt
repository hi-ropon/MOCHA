### テキスト
#### CC-Link IE Field 通信設定
使用するマスターに合わせて、以下の項目を設定します。

- ネットワークタイプ: CC-Link IE Field
- ネットワークNo.
- 局番
- 通信速度（自動、1Gbps固定）
- MACアドレス（製品固有）

※ネットワークモジュールに割り付けられている固有の識別番号で設定不可。

#### CC-Link IE TSN 通信設定
使用するマスターに合わせて、以下の項目を設定します。

- ネットワークタイプ: CC-Link IE TSN
- IPアドレス
- サブネットマスク
- デフォルトゲートウェイ
- 通信速度（1Gbpsまたは100Mbps）
- MACアドレス（製品固有）

※ネットワークモジュールに割り付けられている固有の識別番号で設定不可。

### 表
#### CC-Link IE Field 設定項目

| 項目           | 初期値             | 設定可能範囲           |
| :------------- | :----------------- | :--------------------- |
| ネットワークタイプ | CC-Link IE Field   | 変更不可               |
| ネットワークNo.   | 1                  | 1～239                 |
| 局番           | 1                  | 1～120                 |
| 通信速度       | 自動               | 設定不要（1Gbps固定）  |
| MACアドレス    | 製品固有           | 設定不可               |

#### CC-Link IE TSN 設定項目

| 項目               | 初期値           | 設定可能範囲               |
| :----------------- | :--------------- | :------------------------- |
| ネットワークタイプ   | CC-Link IE TSN   | 変更不可                   |
| IPアドレス          | 192.168.0.1      | 0.0.0.0～255.255.255.255   |
| サブネットマスク    | 255.255.255.0    | 0.0.0.0～255.255.255.255   |
| デフォルトゲートウェイ | 0.0.0.0          | 0.0.0.0～255.255.255.255   |
| 通信速度            | 1[Gpbs]          | 1[Gpbs] / 100[Mbps]        |
| MACアドレス         | 製品固有         | 設定不可                   |
