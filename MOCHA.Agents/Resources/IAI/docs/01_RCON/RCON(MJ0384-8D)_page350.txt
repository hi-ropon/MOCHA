### テキスト
#### 4.16 Ethernetコネクターの接続
Ethernetコネクターの接続方法について説明します。

- コントローラー側コネクター正面図の写真と、ケーブル側コネクター正面図のイラストが掲載されています。

### 表
#### Ethernet接続コネクター仕様

| コネクター名称 | Ethernet接続コネクター | 備考 |
| :--- | :--- | :--- |
| ケーブル側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上シールド付 8P8Cモジュラープラグ（RJ-45） | 用意してください |
| コントローラー側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5以上シールド付 8P8Cモジュラージャック（RJ-45） |  |

#### ピン配列と信号内容

| ピン番号 | 信号名 | 内容 | 適合電線径 |
| :---: | :--- | :--- | :--- |
| 1 | TXP | 送信データ＋ | Ethernetケーブルは、カテゴリー5以上のストレートSTPケーブルを使用してください。 |
| 2 | TXN | 送信データ－ |  |
| 3 | RXP | 受信データ＋ |  |
| 4 | NC | 未接続 |  |
| 5 | NC | 未接続 |  |
| 6 | RXN | 受信データ－ |  |
| 7 | NC | 未接続 |  |
| 8 | NC | 未接続 |  |
