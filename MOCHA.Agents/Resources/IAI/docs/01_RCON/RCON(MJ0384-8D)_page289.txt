### テキスト
#### カテゴリー1（RCON-SC単相200V）を接続する場合
- 使用機器例：TB-02D（またはダミープラグDP-4S）
- 電磁接触器により24V系と200V系のモーター電源を遮断する構成
- ゲートウェイユニットの電源入力コネクターMP+24V、RCON-PS2のAC200Vを遮断
- 24Vドライバーユニットの駆動源遮断コネクターMPO、MPIを遮断した回路も可能
- RCON-PS2はAC200Vを遮断すること

### 図
```mermaid
flowchart TD
    A[TB-02D (またはダミープラグ DP-4S)] -->|CB-TB1-GC□□□| B[ポジションコントローラー用TPアダプター RCB-LB-TGS]
    B -->|接続ケーブル CB-CON-LB□□□| C[RCONシステム ゲートウェイユニット RCON-GWG]
    C -->|SIOコネクター| D[システムI/Oコネクター]

    C -->|モーター電源コネクター| F[RCON-PS2]
    F -->|MP+24V, 0V| G[電磁接触器]
    G -->|24V系モーター電源| H[モーター電源 AC200V]
    H -->|L1, L2| I[電磁接触器]
    I -->|200V系モーター電源| J[RCON-SC]
```

### 図の説明
- TB-02DまたはダミープラグDP-4Sからポジションコントローラー用TPアダプターRCB-LB-TGSへ接続
- RCB-LB-TGSはゲートウェイユニットRCON-GWGに接続され、システムI/Oコネクターを介して安全スイッチ（S1, S2, STOP+/-）と接続
- モーター電源はRCON-PS2を経由し、電磁接触器で24V系と200V系の電源を遮断
- 200V系モーター電源はRCON-SCに供給される
