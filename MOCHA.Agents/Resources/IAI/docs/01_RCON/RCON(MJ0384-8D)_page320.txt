### テキスト
#### EC接続ユニットへの配線（インターフェイスボックスを使用して接続）
インターフェイスボックスを使用してEC接続ユニットへ接続する場合、図に示すように配線を行います。

#### 注意
- コネクターには挿込み向きがあります。
- コネクターの形状に合わせ向きを揃え、クリック感があるまで挿入してください。

### 図
#### ケーブルの接続図
```mermaid
flowchart LR
    A[エレシリンダー] -->|アクチュエーターケーブル| B[インターフェイスボックス変換ケーブル]
    B -->|接続| C[インターフェイスボックス]
    C -->|電源・I/Oケーブル| D[RCON-EC]

    subgraph コネクター挿入指示
        direction TB
        A1["“カチッ”と音がするまで挿入してください"] --> A
        B1["“カチッ”と音がするまで挿入してください"] --> B
        C1["コネクターの溝を合わせて挿入してください"] --> C
        D1["“カチッ”と音がするまで挿入してください"] --> D
    end
```

**図の説明:**
- エレシリンダーからアクチュエーターケーブルを介してインターフェイスボックス変換ケーブルに接続
- インターフェイスボックス変換ケーブルからインターフェイスボックスへ接続
- インターフェイスボックスから電源・I/Oケーブルを通じてRCON-ECへ接続
- 各コネクターは「カチッ」という音がするまで確実に挿入し、向きを合わせる必要がある
</contents>
