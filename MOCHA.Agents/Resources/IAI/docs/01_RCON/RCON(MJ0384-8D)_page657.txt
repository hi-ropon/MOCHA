### テキスト
#### 衝突検出機能（パラメーターNo.168）
この機能はアクチュエーターが衝突した際に、衝突検出アラームを発生させ、移動を停止（サーボOFF）するものです。  
検出はポジションゾーンを設定した範囲内で行われます。  
詳細は「7.2 衝突検出機能」を参照してください。

### 表
#### 衝突検出機能パラメーター設定値一覧

| 設定値 | 内容                                                                                         | アラームレベル     |
| :----- | :------------------------------------------------------------------------------------------- | :----------------- |
| 0      | 検出を行いません（2、4、6を設定した場合も同様）                                             | －                 |
| 1      | ポジションゾーン設定範囲で検出を行います。                                                 |                   |
| 3 (注1) | ポジションゾーン設定範囲で検出を行いますが、以下の場合は検出しません。<br>・一時停止解除後の最初の移動<br>・ポジションゾーン範囲内で停止した状態からの移動 | 動作解除レベル     |
| 5      | ポジションゾーン設定範囲で検出を行います。                                                 |                   |
| 7 (注1) | ポジションゾーン設定範囲で検出を行いますが、以下の場合は検出しません。<br>・一時停止解除後の最初の移動<br>・ポジションゾーン範囲内で停止した状態からの移動 | メッセージレベル   |

注1：この設定により、加速時の電流値による誤検出を回避可能です。
