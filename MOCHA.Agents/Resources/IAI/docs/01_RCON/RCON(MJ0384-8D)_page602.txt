### テキスト
#### 【ステータスデータ】
モニター可能なステータスデータは以下の項目です。

### 表
| 項目                 | 内容                                                                                                   |
|----------------------|------------------------------------------------------------------------------------------------------|
| 出力電圧【V】         | 出力電圧のAD変換値。本電源は負荷に応じて出力電圧を変動させているため、モニター値が時々刻々と変化するが異常ではない。                     |
| 補助巻線電圧【V】     | 電源内部の制御電源電圧のAD変換値。出力電圧側の負荷に応じて出力電圧と同様に変化するが異常ではない。                                         |
| ピークホールド電圧【V】| 出力電圧のAD変換値。AD変換実行時に常に比較を行い、最大電圧値を保存。                                                               |
| 出力電流【A】         | 出力電流のAD変換値。瞬時値を表す。                                                                 |
| ピークホールド電流【A】| 出力電流のピーク値。AD変換実行時に常に比較を行い、最大電流値を保存。                                                                   |
| 負荷率【%】           | 出力電流の積分値と定格出力電流の比率を百分率で表した値。100%超過で過負荷エラーとなり出力電圧遮断。                                         |
| ピークホールド負荷率【%】| 負荷率計算時に常に比較を行い、最大負荷率を保存。                                                                                     |
| FAN回転速度【rpm】    | ファンから入力されるパルス信号よりファン回転数を算出した値。                                                                             |
| PCB温度【℃】          | 二次側の出力コンデンサー近傍の温度センサーのAD変換値。                                                                                 |
| 通算通電時間（日：時） | 通電時間をCPUでカウントし、定期的に内部メモリーに保持した値を積算した時間。                                                             |

### 画像説明
パソコン専用ティーチングソフトの電流モニター画面（型式：RCM-101-*-*の場合）のスクリーンショットが示されている。  
画面内の各ステータスデータの数値が表形式で表示されており、背景色は取得した電源ユニットのステータスにより変化する。  

#### 背景色の変化条件
- 補助巻線電圧（V）：15V以上は黄緑、10V以下は白  
- 負荷率（%）：330Wファン付きは黄緑、220Wファンなしは白  
- FAN回転速度：ファンなしはグレー、ファン正常時は白、警告時は黄色、異常時は赤  

操作方法は「パソコン専用ティーチングソフト取扱説明書（MJ0155）9.8 電源ユニット情報画面」または「IA-OS収録のガイド機能」を参照すること。
