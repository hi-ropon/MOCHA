### テキスト
#### 3.5.5 外形寸法
対象ユニットは以下の通りです。
- SCON拡張ユニット：RCON-EXT
- PIO/SIO/SCON拡張ユニット：RCON-EXT-NP/PN-*
- PIOユニット：RCON-NP/PN-*

※これらのユニットは外形寸法が同じです。

### 表
| 項目     | 仕様                                      |
| :------- | :---------------------------------------- |
| 外形寸法 | W22.6mm × H115mm × D95mm                  |
| 質量     | 約96g（SCON拡張ユニット）                  |
|          | 約110g（PIO/SIO/SCON拡張ユニット）         |
|          | 約105g（PIOユニット）                       |
| 外観図   | 下図参照                                  |
