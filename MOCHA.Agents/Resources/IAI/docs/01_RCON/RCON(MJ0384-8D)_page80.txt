### テキスト
#### 2.2.1 RCONシステムの電源仕様
RCONシステムの電源に関する仕様は以下の通りです。

（1）RCONシステム 各ユニットの制御電源

### 表
| 項目               | 仕様                                                                                     |
|--------------------|------------------------------------------------------------------------------------------|
| 電源電圧           | DC24V ±10%（ゲートウェイユニット）                                                     |
| 制御電源容量（ユニット1台当たり） |                                                                                          |
| ゲートウェイユニット（ターミナルユニット含む） | Ethernetオプション：なし 0.8A<br>Ethernetオプション：あり 1.0A                         |
| 24Vドライバーユニット（全タイプ共通） | ブレーキ：なし 0.2A<br>ブレーキ：あり（1軸仕様）0.4A<br>ブレーキ：あり（2軸仕様）0.6A |
| 200Vドライバーユニット（200V電源ユニット含む） | ブレーキ：なし 0.2A<br>ブレーキ：あり 0.5A                                            |
| SCON拡張ユニット   | 0.1A                                                                                     |
| PIO/SIO/SCON拡張ユニット | 0.1A                                                                                     |
| PIOユニット        | 0.1A                                                                                     |
| EC接続ユニット     | 0.1A                                                                                     |
| 簡易アブソユニット（全タイプ共通） | 0.2A                                                                                     |

### 注釈
- 200V電源ユニットで消費する制御電源電流はわずかであるため、計算上考慮する必要はありません。
- ファンユニット（RCON-FU/RCON-FUH）の電源は、接続する各ユニットの制御電源容量に含まれており、約50mA/台です。
