### テキスト
#### 軸番号設定の追加・削除後の設定手順
5）追加または削除後の軸番号設定を行う。  
以下の選択項目から選択し、「完了」ボタンを押す。

### 表
#### 軸数ごとの選択項目と軸番号設定変更結果

| 軸数   | 選択項目           | 軸番号の設定変更結果                                           |
| :----- | :----------------- | :------------------------------------------------------------- |
| 追加時 | シフト             | 追加した分のドライバーの軸番号を前からシフトする。             |
|        | 未割当番号         | 追加したドライバーに未割当の番号を設定する。                   |
| 削除時 | シフト             | 削除した分のドライバーの軸番号を前からシフトする。             |
|        | 予約ユニットとして残す | 編集したユニットの後ろに予約ユニットを追加し軸番号を残す。     |

### 画像説明
- 上部の2つのスクリーンショットは、軸数追加時（左）と削除時（右）の軸番号設定画面。  
  「追加後の軸番号設定」または「削除後の軸番号設定」の選択肢が表示されている。  
  それぞれ「シフト」「未割当番号」や「予約ユニットとして残す」が選べる。

- 中段の画像は「Unit1にDrv1を追加し、シフトを選択した場合」の軸番号の変化を示す。  
  追加前はUnit1のDrv1が3番、追加後は2番にシフトしている。

- 下段の画像は「Unit1にDrv1を追加し、未割当番号を選択した場合」の軸番号の変化を示す。  
  追加前はUnit1のDrv1が3番、追加後は6番に未割当番号が割り当てられている。
