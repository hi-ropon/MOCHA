### テキスト
#### 停止入力信号のテストパルス
外部から入力されるテストパルスのオフ時間は1ms以下にしてください。

#### 動作シーケンス（通常動作）
停止入力信号1（/SRI1）および停止入力信号2（/SRI2）がON/OFFすることでDRV STOP状態の解除や保持が行われます。  
モーターへのエネルギー供給信号と外部機器モニター出力信号（EDM）のON/OFF状態も連動し、レスポンスタイムは8ms以下です。  
モーター回転数は内部回路遅延時間（数ms）を経てダイナミックブレーキにより減速します。

| 項目                         | 仕様           | 備考               |
|------------------------------|----------------|--------------------|
| 停止入力信号（SRI）          |                |                    |
| オン入力電圧範囲             | 24V±10%        |                    |
| オフ入力電圧範囲             | 0-2V           |                    |
| 入力電流                     | 7.6mA (Typ)    | 1ch当たりの値です。 |
| レスポンスタイム             | 8ms以下        |                    |
| 外部機器モニター出力信号（EDM） |                |                    |
| 電圧範囲                     | 24V±10%        |                    |
| 出力電流                     | 100mA (Max)    |                    |
