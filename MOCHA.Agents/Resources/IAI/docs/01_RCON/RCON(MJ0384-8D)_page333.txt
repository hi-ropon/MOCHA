### テキスト
#### 4.11 外付け回生抵抗ユニットの配線
回生抵抗ユニット付属のケーブルを用いて、以下のように回生抵抗ユニットを接続します。  
1) 1台接続の場合：RESU(D)-2を付属ケーブル（CB-SC-REU）で接続  
2) 2台目以降の接続：RESU(D)-1を付属ケーブル（CB-ST-REU）で接続  

接続ケーブルの詳細は次ページに掲載されています。

### 図
```mermaid
flowchart LR
    A[200V電源ユニット\n外付け回生抵抗コネクター] -->|1) CB-SC-REU| B[RESU-2\nRESUD-2]
    B -->|RB IN| C[RESU-1\nRESUD-1]
    B -->|RB OUT| C
    B -.->|2) CB-ST-REU| C
```
**図の説明:**  
- 電源ユニットの外付け回生抵抗コネクターからRESU-2（RESUD-2）へ付属ケーブル（CB-SC-REU）で接続（1台目）  
- 2台目以降はRESU-2のRB OUTからRESU-1のRB INへ付属ケーブル（CB-ST-REU）で接続
