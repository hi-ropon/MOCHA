### 表
#### 【入出力信号一覧】簡易直値モード（ON=該当ビットが"1"、OFF=該当ビットが"0"）

| 信号種類           | ビット       | 信号名 | 内容                                                                                         | 詳細   |
|--------------------|--------------|--------|----------------------------------------------------------------------------------------------|--------|
| 位置データ指定     | 32ビットデータ | －      | 32ビット符号付整数（単位：0.01mm、ただしDDモーターの場合は0.001度）<br>例）+25.40mmならば2540と設定<br>・設定最大値は+9,999.99mm=999999<br>・16進数で入力する場合、負数は2の補数で指定 | 6.5.2  |
| 指令ポジションNo. | b6-b0        | PC＊＊  | 位置データ指定以外の移動データを、ポジションテーブルで設定。ただしポジションNo.を2進数で指定 | 6.5.2  |
| 制御信号           | b15          | BKRL   | ブレーキ強制解除<br>【ON：ブレーキ強制、OFF：ブレーキ有効】                                   | 6.3(16)|
|                    | b14～b9      | －      | 使用できません                                                                              | －      |
|                    | b8           | JOG+   | +ジョグ<br>【ON：反原点方向移動、OFF：停止】                                               | 6.3(11)|
|                    | b7           | JOG-   | -ジョグ<br>【ON：原点方向移動、OFF：停止】                                                 | 6.3(11)|
|                    | b6           | JVEL   | ジョグ速度/インチング距離切替え<br>【OFF：RCONのパラメーターNo.26“ジョグ速度”、No.48“インチング距離の設定値”を使用<br>ON：RCONのパラメーターNo.47“ジョグ速度2”、No.49“インチング距離2の設定値”を使用】 | 6.3(12)|
|                    | b5           | JISL   | ジョグ/インチング切替え<br>【ON：インチング、OFF：ジョグ】                               | 6.3(13)|
|                    | b4           | SON    | サーボON<br>【ON：サーボON、OFF：サーボOFF】                                               | 6.3(5) |
|                    | b3           | RES    | リセット<br>【ONでリセット実行】                                                          | 6.3(4) |
|                    | b2           | STP    | 一時停止<br>【ON：一時停止、OFF：一時停止解除】                                           | 6.3(9) |
|                    | b1           | HOME   | 原点復帰<br>【ONで原点復帰指令、途中でOFFしても完了まで実行】                             | 6.3(6) |
|                    | b0           | CSTR   | 位置決めスタート<br>【ONで移動指令実行、途中でOFFしても完了まで実行】                     | 6.3(7) |

**説明:**
- 「位置データ指定」は32ビットの符号付整数で位置を指定し、DDモーターの場合は単位が異なる。
- 「指令ポジションNo.」は位置データ指定以外の移動データをポジションテーブルで指定するためのビット。
- 「制御信号」は各ビットに対して特定の制御機能が割り当てられており、ON/OFFで動作を制御する。
- 使用できないビット（b14～b9）は空白で示されている。
