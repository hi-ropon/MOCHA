### テキスト
#### 4.6 安全カテゴリーへの対応について
【1】-3 カテゴリー1（SCON拡張ユニットを使用してSCONを接続する場合）

- 電磁接触器によって、RCONとSCONのモーター電源を遮断する構成例を示しています。
- 本回路例では、RCONの回路はゲートウェイユニットの電源入力コネクターMP+24Vを遮断しています。
- 各ドライバーユニットの駆動源遮断コネクターMPO、MPIを遮断した回路とすることも可能です。

---

### 図（回路図）

```mermaid
flowchart TD
    A[TB-02D (またはダミープラグ：DP-4S)] -->|接続ケーブルCB-CON-LB□□□| B[ポジションコントローラー用TPアダプター RCB-LB-TGS]
    B -->|ENBOUT, ENB1+, ENB1-, ENB2+, ENB2-, ENB3+, ENB3-, EMGOUT, EMG1+, EMG1-, EMG2+, EMG2-, EMG3+, EMG3-| C[RCONシステム ゲートウェイユニット RCON-GWG]
    C -->|SIOコネクター| D[システムI/Oコネクター]
    D -->|S1, S2, STOP+, STOP-| E[モーター電源コネクター]
    E -->|モーター電源| F[モーター]
    subgraph 電磁接触器回路
        G1[電磁接触器] -.->|24V| B
        G2[電磁接触器] -.->|AC100V/AC200V| F
    end
    subgraph SCON接続
        H1[SCON] -->|IN| I1[モーター電源]
        H1 -->|OUT| I2[モーター電源]
        H2[SCON] -->|IN| I3[モーター電源]
        H2 -->|OUT| I4[モーター電源]
    end
```

**図の説明:**
- TB-02DまたはダミープラグDP-4Sからポジションコントローラー用TPアダプター（RCB-LB-TGS）へ接続ケーブルで接続。
- RCB-LB-TGSは複数の信号線（ENB、EMGなど）を持ち、ゲートウェイユニット（RCON-GWG）へ接続。
- ゲートウェイユニットはシステムI/Oコネクター、モーター電源コネクターを介してモーターへ電源供給。
- 電磁接触器により24V電源とモーター電源（AC100V/AC200V）を遮断可能。
- SCONユニットがモーター電源のIN/OUTに接続されている。

---

### 注意事項
- 電磁接触器によってRCONとSCONのモーター電源を遮断することが可能。
- 本例ではゲートウェイユニットの電源入力コネクターMP+24Vを遮断しているが、各ドライバーユニットの駆動源遮断コネクターMPO、MPIを遮断する回路も可能。
