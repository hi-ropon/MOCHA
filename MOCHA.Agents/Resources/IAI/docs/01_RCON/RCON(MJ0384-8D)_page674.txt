### テキスト
#### 8.4.2 ネットワーク設定
ネットワーク設定画面の内容について説明しています。

##### 【1】通信設定
使用するドライバーやネットワークにより表示内容が異なるため、マスターに合わせて設定を行う必要があります。

【CC-Linkの場合】
- ネットワークタイプ：CC-Link（変更不可）
- アドレス：1～64の範囲で設定可能（初期値は1）
- 通信速度：156kbps、625kbps、2.5Mbps、5Mbps、10Mbpsから選択可能（初期値は156kbps）

### 画像説明
ネットワーク設定画面は3つの枠に分かれている。

```plaintext
┌───────────────┐ ┌───────────────┐
│ ① 通信設定    │ │ ② 軸制御設定  │
└───────────────┘ └───────────────┘
          ↓
┌─────────────────────────────┐
│ ③ 占有情報（マスター占有情報）│
└─────────────────────────────┘
```

- 【1】通信設定：ネットワークタイプ、アドレス、通信速度の設定
- 【2】軸制御設定：軸番号やユニット構成設定、動作モード設定
- 【3】占有情報：マスター占有情報の表示

### 表
| 項目           | 初期設定 | 設定可能範囲                                   |
| :------------- | :------- | :--------------------------------------------- |
| ネットワークタイプ | CC-Link  | 変更不可                                      |
| アドレス       | 1        | 1～64                                         |
| 通信速度       | 156kbps  | 156kbps / 625kbps / 2.5Mbps / 5Mbps / 10Mbps |
