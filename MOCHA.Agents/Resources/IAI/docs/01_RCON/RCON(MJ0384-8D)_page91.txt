### テキスト
#### EC接続ユニットへの高推力EC接続数制限について
高推力ECをEC接続ユニット（RCON-EC-4）1台に接続できる軸数は最大2軸までです。また、高推力ECの接続軸数によって、高推力以外のエレシリンダーの接続可能な軸数も変わります。

### 表1：高推力EC接続例

| 高推力EC接続軸数 | RCON-EC-4（1ユニット）への高推力EC接続可否 | 高推力EC以外のエレシリンダー接続可能軸数 |
| :--------------- | :------------------------------------------ | :---------------------------------------- |
| 1軸の場合       | 接続可能                                   | 3軸まで接続可能                           |
| 2軸の場合       | 接続可能                                   | 接続できません                           |
| 3軸以上         | 接続不可                                   | －                                       |
