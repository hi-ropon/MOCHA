### テキスト
#### 6.2 各アドレスの割付け内容
##### 6.2.1 固定領域の制御・状態信号
1. ゲートウェイユニットの制御信号・状態信号について説明。
2. ゲートウェイユニットのアドレス構成では、最初の入出力各2ワードが制御や状態監視のための信号に割り当てられている。
3. アドレスはゲートウェイ先頭からの相対アドレスである。
4. CC-Link、CC-Link IE Field、CC-Link IE TSNとDeviceNetはワードアドレス、PROFIBUS-DP、EtherNet/IP、EtherCATはバイトアドレス、PROFINET-IOは4ワードのモジュールアドレスである。
5. CC-Link系のビットレジスターアドレスの「＊」は0～Fの範囲。
6. CC-Link、CC-Link IE Fieldのb10～b15は16進でbA～bFとなる。
7. PROFIBUS-DP、EtherNet/IP、EtherCATのb8～b15はb0～b7となる（バイトアドレスのため）。
