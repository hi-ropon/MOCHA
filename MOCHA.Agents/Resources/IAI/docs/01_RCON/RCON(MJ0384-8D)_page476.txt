### テキスト
#### 6.2.7 ポジショナー5モードの割付け
ポジショナー5モードの割付けは以下のようになります。

PLC出力＝軸制御信号のアドレス割付けは、CC-Link、DeviceNet、PROFIBUS-DP、EtherNet/IP、EtherCAT、PROFINET-IOごとに異なり、PROFINET-IOは4ワードモジュールのアドレスとなっています。

PLC入力＝軸状態信号のアドレス割付けも同様に各通信方式ごとに割付けられています。

※mは各軸の先頭レジスターアドレス、nは各軸の先頭の相対アドレス、pは各軸の先頭のモジュールアドレスを示します。

#### 注意
- ポジショナー5モードで出力できる現在位置データは最大で3,276.7mm（DDモーターの場合は327.67度）です。
- 最大値を超えた場合、現在位置データが正しく出力されず、誤表示や誤動作を引き起こし、人身事故や装置破損の恐れがあります。
- 最大値を超える動作範囲で使用する場合は、ほかの動作モードを選択してください。

### 表（アドレス割付け概要）

| 信号種別       | 通信方式                         | アドレス形式               | 備考                         |
| -------------- | -------------------------------- | -------------------------- | ---------------------------- |
| PLC出力軸制御信号 | CC-Link, CC-Link IE Field, CC-Link IE TSN, DeviceNet | ワードアドレス             | 各軸の先頭レジスターアドレスm+0、m+1に割付け |
|                | PROFIBUS-DP, EtherNet/IP, EtherCAT | バイトアドレス             | 上位バイトn+0、下位バイトn+1に割付け       |
|                | PROFINET-IO                      | 4ワードモジュールアドレス   | p+0、p+1に割付け             |
| PLC入力軸状態信号 | CC-Link, CC-Link IE Field, CC-Link IE TSN, DeviceNet | ワードアドレス             | 同上                         |
|                | PROFIBUS-DP, EtherNet/IP, EtherCAT | バイトアドレス             | 同上                         |
|                | PROFINET-IO                      | 4ワードモジュールアドレス   | 同上                         |
