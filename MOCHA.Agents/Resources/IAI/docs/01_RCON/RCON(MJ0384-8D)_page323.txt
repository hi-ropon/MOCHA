### テキスト
#### モーター駆動用 DC 電源の電源配線に使用する電線の線径
電源コネクターに配線する電線は、下記の適合電線を使用してください。

※絶縁被覆の温度定格が60℃以上の電線を使用してください。

⑤ モーター駆動用 DC 電源に電源コネクターを挿込み、コネクターのフランジ部分にある固定ねじをマイナスドライバーで締込みます。

### 表
#### 適合電線

| コネクター | 端子名 | 適合電線 線径         |
| :--------- | :----- | :-------------------- |
| （画像あり） | PE     | AWG14（銅線）         |
|            | NC     | 配線しない             |
|            | L2C    | AWG18（銅線）         |
|            | L1C    | AWG18（銅線）         |
|            | L2     | AWG14（銅線）         |
|            | L1     | AWG14（銅線）         |

### 画像説明
- モーター駆動用 DC 電源（型式：PSA-200-*）の電源コネクター（型式：MSTB2.5/6-STF-5.08）を示す写真
- 電源コネクターの端子に電線を挿入し、マイナスドライバーで固定ねじを締める作業を示す拡大写真
- 「マイナスドライバーで締込む」という吹き出しで締め付け方法を説明
