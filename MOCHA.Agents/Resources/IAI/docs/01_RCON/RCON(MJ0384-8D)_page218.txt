### テキスト
#### EC接続ユニット LED表示
EC接続ユニットのLED表示について説明しています。  
LEDは各種状態を色と点灯・点滅で示し、通信状態や異常、動作状態を視覚的に確認できます。

### 表
#### LED表示の状態一覧

| パネル表記 | 表示色 | 状態   | 説明                                                                                   |
|------------|--------|--------|----------------------------------------------------------------------------------------|
| T RUN      | 緑     | 点灯   | 内部バス正常通信中                                                                     |
|            | 緑     | 点滅   | 初期化通信待ち、初期化通信失敗                                                       |
|            | 橙     | 点灯   | 内部バス通信異常発生                                                                 |
| SYS        | 緑     | 点灯   | 正常動作中                                                                           |
|            | 緑     | 消灯   | 制御電源電圧低下、モーター電源電圧低下、非常停止、ECとの定常通信エラー発生           |
|            | 赤     | 点灯   | 制御電源電圧低下、モーター電源電圧低下、非常停止、ECとの定常通信エラー発生           |
|            | 赤     | 消灯   | 正常動作中                                                                           |
| LS1/ALM    | 緑     | 点灯   | 前進端へ移動完了                                                                     |
| (I=1軸目、 | 緑     | 消灯   | 停止                                                                                 |
| II=2軸目、 | 赤     | 点灯   | アラーム発生中                                                                       |
| III=3軸目、|        |        |                                                                                        |
| IV=4軸目)  |        |        |                                                                                        |
| LS0        | 緑     | 点灯   | 後退端へ移動完了                                                                     |
| (I=1軸目、 | 緑     | 消灯   | 停止                                                                                 |
| II=2軸目、 |        |        |                                                                                        |
| III=3軸目、|        |        |                                                                                        |
| IV=4軸目)  |        |        |                                                                                        |

### 画像説明
写真はEC接続ユニットのLED部分を示しており、LS0、LS1/ALM、T RUN、SYSの各LEDが確認できます。  
各LEDの点灯・点滅状態により、通信状態や異常、動作状態を判別可能です。
