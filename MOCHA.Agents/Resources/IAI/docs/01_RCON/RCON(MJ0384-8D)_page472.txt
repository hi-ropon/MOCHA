### テキスト
#### 6.2.5 ポジショナー2モードの割付け
ポジショナー2モードの割付けは以下のように定義されています。

- PLC出力は軸制御信号で、CC-Link、DeviceNet、PROFIBUS-DP、PROFINET-IOなどの各通信方式に対応したアドレス割付けが示されています。
- PLC入力は軸状態信号で、同様に各通信方式に対応したアドレス割付けが示されています。

※mは各軸の先頭レジスタアドレス、nは各軸の先頭の相対アドレス、pは各軸の先頭のモジュールアドレスを表します。

CC-Link系とDeviceNetはワードアドレス、PROFIBUS-DP、EtherNet/IP、EtherCATはバイトアドレス、PROFINET-IOは4ワードモジュールのアドレスとなっています。

入出力信号一覧やアラーム内容一覧はポジショナー1モードの「現在位置データ」項目を除き、ほぼ同じです。ポジショナー1モードのPLC入力信号には「現在位置データ」が含まれますが、ポジショナー2モードには含まれません。

詳細は「6.2.4 ポジショナー1モードの割付け」を参照してください。

### 表（アドレス説明）

| 記号 | 意味 |
| :--- | :--- |

| n | 各軸の先頭の相対アドレス |
| p | 各軸の先頭のモジュールアドレス |

---

**備考**
- CC-Link、CC-Link IE Field、CC-Link IE TSN、DeviceNetはワードアドレス
- PROFIBUS-DP、EtherNet/IP、EtherCATはバイトアドレス
- PROFINET-IOは4ワードモジュールのアドレス
