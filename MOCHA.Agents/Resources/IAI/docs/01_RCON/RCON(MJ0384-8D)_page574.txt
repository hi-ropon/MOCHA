### テキスト
#### 入出力信号のタイミングと運転手順
- 動作途中でST信号をOFFにすると、減速を開始し途中で停止する。  
- 例として、LS信号の検出範囲を大きくした場合の注意点が示されている。  
- LS信号がONした直後にST信号をOFFにするシーケンスの場合、エレシリンダーが前進端または後退端に到達していない可能性がある。

#### 注意
- 減速停止後に再びST信号をONにすると、エレシリンダーは動作を再開する。  
- 検出範囲に入ると、エレシリンダーが動作中でもLS信号はONになる。  
- “ST0”信号と“ST1”信号を同時にONにしないこと。予期せぬ動作の恐れがある。

```mermaid
flowchart LR
    subgraph 信号と動作
        A[ST信号] -->|OFF| B[減速開始]
        B --> C[途中停止]
        D[LS信号 ON] --> E[ST信号 OFF]
        F[エレシリンダーの動き] -->|動作途中で減速開始| C
        C -->|後退端到着前に停止| G[停止位置]

    subgraph 検出範囲
        H[検出範囲] -.-> E
    end
    G -.-> I[本来の減速開始位置]
```

### 図の説明
- **ST信号**: 動作途中でOFFにすると減速開始し途中で停止する。  
- **LS信号**: ONになるとST信号をOFFにするシーケンスがある。  
- **エレシリンダーの動き**: 減速開始後、後退端到着前に停止する。  
- **検出範囲**: LS信号の検出範囲が大きいと、本来の減速開始位置より早く停止する可能性がある。
