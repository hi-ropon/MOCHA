### テキスト
#### EtherNet/IP 通信設定
使用するマスターに合わせて以下の項目を設定する必要があります。

- ネットワークタイプ: EtherNet/IP
- IPアドレス
- サブネットマスク
- デフォルトゲートウェイ
- 通信速度（自動設定）

#### EtherNet/IP（フロントポート仕様）通信設定
フロントポート仕様のEtherNet/IP設定。  
MACアドレスが製品固有で設定不可となっている点が特徴。

---

### 表

#### EtherNet/IP 通信設定項目

| 項目             | 初期値           | 設定可能範囲                     |
| :--------------- | :--------------- | :------------------------------- |
| ネットワークタイプ | EtherNet/IP      | 変更不可                       |
| IPアドレス       | 192.168.0.1      | 0.0.0.0～255.255.255.255       |
| サブネットマスク | 255.255.255.0    | 0.0.0.0～255.255.255.255       |
| デフォルトゲートウェイ | 0.0.0.0          | 0.0.0.0～255.255.255.255       |
| 通信速度         | 自動             | 設定不要（自動ネゴシエーション） |

#### EtherNet/IP（フロントポート仕様）通信設定項目

| 項目             | 初期値                   | 設定可能範囲                     |
| :--------------- | :----------------------- | :------------------------------- |
| ネットワークタイプ | EtherNet/IP(Front Port)  | 変更不可                       |
| IPアドレス       | 192.168.0.1              | 0.0.0.0～255.255.255.255       |
| サブネットマスク | 255.255.255.0            | 0.0.0.0～255.255.255.255       |
| デフォルトゲートウェイ | 0.0.0.0                  | 0.0.0.0～255.255.255.255       |
| 通信速度         | 自動                     | 設定不要（自動ネゴシエーション） |
| MACアドレス      | 製品固有                 | 設定不可                       |

---

### 備考
- MACアドレスはゲートウェイユニットに割り付けられている固有の識別番号であり、設定不可。
