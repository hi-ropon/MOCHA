### テキスト
#### 6.1.1 固定領域の構成
PLC出力からRCONへのデータ領域と、RCONからPLC入力へのデータ領域の構成について説明しています。

- 電源ユニットを接続していない場合でも、データ領域として占有されることが記載されています。
- 注釈として、1本信号はゲートウェイパラメーターによる切替えで機能を変更可能であることが示されています。

### 表
#### 固定領域の構成（PLC出力⇒RCON / RCON⇒PLC入力）

| 領域           | 上位バイト           | 下位バイト           | ワード数 | 上位バイト           | 下位バイト           | ワード数 |
| :------------- | :------------------- | :------------------- | :------- | :------------------- | :------------------- | :------- |
| **ゲートウェイ領域** | ゲートウェイ制御信号0 | ゲートウェイ制御信号1 | 2        | ゲートウェイ状態信号0 | ゲートウェイ状態信号1 | 2        |
| **電源ユニット領域** | 使用できません       | 使用できません       | 6        | 電源ユニット状態信号0 | 電源ユニット状態信号1 | 6        |
|                | 使用できません       | 使用できません       |          | 電源ユニット状態信号2 |                      |          |
|                | 使用できません       | 使用できません       |          | 電源ユニット状態信号3 |                      |          |
|                | 使用できません       | 使用できません       |          | 電源ユニット状態信号4 / 軽故障メンテナンス信号（注1） | 使用できません |          |

### 注釈
- 電源ユニットを接続していない場合でも、データ領域として占有されます。
- 1本信号はゲートウェイパラメーターによる切替えで機能を変更可能です。
