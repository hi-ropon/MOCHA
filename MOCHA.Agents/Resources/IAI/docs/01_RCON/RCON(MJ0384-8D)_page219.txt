### テキスト
#### ジョグスイッチ
ジョグ運転用のスイッチで、上から1軸目、2軸目、3軸目、4軸目となります。  
RCONゲートウェイパラメーターの特殊パラメーターにある「RCON-EC JOGスイッチ」を「1：有効」に設定すると、ゲートウェイユニットに接続中の全EC接続ユニットのジョグスイッチが有効になります（初期値は「0：無効」）。  

JOG+側にスイッチを傾けると＋方向（ポジションデータの前進端に登録された目標位置）に、JOG−側に傾けると−方向（ポジションデータの後退端に登録された目標位置）にジョグ動作を行います。ジョグ速度はポジションデータに登録された速度です。途中でスイッチを離した場合は、ポジションデータに登録された減速度で停止します。ただし、原点復帰未完了の場合は、どちらのスイッチを傾けても原点復帰動作を行い、スイッチを離すと原点復帰動作を中断します。  

ジョグスイッチの操作はMANUモードの場合のみ有効で、AUTOモード時は無効です。また、ティーチングツールでアクチュエーターを操作できる画面を開いている場合もジョグスイッチは無効です。ジョグスイッチによる動作中に操作できる画面を開いた場合、アクチュエーターは減速停止します。

### 表
| 記号  | 説明                                                                                   |
| :---- | :------------------------------------------------------------------------------------- |
| JOG+  | ＋方向（ポジションデータの前進端に登録された目標位置）にジョグ動作                     |
| JOG−  | －方向（ポジションデータの後退端に登録された目標位置）にジョグ動作                     |

### 注意
- ティーチングツールでアクチュエーターを操作できる画面を開いているときに、ティーチングツールとの通信が切断された場合、ジョグスイッチの操作が無効になったままになります。  
- ジョグスイッチの操作を有効に戻すには、RCONシステムの電源を再投入するか、ソフトウェアリセットを行ってください。
