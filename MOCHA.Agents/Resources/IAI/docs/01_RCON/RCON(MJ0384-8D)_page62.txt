#### 1.8 ユニット接続制限事項

##### 1.8.1 モーター電源・制御電源容量の制限
- 詳細は「2.2.3 ユニット接続制限」を参照するよう案内。

##### 1.8.2 ユニット配置制限
- RCONシステムはロック構造のユニット連結方式で、同じ連結コネクターを持つユニット同士は自由な順番で連結可能。
- ただし、以下のユニットには配置上の制限がある。

| ユニット名               | 配置制限内容                                                                                         |
|--------------------------|----------------------------------------------------------------------------------------------------|
| ゲートウェイユニット     | RCONシステムの左端に配置                                                                           |
| SCON拡張ユニット         | ゲートウェイユニットの右隣り<br>（PIO/SIO/SCON拡張ユニットと同時使用不可）                         |
| PIO/SIO/SCON拡張ユニット | ゲートウェイユニットの右隣り<br>（SCON拡張ユニットと同時使用不可）                               |
| PIOユニット              | ゲートウェイユニット、SCON拡張ユニット、もしくはPIO/SIO/SCON拡張ユニットのうち一番右側のユニットの右隣り<br>（最大8台まで接続可能） |
| 24Vドライバーユニット    | ゲートウェイユニット、SCON拡張ユニット、PIO/SIO/SCON拡張ユニット、PIOユニットより右側<br>（パルスモータータイプ、ACサーボモータータイプ、DCブラシレスモータータイプの順番制限なし） |
| EC接続ユニット           | ゲートウェイユニット、SCON拡張ユニット、PIO/SIO/SCON拡張ユニット、PIOユニット、24Vドライバーユニットより右側<br>（最大4ユニットまで接続可能） |
| 200V電源ユニット         | SCON拡張ユニット、24Vドライバーユニット（RCON-PC-*/AC-*/DC-*）とEC接続ユニット（RCON-EC-4）の右隣り<br>SCON拡張ユニット、24Vドライバーユニット、EC接続ユニットがない場合はゲートウェイユニットの右隣り |

- 各ユニットの配置順序や同時使用不可の組み合わせに注意が必要。
