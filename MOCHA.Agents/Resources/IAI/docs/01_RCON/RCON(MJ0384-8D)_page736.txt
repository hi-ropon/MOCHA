### 表
#### ゲートウェイユニットのアラームの原因と対策一覧

| アラームコード | アラーム名称               | 原因/対策                                                                                                                                                                                                                      |
| :------------- | :------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| BA (8BA)       | 軸数/動作モード不一致     | 原因：<br>1) ゲートウェイのパラメータに設定されている軸数と、ゲートウェイに設定されている動作モードから算出した軸数が一致していない<br>2) ドライバーユニットを17軸以上接続している<br>対策：<br>1) パラメーター設定値が不正なので、ゲートウェイのパラメーターを設定しなおす<br>2) ドライバーユニットを16軸以下にして、適切なゲートウェイパラメーターを設定する |
| BC (8BC)       | 付属軸通信異常             | 原因：トータルフレーム通信の初期通信時に通信異常が発生している。<br>1) 連結コネクターまたはSCON接続コネクターが正しく接続されていない<br>2) 内部の信号線またはSCON接続ケーブルが断線している<br>3) ターミナルユニットまたはターミナルコネクター（SCON用）を取付けていない<br>4) ノイズの影響による通信異常<br>対策：<br>1) 2) 各ユニット同士がしっかり連結されているか確認し、再度連結しなおす<br>3) ターミナルユニットまたはターミナルコネクターを取付ける<br>4) ノイズ対策を行う |
| BE (8BE)       | オプションユニット初期通信異常 | 原因：オプションユニット（EC接続ユニット）宛のトータルフレーム通信の初期通信時に通信異常が発生している。<br>1) 連結コネクターが正しく接続されていない<br>2) 内部の信号線が断線している<br>3) ターミナルユニットを取付けていない<br>4) ノイズの影響による通信異常<br>対策：<br>1) 2) 各ユニット同士がしっかり連結されているか確認し、再度連結しなおす<br>3) ターミナルユニットを取付ける<br>4) ノイズ対策を行う |
| D0 (8D0)       | オプションユニット接続台数異常 | 原因：オプションユニット（EC接続ユニット）の接続ユニット数が、ゲートウェイパラメーター設定上の接続ユニット数と異なる。<br>対策：設定上のオプションユニット数とEC接続ユニット数を合わせる。                                                                                     |
| D1 (8D1)       | 非対応オプションユニット接続 | 原因：非対応のオプションユニットを接続した。<br>対策：対応しているオプションユニットを接続する。                                                                                                                             |
| DD (8DD)       | ユニット連結確認信号異常   | 原因：各ユニットが正常に連結されていない可能性がある。<br>1) 連結コネクターまたはSCON接続コネクターが正しく接続されていない<br>2) 内部の信号線またはSCON接続ケーブルが断線している<br>3) ターミナルユニットまたはターミナルコネクター（SCON用）を取付けていない<br>対策：<br>1) 2) 各ユニット同士がしっかり連結されているか確認し、再度連結しなおす<br>3) ターミナルユニットまたはターミナルコネクターを取付ける。 |
