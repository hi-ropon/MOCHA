### テキスト
#### 3.2.2 型式
ドライバーユニットの型式は以下の通りです。  
RCON-PC/AC/DCは1軸仕様または2軸仕様を選択可能。  
RCON-PCFは1軸仕様のみ。

（1）型式の見方

【24Vドライバーユニット】  
型式は「RCON - タイプ - 軸数」で表される。

【ファンユニット】  
型式は「RCON - FU」

### 図
```mermaid
flowchart LR
    A[RCON] --> B[タイプ]
    B --> PC[PC: パルスモータータイプ]
    B --> PCF[PCF: パルスモーター56SP/60P/86P対応タイプ]
    B --> AC[AC: ACサーボモータータイプ]
    B --> DC[DC: DCブラシレスモータータイプ]
    A --> C[軸数]
    C --> "1"[1軸仕様]
    C --> "2"[2軸仕様]
```

**図の説明:**  
- タイプは4種類（PC、PCF、AC、DC）  
- 軸数は1または2軸仕様（PCFは1軸のみ）
