### テキスト
#### 〔4〕押付け方向の設定を間違えた場合
押付け方向の設定を誤ると、動作が「位置決め幅×2」分ずれてしまうため注意が必要である。

#### 〔5〕押付け後の戻り動作を相対座標指定で行った場合
相対座標指定の基準位置は、押付け完了して停止している現在位置ではなく、押付けを実行したポジションNo.の目標位置となる。  
例として、ポジションNo.を相対座標の-40mmに設定すると、280mmの目標位置から40mm引いた240mmの位置に移動する。ただし押付け指定している場合は停止位置からの相対移動となるため注意が必要。

### 図

```mermaid
flowchart LR

    B --> C[位置決め幅]
    C --> D[位置決め幅]
    D --> E[位置決め幅]
    subgraph 実際の動作
        B -->|実際の動作| C
    end
    subgraph 意図した動作
        E -.->|意図した動作| F[目標位置]
    end
    C --> G[実際に移動する位置（空振り）]
```
**図の説明:**  
速度が一定に上昇し、移動距離を進むが、押付け方向の設定ミスにより実際の動作は位置決め幅の2倍分ずれてしまう。実際に移動する位置は目標位置より手前にずれ、空振りが発生する。

#### 押付け後の戻り動作を相対座標指定で行った場合の速度-位置グラフ
```mermaid

    A[速度] --> B[240mm（戻り動作）]
    B --> C[280mm（目標位置）]
    C --> D[押付け完了位置]
```
**図の説明:**  
速度が一定に上昇し、240mmの位置まで戻り動作を行い、その後280mmの目標位置を経て押付け完了位置に至る。
