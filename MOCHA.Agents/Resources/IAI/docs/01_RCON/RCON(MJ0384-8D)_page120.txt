### テキスト
#### システムI/Oコネクター
外部からのAUTO/MANU切替入力、STOP入力、PSA-24とのシリアル通信用のコネクターです。  
出荷時は以下のピンが短絡されています。  
- 1番ピンと6番ピン  
- 2番ピンと7番ピン  
- 3番ピンと8番ピン  

### 画像
システムI/Oコネクターのピン配置写真（ピン番号が表示されている）

### 表
#### 基板側・ケーブル側コネクター型番

| 種類           | 型番                                   |
| :------------- | :------------------------------------ |
| 基板側コネクター | DMC1.5/5-G1-3.5P20THR（フェニックス・コンタクト） |
| ケーブル側コネクター | DFMC1.5/5-ST-3.5（フェニックス・コンタクト）       |

#### ピン番号・信号名・説明

| ピン番号 | 信号名       | 説明                                   |
| :------- | :----------- | :------------------------------------ |
| 1        | S2           | ティーチングボックス用STOPスイッチ接点入力 |
| 2        | S1           | ティーチングボックス用STOPスイッチ接点出力 |
| 3        | AUTO/MANU－  | AUTO/MANU信号入力                      |
| 4        | GND          | 0V                                    |
| 5        | PS24_SD－    | PSA-24用RS-485差動信号－側            |
| 6        | STOP＋       | STOP信号用＋24V電源出力                |
| 7        | STOP－       | STOP信号入力                          |
| 8        | AUTO/MANU＋  | AUTO/MANU用電源                       |
| 9        | NC           | 未接続                               |
| 10       | PS24_SD＋    | PSA-24用RS-485差動信号＋側            |

#### ケーブル側コネクター適合電線

| 項目               | 仕様           |
| :----------------- | :------------- |
| 適合電線           | AWG24～16（銅線） |
| ストリップ長       | 10.0mm        |
| 絶縁被覆の温度定格 | 60℃以上       |

#### 電気仕様

| 項目       | 仕様                   |
| :--------- | :--------------------- |
| STOP－入力 | DC24V±10% / 10mA以下  |
| S1、S2入力 | DC24V±10% / 0.1A以下   |
