### テキスト
#### DC24V電源がプラス接地されたRCONゲートウェイユニットへのパソコン接続時の注意
- ゲートウェイユニットのDC24V電源のプラス側が接地されている場合、パソコンをゲートウェイユニットのSIOコネクターに接続する際は、SIOアイソレーター（RCB-ISL-SIO）を必ず使用すること。
- SIOアイソレーターを使用せずに接続すると、電源の短絡が発生し、パソコンの故障原因となる。

### 図
```mermaid
flowchart LR
    PC[パソコン]
    ConvUnit[変換ユニット\n(RCB-CV-USB)]
    Iso[ SIO アイソレーター\n(RCB-ISL-SIO)]
    Gateway[ゲートウェイユニット]
    Power[DC24V 電源]

    PC -- USBケーブル\n(CB-SEL-USB030) --> ConvUnit
    ConvUnit -- RS485通信 --> Iso
    Iso -- 絶縁 --> Gateway
    Gateway -- +24V / 0V --> Power

    subgraph ケーブル
        IsoComm[アイソレーター通信ケーブル\nCB-RCB-SIO***\n別途必要]
        PCSoft1[パソコン専用ティーチングソフト\n(RCM-101-USB)付属ケーブル\n(USBケーブル + 変換ユニット)]
        PCSoft2[パソコン専用ティーチングソフト\n(RCM-101-USB)付属ケーブル\n(通信ケーブル CB-RCA-SIO050)]
    end

    ConvUnit -.-> PCSoft1
    Iso -.-> IsoComm
    Gateway -.-> PCSoft2
```

**図の説明:**
- パソコンはUSBケーブル（CB-SEL-USB030）を介して変換ユニット（RCB-CV-USB）に接続
- 変換ユニットはRS485通信でSIOアイソレーター（RCB-ISL-SIO）に接続
- SIOアイソレーターは絶縁を行い、ゲートウェイユニットに接続
- ゲートウェイユニットはDC24V電源（プラス接地）に接続されている
- 各接続には専用ケーブルが必要であることを示す

### 注意
- RS232変換ユニット（RCB-CV-MW）は使用できない。
