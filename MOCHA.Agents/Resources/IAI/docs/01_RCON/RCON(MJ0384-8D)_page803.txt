### テキスト
#### 11.1.5 Ethernet 接続設定方法
Ethernetポート搭載仕様（オプション：ET）の場合の、ゲートウェイパラメーター設定ツールでのEthernet接続設定方法について説明しています。

【RCON ゲートウェイパラメーターの設定】  
●用意するもの  
- RCON システム/パソコン/RCM-101 付属ケーブル

1. ドライバーユニットを削除する  
ゲートウェイパラメーター設定ツールを起動します。  
1) 『ゲートウェイパラメーター設定ツール』のアイコンをダブルクリックします。

2) SelectGwType画面から「RCON」を選択しクリックします。

### 図（操作手順のイラスト）
- ノートパソコンの画面に「ゲートウェイパラメーター設定ツール」のアイコンがあり、そこをダブルクリックする操作を示す図
- SelectGwType画面のスクリーンショットにて、ドロップダウンリストから「RCON」を選択しクリックする操作を示す図

```mermaid
flowchart TD
    A[ゲートウェイパラメーター設定ツールのアイコン] -->|ダブルクリック| B[ツール起動]
    B --> C[SelectGwType画面表示]
    C --> D{ユニットタイプ選択}
    D -->|RCON選択| E[OKクリック]
```

**図の説明:**  
- ユーザーはまずツールのアイコンをダブルクリックして起動する。  
- 起動後、SelectGwType画面でユニットタイプを選択し、その中から「RCON」を選んでクリックする。
