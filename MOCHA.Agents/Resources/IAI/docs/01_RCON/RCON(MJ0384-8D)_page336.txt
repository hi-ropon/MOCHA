### テキスト
#### 4.12.2 CC-Link IE Field
CC-Link IE Fieldの接続コネクターについて説明しています。  
ケーブル側およびコントローラー側のコネクターは、Ethernet ANSI/TIA/EIA-568-Bカテゴリー5e以上のシールド付き8P8Cモジュラープラグ（RJ-45）およびモジュラージャック（RJ-45）を使用します。  
Ethernetケーブルはカテゴリー5e以上のストレートSTPケーブルを使用してください。

### 表
#### CC-Link IE Field 接続コネクター仕様

| コネクター名称 | CC-Link IE Field 接続コネクター | 備考 |
| :--- | :--- | :--- |
| ケーブル側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5e以上 シールド付8P8Cモジュラープラグ（RJ-45） | 用意してください |
| コントローラー側 | Ethernet ANSI/TIA/EIA-568-B カテゴリー5e以上 シールド付8P8Cモジュラージャック（RJ-45） |  |
