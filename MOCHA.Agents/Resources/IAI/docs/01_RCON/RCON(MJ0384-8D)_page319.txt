### テキスト
#### 4.8.3 配線方法（24V ACサーボモーター搭載機種）
エレシリンダー大型スライダータイプをEC接続ユニットに接続する場合、24V電源に加えモーター駆動用DC電源と接続する必要があります。

(1) RCON-ECに直接接続  
アクチュエーターケーブルをEC接続ユニット（RCON-EC）に接続します。

#### 注意
- コネクターには挿込み向きがあります。  
  コネクターの形状に合わせ向きを揃え、クリック感があるまで挿入してください。

### 図
```mermaid
flowchart LR
    A[エレシリンダー] -->|アクチュエーターケーブル| B[RCON-EC (EC接続ユニット)]
    B --> C[コネクターのツメ]
    C -.-> D[ツメを左側に合わせて“カチッ”と音がするまで挿入]
```

### 図の説明
- エレシリンダーからアクチュエーターケーブルを用いてRCON-ECに接続する。
- コネクターのツメを左側に合わせ、カチッと音がするまで確実に挿入する必要がある。
