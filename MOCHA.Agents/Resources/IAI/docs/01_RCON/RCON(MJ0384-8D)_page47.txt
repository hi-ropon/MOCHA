### テキスト
#### 1.3.3 構成ユニット
ゲートウェイユニット、ターミナルユニット、ファンユニットの各タイプについて説明しています。

### 表
#### ゲートウェイユニット、ターミナルユニット、ファンユニットのタイプ別仕様

| タイプ | SELユニット | ターミナルユニット (24Vユニットのみ使用の場合) | 200V用ターミナルユニット |
| :--- | :--- | :--- | :--- |
| 型式 | RCON-GW-* | RCON-GW-TR<br>SELユニットに付属 | RCON-GW-TRS<br>200V電源ユニットに付属 |
| 外観 | ![SELユニット](画像) | ![ターミナルユニット24V](画像) | ![ターミナルユニット200V](画像) |
| 最大接続ユニット数 | 1ユニット | 1ユニット<br>RCON-GW-TR / RCON-GW-TRS どちらか一つ | 1ユニット |

| タイプ | ファンユニット (電源、24Vドライバー、SELユニット用) | 200Vドライバー用ファンユニット |
| :--- | :--- | :--- |
| 型式 | RCON-FU | RCON-FUH |
| 外観 | ![ファンユニット](画像) | ![200Vドライバー用ファンユニット](画像) |

### 画像説明
- SELユニットはRCON-GW-*型式で、ゲートウェイユニットの基本ユニット。
- ターミナルユニットは24V用（RCON-GW-TR）と200V用（RCON-GW-TRS）があり、それぞれSELユニットまたは200V電源ユニットに付属。
- ファンユニットは24Vドライバー用（RCON-FU）と200Vドライバー用（RCON-FUH）がある。
