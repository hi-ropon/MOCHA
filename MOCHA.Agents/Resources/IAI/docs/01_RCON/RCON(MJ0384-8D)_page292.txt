### テキスト
#### カテゴリー2（RCON-SC単相200V）を接続する場合
- TB-02D（またはダミープラグ：DP-4S）を使用  
- 電磁接触器によって24V系と200V系のモーター電源を遮断することが重要  
- 本回路例ではゲートウェイユニットの電源入力コネクターMP+24V、RCON-PS2のAC200Vを遮断している  
- 24Vドライバーユニットの駆動源遮断コネクターMPO、MPIを遮断した回路とすることも可能  
- RCON-PS2はAC200Vを遮断する必要がある  

### 図
```mermaid
flowchart TD
    A[TB-02D (またはダミープラグ DP-4S)] -->|接続ケーブル CB-TB1-GC□□□| B[ポジションコントローラー用TPアダプター RCB-LB-TGS]
    B -->|接続ケーブル CB-CON-LB□□□| C[RCONシステム ゲートウェイユニット RCON-GWG]
    B --> D[G9SA-301 (OMRON) リセットSW・イネーブルSW・非常停止SW接続]
    D --> E[電磁接触器]
    E --> F[モーター電源 AC200V]
    C -->|システムI/Oコネクター| G[システムI/O]
    C -->|モーター電源コネクター| H[RCON-PS2]
    H --> I[モーター電源]
    H --> J[RCON-SC]
    J --> K[モーター電源 L1, L2]
```

### 図の説明
- ポジションコントローラー用TPアダプターRCB-LB-TGSは、イネーブルスイッチ、リセットスイッチ、非常停止スイッチを備えた2つのOMRON製G9SA-301リレーを内蔵  
- 電磁接触器により24V系と200V系のモーター電源を遮断  
- RCONシステムのゲートウェイユニットRCON-GWGはシステムI/Oコネクターとモーター電源コネクターを持つ  
- RCON-PS2はモーター電源のAC200Vを遮断する役割を持つ  
- モーター電源はRCON-SCを経由して供給される
