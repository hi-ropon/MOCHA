### テキスト
#### 2.2.3 ユニット接続制限
【1】電流制限値  
選定計算用の電流制限値は以下の通り。  
RCONシステムの構成に基づき、各ユニットの制御電源とモーター電源を計算し、制限値を超えていないか確認すること。  
ゲートウェイユニットは計算対象に含まれない。  

#### 注意事項  
- ゲートウェイユニットは計算対象外。  
- 以下に計算例を示す。

### 表
#### 選定計算用の電流制限値

| 項目               | 選定計算用の電流制限値 |
| :----------------- | :--------------------- |
| 制御電源（CP）     | 9.0A 以下               |
| 24V モーター電源（MP） | 37.5A 以下              |
