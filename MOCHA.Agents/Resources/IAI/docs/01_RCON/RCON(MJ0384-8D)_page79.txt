### テキスト
#### 2.2 電源仕様

- 電源容量は「制御電源容量」と「モーター電源容量」に分けられる。
- DC24V電源はゲートウェイユニットの制御電源コネクターとモーター電源コネクターから入力する。ただし、制御電源とモーター電源の0Vは必ず共通にする必要がある。
- AC200Vアクチュエーターのモーター電源は、200V電源ユニット（RCON-PS2-3）から入力する。

- 必要な電源容量の計算式：
  ```
  必要電源容量 = “使用しているユニットの制御電源容量の和” + “接続しているアクチュエーターのモーター電源容量の和”
  ```
  ただし、200Vドライバーユニットの場合は「使用しているユニットの制御電源容量の和」のみで算出し、モーター電源は別途考える。

- DC24V電源の定格電流値はモーター電源容量の定格電流合計値を上回る必要がある。ピーク電流値もモーター電源容量合計値を上回る必要がある。
- 複数軸接続時はアクチュエーターの動作タイミングが異なる場合、定格電流・最大電流が同時に流れないため単純な合計にはならない。

- 24V系については、アクチュエーターの動作条件と動作パターンを設定すると必要電源容量を自動計算できる「カリキュレーターソフト」が用意されている。
- 操作方法など詳細は「カリキュレーター取扱説明書（MJ0381）」を参照。
