### テキスト
#### 外部出力仕様
SCON拡張ユニットなどの外部出力仕様について説明しています。

### 表
#### 外部出力仕様の主要項目

| 項目           | 仕様               |
| :------------- | :----------------- |
| 出力点数       | 16点               |
| 定格負荷電圧   | DC24V±10%          |
| 最大電流       | 50mA/1回路         |
| 絶縁方式       | フォトカプラー     |

### 図
#### NPN仕様回路図
```mermaid
flowchart LR
    内部回路 -->|LED| フォトカプラー
    フォトカプラー -->|トランジスタ| トランジスタ
    トランジスタ -->|出力端子15| 出力端子15
    出力端子15 -->|負荷| 負荷
    負荷 -->|外部電源DC24V| 外部電源DC24V
    出力端子P24 -->|接続| 負荷
```

#### PNP仕様回路図
```mermaid
flowchart LR
    内部回路 -->|LED| フォトカプラー
    フォトカプラー -->|トランジスタ| トランジスタ
    トランジスタ -->|端子15| 端子15
    外部電源DC24V -->|接続| 出力端子P24
    端子15 -->|負荷| 負荷
    負荷 -->|接続| 外部電源DC24V
```

**図の説明:**
- NPN仕様は、フォトカプラーのトランジスタが負荷の片側を接地側に接続し、負荷のもう一方が外部電源に接続される構成。
- PNP仕様は、フォトカプラーのトランジスタが負荷の片側を外部電源側に接続し、負荷のもう一方が接地側に接続される構成。
