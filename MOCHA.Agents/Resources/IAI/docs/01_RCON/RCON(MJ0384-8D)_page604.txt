### テキスト
#### 7.5.2 過負荷警告
本機能を活用することで、グリースの枯渇や部品の消耗などにより生じるモーター温度変化をモニターできます。  
あらかじめ設定した値を超えた場合に警告を出し、故障や不具合発生前の異変を検知可能です。

【ドライバーユニットの場合】  
- 警告出力により、グリースアップ時期、部品交換時期、メカ調整実施時期などを検知できます。

### 図
```mermaid
flowchart LR
    A[モーター温度変化（正常時）] -->|時間経過| B[モーター温度変化（異常時）]
    B --> C{過負荷警告値到達}
    C -->|警告出力| D[ロボシリンダ継続動作]
    C -->|アラーム出力| E[ロボシリンダ停止]
    E --> F[リセット操作必要]
    D --> G[メンテナンス実施]
    G --> H[モーター温度低下]
```

**図の説明:**  
- 青線は正常時のモーター温度変化例を示す。  
- オレンジ線は過負荷警告発生時の温度変化を示し、警告出力が発生。  
- 赤線は過負荷アラーム検出値（100%）を超えた場合の温度変化で、アラーム出力が発生しロボシリンダは停止。  
- 警告出力後にメンテナンスを実施することで故障や不具合を防止可能。  
- 過負荷「警告」値はパラメータロードレベル比で設定し、50～99%の範囲で任意設定可能。

### 補足
- 警告出力から以下の内容を検知可能：  
  - グリースアップ時期  
  - 部品交換時期  
  - メカ調整実施時期
