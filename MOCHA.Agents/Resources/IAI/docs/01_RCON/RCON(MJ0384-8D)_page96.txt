### テキスト
#### 2.3.3 CC-Link IE TSN インターフェイス仕様
CC-Link IE TSNのインターフェイス仕様について説明しています。

#### 注釈
- 通信ケーブルは別途用意する必要があることが記載されています。

### 表
#### CC-Link IE TSN インターフェイス仕様

| 項目                     | 仕様                                                                                   |
|--------------------------|----------------------------------------------------------------------------------------|
| 通信規格                 | IEEE802.3ab（1000BASE-T）準拠                                                        |
| 通信速度                 | 1Gbps                                                                                  |
| 通信制御方式             | 時分割方式、タイムマネージド・ポーリング方式                                          |
| 通信ケーブル長           | 各セグメント間の距離：100m以内（ツイストペアケーブル（IEEE 802.3準拠）の場合）        |
| 1ネットワークあたりの接続ノード数 | 64,770台（マスター局とスレーブ局の合計）                                             |
| トポロジー               | ライン、スター、ライン・スター混在、リング、リング・スター混在、メッシュ              |
| 通信ケーブル※1          | ANSI/TIA/EIA-568-B（カテゴリー5e以上）<br>（アルミテープと編組の二重遮蔽シールドケーブル推奨） |
| 接続コネクター           | RJ-45コネクター2個                                                                     |
| 1局あたりの最大サイクリックサイズ | 各局、入出力合計で最大4G（4,294,967,296）オクテット                                  |
| 認証クラス               | B                                                                                      |

※1 通信ケーブルは別途用意してください。
