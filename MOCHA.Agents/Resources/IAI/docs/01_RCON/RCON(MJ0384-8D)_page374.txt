### テキスト
#### オプションユニットを接続する場合
オプションユニット（EC接続ユニット、PIO/SIO/SCON拡張ユニット、PIOユニット）を接続する際は、「オプションユニット台数」に接続するオプションユニットの数を設定します。  
SCON拡張ユニット（RCON-EXT）はオプションユニット台数に含まれません。  
オプションユニット数は最大8台まで設定可能です。PIO/SIO/SCON拡張ユニットは1台のみ、EC接続ユニットは最大4台まで接続可能です。  
EC接続ユニットの軸番号は各ドライバーユニット（RCON-PC・PCF・AC・DC・SC）やSCONより後ろに割り付けられ、EC接続ユニットの後ろに他のユニットを移動・追加することはできません。  
また、EC接続ユニットの先頭軸番号が15を超える場合は接続できません。

### 注意
| 注意事項 |
| :--- |
| RCONシステムの実際の接続軸数とゲートウェイパラメーター設定ツールで設定して転送した軸数が一致しない場合、パソコン専用ティーチングソフトは接続できません。 |
| 実際のユニット構成や接続軸数に合わせて、ゲートウェイパラメーターを適切に設定して転送してください。 |
