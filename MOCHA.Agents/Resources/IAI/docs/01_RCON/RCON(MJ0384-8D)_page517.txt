### テキスト
#### 設定値と信号の出力範囲
ゾーンの＋側と－側の設定値の差により、ゾーン出力範囲が異なります。

1) ＋側設定値 ＞ －側設定値の場合  
　－側設定値～＋側設定値の範囲でゾーン信号がON、範囲外でOFFとなる。

2) ＋側設定値 ＜ －側設定値の場合  
　＋側設定値～－側設定値の範囲でゾーン信号がOFF、範囲外でONとなる。

```
設定値
ゾーン設定＋側：30mm
ゾーン設定－側：70mm
```

#### 【多回転仕様のロータリーアクチュエーターがインデックスモードの場合】
```
左図：
0度 → 70度 → 315度（反時計回り）
設定値
ゾーン設定＋側：70度
ゾーン設定－側：315度

右図：
0度 → 315度 → 70度（時計回り）
設定値
ゾーン設定＋側：315度
ゾーン設定－側：70度
```

### 注意
- 本信号は、原点復帰完了後に有効になります。電源投入しただけでは出力しません。
