### テキスト
#### 5.7.3 ジョグ移動
アクチュエーターをジョグ（JOG）動作させる方法について説明しています。

1. 「前進」ボタンをクリックすると、アクチュエーターがストロークエンド側に動作します。
2. 「後退」ボタンをクリックすると、アクチュエーターが原点方向に動作します。
3. 赤枠部分から3段階の速度調整が可能です。

### 図
#### ポジションデータ編集画面のジョグ操作部分
```mermaid
flowchart LR
    A[ジョグ動作選択] --> B{速度選択}
    B -->|高速| C[高速移動]
    B -->|中速| D[中速移動]
    B -->|低速| E[低速移動]
    C --> F[前進ボタン押下]
    D --> F
    E --> F
    F --> G[アクチュエーター動作]
```

#### アクチュエーター動作方向イメージ
```mermaid
flowchart LR
    subgraph ストロークエンド側に動作
        direction LR
        A[前進ボタン] --> B[アクチュエーターがストロークエンド側に移動]
    end

    subgraph 原点側に動作
        direction LR
        C[後退ボタン] --> D[アクチュエーターが原点側に移動]
    end
```

### 図の説明
- ポジションデータ編集画面のジョグ操作部分では、速度（高速・中速・低速）を選択し、前進・後退ボタンでアクチュエーターを動かす。
- アクチュエーターは「前進」ボタンでストロークエンド側に、「後退」ボタンで原点側に動作する。
- 速度は3段階から選択可能で、操作中は指定速度で移動する。
