### テキスト
#### 6.5.8 多回転ロータリーアクチュエーターの近回り制御
【1】近回り選択の設定  
近回り選択はパラメーターNo.80「回転軸近回り選択」で有効/無効の設定が可能です。  
有効にすると同一方向にだけ運転を行うことも可能となります。

【運転例】  
ポジション1 → 2 → 3 → 4 → 1 と順番に運転した場合、近回り選択の有効・無効で動作が異なります。

- 無効の場合  
  ポジション1から2へ進み、次に3へは戻る動作（逆回転）を伴い、4から1へ戻る際も逆回転を行います。

- 有効の場合  
  ポジション1から2へ進み、3へは逆回転で戻りますが、4から1へは逆回転せず同一方向に回転します。

#### 運転順序（無効の場合）
```mermaid
flowchart LR
    P1(1) --> P2(2)
    P2 --> P3(3)
    P3 --> P4(4)
    P4 --> P1(1)
    P3 -.逆回転.-> P2
    P4 -.逆回転.-> P3
    P1 -.逆回転.-> P4
```

#### 運転順序（有効の場合）
```mermaid
flowchart LR
    P1(1) --> P2(2)
    P2 --> P3(3)
    P3 -.逆回転.-> P2
    P3 --> P4(4)
    P4 --> P1(1)
```
