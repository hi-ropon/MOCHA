### 表：ドライバーユニット・簡易アブソユニットのアラームの原因と対策

| アラームコード | アラーム名称                         | 原因/対策                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| -------------- | ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0D8            | 偏差オーバーフロー                 | 原因：位置偏差カウンターがオーバーフローした<br>1) 移動中に外力などの影響、または過負荷により速度が低下、または停止した<br>2) 電源投入後の励磁検出動作が不安定な状態<br>3) 電源電圧が低下した<br>4) サーボゲイン番号が小さすぎる<br>対策：アクチュエーターが指令通りに動作できないときに発生<br>1) ワークが周辺物に干渉していないか、ブレーキが解除されているかなど負荷状況を確認し原因を取除く<br>2) 過負荷が考えられるため、搬送質量を見直し電源再投入<br>3) 電源電圧を確認<br>4) サーボゲイン番号を調整 |
| 0D9            | ソフトウェアストロークリミットオーバーエラー | 原因：アクチュエーターの現在位置がソフトウェアストロークリミットを越えた<br>対策：ソフトウェアストロークリミットの範囲内に戻す                                                                                                                                                                                                                                                                                                                                                                   |
| 0DC            | 押付け動作範囲オーバーエラー     | 原因：押付け完了後に押戻す力が強すぎて、押付け開始設定位置まで押戻された<br>対策：押戻す力が小さくなるように再設定および調整                                                                                                                                                                                                                                                                                                                                                                       |
| 0DF (Pドライバー限定) | 衝突検出                         | 原因：アクチュエーターの衝突を検出した<br>対策：衝突の原因を取除く。想定外の検出である場合、衝突検出機能の再調整を実施してください。[7.2 衝突検出機能]参照                                                                                                                                                                                                                                                                                                                                       |
| 0ED (P/Aドライバー限定) | アブソリュートエンコーダー異常検出1 | 原因：アブソデータの読出しまたは保存の際、現在位置が変化した<br>対策：アクチュエーターに振動などが加わらないようにしてください                                                                                                                                                                                                                                                                                                                                                                   |
| 0EE (P/A/Sドライバー限定) | アブソリュートエンコーダー異常検出2 | 原因：バッテリーレスアブソ仕様、または簡易アブソ仕様でエンコーダーが位置情報を正常に検出できない<br>1) バッテリーレスアブソ仕様でモーター交換後、または簡易アブソ仕様で初めての電源投入時（アブソリセット実行前）<br>2) アブソバッテリー電圧の低下（簡易アブソ仕様）<br>3) アクチュエーター接続ケーブル、アクチュエーター側付属ケーブルの断線、コネクター接続不良、またはケーブル抜差し<br>4) ドライバーユニットのパラメーターを変更した<br>対策：2) 72時間以上の電源供給を行いバッテリー充電後アブソリュートリセットを実施。頻発する場合はバッテリー交換。1),3),4)の場合はアブソリュートリセットを実施。[9.4.2 アブソバッテリーの交換]、[9.5 アブソリュートリセット手順]参照 |
| 0EF (P/Aドライバー限定) | アブソリュートエンコーダー異常検出3 | 原因：1) 簡易アブソ仕様でエンコーダーが位置情報を正常に検出できない（エンコーダーオーバースピードエラー）<br>2) 電源遮断時に外的要因により回転速度設定以上の速度で現在位置が変化した<br>対策：回転速度設定を現在よりも高回転対応の設定に変更。発生した場合はアブソリュートリセットが必要。[9.5 アブソリュートリセット手順]参照 |
