### テキスト
#### 5.7 パソコン専用ティーチングソフトでの動作確認
##### 5.7.1 ポジションデータ編集画面を開く
1. メイン画面左端にあるツリービューの軸 No.を右クリックすると各項目が展開します。  
2. 「ポジションデータ編集」をクリックします。  
3. ポジションデータ編集画面が表示されます。

### 図
```mermaid
flowchart TD
    A[メイン画面左端のツリービューの軸 No.を右クリック] --> B[各項目が展開]
    B --> C[ポジションデータ編集をクリック]
    C --> D[ポジションデータ編集画面が表示]
```

**図の説明:**  
- メイン画面のツリービューで軸 No.を右クリックし、メニューを展開  
- メニューから「ポジションデータ編集」を選択  
- ポジションデータ編集画面が開く流れを示す
