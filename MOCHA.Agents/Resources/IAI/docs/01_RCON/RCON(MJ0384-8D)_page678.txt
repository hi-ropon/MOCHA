<overview>
本ページは「RCONゲートウェイパラメーター」に関する説明で構成されており、主に2つの通信設定セクションに分かれています。  
1つ目は「PROFIBUS-DP」の通信設定、2つ目は「PROFINET IO」の通信設定です。  
各セクションはテキスト説明、設定画面のスクリーンショット、そして設定項目の初期値と設定可能範囲を示した表で構成されています。  
グラフやフローチャートは含まれていません。
</overview>

<contents>
### テキスト
#### PROFIBUS-DP 通信設定
使用するマスターに合わせて以下の項目を設定します。  
- ネットワークタイプ: PROFIBUS-DP  
- アドレス  
- 通信速度  

#### PROFINET IO 通信設定
使用するマスターに合わせて以下の項目を設定します。  
- ネットワークタイプ: PROFINET IO  
- 通信速度  
- MACアドレス  

※MACアドレスはネットワークモジュールに割り付けられている固有の識別番号であり、設定不可。

### 表
#### PROFIBUS-DP 設定項目

| 項目           | 初期値       | 設定可能範囲          |
| :------------- | :----------- | :-------------------- |
| ネットワークタイプ | PROFIBUS-DP | 変更不可              |
| アドレス       | 0            | 0～65,535             |
| 通信速度       | 自動         | 設定不要（1Gbps固定） |

#### PROFINET IO 設定項目

| 項目           | 初期値       | 設定可能範囲          |
| :------------- | :----------- | :-------------------- |
| ネットワークタイプ | PROFINET IO | 変更不可              |
| 通信速度       | 自動         | 設定不要（100Mbps固定）|
| MACアドレス    | 製品固有     | 変更不可              |
