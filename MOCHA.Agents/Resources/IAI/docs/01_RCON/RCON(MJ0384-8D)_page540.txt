### テキスト
#### 6.4 EC接続ユニットの入出力信号
【3】ロボポンプ（RP）  
●パラメーターNo.5 PIOパターン選択が“0：ダブルソレノイド”の場合  
信号のON/OFFにより離脱モードや吸着モードに遷移することを示しています。  

●パラメーターNo.5 PIOパターン選択が“1：シングルソレノイド”の場合  
吸着/離脱指令信号のON/OFFでモード遷移を検知し、アラームリセットや吸着確認信号の動作について説明しています。

### 表
#### パラメーターNo.5 PIOパターン選択が“0：ダブルソレノイド”の場合

| 区分 | Bit | 信号名称       | 信号略称 | 機能概要                                                                                                   |
| :--- | :-- | :------------- | :------- | :--------------------------------------------------------------------------------------------------------- |
| 指令 | 0   | 離脱指令       | ST0      | 離脱指令用の入力信号。ONエッジで離脱モード（モーター停止、ソレノイド駆動で真空配管の弁を開いて大気圧に戻す）に遷移。 |
|      | 1   | 吸着指令       | ST1      | 吸着指令用の入力信号。ONエッジで吸着モード（モーター起動、ポンプ動作）に遷移。                             |
|      | 2   | アラームリセット | RES      | ONにするとアラームをリセット。内容により解除できない場合あり。詳細は取扱説明書参照。                         |
| 応答 | 0   | 吸着確認       | PE1      | 吸着指令後、真空圧が吸着確認レベル以下でON。真空圧が離脱確認レベルより大きくなるとOFF。                   |
|      | 1   | アラーム（B接点） | *ALM     | 正常時ON。アラーム発生でOFF。OFF時は速やかに安全対策を。詳細は取扱説明書参照。                             |

#### パラメーターNo.5 PIOパターン選択が“1：シングルソレノイド”の場合

| 区分 | Bit | 信号名称         | 信号略称 | 機能概要                                                                                                   |
| :--- | :-- | :--------------- | :------- | :--------------------------------------------------------------------------------------------------------- |
| 指令 | 0   | 吸着/離脱指令    | ST1      | 信号レベルで吸着/離脱指令を検知。ON(High)で吸着モード、OFF(Low)で離脱モードに遷移。                         |
|      | 1   | アラームリセット | RES      | ONでアラームリセット。解除できない場合あり。詳細は取扱説明書参照。                                         |
| 応答 | 0   | 吸着確認         | PE1      | 吸着/離脱指令ON時、真空圧が吸着確認レベル以下でON。離脱確認レベルより大きくなるとOFF。                     |
|      | 1   | アラーム（B接点） | *ALM     | 正常時ON。アラーム発生でOFF。OFF時は速やかに安全対策を。詳細は取扱説明書参照。                             |
