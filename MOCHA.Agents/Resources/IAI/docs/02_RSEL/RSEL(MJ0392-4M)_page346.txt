#### PROFINET IO接続仕様
PROFINET IOの物理インターフェイスの写真と、MSおよびNSのLED表示部の画像が掲載されています。

##### PROFINET IOインターフェイス仕様表

| 項目               | 内容                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 対応プロトコル     | PROFINET IO                                                                           |
| コンフォーマンスクラス | A（RT通信）                                                                           |
| デバイスクラス     | IOデバイス                                                                             |
| IPアドレス         | 0.0.0.0～255.255.255.255（マスターなどからPROFINETネットワーク経由で設定）             |
| 通信速度           | 100Mbps                                                                               |
| 通信モード         | 100BASE-TX                                                                            |
| ケーブル           | カテゴリー5以上                                                                        |
| コネクター         | RJ-45×1                                                                               |

**説明:**  
PROFINET IOはRT（リアルタイム）通信に対応したIOデバイスとして動作し、IPアドレスはネットワーク経由で設定されます。通信速度は100Mbps、通信モードは100BASE-TXで、カテゴリー5以上のケーブルを使用し、コネクターはRJ-45が1つ搭載されています。
