### テキスト
#### ジョグスイッチについて
- ジョグ運転用のスイッチである。
- ドライバーユニットパラメーターNo.194「JOGスイッチ」を“0”に設定すると本スイッチが有効になる（初期値は“0：有効”）。
- JOG+側にスイッチを傾けると＋方向に、JOG−側に傾けると−方向にジョグ動作を行う。
- スイッチを傾け続けると段階的にジョグ速度が速くなる。
- 原点復帰未完了の場合、ジョグ速度は原点復帰未完了時最大JOG速度（MC共通パラメーターNo.3）となる。
- サーボON中にジョグスイッチでジョグ動作をすると、完了後もサーボONのまま。サーボOFF中の場合は、サーボONしてからジョグ動作し完了後にサーボOFFとなる。
- ジョグスイッチの操作はMANUモード時のみ有効で、AUTOモード時は無効。
- ティーチングツールでアクチュエーター操作画面を開いている場合もジョグスイッチは無効。
- ジョグスイッチ動作中に操作画面を開くとアクチュエーターは減速停止する。

### 表
| 記号  | 説明                         |
| :---- | :--------------------------- |
| JOG+  | ＋方向にジョグ動作（原点の反対方向） |
| JOG−  | －方向にジョグ動作（原点方向）       |

### 注意事項
- ティーチングツールでアクチュエーター操作画面を開いているときに通信が切断されると、ジョグスイッチ操作が無効のままになる。
- ジョグスイッチ操作を有効に戻すには、RSELシステムの電源再投入かソフトウェアリセットが必要。
- ジョグスイッチ操作にはRSEL軸の設定が必要（詳細は「5.3 軸設定」を参照）。
