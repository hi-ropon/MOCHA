### テキスト
#### 4.9 USBコネクターの接続
パソコンをUSBポートに接続して、パソコン専用ティーチングソフトを使用できます。  
アクチュエーターを動作させる場合は、ティーチングコネクターにダミープラグ（DP-4S）を挿入してください。

### 図
```mermaid
flowchart LR
    PC[パソコン] -->|USBケーブル (A、mini-B) (お客様で用意)| Controller[コントローラー]
    Controller -->|ダミープラグ DP-4S| DummyPlug[ティーチングコネクター]
```

**USB接続コネクターのピン配置図**  
USBコネクターの端子が縦に並び、上からピン番号5（GND）と1（Vbus）が示されています。

### 表
#### USB接続コネクター仕様

| 項目           | 内容                          | 備考                   |
| :------------- | :---------------------------- | :--------------------- |
| 使用コネクター | USB mini-B                   | 51387-0530（モレックス） |
| コネクター名称 | USB                          |                        |
| 通信規格       | USB2.0準拠                   | 12Mbps（フルスピード）  |
| 最大接続距離   | 5m                           |                        |
| 被接続ユニット | パソコン（USBポート）         |                        |

#### ピン配置

| ピン番号 | 信号名称 | 信号内容                          |
| :------- | :------- | :-------------------------------- |
| 1        | Vbus     | USB電源入力（+5V）                |
| 2        | D-       | USB差動送受信データ−側            |
| 3        | D+       | USB差動送受信データ＋側            |
| 4        | ID       | USB ID（識別）端子（ケーブル側はオープン） |
| 5        | GND      | 電源グラウンド                    |
