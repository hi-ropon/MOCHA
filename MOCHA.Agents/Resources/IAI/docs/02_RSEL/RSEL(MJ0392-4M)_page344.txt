### テキスト
#### EtherNet/IP接続仕様
EtherNet/IPインターフェイス仕様について説明しています。対応プロトコルはCIP、TCP/IPで、デバイスタイプはGeneric Deviceです。IPアドレスは1.0.0.1～254.255.255.254の範囲で、I/Oパラメーターによるソフトウェア設定が可能です。ポート番号は2222（UDP）と44818（TCP/UDP）を使用します。通信速度は10/100Mbpsで、通信モードは10BASE-T/100BASE-TX（半二重/全二重）です。ケーブルはカテゴリー5以上を推奨し、コネクターはRJ-45×1です。

※補足として、特定のオクテット値は予約アドレスのため使用不可、EtherNet/IP通信で使用するポート番号、STPケーブル推奨の注意書きがあります。

### 表
#### EtherNet/IPインターフェイス仕様

| 項目           | 内容                                                                                  |
| -------------- | ------------------------------------------------------------------------------------- |
| 対応プロトコル | CIP、TCP/IP                                                                          |
| デバイスタイプ | Generic Device                                                                       |
| IPアドレス     | 1.0.0.1～254.255.255.254（I/Oパラメーターによるソフトウェア設定）                    |
| ポートNo.      | 2222（UDP）、44818（TCP/UDP）                                                        |
| 通信速度       | 10/100Mbps（I/Oパラメーターによるソフトウェア設定）                                  |
| 通信モード     | 10BASE-T/100BASE-TX（半二重/全二重）（I/Oパラメーターによるソフトウェア設定）        |
| ケーブル       | カテゴリー5以上（STPケーブル推奨）                                                   |
| コネクター     | RJ-45×1                                                                              |

#### I/OスロットステータスLED

| LED名称 | 色   | 状態   | 内容                             |
| ------- | ---- | ------ | -------------------------------- |
| NS      | 緑   | 点灯   | オンライン状態、正常通信中       |
| NS      | 緑   | 点滅   | オンライン状態、コネクションなし |
| NS      | 橙   | 点灯   | IPアドレス重複、致命的なリンクエラー |
| NS      | 橙   | 点滅   | コネクションタイムアウト         |
| NS      | -    | 消灯   | 電源供給なし、IPアドレス未設定   |
| MS      | 緑   | 点灯   | 通常運転                       |
| MS      | 緑   | 点滅   | コンフィグレーション未設定または不完全、要試運転 |
| MS      | 橙   | 点灯   | 回復不可能な異常               |
| MS      | 橙   | 点滅   | 回復可能な異常                 |
| MS      | -    | 消灯   | 電源供給なし                   |
