### テキスト
#### 5.3.4 各エリアの説明
##### 物理軸エリア
物理エリアには、ドライバーユニットおよび実装済みの軸画像が表示されます。

1. ドライバーユニット種別  
ドライバーユニットが実装されている順番に表示されます。  
ただし、SCONシリーズについては必ず最終ドライバーユニットとして表示されます。

2. 実装済みの軸画像  
実装済みの軸画像には以下の情報が表示されます。

### 画像
#### 図 5.3-4 物理エリアイメージ（ドライバー5台接続）
- ドライバーユニット種別ごとに区切られたエリアに、各ユニットの軸画像が配置されている。  
- 「No Image」と表示されている箇所は軸画像が未実装の部分。

#### 図 5.3-5 実装済みの軸画像
- 軸画像の各部にラベル（A～E）が付されており、表で説明されている。

### 表
| ラベル | 内容 |
| :--- | :--- |
| A | ドライバーユニット番号（Unit0～Unit7） |
| B | ドライバーユニット内物理軸番号（Drv0～Drv1） |
| C | アクチュエーター画像（画像がない場合は「No Image」の画像が表示される） |
| D | アクチュエーター名称（名称を取得できない場合はUnknownが表示される） |
| E | 軸設定状況に割当てられたAxis番号（Axis1～Axis8）（割当てがある場合のみ半透明の文字で表示） |

</contents>
