### テキスト
#### (4) ブレーキリリーススイッチ
- ブレーキ強制解除用のスイッチで、Iは1軸目、IIは2軸目を示す。
- 通常はNOM側に設定。NOM側ではサーボONでブレーキ解除、サーボOFFでブレーキロックとなる。
- RLS側ではサーボON/OFFに関係なく強制解除（制御電源OFF時を除く）。

#### 警告
- ブレーキ解除は十分注意して行うこと。不用意に行うとスライダーやロッドの落下により、けがやアクチュエーター本体、ワークや装置の破損原因となる。
- ブレーキ解除後は必ずブレーキを有効状態に戻すこと。解除状態のまま運転すると非常に危険。

#### (5) モーター・エンコーダーコネクター
- アクチュエーターと接続するコネクター。Iは1軸目、IIは2軸目を示す。
- RSELシステムではティーチングツールを使い、使用前に軸番号設定が必要（[5.3 軸設定]参照）。

### 表
| 記号 | 説明 |
| :--- | :--- |
| RLS | ブレーキリリース（Brake Release） |
| NOM | ブレーキロック（Normal） |
