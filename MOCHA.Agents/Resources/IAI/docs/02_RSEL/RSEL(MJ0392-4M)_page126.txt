### テキスト
#### 3.2.2 型式
ドライバーユニットの型式は以下の通りです。  
RCON-PC/AC/DCは1軸仕様または2軸仕様を選択可能。  
RCON-PCFは1軸仕様のみ。

(1) 型式の見方

【24Vドライバーユニット】  
RCON - タイプ - 軸数

【ファンユニット】  
RCON - FU

### 表
#### 24Vドライバーユニットのタイプと軸数対応表

| タイプ | 説明                         |
| :----- | :--------------------------- |
| PC     | パルスモータータイプ         |
| PCF    | パルスモーター56SP/60P/86P対応タイプ |
| AC     | ACサーボモータータイプ       |
| DC     | DCブラシレスモータータイプ   |

| 軸数 | 説明     |
| :--- | :------- |
| 1    | 1軸仕様  |
| 2    | 2軸仕様  |
