### テキスト
#### ベース座標系の設定とポジション指定移動命令の実行
- ワーク座標系No.0の選択とポジションタイプにRect（直交）を設定した上で、ポジション指定移動命令を実行する。
- SELプログラム内でワーク座標系No.を選択する場合はSLWK命令を使用する。
- 選択したワーク座標系No.はプログラム終了後や電源再投入後も有効。

#### SELプログラム例
| 命令 | 値 | 説明 |
| --- | --- | --- |
| VEL | 100 | 速度100mm/s |
| ACC | 0.3 | 加速度0.3G |
| DCL | 0.3 | 減速度0.3G |
| SLWK | 0 | ワーク座標系No.0（ベース座標系）を選択 |
| MOVP | 1 | ポジションNo.1へPTP動作で移動 |
| MOVP | 2 | ポジションNo.2へPTP動作で移動 |
| EXIT | - | プログラム終了 |

### 画像説明
- 画面キャプチャには、ワーク座標系選択No.のドロップダウンメニューが表示されており、No.0が選択されている。
- ポジションタイプが「Rect」となっている複数のポジションNo.がリスト表示されている。
- 速度、加速度、減速度の設定欄があり、それぞれ0.000や0.30などの値が入力されている。
