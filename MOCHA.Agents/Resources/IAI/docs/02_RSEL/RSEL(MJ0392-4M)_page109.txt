<contents>
### テキスト
#### IOスロット
PIOやフィールドネットワーク接続をするためのコネクターを装着するスロットです。

#### PIO（NPN/PNP）
PIOコネクターの写真が示されており、端子番号「1A」「1B」「20A」「20B」が表示されています。

### 表
#### コネクター仕様

| 項目           | 内容                     |
| -------------- | ------------------------ |
| 型式           | HIF6-40PA-1.27DS（71）  |
| メーカー       | ヒロセ電機               |

#### コネクター一般仕様

| 項目       | 内容    |
| ---------- | ------- |
| 定格電流   | 0.5A    |
| 定格電圧   | AC125V  |
| 挿抜寿命   | 500回   |

#### LED仕様

| LED名称 | 色   | 状態 | 内容                              |
| ------- | ---- | ---- | --------------------------------- |
| RUN     | 緑   | 点灯 | 初期化完了により点灯、正常動作中  |
| ERR     | 橙   | 点灯 | PIO電源（DC24V）電圧低下異常     |

### 画像説明
PIOコネクターの写真には、端子番号が「1A」「1B」「20A」「20B」と表示されており、RUNとERRのLEDインジケーターが確認できます。
