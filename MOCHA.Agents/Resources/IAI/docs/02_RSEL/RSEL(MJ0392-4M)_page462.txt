### テキスト
#### 7.1 座標系概要
直交型6軸ロボットの直交座標系機能が有効な場合は、直交型6軸ロボットに限って直交座標系の位置決め機能を付加することができます。付加軸は対象外です。

#### 7.1.1 座標系一覧
直交座標系は以下のように分類されます。

### 図
```mermaid
flowchart TD
    A[座標系] --> B[各軸座標系]
    B --> C[直交座標系]
    C --> D[ベース座標系]
    C --> E[ワーク座標系]
    C --> F[ツール座標系（メカニカルインターフェイス座標系）]
```

**図の説明:**
- 座標系は「各軸座標系」と「直交座標系」に大別される。
- 直交座標系はさらに「ベース座標系」「ワーク座標系」「ツール座標系（メカニカルインターフェイス座標系）」に分類される。
