#### 3.7.7 注意事項

【パラメーター変更時の注意事項】  
以下のパラメーターを変更した場合、アブソリュートエラーが発生するため、変更後は再度アブソリュートリセットを行う必要がある。  
対象パラメーターは以下の通り。  
1. ドライバーユニットパラメーターNo.5 “原点復帰方向”  
2. ドライバーユニットパラメーターNo.22 “原点復帰オフセット量”  
3. ドライバーユニットパラメーターNo.77 “ボールねじリード長”  
4. ドライバーユニットパラメーターNo.78 “軸動作種別”  

【アブソバッテリーの取扱い】  
以下の安全上の注意事項を厳守すること。  
1. 絶対に分解しないこと。電解液は強アルカリ液であり、皮膚や衣類を傷める。  
2. ショート（＋極と－極の直接接触）は絶対に避けること。機器損傷や発熱、火傷の恐れがある。  
3. 火中投入は破裂の恐れがあるため避けること。また水中投入も電池機能を失うため避ける。  
4. 直接はんだ付けしないこと。電池キャップ内の安全弁損傷による安全機構破壊の恐れがある。  
5. コネクター接続状態で長期間電源遮断状態が続くと深放電され、漏液や性能・寿命低下の恐れがある。長期間電源遮断する場合はコネクターを抜くこと。  
6. 廃棄時はリサイクル協力店の回収箱に入れるなど適切な処置を行うこと。
