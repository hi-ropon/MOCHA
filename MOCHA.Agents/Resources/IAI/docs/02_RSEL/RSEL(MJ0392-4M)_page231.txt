### テキスト
#### 4.3.2 三相仕様電源回路
RSELシステムの三相仕様電源回路の展開接続図（例）を示しています。

- 24V電源はモーター電源コネクターと制御電源コネクターに接続されている。
- 0Vは共通のグランドとして接続されている。
- 200V三相電源はL1、L2、L3の3相線とPE（保護接地）線があり、サーキットブレーカーを介して200Vドライバー用モーター電源コネクターに接続されている。

### 図（回路図）
```mermaid
flowchart TD
    subgraph RSELシステム
        SEL[SELユニット (RSEL-G)]
        D24[24Vドライバーユニット (RCON-PC/PCF/AC/DC)]
        P200[200V電源ユニット (RCON-PS2-3)]
        D200[200Vドライバーユニット (RCON-SC)]
    end

    %% 24V電源系統
    24V[24V電源] -->|モーター電源コネクター| SEL
    24V -->|制御電源コネクター| SEL
    0V[0V] --> SEL
    SEL -->|FG| Ground[接地]

    %% 200V三相電源系統
    L1 -->|サーキットブレーカー| Motor200V[200Vドライバー用モーター電源コネクター]
    L2 --> Motor200V
    L3 --> Motor200V
    PE --> Motor200V

    Motor200V --> P200
```

**図の説明:**  
- 24V電源はモーター電源コネクターと制御電源コネクターを通じてSELユニットに供給される。  
- 0Vは共通のグランドとして接続され、FG（フレームグラウンド）も接地されている。  
- 200V三相電源はサーキットブレーカーを経由し、200Vドライバー用モーター電源コネクターに接続されている。  
- 200V電源ユニットと200Vドライバーユニットはそれぞれ別のブロックとしてRSELシステム内に配置されている。
