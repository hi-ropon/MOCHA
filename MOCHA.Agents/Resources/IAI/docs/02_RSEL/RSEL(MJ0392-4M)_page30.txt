### テキスト
#### 取扱い上の注意
13. 以下のアクチュエーターのブレーキ付き仕様の場合、エンコーダーケーブルの接続には必ずブレーキボックスを使用してください。

（注）ブレーキボックスを使用しない場合、アクチュエーターのブレーキ故障の原因となります。  
ブレーキボックスは以下2種類の型式があり、接続するアクチュエーターにより使用するブレーキボックスが異なります。

- ブレーキボックス型式：RCB-110-RA13-0  
  ・ボールねじナット回転型タイプ NSシリーズの内 MZMS/MZMM/LZMS/LZMMタイプ  
  ・ロボシリンダー超高推力ロッドタイプ RCS2-RA13R  
  ・ISB/ISPB アクチュエーターの特注品で、DC24Vのブレーキ作動電圧のブレーキ付きモーターを使用しているもの

- ブレーキボックス型式：IA-110-DD-4  
  ・DDAのブレーキ付きタイプ

各配線の詳細は、[4.7.2 ブレーキボックスの配線]を参照してください。

#### ブレーキボックス型式：IA-110-DD-4 接続図
```mermaid
flowchart LR
    A[アクチュエーター] -->|モーターケーブル (CB-X2-MA□□□ / CB-XMC1-MA□□□)| B[ブレーキボックス (IA-110-DD-4)]
    A -->|エンコーダーケーブル (CB-X3-PA□□□)| B
    B -->|ブレーキボックス・アクチュエーター間接続ケーブル (CB-DDB-BK□□□)| B
    B -->|エンコーダーケーブル (CB-X3-PA010)| C[RSELシステム]
    C -->|接続| D[RCON-SC-1]
```

**図の説明:**  
- アクチュエーターからモーターケーブルとエンコーダーケーブルがブレーキボックスに接続される。  
- ブレーキボックスからエンコーダーケーブルを介してRSELシステムに接続される。  
- RSELシステムはRCON-SC-1に接続されている。  
- 型式ごとにケーブルの種類や接続形態が異なる。
