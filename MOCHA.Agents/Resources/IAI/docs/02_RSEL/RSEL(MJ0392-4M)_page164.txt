### 表
#### LED表示のパネル表記と状態説明

| パネル表記 | 表示色 | 状態   | 説明                         |
| :--------- | :----- | :----- | :--------------------------- |
| T RUN      | 緑     | 点灯   | 内部バス正常通信中           |
| T RUN      | 緑     | 点滅   | 初期化通信待ち、初期化通信失敗 |
| T RUN      | 橙     | 点灯   | 内部バス通信異常発生         |
| SYS        | 緑     | 点灯   | サーボON                     |
| SYS        | 緑     | 消灯   | サーボOFF                    |
| SYS        | 赤     | 点灯   | アラーム発生中、STOP入力中   |
