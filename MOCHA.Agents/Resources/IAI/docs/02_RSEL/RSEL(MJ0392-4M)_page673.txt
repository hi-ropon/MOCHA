### テキスト
#### 11.2 エラーレベル管理
アラームはエラーの内容により、以下の5種類にレベル分けされています。  
- メッセージレベル  
- 動作解除レベル  
- コールドスタートレベル  
- システムダウンレベル  
- シークレットレベル  

※シークレットレベルはエラーではなく、エラー発生時の原因解析のために必要に応じて内部ステータスをシークレットレベルとしてエラーリストに登録しています。

### 表
| エラーレベル | 発生元       | エラーNo.（HEX）  |
| :----------- | :----------- | :---------------- |
| シークレット (S) | コントローラー | 100～1AF          |
|               | TP/PC        | 1B0～1FF          |
| メッセージ (M) | コントローラー | 200～2FF          |
|               | TP/PC        | 300～3FF          |
| 動作解除 (A)   | コントローラー | 400～4FF          |
|               | TP/PC        | 500～5FF          |
| コールドスタート (C) | コントローラー | 600～6FF          |
|               | TP/PC        | 700～7FF          |
| システムダウン (D) | コントローラー | F00～FCF          |
|               | TP/PC        | FD0～FFF          |

PC：パソコン専用ティーチングソフト  
TP：ティーチングボックス
