#### 1ワードモジュールの使用例③
自動割付けにおいて、入力1024点、出力1024点の場合の使用モジュール数の計算例が示されています。

- Input 1wordモジュール使用数：1024 ÷ 16 = 64
- Output 1wordモジュール使用数：1024 ÷ 16 = 64

この場合、PLC側登録モジュールが64を超えるため使用できません。

そのため、PLC側登録モジュールが64を超える場合は、4ワードモジュールを使用する必要があると説明しています。
