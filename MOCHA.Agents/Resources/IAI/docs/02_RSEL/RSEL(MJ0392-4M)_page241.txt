### テキスト
#### (2) RCP6（高推力）/ RCP5（高推力）/ RCP4シリーズとRSELの接続方法
「4.4.4 RSEL接続ケーブル一覧 配線図B」に基づく接続例を示す。

#### (3) 上記以外のRCP/RCAシリーズとRSELの接続
「4.4.4 RSEL接続ケーブル一覧 配線図D」に基づく接続例を示す。

### 図
#### (2) RCP6/5/4シリーズとRSEL接続例
```mermaid
flowchart LR
    A[アクチュエーター (RCP6-SA8C) ※高推力] --> B[変換ケーブル CB-CAN-AJ002]
    B --> C[接続ケーブル CB-ADP(2)C-MPA□□□(-RB)]
    C --> D[RSELシステム]
    D --> E["“カチッ”と音がするまで挿込む"]
    C --> F["“カチッ”と音がするまで挿込みカバーをする"]
    B --> G["“カチッ”と音がするまで挿込む"]
```
- アクチュエーターから変換ケーブル、接続ケーブルを経由してRSELシステムに接続  
- 各接続部は「“カチッ”と音がするまで挿込む」ことを強調  
- 接続ケーブルのカバーも確実に装着することを指示

#### (3) RCP/RCAシリーズとRSEL接続例
```mermaid
flowchart LR
    A[アクチュエーター (RCA)] --> B[アクチュエーター付属モーターエンコーダーケーブル CB-ASEP2-MPA□□□]
    B --> C[コネクター変換ユニット RCM-CV-APCS]
    C --> D[接続ケーブル CB-ADPC(2)-MPA002(-RB)]
    D --> E[RSELシステム]
    E --> F["“カチッ”と音がするまで挿込む"]
    D --> G["“カチッ”と音がするまで挿込みカバーをする"]
    C --> H["“カチッ”と音がするまで挿込みカバーをする"]
```
- アクチュエーターからモーターエンコーダーケーブル、コネクター変換ユニット、接続ケーブルを経由してRSELシステムに接続  
- 各接続部は「“カチッ”と音がするまで挿込む」ことを強調  
- ケーブルのカバーも確実に装着することを指示

### 図の説明
- 各接続部は確実に接続し、音がするまで差し込むことが重要  
- ケーブルのカバーは必ず装着し、接続の安定性を確保する  
- RCP6/5/4シリーズは変換ケーブルを使用し、RCP/RCAシリーズはコネクター変換ユニットを使用する点が異なる
