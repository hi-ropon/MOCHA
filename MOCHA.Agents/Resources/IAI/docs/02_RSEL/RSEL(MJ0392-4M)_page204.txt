### テキスト
#### ブレーキリリーススイッチ
- ブレーキ強制解除用のスイッチで、上から1軸目、2軸目、3軸目、4軸目の順に配置されている。
- 通常使用時はNOM側に設定する。
- NOM側ではサーボONでブレーキ解除、サーボOFFでブレーキロックとなる。
- RLS側ではサーボON/OFFに関係なく強制解除となる（制御電源OFF時を除く）。

#### 警告
- ブレーキの解除は十分に注意して行うこと。不用意に解除するとスライダーやロッドの落下により、けがやアクチュエーター本体、ワークまたは装置の破損の原因となる。
- ブレーキ解除後は必ずブレーキを有効状態に戻すこと。解除状態のまま運転すると非常に危険であり、スライダーやロッドの落下によるけがや装置破損の原因となる。

### 表
| 記号 | 説明                         |
| :--- | :--------------------------- |
| RLS  | ブレーキリリース (Brake Release) |
| NOM  | ブレーキロック (Normal)      |
