### テキスト
#### 4.1.1 ユニット連結
RSELシステムのユニット連結方法について説明しています。ユニット連結はDINレール接続前に行うことが推奨されています。

連結手順は以下の通りです。  
1. 連結アッパーパーツ・連結ボトムパーツの操作部をパネル側に回し、パネル側端に位置させる。  
2. 破線〇部二箇所に連結アッパーパーツ・連結ボトムパーツ、点線□部4箇所に位置決めボス、計6箇所が嵌合するように二つのユニットの位置を合わせる。  
3. 位置合わせができたら、接続コネクター×2がしっかり接続されるように挿入する。  
4. 連結アッパーパーツ・連結ボトムパーツの操作部を背面側に回し、クリック感があるところまでしっかり回す。

### 図
- ユニットの写真2枚（正面と側面）  
- 連結アッパーパーツ操作部、連結ボトムパーツ操作部の位置と回転方向（パネル側⇔背面側）を赤矢印で示す。  
- 位置決めボス（点線□）と連結部（破線〇）の位置を示したユニットの平面図。

```mermaid
flowchart LR
    A[連結アッパーパーツ操作部] -- パネル側に回す --> B[パネル側端に位置]
    B --> C[ユニットの位置合わせ]
    C --> D[接続コネクター×2を挿入]
    D --> E[操作部を背面側に回す]
    E --> F[クリック感があるまで回す]
```

**図の説明:**  
- 操作部はパネル側に回して位置決めし、ユニットを合わせて接続コネクターを挿入。  
- その後、操作部を背面側に回して固定する。  
- 位置決めボスと連結部が6箇所で嵌合することで正確な連結が可能。
