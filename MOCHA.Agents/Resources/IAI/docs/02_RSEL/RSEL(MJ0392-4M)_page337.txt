### テキスト
#### 6.1 インターフェイス仕様
I/OスロットにPIOモジュールまたは各フィールドネットワークモジュールが装着可能であることを説明しています。  
装着されているモジュールの型式一覧が以下の表に示されています。  

また、SELユニットにPIO/SIO/SCON拡張ユニットやPIOユニットを連結することで、PIOの点数拡張やフィールドネットワークとPIOの併用が可能であることが記載されています。  
PIO/SIO/SCON拡張ユニットは1台、PIOユニットは最大8台（拡張ユニット使用時は7台）まで連結可能です。

### 表
#### 装着可能なモジュール一覧

| 型式           | 搭載モジュール               | 備考                         |
| :------------- | :--------------------------- | :--------------------------- |
| RSEL-G-E       | I/Oスロット 不使用           |                              |
| RSEL-G-NP      | PIO (NPN) 仕様              | NPN IN16/OUT16               |
| RSEL-G-PN      | PIO (PNP) 仕様              | PNP IN16/OUT16               |
| RSEL-G-CC      | CC-Link 接続仕様            |                              |
| RSEL-G-CC2     | CC-Link 接続仕様            | 二股コネクターが付属されます。 |
| RSEL-G-CIE     | CC-Link IE Field 接続仕様   |                              |
| RSEL-G-DV      | DeviceNet 接続仕様          |                              |
| RSEL-G-DV2     | DeviceNet 接続仕様          | 二股コネクターが付属されます。 |
| RSEL-G-EC      | EtherCAT 接続仕様           |                              |
| RSEL-G-EP      | EtherNet/IP 接続仕様        |                              |
| RSEL-G-PR      | PROFIBUS-DP 接続仕様        |                              |
| RSEL-G-PRT     | PROFINET IO 接続仕様        |                              |
