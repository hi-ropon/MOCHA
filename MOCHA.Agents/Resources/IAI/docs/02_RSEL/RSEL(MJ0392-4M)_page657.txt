### テキスト
#### 非常用プログラムの起動種別設定
非常用プログラムの起動種別は「その他パラメーターNo.5」で設定可能です。

#### 10.7.5 非常停止が解除されたら、自動復旧（再起動）したい
非常停止解除時に自動的に再起動（ソフトウェアリセット）を行い、プログラムを実行する設定について説明しています。

#### 10.7.6 非常停止が解除されたら、自動復旧（エラーリセット）したい
非常停止解除時に自動的にエラーリセットを行い、プログラムを実行する設定について説明しています。

### 表

#### 非常用プログラムの起動種別（その他パラメーターNo.5）

| パラメーターNo. | 内容                                                                                     |
| :-------------- | :--------------------------------------------------------------------------------------- |
| その他パラメーターNo.5 | 0：プログラム実行中、全動作解除要因発生時<br>1：全動作解除要因発生時（プログラム実行の有無と無関係に起動）<br>2：プログラム実行中、動作レベル以上のエラー発生時に、全動作解除要因発生時<br>3：動作レベル以上のエラー発生時に、全動作解除要因発生時（プログラム実行の有無と無関係に起動） |

#### 10.7.5 非常停止解除で自動復旧（再起動）設定

| パラメーターNo.       | 設定値 | 内容                                   |
| :-------------------- | :----- | :------------------------------------ |
| その他パラメーターNo.1 | 実行プログラムNo. |                                      |
| その他パラメーターNo.10 | 3      | 非常停止解除で再起動（ソフトウェアリセット）を行う |
| I/OパラメーターNo.33   | 1      | AUTOモード時、再起動後、プログラムを実行する |

#### 10.7.6 非常停止解除で自動復旧（エラーリセット）設定

| パラメーターNo.       | 設定値 | 内容                                               |
| :-------------------- | :----- | :------------------------------------------------ |
| その他パラメーターNo.1 | 実行プログラムNo. |                                                  |
| その他パラメーターNo.10 | 4      | 非常停止解除でエラーリセットを行う                 |
| I/OパラメーターNo.33   | 1      | AUTOモード時、エラーリセット後、プログラムを実行する |
| I/OパラメーターNo.44   | 0      | 駆動源遮断要因（非常停止）解除で、駆動源遮断を復旧する |
