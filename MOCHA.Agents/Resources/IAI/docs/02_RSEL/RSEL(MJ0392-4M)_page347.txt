### テキスト・表
#### I/OスロットステータスLEDの状態説明

| LED名称       | 色   | 状態   | 内容                                           |
|--------------|------|--------|------------------------------------------------|
| NS           | 緑   | 点灯   | 通常運転中                                     |
|              |      | 点滅1  | 診断イベント中                                 |
|              |      | 点滅2  | エンジニアリング・ツールがノードを識別中     |
| NS           | 橙   | 点灯   | 例外エラー                                     |
|              |      | 点滅1  | コンフィグレーションエラー                     |
|              |      | 点滅2  | IPアドレスエラー（IPアドレス未設定）          |
|              |      | 点滅3  | ステーション名エラー                           |
|              |      | 点滅4  | 内部エラー                                     |
| NS           | -    | 消灯   | 電源OFFまたは初期化未完了                      |
| NS           | 緑   | 点灯   | オンライン（RUN）（上位IOコントローラーがRUN状態） |
|              |      | 点滅   | オンライン（STOP）（上位IOコントローラーがSTOP状態） |
|              | -    | 消灯   | オフライン（電源OFFまたは上位IOコントローラーと未接続） |
| Link/Activity| 緑   | 点灯   | リンク確立（通信なし）                         |
|              |      | 点滅   | リンク確立（通信中）                           |
|              | -    | 消灯   | リンク確立していません                         |

### グラフ
#### LEDの点滅パターン（点滅1～4）

```mermaid
flowchart TD
    subgraph 点滅1
        A1[ON 0-0.75s] --> A2[OFF 0.75-1s]
    end
    subgraph 点滅2
        B1[ON 0-0.75s] --> B2[OFF 0.75-1s] --> B3[ON 1-1.25s] --> B4[OFF 1.25-1.5s]
    end
    subgraph 点滅3
        C1[ON 0-0.75s] --> C2[OFF 0.75-1s] --> C3[ON 1-1.25s] --> C4[OFF 1.25-1.5s] --> C5[ON 1.5-1.75s] --> C6[OFF 1.75-2s]
    end
    subgraph 点滅4
        D1[ON 0-0.75s] --> D2[OFF 0.75-1s] --> D3[ON 1-1.25s] --> D4[OFF 1.25-1.5s] --> D5[ON 1.5-1.75s] --> D6[OFF 1.75-2s] --> D7[ON 2-2.25s] --> D8[OFF 2.25-2.5s]
    end
```

**図の説明:**
- 点滅1は0.75秒点灯し、0.25秒消灯の1回点滅
- 点滅2は0.75秒点灯、0.25秒消灯、再度0.25秒点灯、0.25秒消灯の2回点滅
- 点滅3は点滅2のパターンにさらに1回点滅を追加した3回点滅
- 点滅4は点滅3のパターンにさらに1回点滅を追加した4回点滅

以上がI/OスロットステータスLEDの色・状態・点滅パターンの詳細説明です。
