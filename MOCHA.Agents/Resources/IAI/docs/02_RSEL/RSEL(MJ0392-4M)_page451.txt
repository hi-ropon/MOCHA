### テキスト
#### オフライン時の設定方法（ツールボックスから選択）

1. Auto Connect Project画面の「ツールボックス」の「グループ」から **SEL Controller** をクリックします。

2. 「ツールボックス」下部のリストから対応するデバイスを、ダブルクリック、もしくは「EtherCAT」タブの「Master」へドラッグ＆ドロップします。  
対応するデバイスは、EtherCAT入出力使用ポート数を参照して選択してください。

3. Auto Connect Project画面の「EtherCAT」タブに、スレーブが追加されます。

### 表
#### EtherCAT入出力使用ポート数とESIファイル対応デバイス

| EtherCAT 入出力使用ポート数 | ESIファイル対応デバイス               |
| :-------------------------- | :---------------------------------- |
| 8～256                      | SEL-ECT（32byte/32byte）             |
| 264～512                    | SEL-ECT（64byte/64byte）             |
| 520～768                    | SEL-ECT（96byte/96byte）             |
| 776～1024                   | SEL-ECT（128byte/128byte）           |

※入力使用ポート数・出力使用ポート数で対応するデバイスが異なる場合、ポート数の大きい方のデバイスを適用します。
