### テキスト
#### 6.1.2 PIO/SIO/SCON 拡張ユニット、PIOユニット

- 図中の赤枠①は「トータルフレーム通信ステータスLED」、赤枠②は「PIOユニットステータスLED」を示す。  
- PIOコネクターとSIO接続コネクター（PIO/SIO/SCON拡張ユニットのみ搭載）がある。

---

### 表1：インターフェイス仕様（LED名称・色・状態）

| LED名称                         | 色（緑） | 色（赤） | 状態                     |
| :----------------------------- | :------- | :------- | :----------------------- |
| ①トータルフレーム通信ステータスLED | 点滅     | 消灯     | コンフィグレーション通信中 |
|                                | 点灯     | 消灯     | 正常通信中               |
|                                | 消灯     | 点灯     | トータルフレーム通信エラー |
|                                | 消灯     | 消灯     | アップデート中           |
| ②PIOユニットステータスLED       | 点灯     | 消灯     | 正常動作中               |
|                                | 消灯     | 点灯     | 制御電源電圧低下         |
|                                | 消灯     | 点灯     | IO電源電圧低下           |
|                                | 点滅     | 消灯     | アップデート中           |

---

### 表2：SIOインターフェイス仕様

| 項目             | 内容                             |
| :--------------- | :------------------------------- |
| インターフェイス規格 | RS-485                          |
| 通信方式         | 調歩同期式 半二重通信             |
| ボーレート (bps)  | 9.6k / 19.2k / 38.4k / 57.6k / 115.2k / 230.4k |
| データ長         | 7または8ビット                   |
| スタートビット     | 1ビット                         |
| ストップビット     | なし、奇数または偶数             |
