### テキスト
#### ブレーキ解除スイッチコネクター1、2
- ケーブル側と使用コネクターの型式が示されています。
- スイッチは定格DC30V 1.5mA以上のものを使用する必要があります。

### 表
#### コネクター情報
| コネクター | 型式 | 備考 |
| :--- | :--- | :--- |
| ケーブル側 | XAP-02V-1<br>コンタクト BXA-001T-P0.6<br>（日本圧着端子） |  |
| 使用コネクター | S02B-XASS-1（日本圧着端子） |  |

#### ピン情報
| ピン番号 | 信号名 | 内容 | 適合電線 線径 |
| :--- | :--- | :--- | :--- |
| 1 | BKRML | ブレーキ解除スイッチ入力 | 0.08～0.3mm² (AWG28～22) |
| 2 | COM | ブレーキ解除スイッチ入力用コモン | 0.08～0.3mm² (AWG28～22) |
