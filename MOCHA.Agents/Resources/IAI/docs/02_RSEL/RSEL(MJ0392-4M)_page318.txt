### テキスト
#### 5.10.6 入出力信号のタイミング
PLCのシーケンスプログラムでアクチュエーター（エレシリンダー）を動かす際、制御信号をONにしてからエレシリンダーに届くまでの時間と、エレシリンダーの状態信号がPLCに返ってくるまでの最大応答時間は以下の式で表される。

- PLC → エレシリンダーの最大応答時間：Yt + Tt + Ft  
- エレシリンダー → PLCの最大応答時間：Gt + Tt + Xt  

各パラメータの意味は以下の通り。  
- Yt：マスター局 → リモートI/O局の伝送遅れ時間  
- Xt：リモートI/O局 → マスター局の伝送遅れ時間  
- Tt：SELユニット内処理時間 + 伝送時間（5ms）  
- Ft：エレシリンダーの入力フィルター時間（6～26ms）  
- Gt：RCON-EC-4の入力フィルター時間（6～26ms）  

※Ft、Gtは接続するエレシリンダーの機種によって異なる。  

マスター局とリモートI/O局間の伝送遅れ時間（Yt、Xt）については、各種フィールドネットワークマスターユニットやPLCの取扱説明書を参照すること。
