### テキスト
- (Rx, Ry, Rz) = (90deg, 90deg, -90deg)の場合の座標系の変化について説明しています。
- 回転は3段階に分けて行われます。
  1. X軸周りに90度回転
  2. Y軸周りに90度回転
  3. Z軸周りに-90度回転

### 図
```mermaid
flowchart LR
    A[(Rx, Ry, Rz) = (0, 0, 0)] -->|① X軸周り 90deg 回転| B[(Rx, Ry, Rz) = (90, 0, 0))]
    B -->|② Y軸周り 90deg 回転| C[(Rx, Ry, Rz) = (90, 90, 0))]
    C -->|③ Z軸周り -90deg 回転| D[(Rx, Ry, Rz) = (90, 90, -90))]

    subgraph 初期座標系
        A
    end
    subgraph 1回転後
        B
    end
    subgraph 2回転後
        C
    end
    subgraph 3回転後
        D
    end
```

**図の説明:**  
- 初期座標系はX, Y, Z軸が標準の向きにある状態。  
- ①の操作でX軸を中心に90度回転し、新しい座標系X', Y', Z'が得られる。  
- ②の操作でY軸を中心に90度回転し、新しい座標系X'', Y'', Z''が得られる。  
- ③の操作でZ軸を中心に-90度回転し、新しい座標系X''', Y''', Z'''が得られる。  
- 各回転は矢印と回転方向の赤い曲線で示されている。
