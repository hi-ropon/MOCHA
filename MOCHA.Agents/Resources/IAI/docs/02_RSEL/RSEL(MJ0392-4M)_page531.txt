### テキスト
#### 9.4.2 過負荷警告
本機能を活用することで、グリスの枯渇や部品の消耗などにより生じるモーター温度変化をモニターできます。  
あらかじめ設定した値を超えた場合に警告を出し、故障や不具合発生前の異変を検知可能です。

### 図
```mermaid
flowchart LR
    A[負荷増] --> B[モーター温度変化例（異常状態）]
    B --> C{過負荷警告発生}
    C -->|継続動作| D[メンテナンスを実施]
    C -->|警告出力| E[警告値設定範囲 50%～99%]
    B --> F{アラーム発生}
    F -->|停止| G[ロボシリンダー停止]
    G --> H[リセット操作必要]
    I[正常時のモーター温度変化例] --> J[時間経過]

    style A fill:#1f77b4,stroke:#333,stroke-width:2px,color:#fff
    style F fill:#d62728,stroke:#333,stroke-width:2px,color:#fff
    style C fill:#ff7f0e,stroke:#333,stroke-width:2px,color:#fff
    style D fill:#17becf,stroke:#333,stroke-width:2px,color:#fff
```

**図の説明:**  
- 青線は正常時のモーター温度変化例を示す。  
- オレンジ線は過負荷警告発生時の温度変化を示し、警告出力が行われる。  
- 赤線は過負荷アラーム検出値（100%）を超えた場合の温度変化で、アラームが発生しロボシリンダーが停止する。  
- 警告出力後にメンテナンスを実施することで故障や不具合を防止可能。  
- 警告値はパラメータロードレベル比で設定エラー検出レベルを100%とし、50%～99%の間で任意に設定できる。
