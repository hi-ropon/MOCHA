### テキスト
#### 2.2.2 ユニット接続制限
（1）電流制限値  
選定計算用の電流制限値は以下のとおりです。  
RSELシステムの構成に基づいて各ユニットの制御電源とモーター電源を計算した結果が、選定計算用の電流制限値を超えていないか確認してください。  
なお、SELユニットは計算対象に含みません。

#### 【制御電源】  
※SELユニットは計算対象に含みません。

### 表
#### 選定計算用の電流制限値

| 項目               | 選定計算用の電流制限値 |
| :----------------- | :--------------------- |
| 制御電源（CP）     | 9.0A 以下              |
| 24V モーター電源（MP） | 37.5A 以下             |
