### テキスト
#### CC-Link IE Field 接続仕様
CC-Link IE Fieldの接続仕様について説明しています。  
写真に示された接続部には、2つのポート（P1、P2）があり、それぞれにLINKとL ERのLEDが配置されています。  
また、MS（マスタースイッチ）とNS（ノードスイッチ）のスイッチが存在します。

#### インターフェイス仕様
CC-Link IE Fieldのインターフェイス仕様の概要を示しています。

### 表
#### インターフェイス仕様

| 項目               | 内容                                      |
|--------------------|-------------------------------------------|
| イーサネット規格   | IEEE802.3ab（1000BASE-T）準拠             |
| 通信速度           | 1Gbps                                    |
| 通信制御方式       | トークンパッシング方式                     |
| トポロジー（接続形態） | ライン、スター、リング、ライン・スター混在 |
| 局種別             | インテリジェントデバイス局                 |
| 最大接続台数       | 254台（マスター局とスレーブ局の合計）      |
| 最大ネットワーク数 | 239                                      |
| 最大局間距離       | 100mm                                    |

| LED名称 | 色   | 状態   | 内容                                   |
|---------|------|--------|--------------------------------------|
| NS      | 緑   | 点灯   | 正常動作中                           |

| NS      | 橙   | 消灯   | 正常動作中、電源未投入               |
| NS      | 緑   | 点灯   | サイクリック伝送を実施中             |
| NS      | 緑   | 点滅   | サイクリック伝送停止中               |
| NS      | 緑   | 消灯   | サイクリック伝送未実施、解列中、電源未投入 |
| NS      | 橙   | 点灯   | 受信データが異常（L ERと同時点灯）  |
| NS      | 橙   | 消灯   | 受信データが正常、電源未投入         |
