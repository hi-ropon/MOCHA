### テキスト
#### 5.3.3 簡易設定方法
初回起動時は、実装されたドライバーユニット内の物理軸順に論理軸が割当てられます。  
変更の必要がなければ、「はい」ボタンを押してください。  
個別に論理軸の割当てを行う場合は、「いいえ」ボタンを押してください。

### 図
```mermaid
flowchart TD
    A[初回起動画面] --> B[物理軸順に論理軸割当]
    B --> C{変更の必要あり?}
    C -->|いいえ| D[「はい」ボタンを押す]
    C -->|はい| E[「いいえ」ボタンを押す]
```

**図の説明:**  
- 初回起動画面では、物理軸順に論理軸が割り当てられる。  
- 変更不要なら「はい」ボタンを押す。  
- 変更する場合は「いいえ」ボタンを押す。  

図中の画面イメージには、各軸のユニット番号とドライバー番号、軸の状態（画像あり・No Image）が表示されている。
