### テキスト
#### (4) ブレーキリリーススイッチ
- ブレーキ強制解除用のスイッチ。
- 通常はNOM側に設定。NOM側ではサーボONでブレーキ解除、サーボOFFでブレーキロック。
- RLS側ではサーボON/OFFに関係なく強制解除（制御電源OFF時を除く）。

#### 警告
- ブレーキ解除は十分注意して行うこと。不用意に行うとスライダーやロッドの落下により、けがやアクチュエーター本体、ワーク、装置の破損原因となる。
- ブレーキ解除後は必ずブレーキを有効状態に戻すこと。解除状態のまま運転すると非常に危険。

#### (5) モーターコネクター
- アクチュエーターのモーターケーブルと接続するためのコネクター。

### 表
#### ブレーキリリーススイッチの記号と説明
| 記号 | 説明                         |
| :--- | :--------------------------- |
| RLS  | ブレーキリリース (Brake Release) |
| NOM  | ブレーキロック (Normal)      |
