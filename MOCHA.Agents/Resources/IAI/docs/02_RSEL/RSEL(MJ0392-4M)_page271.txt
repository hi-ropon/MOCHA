### テキスト
#### 電源入力端子台
ブレークボックスへ電源を供給するには、電源入力端子台への配線を行う必要があります。

### 表
#### 電源入力端子台の名称とピンアサイン
| ピン番号 | 信号名 | 説明         |
| :------- | :----- | :----------- |
| 1        | L      | AC電源 L相   |
| 2        | N      | AC電源 N相   |
| 3        | PE     | 保護接地     |

#### 推奨電源サイズと圧着端子
| 項目       | 説明                         |
| :--------- | :--------------------------- |
| 推奨電線サイズ | L,N : AWG18 以下<br>PE : AWG14 以下 |
| 圧着端子     | 内径 : φ4.0mm 以上、外径 : φ8.5mm 以下 |
