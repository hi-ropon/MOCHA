### テキスト
#### 停止入力信号のテストパルス
外部から入力されるテストパルスのオフ時間は1ms以下にしてください。

#### 動作シーケンス
●通常動作の場合の停止入力信号1（/SRI1）、停止入力信号2（/SRI2）、モーターへのエネルギー供給、外部機器モニター出力信号（EDM）、モーター回転数の動作タイミングを示しています。

### 表
#### 電気的仕様

| 項目                     | 仕様           | 備考               |
|--------------------------|----------------|--------------------|
| 停止入力信号（SRI）      |                |                    |
| オン入力電圧範囲         | 24V±10%        |                    |
| オフ入力電圧範囲         | 0-2V           |                    |
| 入力電流                 | 7.6mA (Typ)    | 1ch当たりの値です。 |
| リアクションタイム       | 8ms以下        |                    |
| 外部機器モニター出力信号（EDM） |          |                    |
| 電圧範囲                 | 24V±10%        |                    |
| 出力電流                 | 100mA (Max)    |                    |

### 図
#### 動作シーケンス（タイミングチャート）

```mermaid
flowchart LR
    subgraph 停止入力信号1 (/SRI1)
        A1[ON] -->|DRV STOP解除| B1[DRV STOP状態]
        A2[OFF] --> B1
        B1 --> C1[DRV STOP解除]
    end

    subgraph 停止入力信号2 (/SRI2)
        A3[ON] -->|DRV STOP解除| B2[DRV STOP状態]
        A4[OFF] --> B2
        B2 --> C2[DRV STOP解除]
    end

    subgraph モーターへのエネルギー供給
        A5[ON] --> B3[供給中]
        A6[OFF] --> B4[停止]
    end

    subgraph 外部機器モニター出力信号 (EDM)
        A7[ON] --> B5[出力中]
        A8[OFF] --> B6[停止]
    end

    subgraph モーター回転数
        D1[回転数] -->|内部回路遅延時間：数ms| D2[減速開始]
        D2 --> D3[ダイナミックブレーキ]
    end

    %% 矢印でタイミング関係を示す
    B1 -.-> B3
    B2 -.-> B3
    B3 -.-> B5
    B4 -.-> B6
```

**図の説明:**  
- 停止入力信号1、停止入力信号2がONになるとDRV STOP状態が解除される。  
- モーターへのエネルギー供給は停止入力信号の状態に連動し、OFFになると停止状態に移行。  
- 外部機器モニター出力信号（EDM）はモーターのエネルギー供給状態に連動。  
- モーター回転数は内部回路の遅延時間（数ms）を経て減速し、ダイナミックブレーキがかかる。  
- 停止入力信号のリアクションタイムは8ms以下であることが示されている。
