### テキスト
#### 配線例について
図のように配線することで、スイッチOFF時にドライバー機能を停止できます。  
注1：0VはSELユニットの24V電源と共通にしてください。

#### 注意事項
- ドライバー停止機能（DRV STOP）を使用して複数ユニットを同時に停止解除する場合は、ユニットごとに停止解除信号を1ms以上ずらして使用してください。  
  （過電流エラーを誤検出する可能性があります）  
- 製造番号B20978900以降の製品であれば、同時に停止解除を行っても問題ありません。  
- ドライバー停止機能はSafe Torque Off機能（EN/IEC61800-5-2）ではありません。

### 図
```mermaid
flowchart LR
    A[24V電源] --> B[スイッチ]
    B --> C[コネクター端子3,4 (SRI1- / SRI1+)]
    B --> D[コネクター端子5,6 (SRI2- / SRI2+)]
    E[0V (SELユニットの24V電源と共通)] --> C
    E --> D
    F[コネクター端子7,8 (EDM-, EDM+)]
```
**図の説明:**  
- 端子3,4は停止入力信号1（SRI1- / SRI1+）  
- 端子5,6は停止入力信号2（SRI2- / SRI2+）  
- 端子7,8は外部機器モニター出力信号（EDM-, EDM+）
