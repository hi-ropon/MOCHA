### テキスト
#### 4.1.2 ファン取付け
1. RSELシステムとファンユニットの取付け向きを合わせます。
2. ファンユニットのつめを電源ユニットへ右図のようにひっかけます。
3. ファンユニットをRSELシステム正面側に倒します。
4. ファンユニットを上から押え、はめ込みます。

※ RCON-PS2-3、RCON-SC-1には出荷時からファンユニットが取付けられています。

### 画像説明
- 1枚目の写真はファンユニットのつめを電源ユニットにひっかけている様子を示しています。
- 2枚目の写真はファンユニットを正面側に倒し、上から押さえてはめ込む作業を示しています。

    B --> C[ファンユニットをRSELシステム正面側に倒す]
    C --> D[ファンユニットを上から押さえてはめ込む]

**図の説明:**
- ファンユニットの取り付けは4つのステップで行う。
- 向きを合わせ、つめを引っ掛け、倒してから押し込む順序で作業を進める。
