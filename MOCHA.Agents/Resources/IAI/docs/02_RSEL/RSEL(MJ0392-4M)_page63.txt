### テキスト
#### 2.1.3 型式
(1) SELユニットの型式構成について説明している。

型式は「RSEL - G - □ - □ - □」の形式で表され、各項目は以下の意味を持つ。
- G：安全カテゴリー対応タイプ（固定）
- I/O種類
- I/Oケーブル長さ
- オプション

### 表
#### SELユニット型式の各要素

| 項目 | 内容 |
| :--- | :--- |
| タイプ | G：安全カテゴリー対応タイプ |
| I/O種類 | E：未使用<br>NP：PIO仕様（NPN 16/16）<br>PN：PIO仕様（PNP 16/16）<br>CC：CC-Link接続仕様<br>CC2：CC-Link接続仕様（二股コネクター付属）<br>CIE：CC-Link IE Field接続仕様<br>DV：DeviceNet接続仕様<br>DV2：DeviceNet接続仕様（二股コネクター付属）<br>EC：EtherCAT接続仕様<br>EP：EtherNet/IP接続仕様<br>PR：PROFIBUS-DP接続仕様<br>PRT：PROFINET IO接続仕様 |
| I/Oケーブル長さ | 0：ケーブルなし<br>2：2m（標準）<br>3：3m<br>5：5m |
| オプション | FU*：ファンユニット装着（装着数1～5指定）<br>TRN：ターミナルユニットなし |
