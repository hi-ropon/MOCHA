### テキスト
#### 4.4.8 フィールドネットワークの配線
本書では三菱電機製PLCを上位PLCとして、CC-Linkマスターユニットと接続する場合の例を紹介しています。

【上位PLCとRSELシステム2台の接続】

### 図
```mermaid
flowchart LR
    A[上位PLC (お客様で用意)] -->|CC-Link専用ケーブル (お客様で用意)| B[RSELシステム1台目]
    B -->|CC-Link専用ケーブル (お客様で用意)| C[RSELシステム2台目]
    B --> D[アクチュエーター]
    C --> E[アクチュエーター]
    B -.-> F[終端抵抗 (SELユニットに付属)]
    C -.-> F
    A -.-> G[CC-Linkマスターユニット (お客様で用意)]
```

**図の説明:**  
- 上位PLCは三菱電機製PLCで、お客様が用意します。  
- 上位PLCとRSELシステム間はCC-Link専用ケーブルで接続します（ケーブルもお客様で用意）。  
- RSELシステムは2台接続され、それぞれにアクチュエーターが接続されています。  
- CC-Linkマスターユニットは上位PLCに接続されます。  
- 終端抵抗はSELユニットに付属していますが、CC-Linkマスターユニットの終端抵抗はお客様で用意する必要があります。
