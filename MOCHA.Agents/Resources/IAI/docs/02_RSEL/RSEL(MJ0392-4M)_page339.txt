### 表
#### インターフェイス仕様

| 項目               | 内容                                                                                  |
|--------------------|---------------------------------------------------------------------------------------|
| CC-Linkバージョン   | Ver.1.10 / Ver.2.00（CC-Link使用I/O点数によって決定）                                |
| 局種別             | リモートデバイス局                                                                     |
| 占有局数           | 1～4局（CC-Link使用I/O点数によって決定）                                             |
| 拡張サイクリック設定 | 1/4倍（CC-Link使用I/O点数によって決定）                                              |
| 局番               | 1～64（I/Oパラメーターによるソフトウェア設定）                                        |
| 通信速度           | 10M / 5M / 2.5M / 625k / 156kbps（I/Oパラメーターによるソフトウェア設定）             |

---

#### I/OスロットステータスLED

| LED名称 | 色   | 状態 | 内容                                                                                   |
|---------|------|------|----------------------------------------------------------------------------------------|
| RUN     | 緑   | 点灯 | ネットワーク加入後のリフレッシュ＆ポーリング正常受信またはリフレッシュ正常受信          |
| ERR     | 橙   | 点灯 | エラー発生（CRCエラー／局番設定エラー／ボーレート設定エラー）                          |
| ERR     | 橙   | 点滅 | リセット解除時の局数およびボーレート設定値から値が変化                                |
