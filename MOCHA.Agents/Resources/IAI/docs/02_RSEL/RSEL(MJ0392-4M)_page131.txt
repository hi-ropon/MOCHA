### テキスト
#### ジョグスイッチ
ジョグ運転用のスイッチで、Iは1軸目、IIは2軸目を表します。  
ドライバーユニットパラメーターNo.194「JOGスイッチ」を“0”に設定すると本スイッチが有効になります（初期値は“0：有効”）。  

JOG+側にスイッチを傾けると＋方向に、JOG−側に傾けると−方向にジョグ動作を行います。  
スイッチを傾け続けると段階的にジョグ速度が速くなります。ただし、原点復帰未完了の場合はジョグ速度は原点復帰未完了時最大JOG速度（MC共通パラメーターNo.3）となります。  

サーボON中にジョグスイッチでジョグ動作をすると完了後もサーボONのままですが、サーボOFF中の場合はサーボONしてからジョグ動作を行い、完了後にサーボOFFします。  

ジョグスイッチの操作はMANUモード時のみ有効で、AUTOモード時は無効です。  
また、ティーチングツールでアクチュエーターを操作できる画面を開いている場合もジョグスイッチは無効となり、ジョグスイッチによる動作中に操作できる画面を開いた場合はアクチュエーターは減速停止します。  

#### 注意事項
- ティーチングツールでアクチュエーター操作画面を開いているときに通信が切断されると、ジョグスイッチ操作が無効のままになります。  
- ジョグスイッチ操作を有効に戻すには、RSELシステムの電源再投入かソフトウェアリセットが必要です。  
- ジョグスイッチ操作にはRSEL軸の設定が必要です。詳細は「5.3 軸設定」を参照してください。

### 表
| 記号  | 説明                         |
| :---- | :--------------------------- |
| JOG+  | ＋方向にジョグ動作（原点の反対方向） |
| JOG−  | －方向にジョグ動作（原点方向）       |
