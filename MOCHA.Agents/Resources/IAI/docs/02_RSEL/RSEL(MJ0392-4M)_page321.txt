### テキスト
#### 5.10.8 設定例
SELユニットにEC接続ユニット（RCON-EC-4）を接続し、エレシリンダーを動作させるまでの設定例を説明しています。

##### 1) RSELシステム構成と入出力ポート構成例
（1）RSELシステム構成について説明しています。

### 表
#### RSELシステム構成

| ユニット構成           | 台数 | 設定内容など                                                                                   |
|------------------------|------|----------------------------------------------------------------------------------------------|
| RSELユニット           | -    | ・自動割付け<br>・フィールドバス入力64点/出力64点有効<br>・入力/出力ともフィールドバス最終16点を2台目のEC接続ユニット制御用に使用 |
| 24Vドライバーユニット  | 4    | 各ユニット2軸接続                                                                             |
| PIOユニット            | 2    |                                                                                              |
| EC接続ユニット         | 2    | 各ユニット2軸接続（残り2軸はEC未接続）<br>EC接続ユニット#1：1軸目/4軸目にEC接続 SELプログラムで制御<br>EC接続ユニット#2：1軸目/2軸目にEC接続 上位機器（PLC）から制御 |

### 図
```mermaid
flowchart TD
    A[上位機器（PLC）] -->|フィールドネットワーク (CC-Link、EtherNet/IPなど)| B[RSELシステム]

    end

```

**図の説明:**
- 上位機器（PLC）からフィールドネットワークを介してRSELシステムに接続
- RSELシステムはSELユニット、PIOユニット、ドライバーユニット、EC接続ユニット、ターミナルユニットで構成
- EC接続ユニット#1はSELプログラムで制御される軸（第9軸、第12軸）を持つ
- EC接続ユニット#2は上位機器（PLC）から制御される軸（第13軸、第14軸）を持つ
