### テキスト
#### 5.3.6 軸グループ設定
軸グループ設定とは、複数の軸を「軸グループ」という単位で管理し、独立したポジションデータを設定できる機能です。これにより、同じ軸構成が2つある場合でも同じ内容のポジションデータで動作させることが可能になります。また、組み合わせ軸と付加軸のポジションデータを分けて管理できるため、管理が容易になります。

例えば、3軸の組み合わせが2組ある場合、従来はAxis1～Axis6にポジションデータを入力していましたが、グループ化により3軸のポジションデータを2つ持つことが可能になります。SELプログラムでは使用するグループを宣言し、使用するポジションデータを指定します。

「軸グループ設定」ボタンをクリックすると、論理軸エリアに2つのグループが表示され、それぞれに軸画像を割り当てられるようになります。同じ軸グループは物理軸エリアの軸画像の背景色が同じになります。

#### 図5.3-8 軸グループ1と2に3軸ずつ割り当てた状態
- 2つの軸グループが表示され、それぞれに3軸ずつ割り当てられている。
- 物理軸エリアの軸画像の背景色がグループごとに異なり、視覚的に区別可能。

```mermaid
flowchart TD
    A[軸グループ設定ボタンをクリック] --> B[論理軸エリアに2つのグループ表示]
    B --> C[各グループに軸画像をドラッグ＆ドロップで割り当て]
    C --> D[物理軸エリアの軸画像背景色がグループごとに変わる]
    D --> E[SELプログラムで使用するグループを宣言・指定]
```

**図の説明:**
- ユーザーは「軸グループ設定」ボタンを押すことで、論理軸エリアに複数の軸グループを表示できる。
- 各グループに対して軸画像を割り当てることで、ポジションデータをグループ単位で管理可能。
- 物理軸エリアの背景色でグループの識別が容易になる。
- SELプログラムで使用するグループを指定し、制御に反映させる。
