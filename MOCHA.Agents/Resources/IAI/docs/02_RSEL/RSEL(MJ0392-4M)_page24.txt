### テキスト
#### 注意表示について
各機種の取扱説明書には、安全事項を「危険」、「警告」、「注意」、「お願い」の4つのランクに分けて表示しています。

### 表
#### 注意表示のレベルと内容

| レベル | 危害・損害の程度 | シンボル |
| :--- | :--- | :--- |
| 危険 | 取扱いを誤ると、死亡または重傷に至る危険が差迫って生じると想定される場合 | ![危険シンボル](画像内の危険マーク) 危　険 |
| 警告 | 取扱いを誤ると、死亡または重傷に至る可能性が想定される場合 | ![警告シンボル](画像内の警告マーク) 警　告 |
| 注意 | 取扱いを誤ると、傷害または物的損害の可能性が想定される場合 | ![注意シンボル](画像内の注意マーク) 注　意 |
| お願い | 傷害の可能性はないが、本製品を適切に使用するために守っていただきたい内容 | ![お願いシンボル](画像内のお願いマーク) お願い |
