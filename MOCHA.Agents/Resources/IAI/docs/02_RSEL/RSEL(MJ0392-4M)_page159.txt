### テキスト
#### 3.4.2 型式
(1) 型式の見方

【200Vドライバーユニット】  
200Vドライバーユニットの型式は以下の通り。

```
RCON － SC － 1
シリーズ  タイプ  軸数
          AC200Vサーボモータータイプ
          SC：60-750Wモーター
          1：1軸
```

##### タイプ：200V AC 60-750Wモーター 1軸

| モーター出力(W) | 説明                         |
|-----------------|------------------------------|
| 60              | 60Wサーボモーター            |
| 100             | 100Wサーボモーター           |
| 100S            | 100Wサーボモーター（LSA用） |
| 150             | 150Wサーボモーター           |
| 200             | 200Wサーボモーター           |
| 200S            | 200Wサーボモーター（LSA、DD用） |
| 300S            | 300Wサーボモーター（LSA用） |
| 400             | 400Wサーボモーター           |
| 600             | 600Wサーボモーター           |
| 750             | 750Wサーボモーター           |

【200Vドライバー用ファンユニット】  
型式は「RCON－FUH」
