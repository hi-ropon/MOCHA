### テキスト
#### 外部入力仕様
SELユニット、ターミナルユニットの外部入力仕様について説明しています。

### 表
#### 外部入力仕様一覧

| 項目         | 仕様                              |
|--------------|---------------------------------|
| 入力点数     | 16点                            |
| 入力電圧     | DC24V±10%                      |
| 入力電流     | 4mA/1回路                      |
| オン/オフ電圧 | オン電圧：最小 DC18V（3.5mA）<br>オフ電圧：最大 DC6V（1mA） |
| 絶縁方式     | フォトカプラー                  |

### 図
#### NPN仕様回路図
```mermaid
flowchart LR
    A[+24V] -->|ダイオード| B(各入力)
    B --> C[5.6kΩ 抵抗]
    C --> D[ロジック回路]
    D --> E[フォトカプラー]
    E --> F[内部回路]
    B --> G[0V]
```

#### PNP仕様回路図
```mermaid
flowchart LR
    A[+24V] --> B(各入力)
    B --> C[5.6kΩ 抵抗]
    C --> D[ロジック回路]
    D --> E[フォトカプラー]
    E --> F[内部回路]
    B -->|ダイオード| G[0V]
```

**図の説明:**  
- NPN仕様は入力側にダイオードがあり、5.6kΩの抵抗を経てロジック回路に入力されます。  
- PNP仕様は入力側にダイオードが0V側に配置されており、同様に5.6kΩの抵抗を経てロジック回路に入力されます。  
- 両仕様ともフォトカプラーを介して内部回路に信号が伝達されます。
