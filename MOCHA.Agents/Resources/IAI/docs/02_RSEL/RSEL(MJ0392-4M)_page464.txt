### テキスト
#### 7.1.3 各軸座標系
直交型6軸ロボットの各軸座標系は、ロボットの構成軸に設定された座標系です。

### 図
#### CRS-XB（XYベース固定）
- ロボットのXYベースが固定された構成
- 各軸C1、C2、C3の正方向を赤い矢印で示す

#### CRS-XG（XYベース固定＋ガントリー）
- ロボットのXYベースが固定され、さらにガントリー構造を持つ構成
- 各軸C1、C2、C3の正方向を赤い矢印で示す

#### 軸の回転方向と座標系の模式図
```mermaid
flowchart TD
    subgraph 軸方向
        direction_Y["C2 (Y軸) 方向 (+) / (-)"]
        direction_R["R軸 回転方向 (+) / (-)"]
        direction_B["B軸 回転方向 (+) / (-)"]
        direction_T["T軸 回転方向 (+) / (-)"]
    end

    direction_Y --> R_axis["R軸 回転"]
    R_axis --> B_axis["B軸 回転"]
    B_axis --> T_axis["T軸 回転"]

    Mechanical_Interface["メカニカルインターフェイス"]
    B_axis --> Mechanical_Interface
    T_axis --> Mechanical_Interface
```

**図の説明:**  
- 赤い矢印は各軸の正方向を示す。  
- 回転軸（R軸、B軸、T軸）はそれぞれ正方向・負方向の回転を示す。  
- メカニカルインターフェイスはB軸とT軸の回転軸原点に位置する。
