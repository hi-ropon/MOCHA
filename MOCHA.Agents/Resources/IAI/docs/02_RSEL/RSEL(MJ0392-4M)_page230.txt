#### 4.3 展開接続図（例）
##### 4.3.1 単相仕様電源回路

### 図
```mermaid
flowchart TD
    A[単相電源入力] --> B[サーキットブレーカー]
    B --> C[200Vドライバー用モーター電源コネクター]
    C --> D[200V電源ユニット (RCON-PS2-3)]
    D --> E[200Vドライバーユニット (RCON-SC)]

    subgraph RSELシステム
        F[SELユニット (RSEL-G)]
        G[24Vドライバーユニット (RCON-PC/PCF/AC/DC)]
    end

    H[24V電源] --> G
    G --> F

    style F fill:#fff,stroke:#000,stroke-width:1px
    style G fill:#ccc,stroke:#000,stroke-width:1px
    style D fill:#fff,stroke:#000,stroke-width:1px
    style E fill:#ccc,stroke:#000,stroke-width:1px

    subgraph 電源ライン
        I[24V] -->|モーター電源コネクター| F
        J[24V] -->|制御電源コネクター| F
        K[0V] -->|共通接地| F
    end
```

**図の説明:**
- 単相電源（L, N）からサーキットブレーカーを経て200Vドライバー用モーター電源コネクターに接続
- 200V電源ユニット（RCON-PS2-3）および200Vドライバーユニット（RCON-SC）に電源供給
- 24V電源は24Vドライバーユニット（RCON-PC/PCF/AC/DC）に供給され、SELユニット（RSEL-G）に接続
- 各ユニットの電源コネクターは24V、0V、FG（フレームグラウンド）で構成されている
