### テキスト
#### PROFIBUS-DP
D-subコネクター9ピン（ソケット）を使用した通信インターフェースについて説明しています。

### 画像
D-sub 9ピンコネクターの写真が掲載されており、ピン番号が示されています。

### 表
#### コネクタータイプとピン配列

| ピン番号 | 信号名 | 説明 |
| :---: | :--- | :--- |
| 1 | NC | 未接続 |
| 2 | NC | 未接続 |
| 3 | B-Line | 通信ラインB（RS-485） |
| 4 | RTS | 送信要求 |
| 5 | GND | シグナルGND（絶縁） |
| 6 | +5V | +5V出力（絶縁） |
| 7 | NC | 未接続 |
| 8 | A-Line | 通信ラインA（RS-485） |
| 9 | NC | 未接続 |
| ハウジング | シールド | ケーブルシールド（コントローラー内部FGと接続） |

#### LED名称ごとの色・状態・内容

| LED名称 | 色 | 状態 | 内容 |
| :--- | :---: | :--- | :--- |
| MS | 緑 | 点灯 | 初期化完了 |
| MS | 緑 | 点滅 | 初期化完了（診断イベントあり） |
| MS | 橙 | 点灯 | エラー発生（例外エラー） |
| NS | 緑 | 点灯 | オンライン状態 |
| NS | 緑 | 点滅 | オンライン状態（クリアー状態） |
| NS | 橙 | 点灯 | エラー発生（パラメーター化エラー、PROFIBUSコンフィギュレーションエラー） |
