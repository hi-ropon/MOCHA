### テキスト
#### 4.4 配線方法
(2) -1. EC接続ユニットへ直接接続する場合  
超小型エレシリンダーのアクチュエーターケーブルをEC接続ユニット（RCON-EC）に接続します。

#### 注意
- コネクターには挿込み向きがあります。  
  コネクターの形状に合わせ向きを揃え、クリック感があるまで挿入してください。

### 図
#### ケーブルの接続図
```mermaid
flowchart LR
    A[アクチュエーター本体] --> B[アクチュエーターケーブル]
    B --> C[RCON-EC]
    C --> D[ツメ]
    D -.-> E[ツメを左側に合わせ、クリック感があるまで挿入]
```

**図の説明:**  
- アクチュエーター本体からアクチュエーターケーブルを介してEC接続ユニット（RCON-EC）に接続されている。  
- コネクターのツメを左側に合わせ、クリック感があるまでしっかり挿入する必要がある。  
- 図中の拡大写真でツメの位置が示されている。
