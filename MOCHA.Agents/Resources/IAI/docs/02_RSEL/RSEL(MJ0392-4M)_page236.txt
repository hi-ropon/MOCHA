### テキスト
#### 4.4.3 アクチュエーターの型式確認
【アクチュエーター型式とドライバーユニット型式の確認】  
アクチュエーターを接続する前に、ドライバーユニットとの組み合わせが一致しているか必ず確認すること。  
接続可能なアクチュエーター型式は、ドライバーユニット正面下側の製番シールに記載されている。  

用意する物  
- RSELユニット  
- アクチュエーター  
- モーター・エンコーダーケーブル  

### 図
```mermaid
flowchart LR
    A[アクチュエーター製番シール内 "MODEL" 記載の型式] --> B{型式一致?}
    B -- はい --> C[ドライバーユニット製番シール "Actuator" 部記載の型式]
    B -- いいえ --> D[接続不可]
    
    subgraph アクチュエーター側面
        A
    end
    
    subgraph ドライバーユニット正面
        C
    end
```

**図の説明:**  
- アクチュエーター側面の製番シールに記載された"MODEL"の型式と、  
- ドライバーユニット正面の製番シール"Actuator"部に記載された型式が一致しているかを確認する。  
- 一致していれば接続可能、不一致の場合は接続不可。
