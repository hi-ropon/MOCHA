### テキスト
#### 3.3.2 型式
（1）型式の見方

【200V電源ユニット】  
型式は「RCON - PS2 - 3 - オプション」で表されます。  
- PS2：200V電源ユニットのタイプ  
- 3：三相／単相 200Vの電源電圧  
- オプション：ターミナルユニットの有無（例：TRNはターミナルユニットなし）

【200V用ターミナルユニット】  
型式は「RCON - GW - TRS」で表されます。

### 画像説明
200V用ターミナルユニットの写真に以下のユニットがラベル付けされています。

| ユニット名               | 型式           |
|--------------------------|----------------|
| ファンユニット           | RCON-FU        |
| 200Vドライバー用ファン   | RCON-FUH       |
| 200Vドライバーユニット   | RCON-SC-1      |
| 200V電源ユニット         | RCON-PS2-3     |
| 200V用ターミナルユニット | RCON-GW-TRS    |

※最大接続ユニット数は1ユニットです。
