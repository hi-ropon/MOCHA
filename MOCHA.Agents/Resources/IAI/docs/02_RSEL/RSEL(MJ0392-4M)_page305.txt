### テキスト
#### プログラム起動手順
1. プログラムNo.の入力は、RSELシステムが起動後（正常起動＝RDY信号ON後）、0.01秒以上経過してから行うこと。  
2. 起動するプログラムNo.は、I/OパラメーターNo.30で設定した入力ポートにセットする。  
3. プログラムNo.をセットしてから0.05秒以上経過後に、入力ポート000（プログラムスタート）をONにする。  
4. 入力ポート000（プログラムスタート）は0.1秒以上ONにする。  
5. 設定したプログラムが起動する。

```mermaid
    B --> C[プログラムスタート（上位から出力）]
    C --> D[プログラム実行]

    style D fill:#b3e5fc

    %% 時間間隔の注釈
    A -.->|② Min0.01s| B
    B -.->|③ プログラム番号セット| C
    C -.->|④ Min0.05s| C
    C -.->|⑤ Min0.1s| D
```

**図の説明:**  
- READY出力（RSEL出力）がONしてから最低0.01秒経過後にプログラムNo.を入力ポートにセットする。  
- プログラムNo.セット後、0.05秒以上経過してから入力ポート000（プログラムスタート）をONにする。  
- 入力ポート000は0.1秒以上ONに保持し、その後プログラムが実行される。
