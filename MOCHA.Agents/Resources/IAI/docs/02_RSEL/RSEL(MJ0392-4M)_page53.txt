### テキスト
#### 1.4.2 放熱および設置
制御盤の大きさ、RSELシステムの配置、冷却などを考慮し、周囲温度が0～55℃になるように設計・製作を行うこと。  
（ファンユニットなしの場合はディレーティングがある）  
詳細は「1.3.5 使用温度範囲（1-10ページ）」を参照。  
簡易アブソユニットや拡張ユニットに接続したSCON-CBを同じ制御盤に設置する場合は、周囲温度が0～40℃になるように設計・製作を行うこと。  
特に簡易アブソユニット（バッテリー）周辺は低温・高温で性能が低下するため、できるだけ常温（約20℃）が推奨温度。

### 表
| 項目   | 仕様                     |
| :----- | :----------------------- |
| 設置   | 設置方向                 | 垂直設置（排気側が上側） |
|        | 設置方法                 | DINレール取付け          |
|        | 設置条件                 | 下図参照                 |
| 接地   |                         | 機能接地のみ             |

### 図（設置間隔と排気・吸気のクリアランス）
- 左図：コントローラー正面図  
  - 両側面に10mm以上の空間を確保  
- 右図：コントローラー側面図  
  - 上方：100mm以上  
  - 後方：100mm以上  
  - 下方：50mm以上  
  - 排気は上側、吸気は下側に配置  

**注意点**  
複数のコントローラーを上下に配置する場合、下側のコントローラーの排気が上側のコントローラーの吸気にならないようにすること。
