### グラフ
「過熱」アラームの原因説明に添付されたグラフは、使用温度環境（℃）に対する動作Duty（%）を示しています。

```mermaid
flowchart LR
    A[使用温度環境(℃)] --> B[動作Duty(%)]
    B --> C{ファンの有無}
    C -->|ファンあり| D[高い動作Dutyを維持可能]
    C -->|ファンなし| E[50℃付近から動作Dutyが急激に低下]
```

**グラフ説明:**  
- 青線（ファンなし）は50℃を超えると動作Dutyが急激に低下する。  
- 赤点線（ファンあり）はより高温環境でも動作Dutyを維持可能。
