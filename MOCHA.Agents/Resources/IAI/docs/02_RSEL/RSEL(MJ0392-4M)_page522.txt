### テキスト
#### PSA-24 通信モニターのパラメーター設定
PSA-24を接続し、RSEL経由で電源のステータスデータをモニターする場合、パラメーターNo.61「電源ユニット台数」に接続するPSA-24の台数を設定します。

パラメーター設定後、パソコン専用ティーチングソフトを使用する場合は、「モニター(M)」→「電源ユニット情報(P)」を選択すると、電源ユニット情報が表示されます。

#### ステータスデータ
モニター可能なステータスデータは以下の項目です。

### 表
#### その他パラメーター

| No. | パラメーター名称       | 出荷時設定値 | 入力範囲 | 単位 | 備考                                  |
|:----|:----------------------|:-------------|:---------|:-----|:-------------------------------------|
| 61  | 電源ユニット接続台数 | 0            | 0～5     |      | 0：未接続 1～5：電源ユニット接続台数 |

#### ステータスデータ一覧

| 項目               | 内容                                                                                                                         |
|--------------------|------------------------------------------------------------------------------------------------------------------------------|
| 出力電圧           | 出力電圧のAD変換値。本電源は負荷に応じて出力電圧を変動させているため、モニター値が時々刻々と変化するが異常ではない。          |
| 補助巻線電圧       | 電源内部の制御電源電圧のAD変換値。出力電圧側の負荷に応じて出力電圧と同様に変化するが異常ではない。                            |
| ピークホールド電圧 | 出力電圧のAD変換値。AD変換実行時に常に比較を行い、最大電圧値を保存する。                                                    |
| 出力電流           | 出力電流のAD変換値。瞬時値を表す。                                                                                            |
| ピークホールド電流 | 出力電流のピーク値。AD変換実行時に常に比較を行い、最大電流値を保存する。                                                    |
| 負荷率             | 出力電流の積分値と定格出力電流の比率を百分率で表した値。100%超過時は過負荷エラーとして出力電圧が遮断される。                  |
| ピークホールド負荷率 | 負荷率計算時に常に比較を行い、最大負荷率を保存する。                                                                        |
| FAN回転速度        | ファンから入力されるパルス信号よりファン回転数を算出した値。                                                                  |
| PCB温度             | 二次側の出力コンデンサー近傍の温度センサーのAD変換値。                                                                        |
| 通算通電時間       | 通電時間をCPUでカウントし、定期的に内部メモリーに保持した値を積算した時間。                                                    |

### 画像
電源ユニット情報表示画面のスクリーンショットがあり、No.0～No.4までの各電源ユニットの以下の情報が表形式で表示されている。  
- 出力電圧[V]  
- 補助巻線電圧[V]  
- ピークホールド電圧[V]  
- 出力電流[A]  
- ピークホールド電流[A]  
- 負荷率[%]  
- ピークホールド負荷率[%]  
- FAN回転速度[rpm]  
- PCB温度[℃]  
- 通算通電時間[Day:Hour]
