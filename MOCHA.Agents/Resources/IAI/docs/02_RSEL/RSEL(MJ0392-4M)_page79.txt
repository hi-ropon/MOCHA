### テキスト
#### 200V電源ユニット（RCON-PS2-3）の接続制限
RCON-SC-1を使用して200Vアクチュエーターを接続する場合のモーター電源に関する制約について説明しています。接続するアクチュエーターのモーターW数の合計が、選定計算用の制限値を超えていないかを確認する必要があります。

また、特定のアクチュエーターは計算用モーターW数で計算することが求められています。

### 表
#### 200Vモーター電源の選定計算用制限値

| 項目                 | 選定計算用の制限値 |
| :------------------- | :----------------- |
| 200Vモーター電源（単相） | 1,600W             |
| 200Vモーター電源（三相） | 2,400W             |
