### テキスト
#### 直交型6軸ロボットアブソリュートリセット手順

③ 干渉回避姿勢へジョグ画面が表示されるので、各軸の「←（－）」「→（＋）」をクリックし、原点復帰時に干渉しない位置・姿勢に移動させる。  
“ジョグ速度”および“インチング距離”は必要に応じて変更可能。干渉しない位置・姿勢に移動後、「OK」をクリックする。

④ 非常停止、ブレーキリリースを行い、R軸の基準姿勢近傍まで近づける。  
R軸の基準姿勢は図8.3-10を参照。基準姿勢でストッパーボルトを取り付ける（図8.3-11参照）。  
ストッパーボルトを取り付けた状態で、ブレーキロック、非常停止解除を行い、確認画面の「OK」をクリックする。

```mermaid
flowchart TD
    A[干渉回避姿勢ジョグ画面表示] --> B[各軸の「←（－）」「→（＋）」をクリック]
    B --> C[干渉しない位置・姿勢に移動]
    C --> D[ジョグ速度・インチング距離を必要に応じて変更]
    D --> E[「OK」クリック]

    F[非常停止・ブレーキリリース実施] --> G[R軸基準姿勢近傍まで移動]
    G --> H[基準姿勢でストッパーボルト取り付け]
    H --> I[ブレーキロック・非常停止解除]
    I --> J[確認画面の「OK」クリック]
```

**図の説明:**  
* 干渉回避姿勢ジョグ画面で各軸の移動ボタンを操作し、干渉しない位置・姿勢に調整する。  
* 非常停止解除のためにブレーキリリースを行い、R軸の基準姿勢近傍まで移動し、ストッパーボルトを取り付ける。  
* その後、ブレーキロック・非常停止解除を行い、確認画面で「OK」を押す。

### 画面イメージ説明
- 図8.3-8 干渉回避姿勢ジョグ画面  
  ジョグ速度（mm/sec）とインチング距離（mm）を設定し、X軸～T軸までの各軸に対して「←（－）」「→（＋）」ボタンが配置されている。

- 図8.3-9 確認画面  
  非常停止、ブレーキリリース、位置決め治具取り付け、ブレーキロック、非常停止解除の手順を案内し、「OK」「キャンセル」ボタンがある。
