### テキスト
#### PROFINET IO接続仕様
- マスターのコンフィグレーション時には、4ワード（=8バイト）モジュールまたは1ワード（=2バイト）モジュールを選択可能。
- 1ワードモジュール使用時は、モジュール数を64以下（入出力使用点数合計1024点以下）に制限。
- InputモジュールからOutputモジュールの順で登録が必要。

#### 例
- 入力208点、出力400点を使用する場合のモジュール使用数計算例。
- Input 4wordモジュール使用数：208÷64=3.25 → 4個
- Output 4wordモジュール使用数：400÷64=6.25 → 7個
- よって、PLC側登録モジュールはInput 4wordを4個、Output 4wordを7個使用。
