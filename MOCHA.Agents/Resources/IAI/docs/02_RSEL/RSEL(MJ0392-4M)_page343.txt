### テキスト・表
#### I/OスロットステータスLEDの状態説明

| LED名称       | 色   | 状態           | 内容                                      |
|--------------|------|----------------|-------------------------------------------|
| RUN          | 緑   | 点灯           | EtherCAT通信 “OPERATION” 状態             |
|              |      | ブリンキング※1 | EtherCAT通信 “PRE-OPERATION” 状態         |
|              |      | シングルフラッシュ※2 | EtherCAT通信 “SAFE-OPERATION” 状態       |
|              | 橙   | 点灯           | 故障（回復不可能）                         |
|              | -    | 消灯           | EtherCAT通信 “INIT” 状態、電源供給なし     |
| ERR          | 橙   | 点灯           | 回復不可能な異常                           |
|              |      | ブリンキング※1 | ネットワーク構成異常                       |
|              |      | ダブルフラッシュ※3 | ウォッチドグタイムアウト                   |
|              | -    | 消灯           | 異常なし、電源供給なし                     |
| Link/Activity| 緑   | 点灯           | リンク中（通信トラフィック未検出）         |
|              |      | フリッカリング※4 | リンク中（通信トラフィック検出）           |
|              | -    | 消灯           | リンクなし、電源供給なし                   |

### グラフ
LEDの点滅パターンのタイミング図：

```mermaid
flowchart LR
    blinking[ブリンキング※1]
    single_flash[シングルフラッシュ※2]
    double_flash[ダブルフラッシュ※3]
    flickering[フリッカリング※4]

    blinking -->|ON 200ms| off1[OFF 200ms]
    off1 --> blinking

    single_flash -->|ON 200ms| off2[OFF 1000ms]
    off2 -->|ON 200ms| off3[OFF]
    
    double_flash -->|ON 200ms| off4[OFF 200ms]
    off4 -->|ON 200ms| off5[OFF 1000ms]
    off5 --> off6[OFF]

    flickering -->|ON 50ms| off7[OFF 50ms]
    off7 --> flickering
```

**図の説明:**
- ※1 ブリンキング：200ms点灯→200ms消灯を繰り返す
- ※2 シングルフラッシュ：200ms点灯→1000ms消灯→200ms点灯→消灯
- ※3 ダブルフラッシュ：200ms点灯→200ms消灯→200ms点灯→1000ms消灯
- ※4 フリッカリング：50ms点灯→50ms消灯を高速で繰り返す
