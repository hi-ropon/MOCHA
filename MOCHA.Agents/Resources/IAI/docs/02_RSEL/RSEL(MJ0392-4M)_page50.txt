### テキスト
#### 1.3.4 制御部仕様
RSELシステムの制御部仕様について説明しています。

### 表
#### 制御部仕様一覧

| 項目                     | 内容                                                                                         |
|--------------------------|----------------------------------------------------------------------------------------------|
| 安全回路構成             | 二重化可能                                                                                   |
| 駆動源遮断方式           | 内部半導体遮断、外部全軸一括遮断                                                             |
| 非常停止入力             | B接点入力（外部給電、二重化可、内部給電より選択可能）                                       |
| イネーブル入力           | B接点入力（外部給電、二重化可、内部給電より選択可能）                                       |
| 速度設定                 | 1mm/s～上限はアクチュエーター仕様による                                                     |
| 加減速設定               | 0.01G～上限はアクチュエーター仕様による                                                    |
| 軸グループ数             | 2（1グループ最大8軸）                                                                        |
| プログラム言語           | スーパーSEL言語                                                                             |
| プログラム数             | 512（入力信号ではBCD指定で99、バイナリー指定では255まで指定可）                             |
| プログラムステップ数     | 20,000ステップ                                                                             |
| マルチタスクプログラム数 | 16プログラム                                                                                |
| ポジション数             | 36,000ポジション（軸グループ数により可変）                                                |
| シンボル定義数           | 2,000                                                                                      |
| シンボル使用数           | 20,000                                                                                     |
| シンボル文字数           | 半角40文字、全角20文字                                                                     |
| ステップコメント文字数   | 半角32文字、全角16文字                                                                     |
| データ記憶素子           | FlashROM + 不揮発性RAM（FRAM）（バッテリー不要）                                           |
| データ入力方法           | ティーチングボックス、またはパソコン専用ティーチングソフト                                 |
| 対応ティーチングボックス | TB-02/TB-03                                                                               |
| 対応パソコン専用ソフト   | XSELパソコン専用ティーチングソフト                                                        |
| 標準入出力               | （I/Oスロット選択）16IN/16OUT PIOボード（NPN/PNP）                                       |
| 拡張入出力               | PIOユニット接続を最大8台接続可能                                                          |
| シリアル通信機能         | ティーチングポート（最大115.2kbps）、PSA-24通信、USBポート（Mini-B:12Mbpsフルスピード）、Ethernet（RJ-45）、SIO接続コネクター（最大230.4kbps） |
| 対応フィールドネットワーク | CC-Link、CC-Link IE Field、DeviceNet、EtherCAT、EtherNet/IP、PROFIBUS-DP、PROFINET IO（スレーブ局） |
| Ethernet                 | 10/100BASE-T（RJ-45コネクター）、XSELシリアル通信プロトコル（フォーマットB）※1、SELメッセージ通信 |
| USB                      | USB2.0（Mini-B）、XSELシリアル通信プロトコル（フォーマットB）※1                          |
| カレンダー機能           | 保持時間：約10日、充電時間：約100時間                                                     |
| SDカード                 | SD/SDHC（アップデート機能のみ使用）                                                       |

※1: XSELシリアル通信プロトコル（フォーマットB）は1ポートのみ通信可能。ティーチングポート（優先度：高）、USB、Ethernet（優先度：低）の優先度として、低い優先度は無応答。
