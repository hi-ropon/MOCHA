### テキスト
#### 3.5.2 型式
(1) 型式の見方について説明。

【SCON拡張ユニット】  
型式は「RCON - EXT」

【PIO/SIO/SCON拡張ユニット】  
型式は「RCON - EXT - □ - □」  
- 3つ目の□はI/O仕様を示す  
- 4つ目の□はI/Oケーブル長を示す

【PIOユニット】  
型式は「RCON - □ - □」  
- 2つ目の□はI/O仕様を示す  
- 3つ目の□はI/Oケーブル長を示す

### 表
#### I/O仕様
| 記号 | 内容          |
| :--- | :------------ |
| NP   | I/O NPN仕様   |
| PN   | I/O PNP仕様   |

#### I/Oケーブル長
| 記号 | 内容          |
| :--- | :------------ |
| 0    | ケーブルなし  |
| 2    | 2m（標準）    |
| 3    | 3m            |
| 5    | 5m            |
