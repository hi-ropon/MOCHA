#### CC-Link IE Field 接続仕様

マスターのコンフィグレーション時に、リモート入出力領域（RX/RY）およびリモートレジスター入出力領域（RWr/RWw）のサイズを以下の算出方法に従って設定する必要があります。

##### 【算出方法】
1. 入力使用ポート数（I/OパラメーターNo.14）、リモートレジスター入力ワード数（I/OパラメーターNo.184）を用いて、リモート入力ビット数を計算する。

```
リモート入力ビット数 = 入力使用ポート数（I/OパラメーターNo.14）
                      - リモートレジスター入力ワード数（I/OパラメーターNo.184） × 16
```

2. 出力使用ポート数（I/OパラメーターNo.15）、リモートレジスター出力ワード数（I/OパラメーターNo.185）を用いて、リモート出力ビット数を計算する。

```
リモート出力ビット数 = 出力使用ポート数（I/OパラメーターNo.15）
                      - リモートレジスター出力ワード数（I/OパラメーターNo.185） × 16
```

3. 表Aより、リモート入力ビット数、リモート出力ビット数のいずれか大きいビット数が含まれる範囲用のサイズを検索する。検索結果がリモート入出力領域（RX/RY）のサイズとなる。

4. 表Bより、リモートレジスター入力ワード数とリモートレジスター出力ワード数を比較し、大きい方の値が含まれる範囲用のサイズを検索する。検索結果がリモートレジスター入出力領域（RWr/RWw）のサイズとなる。
