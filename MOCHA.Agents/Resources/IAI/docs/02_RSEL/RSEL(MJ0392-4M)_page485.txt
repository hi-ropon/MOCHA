### テキスト
#### ツール座標系の設定と使用方法
- ツール座標系定義データの設定・番号選択とポジションタイプに「Rect（直交）」を設定し、ポジション指定移動命令を実行する。  
- SELプログラム内でツール座標系No.を選択する場合はSLTL命令を使用。  
- 選択したツール座標系選択No.はプログラム終了後や電源再投入後も値が保持される。  
- ツール座標系オフセット量を設定する場合はDFTL命令を使用。

#### SELプログラム例
| 命令 | パラメータ | 説明 |
| --- | --- | --- |
| VEL | 100 | 速度100mm/s |
| ACC | 0.3 | 加速度0.3G |
| DCL | 0.3 | 減速度0.3G |
| SLTL | 1 | ツール座標系No.1を選択 |
| SLWK | 1 | ワーク座標系No.1を選択 |
| MOVP | 1 | ポジションNo.1へPTP動作で移動 |
| MOVP | 2 | ポジションNo.2へPTP動作で移動 |
| EXIT | - | プログラム終了 |
