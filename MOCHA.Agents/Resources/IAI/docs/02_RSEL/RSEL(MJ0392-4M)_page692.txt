### 表
#### エラー表（エラーNo.31C～321）

| エラーNo. | エラー名称               | 内容・原因 / 対処方法                                                                                              |
| --------- | ------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| 31C       | 軸No.エラー              | 【内容・原因】<br>①有効でない軸No.を指定した<br>【対処方法】<br>①有効な軸No.を指定する                         |
| 31D       | 有効軸無しエラー         | 【内容・原因】<br>①設定と実際に接続された軸数が一致していない<br>②有効な軸が存在しない<br>【対処方法】<br>①軸コンフィグレーションを行う<br>②ドライバーユニット構成を確認する |
| 31E       | 各軸系時ティーチ禁止エラー | 【内容・原因】<br>①各軸座標系で、現在位置取込み操作を行った<br>【対処方法】<br>①各軸座標系で、現在位置取込み操作を行わない |
| 31F       | ペアデータ不整合         | 【内容・原因】<br>パラメーター“ソフトリミット＋側”、“ソフトリミット－側”（対となっているデータ）に不適切な値が入力された。<br>【対処方法】<br>パラメーター編集画面で以下の条件を満たすよう入力値を訂正すること<br>①パラメーター“ソフトリミット＋側”入力値＝“ソフトリミット－側”＋0.90mm以上<br>②パラメーター“ソフトリミット－側”入力値＝“ソフトリミット＋側”－0.90mm以下 |
| 320       | 絶対値過小               | 【内容・原因】<br>“位置決め幅”に不適切な値が入力された。<br>【対処方法】<br>“位置決め幅”に設定できる値はアクチュエーター最小位置決め幅より大きな値を入力すること。<br>最小位置決め幅は以下の式で求められる。<br>■最小位置決め幅[mm] = ボールねじリード長[mm] ÷ アクチュエーターエンコーダーパルス数[pulse] × 係数（PCON系=3、ACON、SCON系=1） |
| 321       | 最低速度未満入力警告     | 【内容・原因】<br>アクチュエーターボールねじリードとエンコーダー分解能によって決まる「最低速度」未満の速度が入力された（アラームではない）。<br>【対処方法】<br>警告が出ない速度（最低速度以上）に変更することを推奨。<br>最低速度[mm/s] = アクチュエーターボールねじリード[mm] ÷ アクチュエーターエンコーダーパルス数[pulse] ÷ 0.001[s]<br>最低速度未満での動作は不安定動作や異音・振動の原因となる。 |
