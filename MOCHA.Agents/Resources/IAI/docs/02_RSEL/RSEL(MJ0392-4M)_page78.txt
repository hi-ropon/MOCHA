### テキスト
#### 24Vモーター電源
24Vモーター電源の電流容量について、各例における定格電流の計算結果を示しています。  
また、注意事項として以下が記載されています。  
- 全軸が同時に加減速動作だけを行う動作パターンで、動作デューティーが100%の場合は、モーター電源は最大電流値で計算する必要がある。  
- モーター電源をより細かく算出する場合は、「カリキュレーターソフト」を使用し、アクチュエーターの動作条件と動作パターンを設定すると必要な電源容量を自動計算できる。  
- 操作方法など詳細は「カリキュレーター取扱説明書（MJ0381）」を参照すること。

### 表
#### 24Vモーター電源の例（定格電流計算結果）

| 例 | 内容 | 電流計算 | 判定 |
| :--- | :--- | :--- | :--- |
| 例6 | RCON-PC（パワーコンあり）×8軸 | 2.3A×8軸=18.4A | OK |
| 例7 | RCON-PCF×7軸、または6軸の場合 | 5.7A×7軸=39.9A | NG |
|  |  | 5.7A×6軸=34.2A | OK |
| 例8 | RCON-PC（パワーコンなし）、RCON-AC、RCON-DC | どのユニットも定格電流が低く、8軸で電流は制限値を超えない | OK |
| 例9 | RCON-PCF×6軸、RCON-PC（パワーコンあり）×2軸 または、RCON-PCF×6軸、RCON-AC×2軸 | 5.7×6＋2.3×2=38.8A | NG |
|  |  | 5.7×6＋1.3×2=36.8A | OK |

### 注意事項
- 全軸が同時に加減速動作だけを行う動作パターンで、動作デューティーが100%の場合は最大電流値で計算する必要がある。  
- 詳細な電源容量計算には「カリキュレーターソフト」を使用し、動作条件と動作パターンを設定することで自動計算が可能。  
- 操作方法は「カリキュレーター取扱説明書（MJ0381）」を参照。
