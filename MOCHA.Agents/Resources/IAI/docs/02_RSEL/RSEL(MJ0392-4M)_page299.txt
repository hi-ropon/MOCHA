### テキスト
#### 5.7.6 メモリーマップ
RSELには、保持メモリーによる保存領域とフラッシュメモリーによる保存領域があります。  
パソコン専用ティーチングソフトまたはティーチングボックスからデータ転送を行っても、メモリーに書き込まれるだけであり、電源OFFやコントローラーリセットによりデータは消去されます。  
確実に保存するためには、フラッシュ書込みを行う必要があります。  

プログラム、パラメーター、シンボルは再起動時にフラッシュメモリーから読み込まれます。  
フラッシュメモリーへ書き込みをしないと、メモリーのデータは編集前のデータに戻ってしまいます。  
コントローラーは常にメインCPUメモリーのデータに従い動作します（パラメーターは除く）。  

（注）ポジションデータ先頭から10%は不揮発性RAMに保存し、残りはフラッシュメモリーに保存します。  
ポジションデータごとのコメントはポジションNo.に関係なく最大10,000ポジションに使用でき、フラッシュメモリーに保存されます。

```mermaid
    C -->|リセット時転送| B

    D -->|リセット時転送| B
    D -->|転送| G[ドライバーユニット不揮発性RAM]

    H[電源ONの間データを保持 リセットによりデータ消去] --> B

    I[電源OFFしてもデータを保持] --> F
    I --> G

    subgraph PCソフト TP
    end
```

**図の説明:**  
- パソコンやティーチングボックスで編集したデータはRSELユニットのRAMメモリに転送される。  
- RAMメモリ内のプログラム、パラメーター、シンボルはフラッシュメモリーに書き込まれ、再起動時に読み込まれる。  
- 電源ON時はRAMメモリのデータを保持し、リセットで消去される。  
- 電源OFF時でもフラッシュメモリーと不揮発性RAMにデータが保持される。  
- ポジションデータの先頭10%は不揮発性RAMに保存され、残りはフラッシュメモリーに保存される。
