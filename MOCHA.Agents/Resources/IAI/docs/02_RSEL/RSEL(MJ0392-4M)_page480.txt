### テキスト
#### 7.2.6 座標系定義データ
（1）ワーク座標系定義データ  
ワーク座標系定義データは15種類登録可能で、番号0を選択するとワーク座標無し（ベース座標系）で動作します。  
ワーク座標系定義データのフォーマットは画像の編集画面の通りです。  

SELプログラム内でワーク座標系定義データを操作するにはDFWK・GTWK命令を使用します。  
詳細は「SEL言語プログラミングマニュアル（MJ0224）」を参照してください。

### 画像
- 「座標系定義」編集画面のスクリーンショット  
  - No. 1～15までの行があり、各行にX, Y, Z, Rx, Ry, Rzの数値を入力可能  
  - 入力範囲は-99,999.999～99,999.999

### 表
| 項目 | 入力内容 | 備考 |
| :--- | :--- | :--- |
| X | ベース座標系からのX軸オフセット量 | ワーク座標の原点位置の指定 |
| Y | ベース座標系からのY軸オフセット量 | ワーク座標の原点位置の指定 |
| Z | ベース座標系からのZ軸オフセット量 |  |
| Rx | ベース座標系からのX軸回転量 | ワーク座標軸の姿勢の指定 |
| Ry | ベース座標系からのY軸回転量 | ワーク座標軸の姿勢の指定 |
| Rz | ベース座標系からのZ軸回転量 |  |
