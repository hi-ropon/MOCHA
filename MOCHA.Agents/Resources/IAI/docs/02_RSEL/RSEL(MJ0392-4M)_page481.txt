### テキスト
#### ツール座標系定義データ
- ツール座標系定義データは15種類登録可能。  
- ツール座標系No.0を選択した場合はツールなし（メカニカルインターフェイス座標系）として動作。  
- データフォーマットは画像の編集画面の通り。

#### SELプログラムでの操作
- ツール座標系定義データを操作するにはDFTL・GTTL命令を使用。  
- 詳細は「SEL言語プログラミングマニュアル（MJ0224）」を参照。

### 画像
- ツール座標系定義編集画面のスクリーンショット。  
- 15行のデータ入力欄があり、各行にX, Y, Zのオフセット量とRx, Ry, Rzの回転量を入力可能。  
- 入力範囲は-99,999.999～99,999.999。

### 表
#### 各セルの入力内容

| 項目 | 入力内容 | 備考 |
| :--- | :--- | :--- |
| X | メカニカルインターフェイス座標系からのX軸オフセット量 | ツール座標の原点位置の指定 |
| Y | メカニカルインターフェイス座標系からのY軸オフセット量 | ツール座標の原点位置の指定 |
| Z | メカニカルインターフェイス座標系からのZ軸オフセット量 | ツール座標の原点位置の指定 |
| Rx | メカニカルインターフェイス座標系からのX軸回転量 | ツール座標軸の姿勢の指定 |
| Ry | メカニカルインターフェイス座標系からのY軸回転量 | ツール座標軸の姿勢の指定 |
| Rz | メカニカルインターフェイス座標系からのZ軸回転量 | ツール座標軸の姿勢の指定 |
