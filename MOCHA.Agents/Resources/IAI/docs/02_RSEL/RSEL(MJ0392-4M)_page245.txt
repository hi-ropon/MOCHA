### テキスト
#### インターフェイスボックスからRCON-ECへ接続する場合
インターフェイスボックスを使用してEC接続ユニットへ接続する場合の配線方法を示しています。

#### 注意
- コネクターには挿込み向きがあります。
- コネクターの形状に合わせ向きを揃え、クリック感があるまで挿入してください。

### 図
#### ケーブルの接続図
```mermaid
flowchart LR
    A[アクチュエーター本体] -->|アクチュエーターケーブル| B[インターフェイスボックス変換ケーブル]
    B -->|クリック感があるまで挿入してください| C[インターフェイスボックス]
    C -->|コネクターの溝を合わせて挿入してください| D[電源・I/Oケーブル]
    D -->|クリック感があるまで挿入してください| E[RCON-EC]
```

### 図の説明
- アクチュエーター本体からアクチュエーターケーブルを介してインターフェイスボックス変換ケーブルに接続
- インターフェイスボックス変換ケーブルはクリック感があるまでインターフェイスボックスに挿入
- インターフェイスボックスから電源・I/Oケーブルを接続し、コネクターの溝を合わせて挿入
- 電源・I/Oケーブルはクリック感があるまでRCON-ECに挿入
