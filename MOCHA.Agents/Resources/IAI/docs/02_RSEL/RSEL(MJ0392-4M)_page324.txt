### テキスト
#### 5.10 エレシリンダーの運転
【3】パラメーター設定

(1) I/Oパラメーターの設定  
I/OパラメーターNo.1「入出力ポート割付け種別」、No.14「I/O1フィールドバスリモート入力使用ポート数」、No.15「I/O1フィールドバスリモート出力使用ポート数」を以下のように設定します。

(2) オプションユニット有効設定  
SELユニットに接続するオプションユニット（PIO/SIO/SCON拡張ユニット、PIOユニット、EC接続ユニット）の台数を設定します。設定はパソコン専用ティーチングソフトまたはティーチングボックス（TB-02/03）を使用し、I/OパラメーターNo.186「オプションユニット接続台数」に台数を入力します。  
なお、SCON拡張ユニット（RCON-EXT）は接続台数に含みません。

例：  
オプションユニット（PIO/SIO/SCON拡張ユニット1台、PIOユニット1台、EC接続ユニット2台）が4台あるため、I/OパラメーターNo.186には「4」を設定します。

### 表
#### I/Oパラメーター設定一覧

| No. | パラメーター名称                         | 設定値 | 備考               |
| :-- | :------------------------------------- | :----- | :----------------- |
| 1   | 入出力ポート割付け種別                   | 1      | “自動割付け”設定   |
| 14  | I/O1 フィールドバスリモート入力使用ポート数 | 64     |                    |
| 15  | I/O1 フィールドバスリモート出力使用ポート数 | 64     |                    |
