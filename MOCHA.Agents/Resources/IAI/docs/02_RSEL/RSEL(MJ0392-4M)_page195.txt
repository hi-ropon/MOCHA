### テキスト
#### 3.5.7 外形寸法
対象ユニットは以下の3種類です。
- 【SCON拡張ユニット】RCON-EXT
- 【PIO/SIO/SCON拡張ユニット】RCON-EXT-NP/PN-*
- 【PIOユニット】RCON-NP/PN-*

これらのユニットの外形寸法は同一であることが記載されています。

### 表
| 項目     | 仕様                                                                                   |
| :------- | :------------------------------------------------------------------------------------- |
| 外形寸法 | W22.6mm × H115mm × D95mm                                                             |
| 質量     | 約96g（SCON拡張ユニット）<br>約110g（PIO/SIO/SCON拡張ユニット）<br>約105g（PIOユニット） |
| 外観図   | 下図参照                                                                             |

※備考：SCON拡張ユニット、PIO/SIO/SCON拡張ユニット、PIOユニットの外形寸法は同じである。

### 図
対象ユニットの外観図が複数方向（正面、側面、上面など）から示されており、寸法（mm単位）が詳細に記載されています。

- 幅22.6mm、高さ115mm、奥行き95mmが主な寸法
- その他、細かい部分寸法（例：3.5mm、7mm、1mm、39.6mm、35.2mm、98.8mmなど）も明示

（図は寸法線付きのユニット外観図であり、各方向からの形状と寸法を示す技術図面）
