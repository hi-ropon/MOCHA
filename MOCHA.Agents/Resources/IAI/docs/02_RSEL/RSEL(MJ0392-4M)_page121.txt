### テキスト
#### LEDの点滅パターン
- 点滅1～4までの4種類の点滅パターンが示されている。
- 各点滅はON/OFFの時間が秒単位で示されており、点滅回数が増えるごとにON/OFFの繰り返しが増加する。

### 図（点滅パターン）
```mermaid
flowchart LR
    subgraph 点滅1
        A1[OFF 0秒] --> A2[ON 0.75秒] --> A3[OFF 1秒]
    end
    subgraph 点滅2
        B1[OFF 0秒] --> B2[ON 0.75秒] --> B3[OFF 1秒] --> B4[ON 1.25秒] --> B5[OFF 1.5秒]
    end
    subgraph 点滅3
        C1[OFF 0秒] --> C2[ON 0.75秒] --> C3[OFF 1秒] --> C4[ON 1.25秒] --> C5[OFF 1.5秒] --> C6[ON 1.75秒] --> C7[OFF 2秒]
    end
    subgraph 点滅4
        D1[OFF 0秒] --> D2[ON 0.75秒] --> D3[OFF 1秒] --> D4[ON 1.25秒] --> D5[OFF 1.5秒] --> D6[ON 1.75秒] --> D7[OFF 2秒] --> D8[ON 2.25秒] --> D9[OFF 2.5秒]
    end
```

### 表
#### LED名称ごとの色・状態・内容

| LED名称 | 色     | 状態   | 内容                                         |
| :------ | :----- | :----- | :------------------------------------------- |
| MS      | 緑     | 点灯   | 正常動作中                                   |
|         | 緑     | 点滅1  | 診断イベントが存在                           |
|         | 緑     | 点滅2  | エンジニアリングツールがノード識別に使用   |
|         | 橙     | 点灯   | 重故障（EXCEPTION）状態                      |
|         | 橙     | 点滅1  | コンフィグレーションエラー                   |
|         | 橙     | 点滅2  | IPアドレスエラー（IPアドレス未設定）        |
|         | 橙     | 点滅3  | ステーション名エラー（ステーション名未設定）|
|         | 橙     | 点滅4  | 内部重故障エラー                             |
|         | 緑/橙  | 消灯   | 電源オフ状態、モジュールがセットアップ/イニシャル状態 |
| NS      | 緑     | 点灯   | 通信成立、RUN状態                            |
|         | 緑     | 点滅   | 通信成立、STOP状態                           |
|         | 緑/橙  | 消灯   | 電源オフ、コネクションなし                   |
