### テキスト
#### 10.7.18 全単軸アクチュエーターが原点位置であることを出力したい
全単軸アクチュエーターが原点位置であることを確認できます。

#### 10.7.19 全単軸アクチュエーターが原点復帰完了状態であることを出力したい
全単軸アクチュエーターが原点復帰完了状態であることを確認できます。

### 表
#### 10.7.18 全単軸アクチュエーターが原点位置であることを出力したい場合の状態と出力ポート対応

| 状態                         | 出力ポート 304 |
| :--------------------------- | :------------- |
| 全有効軸が原点位置           | ON             |
| 有効軸のいずれかが原点位置ではない | OFF            |

#### 10.7.18 パラメーター設定

| パラメーターNo.       | 設定値 | 内容                             |
| :-------------------- | :----- | :------------------------------ |
| I/O パラメーター No.50 | 1      | 全単軸アクチュエーター原点位置で出力に設定 |

#### 10.7.19 全単軸アクチュエーターが原点復帰完了状態であることを出力したい場合の状態と出力ポート対応

| 状態           | 出力ポート 304 |
| :------------- | :------------- |
| 原点復帰完了   | ON             |
| 原点復帰未完了 | OFF            |

#### 10.7.19 パラメーター設定

| パラメーターNo.       | 設定値 | 内容                                   |
| :-------------------- | :----- | :------------------------------------ |
| I/O パラメーター No.50 | 2      | 全単軸アクチュエーター原点復帰完了状態で出力に設定 |
