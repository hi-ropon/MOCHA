### テキスト
#### 2.1.5 直交型6軸ロボット（CRSシリーズ）用セット型式
直交型6軸ロボット用にコントローラーを用意する場合、セット型式での指定が可能です。

（1）セット型式  
RSEL - タイプ - I/O種類 - I/Oケーブル長さ - オプション

（3）付加軸  
直交型6軸ロボット（CRSシリーズ）6軸分にさらに2軸分のドライバーユニットを追加接続可能。追加接続するドライバーユニットは別途手配が必要。

### 表
#### セット型式の構成要素

| 項目 | 内容 |
| :--- | :--- |
| タイプ | CRSシリーズの各種タイプ（例：SXBA、SXGA、SXBBなど） |
| I/O種類 | 未使用、PIO仕様（NPN/PNP）、CC-Link接続仕様、DeviceNet接続仕様、EtherCAT接続仕様、EtherNet/IP接続仕様、PROFIBUS-DP接続仕様、PROFINET IO接続仕様など |
| I/Oケーブル長さ | 0（ケーブルなし）、2m（標準）、3m、5m |
| オプション | ファンユニット（RCON-PC用） |
