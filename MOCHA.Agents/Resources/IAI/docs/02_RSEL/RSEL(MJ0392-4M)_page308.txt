### テキスト
#### 5.10.3 エレシリンダーの制御方式
エレシリンダーは上位機器からのI/OまたはSELプログラムによる制御が可能です。どちらの制御方法を使うかはEC接続ユニットごとにパラメーターで設定します。

- 注1：エレシリンダーは1軸ずつしか変更できません。
- 3ポジション切替仕様（MF）の場合は別の取扱説明書（MJ3837）を参照。

##### 上位機器のI/O制御によるエレシリンダー動作
【動作イメージ】
1. PLCからRSEL（EC接続ユニット）に移動指令信号を入力（前進または後退）
2. RSELから各エレシリンダーに移動指令信号を入力
3. エレシリンダーが動作
4. エレシリンダーから位置検出信号が出力（前進端または後退端）
5. RSELから位置検出信号がPLCに出力される

エレシリンダーは移動指令信号（ST0/ST1）がONの間動作を継続し、動作完了後に位置検出信号（LS0/LS1）をONにします。動作完了前に移動指令信号をOFFにすると動作を中断し減速停止します。原点復帰動作も同様です。

```mermaid
    C -->|③ 動作| C
    C -->|④ 前進端検出 or 後退端検出| B
    B -->|⑤ 前進端検出 or 後退端検出| A

    subgraph ケーブル
        B ---|EC接続ユニット用ケーブル| C
    end
```

**図の説明:**
- 上位機器（PLCなど）からRSEL（EC接続ユニット）へ前進または後退の移動指令信号を送る。
- RSELからエレシリンダーへ同様の指令信号を送る。
- エレシリンダーが動作し、動作完了時に位置検出信号をRSELに返す。
- RSELは位置検出信号を上位機器に返す。
- EC接続ユニット用ケーブルでRSELとエレシリンダーが接続されている。
