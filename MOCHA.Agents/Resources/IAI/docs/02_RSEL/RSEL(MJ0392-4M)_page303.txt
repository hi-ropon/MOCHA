### テキスト
#### 注意事項
電源投入後、不特定のタイミングでロボットが動作すると危険なため、安全のために押しボタンスイッチなどによる起動確認信号が入力されてからプログラムを起動するなどのインターロックを施す必要があることが説明されています。

#### 動作例
1. パラメーターNo.33に「2」を設定する。入力ポート003をONにするとプログラムが起動し、OFFにすると停止する。  
2. RSELシステムの動作モード設定スイッチをAUTO側に設定する。  
3. 電源を投入する。  
4. RSELシステムが起動後（正常起動＝RDY信号ON後）、0.01秒以上経過を確認してから入力ポート003をONにする。  
5. 設定したプログラムが起動する。  
6. 入力ポート003をOFFにするとREADY出力がOFFし、プログラムが終了する。

### 図
```mermaid
flowchart LR
    A[電源投入] --> B[RSELシステム起動 (RDY信号ON)]
    B --> C[0.01秒以上経過確認]
    C --> D[入力ポート003 ON]
    D --> E[プログラム実行 (Min 0.01s)]
    E --> F[入力ポート003 OFF]
    F --> G[READY出力 OFF]
    G --> H[プログラム終了]
```

### タイミングチャート説明
- READY出力（RSELの出力）は電源投入後にONとなり、入力ポート003がONになるまで維持される。  
- 入力ポート003がONになるとプログラムが実行され、最低0.01秒間継続する。  
- 入力ポート003がOFFになるとREADY出力がOFFし、プログラムが終了する。  
- なお、I/OパラメーターNo.47が0以外の場合、READY出力は出力ポートNo.301から出力される。
