### テキスト
#### 5.9 運転方法
運転には、設定されたプログラムNo.の自動起動と外部からプログラムNo.を指定して起動する2つの方法があります。

#### 5.9.1 電源投入による自動起動
電源投入後、RSELシステムに登録したプログラムNo.を自動起動できます。

1. ティーチングツールを接続し、I/OパラメーターNo.33「入力機能選択003」を「1」に、その他パラメーターNo.1「オートスタートプログラムNo.」に起動するプログラムNo.を設定してください。

2. RSELシステムの動作モード設定スイッチをAUTO側に設定してください。

3. 電源を再投入してください。

4. RSELシステムが再起動後（正常起動＝RDY信号ON後）、設定したプログラムが起動します。

### 表
#### パラメーター設定一覧

| パラメーター種別、およびNo. | 設定値 | 内容 |
| :--------------------------- | :----- | :--- |
| I/OパラメーターNo.33 | 0 | 入力ポート003を汎用入力にします。 |
|  | 1（出荷時） | AUTOモード時、電源投入またはソフトウェアリセットを行うと、その他パラメーターNo.1に設定した番号のプログラムを実行します。入力ポート003は汎用入力として使用可能。 |
|  | 2 | 入力ポート003をONすると、その他パラメーターNo.1に設定した番号のプログラムを実行。OFFにすると実行中の全プログラムを停止。 |
| I/OパラメーターNo.286 | -1（出荷時） | I/OパラメーターNo.33「入力機能選択003」の機能割付けポートNo.を指定。 |
| その他のパラメーターNo.1 | 1～512 | 自動実行させるプログラム番号を設定。 |

### 図
#### 電源投入後のプログラム起動タイミング

```mermaid
flowchart LR
    電源ON --> READY出力(RSELの出力)
    READY出力 --> プログラム実行[プログラム実行]
    プログラム実行 -->|Min0.01s| プログラム
```

**図の説明:**
- 電源ON後、READY出力（RSELの出力）がONになる。
- READY信号ON後、設定されたプログラムが最短0.01秒で実行開始される。
- 注記：I/OパラメーターNo.47が0以外の場合は、出力ポートNo.301から出力される。
