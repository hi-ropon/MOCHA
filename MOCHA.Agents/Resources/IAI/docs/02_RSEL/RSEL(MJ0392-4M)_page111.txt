### テキスト
#### 外部出力仕様
SELユニット、ターミナルユニットの外部出力に関する仕様を示しています。

### 表
#### 外部出力仕様

| 項目           | 仕様               |
| :------------- | :----------------- |
| 出力点数       | 16点               |
| 定格負荷電圧   | DC24V±10%          |
| 最大電流       | 50mA/1回路         |
| 絶縁方式       | フォトカプラー     |

### 図
#### NPN仕様回路図

```mermaid
flowchart LR
    subgraph 内部回路
        A[フォトカプラー]
        B[トランジスタ]
    end
    A --> B
    B --> C[P24 出力端子]
    B --> D[15Ω 抵抗]
    D --> E[負荷]
    E --> F[外部電源 DC24V]
    F --> G[N端子]
    G --> B
```

#### PNP仕様回路図

```mermaid
flowchart LR
    subgraph 内部回路
        A[フォトカプラー]
        B[トランジスタ]
    end
    A --> B
    B --> C[P24 端子]
    C --> F[外部電源 DC24V]
    F --> E[負荷]
    E --> D[15Ω 抵抗]
    D --> G[N端子]
    G --> B
```

**図の説明:**
- NPN仕様は内部回路からトランジスタを介して出力端子P24に接続し、負荷を通じて外部電源DC24VのN端子に接続される構成。
- PNP仕様は内部回路からトランジスタを介して出力端子P24に接続し、外部電源DC24Vから負荷を経由してN端子に接続される構成。
</contents>
