### テキスト
#### 9.5.3 インターフェイス仕様
コントローラー部とビジョンシステムとの接続は、パラレル通信とシリアル通信を両方使用して接続します。

### 表
| 項目       | 用途                                                                 | 備考                                         |
|------------|----------------------------------------------------------------------|----------------------------------------------|
| パラレル   | PIO入力1点<br>ビジョンシステムの初期化完了ステータス入力（必要に応じて使用） | ビジョンシステムとのハンドシェイク用（PIOボード選択またはPIOユニット購入が必要） |
|            | PIO出力1点<br>ビジョンシステムへの撮像指令出力（必須）               |                                              |
| シリアル   | Ethernet（TCP）<br>撮像ワークデータの受信（必須）                    | Ethernetに標準対応                            |

### 図
コントローラー背面の接続部位を示す図。
