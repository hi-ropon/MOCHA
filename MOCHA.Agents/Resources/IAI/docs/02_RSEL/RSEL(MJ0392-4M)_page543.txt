### テキスト
#### 9.5.5 関連エラーコード
ビジョンシステムI/Fに関連するパラメーターの一覧を示す。

### 表
#### 動作解除レベルエラー一覧

| エラーNo. | エラー名称                         | 内容・原因                                                                                                   | 対処方法                                                                                                  |
| -------- | -------------------------------- | ---------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| 40D      | ビジョンシステムエラー           | 【内容・原因】<br>パラメーター設定ミス<br>①I/OパラメーターNo.351 “ビジョンシステムI/F1機能選択1”の設定が使用未許可設定("0")になっている | 【対処方法】<br>①I/OパラメーターNo.351 “ビジョンシステムI/F1機能選択1”の設定を使用許可設定("1")に設定する |
| 40E      | ビジョンシステムレスポンスタイムアウトエラー | 【内容・原因】<br>ビジョンシステムからの通信レスポンスを確認できない<br>①パラメーター設定ミス<br>②ビジョンシステム側の設定ミス | 【対処方法】<br>①-1 IOパラメーターNo.129 ビット4-7：TCP/IPメッセージ通信の値が1(使用)であることを確認する<br>①-2 IOパラメーターNo.160-163 ビジョンシステムI/F接続先IPアドレスの設定値が正しいことを確認する<br>①-3 IOパラメーターNo.164 ビジョンシステムI/F接続先ポート番号の設定値が正しいことを確認する<br>②ビジョンシステムが撮像指令に対してデータ送信しているか確認する |
| 40F      | 受信伝文エラー（ビジョンシステム） | 【内容・原因】<br>①ビジョンシステムから未サポートの識別コードを受信した<br>②ビジョンシステムから不正なデータを受信した | 【対処方法】<br>①、②ビジョンシステムの送信データが正しいことを確認する |
| 410      | 受信ワーク数エラー（トラッキング・ビジョンシステムI/Fデータ通信） | 【内容・原因】<br>①ビジョンシステムから受信したワーク数が、1回撮像当たりのワーク数上限を超えた | 【対処方法】<br>①コンベア上のワークの間隔を広げ、上限を超えないようにする |
| 411      | ビジョンシステムI/F内部処理異常 | 【内容・原因】<br>①何らかの要因でワーク状態がビジーになった<br>②何らかの要因でワーク情報ハンドリング処理がタイムアウトした | 【対処方法】<br>①、②電源再投入<br>それでも改善されない場合はSELユニット交換 |
| 412      | ビジョンシステム指定エラー       | 【内容・原因】<br>①使用しようとしているビジョンシステムI/Fが使用中のビジョンシステムI/Fと異なる<br>②指定したビジョンシステムI/Fは他タスクで使用中 | 【対処方法】<br>①使用中のビジョンシステムI/Fを開放してから使用するビジョンシステムI/Fを指定する（開放=SLVS命令操作）<br>②使用したいビジョンシステムの他タスクでの使用が完了してから使用する |
