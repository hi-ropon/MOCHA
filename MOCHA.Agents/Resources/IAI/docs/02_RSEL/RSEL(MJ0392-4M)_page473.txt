#### 7.2 直交座標系機能使用時の注意事項

● CP動作のポジション設定  
命令語によって指定可能な座標系（ポジションタイプ）が異なります。

・各軸座標系（Joint）が指定可能なCP動作命令  
MOVL MVLI PATH PSPL CIR CIR2 ARC2 ARCC ARCD ARCS CIRS PUSH ARCH

・直交座標系（Rect）が指定可能なCP動作命令  
MOVL MVLI TMLI PATH PSPL ARCS CIRS ARCH

命令語の詳細については、「SEL言語プログラミングマニュアル（MJ0224）」を参照してください。
