### テキスト
#### 1.3.1 基本仕様
RSELシステムの基本仕様について説明しています。

### 表
#### RSELシステム基本仕様一覧

| 項目                     | 仕様                                                                                          |
|--------------------------|-----------------------------------------------------------------------------------------------|
| 電源電圧                 | DC24V ±10% ※1<br>AC200-230V ±10%                                                           |
| 電源電流                 | システム構成により異なる<br>（詳細は[2.2 電源容量（2-12ページ）]参照）                         |
| 制御軸数                 | ドライバーユニット制御：0～8軸（ドライバーユニットは任意の組み合わせ可能）<br>EC接続ユニット制御含め最大16軸まで接続（※2） |
| 対応フィールドネットワーク | CC-Link、CC-Link IE Field、DeviceNet、EtherCAT、EtherNet/IP、PROFIBUS-DP、PROFINET IO（スレーブ局） |
| 構成ユニット             | RCON-Aドライバーユニット、RCON-Dドライバーユニット<br>RCON-Pドライバーユニット、RCON-P高能力ドライバーユニット<br>RCON-SC電源ユニット・ドライバーユニット<br>SCON拡張ユニット、PIO/SIO/SCON拡張ユニット、PIOユニット<br>EC接続ユニット、簡易アブソユニット、ターミナルユニット<br>（詳細は[2.1.2 構成ユニット]参照） |
| 非常停止／イネーブル動作 | SELユニットのSTOP信号入力でシステム一括対応、各ドライバーユニット、EC接続ユニットによって1軸ごと駆動源遮断できるコネクターを搭載 |
| 法令・規格               | CEマーキング、UL認証                                                                        |
| 外形寸法                 | システム構成により異なる                                                                     |
| 各ユニット間の接続       | ユニット連結方式<br>（詳細は[4.1.1 ユニット連結（4-2ページ）]参照）                          |
| 設置取付け方法           | DINレール（35mm）取付け                                                                      |

### 注釈
- ※1 電圧降下によるトラブル回避のため、配線インピーダンスに注意すること。電源側24V電圧とコントローラー側24V電圧の差は1V未満が目安。24V電源とコントローラー間のケーブルは適合線径で極力太いものを選定し、極力短くすること。
- ※2 ECのみ16軸の接続も可能。
