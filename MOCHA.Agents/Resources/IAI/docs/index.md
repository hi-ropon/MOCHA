# IAI（アイエイアイ）コントローラマニュアル インデックス

このファイルはIAI社製コントローラのマニュアルインデックスです。
IAI Document Readerとiai_agentが各マニュアルページを検索・読み込むために使用されます。

## マニュアル構成概要

IAIコントローラシリーズは同様の基本機能を持つ制御機器群です。各製品は共通のプログラミング概念と操作方法を共有しており、用途に応じて最適化された機能を提供します。

### 01_RCON（RCONコントローラ）
**フォルダ:** `01_RCON/`  
**ファイル数:** 886ページの詳細マニュアル  
**対象ページ:** 25-609  
**マニュアル型番:** MJ0384-8D

**主な内容:**
- **取扱い上の注意**: 使用条件・環境、ティーチングツール選択、データバックアップ、フィールドネットワーク初期設定
- **駆動源遮断システム**: 24V/200V系統別遮断方式、外部遮断インターフェイス（MPI/MPO）、内部半導体遮断回路、ドライバー停止回路（8ms応答）
- **ユニット構成**: ゲートウェイユニット、ドライバーユニット（PC/PCF/AC/DC/SC）、EC接続ユニット、TPアダプター
- **フィールドネットワーク対応**: 7種類のネットワーク、6種類の動作モード、制御シーケンス連携
- **入出力制御**: PLC信号制御、アドレス割付、制御信号（ST0/ST1/RES）、4軸同時制御対応
- **ティーチング・プログラミング**: パソコン専用ソフト、ティーチングボックス対応、外部メモリー保存
- **安全回路**: 非常停止回路、予期しない動作防止、制御シーケンス整合性確認
- **トラブルシューティングとメンテナンス**: 故障診断、アラーム解除、復旧手順

**検索キーワード:** RCON、フィールドネットワーク、ドライバーユニット、EC接続、TPアダプター、制御信号、アドレス割付、ティーチングツール、駆動源遮断、接続、プログラミング、設定、トラブル
