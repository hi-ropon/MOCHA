### テキスト
#### 6.1 パラメーター

【25】押付け速度（パラメーターNo.34）  
押付け動作中の速度を設定するパラメーターです。出荷時はアクチュエーター仕様に応じた設定がされています。詳細は「10.3 接続可能なアクチュエーターの仕様一覧」を参照してください。  
設定変更が必要な場合は、アクチュエーターの最高押付け速度以下で使用してください。速度を変更すると所定の押付け力が得られなくなる場合があります。遅い速度を設定する場合も5mm/sを限度としてください。

**注意**  
- ポジションテーブルの位置決め速度がこのパラメーター以下に設定されていると、押付け速度は位置決め速度と同一の速度となります。

【26】セーフティー速度（パラメーターNo.35）  
ティーチングツールで、セーフティー速度選択中の手動操作の最高速度を設定するパラメーターです。安全のため必要以上の設定をしないでください。

### 図
押付け速度の速度変化を示すグラフ。速度が位置決め速度から押付け速度に変化し、ワークに到達するまでの位置決め幅を示している。

```mermaid
flowchart LR
    A[速度上昇] --> B[位置決め速度維持]
    B --> C[押付け速度に減速]
    C --> D[ワーク到達]
    D --> E[位置決め幅]
```

**図の説明:**  
- 速度は最初に上昇し、一定の位置決め速度で維持される  
- その後押付け速度まで減速し、ワークに到達するまでの距離が位置決め幅として示されている
