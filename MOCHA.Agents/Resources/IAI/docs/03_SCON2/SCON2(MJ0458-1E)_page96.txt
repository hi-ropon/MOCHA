### テキスト
#### 3.1 ポジショナーモード（PIOモード）

2. PIOパターン1 …… 教示モード（教示タイプ）  
- DC24V（PNP/NPN仕様）供給  
- 指令ポジションNo.1、No.2、No.4、No.8、No.16、No.32の信号線があり、それぞれ色分けされた配線で接続されている  
- 教示モード、ジョグ/インチング切替え、ジョグ移動＋、ジョグ移動－、運転モード切替え、原点復帰、一時停止、スタート/現在位置書込み、リセット、サーボONの信号線がある  
- ＊は負論理信号を示し、入力信号がOFFの時に処理され、出力信号は電源が入っている状態で通常ON、信号出力時はOFFになる

3. PIOパターン2 …… 256点モード（位置決め点数256点タイプ）  
- パターン1に加え、指令ポジションNo.64、No.128の信号線が追加されている  
- ブレーキ強制解除、運転モード切替え、原点復帰、一時停止、スタート、リセット、サーボONの信号線もある  
- ＊は負論理信号の説明はパターン1と同様

### 信号説明（共通）
| 信号名 | 内容 |
| :--- | :--- |
| 指令ポジションNo.x | 指令位置信号（xは番号） |
| 教示モード | 教示動作モードの切替え信号 |
| ジョグ/インチング切替え | ジョグ操作の切替え信号 |
| ジョグ移動＋/－ | ジョグ操作の移動方向信号 |
| 運転モード切替え | 運転モードの切替え信号 |
| 原点復帰 | 原点復帰動作信号 |
| 一時停止 | 動作の一時停止信号 |
| スタート/現在位置書込み | 動作開始または現在位置の書込み信号 |
| リセット | システムリセット信号 |
| サーボON | サーボモータON信号 |
| 完了ポジションNo.x | 指令位置の完了信号 |
| 移動中 | 移動中状態信号 |
| 教示モードステータス | 教示モードの状態表示 |
| ポジションゾーン/ゾーン1 | ポジションゾーンの状態 |
| 運転モードステータス | 運転モードの状態表示 |
| 原点復帰完了 | 原点復帰完了信号 |
| 位置決め完了/書込み完了 | 位置決めまたは書込み完了信号 |
| サーボONステータス | サーボON状態表示 |
| 非常停止ステータス | 非常停止状態表示 |
| アラーム | アラーム信号 |
| 軽故障アラーム | 軽故障アラーム信号 |

**注記**  
＊は負論理信号を示し、入力信号がOFFの時に処理され、出力信号は電源が入っている状態で通常ON、信号出力時はOFFになる。
