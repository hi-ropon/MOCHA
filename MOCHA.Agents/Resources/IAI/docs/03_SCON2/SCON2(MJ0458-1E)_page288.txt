### テキスト
#### 5.3.2 反転動作
衝突検出時に反転動作ポジションNo.に任意のポジションNo.を指定することで、衝突検出後に反転動作を行います。  
反転動作に用いるポジションデータは「位置」「速度」「加速度」「減速度」「位置決め幅」のみで、設定した「位置」を基に現在位置から進行方向の逆方向に相対移動します。  
移動先がソフトウェアストロークリミット範囲外の場合は、相対移動距離を範囲内に制限します。  
指定ポジションデータに不足がある場合（速度・加速度・減速度・位置決め幅のいずれかが範囲外）はエラーNo.0A2「ポジションデータ異常」を出力します。  

反転動作中は衝突検出無効であり、別の移動指令は送れません。ジョグスイッチ機能も無効です。  
また、反転動作中は衝突からの退避を目的とするため一時停止指令も無効となります。  

反転動作完了後にパラメーターNo.225「衝突検出機能出力選択」の設定値により、エラーNo.0DF「衝突検出」（アラーム）またはエラーNo.049「衝突警告」（ワーニング）を出力します。  
反転動作中にサーボOFF指令・減速停止指令が行われた場合は反転動作を中止し、同様にエラーNo.0DFまたはNo.049を出力します。  

反転動作と出力選択の組み合わせによる衝突検出後の挙動は以下の通りです。

### 表
| 反転動作 | 出力選択 | 衝突検出後の挙動 |
| :--- | :--- | :--- |
| 有効 | アラーム | 反転動作完了後アラームを出力しサーボOFF停止 |
| 有効 | ワーニング | 反転動作完了後ワーニングを出力しサーボON状態を維持したまま停止 |
| 無効 | アラーム | 衝突検出後アラームを出力しサーボOFF停止 |
| 無効 | ワーニング | 衝突検出後ワーニングを出力する（停止しない） |

※反転動作有効時に反転動作用ポジションデータの「位置」に“0”を設定すると、衝突検出後反転動作を行わずに停止します。
