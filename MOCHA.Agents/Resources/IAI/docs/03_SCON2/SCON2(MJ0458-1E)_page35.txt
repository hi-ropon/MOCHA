### テキスト
#### 各部の名称と機能

9. 状態表示LED（PWR、SV、ALM、WRG、STOP、SAFE）  
コントローラーの運転状態を表示します。

10. PIOコネクター（PIO）  
制御用の入出力信号接続用のコネクターです。フィールドネットワーク仕様の場合、各ネットワークのインターフェイスになります。

11. USBコネクター  
パソコン専用ティーチングソフトを接続するコネクターです。

12. 安全機能用IOコネクター（SAFE）  
安全機能を使用する場合、安全回路を接続するコネクターです。外部安全要求信号が入力されるとSS1-t動作がハードウェア的に実行され、安全状態に移行します。

13. マルチファンクションコネクター（MF）  
パルス列入力、フィードバックパルス出力、ロードセル荷重データのアナログ出力を使用するためのコネクターです。

14. 機能安全ユニット接続コネクター  
機能安全ユニットを取付けるコネクターです。

### 表
#### 状態表示LEDの状態

| LED       | 状態                         |
| :-------- | :--------------------------- |
| PWR（緑） | コントローラー正常起動       |
| SV（緑）  | サーボON（自動サーボOFF中点滅） |
| ALM（橙） | アラーム発生中               |
| WRG（橙） | ワーニング発生               |
| STOP（赤）| 非常停止中                   |
| SAFE（赤）| 安全停止中                   |
