### テキスト
#### 8.4.3 パラメーター
ドライブレコーダー機能に関連するパラメーターの説明です。

【1】ドライブレコーダーモード選択（パラメーターNo.210）  
ドライブレコーダー機能で取得するデータ数を選択します。取得するデータ種別は変更不可です。

【2】ドライブレコーダー保存履歴数選択（パラメーターNo.211）  
ドライブレコーダー機能で保存可能なアラーム履歴数を設定します。

---

### 表

#### パラメーターNo.210 ドライブレコーダーモード選択

| No. | 名称                     | 単位 | 入力範囲                      | 工場出荷時の初期値 |
|:----|:-------------------------|:-----|:-----------------------------|:-------------------|
| 210 | ドライブレコーダーモード選択 | —    | 0：無効、1：4CH、2：8CH、3：2CH | 2                  |

#### パラメーターNo.210 取得データ種別一覧

| CH数 | データ種別             | 単位           |
|:-----|:-----------------------|:---------------|
| 0    | ドライブレコーダー機能無効 |                |
| 1    | 1 現在位置             | mm or pulse    |
|      | 2 フィードバック電流値   | % or mA        |
|      | 3 現在指令位置         | mm or pulse    |
|      | 4 現在速度             | mm/s or pps    |
| 2    | 5 運転計画速度         | mm/s or pps    |
|      | 6 現在荷重             | N              |
|      | 7 DCバス電圧値         | V              |
|      | 8 推定回生放電電力量   | W              |

#### チャンネル数ごとのデータ取得可能時間

| CH数 | 取得可能時間 |
|:-----|:-------------|
| 2    | 600秒        |
| 4    | 300秒        |
| 8    | 150秒 ←初期値 |

※データ取得周期を10msとした場合。
