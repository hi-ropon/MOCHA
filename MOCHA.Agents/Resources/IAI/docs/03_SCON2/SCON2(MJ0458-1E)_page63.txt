### テキスト
#### 2.5.1 PIO入出力インターフェイス
PIO入出力インターフェイスの仕様について説明しています。

### 表
|  | 入力部 | 出力部 |
| :--- | :--- | :--- |
| 仕様 | 入力電圧 | DC24V±10% | 負荷電圧 | DC24V |
|  | 入力電流 | 4mA 1回路 | 最大負荷電流 | 50mA 1回路 |
|  | ON/OFF電圧 | ON電圧 Min. DC18V<br>OFF電圧 Max. DC6V | 漏れ電流 | Max. 0.1mA/1点 |
