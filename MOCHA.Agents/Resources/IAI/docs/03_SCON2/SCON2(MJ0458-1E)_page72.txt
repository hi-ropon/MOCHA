### テキスト
#### 2.6.6 アブソリュートバッテリー
アブソリュート仕様のコントローラーにはアブソリュートバッテリーが付属しています。  
このバッテリーはアブソリュートデータのバックアップ用電源として機能します。  
コントローラー前面パネルのアブソバッテリー接続コネクターに接続してください。


### 表
#### アブソエンコーダーバックアップ仕様

| 項目                         | 仕様                                                                                   |
|------------------------------|----------------------------------------------------------------------------------------|
| バッテリー分類               | 塩化チオニルリチウム電池                                                               |
| バッテリーメーカー           | 東芝ライフスタイル株式会社 または、日立マクセル株式会社                                 |
| バッテリー型式（IAI型式）    | AB-5                                                                                   |
| バッテリー公称電圧           | 3.6V                                                                                   |
| バッテリー標準容量           | 2000mAh                                                                               |
| バッテリー交換時期の目安 (周囲温度40℃) | 使用後2年（コントローラーに通電せず放置した場合） 使用後4年（コントローラーへの通電時間50%の場合） |
| 異常検出                     | 電圧低下警告信号＊ALMLの出力：3.1V（目安値） アラーム出力＊ALMの出力：2.5V（目安値）       |
| 警告→アラームの猶予時間の目安 | 周囲温度20℃の時、7日間 周囲温度40℃の時、2.5日間                                      |
| バッテリー交換時のアブソリュートデータ保持時間 | 15分（この時間以内で交換してください。）                                               |

#### 注釈
- バッテリーは定期的に交換してください。  
- ※1 異常検出：アブソリュートバッテリーの電圧が低下してくると、電圧に応じて異常検出が行われます。
