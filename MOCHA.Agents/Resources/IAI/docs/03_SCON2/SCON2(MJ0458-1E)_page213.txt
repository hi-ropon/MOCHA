### テキスト
#### 押付け動作中のトルクレベル検出（PIOパターン7で有効）

■制御方法  
ポジションデータの“しきい”にパーセント[%]で設定した押付け力を監視し、押付け力が次の条件となったとき、トルクレベルステータス信号TRQSをONにします。  
その際ポジションゾーン内であれば負荷出力判定信号LOADもONします。  
別のポジションへの移動指令またはサーボOFFにより、この信号はOFFします。  
本信号は一度ONすると、次の移動指令までONを維持します。

●条件  
（押付け力が力判定マージン範囲内の累積時間）  
−（押付け力が力判定マージン範囲外の累積時間）255ms（パラメーターNo.50）

### 図（押付け力と信号のタイミングチャート）

```mermaid
flowchart LR
    A[ワークに接触] --> B[押付け力上昇]
    B --> C{力判定マージンを超える押付け力発生}
    C --> D[TRQS ON]
    D --> E[負荷出力判定(TRQS) ON]
    E --> F[押付け完了信号(PEND) ON]

    subgraph 押付け力の時間変化 [押付け力]
        direction LR
        A -->|100ms| B -->|20ms| C -->|180ms| F
    end

    subgraph 信号状態
        TRQS[TRQS信号]
        PEND[PEND信号]
    end

    B --> TRQS
    C --> PEND
```

**図の説明:**  
- 横軸は時間[ms]、縦軸は押付け力の大きさを示す。  
- 押付け力はワークに接触後、100msで力判定マージンの下限を超え、20msで力判定マージンの上限を超える押付け力が発生。  
- その後180ms間、押付け力が判定マージン範囲内に維持される。  
- この合計255ms以上の条件で、負荷出力判定信号（TRQS）と押付け完了信号（PEND）がONになる。  
- TRQS信号はポジションゾーン内でONし、移動指令またはサーボOFFでOFFされる。  
- PEND信号は押付け完了を示す。
