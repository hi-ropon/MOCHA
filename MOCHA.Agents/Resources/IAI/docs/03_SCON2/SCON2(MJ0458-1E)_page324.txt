### テキスト
#### 6.1 パラメーター
S字は、加速時間を1周期とする正弦波形となります。  
本パラメーターで振幅の度合を指定します。

### 表
| 設定[%] | 振幅の度合 |
| :---: | :--- |
| 0 | S字モーションなし（イメージ図の点線） |
| 10 | 正弦波の振幅×0.1（イメージ図の実線） |
| 50 | 正弦波の振幅×0.5（イメージ図の1点鎖線） |
| 100 | 正弦波の振幅×1（イメージ図の2点鎖線） |

### グラフ
速度を縦軸、時間を横軸としたグラフで、0秒から2秒以内の加速時間における速度変化を示しています。  
設定値10、50、100に応じて速度の上昇曲線が異なり、設定値が大きいほど速度の変化が大きくなっています。

### 注意
- 必ずアクチュエーターが停止した状態で指令してください。  
- 移動中の速度変更などを行ったため、アクチュエーター動作中にS字モーションを設定したポジション指令、または直接数値指令を行っても、S字モーション制御ではなく台形制御になります。  
- ロータリーアクチュエーターのインデックスモードでは、S字モーション制御は無効です。指定しても台形制御となります。  
- 加速時間、または減速時間が2秒を超える設定の場合、S字モーション制御を指令しないでください。台形制御になります。  
- 加速中、または減速中に一時停止を行わないでください。速度変化（加速）を起こし、危険を伴う場合があります。
