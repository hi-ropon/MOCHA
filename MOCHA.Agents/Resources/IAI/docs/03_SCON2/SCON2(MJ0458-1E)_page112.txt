### テキスト
#### 3.3 配線方法
##### 3.3.1 電源回路の配線
- 主電源回路（電源コネクター）について、コントローラーの仕様に合わせて電源を供給する必要がある。
- 負荷電流は接続するアクチュエーターなどにより異なるため、仕様に適合したサーキットブレーカーおよび漏電ブレーカーを選定すること。
- 電圧は購入後に変更できない（AC100～115VまたはAC200～230V）。

### 表1：電源種別と仕様
| 電源種別   | 仕様                                  |
| :--------- | :----------------------------------- |
| モーター電源 | 100V仕様：AC100～115V±10% 50/60Hz    |
| 制御電源   | 200V仕様：AC200～230V±10% 50/60Hz    |

### 図：配線イメージ
- 漏電ブレーカー（お客様準備）
- サーキットブレーカー（お客様準備）
- ノイズフィルター（お客様準備）
- サージプロテクター（お客様準備）
- 電源コネクターに接続される配線のイメージ図
