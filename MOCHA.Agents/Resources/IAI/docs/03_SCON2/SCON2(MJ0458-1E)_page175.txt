### テキスト
#### 多回転仕様のロータリーアクチュエーターの近回り制御
- 近回り制御の設定はインデックスモード設定時に可能（パラメーターNo.79「回転軸モード選択」の「1：インデックスモード」設定時）。
- 詳細は「4.2【3】多回転仕様のロータリーアクチュエーターの運転モードと指令の制限」を参照。
- 近回り選択はパラメーターNo.80「回転軸近回り選択」で有効/無効を設定可能。
- 有効にすると同一方向のみの運転も可能になる。

#### 運転例
- ポジション1→2→3→4→1の順に運転した場合の動作比較（近回り選択無効時と有効時）。

### 図
#### ポジション配置図
```mermaid
flowchart LR
    A[ポジション No.1] --- B[ポジション No.2]
    B --- C[ポジション No.3]
    C --- D[ポジション No.4]
    D --- A
```
- 円形に4つのポジションが均等に配置されている。

#### 近回り選択の動作フロー
```mermaid
flowchart LR
    subgraph 無効の場合
        A1((1)) -->|時計回り| B1((2))
        B1 -->|反時計回り| C1((3))
        C1 -->|時計回り| D1((4))
        D1 -->|時計回り| E1((1))
    end

    subgraph 有効の場合
        A2((1)) -->|時計回り| B2((2))
        B2 -->|反時計回り| C2((3))
        C2 -->|反時計回り| D2((4))
        D2 -->|時計回り| E2((1))
    end
```
- 無効時はポジション3→4間が時計回り、4→1間も時計回りに動作。
- 有効時はポジション3→4間が反時計回りに動作し、近回り運転となる。

### 表
#### ポジション番号と位置データ
| ポジション No. | 位置（度） |
| :------------- | :--------- |
| 1              | 0          |
| 2              | 90         |
| 3              | 180        |
| 4              | 270        |

- 位置データは1度=1mmとして入力。例：1.2は1.2度として扱う。
