### テキスト
#### アブソバッテリー仕様
- ホルダーなし型式：AB-5
- ホルダー付き型式：AB-5-CS3

コントローラーの電源投入後は正常時ON、異常検出時にOFFとなる。
PLCなどで＊ALML信号によるランプ表示を行い、アラーム発生前にバッテリー交換を推奨。
アラーム発生時はバッテリー交換後にアブソリュートリセットが必要。

### 表
| 電圧           | PIO信号                         | アラーム                         |
| -------------- | ------------------------------- | ------------------------------- |
| 3.1V（目安値） | 電圧低下警告信号＊ALML（注2）OFF | ―                               |
| 2.5V（目安値） | アラーム出力＊ALM（注2）OFF      | OEE “アブソリュートエンコーダー異常検出2” |

注2：＊ALMLおよび＊ALMは負論理の信号。

### グラフ

```mermaid
    ALML_ON[＊ALML信号 ON] -.-> B
    ALML_OFF[＊ALML信号 OFF] -.-> C
    ALM_ON[＊ALM信号 ON] -.-> C
    ALM_OFF[＊ALM信号 OFF] -.-> D

    note1[アブソリュートリセット不要] --> C
    note2[＊ALM OFF アブソリュートリセット必要] --> E
```

**図の説明:**
- バッテリー電圧が3.1Vを下回ると電圧低下警告信号（＊ALML）がOFFになる。
- 2.5Vを下回るとアラーム信号（＊ALM）がOFFとなり、アラームが発生。
- アラーム発生後はバッテリー交換とアブソリュートリセットが必要。
- 警告信号出力時はアブソリュートリセット不要。
