### テキスト
#### ポジショナーモードの信号制御手順
- 減速度、位置決め幅、加減速モード、停止モードの各パラメーター番号と初期値が示されています。
- 書込み完了後、コントローラーから書込み完了信号WENDが出力されるため、PWRT信号をOFFにする必要があります。
- PWRT信号OFFによりWEND信号もOFFになります。
- WENDがONを確認してからPWRTをOFFにしないと正しいデータ書込みが行われません。

#### 注意事項
- ポジションNo.の入力からPWRT信号ONまで6ms以上の時間を設けること。PLCのスキャンタイムの2～4倍の設定推奨。
- 原点復帰が完了していない状態でPWRT信号をONにするとアラーム093が発生。
- WEND信号がONする前にPWRT信号がOFFされた場合、正しいデータ書込みが行われない。
- IA-OSなどのティーチングツールでポジションテーブル画面を開いたまま書込み処理を行うと画面上のデータは更新されない。更新・確認には以下の処理が必要：
  1. IA-OSのポジションデータ再読出しボタンをクリック。
  2. ティーチングボックスでユーザー調整画面に切替え、調整No.に「4」を入力し、ソフトウェアリセット後ポジションテーブル画面に戻る。

### 図
```mermaid
flowchart LR
    A[指令ポジションNo. PC1～PC** (PLC→コントローラー)] -->|信号入力| B[現在値書込み信号 PWRT (PLC→コントローラー)]
    B --> C[現在位置座標書込み処理 (コントローラー)]
    C --> D[書込み完了信号 WEND (コントローラー→PLC)]

    B -.->|T1≥6ms| C
    D -->|WEND ONによりOFFしてください| B
    D -->|PWRTのOFFによりOFFします| B
```

**図の説明:**  
- PLCからコントローラーへ指令ポジションNo.が入力される。  
- 続いてPWRT信号がONされ、コントローラーは現在位置座標の書込み処理を行う。  
- 書込み完了後、コントローラーからPLCへWEND信号がONされる。  
- WEND信号ONを確認後、PWRT信号をOFFにする必要がある。  
- PWRT信号OFFによりWEND信号もOFFになる。  
- 指令ポジションNo.入力からPWRT信号ONまで6ms以上の間隔を設けることが重要。
