### テキスト
#### ポジション3運転回路
ポジションNo.3へ移動するための「起動」→「移動中」→「位置決め完了」の信号処理と管理を行うメイン回路です。ポジションNo.1と同様のシーケンス回路となっています。

### 図
```mermaid
flowchart TD
    subgraph 起動
        A1[AUX14: ポジション2位置決め完了] --> A2[AUX5: 完了ポジションNo.2]
        A2 --> A3[AUX16: ポジション3位置決めスタート確認]
        A3 --> A4[OUT1: STARTL RC運転表示]
        A4 --> A5[AUX15: ポジション3位置決めスタート補助]
    end

    subgraph 移動中
        B1[AUX15: ポジション3位置決めスタート補助] --> B2[IN13: PEND位置決め完了]
        B2 --> B3[AUX17: ポジション3位置決め完了]
        B3 --> B4[OUT1: STARTL RC運転表示]
        B4 --> B5[AUX16: ポジション3位置決めスタート確認]
    end

    subgraph 完了
        C1[AUX16: ポジション3位置決めスタート確認] --> C2[AUX6: 完了ポジションNo.3]
        C2 --> C3[AUX9: ポジション1位置決めスタート補助]
        C3 --> C4[OUT1: STARTL RC運転表示]
        C4 --> C5[AUX17: ポジション3位置決め完了]
    end

    A5 --> B1
    B5 --> C1
```

### 図の説明
- **起動**: ポジション2の位置決め完了をトリガーにポジション3の位置決めスタートを補助し、RC運転表示を開始
- **移動中**: ポジション3の位置決め完了を検出しつつ、スタート補助信号を維持
- **完了**: ポジション3の位置決め完了を確定し、ポジション1の再起動信号を送出
