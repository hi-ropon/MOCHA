### テキスト
#### 5.2 節電機能（自動サーボOFF機能）
この設定はパラメーターNo.39によって行います。  
（注）自動サーボOFF中は、前面パネルのコントローラー状態表示LEDのSVが緑点滅します。

### 表1
#### パラメーターNo.39設定値とPEND信号の内容、自動サーボOFF中の信号出力状態

| パラメーターNo.39設定値 | PEND信号の内容       | 自動サーボOFF中の信号出力状態 |       |       |
|-------------------------|----------------------|------------------------------|-------|-------|
|                         |                      | PEND                         | PM1～PM** | PE**  |
| 0                       | 位置決め完了信号     | OFF                          | OFF   | OFF   |
| 1                       | インポジション信号   | ON                           | ON    | ON    |

---

### 表2
#### (1) パラメーターNo.39=0のときの動作

| アクチュエーターの動作       | 位置決め動作 | 自動サーボOFF待機 | サーボOFF | 位置決め動作 |
|-----------------------------|--------------|------------------|----------|--------------|
| サーボの状態                 | ON           | ON               | OFF      | ON           |
| 完了ポジションNo.出力       | PM1～**=0   | PM1～**=出力     | PM1～**=0 | PM1～**=0   |
| （現在位置番号出力）         |              |                  |          |              |
| 位置決め完了信号PEND         | OFF          | ON               | OFF      | OFF          |

- サーボOFF遅延時間はパラメーターNo.36～38で設定

---

### 表3
#### (2) パラメーターNo.39=1のときの動作

| アクチュエーターの動作       | 位置決め動作 | 自動サーボOFF待機 | サーボOFF | 位置決め動作 |
|-----------------------------|--------------|------------------|----------|--------------|
| サーボの状態                 | ON           | ON               | OFF      | ON           |
| 完了ポジションNo.出力       | PM1～**=0   | PM1～**=出力     | PM1～**=出力 | PM1～**=0   |
| （現在位置番号出力）         |              |                  |          |              |
| 位置決め完了信号PEND         | OFF          | ON               | ON       | OFF          |

- サーボOFF遅延時間はパラメーターNo.36～38で設定
