#### アラーム一覧（アラームコード、名称、原因・対策）

|:--------------|:------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 08F           | 安全ユニット マイコンB異常       | 原因: 安全ユニットのマイコンBが異常を検出した。<br>対策: <br>1. 二重化の安全入力信号が不一致（片方ON、片方OFF）、配線および信号を確認。<br>2. 安全入出力コネクターの電源を確認。<br>3. 原点復帰完了後にSLS、SLP、SDI+、SDI-、SS1-t、SS2-tを実行。<br>4. 安全パラメーターの確認または再書込み。<br>5. アクチュエーターが指令通りに動作しない場合は周辺物の干渉やブレーキ解除状況を確認。<br>6. 電源再投入でも復旧しない場合はコントローラー、安全ユニット、モーターユニットの交換が必要。 |
| 09B           | 内部通信エラー                 | 原因: R-Unitのゲートウェイユニットとドライバーユニット間で内部通信異常が発生。<br>1. コネクターの取付不良、接触不良。<br>2. 周辺機器からのノイズ影響。<br>3. ゲートウェイユニットまたはドライバーユニットの故障。<br>対策: <br>1. 脱着してアラーム再発を確認。<br>2. 周辺機器の電源遮断し動作確認。ノイズ対策や見直しを実施。<br>3. ユニット交換。                                                                                                                                                                                                                     |
| 09C           | エンコーダーFRAM 読書き後サーボON指令 | 原因: ユーザーメモ編集後、再起動または電源再投入せずにサーボONを行った。<br>対策: 再起動または電源再投入を行う。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| 09E           | アクチュエーター誤接続検出         | 原因: コントローラーのパラメーターと接続中のアクチュエーターが不一致。<br>対策: 正しい組合せのアクチュエーターにつなぎ直す。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 09F           | 周期同期エラー                 | 原因: R-Unitとの通信異常。<br>1. 連結コネクターまたはSCON接続コネクターが正しく接続されていない。<br>2. 信号線またはSCON接続ケーブル断線。<br>3. ターミナルユニットまたはSCON用コネクター未取付。<br>4. ノイズの影響。<br>対策: <br>1. ユニット同士の連結確認、再連結。<br>2. 電源再投入後も再発する場合は連絡。<br>3. ターミナルユニットやコネクター取付。<br>4. ケーブル引回し変更やノイズ対策。                                                                                                                                            |
| 0A1           | パラメーターデータ異常            | 原因: パラメーター領域のデータ入力範囲が不適切。<br>例: ソフトリミット+側200.3mmに対し、-側300mmと誤入力し大小関係が不適切。<br>対策: 適切な値に変更する。                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
