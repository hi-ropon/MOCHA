### テキスト
#### 【12】指令ポジションNo.出力回路
準備回路の結果により、PLCからSCON2へポジションNo.をバイナリーコードに変換して出力します。

#### 【13】スタート信号出力回路
ポジションNo.を出力後、20ms遅延してPLCからSCON2へスタート信号を出力します。

### 図
#### 【12】指令ポジションNo.出力回路のラダー図
```mermaid
flowchart TD
    A[ポジションNo.1セット (AUX18)] --> OUT8[OUT8 PC1 指令ポジション番号1]
    B[ポジションNo.2セット (AUX19)] --> OUT9[OUT9 PC2 指令ポジション番号2]
    C[ポジションNo.3セット (AUX20)] -.-> B
    C -.-> A
```
**図の説明:**
- AUX18, AUX19, AUX20の信号でポジション1～3をセットし、対応するOUT8, OUT9へバイナリーコードで出力している。

#### 【13】スタート信号出力回路のラダー図（タイミング説明付き）
```mermaid
flowchart LR
    subgraph ポジション位置決めスタート補助
        AUX9[ポジション1]
        AUX12[ポジション2]
        AUX15[ポジション3]
    end
    AUX9 -->|信号2| HANDSHAKE[ハンドシェイク]
    AUX12 -->|信号5| HANDSHAKE
    AUX15 -->|信号8| HANDSHAKE
    HANDSHAKE -->|20ms遅延| TIMER[TIM5 200ms設定]
    TIMER --> OUT13[スタート信号出力 (OUT13 CSTR)]
```
**図の説明:**
- ポジション1,2,3のスタート補助信号（2,5,8）がハンドシェイクにより制御される。
- 20msの遅延後、タイマーTIM5が動作し、スタート信号を出力する。
- PLCのスキャンタイムの2～4倍の設定で、SCONの入力条件を満たすようにしている。
- ハンドシェイクにより、メイン回路の移動中回路がONすると補助信号がOFFになる制御を行っている。
