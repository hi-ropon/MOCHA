### テキスト
#### 2.3 基本仕様
##### 2.3.1 基本仕様一覧
本節では、SCON2-CGの基本仕様を一覧表形式で示しています。

### 表
#### 基本仕様一覧（SCON2-CG）

| 項目 | SCON2-CG |
| :--- | :--- |
| 対応モーター容量 | 電源電圧 AC100V: 60～200W<br>電源電圧 AC200V: 60～750W |
| 電源電圧 | 単相 AC100～115V<br>単相 AC200～230V<br>（電源変動±10％以内） |
| 突入電流 | 電源電圧 AC100V:<br>制御側：30A（～25℃）、70A（～60℃）(注1)<br>駆動側：30A（～25℃）、80A（～60℃）(注1)<br>電源電圧 AC200V:<br>制御側：30A（～25℃）、60A（～60℃）(注1)<br>駆動側：30A（～25℃）、70A（～60℃）(注1) |
| 漏れ電流（注2）<br>（電源ラインにノイズフィルターを接続した場合の1次側） | 3.5mA |
| 負荷容量、発熱量 | [2.3.2 電源容量と発熱量]参照 |
| 電源周波数 | 50 / 60Hz |
| PIO電源（注3） | DC24V ±10% |
| 電磁ブレーキ用電源（ブレーキ付きアクチュエーターの場合） | DC24V ±10% 1A（最大）<br>（外部から供給） |
| 瞬時停電耐性 | 20ms（50Hz）、16ms（60Hz） |
| モーター制御方式 | 正弦波 PWM ベクトル電流制御 |
| 対応エンコーダー | インクリメンタルシリアルエンコーダー<br>アブソリュートシリアルエンコーダー<br>バッテリーレスアブソエンコーダー<br>ABZ（UVW）パラレルエンコーダー |
| アクチュエーターケーブル長 | 最大 20m |
| シリアル通信インターフェイス | SIOコネクター:<br>RS-485：1CH・Modbusプロトコル RTU/ASCII準拠<br>速度：9.6～230.4Kbps<br>パルス列以外のモードでシリアル通信による制御可能<br>USBコネクター:<br>通信規格：USB2.0、速度12MHz、コネクター：miniB |
| 外部インターフェイス | PIO仕様:<br>DC24V専用信号入出力（NPN/PNP選択）…入力最大16点、出力最大16点<br>フィールドネットワーク仕様:<br>CC-Link、CC-Link IE Field、DeviceNet、EtherCAT、EtherNet/IP、PROFINET IO、MECHATROLINK-III<br>その他:<br>R-Unit接続仕様 |
| マルチファンクションコネクター | 入力パルス周波数:<br>差動方式（ラインドライバー方式）：Max. 2.5Mpps<br>オープンコレクター方式：Max. 200Kpps（AK-04オプション使用時）<br>指令パルス倍率（電子ギア：A/B）<br>A、Bの設定範囲（パラメーターに設定）：1～99,999,999<br>フィードバックパルス:<br>差動方式（ラインドライバー方式）：Max. 2.5Mpps<br>オープンコレクター方式：Max. 500Kpps（JM-08オプション使用時）<br>アナログ出力:<br>1系統（荷重データ）<br>4～20mA電流出力（±1%）<br>負荷抵抗10～600Ω |
| データ設定、入力方法 | パソコン対応ソフト、タッチパネルティーチング、ティーチングボックス |
| データ保持メモリー | ポジションデータ、パラメーターを不揮発性メモリーへ保存（書込み回数制限なし） |

**注釈**
- (注1) 突入電流の温度条件についての注記
- (注2) 漏れ電流は電源ラインにノイズフィルターを接続した場合の1次側での値
- (注3) PIO電源の仕様に関する注記
