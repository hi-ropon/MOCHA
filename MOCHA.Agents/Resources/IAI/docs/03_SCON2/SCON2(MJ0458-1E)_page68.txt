#### 2.6.4 ブレーキボックス
- 型番：RCB-110-RA13-0
- ブレーキボックス1個で2軸分のブレーキを制御可能。
- ブレーキボックス接続指示のあるアクチュエーターを接続する場合に必要。

#### 仕様
| 項目           | 仕様                                      |
| -------------- | ----------------------------------------- |
| 本体寸法       | 162×94×65.5mm                            |
| 電源電圧、電流 | DC24V±10%　1A                            |
| 接続ケーブル   | エンコーダーケーブル（型番 CB-RCS2-PLA010）1m |
| 制御軸数       | 2                                        |
