### テキスト
#### ポジショナーモードの運転（制振No.）
アクチュエーターに取り付けられた負荷の振動（共振）を抑制します。3種類の振動に対応可能で、1つの振動に対して4個のパラメーターが設定されており、これを1セットとしています。ポジションテーブルには振動抑制に必要なポジションNo.に該当するパラメーターセットを設定します。

#### 注意事項
- 抑制できる振動周波数（対象の固有振動数）は0.5Hz～30Hzです。
- 本コントローラーに接続されたアクチュエーターによる負荷の振動が対象で、それ以外の振動は抑制できません。
- アクチュエーターの動作方向と同一方向の振動が対象で、それ以外の方向の振動は抑制できません。
- 原点復帰および押付け動作は対象外です。
- パルス列制御モードには対応していません。
- 振動周波数の設定が低い場合、タクトタイムが長くなることがあります。約6Hz以下では位置決め収束時間が150ms以上になります。

### 表
#### 制振No.とパラメーター設定

| 設定 | 制振周波数（固有振動数）           | パラメーターNo.  |
| :--- | :------------------------------- | :-------------- |
| 0    | 通常位置制御（制振なし）           | ―               |
| 1    | 制振制御パラメーターセット1        | 97～100         |
| 2    | 制振制御パラメーターセット2        | 101～104        |
| 3    | 制振制御パラメーターセット3        | 105～108        |
