### テキスト
#### 3.3.5 回生抵抗ユニットの接続
回生抵抗ユニット付属のケーブルで、以下のように回生抵抗ユニットを接続してください。
1. 1台接続：付属ケーブル（CB-SC-REU）で接続
2. 2台目以降の接続：付属ケーブル（CB-ST-REU）で接続

●配線イメージ  
回生抵抗ユニット接続コネクターにケーブルを接続し、1台目はCB-SC-REUケーブルで接続、2台目以降はCB-ST-REUケーブルで接続します。終端抵抗は不要です。

### 図
```mermaid
flowchart LR
    A[回生抵抗ユニット接続コネクター] -->|1. CB-SC-REUケーブル| B[RESU-2 / RESUD-2]
    B -->|2. CB-ST-REUケーブル| C[RESU-1 / RESUD-1]
    B -- RB IN --> C
    C -- RB OUT --> B
```
**図の説明:**  
- 1台目はCB-SC-REUケーブルで接続  
- 2台目以降はCB-ST-REUケーブルで接続  
- RB IN/OUT端子で回生抵抗ユニット同士を接続  
- 終端抵抗は不要
