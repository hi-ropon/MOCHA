### テキスト
#### 4.5 アブソリュートリセット
アブソリュートリセット処理の動作を示すタイミングチャートが掲載されています。

**注釈**
- 注1：PIO用24V電源（ブレーキ付アクチュエーター仕様の場合は、ブレーキ用24V電源も対象）は、制御電源・モーター電源より先に投入してください。
- 注2：制御電源とモーター電源は同一電源とし、同時に投入してください。

### 図（タイミングチャート）
```mermaid
flowchart TD
    A[安全回路の状態] --> B[非常停止解除状態（モーター駆動電源通電状態）]
    C[PIO用24V電源投入] --> B
    D[ブレーキ電源投入] --> B
    E[制御電源投入] --> B
    F[モーター電源投入] --> B
    G[アラームリセット（手順④）] --> H[アラーム信号(*ALM) / アラームコード出力(PM8～PM1)]
    H --> I[ALM LED]
    J[*一時停止信号(STP)] --> K[一時停止が解除されている]
    L[サーボON信号(SON)] --> M[サーボONステータス(SV)]
    N[位置決め完了信号(PEND)] --> O[原点復帰（手順⑤）]
    O --> P[原点復帰完了信号(HEND)]
    P --> Q[原点復帰動作]
    Q --> R[移動開始]
    R --> S[メカエンド]
    S --> T[原点位置]
    T --> U[ティーチングボックス、パソコン操作にてポジションテーブル作成]
```

**図の説明**
- 各信号の投入・解除タイミングを波形で示し、アブソリュートリセットの動作手順を視覚的に理解できる。
- 安全回路の状態から非常停止解除状態へ移行し、各電源投入の順序が示されている。
- アラームリセット後の信号変化、サーボON信号の動作、位置決め完了信号、原点復帰動作の流れが示されている。
- 最終的にティーチングボックスやパソコン操作でポジションテーブルを作成する工程に至る。
