### テキスト
#### 自動サーボOFF遅延時間1、2、3（パラメーターNo.36、No.37、No.38）
節電機能使用時に、位置決め完了後に自動サーボOFFするまでの時間を設定します。詳細は「5.2 節電機能（自動サーボOFF機能）」を参照してください。

#### 位置決め完了信号出力方式（パラメーターNo.39）
使用する位置決め完了信号の種類を選択するパラメーターです。PIOパターン=5（電磁弁タイプ2（3点タイプ））以外を選択しているときに有効です。  
位置決め完了信号は2種類あり、位置決め完了後のサーボON中またはサーボOFFにより出力状態が異なります。

#### 位置決め完了信号の設定値と動作

| 設定値 | 信号識別 | サーボON時（位置決め完了時）                                                                 | サーボOFF時       |
| :----- | :------- | :------------------------------------------------------------------------------------------ | :---------------- |
| 0      | PEND     | 現在位置が位置決め幅の範囲外となってもOFFしません。                                         | 無条件にOFF       |
| 1      | INP      | 現在位置が位置決め幅の範囲内でON、範囲外でOFFします。                                       |                  |
| パルス列制御モード | INP AUTO | 現在位置が位置決め幅（パラメーターNo.10）の範囲内でON、範囲外でOFFします。                 | 無条件にOFF       |
|        | INP MANU | 現在位置が位置決め幅（パラメーターNo.10）の範囲内でON、範囲外でOFFします。                 |                  |

注1：パルス列制御モードでは、AUTOのときは強制的にINPになりますが、サーボOFF状態ではOFFとなります。

### 補足説明
完了ポジションNo.出力PM1～PM＊＊または現在位置No.出力PE0～PE6も同様の出力形態となります。
