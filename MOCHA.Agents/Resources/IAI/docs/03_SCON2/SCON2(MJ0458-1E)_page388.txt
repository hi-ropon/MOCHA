| 087            | ロードセルキャリブレーション処理中移動指令 | 原因: ロードセルキャリブレーション中に移動指令を行った。<br>対策: ロードセルキャリブレーションが終了したことをキャリブレーション完了（CEND）信号で確認した後、CLBR信号をOFFにしてから移動指令を行ってください。   |
| 088            | ロードセル無効時カセンサー押付け指令    | 原因: ロードセル無効状態で力制御による押付け指令を行った。<br>対策: コントローラーを再起動し、ロードセルが有効な状態になってから力制御による押付け指令を行ってください。                                               |
| 090            | サーボON状態でのソフトウェアリセット指令 | 原因: サーボON状態のときにソフトウェアリセット指令を行った。<br>対策: サーボOFF状態（SV信号がOFFの状態）を確認してからソフトウェアリセット指令を行ってください。                                                     |
| 091            | ティーチ時ポジションNo.異常           | 原因: ティーチング（教示）時、範囲外のポジションNo.の指定を行った。<br>対策: 指定ポジションNo.を63以下にしてください。                                                                                               |
| 092            | 移動中PWRT信号検出                   | 原因: PIOパターン1の教示モードで、現在位置書込み信号（PWRT）がジョグ移動中のときに入力された。<br>対策: 停止中（移動中信号MOVEがOFF状態）を確認してから入力してください。                                               |

| 095            | 基準位置移動指令でのDCLR信号検出       | 原因: 1.偏差カウンタークリアー（DCLR）信号がONの状態で基準位置移動指令（RSTR）信号をONした。<br>2.基準位置移動中にDCLR信号がONされた。<br>対策: 適切なタイミングでDCLR信号をONしてください。                   |
