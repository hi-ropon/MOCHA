### テキスト
#### 9.2 安全機能 SS1-t 仕様

（2）電気的仕様  
- 安全要求入力信号（SRI）と外部機器モニター出力信号（EDM）の電気的仕様について記載。  
- 安全入力信号のテストパルスは外部から入力されるもので、オフ時間は1ms以下にする必要がある。  
- 安全機能用I/Oケーブルは別売り品で、安全機能用I/Oコネクターと安全機器間の接続用。ケーブル長は3.0m。

#### （3）安全入力信号のテストパルス  
- 外部から入力されるテストパルスのオフ時間は1ms以下にしてください。

#### （4）安全機能用I/Oケーブル（別売り品）  
- 安全機能用I/Oコネクターと安全機器間の接続用ケーブル。  
- ケーブル長：3.0m  
- 型番ラベル：CB-SC-STO＊＊＊

### 表
#### 電気的仕様

| 項目                         | 仕様           | 備考               |
|------------------------------|----------------|--------------------|
| **安全要求入力信号（SRI）** |                |                    |
| オン入力電圧範囲             | 24V±10%        |                    |
| オフ入力電圧範囲             | 0-2V           |                    |
| 入力電流                     | 7.6mA（Typ）   | 1chあたりの値です。 |
| リアクションタイム           | 500ms以下      | SS1-t仕様          |
| **外部機器モニター出力信号（EDM）** |          |                    |
| 電圧範囲                     | 24V±10%        |                    |
| 出力電流                     | 100mA（Max.）  |                    |
