### テキスト
#### 3.4 フィールドネットワークコネクターの接続
各フィールドネットワークの配線方法については、以下の取扱説明書を確認の上作業を行ってください。

### 表
| フィールドネットワーク名 | 内容                                                                                     | 参照           |
|--------------------------|------------------------------------------------------------------------------------------|----------------|
| CC-Link                  | PIO同様の制御信号を用いたI/O通信、または数値データ通信でアクチュエーターの制御を行うことが可能です。 | 別冊 MJ0469    |
| CC-Link IE Field         | 同上                                                                                     | 同上           |
| DeviceNet                | 同上                                                                                     | 同上           |
| EtherCAT                 | 同上                                                                                     | 同上           |
| EtherNet/IP              | 同上                                                                                     | 同上           |
| MECHATROLINK-Ⅲ（標準 I/O プロファイル） | 同上                                                                                     | 同上           |
| PROFINET IO              | 同上                                                                                     | 同上           |
| EtherCAT（CiA401 Drive Profile） | 定周期的な通信で複数のアクチュエーターの同期制御を行うことが可能です。                              | 別冊 MJ0472    |
| MECHATROLINK-Ⅲ（標準サーボプロファイル） | 同上                                                                                     | 別冊 MJ0471    |
| RS-485                   | 汎用プロトコル“Modbus”通信を用いてアクチュエーターを制御します。                              | 別冊 MJ0162    |
| Modbus/TCP               | TCP/IPを用いたModbus通信です。EtherNet/IP仕様を選択の場合に使用できます。                     | 別冊 MJ0469、MJ0162 |

### テキスト（注記）
- 本コントローラーはスレーブユニット（子局）となります。  
- 各ネットワークの詳細は、各社のマスターユニットおよび搭載されるPLCの取扱説明書を確認してください。  
- フィールドネットワークの取扱いについては、取扱説明書が別冊となっているため、本書とあわせて利用してください。
