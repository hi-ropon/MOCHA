### テキスト
#### パルス列制御モードのI/O信号と各機能
パルス列制御モードにおけるフラットケーブルの信号割付けは、PIOパターン（インクリ／アブソ）に合わせて設定する必要があります。  
アクチュエーターの仕様に応じてPIOパターンを設定し、本表に従って外部機器（PLCなど）と接続を行います。  
「○」は使用可、「×」は使用不可を示します。

### 表の説明（抜粋）
| 信号略称 | 機能の概要 |
| :------- | :--------- |
| SON | サーボON信号。ONの間サーボON、OFFの間サーボOFF。 |
| RES | アラームリセット信号。ONでアラームリセット。 |
| HOME | 原点復帰信号。ONで原点復帰動作。 |
| TL | トルク制限選択信号。パラメーター設定値でトルク制限。 |
| CSTP | 強制停止信号。10ms以上連続ONで強制停止。 |
| DCLR | 偏差カウンタークリア信号。 |
| BKRL | ブレーキ強制解除信号。 |
| RMOD | 運転モード切替信号。AUTOモード時に切替可能。 |
| RSTR | 基準位置移動指令信号。パラメーター設定位置へ移動。 |
| PWR | システム準備完了出力。 |
| SV | サーボON状態出力。 |
| INP | 位置決め完了出力。 |
| HEND | 原点復帰完了出力。 |
| TLR | トルク制限中出力。 |
| *ALM | コントローラーアラーム状態出力（負論理）。 |
| *EMGS | 非常停止状態出力（負論理）。 |
| RMDS | 運転モード状態出力。 |
| ALM1～ALM8 | アラームコード出力。 |
| *ALMI | 軽故障アラーム出力（負論理）。 |
| REND | 基準位置移動完了出力。 |
| ZONE1, ZONE2 | ゾーン信号出力。 |
