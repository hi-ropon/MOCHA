### テキスト
#### 2.8.2 設置および取付け
制御盤の大きさ、コントローラーの配置、冷却などを考慮し、周囲温度が0～60℃になるように設計・製作を行うこと。

### 表
| 項目 | 内容 |
| :--- | :--- |
| 設置 | 設置方向 | 垂直設置（排気側が上側） |
|  | 設置方法 | ねじ取付、もしくは、DINレール取付け |
|  | 設置条件 | 図、設置条件、および、設置条件注記参照 |
| 接地 |  | 必須（アースが不完全な場合、感電の恐れあり） |

### 図
- 左図はコントローラーを複数並べて設置する場合の間隔を示す。  
  - コントローラー間は1mm以上  
  - 両端から壁面までは50mm以上  
  - 天井と床の位置も示されている  
- 右図はコントローラー単体設置時の周囲空間を示す。  
  - 天井側、床側ともに100mm以上の空間確保が必要  
  - 配線スペースを十分確保することを指示

```mermaid
flowchart TD
    A[設置方向] -->|垂直設置| B[排気側が上側]
    A --> C[設置方法]
    C -->|ねじ取付| D[コントローラー固定]
    C -->|DINレール取付け| D
    D --> E[設置条件参照]
    F[接地] -->|必須| G[感電防止]
```

**図の説明:**  
- 設置は垂直設置で排気側が上になるようにする。  
- 取付方法はねじまたはDINレール。  
- 接地は必須で、アースが不完全だと感電の恐れがある。  
- コントローラー間隔や周囲空間はファン交換作業性を考慮し、最低限の寸法を確保する必要がある。  
- 作業性を考慮しない場合でも50mm以上の間隔を確保することが推奨されている。
