### テキスト
#### 移動指令種別（パラメーターNo.27）
- No.27  
- 名称：移動指令種別  
- 単位：なし  
- 入力範囲：0：レベル、1：エッジ  
- 工場出荷時の初期値：0  

PIOパターン4は電磁弁モード1（7点タイプ）、PIOパターン5は電磁弁モード2（3点タイプ）、PIOパターン7はカセンサー使用押付けモード2（電磁弁モード）となっている場合のスタート信号（ST0～ST6、PIOパターン5はST0～ST2）入力方式を設定します。

### 表
| 設定値 | 入力方式 | 内容 |
| :---: | :---: | :--- |
| 0 | レベル | 入力信号のONで移動を開始し、移動途中でOFFになると減速停止し動作完了となる。 |
| 1 | エッジ | 入力信号の立ち上がりエッジで移動を開始し、移動途中でOFFになっても停止せず目標位置に到達する。 |

### グラフ
#### レベル方式
```mermaid
flowchart LR
    A[移動指令入力 (ST0～ST6)] -->|ON| B[アクチュエーター移動開始]
    B --> C[移動途中でOFF]
    C --> D[減速停止]
```

#### エッジ方式
```mermaid
flowchart LR
    A[移動指令入力 (ST0～ST6)] -->|立ち上がりエッジ| B[アクチュエーター移動開始]
    B --> C[移動途中でOFF]
    C --> D[停止せず移動継続]
    D --> E[動作完了（目標位置）]
    E -->|到達完了 (PE0～PE6)| F[停止]
```

**図の説明:**  
- レベル方式は入力信号がONの間アクチュエーターが動作し、途中でOFFになると減速停止して動作完了となる。  
- エッジ方式は入力信号の立ち上がりエッジで動作開始し、途中でOFFになっても停止せず目標位置まで移動を継続する。
