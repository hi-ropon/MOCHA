### テキスト
#### ブレーキ電源供給（電源コネクター）
- ブレーキ付きアクチュエーター使用時は、DC24V±10%、最大1Aを供給すること。
- 配線方法：
  1. マイナスドライバーなどで突起部分を押し込み、挿入口を開く。
  2. 適合電線径を満たす電線を10mmストリップし、開口部に挿入。
  3. マイナスドライバーで端子の突起部から離す（挿入口が閉じて配線を固定）。
  4. 同様の手順で全配線を行い、付属コネクターをシステムI/Oコネクターに挿入。

### 図
```mermaid
flowchart LR
    A[マイナスドライバー] --> B[突起部分を押し込み挿入口を開く]
    B --> C[10mmストリップした電線を挿入]
    C --> D[マイナスドライバーで突起部から離す]
    D --> E[挿入口が閉じて配線固定]
    E --> F[付属コネクターをシステムI/Oコネクターに挿入]
```
**図の説明:**  
- マイナスドライバーで挿入口を開き、10mmストリップした電線を挿入し、挿入口を閉じて配線を固定。  
- 付属コネクターをシステムI/Oコネクター（型式：B2CF_3.50/08/180_SN_BK_BX）に接続。

### 表
#### システムI/Oコネクター信号名と内容、適合電線径

| 信号名 | 内容               | 適合電線径                     |
| :----- | :----------------- | :----------------------------- |
| BK+    | ブレーキ電源入力＋側 | 1.25～0.5mm²（AWG16～20）      |
| BK－   | ブレーキ電源入力－側 | 1.25～0.5mm²（AWG16～20）      |

### システムI/Oコネクター ピン配列
```
S1側: STOP+ / ENB+ / BK+
S2側: STOP－ / ENB－ / BK－
```
