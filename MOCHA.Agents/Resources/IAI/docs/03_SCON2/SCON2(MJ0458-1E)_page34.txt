### テキスト
#### 各部の名称と機能
3. 充電状態表示LED（CHARGE）  
コントローラー内部の充電状態を表示します。  
**注意**  
- 本LED点灯中は感電防止のため、コントローラーや回生抵抗ユニットに触れないでください。

4. 電源コネクター（PWR）  
コントローラーに動力と制御用電源の供給を行うためのコネクターです。  
[3.3.1 電源回路の配線]参照

5. システムI/Oコネクター（SYS）  
停止スイッチ、イネーブルスイッチ、ブレーキ電源を接続するためのコネクターです。  
[3.3.2 非常停止回路（システムI/O）の配線]参照

6. SIOコネクター（SIO）  
パソコン専用ティーチングソフトなどのティーチングツールおよびR-unitなどの通信ケーブル接続用コネクターです。  
[3.3.6 SIOコネクターの接続]参照

7. 動作モード設定スイッチ（MANU/AUTO）  
インターロック用のスイッチです。

8. ジョグスイッチ  
JOG＋側にスイッチを傾けると＋方向に、JOG－側に傾けると－方向にジョグ動作を行います。

### 表
#### 動作モード設定スイッチの設定と機能

| スイッチの設定 | 機能                                                                                   |
| :------------- | :------------------------------------------------------------------------------------- |
| AUTO           | PIO信号による自動運転が可能となります。<br>ティーチングツールからは、モニター操作にかぎり行うことが可能です。 |
| MANU           | ティーチングツールからの手動操作が可能です。                                         |
