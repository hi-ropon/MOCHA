### テキスト
#### 3.2.3 配線

（1）主電源回路  
コントローラーの電源電圧値（AC100V / 200V）は変更不可。

（2）ブレーキ電源供給回路  
ブレーキ付きアクチュエーター使用時はDC24Vを供給すること。

### 図
#### （1）主電源回路
```mermaid
flowchart LR
    L1[L1] -->|漏電ブレーカー| A[サーキットブレーカー]
    L2[L2] -->|漏電ブレーカー| A
    A --> B[ノイズフィルター]
    B --> C[SCON2 電源コネクター]
    B --> D[サージプロテクター]
    C --> E[L1c 制御電源]
    C --> F[L2c 制御電源]
    C --> G[L1 モーター電源]
    C --> H[L2 モーター電源]
    C --> I[PE 接地]
```
**図の説明:**  
- L1、L2から漏電ブレーカーを経てサーキットブレーカーへ  
- サーキットブレーカーからノイズフィルター、サージプロテクターへ接続  
- ノイズフィルターからSCON2電源コネクターへ接続し、制御電源（L1c、L2c）とモーター電源（L1、L2）に分岐  
- PEは接地線

#### （2）ブレーキ電源供給回路
```mermaid
flowchart LR
    DC24V --> SCON2[システムIOコネクター]
    SCON2 --> BKplus[BK+]
    SCON2 --> BKminus[BK-]
    BKminus --> Zero[0V]
```
**図の説明:**  
- DC24VがシステムIOコネクターのBK+端子に供給され、BK-端子から0Vに戻る回路  
- ブレーキ付きアクチュエーター使用時にDC24Vを供給する必要あり
</contents>
