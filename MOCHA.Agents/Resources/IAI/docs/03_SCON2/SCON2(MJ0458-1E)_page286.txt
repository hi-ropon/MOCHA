### テキスト
#### 衝突判定電流値
衝突判定電流値はフィードバック電流値と比較し、フィードバック電流値が衝突判定電流値を超えた場合に電流に関する衝突検出と判定される。判定電流値が「0」に設定されている場合は判定無効となり、「0」より大きい値が設定されている場合に判定が有効となる。

アクチュエーターは加減速域で電流値が上昇するため、電流判定のみでの衝突検出は誤検出の可能性があるが、判定距離を同時に設定することで加減速域での誤検出を防止できる。

なお、判定値に「0（無効）」を同時に設定した場合はエラーNo.0A1「パラメータデータ異常」となる。

### グラフ
- 縦軸：電流（％）
- 横軸：時間（ms）

グラフはフィードバック電流値の時間変化を示し、以下のポイントが示されている。
- 衝突判定電流値のラインが引かれている。
- 衝突発生時に電流が判定値を超える。
- 負荷変動により電流が一時的に下がる。
- 移動開始カウント開始から2msでカウントアップ、1msで減算、5msまでカウントアップの動作が示されている。
- 4ms後に衝突検出が行われる。
- 反転動作中の時間帯も示されている。

```mermaid
flowchart LR
    A[移動開始カウント開始] --> B[2までカウントアップ]
    B --> C[1まで減算]
    C --> D[5までカウントアップ]
    D --> E[衝突検出]
```

**図の説明:**
- フィードバック電流値が衝突判定電流値を超えた時点で衝突発生と判定。
- 負荷変動により電流が一時的に低下するが、判定は継続される。
- 衝突検出は一定時間のカウント処理を経て行われる。
