### テキスト
#### 4.2 ポジショナーモードの運転
1) 用途例として、150mmの位置から0mmの位置への移動途中で速度変更を行う。まずポジションNo.1に一段目の速度による目標位置への位置決めを設定。位置決め幅は目標位置に対してどこで変速させるかを設定し、動作例では100mmに設定。したがってポジションNo.1では目標位置の100mm手前で位置検知出力信号LS1がONになる。

2) ポジションNo.0には2段目の速度による目標位置への位置決めを設定。

3) ポジションNo.1（ST1信号）を起動し、ポジションNo.2の位置検知出力信号LS1によってポジションNo.0（ST0信号）を起動後、ST1信号をOFF。ST1信号がOFFした時点でポジションNo.0の動作に切り替わる。

動作例ではポジションNo.0と1の目標位置を同じにしているが、異なっても問題ない。ただし同じにすると目標位置に対してどこで変速を行うかが分かりやすい。入力信号の受付タイミングにより若干変速が遅れることがあるため、位置決め幅を調整することが推奨される。

ポジションNo.2に位置決め完了状態からポジションNo.1へ移動中に変速しポジションNo.0へ移動する場合のタイムチャートを示す。

```mermaid
    C --> D[ポジションNo.0に位置決め完了]

    subgraph 信号

        ST1[スタート信号 ST1 (PLC→コントローラー)]
        ST2[スタート信号 ST2 (PLC→コントローラー)]
        LS0[位置検知出力 LS0 (コントローラー→PLC)]
        LS1[位置検知出力 LS1 (コントローラー→PLC)]
        LS2[位置検知出力 LS2 (コントローラー→PLC)]
    end

    ST1 -->|ON| B
    LS1 -->|位置決め幅を変速開始位置に設定| B
    ST0 -->|ON| C
    ST1 -->|OFF| C
    C --> D

    classDef signal fill:#f9f,stroke:#333,stroke-width:1px;
    class ST0,ST1,ST2,LS0,LS1,LS2 signal;
```

**図の説明:**
- スタート信号ST1がONになるとポジションNo.1の速度で移動開始。
- 位置検知出力LS1がONになる位置で変速開始位置を設定。
- ポジションNo.0のスタート信号ST0がONになり、ST1がOFFになるとポジションNo.0の速度で移動に切り替わる。
- 最終的にポジションNo.0の位置決めが完了する。
