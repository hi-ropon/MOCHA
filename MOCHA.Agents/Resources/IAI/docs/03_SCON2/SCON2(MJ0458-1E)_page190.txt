### テキスト
#### 4.2 ポジショナーモードの運転

■制御方法  
ポジションデータの「負荷電流しきい値」に[%]で設定した押付け力を監視し、押付け力が次の条件となったとき、トルクレベルステータス信号TRQSをONします。その際ポジションゾーン内であれば負荷出力判定信号LOADもONします。  
別のポジションへの移動指令またはサーボOFFにより、この信号はOFFします。本信号は一度ONすると、次の移動指令までONを維持します。

●条件  
（押付け力が力判定マージン範囲内の累積時間）  
－（押付け力が力判定マージン範囲外の累積時間）≧255ms（パラメーターNo.50）

### 図（押付け力と信号の時間変化）

```mermaid
flowchart LR
    A[ワークに接触] --> B[押付け力上昇]
    B --> C{力判定マージンを超える押付け力発生}
    C --> D[TRQS ON]
    C --> E[LOAD ON（ポジションゾーン内の場合）]
    D --> F[TRQS維持]
    E --> F
    F --> G[別ポジション移動指令またはサーボOFFでOFF]

    subgraph 押付け力の時間変化
        direction LR
        H[力判定マージン-側(FJML)] --> I[押付けしきい値[%]] --> J[力判定マージン+側(FJMM)]
    end

    subgraph 時間軸[時間(ms)]
        K[100ms] --> L[20ms] --> M[180ms]
    end

    note right of M
        100ms - 20ms + 180ms ≧ 255ms (PSWT/LDWT)
        負荷出力判定(TRQS) ON
        押付け完了信号(PEND) ON
    end
```

**図の説明:**  
- 押付け力はワークに接触後、力判定マージンの範囲を超えて発生し、一定時間（255ms以上）累積するとTRQS信号がONになる。  
- TRQS信号は別の移動指令またはサーボOFFまでONを維持する。  
- ポジションゾーン内であればLOAD信号もONとなる。  
- 時間軸は押付け力の累積時間を示し、100ms、20ms、180msの区間で判定されている。
