### テキスト
#### RCON接続仕様
インターフェイスは通常のSCON2と基本的に同様ですが、製品の仕様上、以下のスイッチやコネクターが無効となります。

### 表
| 無効スイッチ／コネクター | 制限事項 |
| :----------------------- | :------- |
| 6. SIOコネクター | ティーチングツールは、RCONゲートウェイユニットのSIOコネクター経由で接続します。TPアダプター（RCB-LB-TGS）もゲートウェイユニットに接続します。 |
| 7. 動作モード設定スイッチ | 動作モードは、RCONゲートウェイユニットの動作モード設定スイッチで切替えます。 |
| 11. USBコネクター | ティーチングは、RCONゲートウェイユニットのSIOコネクター経由で行ないます。 |
| 13. マルチファンクションコネクター | パルス列制御は、RCON接続仕様では使用できません。 |
