### テキスト
#### 6.1 パラメーター
パラメーター設定に関する説明があり、ドライブレコーダー機能や衝突判定に関するパラメーターの概要が記載されています。各パラメーターの詳細は別章の該当機能説明を参照するよう案内されています。

---

### 表

#### ドライブレコーダー保存履歴数選択（パラメーターNo.211）

| No.  | 名称                     | 単位 | 入力範囲               | 工場出荷時の初期値 |
| :--- | :----------------------- | :--- | :--------------------- | :----------------- |
| 211  | ドライブレコーダー保存履歴数選択 | －   | 0：3履歴<br>1：6履歴<br>2：12履歴 | 0                  |

- ドライブレコーダー機能で保存可能なアラーム履歴数を設定する。
- 詳細は「8.4 ドライブレコーダー機能」を参照。

---

#### ドライブレコーダーサンプリング周期（パラメーターNo.212）

| No.  | 名称                     | 単位 | 入力範囲   | 工場出荷時の初期値 |
| :--- | :----------------------- | :--- | :--------- | :----------------- |
| 212  | ドライブレコーダーサンプリング周期 | ms   | 1～1,000   | 10                 |

- ドライブレコーダー機能でデータを取得する周期を変更する。
- 詳細は「8.4 ドライブレコーダー機能」を参照。

---

#### 衝突判定電流値1、2、3（パラメーターNo.213、217、221）

| No.  | 名称           | 単位 | 入力範囲               | 工場出荷時の初期値 |
| :--- | :------------- | :--- | :--------------------- | :----------------- |
| 213  | 衝突判定電流値1 | %    | 0～<br>アクチュエーターによる |                   |
| 217  | 衝突判定電流値2 | %    |                         | 100                |
| 221  | 衝突判定電流値3 | %    |                         |                   |

- 設定した電流値以上かつパラメーターNo.215、219、223の「衝突判定時間1～3」で設定した時間に達したときに衝突を検出する。
- 設定値が「0」の場合は無効。
- 詳細は「5.3 衝突検出」を参照。

---

#### 衝突判定距離1、2、3（パラメーターNo.214、218、222）

| No.  | 名称           | 単位 | 入力範囲               | 工場出荷時の初期値 |
| :--- | :------------- | :--- | :--------------------- | :----------------- |
| 214  | 衝突判定距離1   | mm   | 0.00～<br>アクチュエーターによる |                   |
| 218  | 衝突判定距離2   | mm   |                         | 5.00               |
| 222  | 衝突判定距離3   | mm   |                         |                   |

- 「仮想位置差分値」（現在位置と仮想位置の差分の絶対値）が設定した衝突判定距離に達したとき、パラメーターNo.215、219、223の「衝突判定時間1～3」のカウントを開始する。
- 設定値が「0」の場合は無効。
- 詳細は「5.3 衝突検出」を参照。
