### テキスト
#### 4.2 ポジショナーモードの運転 - 押付け動作の完了判定

(1) 標準の場合（PIOパターン4）の場合  
ポジションテーブルの「押付け」にパーセント[%]で設定したトルク（電流制限値）を監視し、押付け動作中の負荷電流が条件を満たすと押付け完了信号PENDをONにします。  
ワークが停止していなくても条件を満たすとPENDはONします。  
電流が押付け[%]に達した累積時間が255ms以上（パラメーターNo.6）で判定されます。

(2) カセンサー使用押付け（PIOパターン7）の場合  
押付け動作中の押付け力が、ポジションテーブルの「押付け」にパーセント[%]で設定した押付け力に対し、条件を満たすと押付け完了信号PENDをONにします。  
アクチュエーターが停止していなくても条件を満たすとPENDはONします。  
条件は押付け力が力判定マージン＋－の範囲にある累積時間が255ms以上（パラメーターNo.6）で判定されます。

#### (1) 標準の場合（PIOパターン4）の電流と時間の関係グラフ
```mermaid
flowchart LR
    A[動作開始] --> B[アプローチ動作終了 押付け開始]
    B --> C[ワークに接触]
    C --> D[押付け[%]に達した累積時間 ≥ 255ms]
    D --> E[押付け完了 (PEND出力)]

    subgraph 時間軸
        direction LR
        T1[200ms] --> T2[20ms] --> T3[75ms]
    end

    C -.->|ワークの移動により電流減少| D
```
**図の説明:**  
- 動作開始からアプローチ動作終了後に押付け開始  
- ワークに接触し、押付け[%]に達した累積時間が255ms以上で押付け完了信号PENDが出力される  
- ワークの移動により電流が一時的に減少することがある

#### (2) カセンサー使用押付け（PIOパターン7）の押付け力と時間の関係グラフ
```mermaid
flowchart LR
    A[ワークに接触] --> B[押付け力上昇]
    B --> C[力判定マージン＋[%]]
    C --> D[押付け設定値[%]]
    D --> E[力判定マージン－[%]]

    subgraph 時間軸
        direction LR
        T1[150ms] --> T2[50ms] --> T3[135ms] --> T4[30ms] --> T5[50ms]
    end

    B -.->|ワークの移動により電流減少| C
    D -.->|ワークが外力により押し戻され押付け力増大| E
```
**図の説明:**  
- ワークに接触後、押付け力が上昇し力判定マージン＋の範囲に入る  
- その後押付け設定値を中心に力判定マージン＋－の範囲内で変動  
- ワークの移動により押付け力が減少したり、外力で押し戻され押付け力が増大することがある  
- 累積時間が255ms以上で押付け完了信号PENDがONになる
