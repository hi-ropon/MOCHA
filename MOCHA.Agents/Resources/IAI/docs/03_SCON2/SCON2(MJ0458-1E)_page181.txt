### テキスト
#### 注意事項
- PIOパターン6のカセンサー使用押付けを行う場合は、ロードセルのキャリブレーションが必要です。詳細は「4.2.7 カセンサー使用押付けの運転準備（ロードセルのキャリブレーション）」を参照してください。

#### 位置決め完了信号PENDの処理について
- 制御方法は位置決めと同様ですが、PEND信号の処理が異なります。
- 位置決め完了信号PENDは押付けにより軸が停止（押付け完了）したときに出力されます。
- ワークに押当たらない場合（空振り）は、「位置決め幅」の設定分だけ移動して停止しますが、PENDはONしません。

### 図（タイミングチャート）
```mermaid
flowchart LR
    subgraph 信号
        A[指令ポジション No. PC1～PC** (PLC→コントローラー)]
        B[スタート信号 CSTR (PLC→コントローラー)]
        C[完了ポジション No. PM1～PM** (コントローラー→PLC)]
        D[位置決め完了信号 PEND (コントローラー→PLC)]
        E[移動中信号 MOVE (コントローラー→PLC)]
    end

    subgraph アクチュエーターの動作
        F[アプローチ動作]
        G[押付け動作]
        H[押付け完了]
    end

    A -->|T1≥6ms| B
    B --> C
    C --> D
    D --> E

    F --> G --> H

    note1["注1: ポジションNo.の入力からCSTRのONまで6ms以上空けること"]
    note2["注2: 完了ポジションNo.出力は移動中0"]

    B -.-> note1
    C -.-> note2

    %% 信号の説明
    C ---|PM1～PM**=0| D
    D ---|PENDのOFFによりOFFしてください| B

    %% アクチュエーター動作詳細
    F -->|座標値設定による位置決め動作| G
    G -->|位置決め幅による移動| H
    H -->|押付け停止| end
```

**図の説明:**  
- 指令ポジションNo.入力後、6ms以上の遅延を置いてスタート信号CSTRがONになる。  
- 完了ポジションNo.は移動中は0で、押付け動作中に設定値となる。  
- 位置決め完了信号PENDは押付け完了時にONし、空振りの場合はONしない。  
- アクチュエーターは「アプローチ動作」→「押付け動作」→「押付け完了」の順に動作する。

### 注釈
| 注釈番号 | 内容 |
| :--- | :--- |
| 注1 | ポジションNo.の入力からCSTRのONまで6ms以上の時間を空けること。PLCのタイマー処理やスキャンタイムも考慮する必要がある。 |
| 注2 | 完了ポジションNo.の出力は移動中は0となる。 |
