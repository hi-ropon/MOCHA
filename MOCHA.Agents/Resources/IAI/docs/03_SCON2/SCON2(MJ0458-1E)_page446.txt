### テキスト
#### 一時停止回路の説明
- 一時停止はオルタネイトスイッチと同様の動作で、一度目のONで一時停止、再度ONで解除される。
- 同じ押しボタンで一時停止と解除を切り替える。
- 押しボタンが押されると「一時停止指示＋一時停止ランプ点灯」状態となり、もう一度押すと「一時停止解除指示＋一時停止ランプ消灯」となる。

### 図（回路図）
```mermaid
flowchart TD
    A[回路を分かり易くするため、b接点入力をa接点に置き換える回路] --> B[一時停止ボタン押下でAUX1がON]
    B --> C{ランプが消えていたら点ける回路}
    C --> D[インターロック: ボタン押下中は1スキャンおきにAUX2とAUX3が交互にON]
    D --> E{ランプが消えていたら消す回路}
    E --> F[一時停止中、リセットが行われなければ残移動量のキャンセルを待つタイマー]
    F --> G[一時停止中、非常停止になったら一時停止解除]
```

### 図内注釈
- AUX1: 一時停止補助1 (a018 a023)
- AUX2: 一時停止補助2 (a017 b024 a026)
- AUX3: 一時停止解除 (b019 a022 b028)
- HOLDL: 一時停止表示 (b016 a021 a027 b032 a034)
- *STP: 一時停止信号

### 図の説明
- 回路はb接点入力をa接点に置き換え、押しボタンが押されるとAUX1がONになる。
- ランプが消えている場合は点灯させる回路があり、インターロック機能でボタン押下中はAUX2とAUX3が交互にONし続ける。
- ランプが消えている場合は消灯させる回路がある。
- 一時停止中はリセット待ちタイマーが動作し、非常停止が発生すると一時停止を解除する。
