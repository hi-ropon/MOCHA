### テキスト・表
#### 8.3 アラームについて

| アラームコード | アラーム名称           | 原因/対策                                                                                                                                                                                                                                                                                                                                                  |
|:--------------|:---------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0A7           | 指令減速度異常       | **原因:** 移動中に減速度を低く変更した際、減速距離が不足しており、変更後の減速度で現在位置から減速するとソフトリミットを超えた。<br>（図参照）<br>**対策:** 移動途中で速度変更する際の次の移動指令のタイミングが遅いため、減速度変更のための移動指令のタイミングを早くすること。 |
| 0AB           | 指令速度異常         | **原因:** モーションネットワークにてPLCからの指令がアクチュエーターの最高速度を超えている。<br>**対策:** PLCのパラメーター、プログラムを見直すこと。                                                                                                                                                                                                 |
| 0B5           | Z相位置異常          | **原因:** 原点復帰時にZ相を検出した位置が規定想定外。<br>**対策:** エンコーダー不良の可能性があるため連絡。                                                                                                                                                                                                                                           |
| 0BA           | 原点センサー未検出   | **原因:** 原点センサー付きアクチュエーターの原点復帰動作が正常完了していない。<br>1. 原点復帰途中でワークが周囲と干渉。<br>2. アクチュエーターの摺動抵抗が大きい。<br>3. 原点センサーの取付不良、故障、断線。<br>**対策:** ワークが干渉していない場合は2,3を確認しメンテナンスや修理を実施。 |
| 0BE           | 原点復帰タイムアウト | **原因:** 原点復帰動作開始後、一定時間経過しても原点復帰が完了しない。<br>**対策:** 通常の動作ではなく、コントローラーとアクチュエーターの組み合わせ誤りの可能性。                                                                                                                                                                             |
| 0BF           | クリープセンサー未検出 | **原因:** 原点復帰動作時にクリープセンサー検出前に原点センサー検出、またはメカエンドに到達した状態。<br>1. クリープセンサー取付位置不適切。<br>2. クリープセンサー不良。<br>3. ケーブル断線やコネクター不良。<br>4. 干渉により負荷が大きく動かない。<br>**対策:** 取付位置調整、センサー交換、導通チェック、負荷確認。 |
| 0C0           | 実速度過大           | **原因:** モーター回転数が許容回転数を超えた。<br>1. アクチュエーターの摺動抵抗が局部的に大きい。<br>2. 瞬間的外力加わり急激な速度上昇。<br>**対策:** 組付け状態の異常確認、動作方向への外力加わりの可能性確認。                                                                                                                                         |

### 図

flowchart TD
    A[ソフトリミットを超えないときの減速開始位置] --> B[減速開始]
    B --> C[速度低下]
    C --> D[ソフトリミット]
    E[この位置で指令するとソフトリミットを超える] -.-> D

**図の説明:**  
移動中の減速開始位置がソフトリミットを超えない場合の速度低下の流れと、減速開始位置が遅くてソフトリミットを超えてしまう場合の違いを示している。
