### テキスト
#### 4.3 パルス列制御モード
【グリッパーの原点復帰動作】

- フィンガーアタッチメントはアクチュエーターの付属品ではなく、お客様で用意する必要がある。

1) HOME信号のONにより、原点復帰速度（20mm/s）でメカエンド（外側）に向かって移動する。
2) メカエンドから反転移動し、原点位置で停止する。このときの移動量はパラメーターNo.22「原点復帰オフセット量」の設定値となる。

#### 注意
- 原点逆仕様の場合は、動作方向が逆になる。
- パラメーターNo.22「原点復帰オフセット量」を変更する場合は、必ず「6.1.2 [18]」を参照すること。

### 図
```mermaid
flowchart LR
    A[HOME信号ON] --> B[原点復帰速度(20mm/s)でメカエンド(外側)へ移動]
    B --> C[メカエンドから反転移動]
    C --> D[原点位置で停止]
    D --> E[移動量はパラメーターNo.22「原点復帰オフセット量」]
```

**図の説明:**
- HOME信号ONでグリッパーが外側のメカエンドに向かって移動し、
- メカエンド到達後に反転して原点位置で停止する動作を示す。
- 停止位置はパラメーターNo.22の設定値に依存する。
