### テキスト
#### 基準位置移動（RSTR、REND）
- RSTR信号は任意に設定した基準位置へ移動するための指令信号で、立ち上がり（ONエッジ）で処理される。
- 基準位置への運転が行われ、移動完了後に基準位置移動完了信号RENDがONになる。
- アブソ仕様のアクチュエーターは原点復帰を行わないため、この位置で上位コントローラーの位置管理を行う必要がある。
- アブソ仕様のアクチュエーターにはパラメーターNo.167を設定する。

##### 基準位置への移動条件（注1）
| 項目 | 内容 |
| :--- | :--- |
| 位置 | パルス列基準位置（パラメーターNo.167設定位置） |
| 速度 | PIOジョグ速度（パラメーターNo.26設定速度） |

##### REND信号OFF条件（注2）
1. RSTR信号がON  
2. サーボOFF時  
3. 強制停止（CSTP）、原点復帰（HOME）、偏差カウンタークリアー（DCLR）のいずれかの信号でONエッジ検出  
4. AUTOからMANUにモード変更  

#### 注意事項
- HENDがOFF状態でRSTR信号をONにするとアラームになる。  
- DCLR信号がON状態でRSTR信号をONにすると基準位置移動指令でDCLR信号検出アラームになる。
