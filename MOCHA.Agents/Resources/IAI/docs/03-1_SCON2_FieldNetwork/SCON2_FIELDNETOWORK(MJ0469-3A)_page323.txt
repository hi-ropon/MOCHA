### テキスト
#### サーボゲインパラメーターセット選択（GSL0、GSL1）PLC出力信号
あらかじめ、サーボゲインパラメーター（6種類）を4セット登録しておくことで、ポジション移動ごとに選択したセットで運転可能です。  
詳細は、「SCON2取扱説明書（MJ0458）」を参照してください。

### 表
| GSL1 | GSL0 | 機能                   | 備考     |
| :--- | :--- | :--------------------- | :------- |
| OFF  | OFF  | パラメーターセット0選択 | 出荷時設定 |
| OFF  | ON   | パラメーターセット1選択 |          |
| ON   | OFF  | パラメーターセット2選択 |          |
| ON   | ON   | パラメーターセット3選択 |          |
