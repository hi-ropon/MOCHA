### テキスト
#### ステータスLED
コントローラー前面に設置されているNS（Network Status）、MS（Module Status）の2つのLEDにより、スレーブ（各コントローラー）の状態やネットワークの状態を知ることができます。

- NSはDeviceNetの通信状態を表します。
- MSはコントローラーの状態、ネットワークモジュールの状態およびDeviceNet通信の異常発生を表します。

電源投入時にはセルフテストが動作し、LEDは以下の順序で表示されます。
1. NS、MSを消灯
2. MSを緑点灯（約0.25秒間）
3. MSを橙点灯（約0.25秒間）
4. MSを消灯
5. NSを緑点灯（約0.25秒間）
6. NSを橙点灯（約0.25秒間）
7. NSを消灯

セルフテスト完了後、正常に通信を開始するとMSとNSの両方のLEDが緑点灯します。

---

### 表
#### ステータスLEDの表示内容

| パネル表記 | 表示色 | 状態 | 説明 |
| :--- | :--- | :--- | :--- |
| **NS** | 緑 | 点灯 | コネクションが確立し、正常に通信中 |
|  | 緑 | 点滅 | オンライン状態になっているが、コネクションが確立していない。通信停止中。（ネットワークは正常） |
|  | 橙 | 点灯 | ノードアドレスの重複またはBusoff検出 |
|  | 橙 | 点滅 | 通信異常（通信タイムアウト検出） |
|  | - | 消灯 | オンライン状態になっていない、または電源が供給されていません。 |
| **MS** | 緑 | 点灯 | 正常動作中 |
|  | 緑 | 点滅 | ハードウェア異常、電源再投入で回復できる場合があります。 |
|  | 橙 | 点灯 | ハードウェア異常、通信モジュールの交換が必要です。 |
|  | 橙 | 点滅 | ユーザー設定異常、コンフィグレーション異常などの軽微な異常です。再設定などで回復できます。 |
|  | - | 消灯 | DeviceNetの初期化中、または電源が供給されていません。 |

---

#### 注釈
- 通信ケーブル上でのエラー発生率が非常に高い状態（Busoff）を検出すると異常を検知します。内部のエラーカウンタはマスターの起動時やリスタート時にクリアされます。
