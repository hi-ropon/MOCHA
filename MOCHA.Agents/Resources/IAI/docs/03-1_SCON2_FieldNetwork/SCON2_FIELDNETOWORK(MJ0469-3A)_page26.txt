#### 1.2 システム構成
システム構成は以下のとおりです。

### 図
```mermaid
flowchart TD
    PLC[PLC] -->|フィールドネットワーク| SCON2[SCON2]
    SCON2 -->|電源供給| DC24V[DC24V 電源<br>型式：PSA-24<br>（ブレーキ解除用）]
    SCON2 -->|電源供給| AC100V[AC100V/200V]
    SCON2 -->|接続| 回生抵抗ユニット[回生抵抗ユニット<br>型式：RESU-2]
    SCON2 -->|接続| アブソバッテリー[アブソバッテリー<br>バッテリー単体：AB-5<br>バッテリーホルダー付：AB-5-CS3]
    SCON2 -->|接続可能| アクチュエーター[接続可能アクチュエーター]
    アクチュエーター --> RCS[RCSシリーズ]
    アクチュエーター --> IS[ISシリーズ]
    アクチュエーター --> LS[LSシリーズ]
    アクチュエーター --> NS[NSシリーズ]
    アクチュエーター --> IF[IFシリーズ]
    アクチュエーター --> DD[DDシリーズ]
    ティーチングツール[ティーチングツール]
    ティーチングツール --> ティーチングボックス[ティーチングボックス<br>型式：TB-02 TB-03]
    ティーチングツール --> パソコン専用ティーチングソフト[パソコン専用ティーチングソフト<br>型式：IA-OS]
```

**図の説明:**
- PLCはフィールドネットワークを介してSCON2に接続されている。
- SCON2はAC100V/200V電源とDC24V電源（PSA-24型、ブレーキ解除用）に接続されている。
- SCON2は回生抵抗ユニット（RESU-2型）とアブソバッテリー（AB-5単体およびAB-5-CS3バッテリーホルダー付）に接続されている。
- SCON2は複数の接続可能アクチュエーター（RCSシリーズ、ISシリーズ、LSシリーズ、NSシリーズ、IFシリーズ、DDシリーズ）に接続されている。
- ティーチングツールはティーチングボックス（TB-02、TB-03型）とパソコン専用ティーチングソフト（IA-OS型）で構成されている。
