### テキスト
#### 2.5.1 EtherNet/IP インターフェイス仕様
EtherNet/IPの詳細な説明については、「マスターユニットの取扱説明書」を参照するよう案内があります。

### 表
#### EtherNet/IP インターフェイス仕様

| 項目               | 仕様                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 通信規格           | IEC61158（IEEE802.3）                                                                  |
| 通信速度           | 10BASE-T/100BASE-T（オートネゴシエーション設定を推奨）                                |
| 通信ケーブル長     | EtherNet/IP接続仕様による（ハブと各ノード間の距離：100m以内）                         |
| コネクション数     | マスターユニットによる                                                                 |
| 設定可能ノードアドレス | 0.0.0.0 ～ 255.255.255.255                                                           |
| 通信ケーブル 注1   | カテゴリー5以上（アルミテープと編組の二重遮蔽シールドケーブル推奨）                   |
| 接続コネクター     | RJ-45コネクター1個                                                                    |

注1 通信ケーブルは別途用意する必要がある。
