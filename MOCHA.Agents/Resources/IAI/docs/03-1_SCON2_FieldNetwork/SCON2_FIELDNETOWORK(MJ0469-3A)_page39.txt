### テキスト
#### 2.1 CC-Link
CC-Linkは、Control & Communication Linkの略称で、1996年に三菱電機株式会社によって開発されたRS-485ベースのオープンフィールドネットワーク規格の一つです。  
マスタースレーブ方式のネットワークで、リンクデバイスや通信制御はマスター側で集中管理し、スレーブ側での設定は不要です。  
通信機能には、一定周期でデータを更新するサイクリック通信機能と、任意のタイミングで更新するトランジェント伝送機能があります。  
通信の二重化にも対応しており、待機マスター局を使用することでマスター局が電源断した場合も通信を継続可能です。  

※マスター局に設定するSCON2の局情報は、Ver.1リモートデバイス局に設定してください。

#### 2.1.1 CC-Linkインターフェイス仕様
CC-Linkのインターフェイス仕様は以下の通りです。

| 項目           | 仕様                                                                 |
| :------------- | :------------------------------------------------------------------- |
| 通信規格       | CC-Link Ver1.10                                                      |
| 局種別         | リモートデバイス局（最大4局占有）                                  |
| 通信速度       | 10Mbps / 5Mbps / 2.5Mbps / 625kbps / 156kbps                       |
| 通信方式       | ブロードキャストポーリング方式                                      |

| 通信ケーブル長 | 通信速度（bps） | 10M | 5M | 2.5M | 625k | 156k                      |
|                | 総ケーブル長（m） | 100 | 160 | 400 | 900 | 1200                      |
| 通信ケーブル   | 専用ケーブルを使用してください                                      |
| コネクター     | MSTB2.5/5-STF-5.08 AU（フェニックス・コンタクト）                 |

注1：T分岐通信の場合は、「マスターユニットおよび搭載されるPLCの取扱説明書」を参照してください。  
注2：ケーブル側コネクターはネットワークオプションとして標準で付属します。
