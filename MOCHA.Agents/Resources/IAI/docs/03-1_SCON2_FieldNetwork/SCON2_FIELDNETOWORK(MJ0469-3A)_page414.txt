### テキスト
#### ファンクションコード
SCON2のメモリー領域は、ワード単位で読み書きを行うModbusレジスター領域と、ビット（コイル）単位で書き込みを行うModbusステータス領域によって構成されており、構成によってファンクションコードが異なります。

### 表
#### ファンクションコード一覧

| メモリー領域       | アクセス単位 | コード | 機能                           |
| :----------------- | :----------- | :----- | :----------------------------- |
| Modbusステータス   | ビット       | 01H    | コイルの読出し                 |
|                    |              | 02H    | ステータスの読出し             |
| Modbusレジスター   | ワード       | 03H    | 複数保持レジスターの読出し     |
|                    |              | 04H    | 入力レジスターの読出し         |
|                    |              | 06H    | 保持レジスターへの書込み       |
|                    |              | 10H    | 複数保持レジスターへの一括書込み |

</contents>
