### テキスト
#### 1. ステータスLEDについて
コントローラー前面に設置されているNS（Network Status）とMS（Module Status）の2つのLEDにより、スレーブ（各コントローラー）の状態やネットワークの状態を把握できます。  
- NSはEtherNet/IPの通信状態を示す。  
- MSはコントローラーの状態、ネットワークモジュールの状態、およびEtherNet/IP通信の異常発生を示す。

以下にLEDの表示内容を示す。

### 表
#### ステータスLEDの表示内容

| 表記 | 表示色 | 状態 | 説明 |
| :--- | :--- | :--- | :--- |
| **NS** | - | 消灯 | 電源オフ、またはIPアドレス未設定 |
|  | 緑 | 点灯 | オンライン、コネクション成立 |
|  | 緑 | 点滅 | オンライン状態だがコネクション未確立。通信停止中（ネットワークは正常）。マスターユニットの状態を確認。 |
|  | 橙 | 点灯 | 致命的なエラー、IPアドレス重複エラー |
|  | 橙 | 点滅 | コネクションタイムアウト<br>※IPアドレス設定、通信ラインの配線状態、ハブの電源、ノイズ対策などを確認 |
| **MS** | - | 消灯 | 電源オフ |
|  | 緑 | 点灯 | 正常に通信中 |
|  | 緑 | 点滅 | スキャナー（マスター）とのコネクション未確立。構成情報の設定確認、スキャナーのアイドル状態確認。 |
|  | 橙 | 点灯 | 回復不可能故障、ハードウェア異常。ボード交換が必要。 |
|  | 橙 | 点滅 | 回復可能故障、コンフィグレーション異常、設定不正などの軽微な異常。再設定などで回復可能。 |
