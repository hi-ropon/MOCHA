### テキスト
#### ステータスLED
コントローラー前面に設置されているERR、RUNの2つのLEDにより、各コントローラーの状態やネットワークの状態を把握できます。

- ERRはコントローラーの状態、ネットワークモジュールの状態およびCC-Link通信の異常発生を示します。
- RUNはCC-Linkの通信状態を示します。

### 表
#### ステータスLEDの表示内容

| パネル表記 | 表示色 | 状態           | 説明                                                                                   |
|------------|--------|----------------|----------------------------------------------------------------------------------------|
| ERR        | 橙     | 点灯           | 1. CRCエラー<br>2. リセット解除時、局番設定エラー（0または占有局数を含んで65局以上）<br>3. リセット解除時、ボーレート設定エラー<br>4. 電源投入またはソフトウェアリセットからCC-Link初期化終了までの間 |
|            |        | 点滅 (0.4s点滅) | リセット解除時の局番およびボーレート設定値から値が変化した                                      |
|            |        | 消灯           | 1. 正常交信<br>2. ハードウェアリセット中                                                    |
| RUN        | 緑     | 点灯           | ネットワーク加入後のリフレッシュ＆ポーリング正常受信、またはリフレッシュ正常受信                      |
|            |        | 消灯           | 1. ネットワーク未加入<br>2. チャネルのキャリア検出<br>3. タイムオーバー<br>4. ハードウェアリセット中 |

</contents>
