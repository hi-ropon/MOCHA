### テキスト
#### プレスプログラム原点移動信号（PHOM, MPHM, PORG）
- PHOM（プレスプログラム原点移動）: PLC出力信号  
- MPHM（プレスプログラム原点移動中）: PLC入力信号  
- PORG（プレスプログラム原点位置）: PLC入力信号  

PHOM信号をONにすると、立ち上がり（ONエッジ）で処理され、PLCの指令プログラムNo.レジスターに入力されているプログラムNo.の原点位置に移動する。  
原点移動時の速度はパラメーターNo.8「速度初期値」、加減速度はパラメーターNo.9「加減速度初期値」の設定が使用される。  
プログラム原点へ移動中はMPHM信号がONになる。MPHM信号がONしたらPHOM信号はOFFにする必要がある。  
原点位置への移動完了後、PORG信号がONになる。  

※MPHM信号は減圧動作中や戻り動作中もONになる。

#### 注意事項
- プレスプログラム実行中にプレスプログラム原点移動信号を入力すると、プログラムアラーム（4C：プログラム実行中移動指令）が発生する。

        B --> C[プレスプログラム原点移動中 PRUN (SCON2 → PLC)]
        C --> D[プレスプログラム原点位置 PORG (SCON2 → PLC)]
    end

    subgraph アクチュエーター動作
        E[速度]
        F[時間]
    end

    B -.-> C
    C -.-> B
    B -->|開始| E
    D -->|終了| E

**図の説明:**  
- 指令プログラムNo.設定値が変化するとPHOM信号が立ち上がる。  
- PHOM信号の立ち上がりでプレスプログラム原点移動が開始され、PRUN信号（原点移動中）がONになる。  
- PRUN信号がONの間、アクチュエーターは速度を上げて原点位置へ移動。  
- 移動完了後、PORG信号がONになる。  
- MPHM信号はPRUN信号に対応し、移動中はONとなる。
