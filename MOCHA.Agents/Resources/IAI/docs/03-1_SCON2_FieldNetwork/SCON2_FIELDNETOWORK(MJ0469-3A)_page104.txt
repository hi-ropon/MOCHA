### テキスト
#### 1. ステータスLED
コントローラー前面に設置されたLEDにより、通信ボードやネットワークの状態を確認可能。

#### 2. MECHATROLINK-Ⅲ通信コネクター
MECHATROLINK-Ⅲ通信モジュールに実装されているコネクターの説明。

### 表
#### ステータスLEDの表示状態

| LED  | 色   | 表示状態 | 説明                                                                                   |
| :--- | :--- | :------- | :------------------------------------------------------------------------------------- |
| CON  | 緑   | ○        | CONNECT受信（マスターと接続状態(P2以上)）で点灯。DISCONNECT受信またはコントローラー再起動で消灯。いずれか1軸でも接続すると点灯。 |
|      | －   | ×        | マスターと接続できていない。全軸がP1以下の場合消灯。                                   |
| ERR  | 橙   | ○        | コマンドアラーム（ワーニング除く）発生で点灯。アラーム解除で消灯。いずれか1軸でもアラーム発生で点灯。 |
|      | －   | ×        | 正常（全軸ともコマンドアラーム未発生）                                                   |
| LK1  | 緑   | ○        | 他のMECHATROLINK-Ⅲ対応機器と物理的に接続された場合に点灯（断線等の確認用）             |
| LK2  | 緑   | ○        | 他のMECHATROLINK-Ⅲ対応機器と物理的に接続された場合に点灯（断線等の確認用）             |

※通信フェーズはP1～P2の2種類あり、詳細は「7.1.1 基本通信フローとコマンド」を参照。

#### MECHATROLINK-Ⅲ通信コネクター ピン配置と信号名称

| ピン番号 | 信号名称     | 信号略称 |
| :------- | :----------- | :------- |
| 1        | 送信データ＋ | TXD＋    |
| 2        | 送信データ－ | TXD－    |
| 3        | 受信データ＋ | RXD＋    |
| 4        | －           | 未使用   |
| 5        | －           | 未使用   |
| 6        | 受信データ－ | RXD－    |
| 7        | －           | 未使用   |
| 8        | －           | 未使用   |
