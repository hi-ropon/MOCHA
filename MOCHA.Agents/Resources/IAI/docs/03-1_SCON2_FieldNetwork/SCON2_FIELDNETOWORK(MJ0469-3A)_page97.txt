### テキスト
#### (2) サーボプレス仕様
サーボプレス仕様の設定範囲は0～2です。

#### 【2】フィールドバス通信速度（パラメーターNo.86）
通信速度は接続先PLCの仕様に従って設定します。（オートネゴシエーション）

#### 注意
- EtherNet/IPユニットのリンク設定は、接続するスイッチングハブの通信モード設定と合致させる必要があります。合致していない場合、不安定なリンク状態となり正常な通信ができません。
- コントローラーのパラメーターで自動ネゴシエーションに設定することを推奨します。

### 表
#### サーボプレス仕様設定値一覧

| 設定値（出荷時設定） | モード名               | 占有バイト数 | 内容                                                                                   |
| :-------------------: | :-------------------- | :----------: | :------------------------------------------------------------------------------------ |
| 0                     | リモートI/Oモード     | 2            | PIO（24V入出力）による運転をEtherNet/IPによって行います。                             |
| 1                     | ポジション/簡易直値モード | 32           | プレスプログラムの起動、判定結果読取りなどのサーボプレス用機能に加え、直接数値による移動や現在荷重データ読取りなどの全機能をサポートする方式です。 |
| 2                     | ハーフ直値モード       | 32           | プレスプログラムの“加圧ステージ”を、直接数値で指定する運転モードです。               |

※1 これ以外の値を入力すると入力値過大エラーとなります。

#### フィールドバス通信速度設定値一覧

| 設定値（出荷時設定） | 通信速度               |
| :-------------------: | :-------------------- |
| 0                     | 自動ネゴシエーション（推奨） |
| 1                     | 10Mbps 半二重          |
| 2                     | 10Mbps 全二重          |
| 3                     | 100Mbps 半二重         |
| 4                     | 100Mbps 全二重         |
| 上記以外              | ボーレート設定エラー    |
