### テキスト
#### ネットワークタイプ（パラメーターNo.87）
コントローラーに搭載しているネットワークモジュールの種類を指定します。初期値から変更しないでください。

#### フィールドバス入出力フォーマット（パラメーターNo.90）
設定を変更することで、PLCの入出力データレジスターとの交信域で、2ワード内のデータをバイト単位で入替えて送受信することができます。

### 表
#### フィールドバス入出力フォーマット設定値一覧

| 設定値 | 内容                                                                                         |
| :----- | :------------------------------------------------------------------------------------------- |
| 0      | 入替えは行いません。PLCにはそのまま送信されます。                                           |
| 1      | 上位ワードの上位バイトと下位バイトを入替え、また下位ワードの上位バイトと下位バイトを入替えます。 |
| 2      | ワードレジスターの場合、上位ワードと下位ワードを入替えます。                               |
| 3 (出荷時設定) | 上位ワードの上位バイトと下位バイトを入替え、また下位ワードの上位バイトと下位バイトを入替えます。ダブルワードの場合、さらに上位ワードと下位ワードを入替えます。 |

### テキスト
#### IPアドレス（パラメーターNo.140）
IPアドレスは、パラメーターNo.140「IPアドレス」で設定します。  
設定可能範囲は、0.0.0.0 ～ 255.255.255.255です。出荷時は192.168.0.1に設定されています。

※ IPアドレスの値はティーチングツールにより表示が異なります。  
パソコン専用ティーチングソフトの場合、ドット付き10進記法で表示されます。  
ティーチングボックスの場合、8桁の16進数で表示されます。  
例：IPアドレス 192.168.0.1（ドット付き10進記法）⇔ C0A80001（16進数）

#### 注意
- IPアドレスが他の機器と重複しないようにしてください。  
- IPアドレスが重複している場合は、マスターユニット側で通信異常が発生します。  
- 詳細は「PLCならびにマスターユニットの取扱説明書」を参照してください。
