<overview>
このページは、EtherCAT通信ボードのステータスLEDの説明で構成されています。  
主にテキスト説明と表で構成されており、表は「RUN（ラン）」と「ERR（エラー）」の2種類のLED状態について、LEDの色や点灯パターンごとの状態内容を示しています。  
グラフや図はなく、LEDの点灯タイミングに関する補足が記載されています。
</overview>

<contents>
### テキスト
#### ステータスLED
コントローラー前面に設置されたLEDにより、通信ボードの状態やネットワークの状態を確認できます。

### 表
#### LED名称ごとの色（状態）と内容

| LED名称 | 色（状態） | 内容 |
| :--- | :--- | :--- |
| **RUN（ラン）** | －（消灯） | 初期化状態（EtherCAT通信 初期化状態）、または電源OFF |
|  | 緑（点灯） | 正常運転状態（EtherCAT通信 オペレーション状態） |
|  | 緑（点滅）※1 ON：200ms OFF：200ms | EtherCAT通信 プレオペレーション状態 |
|  | 緑（点滅）※2 ON：200ms OFF：1000ms | EtherCAT通信 セーフオペレーション状態 |
|  | 緑（明滅）※4（Flickering） | EtherCAT通信 “BOOT”状態 |
|  | 橙（点灯） | モジュール異常 |
| **ERR（エラー）** | －（消灯） | 異常なし、または電源OFF |
|  | 橙（点灯） | 通信部品（モジュール）異常 |
|  | 橙（点滅）※1 ON：200ms OFF：200ms | 通信設定異常（マスターから受取った情報が設定できない） |
|  | 橙（点滅）※2 ON：200ms OFF：1000ms | 同期イベント異常 |
|  | 橙（点滅）※3 ON：200ms×2回 OFF：1000ms | 通信異常（ウォッチドッグタイマー・タイムアウト） |

※ LEDの点灯タイミングは次ページに掲載されています。
</contents>
