### テキスト
#### 3.4.2 フル機能モード
サーボプレス機能の全機能をフィールドバス通信で行うモード運転方式です。  
入出力エリアに各値を設定してください。  
本モードで制御可能なアクチュエーターの有効な主要機能は次の表のとおりです。

### 表
#### 動作モードと主要機能

| 主要機能               | ○:直接制御 | ×:無効 |
|------------------------|------------|--------|
| 位置データ指定運転     | ○          |        |
| 速度・加減速度直接指定 | ○          |        |
| 現在位置読取り         |            | ×      |
| 加圧荷重直接指定       | ○          |        |
| 現在速度読取り         | ○          |        |
| プログラム No.指定運転 | ○          |        |
| 判定結果読取り         | ○          |        |
| 現在荷重データ読取り   | ○          |        |
| 過負荷レベルモニター   | ○          |        |
| サーボゲイン切替え     | ○※1       |        |

※1 サーボゲインは、1つのプレスプログラムに1つのサーボゲインを登録できます。
</contents>
