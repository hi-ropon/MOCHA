### テキスト
#### 配線方法
CC-Link IE Fieldネットワーク接続用のRJ-45コネクター2個は、図のように接続します。コネクターはP1、P2がありますが、IN/OUTの概念はなく、上位に接続する側を任意に設定可能です。

1. CC-Link IE Field用ケーブルをマスターユニットに接続し、もう一方のコネクターをSCON2のP1コネクターに挿入します。
2. SCON2のP2コネクターにEtherCAT用ケーブルのコネクターを接続します。
   - もう一方のコネクターは次に接続するスレーブユニットのコネクターに挿入します。

通信ケーブルのコネクターは、カチッと音がして固定されるまで確実に接続してください。

#### Point！
- Ethernetケーブルはカテゴリー5e以上のストレートケーブルで、100m以内を推奨（アルミテープと編組の二重遮蔽シールドケーブル推奨）
- スレーブ側の終端コネクターには何も接続しない（終端抵抗は不要）
- マスター側の配線については、使用製品の取扱説明書を参照

### 図
```mermaid
flowchart LR
    PLC[PLC] -->|CC-Link IE Field ケーブル| SCON2_P1[SCON2 P1コネクター]
    SCON2_P1 -->|SCON2内部| SCON2_P2[SCON2 P2コネクター]
    SCON2_P2 -->|EtherCAT ケーブル| Slave_P1[スレーブユニット P1コネクター]
    Slave_P1 --> Slave_P2[スレーブユニット P2コネクター]
```

**図の説明:**
- PLCからSCON2のP1コネクターへCC-Link IE Field用ケーブルを接続
- SCON2のP2コネクターからEtherCAT用ケーブルをスレーブユニットのP1コネクターへ接続
- スレーブユニットのP2コネクターは次のスレーブユニットへ接続可能
- CC-Link IE Fieldは終端処理不要
