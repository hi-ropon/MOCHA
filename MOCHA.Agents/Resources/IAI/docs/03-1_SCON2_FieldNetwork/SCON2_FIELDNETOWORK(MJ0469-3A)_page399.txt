### テキスト
#### 7.1.4 共通コマンドの仕様

【無効（NOP コード：00H）】  
- ネットワーク管理時に無効コマンドとして使用される。  
- レスポンスは現在の状態を返す。  
- 電源投入からイニシャライズ処理完了まで、レスポンスはNOPとなり、CMD_STATのCMDRDYビットは“0”になる。  
- DISCONNECTコマンド以外は受け付けない。

### 表
#### 【NOP】コマンドフォーマット

| コマンドフォーマットのバイト | コマンド    | レスポンス | 備考               |
| :---------------------------- | :---------- | :--------- | :----------------- |
| 0                             | 00H         | 00H        |                    |
| 1                             | WDT         | RWDT       | [7.1.3 (2)]参照    |
| 2                             | CMD_CTRL    | CMD_STAT   | [7.1.3 (3)]参照    |
| 3                             |             |            |                    |
| 4 - n※1                      | 予約        | 予約       |                    |

※1 コマンドフォーマットのバイトモードにより数値が変わる。  
- 16バイトモード：n=15  
- 32バイトモード：n=31  
- 48バイトモード：n=47
