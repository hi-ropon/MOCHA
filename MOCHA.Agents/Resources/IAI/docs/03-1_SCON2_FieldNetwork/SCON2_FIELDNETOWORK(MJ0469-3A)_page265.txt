### テキスト
#### 入出力信号割付けについて
- 表中のONは該当ビットが“1”、OFFは“0”を表す。

### 表
#### PLC出力の信号種類と内容

| 信号種類       | ビット           | 記号 | 内容                                                                                                                                                                                                                     | 詳細   |
|----------------|------------------|------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| 目標位置       | 32ビットデータ   | －   | - 32ビット符号付整数<br>- 目標位置を絶対座標上の位置で指定<br>- 単位：0.001mm<br>- 指定可能範囲：-9999999～9999999<br>- 入力例：+25.410mmなら25410と指定<br>- パラメーターのソフトリミット内側(0.2mm)を超えた値は制限される<br>- 16進数入力時は負数は2の補数で入力 | 4.3.2  |
| 位置決め幅     | 32ビットデータ   | －   | - 32ビット整数<br>- 単位：0.001mm<br>- 指定可能範囲：1～9999999<br>- 入力例：+25.400mmなら25400と指定<br>- 目標位置からの許容範囲で位置決め完了とみなす                                                                 | 4.3.2  |
| 速度           | 32ビットデータ   | －   | - 32ビット整数<br>- 直値移動指令時の速度指定<br>- 単位：0.01mm/s<br>- 設定可能範囲：0～999999<br>- 0指定時は移動中なら減速停止、停止中なら停止状態維持<br>- 入力例：25.41mm/sなら2541と指定<br>- 最大速度超過でアラーム発生                                         | 4.3.2  |
| 加速度         | 16ビットデータ   | －   | - 16ビット整数<br>- 移動時の加減速度指定（加速度と減速度は同値）<br>- 単位：0.01G<br>- 指定可能範囲：1～999<br>- 入力例：0.30Gなら30と指定                                                                                 | 4.3.2  |
| 減速度         | 16ビットデータ   | －   | - 16ビット整数<br>- 移動時の減速度指定（加速度と減速度は同値）<br>- 単位：0.01G<br>- 指定可能範囲：1～999<br>- 入力例：0.30Gなら30と指定<br>- 最大加減速度超過でアラーム発生                                                         | 4.3.2  |
| 指令プログラムNo. | 16ビットデータ   | －   | - 16ビット整数<br>- プログラム起動指令またはプログラム原点移動指令のプレスプログラムNo.指定<br>- 指定可能範囲：0～63<br>- 指定後、プレスプログラムスタート信号(PSTR)をONでプログラム実行<br>- プレスプログラム原点移動信号(PHOM)をONで原点移動 | 4.3.2  |
