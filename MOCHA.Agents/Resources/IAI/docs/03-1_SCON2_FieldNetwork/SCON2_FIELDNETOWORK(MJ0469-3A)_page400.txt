### テキスト
#### 無効ID読出し（ID_RD コード：03H）
コントローラーの各IDの読出しを行うコマンドです。読出すIDのコードをID_CODEに設定します。

次の場合、アラームとなりコマンドを受け付けません。
1. ID_CODEがIDコード詳細表の範囲外を指定している（CMD_ALM=9）
2. OFFSETとSIZEの示す範囲が、読出すIDデータの範囲を超えた（CMD_ALM=9）
3. SIZEが24バイトを超えている（CMD_ALM=9）
4. OFFSETした範囲が、読出すIDデータの範囲を超えた（CMD_ALM=9）

### 表
#### 【ID_RD】コマンドフォーマット

| コマンドフォーマットのバイト | コマンド   | レスポンス | 備考                                           |
| :---------------------------- | :--------- | :--------- | :--------------------------------------------- |
| 0                             | 03H        | 03H        |                                                |
| 1                             | WDT        | RWDT       | [7.1.3【2】]参照                               |
| 2                             | CMD_CTRL   | CMD_STAT   | [7.1.3【3】]参照                               |
| 3                             |            |            |                                                |
| 4                             | ID_CODE    | ID_CODE    |                                                |
| 5                             | OFFSET     | OFFSET     | OFFSETを設定すると、IDデータの途中から指定のサイズだけ読出すことができます。 |
| 6                             | SIZE       | SIZE       |                                                |
| 7                             |            |            |                                                |
| 8 - n※1                      | 予約       | ID         |                                                |

※1 コマンドフォーマットのバイトモードによりnの数値が変わります。  
・16バイトモード：n=15  
・32バイトモード：n=31  
・48バイトモード：n=47
