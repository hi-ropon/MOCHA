### テキスト
#### 4.3 プレスプログラムの運転方法

- 図はPLCとコントローラー間の各種信号の送受信タイミングを示しています。
- 信号は「軸移動許可ENMV」「加圧動作制御モード」「加圧動作停止方法」「目標位置データ」「速度データ」「加速度データ」「減速度データ」「加圧荷重データ」「プレス直値スタート指令DPST」「加圧動作中DPRS」「加圧動作終了DPED」「現在荷重FBFC」「現在位置PNOW」「アクチュエーター動作（速度波形のイメージ）」に分類されます。
- 注釈
  - ※1 T1：上位コントローラーのスキャンタイムを考慮し、間隔を空ける必要がある。
  - ※2 Mtは「使用するフィールドネットワークの仕様」を参照。

### 図（タイミングチャート）
```mermaid
flowchart TD
    subgraph PLCからコントローラーへ
        ENMV[軸移動許可 ENMV]
        MODE[加圧動作制御モード：力制御モード]
        STOP[加圧動作停止方法：荷重停止]
        TARGET[目標位置データ]
        SPEED[速度データ]
        ACCEL[加速度データ]
        DECEL[減速度データ]
        LOAD[加圧荷重データ]
        DPST[プレス直値スタート指令 DPST]
    end

    subgraph コントローラーからPLCへ
        DPRS[加圧動作中 DPRS]

        FBFC[現在荷重 FBFC]
        PNOW[現在位置 PNOW]
    end

    subgraph アクチュエーター動作
        ACT[速度波形のイメージ]
    end

    ENMV -->|1)| MODE -->|2)| STOP -->|3)| TARGET -->|4)| SPEED -->|5)| ACCEL -->|6)| DECEL -->|7)| LOAD -->|8)| DPST -->|9)| DPRS -->|10)| DPRS -->|11)| DPRS -->|12)| DPED -->|13)| DPED
    FBFC -.->|交差| FBFC
    PNOW -.->|交差| PNOW
    ACT -.->|速度波形| ACT

    note1["※1 T1：上位コントローラーのスキャンタイムを考慮し、間隔を空けてください。"]
    note2["※2 Mtは「使用するフィールドネットワークの仕様」を参照してください。"]

    DPST ---|T1| DPRS
    DPRS ---|≤1Mt| DPED
    FBFC ---|≤1Mt| FBFC
    PNOW ---|≤1Mt| PNOW
```

**図の説明:**
- 青矢印はPLCからコントローラーへの信号送信の順序を示す。
- オレンジの線はコントローラーからPLCへの信号送信を示す。
- 黒線はアクチュエーターの速度波形イメージを示す。
- 信号の送受信タイミングはT1やMtの間隔を空ける必要がある。
