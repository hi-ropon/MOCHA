### テキスト
#### 7.2.2 通信仕様
SCON2のModbus/TCP通信仕様について説明しています。  
その他の仕様は「シリアル通信（Modbus版）取扱説明書」を参照するよう案内しています。  
また、SCON2ではping応答も使用可能であることを記載しています。

※ Pingとは、ICMP（Internet Control Message Protocol）のechoコマンドを利用し、  
指定した相手先（IPアドレスまたはホスト名）に文字列を送り、戻りの有無でネットワーク接続を確認するものです。

### 表
#### SCON2のModbus/TCP通信仕様

| 項目             | 内容                         |
| :--------------- | :--------------------------- |
| 伝送速度         | 10M/100Mbps                  |
| 物理層           | 100BASE-TX                   |
| トポロジー       | スター型                     |
| 通信ケーブル     | ツイストペア：カテゴリー5以上 |
| LANコネクター    | RJ-45                        |
| 通信プロトコル   | Modbus/TCP                   |
| 使用するポート番号 | 502                          |
