### テキスト
#### 3.4 プレスプログラム仕様の動作モード
プログラム制御ステージの動作について説明しています。

### 図
プログラム制御ステージの動作フローを示しています。時間軸に沿って以下の7つのステージが順に進行します。

```mermaid
flowchart LR
    A[Prg原点] --> B[1.アプローチ]
    B --> C[2.探り]
    C --> D[3.加圧]
    D --> E[4.停止]
    E --> F[5.減圧]
    F --> G[6.戻り]
    G --> H[7.待機]
    H -->|時間| I[Prg原点]
```

**図の説明:**
- 1.アプローチ：プログラム原点から位置へ移動開始
- 2.探り：対象物の検出や位置調整
- 3.加圧：対象物に圧力を加える動作
- 4.停止：加圧後の停止状態
- 5.減圧：圧力を減らす動作
- 6.戻り：原点へ戻る動作
- 7.待機：次の動作まで待機

### 表
#### 加圧動作モード一覧

| No. | 加圧モード   | 停止方式       | 停止状態       |
| :-- | :----------- | :------------- | :------------- |
| 1   | 速度制御     | 位置停止       | 位置決め停止   |
| 2   | 速度制御     | 距離停止       | 位置決め停止   |
| 3   | 速度制御     | 荷重停止       |                |
| 4   | 速度制御     | 増分荷重停止   |                |
| 5   | 力制御       | 位置停止       | 押付け動作継続 |
| 6   | 力制御       | 距離停止       | 押付け動作継続 |
| 7   | 力制御       | 荷重停止       | 押付け動作継続 |
| 8   | 力制御       | 増分荷重停止   | 押付け動作継続 |
| 9   | 力制御       | 位置停止2      | 位置決め停止   |

- 「速度制御」と「力制御」の2種類の加圧モードがあり、それぞれ複数の停止方式が設定可能。
- 停止状態は「位置決め停止」と「押付け動作継続」の2種類に分類される。
