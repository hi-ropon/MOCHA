### テキスト
#### PLC出力のアドレス（nは各軸の先頭アドレス）
PLCの出力アドレスは、各軸の先頭アドレスnを基準に割り当てられています。1ワードは16ビットで構成されており、各ビットの役割が詳細に定義されています。

### 表
#### PLC出力アドレスとビット割り当て

| アドレス | CC,CIE | DV | EC,EP,PRT | ML3 | 内容 |
|---------|--------|----|-----------|-----|------|
| RWw (n+0) | n+0 | n+0 | n+1 | n+4, n+5 | 目標位置（下位ワード） b15〜b0 |
| RWw (n+1) | n+1 | n+2, n+3 | n+6, n+7 | | 目標位置（上位ワード） b15〜b0 |
| RWw (n+2) | n+2 | n+4, n+5 | n+8, n+9 | | 位置決め幅（下位ワード） b15〜b0 |
| RWw (n+3) | n+3 | n+6, n+7 | n+10, n+11 | | 位置決め幅（上位ワード） b15〜b0 |
| RWw (n+4) | n+4 | n+8, n+9 | n+4, n+5 | | 速度 b15〜b0 |
| RWw (n+5) | n+5 | n+10, n+11 | n+6, n+7 | | 加減速度 b15〜b0 |
| RWw (n+6) | n+6 | n+12, n+13 | n+8, n+9 | | 押付け電流制限値 b15〜b0 (ビットごとの値: 256,128,64,32,16,8,4,2,1) |
| RWw (n+7) | n+7 | n+14, n+15 | n+10, n+11 | | 制御信号 b15〜b0 (ビット名: BKRL, RMOD, DIR, PUSH, GSLL, GSLO, -, JOG+, JOG-, JVEL, JISL, SON, RES, STP, HOME, DSTR) |

- 目標位置は負数の場合、2の補数で表されます。
- 制御信号の各ビットは特定の制御機能を表しています。
