### テキスト
#### ジョグ速度/インチング距離切替え（JVEL）【PLC出力信号】
ジョグ動作またはインチング動作が選択されている時のジョグ速度またはインチング距離を指定するパラメーターの切替え信号です。  
- JVELがONでジョグ動作の場合、速度レジスター値が0ならパラメーターNo.47「ジョグ速度2」の値がジョグ速度になる。  
- 速度レジスター値が0でない場合は、その速度レジスター値がジョグ速度になる。  
設定と動作の詳細は「ジョグ/インチング切替え（JISL）」の設定と動作の関係を参照。

#### ジョグ/インチング切替え（JISL）【PLC出力信号】
ジョグ動作とインチング動作の切替え信号。  
- JISL=OFF：ジョグ動作  
- JISL=ON：インチング動作  
JISL信号がジョグ移動中にON（インチング）に切替わると減速停止しインチング機能となる。  
JISL信号がインチング移動中にOFF（ジョグ）に切替わると移動完了後にジョグ機能となる。  
設定と動作の詳細は「JVEL/JISLの設定と動作の関係」を参照。

#### JVEL/JISLの設定と動作の関係
ジョグ速度/インチング距離切替え信号（JVEL）とジョグ/インチング切替え信号（JISL）のON/OFFの関係は以下の表の通り。

### 表
#### JVEL/JISLのON/OFF関係表

| JISLの状態 | OFF（ジョグ動作） | ON（インチング動作） |
| :--- | :--- | :--- |
| **JVELの状態** | **動作条件** | JOG+/JOG-がONのとき | JOG+/JOG-の立ち上がり（ONエッジ）を検出したとき |
| **JVEL=OFF** | 速度 | パラメーターNo.26 “ジョグ速度” | パラメーターNo.26 “ジョグ速度” |
|  | 移動距離 | – | パラメーターNo.48 “インチング距離” |
|  | 加減速度 | 定格値（アクチュエーター依存） | 定格値（アクチュエーター依存） |
| **JVEL=ON** | 速度 | パラメーターNo.47 “ジョグ速度2” | パラメーターNo.47 “ジョグ速度2” |
|  | 移動距離 | – | パラメーターNo.49 “インチング距離2” |
|  | 加減速度 | 定格値（アクチュエーター依存） | 定格値（アクチュエーター依存） |
