### テキスト
#### 2.4 EtherCAT
EtherCAT®とはEthernet for Control Automation Technologyの略称です。BECKHOFF社で開発されたリアルタイムイーサネットを使用したマスターとスレーブ間オープンネットワーク通信のことです。本ネットワークは、ETG（EtherCAT Technology Group）で管理されています。

SCON2のEtherCATモーションネットワーク仕様については、「SCON2モーションネットワーク取扱説明書 EtherCAT編（MJ0472）」を参照してください。

### 表
#### 2.4.1 EtherCAT®インターフェイス仕様

| 項目               | 仕様                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 通信規格           | IEC61158type12                                                                         |
| 物理層             | 100Base-TX（IEEE802.3）                                                                |
| 通信速度           | マスターに自動追従                                                                       |
| 通信ケーブル長     | EtherCAT®接続仕様による（各ノード間距離：100m以内）                                     |
| スレーブタイプ     | I/Oスレーブ                                                                             |
| 設定可能ノードアドレス | Station Alias（設定範囲：1～65535）                                                    |
| 通信ケーブル¹       | カテゴリー5以上（アルミテープと編組の二重遮蔽シールドケーブル推奨）                      |
| 接続コネクター     | RJ-45コネクター2個（入力×1、出力×1）                                                  |
| 接続               | デイジーチェーン限定                                                                     |

注1 通信ケーブルは、別途用意してください。
