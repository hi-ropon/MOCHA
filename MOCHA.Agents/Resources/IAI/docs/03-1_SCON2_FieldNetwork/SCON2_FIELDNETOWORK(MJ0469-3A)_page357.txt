### テキスト
#### プレスプログラムの運転方法における信号一覧
- 軸移動許可 ENMV (PLC→コントローラー)
- 加圧動作制御モード：速度制御モード (PLC→コントローラー)
- 加圧動作停止方法：荷重停止 (PLC→コントローラー)
- 目標位置データ (PLC→コントローラー)
- 速度データ (PLC→コントローラー)
- 加速度データ (PLC→コントローラー)
- 減速度データ (PLC→コントローラー)
- 加圧荷重データ (PLC→コントローラー)
- プレス直値スタート指令 DPST (PLC→コントローラー)
- 加圧動作中 DPRS (コントローラー→PLC)
- 加圧動作終了 DPED (コントローラー→PLC)
- 現在荷重 FBFC (コントローラー→PLC)
- 現在位置 PNOW (コントローラー→PLC)
- アクチュエーター動作（速度波形のイメージ）

#### 注意事項
- T1：上位コントローラーのスキャンタイムを考慮し、間隔を空けること
- Mt：使用するフィールドネットワークの仕様を参照

### 図の説明
- 上から順にPLCからコントローラーへ送信される各種制御信号やデータが示されている。
- それに続き、コントローラーからPLCへ送信される加圧動作中・終了信号や現在の荷重・位置データがタイミングチャートで表現されている。
- アクチュエーターの速度波形イメージも示され、信号の変化に対応した動作イメージがわかる。
- T1は上位コントローラーのスキャンタイムを考慮した待ち時間を示し、Mtはフィールドネットワークの仕様に基づく通信遅延の目安を示している。
