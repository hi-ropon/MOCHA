### テキスト
#### フィールドバスノードアドレス（パラメーターNo.85）
スレーブのノードアドレスを指定します。接続するネットワークモジュールによって設定範囲、出荷時設定値は異なります。

**注意**  
- ノードアドレスが他の機器と重複しないようにしてください。  
- ノードアドレスが重複している場合は、マスターユニット側で通信異常が発生します。  
- 詳細は「PLCならびにマスターユニットの取扱説明書」を参照してください。

### 表
| ネットワーク種別 | 設定範囲               | 出荷時設定値  |
| :--------------- | :--------------------- | :------------ |
| CC               | 1～64                  | 1             |
| CIE              | 1～120                 | 1             |
| DV               | 0～63                  | 63            |
| EC               | 未使用（マスターから設定） | 0             |
| EP               | 未使用（IPアドレスを設定） | -             |
| PRT              | 未使用（マスターから設定） | -             |
| ML3              | 3_H (3) ～ EF_H (239)  | 3_H (3)       |

※1 ECにおいて「0」を設定した場合、モジュール内部EEPROMに書き込まれたノードアドレスで動作します。また、このときマスターからノードアドレスの変更が可能です。
