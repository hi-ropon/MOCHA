### テキスト
#### ネットワークタイプ（パラメーターNo.87）
ネットワークモジュールの種類を指定するパラメーターです。初期値から変更しないよう注意が促されています。

#### フィールドバス入出力フォーマット（パラメーターNo.90）
PLCの入出力ビットレジスターおよび入出力データレジスターとの交信域で、2ワード内のデータをバイト単位で入れ替えて送受信可能です。パラメーターはマスター側のバイトオーダーに合わせて設定します。

工場出荷時にはネットワークごとに設定値が異なります。

### 表
#### ネットワークタイプ（パラメーターNo.87）設定値一覧

| ネットワーク種別 | 設定値 |
| :--- | :--- |
| CC | 1 |
| CIE | 11 |
| DV | 2 |
| EC | 6 |
| EP | 7 |
| ML3 | 10 |
| PRT | 8 |

注記：  
CC：CC-Link  
CIE：CC-Link IE Field  
DV：DeviceNet  
EC：EtherCAT  
EP：EtherNet/IP  
ML3：MECHATROLINK-III（リモートI/O）  
PRT：PROFINET IO  

#### フィールドバス入出力フォーマット（パラメーターNo.90）設定値と内容

| 設定値 | 内容 |
| :--- | :--- |
| 0 | 入替えは行いません。PLCにはそのまま送信されます。 |
| 1 | 上位ワードの上位バイトと下位バイトを入替え、また下位ワードの上位バイトと下位バイトを入替えます。 |
| 2 | ワードレジスターの場合、上位ワードと下位ワードを入替えます。 |
| 3 | 上位ワードの上位バイトと下位バイトを入替え、また下位ワードの上位バイトと下位バイトを入替えます。ワードレジスターの場合、さらに上位ワードと下位ワードを入替えます。 |
