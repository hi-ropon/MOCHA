### テキスト
#### PLC出力のアドレス説明
- アドレスはnを各軸の先頭アドレスとして示している。
- 1ワードは16ビットで構成される。

#### 各データの内容
- 目標位置（下位ワード・上位ワード）
- 位置決め幅（下位ワード・上位ワード）
- 速度
- 加減速度
- 押付け電流制限値
- 制御信号

※目標位置が負数の場合は2の補数で表される。

### 表（ビット割り当て）

| アドレス | CC,CIE (RWw) | DV | EC,EP,PRT | ML3 | 内容 |
| :------- | :----------- | :-:| :-------- | :-- | :--- |
| n+0 (下位ワード) | b15〜b0 |  |  |  | 目標位置（下位ワード） |
| n+1 (上位ワード) | b15〜b0 |  |  |  | 目標位置（上位ワード） |
| n+2 (下位ワード) | b15〜b0 |  |  |  | 位置決め幅（下位ワード） |
| n+3 (上位ワード) | b15〜b0 |  |  |  | 位置決め幅（上位ワード） |
| n+4 | b15〜b0 |  |  |  | 速度 |
| n+5 | b15〜b0 |  |  |  | 加減速度 |

### ビット構成の説明
- 各ワードは16ビットで構成され、b15が最上位ビット、b0が最下位ビット。
- 目標位置や位置決め幅は2ワード（32ビット）で表現される。
- 押付け電流制限値はビットごとに256、128、64、32、16、8、4、2、1の値を持つ。
- 制御信号は各ビットに特定の機能名が割り当てられている。
