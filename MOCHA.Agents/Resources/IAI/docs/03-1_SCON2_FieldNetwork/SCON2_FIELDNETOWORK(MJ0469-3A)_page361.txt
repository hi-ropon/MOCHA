### テキスト
#### 4.3 プレスプログラムの運転方法

- 軸移動許可 ENMV (PLC→コントローラー)
- 目標位置データ (PLC→コントローラー)
- 速度データ (PLC→コントローラー)
- 加速度データ (PLC→コントローラー)
- 減速度データ (PLC→コントローラー)
- 位置決め指令 DSTR (PLC→コントローラー)
- 加圧動作終了 DPED (コントローラー→PLC)
- 位置決め完了 PEND (コントローラー→PLC)
- 現在荷重 FBFC (コントローラー→PLC)
- 現在位置 PNOW (コントローラー→PLC)
- アクチュエーター動作（速度波形のイメージ）

##### 注釈
- ※1 T1：上位コントローラーのスキャンタイムを考慮し、間隔を空けてください。
- ※2 Mtは、「使用するフィールドネットワークの仕様」を参照してください。

```mermaid
flowchart TD

    end

    ENMV -->|1)| TargetPos
    TargetPos -->|2)| Speed
    Speed -->|3)| Accel
    Accel -->|4)| Decel
    Decel -->|5)| DSTR
    DSTR -->|6)| DPED
    DPED -->|7)| PEND
    PEND -->|8)| DPED

    note1["T1: 上位コントローラーのスキャンタイムを考慮し、間隔を空ける"]
    DSTR -.-> note1

    note2["Mt: 使用するフィールドネットワークの仕様を参照"]
    FBFC --- note2
    PNOW --- note2
```

**図の説明:**
- 青矢印はPLCからコントローラーへの信号送信の順序を示し、1)〜6)のステップで軸移動許可から位置決め指令までの流れを表しています。
- オレンジの波形はコントローラーからPLCへの応答信号（加圧動作終了、位置決め完了、現在荷重、現在位置）を示しています。
- 黒の波形はアクチュエーターの動作イメージ（速度波形）を表しています。
- T1は上位コントローラーのスキャンタイムを考慮した待ち時間を示し、Mtはフィールドネットワークの仕様に基づく時間間隔を示しています。
