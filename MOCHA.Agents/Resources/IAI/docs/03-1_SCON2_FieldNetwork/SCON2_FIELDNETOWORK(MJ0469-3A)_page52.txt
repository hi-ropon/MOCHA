### テキスト
#### 2.2 CC-Link IE Field
CC-Link IE Fieldは、伝送媒体としてIEEE802.3ab（1000BASE-T）準拠の高速・大容量なフィールドネットワークです。

SCON2はCC-Link IE Fieldに接続することで、省配線によるシステム構築が可能です。  
制御については、I/O制御のほか、SLMP（Seamless Message Protocol）通信を利用してマスターからパラメーターのバックアップやリストア※1、リモートリセットが可能です。

※1 ポジションデータはバックアップとリストアの対象に含まれません。  
※ CC-Link IE Fieldの詳細は、使用するPLCおよびマスターユニットの取扱説明書を参照してください。  
※ 三菱電機製のCC-Link IE Fieldマスター以外の接続確認は行っていません。他社マスターとの接続はお客様で確認してください。  
※ 本取扱説明書は各コントローラーの取扱説明書とあわせて使用してください。また、本取扱説明書で可能と表現されている以外の使い方はできません。

注：SCON2はCC-Link IE Field上において「インテリジェントデバイス局」として扱われます。

---

### 注意
- SCON2はCC-Link IEの中でも、CC-Link IE Fieldのみに対応しています。
