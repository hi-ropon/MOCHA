### テキスト
#### 機器セットアップ要求（CONFIGコード：04H）
- 共通パラメーターの再計算およびセットアップ要求コマンド。
- SCON2のCONFIG_MODは“0”にのみ対応し、このときCMDRDY=1を返す。
- CONFIG_MODに“0”以外を設定するとアラームとなりコマンドを受け付けない（CMD_ALM=9）。

| コマンドフォーマットのバイト | コマンド    | レスポンス   | 備考               |
| :---------------------------- | :---------- | :----------- | :----------------- |
| 0                             | 04H         | 04H          |                    |
| 1                             | WDT         | RWDT         | [7.1.3 (2)]参照    |
| 2                             | CMD_CTRL    | CMD_STAT     | [7.1.3 (3)]参照    |
| 3                             | CONFIG_MOD  | CONFIG_MOD   | コントローラーによる |
| 4                             | 予約        | 予約         |                    |

※1 コマンドフォーマットのバイトモードにより数値が変わる  
- 16バイトモード：n=15  
- 32バイトモード：n=31  
- 48バイトモード：n=47
