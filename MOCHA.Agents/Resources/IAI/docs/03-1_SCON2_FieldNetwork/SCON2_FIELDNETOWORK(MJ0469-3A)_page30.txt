### テキスト
#### 1.4 動作モードと機能

【1】リモートI/Oモード  
PIO（24V入出力）による運転をフィールドネットワーク通信で行う方式。

【2】ポジション/簡易直値モード  
ポジションNo.を指定して運転する方式。  
制御信号の切替で目標位置を直接数値指定するか、ポジションデータに登録した値を使用可能。  
速度、加減速度、位置決め幅などは事前登録したポジションデータの値を使用。  
設定可能なポジションデータ数は最大384点。

【3】ハーフ直値モード  
目標位置以外に速度、加減速度、押付け電流値を直接数値指定する運転方式。

### 図

#### リモートI/Oモードの接続イメージ
```mermaid
flowchart LR
    PLC1[PLC] -->|フラットケーブル| SCON2_PIO[PIO仕様のSCON2]
    SCON2_PIO -->|通信ケーブル| SCON2_FN[フィールドネットワーク対応のSCON2]
    PLC2[PLC] --> SCON2_FN
```
* PLCとPIO仕様のSCON2はフラットケーブルで接続  
* フィールドネットワーク対応のSCON2とは通信ケーブルで接続

#### ポジション/簡易直値モードの信号フロー
```mermaid
flowchart LR
    PLC -->|指定ポジションNo.0 目標位置:100.00mm| SCON2_FN[フィールドネットワーク対応のSCON2]
    SCON2_FN -->|現在位置:100.00mm 完了ポジションNo.0| PLC
    SCON2_FN --> アクチュエーター
```
* PLCからSCON2へ目標位置とポジションNo.を送信  
* SCON2からPLCへ現在位置と完了ポジションNo.を返送  
* SCON2はアクチュエーターを制御

#### ハーフ直値モードの信号フロー
```mermaid
flowchart LR
    PLC -->|目標位置:100.00mm 位置決め幅:0.10mm 速度指定:100.0mm/s 加減速度:0.30G 押付け電流制限値:50%| SCON2_FN[フィールドネットワーク対応のSCON2]
    SCON2_FN -->|現在位置:55.00mm 指令電流:-100% 現在速度:100.0mm/s アラームコード:0| PLC
    SCON2_FN --> アクチュエーター
```
* PLCからSCON2へ目標位置、速度、加減速度、押付け電流値を直接数値指定  
* SCON2からPLCへ現在位置、指令電流、現在速度、アラームコードを返送  
* SCON2はアクチュエーターを制御

---

### 図の説明
- PLCとSCON2間の通信形態がモードごとに異なることを示している。  
- 各モードでの信号の送受信内容や制御対象（アクチュエーター）との接続関係が視覚的に理解できる。
