### テキスト
#### PLC出力のアドレス構成
- アドレスは各軸の先頭アドレスnを基準に割り当てられている。
- 1ワードは16ビットで構成されている。
- 目標位置は上位ワードと下位ワードに分かれており、負数の場合は2の補数で表現される。

### 表
#### PLC出力アドレスとビット割り当て

| 項目       | アドレス (nを基準) | 備考                     |
|------------|--------------------|--------------------------|
| 目標位置（上位ワード） | RWw(n+0), DV(n+0), EC,EP,PRT(n+0,n+1), ML3(n+4,n+5) | 16ビット×2ワードで表現 |
| 目標位置（下位ワード） | RWw(n+1), DV(n+1), EC,EP,PRT(n+2,n+3), ML3(n+6,n+7) | 16ビット×2ワードで表現 |
| 速度（上位ワード）     | RWw(n+2), DV(n+2), EC,EP,PRT(n+4,n+5), ML3(n+8,n+9) | 16ビット×2ワードで表現 |
| 速度（下位ワード）     | RWw(n+3), DV(n+3), EC,EP,PRT(n+6,n+7), ML3(n+10,n+11) | 16ビット×2ワードで表現 |
| 加速度                 | RWw(n+4), DV(n+4), EC,EP,PRT(n+8,n+9), ML3(n+12,n+13) | 16ビットワード          |
| 減速度                 | RWw(n+5), DV(n+5), EC,EP,PRT(n+10,n+11), ML3(n+14,n+15) | 16ビットワード          |
| 加圧荷重               | RWw(n+6), DV(n+6), EC,EP,PRT(n+12,n+13), ML3(n+16,n+17) | 16ビットワード          |
| 予約（使用不可）       | RWw(n+7), DV(n+7), EC,EP,PRT(n+14,n+15), ML3(n+18,n+19) | 予約領域                |

- 各ワードはb15からb0までの16ビットで構成されている。
- 目標位置の負数は2の補数で表される。

---

以上の内容はPLCの各種制御パラメータのメモリアドレスとビット割り当てを示しており、プレスプログラムの動作モードにおけるデータ構造の理解に役立つ。
