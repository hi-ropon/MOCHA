### テキスト
#### 2.2.1 CC-Link IE Field インターフェイス仕様
CC-Link IE Fieldのインターフェイス仕様は以下の通りです。

### 表
| 項目               | 仕様                                                                                   |
|--------------------|----------------------------------------------------------------------------------------|
| 通信規格           | IEEE802.3ab（1000BASE-T）準拠                                                        |
| 通信速度           | 1Gbps                                                                                  |
| 通信制御方式       | トークンパッシング方式                                                                  |
| 通信ケーブル長     | 各セグメント間の距離：100m以内                                                        |
| 最大接続台数       | 254台（マスター局とスレーブ局の合計）                                                 |
| 最大ネットワーク数 | 239                                                                                    |
| 最大局間距離       | 100m                                                                                   |
| 通信ケーブル※1    | カテゴリー5e以上（アルミテープと編組の二重遮蔽シールドケーブル推奨）                   |
| 接続コネクター     | RJ-45コネクター2個                                                                     |
| サイクリック通信   | RX（スレーブ→マスター）：16384ビット<br>RY（マスター→スレーブ）：16384ビット<br>RWr（スレーブ→マスター）：8192ワード<br>RWw（マスター→スレーブ）：8192ワード |
| トランジェント通信（メッセージ通信） | メッセージサイズ：最大2048バイト                                                   |

※1 通信ケーブルは別途用意してください。
</contents>
