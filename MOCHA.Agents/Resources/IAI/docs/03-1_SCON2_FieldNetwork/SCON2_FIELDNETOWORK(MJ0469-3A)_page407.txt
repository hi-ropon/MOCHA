### テキスト
#### COM_MODEフィールド詳細
COM_MODEフィールドは通信モードの設定を示すビットフィールドで、各ビットの役割と設定内容が説明されています。

### 表
#### COM_MODEフィールド詳細

| ビット | 略称      | 名称           | 内容                                                                                   |
| :----- | :-------- | :------------- | :------------------------------------------------------------------------------------- |
| 0      | －        | 予約           |                                                                                         |
| 1      | SYNCMODE  | 同期設定       | 0：非同期通信（通信フェーズP2へ遷移）<br>1：同期通信（非サポート）                   |
| 2      | DTMODE    | 通信方式       | 0：単送通信<br>1：連送通信（非サポート）                                             |
| 3      | －        | －             |                                                                                         |
| 4～6   | －        | 予約           |                                                                                         |
| 7      | SUBCMD    | サブコマンド設定 | 0：サブコマンド無効<br>1：サブコマンド有効（非サポート）                             |

</contents>
