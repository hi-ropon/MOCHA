### テキスト
#### アラーム・ワーニングクリア（ALM_CLR コード：06H）
- アラーム・ワーニングの解除指令を行うコマンド。
- 現在発生中のアラームを解除可能（ALM_CLR_MOD=0 のみ対応）。
- コントローラーの動作モードがAUTOの場合、通信を含むコントローラーに発生しているアラーム・ワーニングを解除。
- MANUモードの場合は通信に関連したアラーム・ワーニングのみ解除。
- ALM_CLR_MODは「0」のみ対応し、それ以外はアラームとなりコマンドを受け付けない（CMD_ALM=9になる）。

#### 【ALM_CLR】コマンドフォーマット

| コマンドフォーマットのバイト | コマンド       | レスポンス   | 備考               |
| :---------------------------- | :------------- | :----------- | :----------------- |
| 0                             | 06H            | 06H          |                    |
| 1                             | WDT            | RWDT         | [7.1.3 (2)]参照    |
| 2                             | CMD_CTRL       | CMD_STAT     | [7.1.3 (3)]参照    |
| 3                             | ALM_CLR_MOD    | ALM_CLR_MOD  | 0だけ対応          |
| 4                             |                |              |                    |
| 5                             |                |              |                    |
| 6 - n※1                      | 予約           | 予約         |                    |

※1 コマンドフォーマットのバイトモードにより数値が変わる  
- 16バイトモード：n=15  
- 32バイトモード：n=31  
- 48バイトモード：n=47
