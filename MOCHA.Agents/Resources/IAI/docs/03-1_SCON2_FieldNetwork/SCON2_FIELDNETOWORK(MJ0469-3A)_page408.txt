### テキスト
#### コネクション開放要求（DISCONNECT コード：0F_H）
通信（コネクション）を開放し、通信フェーズP1に遷移することを要求するコマンドです。  
本コマンドはどのような状態でも実行可能です。  
CMD_STAT、CMDRDY=0の時に本コマンドが発行された場合は、処理を中断して本コマンドの処理を実行します。  
※ コントローラーの動作モードがAUTOの場合、本コマンドを実行するとサーボOFFし、ブレーキをかけます。

### 表
#### 【DISCONNECT】コマンドフォーマット

| コマンドフォーマットのバイト | コマンド | レスポンス | 備考 |
| :---------------------------- | :------- | :--------- | :--- |
| 0                             | 0F_H     | 0F_H       |      |
| 1 - n※1                      | 予約     | 予約       |      |

※1 コマンドフォーマットのバイトモードにより数値が変わります。  
・16バイトモード：n = 15  
・32バイトモード：n = 31  
・48バイトモード：n = 47
