### テキスト
#### アラーム・ワーニング読出し（ALM_RD コード：05H）
コントローラーのアラームおよびワーニングの読出しコマンドです。  
SCON2のALM_RD_MODは「0」のみ対応します。  
ALM_RD_MODを「0」に設定した場合、現在発生中のアラームコードの読出しを行います。  
読出したアラームコードはレスポンスのバイト8、9に格納されます。  
読出したアラームデータが「0」の場合は、現在アラームが発生していないことを示します。

次の場合はアラームになりコマンドを受け付けません。  
1. ALM_RD_MODが3でALM_INDEXが範囲外を設定（CMD_ALM=9になります）  
2. ALM_RD_MODに0または3以外を設定（CMD_ALM=9になります）

### 表
#### ALM_RD コマンドフォーマット

| コマンドフォーマットのバイト | コマンド       | レスポンス    | 備考                     |
| :---------------------------- | :------------- | :------------ | :----------------------- |
| 0                             | 05H            | 05H           |                          |
| 1                             | WDT            | RWDT          | [7.1.3 (2)]参照          |
| 2                             | CMD_CTRL       | CMD_STAT      | [7.1.3 (3)]参照          |
| 3                             |                |               |                          |
| 4                             | ALM_RD_MOD     | ALM_RD_MOD    | 下表参照                 |
| 5                             |                |               |                          |
| 6                             | ALM_INDEX      | ALM_INDEX     | 0～15（0=最新）          |
| 7                             |                |               |                          |
| 8 - n※1                      | 予約           | ALM_DATA      |                          |

※1 コマンドフォーマットのバイトモードによりnの数値が変わる  
- 16バイトモード：n=15  
- 32バイトモード：n=31  
- 48バイトモード：n=47
