### テキスト
#### EtherCATの配線方法
EtherCAT接続用の2個のRJ-45コネクターは、INにネットワーク上流側、OUTにネットワーク下流側のケーブルを接続します。  
配線方法は以下の通りです。  
1. EtherCAT用ケーブルをマスターユニットに接続し、もう一方のコネクターをSCON2のINコネクターに挿入する。  
2. SCON2のOUTコネクターにEtherCAT用ケーブルのコネクターを接続する。  
※ もう一方のコネクターは次に接続するスレーブユニットのコネクターに挿入します。

### 図
```mermaid

    PLC[PLC (マスターユニット)] -->|EtherCATケーブル| SCON2_IN[SCON2 INコネクター]
    SCON2_IN --> SCON2[SCON2 EtherCAT仕様]
    SCON2 -->|OUTコネクター| SCON2_OUT[SCON2 OUTコネクター]
    SCON2_OUT --> Slave[スレーブユニット INコネクター]
    Slave -->|OUTコネクター| 次スレーブユニット[次のスレーブユニット]
```
**図の説明**  
- マスターユニット側の配線は製品の取扱説明書を参照  
- EtherCATは終端処理が不要であることを示す

### ポイント
| 項目 | 内容 |
| :--- | :--- |
| Ethernetケーブル | カテゴリー5以上のストレートケーブル、100m以内推奨（アルミテープと編組の二重遮蔽シールドケーブル推奨） |
| スレーブ側終端コネクター | 何も接続しない（終端抵抗不要） |
| マスター側終端 | 「EtherCATマスターユニットの取扱説明書」を参照 |
