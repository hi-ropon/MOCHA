### テキスト
#### 3.2 ポジショナー仕様の動作モード
（1）PLCアドレス構成について説明しています。

注釈：
- CC-Linkの場合、占有領域は「システム領域」となり、リモートデバイス局のシステム領域であるため、他の目的には使用できません。
- 占有領域は他の目的に使用できず、アドレスの重複使用に注意が必要です。

### 表
#### SCON2側入カレジスターのPLCアドレス構成

| 項目           | CC、CIE          | DV    | EC、EP、PRT     | ML3           |
| :------------- | :--------------- | :---- | :-------------- | :------------ |
| 占有領域       | RY n0～nF        | －    | －              | －            |
| 占有領域注1    | RY(n+1)0～(n+1)F | －    | －              | －            |
| 目標位置       | RWw (n+0)        | n+0   | n+0、n+1        | n+4、n+5      |
| 指定ポジションNo. | RWw (n+1)        | n+1   | n+2、n+3        | n+6、n+7      |
| 制御信号       | RWw (n+2)        | n+2   | n+4、n+5        | n+8、n+9      |
|                | RWw (n+3)        | n+3   | n+6、n+7        | n+10、n+11    |

#### SCON2側出カレジスターのPLCアドレス構成

| 項目                   | CC、CIE          | DV    | EC、EP、PRT     | ML3           |
| :--------------------- | :--------------- | :---- | :-------------- | :------------ |
| 占有領域               | RX n0～nF        | －    | －              | －            |
| 占有領域注1            | RY(n+1)0～(n+1)F | －    | －              | －            |
| 現在位置               | RWr (n+0)        | n+0   | n+0、n+1        | n+4、n+5      |
| ポジション完了No.（簡易アラームコード） | RWr (n+1)        | n+1   | n+2、n+3        | n+6、n+7      |
| 状態信号               | RWr (n+2)        | n+2   | n+4、n+5        | n+8、n+9      |
|                        | RWr (n+3)        | n+3   | n+6、n+7        | n+10、n+11    |

### 注釈
- 注1：CC-Linkの場合、占有領域はシステム領域となり、リモートデバイス局のシステム領域であるため他の目的には使用できません。
- 占有領域は他の目的に使用できず、アドレスの重複使用に注意が必要です。
