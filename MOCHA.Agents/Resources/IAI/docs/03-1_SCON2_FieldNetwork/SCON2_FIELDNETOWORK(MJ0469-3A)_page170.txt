### テキスト
#### 軸ごとの入出力信号割付け
- 各軸の入出力信号は、入出力エリアの各8ワード（8ワード=16バイト）で構成される。
- 制御信号および状態信号はビット単位のON/OFF信号。
- 目標位置および現在位置は2ワード（32ビット）のバイナリーデータで、PLCでは-9999999～+9999999（単位：0.01mm（DDモーター以外）、0.001°（DDモーター））の数値を扱う。
  - 目標位置データは使用するアクチュエーターのソフトストロークの範囲内（0～有効ストローク長）で設定する。
- 位置決め幅は2ワード（32ビット）のバイナリーデータで、PLCでは1～+9999999（単位：0.01mm（DDモーター以外）、0.001°（DDモーター））の数値を扱う。
- 速度は1ワード（16ビット）のバイナリーデータで、PLCでは0～+65535（単位：1.0mm/sまたは0.1mm/s）の数値を扱うが、アクチュエーターの最大速度を超えない値に設定する必要がある。
- 単位はパラメーターNo.159 FBハーフ直値モード速度単位で設定。

### 表
#### パラメーターNo.159 設定値と速度単位

| パラメーターNo.159 設定値 | 速度単位   |
| :------------------------- | :--------- |
| 0                          | 1.0mm/s    |
| 1                          | 0.1mm/s    |

### テキスト（続き）
- 加減速度は1ワード（16ビット）のバイナリーデータで、PLCでは1～999（単位：0.01G）の数値を扱うが、アクチュエーターの最大加速度および最大減速度を超えない値に設定する。
- 押付け電流制限値は1ワード（16ビット）のバイナリーデータで、PLCでは0（0%）～510（200%）の数値を扱うが、アクチュエーターの押付け電流制限値の指定可能範囲内で設定する。
- 指令電流は2ワード（32ビット）のバイナリーデータ（単位：1mA）。
- 現在速度は2ワード（32ビット）のバイナリーデータ（単位：0.01mm/s）。
- アラームコードは1ワード（16ビット）のバイナリーデータ。
