### テキスト
#### CC-Link IE Fieldの設定項目
- 【3】フィールドバス通信速度（パラメーターNo.86）
  - 通信速度は「1Gbps」で固定されており、設定の必要はない。
- 【4】ネットワークタイプ（パラメーターNo.87）
  - コントローラーに搭載されているネットワークモジュールの種類を指定する。初期値から変更しないこと。
- 【5】フィールドバス入出力フォーマット（パラメーターNo.90）
  - PLCの入出力ビットレジスターおよび入出力データレジスターとの交信域で、2ワード内のデータをバイト単位で入れ替えて送受信可能。
- 【6】FBハーフ直値モード速度単位（パラメーターNo.159）
  - ハーフ直値モード、ハーフ直値2モード、ハーフ直値3モードで制御する場合の速度単位を選択。

### 表
#### フィールドバス入出力フォーマット（パラメーターNo.90）

| 設定値 | 内容                                                                                     |
| :----- | :--------------------------------------------------------------------------------------- |
| 0      | 入れ替えは行わない。PLCにはそのまま送信される。                                         |
| 1      | 上位ワードの上位バイトと下位バイトを入れ替え、また下位ワードの上位バイトと下位バイトを入れ替える。 |
| （出荷時設定） | ワードレジスターの場合、上位ワードと下位ワードを入れ替える。                           |
| 3      | 上位ワードの上位バイトと下位バイトを入れ替え、また下位ワードの上位バイトと下位バイトを入れ替える。ダブルワードの場合、さらに上位ワードと下位ワードを入れ替える。 |

#### FBハーフ直値モード速度単位（パラメーターNo.159）

| 設定値         | 速度単位  |
| :------------- | :-------- |
| 0（出荷時設定） | 1.0mm/s   |
| 1              | 0.1mm/s   |
