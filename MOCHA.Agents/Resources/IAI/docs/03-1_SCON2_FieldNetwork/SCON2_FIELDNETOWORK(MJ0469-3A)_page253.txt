### テキスト
#### 3.4 プレスプログラム仕様の動作モード
##### 3.4.1 リモートI/Oモード
PIO（実際のDC24V入出力）の代わりにフィールドバス通信で各I/O信号をON/OFFして運転するモードです。  
ティーチングツールを使用してプレスプログラムを作成します。  
本モードで制御可能なロボシリンダーの有効な主要機能は表に示されています。

### 表
#### 動作モードと主要機能
| 主要機能               | ○:直接制御 | ×:無効 |
|------------------------|------------|--------|
| 位置データ指定運転     |            | ×      |
| 速度・加減速度直接指定 |            | ×      |
| 現在位置読取り         |            | ×      |
| 加圧荷重直接指定       |            | ×      |
| 現在速度読取り         |            | ×      |
| プログラムNo.指定運転  | ○          |        |
| 判定結果読取り         | ○          |        |
| 現在荷重データ読取り   |            | ×      |
| 過負荷レベルモニター   |            | ×      |
| サーボゲイン切替え     | ○※1       |        |

※1 サーボゲインは、1つのプレスプログラムに1つのサーボゲインを登録できます。
</contents>
