### テキスト
#### 2.6.1 MECHATROLINK-Ⅲインターフェイス仕様
MECHATROLINK-Ⅲインターフェイス接続仕様を示しています。  
SCON2のMECHATROLINK-Ⅲモーションネットワーク仕様については、別冊の「SCON2モーションネットワーク取扱説明書 MECHATROLINK-Ⅲ編（MJ0471）」を参照するよう案内があります。

### 表
#### MECHATROLINK-Ⅲインターフェイス仕様

| 項目                     | 仕様                                      |
|--------------------------|-------------------------------------------|
| 物理層                   | Ethernet                                  |
| 伝送速度                 | 100Mbps                                   |
| 最大伝送距離（局間）     | 100m                                      |
| 最小局間距離             | 0.2m                                      |
| 接続形態                 | カスケード形 / スター形 / Point to Point形 |
| 接続局数（最大スレーブ数）| 62局                                      |
| 伝送周期                 | 0.125、0.250、0.5～4ms（0.5～4は0.5ms単位）|
| 通信周期                 | 0.125～32ms（1ms単位）                    |
| データ長                 | 16、32、48バイト                          |
| ノードアドレス           | 03H～EFH                                  |
| ケーブル                 | MECHATROLINK-Ⅲ専用ケーブル               |
| コネクター               | コントローラー側：インダストリアル ミニI/Oコネクター |
|                          | ケーブル側：インダストリアル ミニI/Oコネクター       |
